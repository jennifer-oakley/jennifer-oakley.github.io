About files

If you have the SWB file share mapped and are connected to the Weatherford network, you can load survey files and push corrections directly from Surveyor's Workbench. You can also download survey files and upload corrections from SWB Orchestrator. See "Downloading files" and "Uploading corrections".

All files uploaded for a run are stored in the SWB file share. A folder will be created for the job, using the job number for the folder name. Each run will have its own folder within the folder for the job. When a new run is created, all of the survey and FMS files for the previous run will be copied the new run folder.

Downloading files

After the rig engineer uploads the raw survey data, the run status on the dashboard will changed to Survey Received. You can manually download raw survey data and FMS files on the Dashboard page.

  • Click the More Actions button (), then select the Download Raw option to download the raw survey data.
  • Click the More Actions button (), then select the Download FMS option to download FMS files.

Uploading corrections

To upload a survey correction file:

  1. Go to the Dashboard page and click the More Actions button () in the row for the job.
  2. Select the Upload Corrections button.
  3. In the Upload Corrections window that opens, click the Choose button.
  4. Navigate to the file you want to upload, then click Open.

The corrections will be uploaded to the run, and the run status on the Dashboard page will be updated to Correction Sent. The rig engineers assigned to the job will receive a notification that new corrections are available, and the run status on the rig dashboard will be updated to Corrections Available. When a rig engineer views the corrections, the run status will reset to Waiting For Surveys. A new entry will also be added to the run log.
