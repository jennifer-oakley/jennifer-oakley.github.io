About user accounts

Before you create the job, you'll need to create user accounts for the rig engineers who will be uploading raw survey data and FMS files.If the new user has a Weatherford email address, they can log in to the rig dashboard using the Weatherford button on the log-in screen. See "Creating Weatherford user accounts". Otherwise, they'll to log in with their email address and password. See "Creating external user accounts".

Creating Weatherford user accounts

To create a Weatherford user account:

  1. Go to the Users page and click the Add User button at the top of the screen.
  2. In the User Details window that opens, enter the email address for the new user in the Existing Email field at the top of the window, then click the Search button to check for existing account details.
  3. Select the Weatherford User option.
  4. Enter the user's email address and their first and last name.
  5. In the Access field, select the level of access that the user should have.
  6. Click the Save button to save your changes and close the window.

The user account will be added to SWB Orchestrator, with the permissions granted by the access level that you assigned to the account. They will be able to log in to SWB Orchestrator using the Weatherford button on the log-in screen, without entering their log-in credentials.

Creating external user accounts

To create an external user account:

  1. Go to the Users page and click the Add User button at the top of the screen.
  2. In the User Details window that opens, enter the email address for the new user in the Existing Email field at the top of the window, then click the Search button to check for existing account details.
  3. Select the External User option.
  4. Enter the user's email address and their first and last name.
  5. In the Access field, select the level of access that the user should have.
  6. In the Password field, enter a password for the account.
  7. Click the Save button to save your changes and close the window.

The user account will be created, with the permissions granted by the access level that you assigned to the account. A welcome email will be automatically sent to the new user with the SWB Orchestrator URL, user name, and password.
