About chat

The Chat window allows you to send direct messages to other SWB Orchestrator users. There are two ways to open the Chat window:

  • To view all active chat conversations, click the Chat button () at the top of the screen.
  • To view the chat for a specific job, go to the Dashboard page and click the Chat button () in the row for the job.

Starting a chat

To start a new chat:

  1. Click the Chat button () at the top of the screen.
  2. In the Chat window that opens, click the Add User button () at the top of the list of chats.
  3. In the Add User window that opens, enter the name or email address of the person you want to send a chat to, then select their name from dropdown list.
  4. Click the Start button to create the chat conversation.

A new chat will appear in the list of conversations. Click on the chat to open it and start sending messages.

Adding people to a group chat

To add people to an existing group chat:

  1. In the Chat window, click on the chat you want to add someone to. The chat conversation will open in the left side of the Chat window.
  2. Click the Chat Settings button () next to the name of the chat.
  3. In the Chat Settings window that opens, click the Add Members button ().
  4. In the Participant List field, select the name of the person you want to add to the chat.
  5. Click the Add Members button.

The next time the user opens their chat window, the group chat will appear in their list of chats.

Archiving a conversation

To archive a chat conversation:

  1. In the Chat window, click on the chat you want to add someone to. The chat conversation will open in the left side of the Chat window.
  2. Click the Chat Settings button () next to the name of the chat.
  3. In the Chat Settings window that opens, click the Delete Chat button ().
  4. Click Yes on the confirmation message that appears.

The chat will be moved from the list of active chats to the list of archived chats. To view archived chats, click the Show Archive Chat button ().

Warning

Once a chat is archived, it cannot be reactivated.
