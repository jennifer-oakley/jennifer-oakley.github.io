About Canada Well imports

If you have well data from GeoScout, you can import the data into 5D after you create a field. Imported data includes sites, slots, wells, and definitive surveys. If the file contains data for plans, targets, casing points, formations, or other data, that data will not be imported.

To import data from GeoScout into 5D, there are three basic steps:

  1. Export the data from GeoScout to CSV file.
  2. Use the CanadaWellImport.exe utility to convert the CSV file into a text file that can be imported into 5D.
  3. Import the text file into 5D. See "Importing Canada Well files".
Note

If you do not have the CanadaWellImport.exe utility, submit a WISE query to request access to the utility. See "Reporting problems".

Importing Canada Well files

To import well data from GeoScout:

  1. In the Data List pane, right-click on the field and select the Import Wells option.
  2. In the Import Into 5D window that opens, go to the Import Format field and select the Canada Well Import option.
  3. In the Site Unit field, select the distance unit used in the import file.
  4. Click the Browse button to select the import file.
  5. In the window that opens, navigate to the file you want to import and click Open.
  6. Click the OK button.
  7. In the Import 5D Items window, review the list of items to be imported.
  8. Click the Import button to begin the import process.

The selected data will be imported into 5D and added to the current field. If a well does not contain a definitive survey, a current plan and a definitive survey will be added to the well. A blank current plan will be added to the imported well.
