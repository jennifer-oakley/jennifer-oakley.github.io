Field properties

Related Topics
Related Topics

  • Field Magnetic Parameters

The table below defines the properties in the Field Data window. This window appears when you create a new database or edit the field properties. To edit the properties of a field, go to Field Items > Field in the main menu and select the Edit Field option.

Property definitions for the Field Data window
Property Name Definition
Name Name of the field
Company Name Name of the client company. If the client company is not listed, enter the name of the company.
Projected Coordinate System

Coordinate system used to map latitude and longitude to northing and easting coordinates.

  • Local (No Map) – Does not use a coordinate system to map the location of the field. When this option is selected, the relationships between grid coordinates, geographic position, and grid convergence angles must be manually entered. Select this option if the projected coordinate system is not yet know.
  • Select new grid – Opens the Select Projected Coordinate System from Datasource window, which allows you to select the appropriate map grid. Select this option if the map grid is not listed. See "Projected coordinate systems".
Field Vertical Reference Datum

Reference datum that all vertical coordinates are relative to, such as elevation and TVD.

  • Mean Sea Level – Average height of the ocean
  • Lowest Astronomical Tide – Height of the ocean during the lowest tide
  • Normaal Amerstdams Peil (NAP) – Average high water mark in the Netherlands
Map Unit Distance units used in the selected map grid
Preferred Site Unit Distance unit that is the preferred unit for the site. This unit will be used as the default for all objects located in the field.
Dogleg Base Length Length used to record dogleg severity in the preferred site unit
Default Magnetic Parameters Magnetic parameters for the field as a whole. If you do not know the magnetic parameters, click the Calculate button to open the geo-magnetic calculator. See "Field Magnetic Parameters".
