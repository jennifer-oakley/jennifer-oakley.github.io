About custom coordinate systems

If you have advanced access to 5D, you can also create a custom coordinate system. There are three main components of a projected coordinate system:

  • Area of Use – The latitude and longitude range over which the coordinate system will be used. See "Creating an area of use".
  • Geodetic – The datum, or approximation to the Earth's shape.
  • Projection – The mapping between points on the spherical earth, described by the Geodetic element, and the flat plane onto which they are projected.

To open the coordinate system editor:

  1. Click the Create Field and Database button () in the main toolbar.
  2. In the Field Data window that opens, go to the Projected Coordinate System field.
  3. Click the Advanced ... button and enter the password provided by Drilling Services.
  4. Click the Add/Edit button.

The coordinate system editor will open in the Edit Datasource window. In this window, you can add new objects, such as areas of use and projected coordinate systems. See "Creating an area of use" and "Creating a projected coordinate system".

Tip

You can also create a copy of an existing object, then edit the copy as needed. To create a copy of an object, select the object you want to copy and click the Copy Object button (). See "Using the Edit Datasource window".

Creating an area of use

To create a custom area of use:

  1. In the Edit Datasource window, go to the left side of the window and expand the Areas of Use folder.
  2. Navigate to the folder where you want the area of use to be stored.
  3. Click the New Object button ().
  4. In the Area of Use Editor window that opens, go to the Name field and enter a name for the area of use.
  5. Go to the Definition tab.
  6. Select the Specify Extents Points option.
  7. Click the ... button next to the Point Style field to select the projected point style.
  8. In the Minimum Point section, enter the longitude and latitude of the minimum point in the area of use. If you selected a three-dimensional point style, you'll also need to select the height of the minimum point.
  9. In the Maximum Point section, enter the longitude and latitude of the maximum point in the area of use. If you selected a three-dimensional point style, you'll also need to select the height of the maximum point.
  10. Click the OK button to close the window.
  11. When you close the Edit Datasource window, click the Save button to save your changes.

The new area of use will be stored in the selected folder. It will also appear under the *Recent* folder.

Tip

You can store all of your custom areas of use in a separate folder. Create a new folder under the Areas of Use folder, then drag your custom areas of use to the new folder. See "Using the Edit Datasource window".

Creating a projected coordinate system

To create a custom projected coordinate system:

  1. In the Edit Datasource window, go to the left side of the window and expand the Coordinate Systems folder.
  2. Select the Projected category, then navigate to the folder where you want the projected coordinate system to be stored.
  3. Click the New Object button ().
  4. In the Projected Coordinate System Editor window that opens, go to the Name field and enter a name for the projected coordinate system.
  5. Go to the Definition tab
  6. Click the ... button next to the Area of Use field to select the area of use for the projected coordinate system.
  7. Click the ... button next to the Point Style field to select the projected point style.
  8. Click the ... button next to the Geodetic field to select the geodetic coordinate system.
  9. In the Projection field, select the appropriate map projection.
  10. Enter the parameters for the projected coordinate system in the table below the Projection field.
  11. If desired, click the Preview button to test the settings for the projected coordinate system. See "Testing a projected coordinate system".
  12. Click the OK button to close the window.
  13. When you close the Edit Datasource window, click the Save button to save your changes.

The new projected coordinate system will be stored in the selected folder. It will also appear under the *Recent* folder. Before you can assign the projected coordinate system to a field, you will need to restart 5D.

Tip

You can store all of your custom projected coordinate systems in a separate folder. Create a new folder under the Projected folder, then drag your custom projected coordinate systems to the new folder. See "Using the Edit Datasource window".

Testing a projected coordinate system

When you create a new projected coordinate system, it's a good idea to test the settings to make sure coordinate points are converted correctly. In the Preview Projected Coordinate System window, you can convert northing/easting points to latitude/longitude points and compare the results to known values.

To test the conversion from a projected point to a geodetic point:

  1. In the Projected Coordinate System Editor window, click the Preview button.
  2. Enter the easting value to be converted in the Easting field.
  3. Enter the northing value to be converted in the Northing field.
  4. Select the Source to Target option.
  5. Click the Convert button.

The converted values will appear under the Geodetic Point section, in the Longitude field and the Latitude field. The text color used for all values will change from red text to black text.

To test the conversion from a geodetic point to a projected point:

  1. In the Projected Coordinate System Editor window, click the Preview button.
  2. Enter the longitude value to be converted in the Longitude field.
  3. Enter the latitude value to be converted in the Latitude field.
  4. Select the Target to Source option.
  5. Click the Convert button.

The converted values will appear under the Projected Point section, in the Easting field and the Northing field. The text color used for all values will change from red text to black text.

Using the Edit Datasource window

The toolbar below the Folders pane, on the left side of the Edit Datasource window, is used to add and delete custom folders.

Functions of the toolbar buttons in the Folders pane
Toolbar Button Function
Adds a new folder to the selected category or folder. To rename the folder, right-click on the folder name and click the Rename button.
Delete the selected folder and all of the objects in the folder. This button is only enabled when a custom folder is selected.

The toolbar below the pane on the right side of the Edit Datasource window is used to add, edit, and delete custom objects, as well as create copies of existing objects.

Functions of the toolbar buttons in the right pane
Toolbar Button Function
Adds a new object to the selected category or folder.
Delete the selected object. This button is only enabled when a custom object is selected.
Opens the editing window for the selected object, which allows you to change the object definition. This button is only enabled when a custom object is selected.
Opens the viewing window for the selected object, which allows you to view the object definition.
Creates a copy of the selected object, which can be edited as needed.
