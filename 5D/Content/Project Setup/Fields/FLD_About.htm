Fields

A field is a geographical area which contains multiple oil or gas wells that produce from a continuous reservoir. In Weatherford 5D, the field is the highest level of the hierarchy in a 5D database. The Data List pane shows the sites, slots, wells, and lease lines that are in the field. In the Data List pane, the field is represented by the Field icon ().

After you create a field, you can add sites, import wells from Compass, change the projected coordinate system, create lease lines, and more. To update a field or add items to it, right-click on the field name in the Data List pane or go to Field Items > Field in the main menu.

  • Import Wells – Imports wells from Compass 2000, Compass 2003, or a Canada well import file. See "Importing wells".
  • Export Wells – Exports previously imported Compass 2003 wells to a Compass 2003 import file. See "Exporting wells".
  • Edit Field – Opens the Field Data window, which allows you to edit the properties of the field. See "Field properties".
  • Move to Projected Coordinate System – Moves the field and all associated sites, slots, wells, and targets to a different coordinate system. See "Changing the coordinate system".
  • Add Sites – Adds a new site to the field. See "New sites".
  • Add Exploration Well – Adds an exploration well to the field. See "Exploration wells".
  • Change Status – Changes the status of the field to closed, active, or planned. By default, new fields are active.
  • Create Lease Line – Adds a lease line to the field. See "Lease lines".
  • Create/Edit Link – Opens the Project Documentation Link window, which allows you to add or edit a documentation link for the field. See "Document links".
  • Survey Tools – Opens the Survey Tool Parameters window, which shows detailed information about the margin of error for each survey tool. See "Survey Tools".
  • Set Default Casing Points – Sets the default casing points for new wells added the field. See "Set default casing points".
  • Casing List – Opens the Casing List window, which allows you to view and edit custom casing sizes. See "Casing list".
  • Apply Geodetic Scale Factor – Applies a geodetic scale factor to all of the wells in the field. The position of the points along the wellbore will be recalculated to account for the curvature of the earth.
  • Reports – Generates reports for the sites and slots in the field. See "Field site reports" and "Field slot reports".
