About projected coordinate systems

When you create a new field, you'll need to select the projected coordinate system for the field. The projected coordinate system determines the mapping between latitude/longitude and northing/easting. It's also used to calculate the convergence angle that maps true north to grid north. See "Selecting a projected coordinate system".

If you have advanced access to 5D, you can also create a custom coordinate system. See "Custom coordinate systems".

Warning

The projected coordinate system can only be changed if the field doesn't contain any sites, slots, or wells. If the field contains any sites, slots, or wells, the projected coordinate system can only be changed by moving the field to another coordinate system. See "Changing the coordinate system".

Selecting a projected coordinate system

To select the projected coordinate system for a field:

  1. Click the Create Field and Database button () in the main toolbar.
  2. In the Field Data window that opens, go to the Projected Coordinate System field.
  3. Select the Select new grid option.
  4. In the Select Projected Coordinate System from Datasource window that opens, use the pane on the left side of the window to navigate to the desired coordinate system.
  5. Select the projected coordinate system for the field, then click OK.

The coordinate system will be listed as the projected coordinate system for the field in the Field Data window. See "Creating a database".

Tip

To view additional information about a coordinate system, select the coordinate system and click the Info ... button.
