About target sets

Before you can add the targets for a well, you'll need to create a target set. On the Edit Targets > Target Set tab, you can add and edit target sets. You can also edit the targets and associate the target set with a well.

To view the properties of a target set, go to the List Targets tab in the Data List pane to open the Edit Targets panel, then select the name of the target set. The positions of the targets for the selected target set will appear in the table on the right side of Data Entry pane. If the Edit Targets tab is not visible, click the Targets Data button () in the main toolbar.

Adding target sets

To add a target set:

  1. Go to the List Targets tab in the Data List pane to open the Edit Targets pane.
  2. On the Target Set tab, click the New button and enter a name of the target set.
  3. In the Reference field, select the reference that target coordinates will be relative to.
  4. In the Style field, select the coordinate format.
  5. In the TVD Reference field, select the vertical reference datum for the TVD of the targets.
  6. Click the Apply button to save your changes.
  7. Click the Save button () to save the database.

The new target set will be listed on the List Targets tab in the Data List pane. After you create the target set, you will need to add targets to it. See "Targets". You'll also need to associate the target set with a well. See "Associating a well with a target set".

Associating a well with a target set

To associate a well with a target set:

  1. Go to the List Wells tab in the Data List and select the well.
  2. Right-click on the well name and select the Edit/View Well option.
  3. In the Edit Well Data window, go to the Target Set field and select the name of the target set.
  4. Click OK to save your changes.

The target set will appear in the well properties. When you join the current plan to a target or create a 2 of 7 plan, the target set will be the default target set. See "Projecting to a target" and "Two of Seven planning".

Using custom colors

To apply custom colors to a target set:

  1. Go to the Edit Targets > Target Set tab in the Data Entry pane.
  2. Select the target set you want to apply custom colors to.
  3. Select the Custom Colours option.
  4. Click the Targets button and select the color you want to use for targets.
  5. Click the Uncert button and select the color you want to use for target uncertainty.
  6. Click the Apply button.

The colors used for the target set will be shown in the 3D Viewer window and the Plot window will be updated. These colors will override any colors selected in the 3D Viewer window or the Plot window.

Properties of target sets

The table below defines the properties of target sets. These properties appear on the left side of the Edit Targets > Target Set tab.

Properties of target sets
Property Name Definition
Name Name of the target set
Comment Any comment relevant to the target set
Reference Reference that target coordinates are relative to
Style

Format used to enter target coordinates.

  • Cartesian – Coordinates entered in terms of northing/easting or north/east offset.
  • Polar – Coordinates entered in terms of range and bearing. Only available when the site or the slot is selected for the coordinate reference.
TVD Reference Vertical reference datum that the target TVD is relative to
Latitude/Longitude Format used to enter coordinates. Only available when the field is selected for the coordinate reference.
Standard Deviations (Target Uncertainty) Expected amount of uncertainty in the target location, which determines the size of the reference ellipsoid.
Custom Colours Indicates that custom colors are used for the targets and the target uncertainty

The table below defines the properties of the targets in a target set. These properties appear in the data table on the right side of the Edit Targets > Target Set tab.

Properties of targets
Property Name Definition
ID Name of the target
TVD TVD of the target
Northing Northing coordinate of the target. Only available when the field is selected for the coordinate reference.
Easting Easting coordinate of the target. Only available when the field is selected for the coordinate reference.
Latitude Latitude of the target. Only available when the field is selected for the coordinate reference.
Longitude Longitude of the target. Only available when the field is selected for the coordinate reference.
North Offset Northing offset of the target. Only available when the site or slot is selected for the coordinate reference, using Cartesian coordinates.
East Offset Easting offset of the target. Only available when the site or slot is selected for the coordinate reference, using Cartesian coordinates.
Range Distance from the coordinate reference along the specified bearing. Only available when the site or slot is selected for the coordinate reference, using polar coordinates.
Bearing Clockwise angle from the local north reference. Only available when the site or slot is selected for the coordinate reference, using polar coordinates.
Shape Type Shape of the 3D volume around the target point. See "Target shapes".
Comment Any comment relevant to the target
Active Indicates that the target is active. If this option is not selected, the target will not be included when you join targets to the well.
Target Uncertainty Indicates that target uncertainty is enabled

Using the Target Set tab

The table below defines the buttons on the Edit Targets > Target Set tab.

Buttons on the Edit Targets > Target Set tab
Button Function
New Creates a new target set. See "Adding target sets".
Delete Deletes the selected target set, as well as all targets in the target set. If the target set is associated with a well, the target set will be automatically dissociated.
Associate Associates the target set with the selected well. Only visible when a well is selected in the Data List pane, and the well is not currently associated with the target set.
Dissociate Removes the association between the target set and the selected well. Only visible when a well is selected in the Data List pane, and the well is currently associated with the target set.
Copies information about the target set to the clipboard, including coordinate references, TVD references, and target coordinates
Apply Saves all changes to the target set
Discard Discards any changes since the target set was last saved
Moves the position of the selected target up in the list. Only active when there is a target selected in the data table.
Moves the position of the selected target down in the list. Only active when there is a target selected in the data table.
