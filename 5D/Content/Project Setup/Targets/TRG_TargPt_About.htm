About targets

Targets are used to define the points that a well needs to reach when it is drilled. On the Edit Targets > Target tab, you can add and edit the targets for the target set selected on the Target Set tab.

Each target is defined by the position of the target and the shape of the 3D volume around the target shape.

  • The target point is the origin of the target shape. The target point defines the position of the target in the field.
  • The target aim point is the point that the well will pass through when you join the target to the well. This point may be the same as the target point, or it may be offset from the target point.
  • The target shape is the uncertainly area around the target point. The target shape defines the allowable margin of error for reaching the target point. See "Target shapes".

To view the properties of a target, go to the Edit Targets > Target tab in the Data Entry pane and select the name of the target. The properties of the target shape for the selected target will appear on the right side of the Data Entry pane. If the Edit Targets tab is not visible, click the Targets Data button () in the main toolbar.

Adding targets

To add a target:

  1. Go to the Edit Targets > Target Set tab in the Data Entry pane.
  2. Select the target set that you want to add targets to, then go to the Target tab.
  3. Click the New button and enter a name for the target.
  4. In the TVD field, enter the TVD of the target point.
  5. In the Relative To section, enter the coordinates of the target point.
  6. If the target aim point is different from the target point, go to the Target Aim Point section and enter the TVD offset and the coordinate offsets.
  7. On the right side of the screen, select the target shape for the uncertainty area. See "Target shapes".
  8. Click the Apply button to save your changes.
  9. Click the Save button () to save the database.

The new target will be listed on the List Targets tab in the Data List pane, under the selected target. After you create the target, you can join the target to the current plan. See "Projecting to a target".

Using the Targets tab

The table below defines the buttons on the Edit Targets > Target tab.

Buttons on the Edit Targets > Target tab
Button Function
New Creates a new target. See "Adding targets".
Delete Deletes the selected target set
Apply Saves all changes to the target
Discard Discards any changes since the target was last saved
