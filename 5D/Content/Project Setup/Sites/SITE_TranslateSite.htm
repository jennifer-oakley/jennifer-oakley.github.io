About translating sites

If the original site location was entered incorrectly or the planned location has changed, you can move the site to a new physical location in the Translate Site window.

The following items are affected when a site is translated:

  • Slots – Translated by the same amount as the site.
  • Well plans – The current plan is translated. A new plan folder is created in the plan archive. Use the Translate Plan Archive option to include all archived plans.
  • Surveys – All surveys, including the definitive survey, are translated.
  • Target sets – Use the Translate Targets option to move targets associated with wells on the site.

To open the Translate Site window, right-click the site name in the Data List pane and select the Translate Site option.

Translating relative to current position

Use this method to translate the site by a fixed distance or angle from its current location. To translate the site relative to its current position:

  1. Right-click the site name in the Data List pane and select Translate Site.
  2. In the Relative To field, select Current Position.
  3. In the Specify By field, choose North/East Offset or Polar.
  4. If using North/East Offset, enter values in the North Offset and East Offset fields.
  5. If using Polar, enter values in the Range and Bearing fields.
  6. In the Elevation section, enter the elevation of the new site location.
  7. In the Unit field, select the unit used to enter the elevation.
  8. In the Datum field, select the datum that the elevation is relative to.
  9. If desired, select the desired options to translate the plan archive and targets in the Translate Options section.
  10. Click OK to apply the translation.

The site and its associated data will be moved according to the specified offsets or polar coordinates, and any selected plan or target data will be translated accordingly.

Translating using absolute coordinates

Use this method to move the site to a known set of absolute coordinates, such as grid or geographic coordinates. To translate the site using absolute coordinates:

  1. Right-click the site name in the Data List pane and select Translate Site.
  2. In the Relative To field, select Absolute.
  3. In the Specify By field, choose Northing/Easting or Latitude/Longitude.
  4. If using Northing/Easting, enter values in the Northing and Easting fields.
  5. If using Latitude/Longitude, enter values in the Latitude and Longitude fields.
  6. In the Elevation section, enter the elevation of the new site location.
  7. In the Unit field, select the unit used to enter the elevation.
  8. In the Datum field, select the datum that the elevation is relative to.
  9. If desired, select the desired options to translate the plan archive and targets in the Translate Options section.
  10. Click OK to apply the translation.

The site and its associated data will be moved to the specified absolute coordinates, and any selected plan or target data will be translated accordingly.

Field definitions

The table below defines the fields in the Translate Site window.

Field definitions for the Translate Site window
Field Description
Relative To

Defines the reference for the translation.

  • Current Position moves the site relative to its existing location.
  • Absolute moves the site to a specific set of coordinates.
Specify By

Defines the format for entering the new location. The available options depend on the selection in Relative To.

  • North / East Offset – Uses north offset and east offset to define the new location.
  • Polar – Uses range and bearing to define the new location.
  • Northing / Easting – Uses northing and easting coordinates to define the new location.
  • Latitude / Longitude – Uses latitude and longitude to define the new location.
North Offset Offset amount to move the site northward from its current location. Only available when the Current Position and North / East Offset options are selected.
East Offset Offset amount to move the site eastward from its current location. Only available when the Current Position and North / East Offset options are selected.
Range Distance from the current site position. Only available when the Current Position and Polar options are selected.
Bearing Direction of the new site location from the current site location. Only available when the Current Position and Polar options are selected.
Northing Northing coordinate for the new site location. Only available when the Absolute and Northing / Easting options are selected.
Easting Easting coordinate for the new site location. Only available when the Absolute and Northing / Easting options are selected.
Latitude Latitude of the new site location. Only available when the Absolute and Latitude / Longitude options are selected.
Longitude Longitude of the new site location. Only available when the Absolute and Latitude / Longitude options are selected.
Elevation Vertical position of the new site location.
Unit

Unit of measurement for the new site elevation.

  • Map – Elevation is entered using map units.
  • Site – Elevation is entered using site units.
Datum

Specifies the reference point for elevation.

  • Mean Sea Level – Elevation is entered relative to mean sea level.
  • Site TVD Reference – Elevation is entered relative to the site's TVD reference.
Translate Plan Archive When selected, all plans in the site’s plan archive are moved along with the current plan.
Translate Targets When selected, all targets in the associated target sets are moved along with the site.
