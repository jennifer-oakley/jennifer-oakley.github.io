About slots

A slot is a location on a site, from which a main well will be drilled. Slots can contain multiple main wells and sidetrack wells. Slots are the third level of the hierarchy in a 5D database. The Data List pane shows the wells that are in each slot. In the Data List pane, a slot is represented by the Slot icon ().

There are two primary sets of parameters used to define the properties of a slot:

  • The Location section defines the location of the slot within the site.
  • The Elevation section defines the elevation of the site.

Adding slots

To add a slot to a site:

  1. In the Data List pane, right-click on the site you want to add a slot to and select the Add Slots option.
  2. In the Add Slot window that opens, enter the name of the slot in the Name field.
  3. In the Coordinate Reference field, select the reference used for the slot location.
  4. Enter the location of the slot, using either latitude/longitude or northing/easting.
  5. In the Elevation section, select the reference the slot elevation is relative too, then enter the elevation of the slot.
  6. In the Elevation Reference field, select the reference used to determine the elevation of the slot.
  7. Update the remaining properties as needed. See "Slot properties".
  8. Click OK to close the window, then click the Save button () to save the database.

The new slot will be listed in the Data List pane, under the List Wells tab. To add wells or edit the slot, go to the Data List pane and right-click on the slot name. See "Slots". The location and elevation of the slot can be edited until you add a well to the slot. After you add a well, you will need to translate the slot to change the location or elevation. See "Translating Slots".

Slot properties

The table below defines the properties in the Add Slot window. This window appears when you create a new slot or edit the slot properties. To edit the properties of a slot, go to Field Items > Slot in the main menu and select the Edit/View Slot option.

Property definitions for the Add Slot window
Section Property Name Definition
Context Site Name of the site that the slot is associated with
-- Name Name of the slot
-- Status

Status of the slot

  • Closed – All main wells and sidetracks in the slot are completed. You will only be able to view the slot and create reports.
  • Active – The slot is in the execution phase. There are no restrictions for editing or creating child items.
  • Planned – The slot is in the planning phase. There are no restrictions for editing or creating child items.
-- Coordinate Reference

Reference used for the location of the slot

  • Map – Coordinates are map coordinates, and entered in map units.
  • Site – Coordinates are relative to the site center, and entered in site units.
Lat/Lon Latitude Latitude of the slot, entered in either decimal degrees or degrees, minutes, and seconds. If you enter the slot location in northing/easting, this value will be automatically calculated. Only available when the Map option is selected for the coordinate reference.
Longitude Longitude of the slot, entered in either decimal degrees or degrees, minutes, and seconds. If you enter the slot location in northing/easting, this value will be automatically calculated. Only available when the Map option is selected for the coordinate reference.
E/N Northing Northing offset of the slot, entered in the unit for the selected coordinate reference. If you entered the slot location in latitude/longitude, this value will be automatically calculated.
Easting Northing offset of the slot, entered in the unit for the selected coordinate reference. If you entered the slot location in latitude/longitude, this value will be automatically calculated.
Elevation Relative To

Vertical reference datum that the slot elevation is relative to.

  • Mean Sea Level – Elevation is relative to mean sea level.
  • Site TVD Reference – Elevation is relative to the TVD reference defined in the site properties.
Slot Elevation Elevation of the slot, relative to the selected vertical reference datum. Elevation should be entered in the units for selected coordinate reference.
Elevation Reference Elevation point used to define the elevation of the slot
Slot Surface Uncertainty Radius Set Slot Surface Uncertainty Radius Option to override the surface uncertainty radius defined for the site
Radius Amount of horizontal uncertainly in the position of the slot
σ Standard deviation of the uncertainty radius
