About magnetic parameters

Magnetic parameters define Earth's magnetic field characteristics at your field location and are required for accurate directional survey calculations. These parameters are used at multiple levels in the data hierarchy:

  • Field level – Default magnetic parameters for all wells in the field. See "Field Magnetic Parameters".
  • Well level – Specific parameters for individual wells (inherits from field by default).
  • Survey level – Parameters for specific surveys (inherits from well by default). See "Survey Magnetic Parameters".

You can set magnetic parameters manually or calculate them automatically using the built-in geo-magnetic calculator, which uses global magnetic field models to determine the correct values for your location and time period.

Note

The coordinate system should use WGS84 reference for accurate calculations.

Note

The BGGM model is only available to users with a valid British Geological Survey license. When using BGGM for the first time, you'll be prompted to provide the path to the BGGM coefficients file. This file path will be stored for future use.

Using the geo-magnetic calculator

The geo-magnetic calculator is available in the Edit Well Data window when creating a new well or editing the well properties. See "New main wells" and "Well properties".

To calculate magnetic parameters:

  1. In the Edit Well Data window, click the Advanced button.
  2. In the Edit Well Data - Advanced window that opens, go to the Magnetic Data section.
  3. Click the Calculate button to open the Geo-Magnetic Calculator window.
  4. Enter the latitude, longitude, and elevation of the field.
  5. Select the magnetic field model you want to use.
  6. In the Model Date field, enter the survey date when directional measurements will be taken.
  7. Select the Copy to 5D on Exit option.
  8. Click Exit to close the window and apply the calculated values.

The calculated values will be copied to the magnetic parameters in the Edit Well Data - Advanced window.

Field definitions

The table below defines the input fields and calculated values in the Geo-Magnetic Calculator window.

Input fields and calculated values in the Geo-Magnetic Calculator window
Parameter Description
Latitude Latitudinal coordinate of the well
Longitude Longitudinal coordinate of the well
Elevation Elevation of the well above sea level
Geo-Magnetic Model Name

Model used to calculate the magnetic parameters
Model Date

Date the magnetic parameters were calculated. This should typically be the survey date when directional measurements are taken, as Earth's magnetic field changes over time.
Field Strength Strength of the magnetic field at the well's location. This value is calculated automatically and cannot be edited.
Dip Angle between the local horizontal vector and the Earth's magnetic field vector. This value is calculated automatically and cannot be edited.
Dec Clockwise bearing of the horizontal component of the Earth's magnetic field vector from true north. This value is calculated automatically and cannot be edited.
Convergence

Angle between the center of the projected coordinate system and true north. This value is calculated automatically and cannot be edited.
Total Correction

Combined correction factor that includes both magnetic declination and grid convergence for converting between magnetic bearings and grid bearings. This value is calculated automatically and cannot be edited.
Checksum

Validation value used to verify the integrity of the calculated magnetic parameters. This value is calculated automatically and cannot be edited.
Copy to 5D on Exit When enabled, copies the calculated magnetic parameters to the Edit Well Data - Advanced window.
