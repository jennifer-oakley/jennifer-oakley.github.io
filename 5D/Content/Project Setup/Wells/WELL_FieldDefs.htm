Well properties

The table below defines the properties in the Edit Well Data window. This window appears when you create a new well or edit the well properties. To edit the properties of a well, go to Field Items > Well in the main menu and select the Edit/View Well option.

Property definitions for the Edit Well Data window
Section Property Name Definition
Context Site Name of the site that the well is associated with
Slot Name of the slot that the well is associated with
Well Name Name of the well
Identifier Unique, industry-approved ID number of the well
Status

Status of the well

  • Closed – The well has reached total depth, and the definitive surveys are finalized. You will only be able to view the well and create reports.
  • Active – The well is in the execution phase. There are no restrictions for editing or creating child items.
  • Planned – The well is in the planning phase. There are no restrictions for editing or creating child items.
-- Target Set Target set assigned to the well. See "Targets & Target Sets".
-- Comment Any relevant comments about the well

The table below defines the properties in the Edit Well Data - Advanced window. This window appears when you click the Advanced button while creating a well or editing the well properties.

Property definitions for the Edit Well Data window
Section Property Name Definition
Vertical Section - Start Position Relative To

Reference datum that the starting position of the vertical section is relative to

  • Site – Start position is relative to the site center.
  • Slot – Start position is relative to the slot center.
North Offset

Distance that northing is offset from the reference datum
East Offset

Distance that easting is offset from the reference datum
Vertical Section - End Position Reference

Reference used to determine the azimuth for the ending position of the vertical section

  • None – Azimuth is not calculated, and must be manually entered.
  • Total Depth – Azimuth is calculated from the total depth of the current plan or the definitive survey.
  • Target – Azimuth is calculated from the target depth of the current plan or the definitive survey. Only available when there is a target set associated with the well.
Well Path Type

Type of well path used to calculate azimuth. Not available when the None option is selected for the end position reference.
Vertical Section Azimuth

Azimuth along which to project the wellpath. This value may be manually entered or calculated, depending on the option selected for the end position reference.

  • If the None option is selected, this value must be manually entered.
  • If the Total Depth or Target option is selected, this value will be calculated when you click the Calculate button.
Dogleg Parameters Increment Amount to increase DLS1 and DLS2, if a curve-hold-curve or curve-hold calculation fails
DLS1 Default dogleg severity of the first curved section of a curve-hold-curve or curve-hold wellpath
Maximum Maximum allowed value for DLS1 and DLS2
DLS2 Default dogleg severity of the second curved section of a curve-hold-curve or curve-hold wellpath
Recompute well path using default dogleg severity Recalculates the wellpath, using the updated dogleg severity parameters
Magnetic Data -- Magnetic parameters for the well. If you do not know the magnetic parameters, click the Calculate button to open the geo-magnetic calculator. See "Well Magnetic Parameters".
-- Open hole diameter Diameter of the open hole section of the wellbore
-- Deepest Constraint Measured depth at which error accumulates in the wellpath. This parameter is not currently used.
-- File Number Unique ID from Wellz, used to uniquely identify wellpaths
