About exporting wellpaths

You can export the current plan or definitive survey to a standardized file format for use in external systems or reports. Export formats include:

  • MMS – A reporting format for surveys based on a modified version of the UKOOA P7/2000 format. This option is only available for the definitive survey. See "Exporting to MMS format".
  • LAS – A general-purpose format based on LAS 2.0, commonly used for well log and deviation data. See "Exporting to LAS Format".
  • REP – A format designed for direct import into AutoCAD. See "Exporting to REP and CSV format".
  • CSV – A simple comma-separated format for exporting raw data to spreadsheets. See "Exporting to REP and CSV format".

Exporting to MMS format

To export the definitive survey to MMS format:

  1. In the Data List pane, right-click the name of the well you want to export.
  2. Go to Export to > File, then select the Definitive Survey option.
  3. Select the MMS option for the export format, then click Next.
  4. In the next window, click the Select File button, then navigate to the location where you want to save the exported file and enter a file name.
  5. Enter the suffix for the well name, the bottomhole lease number, and any remarks you want to add.
  6. Click Export to generate the MMS file.

The definitive survey will be exported to the selected file location. The well name suffix, bottomhole lease number, and your remarks will be added to the exported file.

Exporting to LAS Format

To export a plan or survey to LAS format:

  1. In the Data List pane, right-click the name of the well you want to export.
  2. Go to Export to > File, then select the well path you want to export.
  3. Select the LASoption for the export format, then click Next.
  4. In the next window, click the Select File button, then navigate to the location where you want to save the exported file and enter a file name.
  5. In the Location field, enter the coordinates for the location of the well.
  6. If the well is in Canada:
    • Select the Site Is In Canada option.
    • Enter the Province where the well is located.
  7. If the well is not in Canada:
    • Leave the Site Is In Canada option unchecked.
    • Enter the County, State, and Country where the well is located.
  8. Select the measurement unit for the site (meters or feet). This defaults to the site unit in 5D but can be changed for the export.
  9. Click Export to generate the LAS file.

The plan or survey will be exported to the selected file location. The location information that you entered will be included in the well information section of the LAS file.

Exporting to REP and CSV format

To export a plan or survey to REP or CSV format:

  1. In the Data List pane, right-click the name of the well you want to export.
  2. Go to Export to > File, then select the well path you want to export.
  3. Select the desired export format, then click Next.
  4. In the next window, click the Select File button, then navigate to the location where you want to save the exported file and enter a file name.
  5. If you want to interpolate the points between the salient points, select the Interpolate option and enter the increment at which you want the points to be interpolated.
  6. Click OK to complete the export.

The plan or survey will be exported to the selected file location.
