About proximity reports

Proximity reports show how close a tie-in well is to other selected wells in the field. The report analyzes center-to-center distances between the tie-in well and nearby wells, helping you assess potential collision risks during drilling operations.

Before you can generate proximity reports, you must enable the center-to-center distance option in the reporting settings. See "Enabling proximity reporting".

Enabling proximity reporting

To enable proximity reporting:

  1. Click the Options and Settings button () in the main menu.
  2. Go to the Reporting > Anti-Collision Report page.
  3. In the Risk thresholds section, select the Centre to centre distance option.
  4. Click OK to save your changes.

Proximity reports will be enabled for tie-on wells. See "Generating proximity reports".

Generating proximity reports

To generate a proximity report for a tie-in well:

  1. In the Data List, select the tie-in well.
  2. Right-click the tie-in well and select Report > Proximity Report.
  3. In the well selection dialog, choose the wells you want to include in the proximity analysis.
  4. Click OK to generate the report.

The report will be generated in report format selected in the main toolbar. See "Selecting the report format". After the report finishes generating, it will open automatically.

Report sections

The table below defines the parameters included in the Primary Well section of the report. The Primary Well section shows the analysis results for the tie-on well and the nearest secondary well.

Parameters in the Primary Well section
Parameter Description
Pri MD Measured depth of the tie-on well
Inc. Inclination angle of the wellbore.
Az. Azimuth angle relative to grid North.
TVD True vertical depth below the field vertical reference datum.
TVD (MSL) True vertical depth, relative to mean sea level
CC Center-to-center distance between the primary well and the nearest secondary well
T. Face to Sec Angle between the hi-side vector of the primary well and the center-to-center line between the primary well and the nearest secondary well
Nearest Well Name of the nearest secondary well
Risk Collision risk assessment based on the center-to-center distance between well paths. Risk levels (low, medium, high) are triggered when the distance drops below specified thresholds set in Options and Settings > Anti-Collision Analysis. See "Anti-collision report settings".

Az. Azimuth angle relative to grid North.
TVD True vertical depth below the field vertical reference datum.
TVD (MSL) True vertical depth, relative to mean sea lvel
CC Center-to-center distance between the primary well and the secondary well
