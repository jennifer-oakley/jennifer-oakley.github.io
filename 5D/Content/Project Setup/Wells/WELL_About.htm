Wells

A main well is a wellbore that is drilled from the surface, and does not branch off of another well. Wells can contain plans, surveys, and sidetrack wells. The Data List pane shows the plans and surveys for each well. In the Data List pane, a main well is represented by the Main Well icon (). Sidetrack wells are represented by the Sidetrack Well icon ().

After you create a well, you can add plans and surveys, define the well formations, update the drilling corridor, and more. To update a well or add items to it, right-click on the well name in the Data List pane or go to Field Items > Well in the main menu.

  • New Reactive Plan – Creates a new current plan, starting from the current definitive survey, that joins to a point in the current plan. See "Reactive well planning".
  • Definitive Survey Summary – Opens a summary of all the surveys that make up the definitive summary. See "Definitive Survey".
  • Add Survey – Adds a new survey to the well. See "New surveys".
  • Delete Surveys Recursively – Deletes all surveys from the well. This action cannot be undone.
  • Save – Saves all changes to the current plan.
  • Save As New Current Plan – Saves all changes to a new current plan. The previous current plan will be copied to the plan archive as a minor revision.
  • Edit/View Well – Opens the Edit Well Data window, which allows you to edit the properties of the well. See "Well properties".
  • Change Well Hierarchy – Allows you to change the parent well or slot, and to change a sidetrack well to a main well. See "Change well hierarchy".
  • Add Sidetrack Well – Adds a sidetrack well to the main well. See "Sidetrack wells".
  • Specific Plan – Allows you to create a well plan using a specialized planning method. See "Specific plans".
  • Well Annotations – Adds comments to the well at a specified depth. See "Well Annotations".
  • Well Formations – Allows you to define the formation layers that the wellbore intersects. See "Well Formations".
  • Create/Edit Link – Opens the Project Documentation Link window, which allows you to add or edit a documentation link for the well. See "Document links".
  • Interpolate – Interpolates the properties of a point between salient points in the current plan. See "Plan interpolation".
  • Lease Lines – Allows you to add lease lines to the well, associate existing lease lines with the well, and run a proximity analysis. See "Lease lines".
  • Plot – Opens the Plot window, which allows you to plot the current plan, definitive survey, or traveling cylinder. See "Plotting".
  • Export To – Export options for plans and surveys. The File option exports the current plan or definitive survey to an MMS, LAS, REP, or CSV file. See "Exporting wellpaths to a file". The WITSML option exports plans and surveys to a WITSML XML file. See "WITSML exports".
  • Report – Generates a report for the current plan, the definitive survey, or uncertainty analysis. See "Well reports".
  • Set AC Primary – Designates the current well as the primary well for anti-collision analysis. See "Anti-Collision Analysis".
  • Field Operations – Opens the Field Operations window, which allows you to monitor the wellpath as it is being drilled. See "Field Operations".
  • Export to Wellz Format – Exports the current plan or definitive survey to a Wellz file. See "Wellz Exports".
  • Delete Well – Deletes the well from the slot. Only available when the well does not contain any surveys.
  • Delete Well Recursively – Permanently deletes the well and all data associated with it. This action cannot be undone.
  • Drilling Corridor – Opens the Drilling Corridor window, which allows you to edit the drilling corridor.
