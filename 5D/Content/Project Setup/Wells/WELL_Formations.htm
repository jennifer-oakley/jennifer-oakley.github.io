About well formations

Well formations are the geological boundaries that the wellpath intersects while the well is being drilled. The depth of each formation layer can be entered using wellpath MD, wellpath TVD, or slot TVD. The remaining depth values will automatically calculated, based on either the definitive survey or the current plan. The depth value that you entered will be shaded in gray to indicate that it is the default depth value for the formation layer.

To update the formation data for a well, right-click on the well name in the Data List pane and select the Well Formations option, or go to Field Items > Well > Well Formations in the main menu.

After you add the formation layers for a well, they will appear in the 3D Viewer and reports generated for the well. You can customize the reporting options in the Tools > Options and Settings window. See "Using the 3D Viewer" and "Report settings".

Entering formation points

To manually enter formation data:

  1. In the Data List pane, right-click on the well and select the Well Formations option.
  2. In the Formation Points window that opens, select the wellpath reference you want to use to calculate depth values.
  3. In the data table, enter the name of the formation layer in the Name column.
  4. Enter a depth value in the Ref. to Wellpath - MD, Ref. to Wellpath - TVD, or Ref. to Slot - TVD column. The remaining depth values will be automatically calculated, and the value you entered will be shaded in gray.
  5. Enter the dip angle and direction for the formation layer.
  6. When you are finished entering formation points, click OK to close the window.
  7. Click the Save button () to save the database.

The depth values that you entered will be shaded in gray, and the remaining depth values will be automatically calculated. To view the formations in the 3D Viewer, click the Wellpath Display Options button () and select the Show Formations option. See "Using the 3D Viewer".

To customize the formation data that is included in well reports, go to Tools > Options and Settings in the main menu and select the report you want to customize in the Reporting section. See "Report settings".

Copying formation data from Excel

You can also add formation data by copying the data from Excel and pasting it into the data table in the Formation Points window. The data that you copy from Excel must contain the following columns, in the following order:

  • Name
  • Depth
  • Dip
  • Direction
  • Comment

Depth, dip, and direction values must be numeric, and should not include the unit. The values in the Comment column may be blank, but they must be included.

To copy formation data from Excel:

  1. In Excel, copy the formation data to the clipboard.
  2. In the Data List pane, right-click on the well and select the Well Formations option.
  3. In the Formation Points window that opens, select the wellpath reference you want to use to calculate depth values.
  4. In the data table, right-click on any cell in the first blank row and select the Paste option.
  5. In the Paste Formations window, go to the Type field and select the type of depth values that your data contains.
  6. In the Unit field, select the unit type used for depth values.
  7. Click OK to close the Paste Formations window, then click OK again to close the Formation Points window.
  8. Click the Save button () to save the database.

The depth values included in your data will be shaded in gray. The remaining depth values will be automatically calculated. To view the formations in the 3D Viewer, click the Wellpath Display Options button () and select the Show Formations option. See "Using the 3D Viewer".

To customize the formation data that is included in well reports, go to Tools > Options and Settings in the main menu and select the report you want to customize in the Reporting section. See "Report settings".

Formation properties

The table below defines the fields and options shown in the Formation Points window.

Property definitions for the Formation Points window
Section Field Name Definition
-- Well Path Reference Wellpath reference used to calculate depth values
-- TVD Reference Well TVD reference, defined on the Wellpath page in the Options and Settings window. See "Wellpath Settings".
-- North Reference North reference for the site, defined in the site properties. See "Site properties".
-- Name Name of the formation layer
Ref. to Wellpath MD MD of the top of the formation layer, relative to the wellpath. This value will be automatically calculated when one of the other depth values is entered.
TVD TVD of the top of the formation layer, relative to the wellpath. This value will be automatically calculated when one of the other depth values is entered.
Ref. to Slot TVD TVD of the top of the formation layer, relative to the slot. This value will be automatically calculated when one of the other depth values is entered.
Formation Dip Magnitude of the inclination of the formation layer from horizontal
Direction Bearing of the dip angle from the north reference
-- Comment Any comment relevant to the formation layer
