About editing tie-in points

A tie-in point defines the connection parameters where a well connects to a reference wellbore whose complete trajectory is unknown. The tie-in point is specified by measured depth, location (TVD, North offset, East offset), and attitude (inclination, azimuth) relative to the reference well.

The tie-in point editing process varies depending on whether the well has a blank current plan or an existing plan with multiple points. Wells with existing plans offer additional options for preserving curve specifications and configuring how the tie-in point connects to the rest of the wellpath.

Before you can edit a tie-in point, the well must meet these requirements:

  • Plans only: The well must consist entirely of wellpath plans with no survey data
  • Active current plan: The well must have a current plan with at least one point
  • No sidetracks: The main well can't have any sidetrack wells attached to it

Edit a tie-in point

To edit the tie-in point parameters for an existing well, you'll modify the first point of the current plan to establish the connection data.

  1. In the Data List pane, select the well that contains the tie-in point you want to edit.
  2. Click the List Plans tab in the Data List pane to display the current plan in the Data Table.
  3. Right-click on the first row and select the Edit Tie In option.
  4. In the Tie-In Point window that opens, specify the position and attitude of the tie-in point. See "Field definitions".
  5. If the current plan has multiple points, configure the connection method and curve preservation options as needed.
  6. Click OK to save the tie-in point parameters.

After successfully editing the tie-in point, the current plan data table displays in yellow to distinguish it from normal current plans (blue), archive plans (green), and surveys (red). The tie-in point parameters are now defined and the well can be used for tie-in wellpath planning.

Field definitions

The Tie-In Point window displays different options depending on whether the current plan is blank or contains existing points.

Blank current plans

The table below defines the fields in the Tie-In Point window, when the current plan only contains one point.

Field definitions for the Tie-In Point window, when the current plan only contains one point
Field Description
MD

Measured depth of the tie-on point from the drill floor
TVD

True vertical depth of the tie-on point, measured from the TVD reference point
Inclination

Inclination of well at the tie-on point, with zero degrees being vertical down
Azimuth

Azimuth of well at the tie-on point, measured clockwise from the site North reference
North Offset

North offset of the tie-on point from the site or slot center in the local North direction
East Offset

East offset of the tie-on point from the site or slot center in the local East direction

Existing plans

When the current plan contains multiple points, additional options control how the tie-in point connects to the rest of the wellpath, the Edit Tie-In Point window will show the fields listed below.

Field definitions for the Edit Tie-In Point window, when the current plan contains multiple points
Option Description
Tie-In Point

Position and attitude of the tie-in point

  • MD – Measured depth of the tie-on point from the drill floor
  • Inclination – Inclination of well at the tie-on point, with zero degrees being vertical down
  • Azimuth – Azimuth of well at the tie-on point, measured clockwise from the site North reference
  • TVD – True vertical depth of the tie-on point, measured from the TVD reference point
  • North Offset – North offset of the tie-on point from the site or slot center in the local North direction
  • East Offset – East offset of the tie-on point from the site or slot center in the local East direction
Rejoin Options

Method for handling wellpath geometry beyond the tie-on point

  • Preserve Position – Maintains exact positions and attitudes of all anchor points beyond the connection
  • Preserve Original Curve Specifications – Recreates all sections beyond the connection using their original design parameters
  • Preserve Shape – Maintains the same wellpath shape beyond the tie point as the original plan
Connect By

Method for joining the tie-on point to the current plan

  • Anchor Point – Connects to a specific anchor or ordinary point (ordinary points are automatically promoted)
  • MD – Connects to an interpolated point at the specified measured depth
  • TVD – Connects to an interpolated point at the specified true vertical depth
  • None – Creates a plan containing only the definitive survey
Rejoin to Anchor Point

Method for connecting the tie-on point to the current plan.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the tie-on point and the current plan.
  • Double Curve – Creates a double curve segment between the tie-on point and the current plan.
  • Hold Double Curve – Creates a hold-double-curve segment between the tie-on point and the current plan.
DLS1

Preferred dogleg severity of the shallow curved section.
DLS2

Preferred dogleg severity of the deep curved section.
DLS Iteration Method

Method for preserving dogleg severity values during iteration.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Apply after every increment

Automatically applies changes after each parameter modification
