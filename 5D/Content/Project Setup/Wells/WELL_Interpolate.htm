About interpolation

Wellpath interpolation lets you calculate the properties of any point along the current plan, even if that point isn't already defined in your wellpath data. This is useful when you need to know the exact inclination, azimuth, coordinates, or other parameters at a specific depth.

The interpolation tool works like a calculator – you enter a depth (measured depth or true vertical depth), and 5D computes all the wellpath parameters at that location. You can then copy these values to the clipboard for use in other applications or reports.

This feature is particularly handy when you need to:

  • Check wellpath parameters at casing setting depths
  • Verify coordinates for obstacle avoidance calculations
  • Get precise attitude information for directional drilling operations
  • Extract data points for external analysis or reporting

The interpolation doesn't modify the current plan in any way. It simply calculates and displays the properties at your specified depth based on the existing wellpath geometry.

Interpolating points

To interpolate a point along the current plan:

  1. In the Data List pane, go to the List Wells tab.
  2. Right click the well name you want to interpolate points for and select the Interpolate option.
  3. In the Interpolate window that opens, enter the depth you want to interpolate.
  4. Select the type of depth entered.

The parameters for the interpolated point will be calculated, based on its position in the current plan. To copy the parameters to the clipboard, click on the Copy button.

Field definitions

The table below defines the parameters in the Interpolate window.

Field definitions for the Interpolate window
Field Name

Description
Depth

Depth of the point to be interpolated and the type of depth measurement.

  • MD – Measured depth from wellpath start
  • TVD – True vertical depth from wellpath start
Interpolation Increment Increment with which depth increases or decreases
MD

Measured depth of the interpolated point from the drill floor
TVD

True vertical depth of the interpolated point, measured from the TVD reference point
Inc

Inclination of well at the interpolated point, with zero degrees being vertical down
Az

Azimuth of well at the interpolated point, measured clockwise from the site North reference
N. Offset

North offset of the interpolated point from the site or slot center in the local North direction
E. Offset

East offset of the interpolated point from the site or slot center in the local East direction
DLS

Dogleg severity of the well path at the interpolated point
T. Face

Angle of the toolface from the high side of the wellbore
VS

Projected distance of the interpolated point from the vertical section starting position, along the vertical section azimuth
