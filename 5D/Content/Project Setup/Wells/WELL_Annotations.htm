About well annotations

You can associate annotations with a well path at a specific MD or TVD. These annotations can appear in reports, plots, and the 3D Viewer.

To open the Well Annotation Points window, right-click a well in the Data List pane and select the Well Annotations option.

Note

To display annotations in the 3D Viewer, you must enable the Show Annotations option in the 3D Viewer settings. See "Using the 3D Viewer".

To include annotations in reports, enable Annotations in the Reporting section of the Options and Settings window. See "Report settings".

Adding an annotation

To add a new annotation to a well:

  1. Right-click the well in the Data List pane and select the Well Annotations option.
  2. In the Well Annotation Points window, select the Depth Type from the dropdown list. Annotation points can be indexed by either MD or TVD.
  3. In the Depth field, enter the depth that you want the want the annotation to be associated with.
  4. In the Comment field, enter a description or other comment for the annotation.
  5. Press Enter to commit the entry. A new blank row will be added automatically.
  6. When you are finished adding annotations, click the OK button.

The annotation will appear in the 3D Viewer window next to the well path, aligned with the specified depth, if the Show Annotations option is enabled. The annotation will also be included in plan reports, survey reports, and UTM reports, if the Annotation points option is enabled in the report settings.

Right-click commands

When you right-click on the annotations table in the Well Annotation Points window, the options listed below will appear.

Right-click commands in the Well Annotation Points window
Menu Item Description
Insert Creates a new blank row before the selected row.
Delete Removes the selected row.
Copy and Paste

To copy between annotation tables:

  • Open a well annotations table and select the desired rows.
  • Right-click and select Copy.
  • Open another well and select Paste.

To copy to an external text editor:

  • Select rows and choose Copy.
  • Paste into a text editor such as Notepad. Each entry appears as tab-delimited text (e.g., TVD 100 Sea Bed).
Paste From Clipboard

Pastes copied data into the table. The data must use the following format: depth type, depth, and comment.
