About data organization

The Data List pane is used to navigate between wells, plans, surveys, targets, and lease lines. The data in 5D is organized into a hierarchy, with the field at the top of the hierarchy.

  • The Field level contains the sites, targets, and lease lines for the field.
  • The Site level contains the slots in the site.
  • The Slot level contains the wells in the slot.
  • The Well level contains the plans and surveys for the well.

Organization levels

The table below defines the top level of organization used to organize well data in 5D.

Organization levels for well data
Icon Name Description
Field The field is the region where the drilling operation takes place. All data is positioned within the field using map coordinates and a TVD reference. Fields contain the sites, targets, and lease lines for the field. See "Fields".
Site The site is the structure from which wells are drilled. All measurements within the site are relative to the site center, in a direction relative to grid north or true north. Sites contain the slots in the site. See "Sites".
Slot The slot is the location in the site from which the main well is drilled. Slot positions can be entered using map coordinates or local coordinates. Slots contain the wells in the slot. See "Slots".
Well The well stores all of the plan and survey data. Each well contains a current plan, a definitive survey, a plan archive, and a survey archive. See "Wells".
Target Set A target sets is a group of targets for the field. Target sets can contain multiple targets, such as reservoir entry points and total depth. See "Targets & Target Sets".
Target Point A target defines specific points that a well needs to reach when it is drilled. See "Targets & Target Sets".
Lease Lines A lease line defines the geographical boundaries of the field. See "Lease lines".

Plan Items

The table below defines the organization levels associated with well plans. See "Plans".

Organization levels for well plans
Icon Name Description
Plan Archive The plan archive stores the plan folders and well plans.
Plan Folder Plan folders store the well plans that have been created for the well. Each plan folder represents major revisions in the plan structure. All plans in a plan folder start from the same survey station. The plan folder that contains the current plan is known as the current folder.
Current Plan The current plan is the most up-to-date plan for the well.
Archived Plan An archived plan is a previous revision of the planned wellpath. When you create a new current plan, the existing current plan will become an archived plan.
Definitive Plan The definitive plan is a plan that has been flagged as important, such as the plan which was agreed to between the client and the contractor.

Survey Items

The table below defines the organization levels associated with surveys. See "Surveys".

Organization levels for well surveys
Icon Name Description
Definitive Survey The definite survey is a composite survey created from the surveys in the survey archive.
Survey Archive The survey archive stores all of the well's surveys.
Survey A survey is a list of survey readings taken while drilling the well.
