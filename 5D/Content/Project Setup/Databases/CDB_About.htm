About central database

The central database is an online repository that stores 5D fields and their associated sites, slots, wells, lease lines, and target sets. It is a single database that contains the 5D data for most of the regions in the world. Most users log-in to the same database, but the list of fields shown in the Central Database Console window is filtered, based on the region assigned to their user account. In a few special cases, such as Saudi Arabia, have their own region-specific central database.

When you connect to the central database, you can download a field or any of the items associated with the field. To connect to the central database, click the Connect to database button () in the main toolbar. See "Connecting to the central database".

After you download a field from the central database, 5D will enter Live mode. In Live mode, an item must be locked before it can be edited. When you lock an item, all of the items in the hierarchy below that item will also be locked. For example, if you lock a site, all slots and wells within that site will also be locked. See "Live mode".

The actions that you can take while you are connected to the central database are determined by your level of access to the central database. There are four levels of access for the central database:

  • Basic access allows you to download items for your assigned region. However, you will not be able to edit any of the items. When someone else makes changes, you can download the updates into 5D.
  • Advanced access allows you to download and edit items for your assigned region. When you make changes, the changes will be saved to the central database.
  • Administrator access allows you to download and edit items for your assigned region. You can also manage user accounts for your region and add transformations for your region to the database. See "User information" and "Transformations".
  • Super User access allows you to download and edit items for any region. You can manage user accounts for any region, create new regions, and transformations to the database. See "User information" and "Transformations".

You can only view the fields, transformations, search results, and user accounts for the region assigned to your user account. The region assigned to your user account is shown in the title bar for the Central Database Console window and the main 5D window.

Connecting to the central database

To connect to the central database:

  1. Click the Connect to database button () in the main toolbar.
  2. In the Central Database Login window, enter your user name and password.
  3. In the Region field, select the region that you want to connect to. If the region isn't listed, click the Advanced button to enter the server information.
  4. Click the OK button to log in.

After 5D connects to the database, you can open the Central Database Console window by clicking the Show Console button () in the main toolbar. If you already have a 5D database open, you will need to close the database or save it to the central database before you can open the Central Database Console window. See "Adding fields to the database".

Adding fields to the database

Before you can add a field to the central database, a transformation is required to convert the field's projected coordinate system to the central database's root coordinate system. To verify if the transformation already exists, open the console window and go to the Transformations tab. See "Transformations".

To add a field to the central database:

  1. Connect to the central database. See "Connecting to the central database".
  2. Click the Open Database button () in the main toolbar, then select the field you want to add to the central database.
  3. Click the Save to Database button () in the main toolbar.
  4. In the window that appears, select the folder in the central database that you want to save the field to.
  5. Click OK.

If the transformation to the root coordinate system was successful, you will get a notification that the field was successfully to the central database. The new field will automatically be associated with the region assigned to your account.

Using the central database

When you are connected to a field from the central database, several new buttons will appear in the main toolbar.

Central database icons in the main toolbar
Button Function
Saves the changes to the current database. This button only appears when you are connected to a central database.
Saves the current field to a local file on your computer. This button only appears when you are connected to a central database.
Opens the Central Database Console window. See "Central Database Console".
Indicates that you are connected to a field from the central database. See "Live mode".
Indicates that there are updates available for the field that you are currently connected to. This button only appears when you are connected to a field from the central database.
Locks the selected item in the data list, as well as all child items. This button only appears when you are connected to a field from the central database.
Downloads any updates from the central database. This button only appears when you are connected to a field from the central database.
