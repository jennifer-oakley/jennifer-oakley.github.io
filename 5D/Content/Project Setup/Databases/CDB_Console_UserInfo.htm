About user information

In the Central Database Console window, the User Info tab lists the users with access to the database. There are four levels of access for the central database:

  • Basic access allows you to download items for your assigned region. However, you will not be able to edit any of the items. When someone else makes changes, you can download the updates into 5D.
  • Advanced access allows you to download and edit items for your assigned region. When you make changes, the changes will be saved to the central database.
  • Administrator access allows you to download and edit items for your assigned region. You can also manage user accounts for your region and add transformations for your region to the database. See "Transformations".
  • Super User access allows you to download and edit items for any region. You can manage user accounts for any region, create new regions, and transformations to the database. See "Transformations".

The left side of the window contains the list of user accounts. You can only view the user accounts for the region assigned to your user account. A green circle icon () next to the user name indicates that the user is currently connected to the central database. When you select a user on the User Info tab, you will be able to view details about the user account, such as the user's email address and the role assigned to the account.

If you are an administrator or super user, you will also be able to create new accounts, reset passwords, update account information, disconnect users from the database, and delete user accounts. However, only super users can grant Super User access.

Creating accounts

To create a new user account:

  1. In the Central Database Console window, go to the User Info tab on the left side of the window.
  2. Click the Add user button at the bottom of the window.
  3. In the right side of the window, enter a user name and password for the new account.
  4. In the Role field, select the type of access that the account should have.
  5. In the Email field, enter the user's primary email address.
  6. If you are a super user, go to the Region field and select the region that the user's account should be assigned to.
  7. To allow access to the 5D test folder, select the View test DB option.
  8. Click the Save button at the bottom of the window.

The new account will appear in the list of users on the left side of the window. The new user will be able to log in to the central database, using the assigned user name and password. The new user will only be able to view the fields, transformations, search results, and user accounts for the region assigned to their user account. If you have Administrator access, the user will automatically be assigned to your region.

Resetting passwords

To reset the password for a user account:

  1. In the Central Database Console window, go to the User Info tab on the left side of the window.
  2. In the list of users, click on the account that you want to reset the password for.
  3. In the right side of the window, go to the Password field and enter a new password for the account.
  4. Click the Save button at the bottom of the window.

The password to the user account will be updated to the new password. If the user is currently logged in, they will not be disconnected from the database. However, they will need to use the new password the next time they log in.

Disconnecting users

To disconnect a user from the central database:

  1. In the Central Database Console window, go to the User Info tab on the left side of the window.
  2. In the list of users, right-click on the user account that you want to disconnect from the database.
  3. Select the Disconnect User option.
  4. In the confirmation message that appears, click Yes to confirm disconnecting the account.

The user account will be disconnected from the database. If they have any items locked, the locks will be released.

Deleting user accounts

Note

If the user is currently connected to the central database, you will need to disconnect the account before you can delete it. See "Disconnecting users".

To delete a user account from the central database:

  1. In the Central Database Console window, go to the User Info tab on the left side of the window.
  2. In the list of users, right-click on the user account that you want to delete.
  3. Select the Delete User option.
  4. In the confirmation message that appears, click Yes to confirm deleting the account.

The deleted account will be removed from the list of users. The user will no longer be able to log in to the central database.
