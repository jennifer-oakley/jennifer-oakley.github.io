About 5D databases

In Weatherford 5D, all of the data related to a field is stored in a database. When you create a new database, a new field will be automatically created. The data within the database is organized hierarchically, with the field as the highest level of the hierarchy. This includes sites, slots, wells, plans, surveys, and lease lines. Databases can be stored locally on your computer, or they can be downloaded from the central database. See "Central Database".

Databases are managed from the File menu in the main menu. You can create new databases, open an existing database, save your changes, close the current database, and import and export items from another 5D database.

  • The Create Field option allows you to create a new database. You can also use the Create Field and Database button () in the main toolbar. See "New databases".
  • The Open option allows you to open an existing database. You can also use the Open Database button () in the main toolbar. See "Opening a database".
  • The Import option allows you to import objects from another 5D database into the current database. See "5D Import".
  • The Export option allows you to export objects from the current database into another 5D database. See "5D Export".

The current 5D database will be automatically saved to a backup temporary file every ten minutes. To recover a database to the auto-saved version, click the Open Database button (), then click the Browse Auto-Save button in the window that opens.

Note

If you have a 5D database open, you must close it before you can create or open another one. To close the current database, go to the File menu and select the Close Database option.

Opening a database

To open an existing database:

  1. Click the Open Database button () in the main toolbar.
  2. To change the root directory, click the Select button. Navigate to the desired location, then click Select Folder.
  3. Select the database you want to open in the list that appears, then click OK.

The selected database will be loaded in Weatherford 5D.

Tip

To open a 5D database without changing the root directory, click the Search For File button and navigate to the database location.
