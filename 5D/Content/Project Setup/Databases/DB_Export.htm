About 5D exports

The Export 5D Items window allows you to export wells from your current 5D database to a new database file. This is useful when you need to share specific wells with co-workers or create backup copies of selected wells.

You can export wells at different levels of the hierarchy:

  • Export all wells in a site by selecting the site
  • Export all wells in a slot by selecting the slot
  • Export individual wells from any combination of sites and slots

When you export wells, all associated data for those wells is included in the new database file, such as surveys, plans, and well configurations. However, you can only export at the well level – individual plans or surveys cannot be exported separately.

Note

When you select a site or slot, all wells contained within it are automatically selected for export. If you select a sidetrack well, its parent well is also automatically selected to maintain the well relationship in the exported database.

Exporting wells

To export wells from the 5D database:

  1. Go to File > Export in the main menu.
  2. In the Export 5D Items window that opens, select the wells you want to export.
  3. Click the Export button.
  4. Navigate to the location where you want to save the exported file and enter a file name.
  5. Click Save to start the export.

The wells included in the export will saved to a new 5D database, at the selected file location.

Using the Export 5D Items window

The buttons and options in the Export 5D Items window provide options for navigating and filtering the well hierarchy.

Button definitions for the Export 5D Items window
Button Function
Show Sites Collapses the hierarchy view to show only site-level items
Expand All Expands the full hierarchy to show all sites, slots, and wells
Search Filters the displayed items based on the text entered in the search field
Enable AC Threshold Warning in Exported Field Triggers a warning in the Field Operations window if the separation factor is less than 1.5. The Show AC warning for SF Threshold option must also be enabled in the Field Operations Settings window. See "Field Operation Settings".
Export Creates a new database file containing the selected wells
