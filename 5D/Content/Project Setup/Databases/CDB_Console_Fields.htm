About the fields list

In the Central Database Console window, the Fields tab lists all of the folders and items for your region. The items in the database are organized hierarchically, with folders at the top of the hierarchy. Each folder can contain multiple fields. On the Fields tab, you can download items from the database and manage the folders within the database. See "Downloading items" and "Managing folders and fields".

When you download an item from the database, all of the items listed below it in the data list hierarchy will be downloaded. Any items listed above it in the hierarchy will also be downloaded. For example, if you download a slot, the associated field, site, and wells for that slot will also be downloaded. However, other slots or sites will not be downloaded. If you download a sidetrack well, the parent well will also be downloaded.

After you download an item, you will not be able to edit it until you have locked the item for editing. See "Locking items". When an item is locked, it cannot be edited by another user. Locking an item also locks all items listed below it in the hierarchy. The icons on the right side of the data list indicates the lock status of an item.

  • A green lock icon () indicates that the item is locked by you.
  • A red lock icon () indicates that the item is locked by another user.

Downloading items

To download items from the central database:

  1. In the Central Database Console window, go to the Fields tab on the left side of the window.
  2. Navigate to the folder containing the items you want to download.
  3. Select the highest-level item that you want to download.
  4. Click the Download button in the bottom right corner of the window.

The Central Database Console window will automatically close, and you will enter Live mode. See "Live mode". The selected items will be downloaded, as well as any items above them in the hierarchy. For example, if you download a slot, the associated field, site, and wells for that slot will also be downloaded. However, other slots or sites will not be downloaded. If you downloaded a sidetrack well, the parent well will also be downloaded. To lock the item for editing, click the Lock Item button () in the main toolbar. See "Locking items".

Note

If you already have a field from the central database open, you will only be able to download other items from that field. To download items from another field, you will need to close the field before opening the Central Database Console window.

Managing folders and fields

When you right-click on a folder or a field in the console window, there are several options available.

  • To create a new folder, right-click on any folder and select the Create folder option. In the window that appears, enter a name for the new folder.
  • To change the name of a folder, right-click on the folder and select the Rename folder option.
  • To delete a folder, right-click on the folder and select the Delete folder option. You can only delete empty folders that do not contain any fields.
  • To archive the fields in a folder, right-click on the folder and select the Archive folder option. Archived fields can still be downloaded and updated, but they do not appear in search results by default.
  • To move a field to another folder, right-click on the field and select the Move to folder option.
  • To set the main well, right-click on the well and select the Set main option. When you search for offset wells, the main well will be used as the center of the search radius. See "Offset well search".
  • To delete a field from the central database, right-click on the field and select the Delete field option. You will be prompted to enter a password to delete a field. If you do not have the password for deleting fields, contact your system administrator to have the field deleted.
