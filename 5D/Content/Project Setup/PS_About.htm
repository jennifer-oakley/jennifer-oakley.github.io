Project Setup

The Data List pane is used to navigate between wells, plans, surveys, targets, and lease lines. The data in 5D is organized into a hierarchy, with the field at the top of the hierarchy.

Databases

Databases are managed from the File menu in the main menu. You can create new databases, open an existing database, save your changes, close the current database, and import and export items from another 5D database.

Fields

A field is a geographical area which contains multiple oil or gas wells that produce from a continuous reservoir. In Weatherford 5D, the field is the highest level of the hierarchy in a 5D database. The Data List pane shows the sites, slots, wells, and lease lines that are in the field. In the Data List pane, the field is represented by the Field icon ().

Sites

A site is the physical structure from which wells are drilled. The measurements for the site are relative to a local reference for the site center, as well as a reference for north that is parallel to grid north or true north. Sites are the second level of the hierarchy in a 5D database. The Data List pane shows the slots and wells that are in each site. In the Data List pane, a site is represented by the Site icon ().

Slots

A slot is a location on a site, from which a main well will be drilled. Slots can contain multiple main wells and sidetrack wells. Slots are the third level of the hierarchy in a 5D database. The Data List pane shows the wells that are in each slot. In the Data List pane, a slot is represented by the Slot icon ().

Wells

A main well is a wellbore that is drilled from the surface, and does not branch off of another well. Wells can contain plans, surveys, and sidetrack wells. The Data List pane shows the plans and surveys for each well. In the Data List pane, a main well is represented by the Main Well icon (). Sidetrack wells are represented by the Sidetrack Well icon ().

Targets

Targets are used to define the points that a well needs to reach when it is drilled. Targets can be created before or after you create well. If the target set already exists, you can associate it with the well when it is created. If not, you can create the target set later.

Lease Lines

A lease line is defined by a list of 2D coordinates that form either an open or closed boundary. Open lease lines have different starting and ending coordinates, while closed lease lines have the same first and last coordinate to form an enclosed area.
