About main wells

A main well is a wellbore that is drilled from the surface, and does not branch off of another well. Wells can contain plans, surveys, and sidetrack wells. The Data List pane shows the plans and surveys for each well. In the Data List pane, a main well is represented by the Main Well icon (). Sidetrack wells are represented by the Sidetrack Well icon ().

There are two primary sets of parameters used to define the properties of a main well:

  • The Elevation section defines the elevation of the well.
  • The Advanced section defines the vertical section, dogleg limits, and magnetic data.

Adding main wells

To add a well to a slot:

  1. In the Data List pane, right-click on the slot you want to add a well to and select the Add Main Well option.
  2. In the Add Well window that opens, enter the name of the well in the Name field.
  3. In the Well TVD Reference field, select the reference used to determine the elevation of the well.
  4. In the Relative To field, select the reference that the well elevation is relative to.
  5. In the Elevation field, enter the elevation of the drill floor.
  6. If you have already created a target set for the well, select the name of the target set. See "Targets & Target Sets".
  7. In the Side-Track section, configure the connection to the parent well:
    • Select which parent wellpath to connect the sidetrack to.
    • Enter the kick-off depth and select whether you're specifying MD or TVD
  8. To define the vertical section, dogleg parameters, and magnetic data, click the Advanced button. See "Properties of main wells".
  9. Click OK to close the advanced properties, then click OK again to create the well.
  10. Click the Save button () to save the database.

The new well will be listed in the Data List pane, under the List Wells tab. A current plan and definitive survey will be automatically created for the new well. To add plans, surveys, or sidetrack wells, go to the Data List pane and right-click on the well name. See "Wells".

After you create the well, you will not be able to edit the well elevation in the well properties. To change the elevation of a plan or survey, you'll need to edit the drill floor. See "Drill floor elevation".

Properties of main wells

The table below defines the properties in the Add Well window. This window appears when you create a new well or edit the well properties. To edit the properties of a well, go to Field Items > Well in the main menu and select the Edit/View Well option.

Property definitions for the Add Well window
Section Property Name Definition
Context Site Name of the site that the well is associated with
Slot Name of the slot that the well is associated with
Well Name Name of the well
Identifier Unique, industry-approved ID number of the well
Status

Status of the well

  • Closed – The well has reached total depth, and the definitive surveys are finalized. You will only be able to view the well and create reports.
  • Active – The well is in the execution phase. There are no restrictions for editing or creating child items.
  • Planned – The well is in the planning phase. There are no restrictions for editing or creating child items.
Well TVD Reference Elevation point used to define the elevation of the well
Relative To

Vertical reference datum that the well elevation is relative to.

  • Mean Sea Level – Elevation is relative to mean sea level.
  • Site TVD Reference – Elevation is relative to the TVD reference defined in the site properties.
  • Slot TVD Reference – Elevation is relative to the TVD reference defined in the slot properties.
Elevation Elevation of the well, relative to the selected vertical reference datum
Rig Height Elevation of the drill floor or the kelly bushing, relative to the elevation of the slot. This value is calculated automatically, based on the value entered in the Elevation field.
Inc Inclination at the drill floor or the kelly bushing
Azi Azimuth at the drill floor or the kelly bushing
-- Target Set Target set assigned to the well. See "Targets & Target Sets".
-- Comment Any relevant comments about the well

The table below defines the properties in the Add Well - Advanced window. This window appears when you click the Advanced button while creating a well or editing the well properties.

Property definitions for the Add Well window
Section Property Name Definition
Vertical Section - Start Position Relative To

Reference datum that the starting position of the vertical section is relative to

  • Site – Start position is relative to the site center.
  • Slot – Start position is relative to the slot center.
North Offset

Distance that northing is offset from the reference datum
East Offset

Distance that easting is offset from the reference datum
Vertical Section - End Position Reference

Reference used to determine the azimuth for the ending position of the vertical section

  • None – Azimuth is not calculated, and must be manually entered.
  • Total Depth – Azimuth is calculated from the total depth of the current plan or the definitive survey.
  • Target – Azimuth is calculated from the target depth of the current plan or the definitive survey. Only available when there is a target set associated with the well.
Well Path Type

Type of well path used to calculate azimuth. Not available when the None option is selected for the end position reference.
Vertical Section Azimuth

Azimuth along which to project the wellpath. This value may be manually entered or calculated, depending on the option selected for the end position reference.

  • If the None option is selected, this value must be manually entered.
  • If the Total Depth or Target option is selected, this value will be calculated when you click the Calculate button.
Dogleg Parameters Increment Amount to increase DLS1 and DLS2, if a curve-hold-curve or curve-hold calculation fails
DLS1 Default dogleg severity of the first curved section of a curve-hold-curve or curve-hold wellpath
Maximum Maximum allowed value for DLS1 and DLS2
DLS2 Default dogleg severity of the second curved section of a curve-hold-curve or curve-hold wellpath
Recompute well path using default dogleg severity Recalculates the wellpath, using the updated dogleg severity parameters
Magnetic Data -- Magnetic parameters for the well. If you do not know the magnetic parameters, click the Calculate button to open the geo-magnetic calculator. See "Well Magnetic Parameters".
-- Open hole diameter Diameter of the open hole section of the wellbore
-- Deepest Constraint Measured depth at which error accumulates in the wellpath. This parameter is not currently used.
-- File Number Unique ID from Wellz, used to uniquely identify wellpaths
