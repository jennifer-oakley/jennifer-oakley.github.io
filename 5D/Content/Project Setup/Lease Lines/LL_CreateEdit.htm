About creating lease lines

On the List Lease Lines tab in the Data List pane, you can view and edit the existing lease lines for the field. Each set of lease lines defines the boundary coordinates and offset lease lines for the section. After you create a lease line, you can associate it with wells and include it in proximity analysis to ensure wellpaths maintain appropriate distances from regulatory boundaries.

You can create lease lines at different organizational levels in your field hierarchy:

  • Field level – Lease lines created at the field level can only be entered using absolute coordinates. See "Creating lease lines relative to a field, site, or slot".
  • Site level – Lease lines created at the site level can be entered relative to the field or the site. See "Creating lease lines relative to a field, site, or slot".
  • Slot level – Lease lines created at the slot level can be entered relative to the field, site, or slot. See "Creating lease lines relative to a field, site, or slot".
  • Well level – Lease lines created at the well level can be entered relative to the field, site, slot, or target. Lease lines created at the well level are automatically associated with the well. See "Creating lease lines relative to a target".

Creating lease lines relative to a field, site, or slot

To create a lease line relative to a field, site, or slot:

  1. Right-click on a field, site, or slot in the Data List pane, then select the Create Lease Line option.
  2. In the Create Lease Line window that opens, enter a name for the lease line.
  3. To include an offset for the lease line, select the Lease Line Offset option and enter the offset distance.
  4. In the Relative To field, select the reference for the lease line.
  5. In the Specify By field, select the coordinate reference system you want to use to enter the coordinates.
  6. Enter the boundary coordinates in the coordinates table.
  7. Click Apply to save your changes, then click Close to close the window.

The new lease line will be listed on the List Lease Lines tab, and it will be available for association with the wells in your field. If lease lines are enabled in the 3D Viewer, the new lease line will appear in the 3D visualization. To display lease lines in the 3D Viewer, click the Lease Lines button () in the 3D Viewer, then select the Show Lease Lines option.

Creating lease lines relative to a target

Warning

If there is already a lease line associated with the well, the existing lease line will be selected by default.

To create a lease line relative to a target:

  1. Right-click on the well in the Data List pane, then go to the Lease Lines menu and select the Create Lease Line option.
  2. In the Create Lease Line window that opens, enter a name for the lease line. If the Name field isn't blank, click on the dropdown menu and select the blank line at the top of the list.
  3. To include an offset for the lease line, select the Lease Line Offset option and enter the offset distance.
  4. In the Relative To field, select the Target option, then select the target you want to use as a reference.
  5. In the Specify By field, select the coordinate reference system you want to use to enter the coordinates.
  6. Enter the boundary coordinates in the coordinates table.
  7. Click Apply to save your changes, then click Close to close the window.

The new lease line will be listed on the List Lease Lines tab, and it will be automatically associated with the well. If lease lines are enabled in the 3D Viewer, the new lease line will appear in the 3D visualization. To display lease lines in the 3D Viewer, click the Lease Lines button () in the 3D Viewer, then select the Show Lease Lines option.

Note

The Target option will only be enabled if there is a target set associated with the well. See "Associating a well with a target set".

Managing lease lines and coordinates

After you create a lease line, you can view and edit them on the List Lease Lines tab. To edit a lease line, right-click on the lease line you want to edit, then select the Edit option.

  • To add coordinates, type directly in the blank row at the bottom of the coordinate table.
  • To change the order of the coordinates, use the red arrow buttons to move the coordinates up and down. Lease lines are created by connecting coordinates in table order.
  • To copy coordinates to the clipboard, use the mouse to highlight the rows you want to copy, then right-click and select the Copy option.
  • To paste clipboard data into the coordinate table, right-click in the coordinate table and select the Paste option.
  • To delete coordinates, use the mouse to highlight the rows you want to delete, then right-click and select the Delete option.

Field definitions

The table below defines the fields and options in the Create Lease Line window.

Fields and options in the Create Lease Line window
Field/Option Definition
Name Unique identifier for the lease line. Enter a new name to create a lease line, or select from the drop-down to edit existing ones.
Lease Line Offset When selected, creates a parallel offset line at the specified distance and orientation
Distance Separation distance between the parent lease line and its offset. Only available when the Lease Line Offset option is selected.
Left/Right Orientation of the offset line relative to walking direction along the parent. Only available when the Lease Line Offset option is selected.
Relative To Reference system for coordinate entry (Field, Site, Slot, or Target)
Specify By Coordinate format (Northing/Easting, Lat/Long, Cartesian offsets, or Polar)
Update Shared Coordinates When selected, coordinate edits update the same coordinate across all field lease lines
