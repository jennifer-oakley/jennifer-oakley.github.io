About proximity analysis

Proximity analysis computes the horizontal distance between a wellpath (current plan or definitive survey) and the lease lines associated with it. This analysis helps ensure that wellpaths maintain safe distances from lease boundaries and provides critical data for regulatory compliance.

You can run proximity analysis in two ways:

  • All lease lines – Analyzes against all associated lease lines and shows only the nearest lease line at each analysis point.
  • Specific lease line – Analyzes against a single selected lease line and shows the distance to that specific boundary at each analysis point.

To control the level of detail in the analysis, you can adjust the analysis increment along the wellpath and set the maximum distance for reporting lease line proximities.

Note

Tie-in wells can also use lease line proximity analysis, but the inter-boundary distance and separation factor parameters are omitted from their reports since anti-collision parameters are not computed for tie-in wells.

Running proximity analysis

To run proximity analysis on a well:

  1. In the Data List pane, right-click on the well you want to analyze.
  2. Go to Lease Lines > Proximity Analysis.
  3. In the Proximity Analysis window that opens, configure the analysis options as needed. See "Analysis options".
  4. Click Run Analysis to generate the proximity report.

The proximity report displays in a separate window with detailed distance measurements at each analysis point along the wellpath.

Analysis options

The Proximity Analysis window provides several options to customize your analysis:

Proximity analysis options
Option Description
Lease Line

Dropdown list containing all lease lines associated with the well:

  • All Lease Lines – Analyzes against all associated lease lines, showing the nearest line at each point.
  • Specific lease line name – Analyzes against only the selected lease line.
Well Path Selection Radio buttons to select the wellpath data source for analysis:
  • Current Plan – Uses the active wellpath plan.
  • Definitive Survey – Uses actual survey data.
Comment When a specific lease line is selected, displays the descriptive comment associated with that lease line.
Show Lease Lines Within Sets the maximum center-to-center distance for lease lines to be included in the report. Only lease lines within this distance will appear in the results.
Analysis Increment Sets the measured depth interval between proximity analysis points along the wellpath. Smaller increments provide more detailed analysis but generate larger reports.

Report format and parameters

The proximity analysis report displays the same columns regardless of whether you analyze against all lease lines or a specific lease line. The report includes the following parameters at each analysis point:

Proximity analysis report parameters
Parameter Description
MD Measured depth along the wellpath at the analysis point.
Inclination Wellpath inclination angle at the analysis point.
Azimuth Wellpath azimuth direction at the analysis point.
TVD True vertical depth at the analysis point.
Northing Northing coordinate of the analysis point.
Easting Easting coordinate of the analysis point.
Name Name of the lease line being analyzed (or nearest lease line when analyzing all lines).
Cr-Cr Distance Center-to-center distance between the analysis point and the nearest point on the lease line.
I/B Distance Inside/boundary distance – the separation between the uncertainty ellipsoid of the well and the lease line, measured along the center-to-center line.
Sep. Factor Separation factor – the ratio of center-to-center distance to the uncertainty ellipsoid radius at the analysis point.
Bias Directional bias measurement indicating the relative position of the wellpath with respect to the lease line.
TFace to Leaseline Tool face angle from the wellpath to the lease line, indicating the rotational direction to the boundary.
