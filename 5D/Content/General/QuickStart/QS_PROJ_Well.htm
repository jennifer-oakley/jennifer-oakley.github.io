Adding main wells

A main well is a wellbore that is drilled from the surface, and does not branch off of another well. Wells can contain plans, surveys, and sidetrack wells. The Data List pane shows the plans and surveys for each well. In the Data List pane, a main well is represented by the Main Well icon (). Sidetrack wells are represented by the Sidetrack Well icon ().

There are two primary sets of parameters used to define the properties of a main well:

  • The Elevation section defines the elevation of the well.
  • The Advanced section defines the vertical section, dogleg limits, and magnetic data.

To add a sidetrack well to a well:

  1. In the Data List pane, right-click on the slot you want to add a well to and select the Add Main Well option.
  2. In the Data List pane, right-click on the well you want to add a sidetrack to and select the Add Side Track Well option.
  3. In the Add Well window that opens, enter the name of the sidetrack well in the Name field.
  4. In the Well TVD Reference field, select the reference used to determine the elevation of the well.
  5. In the Relative To field, select the reference that the well elevation is relative to.
  6. In the Elevation field, enter the elevation of the drill floor.
  7. If you have already created a target set for the well, select the name of the target set. See "Targets & Target Sets".
  8. In the Side-Track section, configure the connection to the parent well:
    • Select which parent wellpath to connect the sidetrack to.
    • Enter the kick-off depth and select whether you're specifying MD or TVD
  9. To define the vertical section, dogleg parameters, and magnetic data, click the Advanced button. See "Properties of main wells". See "Field definitions".
  10. Click OK to close the advanced properties, then click OK again to create the sidetrack well.
  11. Click the Save button () to save the database.

The new well will be listed in the Data List pane, under the List Wells tab. The new sidetrack will be listed in the Data List pane under the parent well. A current plan and definitive survey will be automatically created for the new well. To add plans, surveys, or sidetrack wells, go to the Data List pane and right-click on the well name. See "Wells".

After you create the well, you will not be able to edit the well elevation in the well properties. To change the elevation of a plan or survey, you'll need to edit the drill floor. See "Drill floor elevation".
