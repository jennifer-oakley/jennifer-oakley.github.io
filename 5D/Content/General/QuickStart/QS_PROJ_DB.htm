Creating a database

Related Topics
Related Topics
  • Fields
  • New databases

In Weatherford 5D, all of the data related to a field is stored in a database. When you create a new database, a new field will be automatically created. The data within the database is organized hierarchically, with the field as the highest level of the hierarchy. This includes sites, slots, wells, plans, surveys, and lease lines. Databases can be stored locally on your computer, or they can be downloaded from the central database.

To create a local database:

  1. Click the Create Field and Database button () in the main toolbar.
  2. In the Field Data window that opens, enter the name of the field in the Name field.
  3. In the Projected Coordinate System field, select the name of the map grid where the field is located. If the grid is not listed, select the Select new grid option. See "Projected coordinate systems".
  4. In the Field Vertical Reference Datum field, select vertical reference datum for the field.
  5. Select the map unit and preferred site unit.
  6. In the Dogleg Base Length field, enter the length used to record dogleg severity in the preferred site unit.
  7. In the Default Magnetic Parameters section, enter the magnetic parameters for the field. If you do not know the magnetic parameters, click the Calculate button to open the geo-magnetic calculator. See "Field Magnetic Parameters".
  8. Click the OK button to close the window.
  9. To save the database, click the Save button ().

The new database will be saved to the selected directory. The new field will be listed in the Data List pane, under the List Wells tab. To add sites or edit the field, go to the Data List pane and right-click on the field name. See "Fields".
