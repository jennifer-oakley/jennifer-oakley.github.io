About target sets and targets

Targets are used to define the points that a well needs to reach when it is drilled. Targets can be created before or after you create well. If the target set already exists, you can associate it with the well when it is created. If not, you can create the target set later.

There are three basic steps to adding targets to a well:

  • Create a target set. A target set is a group of targets for a well. See "Adding target sets".
  • Add targets to the target set. Targets can include points such as the reservoir entry point and the target depth. See "Adding targets".
  • Associate the target set with a well. The target set associated with a well will be the default target set when you join the current plan to a target or create a 2 of 7 plan. See "Associating a well with a target set".

Adding target sets

To add a target set:

  1. Go to the List Targets tab in the Data List pane to open the Edit Targets pane.
  2. On the Target Set tab, click the New button and enter a name of the target set.
  3. In the Reference field, select the reference that target coordinates will be relative to.
  4. In the Style field, select the coordinate format.
  5. In the TVD Reference field, select the vertical reference datum for the TVD of the targets.
  6. Click the Apply button to save your changes.
  7. Click the Save button () to save the database.

The new target set will be listed on the List Targets tab in the Data List pane. After you create the target set, you will need to add targets to it. See "Adding targets". You'll also need to associate the target set with a well. See "Associating a well with a target set".

Adding targets

To add a target:

  1. Go to the Edit Targets > Target Set tab in the Data Entry pane.
  2. Select the target set that you want to add targets to, then go to the Target tab.
  3. Click the New button and enter a name for the target.
  4. In the TVD field, enter the TVD of the target point.
  5. In the Relative To section, enter the coordinates of the target point.
  6. If the target aim point is different from the target point, go to the Target Aim Point section and enter the TVD offset and the coordinate offsets.
  7. On the right side of the screen, select the target shape for the uncertainty area. See "Target shapes".
  8. Click the Apply button to save your changes.
  9. Click the Save button () to save the database.

The new target will be listed on the List Targets tab in the Data List pane, under the selected target. After you create the target, you can join the target to the current plan. See "Projecting to a target".

Associating a well with a target set

To associate a well with a target set:

  1. Go to the List Wells tab in the Data List and select the well.
  2. Right-click on the well name and select the Edit/View Well option.
  3. In the Edit Well Data window, go to the Target Set field and select the name of the target set.
  4. Click OK to save your changes.

The target set will appear in the well properties. When you join the current plan to a target or create a 2 of 7 plan, the target set will be the default target set. See "Projecting to a target" and "Two of Seven planning".
