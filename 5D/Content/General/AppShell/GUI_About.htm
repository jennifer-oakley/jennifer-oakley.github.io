About the Weatherford 5D interface

The Weatherford 5D interface has four major areas:

  • The Main Menu contains menus and toolbar buttons that are used to access most of the major functions in Weatherford 5D. See "Main toolbar".
  • The Data List pane is used to navigate between wells, plans, surveys, targets, and lease lines. See "Data List".
  • The 3D Viewer pane provides you with a three-dimensional view of the data in the field. See "3D Viewer".
  • The Data Entry pane is used to enter data for plans, surveys, target sets, targets, and anti-collision analysis. See "Data Table".

5D also includes report utilities, which are launched outside of 5D. These utilities generate reports using specific, pre-defined templates. See "Report utilities".

Main toolbar

The main toolbar is used to open and create databases, launch the 3D viewer, open the Data Table pane, and to open the Anti-Collision window.
