About the data table

The data table shows the salient points for plans, surveys, and casing points. When you go to the List Plans tab or the List Surveys tab in the Data List pane, the data table will automatically open. The background color of the table is color-coded according to the type of data shown. Current plans are shown in blue, archived plans are shown in green, surveys are shown in red, and tie-wells are shown in yellow.

Field definitions

The table below defines the columns that are shown in the Data Table pane, when you go to the List Plans tab or the List Surveys tab in the Data List pane.

Column definitions for the Data Table pane
Column Name

Description
MD

Measured depth of the salient point from the drill floor
Inc

Inclination of well at the salient point, with zero degrees being vertical down
Azi

Azimuth of well at the salient point, measured clockwise from the site North reference
TVD

True vertical depth of the salient point, measured from the TVD reference point
N. Offset

North offset of the salient point from the site or slot center in the local North direction
E. Offset

East offset of the salient point from the site or slot center in the local East direction
Point Type

Type of plan point, such as anchor point, survey point, or ordinary point
DLS

Dogleg severity of the well path between this point and the previous salient point
T. Face

Angle of the toolface from the high side of the wellbore
B. Rate

Change in inclination of the well path between this point and the previous salient point
T. Rate

Effective change in azimuth of the well path between this point and the previous salient point
CL

Change in measured depth between this point and the previous point
VS

Projected distance of the salient point from the vertical section starting position, along the vertical section azimuth
Tortuosity

Tortuosity of the wellpath at the salient point
DDI

Rating for the Directional Difficulty Index (DDI)
Comment

Any comments about the salient point
