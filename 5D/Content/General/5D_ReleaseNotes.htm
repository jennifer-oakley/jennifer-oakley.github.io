Weatherford 5D v9.8.0 Release Notes

In this article

What's new

Bugs fixed

Known issues

Welcome to Weatherford 5D v9.8.0. This version of Weatherford 5D contains new features and enhancements, as well as many bug fixes. For a full list of the updates in v9.8.0, refer to the sections below.

What's new

The features in this section are new for Weatherford 5D v9.8.0.

  • Updated the algorithm for calculating the probability of collision.
  • Updated the HDGM model to HDGM 2026.
  • Added the GYRO_PTW and VES GyroFlex v4.1 tools to the list of available survey tools in the Survey Tool Picker window.
  • In the Plot window, added an option to select the master graph object. When the Select Master option is selected, moving or resizing the grid or axes for the graph will move and resize the entire graph. When the Select Master option is not selected, the grid and axes can be resized independently. See "Plotting: Using the Select Master option".
  • Improved the way graphs are drawn and resized in the Plot window, so that the actual graph size more closely matches the area that you draw.
  • Improved the functionality of snapshots in the Plot window. Snapshots are now automatically created when you perform any of the following actions: add or delete a graph or plot object, resize or move a graph, load a chart, reload all wells or information boxes, update the section view or top view properties, or edit the wellpath accessories. When you click the Undo button, the chart will revert back to the most recent snapshot. See "Plot object browser".
  • Updated the keyboard and mouse controls required to move a graph or plot object in the Plot window. To move an object on the chart, you will now need to hold the Shift key while using the mouse to move the object. See "Keyboard and mouse controls".
  • Added the ability to view the targets for offset wells in the Field Operations window. When offset targets are enabled, they will shown in the Section View and Top View graphs. To show targets for offset wells, click the Targets button () in the Graph pane and select the Show Offset Targets option.
  • When the Restrict TVD Range option is enabled in the 3D Viewer window, you can now adjust the increment at which the minimum and maximum TVD changes. To enable the Restrict TVD Range option, click the Wellpath Display Options button () in the toolbar for the 3D Viewer window. WISE 1084038 See "3D Viewer: Using the Restrict TVD Range option".

Bugs fixed

The critical issues in this section have been fixed in Weatherford 5D v9.8.0.

  • Fixed an issue where uncertainty reports for surveys using the Blind Rotary BHA_PTTEP tool displayed extremely large semi-major uncertainty values.
  • Fixed an issue with polyhedron targets where drilling targets displayed incorrect TVD positioning in side view when the target had negative TVD values. WISE 1109748
  • Corrected the magnitude for the DSTG error source on GYRO+MWD_KOC tools.
  • Improved the error messages that appear when 5D is unable to import Compass files due to line feed characters from Unix operating systems. WISE 1107713

Known issues

The issues in this section are known issues for Weatherford 5D v9.8.0.

  • When you attempt to build a section using an azimuth of 270 degrees or more, you may get an error that the section could not be built. As a workaround, use inclination to build the section and set the toolface to the desired azimuth value.
  • The precision of the coordinates in a report may not be displayed correctly.
  • If you run a proximity analysis for a sidetrack well with only one point in the current plan, no results will be shown for anti-collision analysis.
  • When you import wells from Compass 2003, the kick-off depth for sidetrack wells may be incorrect.
  • The warning messages for anti-collision analysis may be inconsistent when there are multiple survey tools used in the same survey.
  • When you project back on a sidetrack well, child wells may become detached from the sidetrack well.
  • If you attempt to import a large Compass 2000 or Compass 2003 file, the import may fail.
