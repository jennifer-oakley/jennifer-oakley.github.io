About rotating well segments

Rotating a well segment changes the azimuth direction of a portion of the wellpath, spinning the selected segment around a vertical axis and changing the compass direction that the wellpath is heading without altering the inclination. This is useful when you need to redirect a section of the wellpath while keeping the same vertical geometry, such as when you need to hit a different target or avoid an obstacle.

This rotation will generally change the attitude of the first point in the well segment, as well as the position and attitude of the other points in the well segment. The specified azimuth change in the Delta Azimuth field will be added to every point in the selected segment. The rotated segment is reconnected to the rest of the wellpath using composite curves.

Rotating well segments

To rotate a well segment:

  1. Press the Ctrl key and select the consecutive points that you want to move in the data table.
  2. Right-click on the highlighted rows and select the Rotate option.
  3. In the Rotate window, go to the Delta Azimuth field and enter the amount of azimuth change that you want to apply to the wellpath segment. Positive values will rotate the wellpath clockwise, and negative values will rotate it counterclockwise.
  4. In the Join From First Rotated Point section, select the type of composite curve you want to use to connect the first point in the well segment to the wellpath above it.
  5. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the first composite curve.
  6. In the S-Curve from Last Rotated Point section, select the type of composite curve you want to use to connect the last point in the well segment to the wellpath below it.
  7. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the second composite curve.
  8. Click the OK button to save your changes and move the well segment.

The well segment will be rotated by the amount specified in the Delta Azimuth field. The positions of the points in the well segment will be updated in the current plan, and the positions of the points below the well segment will be updated as needed to connect the segment to the wellpath.

Field definitions

The table below defines the fields in the Rotate window.

Fields in the Rotate window
Field Name Description
Delta Azimuth Azimuth change to apply to the selected wellpath segment. This value is added to the azimuth of each point in the segment. Positive values rotate clockwise, negative values rotate counterclockwise.
Join from First Rotated Point

Method for connecting the first point in the well segment to the previous anchor point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the first point in the well segment and the previous anchor point.
  • Double Curve – Creates a double curve segment between the first point in the well segment and the previous anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the first point in the well segment and the previous anchor point.
DLS1

Preferred dogleg severity of the shallow curved section for the curve to the first rotated point.
DLS2

Preferred dogleg severity of the deep curved section for the curve to the first rotated point.
DLS Iteration Method

Method for preserving dogleg severity values during iteration for the curve to the first rotated point.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
S-Curve from Last Rotated Point

Method for connecting the last point in the well segment to the next anchor point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the last point in the well segment and the next anchor point.
  • Double Curve – Creates a double curve segment between the last point in the well segment and the next anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the last point in the well segment and the next anchor point.
DLS1

Preferred dogleg severity of the shallow curved section for the curve from the last rotated point.
DLS2

Preferred dogleg severity of the deep curved section for the curve from the last rotated point.
DLS Iteration Method

Method for preserving dogleg severity values during iteration for the curve from the last rotated point.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Apply after every increment

Automatically applies changes after each parameter modification
Preserve Final Attitude

When selected, the wellpath will connect to the end point of the wellpath with the selected curve using the DLS1 and DLS2 values as the preferred dogleg severities. Otherwise, the end point is connected with a curveHold curve using the DLS1 value as preferred dogleg severity. Only available when the second-to-last anchor point is selected.
