About automatic kick-off point planning

Automatic kick-off point (auto-KOP) planning creates a three-section wellpath from the surface to a target using a hold-curve-hold geometry. When you create an auto-KOP plan, the location of the kick-off point is automatically calculated, based on the target coordinates, final inclination, and the dogleg severity for the curved section.

This planning method is particularly useful for simple wellpaths where you know your target location and desired approach angle, but want the system to optimize the kick-off point for the smoothest trajectory. The resulting wellpath consists of:

  • A vertical hold section from surface to the kick-off point.
  • A curved section that builds inclination at the specified dogleg severity.
  • A final hold section that maintains the target inclination to reach the specified coordinates.

Before you can create an auto-KOP plan, the Auto-KOP option must be enabled in the Options and Settings window. See "Enabling auto-KOP planning".

Enabling auto-KOP planning

To enable automatic kick-off point planning:

  1. Click the Options and Settings button () in the main menu.
  2. Go to the Well Parameter Lists page.
  3. In the Specific Well Plans section, select the Auto-KOP option.
  4. Click OK to save your changes.

The Auto-KOP option will appear in the list of available options when you right-click on a well or go to Field Items > Well in the main menu. See "Creating an auto-KOP plan".

Creating an auto-KOP plan

To create a plan with an automatic kick-off point:

  1. Right-click the well in the Data List pane, go to Specific Plan, then select the Auto-KOP option.
  2. In the Auto-KOP window, enter the Cartesian or polar coordinates for the target.
  3. In the Point Inclination field, enter the final inclination for the target.
  4. In the Curve DLS field, enter the desired dogleg severity for the curved section of the composite curve.
  5. Click OK to save your changes.

The kick-off point will be automatically calculated, with a straight segment connecting the kick-off point to the start of the wellpath and a curveHold segment connecting the kick-off point to the target.

Field definitions

The table below defines the fields in the Auto-KOP window.

Fields in the Auto-KOP window
Field Name Description
Cartesian

Cartesian coordinates for the target.

  • TVD – True vertical depth of the target.
  • North Offset – Northing coordinate relative to wellpath start
  • East Offset – Easting coordinate relative to wellpath start
Polar

Polar coordinates for the target.

  • TVD – True vertical depth of the target.
  • Range – Horizontal distance from the wellpath start
  • Bearing – Compass direction from the wellpath start
Point Inclination Final inclination of the target
Curve DLS Dogleg severity for the curved section of the composite curve
Apply after every increment
