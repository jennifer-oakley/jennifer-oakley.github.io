About freezing parameters

You can freeze specific parameters to control how anchor points connect to other points in the wellpath. When you edit an anchor point, 5D uses the frozen parameters to determine how each anchor point reconnects when changes are made to the wellpath above it. The frozen parameters are preserved, while the unfrozen parameters are adjusted to build the updated wellpath.

By freezing specific parameters on anchor points, you can:

  • Control which properties stay fixed during edits (position, attitude, or combinations)
  • Limit the propagation of changes along the wellpath
  • Ensure critical design requirements are maintained (like hitting a target with a specific approach angle)

For example, if you freeze only the position of an anchor point, its attitude can adjust when the wellpath above it is modified. If you freeze both position and attitude, the point becomes a hard constraint that forces 5D to use composite curves to connect properly.

Tip

Frozen parameters are indicated in the data table with an asterisk (*) to help you identify which constraints are active.

Freezing parameters

To freeze parameters for an anchor point:

  1. In the Data List pane, go to the List Wells tab and select the well whose current plan you want to edit.
  2. Go to the List Plans pane. The data table will automatically open at the bottom of the screen.
  3. Right-click on the salient point whose parameters you want to freeze, then select the Freeze Parameters option.
  4. In the Select Parameters window that opens, go to the Curve tab and select the parameters that you want to freeze.
  5. Go to the Curve tab and select the type of composite that should connect to the anchor point.
  6. Click OK.

When the wellpath above the anchor point is edited, the selected parameters will not be affected.

Field definitions

The table below defines the options available on the Curve tab in the Select Parameters window.

Options on the Curve tab
Option Name Description
TVD, North, East Preserves TVD, north offset, and east offset of the anchor point
Delta MD, Inclination, Azimuth Preserves the inclination, azimuth, and the MD distance between the anchor point and the previous anchor point
MD, Inclination, Azimuith Preserves the MD, inclination, and azimuth of the anchor point
Delta TVD, Inclination, Azimuth Preserves the inclination, azimuth, and the TVD distance between the anchor point and the previous anchor point
DLS, Inclination, Azimuth Preserves the dogleg severity, inclination, and azimuth of the anchor point
DLS, Toolface, Delta MD Preserves the dogleg severity, toolface angle, and the MD distance between the anchor point and the previous anchor point
DLS, Toolface, MD Preserves the dogleg severity, toolface angle, and MD of the anchor point
DLS, Toolface, Delta TVD Preserves the dogleg severity, toolface angle, and the TVD distance between the anchor point and the previous anchor point
DLS, Toolface, TVD Preserves the dogleg severity, toolface angle, and TVD
DLS, Toolface, Inclination Preserves the dogleg severity, toolface angle, and inclination
DLS, Toolface, Azimuth Preserves the dogleg severity, toolface angle, and azimuth
