About projecting to a target

You can connect anchor points to target sets or create new targets from existing anchor points, which helps you build wellpaths that hit specific geological targets or reference points.

There are two options are available for working with targets:

  • Join to Target – Connects the wellpath from the selected anchor point to an existing target set. See "Joining to a target".
  • Create Target – Creates a new target for use in future planning. See "Creating a target".

When joining to targets, the wellpath beyond the selected point will be removed, and a new wellpath segment will be created. The new wellpath segment will connect to the first selected target, then thread through the remaining targets in sequence. For multiple targets, 5D automatically calculates optimal approach angles to create smooth, drillable connections between targets.

Joining to a target

Joining to a target connects the wellpath from any anchor point to a target set. Composite curves will be created to connect through multiple targets or reach targets with defined attitudes. Using an approach point allows more control over the connection by creating a curved approach to the first target. You can specify the curve's dogleg severity, toolface angle, and backtrack distance.

To join an anchor point to a target:

  1. Right-click on the point in the data table that you want to project from, then select the Join to Target option.
  2. In the Join To Target window, select the type of curve you want to use to connect the selected point to the first target.
  3. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the curve.
  4. To include an approach point between the anchor point and the first target, select the Insert Approach Point option, then enter the parameters for the curve connecting the approach point to the target.
  5. In the Target Set section, select the target sets and associated targets that you want to project to.
  6. For single targets, select the Define attitude of single target option to use inclination and azimuth to define the target attitude.
  7. Click OK to save your changes.

The selected targets will be added to the current plan, and a composite curve will be created to connected the wellpath to the selected targets.

Note

If you specify backtrack depth as a ΔTVD value, use negative values to place the approach point above the first target. For approach points to work properly, the attitude must be defined for the first target.

Creating a target

To add a target:

  1. Right-click on any point in the data table, then select the Create Target option.
  2. In the Edit Targets pane that opens, enter a name for the target.
  3. In the TVD field, enter the TVD of the target point.
  4. In the Relative To section, enter the coordinates of the target point.
  5. If the target aim point is different from the target point, go to the Target Aim Point section and enter the TVD offset and the coordinate offsets.
  6. On the right side of the screen, select the target shape for the uncertainty area. See "Target shapes".
  7. Click the Apply button to save your changes.
  8. Click the Save button () to save the database.

The new target will be listed on the List Targets tab in the Data List pane, under the selected target. After you create the target, you can join the target to the current plan. See "Projecting to a target".

Field definitions

The table below defines the fields in the Join To Target window.

Fields in the Join To Target window
Field Name Description
Curve to Target Set

Method for connecting the approach point to the target.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the approach point and the target.
  • Double Curve – Creates a double curve segment between the approach point and the target.
  • Hold Double Curve – Creates a hold-double-curve segment between the approach point and the target.
DLS1

Preferred dogleg severity of the shallow curved section.
DLS2

Preferred dogleg severity of the deep curved section.
DLS Iteration Method

Method for preserving dogleg severity values during iteration.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Approach Point

Parameters for the curve connecting the approach point and the target.

  • Insert approach point – When checked, creates a planned approach curve from the composite curve end to the first target
  • Curve DLS – Dogleg severity of the curve connecting the approach point and the target.
  • Toolface – Direction of the curve connecting the approach point and the target.
  • Backtrack – Distance of the target from the approach point, entered in either MD or TVD. If the projected point is entered in TVD, this value should be negative. If the projected point is entered in MD, this value should be positive.
Target Set List of available target sets. Use the Edit Targets to modify a target set.
Select targets to be joined List of targets in the selected target set. By default, all targets in the target set will be selected.
DLS between targets Dogleg severity for curves connecting multiple targets in the target set
Define attitude of single target When selected, allows you to specify the inclination and azimuth for a single target. This option is only available when there is only target selected.
Apply after every increment

Automatically applies changes after each parameter modification
