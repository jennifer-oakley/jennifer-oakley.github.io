About appending anchor points

When you append an anchor point to the current plan, a new section will be added to the end of the wellpath.

There are five types of well segments you can use to append an anchor point:

  • Straight segments extend the wellpath in a straight line, maintaining the same attitude. To append a straight segment to the wellpath, go to the tab.
  • Curve segments create a constant radius curve that curves to a position, to an attitude, or with curvature. To append a curve segment to the wellpath, go to the tab.
  • CurveHold segments create a curve section, followed by a straight section. To append a curveHold segment, go to the tab.
  • CurveHoldCurve segments create two curved sections that are separated by a straight tangent section. To append a curveHoldCurve segment, go to the tab.
  • DoubleCurve segments create two curved section, with no straight section between them. To append a doubleCurve section, go to the tab.

Appending anchor points

To append an anchor point:

  1. Right-click on the last point in the data table, then select the Append option.
  2. In the Append Anchor Point window, go to the tab for the type of well segment you want to add.
  3. Enter the required parameters for position, attitude, and depth. See "Field definitions".
  4. To join the new point to a target, click the Join To Target button. See "Projecting to a target".
  5. Click OK to confirm your changes and update the plan.

The new anchor point will be added to the end of the wellpath, using the specified curve settings.

Tip

When the Append without exiting option is selected, you can quickly add multiple sections without closing the Append Anchor Point window.

Field definitions

The sections below define the fields in the Append Anchor Point window.

Straight segments

The table below defines the fields on the tab, which defines the parameters for straight segments.

Fields on the tab for straight segments
Field Name Description
Depth

Depth of the anchor point and type of depth measurement.

  • ΔMD – Measured depth relative to the selected point
  • ΔTVD – True vertical depth relative to the selected point
  • MD – Absolute measured depth from wellpath start
  • TVD – Absolute true vertical depth from wellpath start
Apply after every increment

Automatically applies changes after each parameter modification
Append without exiting

Appends the new anchor point, without closing the Append Anchor Point window

Curve segments

The table below defines the fields on the tab, which defines the parameters for curve segments.

Fields on the tab for curve segments
Field Name Description
Curve Method

Method for creating the curved section

  • to Position – Curves to a specified position
  • to Attitude – Curves to a specified inclination and azimuth
  • with Curvature – Curves using specified dogleg severity and toolface
Point Position

Position of the new anchor point, entered using either Cartesian or polar coordinates. Only available when the to Position option is selected.

  • TVD – True vertical depth of the point
  • North Offset – Northing coordinate relative to wellpath start
  • East Offset – Easting coordinate relative to wellpath start
  • Range – Horizontal distance from the selected point
  • Bearing – Compass direction from the selected point
Point Attitude

Attitude of the new anchor point. Only available when the to Attitude option is selected.

  • Inclination – Angle from vertical (0° = vertical, 90° = horizontal)
  • Azimuth – Compass direction of the wellpath
Arrive at Depth

Depth at which the curve reaches the specified attitude. Only available when the to Attitude option is selected.

  • ΔMD – Measured depth relative to the selected point
  • ΔTVD – True vertical depth relative to the selected point
  • MD – Absolute measured depth from wellpath start
  • TVD – Absolute true vertical depth from wellpath start
Curve with DLS Dogleg severity for the curved section. Only available when the to Attitude option is selected.
Curvature

Curvature parameters for the curved section. Only available when the with Curvature option is selected.

  • DLS – Dogleg severity (rate of change of wellpath direction)
  • Toolface – Direction of the curve relative to the high side
Depth

Depth where the curved section ends. Only available when the with Curvature option is selected.

  • ΔMD – Measured depth relative to the selected point
  • ΔTVD – True vertical depth relative to the selected point
  • MD – Absolute measured depth from wellpath start
  • TVD – Absolute true vertical depth from wellpath start
Inclination Target inclination for the curved section. Only available when the with Curvature option is selected.
Azimuth Target azimuth for the curved section. Only available when the with Curvature option is selected.
Apply after every increment

Automatically applies changes after each parameter modification
Append without exiting

Appends the new anchor point, without closing the Append Anchor Point window
