About plans

A plan defines the intended trajectory for a well, from the surface location to the target depth. Plans are organized using a hierarchical archive system that automatically manages versioning as your designs evolve. The current plan represents your active design work and is the only plan you can edit.

When you save changes, 5D creates archived plans that preserve your design history in organized plan folders. The plan archive stores all saved plan versions as read-only records, while the definitive plan marks whichever version has special significance (such as client approval or regulatory sign-off). This lets you focus on design work while maintaining a complete record of how your well plan developed over time.

In the Data List pane, plans are represented by different icons: the Current Plan icon () for your active plan, the Archived Plan icon () for saved versions, and the Definitive Plan icon () for plans with special status.

Current plan

To update the current plan, right-click on the current plan in the Data List pane.

  • Save – Saves all changes to the current plan.
  • Save As New Current Plan – Saves all changes to a new current plan. The previous current plan will be copied to the plan archive as a minor revision.
  • Save As Non Current Plan – Saves all changes to a new plan, but does not promote the new plan to the current plan. The current plan will not be affected.
  • Edit Plan Parameters – Opens the Plan Version Information window, which shows the version history of the current plan and allow you to change the plan name.
  • Edit/View Well – Opens the Edit Well Data window, which allows you to edit the properties of the well. See "Well properties".
  • Add Survey – Adds a new survey to the well. See "New surveys".
  • Anti-Collision Primary – Designates the current plan as the primary wellpath for anti-collision analysis. See "Anti-Collision Analysis".
  • Report – Generates a report for the current plan, UTM, or uncertainty analysis. See "Plan reports".
  • Plot – Opens the Plot window, which allows you to plot the current plan or traveling cylinder. See "Plotting".
  • Interpolate – Interpolates the properties of a point between salient points in the current plan. See "Plan interpolation".
  • Export To File – Exports the current plan to an LAS, REP, or CSV file. See "Exporting wellpaths to a file".
  • New Reactive Plan – Creates a new current plan, starting from the current definitive survey, that joins to a point in the current plan. See "Reactive well planning".
  • Proximity Analysis – Opens the Lease Line Proximity window, which calculates the horizontal distance between the definitive survey and the lease lines associated with the well. See "Lease Line Proximity Analysis".
  • Edit Drill Floor – Opens the Edit Drill Floor window, which allows you change the elevation of the drill floor for the current plan. See "Drill floor elevation".
  • Edit Survey Programme – Opens the Edit Survey Programme window, which allows you to set the survey tools used at different depth ranges along the wellpath. These settings can be used to override the survey program for the current plan in an anti-collision analysis. See "Anticipated survey program".
  • File – Imports or exports the current plan to a text file.
  • Drilling Corridor – Opens the Drilling Corridor window, which allows you to edit the drilling corridor.

Plan folder

To update a folder in the plan archive, right-click on the plan folder in the Data List pane or go to Plans / Surveys > Plan Folder in the main menu.

  • Edit Plan Folder – Opens the Plan Folder Information window, which allows you to edit the folder name.
  • Delete Plan Folder – Deletes the plan folder and all plans within the folder. The plan folder that contains the current plan cannot be deleted.

Archived plans

After a plan is archived, you can promote the plan, create reports, export the wellpath, and more. To update an archived plan, right-click on the plan name in the Data List pane or go to Plans / Surveys > Plan in the main menu.

  • Edit Plan Parameters – Opens the Plan Version Information window, which shows the version history of the selected plan and allow you to change the plan name.
  • Set as Current Plan – Promotes the selected plan to the current plan.
  • Set as Definitive Plan – Designates the selected plan as the definitive plan.
  • Delete Plan – Deletes the selected plan from the plan archive.
  • Report – Generates a report for the selected plan or UTM.
  • Plot – Opens the Plot window, which allows you to plot the selected plan or traveling cylinder. See "Plotting".
  • Interpolate – Interpolates the properties of a point between salient points in the selected plan. See "Plan interpolation".
  • Export To File – Exports the selected plan to an LAS, REP, or CSV file. See "Exporting wellpaths to a file".
  • Edit Drill Floor – Opens the Edit Drill Floor window, which allows you change the elevation of the drill floor for the selected plan. See "Drill floor elevation".
  • Edit Survey Programme – Opens the Edit Survey Programme window, which allows you to set the survey tools used at different depth ranges along the wellpath. These settings can be used to override the survey program for the selected plan in an anti-collision analysis. See "Anticipated survey program".
  • Export Wellpath – Exports the selected plan to a text file.
