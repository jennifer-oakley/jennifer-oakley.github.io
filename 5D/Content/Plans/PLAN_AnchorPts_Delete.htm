About deleting anchor points

When you delete an intermediate anchor point, the adjacent anchor points above and below the deleted anchor point will be connected using a composite curve.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the previous anchor point and the next anchor point.
  • Double Curve – Creates a double curve segment between the previous anchor point and the next anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the previous anchor point and the next anchor point.

If you delete the last anchor point, the end of the wellpath will moved back to the previous anchor point.

Note

The first anchor point in the current plan cannot be deleted.

Deleting anchor points

To delete an anchor point or well segment:

  1. Right-click on the point in the data table that you want to delete, then select the Delete option.
  2. In the Delete Anchor Point window, select the type of curve you want to use to connect the adjacent anchor points.
  3. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the curve.
  4. Click OK to delete the anchor point.

The anchor point will be removed from the current plan. The wellpath connecting the adjacent anchor points will be recalculated, using the selected settings.

Field definitions

The table below defines the fields in the Delete Anchor Point window.

Fields in the Delete Anchor Point window
Field Name Description
Rejoin to Next Anchor Point

Method for connecting the anchor points adjacent to deleted anchor point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the previous anchor point and the next anchor point.
  • Double Curve – Creates a double curve segment between the previous anchor point and the next anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the previous anchor point and the next anchor point.
DLS1

Preferred dogleg severity of the shallow curved section.
DLS2

Preferred dogleg severity of the deep curved section.
DLS Iteration Method

Method for preserving dogleg severity values during iteration.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Preserve Final Attitude

When selected, the wellpath will connect to the end point of the wellpath with the selected curve using the DLS1 and DLS2 values as the preferred dogleg severities. Otherwise, the end point is connected with a curveHold curve using the DLS1 value as preferred dogleg severity. Only available when the second-to-last anchor point is selected.
