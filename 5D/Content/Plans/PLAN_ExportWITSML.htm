About WITSML exports

The WITSML export feature allows you to export wellpath plans from 5D to WITSML XML format. WITSML Wellsite Information Transfer Standard Markup Language is an industry-standard format used for exchanging drilling and well data between different systems and software applications.

The exported data will include the full trajectory information, including measured depth, inclination, azimuth, and coordinate data, formatted to meet the WITSML standard. Wellpath data can be exported either to a single file or to multiple files.

  • Single file output: One XML document containing all selected surveys and plans.
  • Multiple file output: Separate XML files for each survey and plan, named according to the trajectory names with non-alphanumeric characters replaced by hyphens.

You can also customize the precision settings and naming conventions for UIDs. See "WITSML Exports".

Exporting to WITSML

To export plan data to WITSML XML:

  1. In the Data List pane, right-click on the well whose plans you want to export.
  2. Go to Export to > WITSML, then select the Surveys/Plans option.
  3. In the Survey Export window that opens, select the plans that you want to export.
  4. In the UID Options field, select the elements to include in the UID for the trajectory stations. If no UID options are selected, station UIDs will be auto-generated GUIDs.
  5. To export all of the selected wellpaths to a single XML file, select the Export to one file option.
  6. Click the Browse button to navigate to the location where you want to save the export file. If the Export to one file option is selected, you'll also need to specify the file name.
  7. Click OK to begin the export.

When the export completes, a confirmation message will appear. The exported WITSML XML files will be saved to the specified location.

Note

When you open the Survey Export window from the List Plans tab, the list of surveys will be hidden.

WITSML export options

The table below defines the properties and options available in the Survey Export window.

Field definitions for the Survey Export window
Field Name Definition
Slot Name WITSML well name. Defaults to the 5D slot name but can be edited to match your organization's naming conventions.
Slot UID Unique identifier for the WITSML well. Defaults to the slot name with non-alphanumeric characters replaced by hyphens.
Well Name WITSML wellbore name. Defaults to the 5D well name but can be edited to match your organization's naming conventions.
Well UID Unique identifier for the WITSML wellbore. Defaults to the well name with non-alphanumeric characters replaced by hyphens.
Report from drill floor

Controls whether definitive surveys and current plans include trajectory data from the tie-on point to surface.

  • For main wells, this option is automatically checked and disabled since main wells always report to surface.
  • For sidetrack wells, this option is available and unchecked by default. Non-definitive surveys only export from their tie-on points regardless of this setting.
Include empty wellpaths Controls whether empty wellpaths, which only contain the top point in the wellpath, are shown in the survey list and plan list. By default, empty wellpaths are hidden from the lists.
Depth Precision Decimal precision for measured depth and true vertical depth values. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
Inclination Precision Decimal precision for wellpath inclination angles. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
Azimuth Precision Decimal precision for wellpath azimuth angles. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
Coordinate Precision Decimal precision for northing and easting coordinate values. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
Lat/Long Precision Decimal precision for latitude and longitude coordinates. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
DLS Precision Decimal precision for dogleg severity calculations. Accepts values from -1 to 6. Use -1 for no rounding, or 0-6 to round to the specified number of decimal places.
Surveys List of surveys for the selected well. The definitive survey is selected by default on first use. Not available when you open the Survey Export window from the List Plans tab.
Plans List of plans for the selected well. The current plan is selected by default on first use. Not available when you open the Survey Export window from the List Surveys tab.
UID Options

Elements to include in the naming convention for trajectory station UIDs. If no options are selected, the UIDs will be automatically generated.

  • MD – Base unique identifiers for trajectory stations on measured depth values. If any UID options are selected, MD must be included to ensure uniqueness.
  • INC – Include inclination values in UID generation for trajectory stations. Only available when MD is selected.
  • AZM – Include azimuth values in UID generation for trajectory stations. Only available when MD is selected.
Export to one file

Controls the number of files that the data is exported to.

  • When selected, combines all selected surveys and plans into a single XML document.
  • When not selected, creates individual XML files for each selected survey and plan.
Browse location

Specifies where to save the exported XML files.

  • When exporting to one file, click the Browse button to specify the exact file location and name.
  • When exporting to separate files, click the Browse button to specify the output directory. Files are automatically named based on survey or plan names with non-alphanumeric characters replaced by hyphens.
