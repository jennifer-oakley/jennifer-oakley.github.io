About projecting backwards

Projecting backwards allows you to work backwards from existing points along a specified trajectory. When you project backwards from a point in the current plan, a composite curve will be created to connect the selected point to the projected point. The length of the curve is controlled by the distance entered in the Backtrack field, entered as either a change in MD or a change in TVD.

There are two methods of projecting backwards:

  • The Project Back option inserts a new anchor point before the selected anchor point, at the depth entered in the Backtrack field.
  • The Edit as Project Back option edits the depth of the selected anchor point by the amount entered in the Backtrack field.
Note

If the backtrack distance is entered as a change in TVD, the value entered should be a negative value.

Projecting backwards to a point

To project backwards towards a point:

  1. Right-click on the point in the data table that you want to project from, then select the Project Back option or the Edit as Project Back option.
  2. In the Project Backwards window, select the type of curve you want to use to connect the selected point to the projected point.
  3. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the curve.
  4. In the Project Point Curve section, enter the parameters for the curve connecting the selected point to the projected point.
  5. Click OK to save your changes.

If you selected the Project Back option, the projected anchor point will be added to the current plan, and a curve will be created to connect the selected point to the projected point.

If you selected the Edit as Project Back option, the anchor point will be moved to the new position, and a curve will be created to connect the edited point to the wellpath.

Note

If you specify backtrack depth as a ΔTVD value, use negative values to place the projected point above the selected point.

Field definitions

The table below defines the fields in the Project Backwards window.

Fields in the Project Backwards window
Field Name Description
Join to Projected Anchor Point

Method for connecting the selected point to the projected point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the selected point and the projected point.
  • Double Curve – Creates a double curve segment between the selected point and the projected point.
  • Hold Double Curve – Creates a hold-double-curve segment between the selected point and the projected point.
DLS1

Preferred dogleg severity of the shallow curved section.
DLS2

Preferred dogleg severity of the deep curved section.
DLS Iteration Method

Method for preserving dogleg severity values during iteration.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Projected Point Curve

Parameters for the curve connecting the selected point and the projected point.

  • Curve DLS – Dogleg severity of the curve connecting the selected point and the projected point.
  • Toolface – Direction of the curve connecting the selected point and the projected point.
  • Backtrack – Distance of the projected point from the selected point, entered in either MD or TVD. If the projected point is entered in TVD, this value should be negative. If the projected point is entered in MD, this value should be positive.
Apply after every increment

Automatically applies changes after each parameter modification
