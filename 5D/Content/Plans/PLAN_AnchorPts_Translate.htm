About translating well segments

Translating a well segment allows you to move portions of your wellpath to new positions while keeping the same shape. This can be useful when you need to avoid obstacles or adjust the wellpath position without changing the overall geometry.

When you translate a segment, each point in the selected region keeps its original inclination and azimuth - you're just shifting the entire section to a new location. The first and last points in the well segment will connect to the wellpath using a composite curve with your preferred dogleg severities. The salient points below the well segment may also move as needed, depending on the frozen parameters set for anchor points below the well segment.

Translating well segments

To move a well segment to a new position:

  1. Press the Ctrl key and select the consecutive points that you want to move in the data table.
  2. Right-click on the highlighted rows and select the Translate option.
  3. In the Translate window, go to the Translate Values section and enter the change in coordinates from the original position of the well segment.
  4. In the Curve to First Translated Point section, select the type of composite curve you want to use to connect the first point in the well segment to the wellpath above it.
  5. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the first composite curve.
  6. In the Curve from Last Translated Point section, select the type of composite curve you want to use to connect the last point in the well segment to the wellpath below it.
  7. In the DLS1 and DLS2 fields, enter the preferred dogleg severities for the second composite curve.
  8. Click the OK button to save your changes and move the well segment.

The positions of the points in the well segment will be updated in the current plan, and the positions of the points below the well segment will be updated as needed to connect the segment to the wellpath.

Field definitions

The table below defines the fields in the Translate window.

Fields in the Translate window
Field Name Description
Translate Values

Change in the position of the well segment, entered using either Cartesian or polar coordinates.

  • Δ TVD – Change in TVD, relative to previous TVD of the well segment.
  • Δ North – Change in the north offset, relative to the current north offset of the well segment.
  • Δ East – Change in the easting offset, relative to the current north offset of the well segment.
  • Range – Horizontal distance from the current range of the well segment.
  • Bearing – Compass direction from the current bearing of the well segment.
Curve to First Translated Point

Method for connecting the first point in the well segment to the previous anchor point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the first point in the well segment and the previous anchor point.
  • Double Curve – Creates a double curve segment between the first point in the well segment and the previous anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the first point in the well segment and the previous anchor point.
DLS1

Preferred dogleg severity of the shallow curved section for the curve to the first translated point.
DLS2

Preferred dogleg severity of the deep curved section for the curve to the first translated point.
DLS Iteration Method

Method for preserving dogleg severity values during iteration for the curve to the first translated point.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Curve from Last Translated Point

Method for connecting the last point in the well segment to the next anchor point.

  • Curve Hold Curve – Creates a curve-hold-curve segment between the last point in the well segment and the next anchor point.
  • Double Curve – Creates a double curve segment between the last point in the well segment and the next anchor point.
  • Hold Double Curve – Creates a hold-double-curve segment between the last point in the well segment and the next anchor point.
DLS1

Preferred dogleg severity of the shallow curved section for the curve from the last translated point.
DLS2

Preferred dogleg severity of the deep curved section for the curve from the last translated point.
DLS Iteration Method

Method for preserving dogleg severity values during iteration for the curve from the last translated point.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between first and second dogleg severity.
  • Preserve DLS1 – Preserves the first dogleg severity.
  • Preserve DLS2 – Preserves the second dogleg severity.
Apply after every increment

Automatically applies changes after each parameter modification
Preserve Final Attitude

When selected, the wellpath will connect to the end point of the wellpath with the selected curve using the DLS1 and DLS2 values as the preferred dogleg severities. Otherwise, the end point is connected with a curveHold curve using the DLS1 value as preferred dogleg severity. Only available when the second-to-last anchor point is selected.
