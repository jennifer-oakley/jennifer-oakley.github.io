About anchor points

When constructing a wellpath, individual sections are connected by anchor points that fully define the wellpath's position and attitude. Anchor points are special salient points that can be edited to change the wellpath geometry. Unlike ordinary points, you can edit the position and attitude of an anchor point. You can also freeze specific parameters to control how anchor points connect to other points in the wellpath. See "Freezing parameters".

Managing anchor points

To update an anchor point in the current plan, right-click on the anchor point in the data table.

  • Add – Adds a point at a specified MD or TVD along the current wellpath trajectory. See "Adding anchor points".
  • Append – Extends the wellpath by adding an anchor point to the end of the wellpath. Only available for the last anchor point in the current plan. Inserts an anchor point between the selected point and the next selectable point. See "Appending anchor points".
  • Delete – Removes the selected anchor point and reconnects the wellpath using composite curves. See "Deleting anchor points".
  • Project Back – Adds a new anchor point by projecting a curved section backwards from the selected point. See "Projecting backwards".
  • Edit Comment – Adds or modifies text comments associated with the anchor point. See "Editing comments".
  • Edit – Modifies the position and/or attitude of the selected anchor point. See "Editing anchor points".
  • Edit as Project Back – Edits the anchor point by projecting backwards from its current position. See "Projecting backwards".
  • Freeze Parameters – Specifies which parameters remain constant when the wellpath is edited. See "Freezing parameters".
  • Insert After –Inserts an anchor point between the selected point and the next point. See "Inserting anchor points".
  • Join to Target – Connects the wellpath from the selected point to a target set. See "Projecting to a target".
  • Create Target – Creates a new target at the anchor point location. See "Creating a target".

Managing ordinary points

To update an ordinary point in the current plan, right-click on the ordinary point in the data table.

  • Add – Adds a point at a specified MD or TVD along the current wellpath trajectory. See "Adding anchor points".
  • Promote – Converts the ordinary point to an anchor point for editing capabilities.
  • Edit Comment – Adds or modifies text comments associated with the anchor point. See "Editing comments".
  • Freeze Parameters – Specifies which parameters remain constant when the wellpath is edited. See "Freezing parameters".
  • Create Target – Creates a new target at the anchor point location. See "Creating a target".

Managing well segments

To select a well segment in the current plan, press the Ctrl key and select multiple consecutive points in the data table. To update the well segment, right-click on the well segment.

  • Translate – Moves the selected wellpath segment by specified displacement values. See "Translating well segments".
  • Translate on Attitude – Moves the selected segment along the current wellpath attitude direction. See "Translating well segments on attitude".
  • Delete – Removes the selected anchor point and reconnects the wellpath using composite curves. See "Deleting anchor points".
  • Rotate – Rotates the selected wellpath segment by a specified azimuth change. See "Rotating well segments".
