About wellplan recovery

If the wellpath has deviated from the current plan, you can recover the wellplan by joining the definitive survey to the current plan. There are two options available in the Wellplan Recovery panel:

  • Recovery In – Calculates the recovery points required to join the definitive survey to the current plan within a specified distance. The position of the recovery points is determined by the specified distance and dogleg severity settings in the Field Operations Settings window. See "Configuring dogleg severity".
  • Recovery By – Calculates the recovery points required to join the definitive survey to the current plan at a specified measured distance. The position of the recovery points is determined by specified depth and the dogleg severity settings in the Field Operations Settings window. See "Configuring dogleg severity".

When you select a recovery method, the recovery survey points will appear in the data table. The recovery wellpath segment will be created using the CurveHoldCurve method. Recovery points will be shaded in orange in the data table.

Note

If the current plan is a reactive plan, you can only create a recovery plan that begins after the existing recovery sections.

Configuring dogleg severity

To configure the settings for dogleg severity:

  1. Click the Settings button () in the toolbar at the top of the Field Operations window.
  2. In the Field Operations Settings window that opens, go to the Survey Settings tab.
  3. In the DLS1 field, enter the desired dogleg severity of the first curved section in the recovery section.
  4. In the DLS2 field, enter the desired dogleg severity of the second curved section in the recovery section.
  5. In the DLS Iteration Method field, select the desired method for preserving dogleg severity in the recovery section.
  6. Click OK to save your changes.

When you create a recovery plan in the Wellplan Recovery panel, the settings in the DLS1 field, the DLS2 field, and the DLS Iteration Method field will be used to determine the position of the recovery survey point.

Creating a recovery plan

To create a recovery plan:

  1. Configure the settings for dogleg severity. See "Configuring dogleg severity".
  2. Go to the Wellplan Recovery panel in the Field Operations window.
  3. Select the method you want to use for recovering the wellplan.
    • To recover within a specified distance, select the Recovery In option.
    • To recover within a specified measured depth, select the Recovery By option.
  4. Enter the distance or depth that you want to recover within.
  5. In the Tool used in recovery field, select the survey tool to you plan to use.

The recovery survey points will appear in the data table, shaded in orange. If you change the recovery method or the recovery distance, the position of the recovery points will be automatically recalculated. To create a new reactive plan based on the recovery plan, click the Record As New Plan button () in the toolbar on the right side of the data table.

Wellplan recovery settings

The table below defines the settings in the Wellplan Recovery panel.

Settings in the Wellplan Recovery panel
Section Property Name Definition
-- Not Required

Recovery is not required. Any existing recovery sections will be deleted from the data table and from the charts in the Graph pane.
-- Recovery In

Joins the last point in the definitive survey to the current plan, within the specified distance
-- Recovery By

Joins the last point in the definitive survey to the current plan, at the specified measured depth
-- Tool used in recovery

Survey tool used to calculate the points in the recovery section
Projected Target

Target that you want to project to. This option is only available when the Projected graph is selected in the Graph pane.
Projection

View shown in the Graph pane. This option is only available when the Projected graph is selected in the Graph pane.

  • Elevation – Section view of the projected wellpath.
  • Top – Top view of the projected wellpath.
  • Along Path – Projects the wellpath onto the plane that is perpendicular to the last point in the data table.
Font Size

Font size used in the Projected Values window in the Graph pane. This option is only available when the Projected graph is selected in the Graph pane.
Travelling Cylinder Range

Radius of the chart shown in the Graph pane. This option is only available when the Trav. Cyl. chart is selected in the Graph pane.
Ring Interval

Distance between the radial marks shown in the Graph pane. This option is only available when the Trav. Cyl. chart is selected in the Graph pane.
MD of Error Ellipse

Measured depth at which the error ellipses of offset wells appear in the Graph pane. This option is only available when the Trav. Cyl. chart is selected in the Graph pane.
Quality Control Inc Max

Maximum value for inclination that will be shown in the Inc chart. This option is only available when the Qual. Ctrl graph is selected in the Graph pane.
Az Min

Minimum value for azimuth that will be shown in the Az chart. This option is only available when the Qual. Ctrl graph is selected in the Graph pane.
Az Max

Maximum value for azimuth that will be shown in the Az chart. This option is only available when the Qual. Ctrl graph is selected in the Graph pane.
DLS/30 m Max

Maximum value for dogleg severity that will be shown in the DLS chart. This option is only available when the Qual. Ctrl graph is selected in the Graph pane.
Consistency % Max

Maximum absolute value for consistency that will be shown in the HS.Cons and the Lat. Cons charts. This option is only available when the Qual. Ctrl graph is selected in the Graph pane.
