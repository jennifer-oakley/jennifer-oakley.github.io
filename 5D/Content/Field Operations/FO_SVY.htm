About survey projection

In the Survey Projection panel, you can project the survey beyond the last point in the definitive survey. There are two options available in the Survey Projection panel:

  • Project To Bit – Calculates the projected position of the bit. The distance between the last survey point and the bit is determined by the projection settings in the Field Operations Settings window. See "Configuring survey projection".
  • Project Beyond Bit – Calculates the projected position of a point beyond the bit. The distance between the bit and the point you are projecting to is determined by the projection settings in the Field Operations Settings window. See "Configuring survey projection".

When you select a projection method, the projected survey point will appear in the data table. Projected survey points will be shaded in green in the data table. The parameters used to calculate the projected points will be shaded in gray.

You can also create projections by projecting to additional points beyond the bit. The distance between the additional projected point and the last real or projected survey point is determined by the Project To Bit value in the projection settings. The additional projected points are shaded in yellow in the data table. See "Adding projections".

Configuring survey projection

To configure the settings for survey projection:

  1. Click the Settings button () in the toolbar at the top of the Field Operations window.
  2. In the Field Operations Settings window that opens, go to the Survey Settings tab.
  3. In the Project to Bit field, enter the estimated distance between the last survey point and the bit.
  4. In the Project Beyond Bit field, enter the distance between the bit and the survey point you want to project to.
  5. Click OK to save your changes.

When you project ahead in the Survey Projection panel, the settings in the Project to Bit field and the Project Beyond Bit field will be used to determine the measured depth of the projected survey points. See "Projecting beyond the survey". When you add additional projections beyond the bit, the value in the Project to Bit field will be used to determine the distance between the additional projected point and the last real or projected survey point. See "Adding projections".

Adding survey points

To add a survey point:

  1. Go to the data table in the Field Operations window.
  2. Double-click inside the first cell in the blank red row below the last survey point.
  3. Enter the MD, inclination, and azimuth of the survey point.

The remaining survey parameters will be automatically calculated, and a new blank row will appear below the new survey point. If there are any points projected beyond the survey, the position of the projected points will be automatically recalculated.

Projecting beyond the survey

To project to points beyond the last point in the definitive survey:

  1. Configure the settings for survey projection. See "Configuring survey projection".
  2. Go to the Survey Projection panel in the Field Operations window.
  3. In the Project To Bit field, select a method for projecting the survey to the drill bit. See "Projection methods".

  4. In the Project Beyond Bit field, select a method for projecting the survey beyond the drill bit.

The projected survey points will appear in the data table, shaded in green. The parameters that were used to calculate the projected points will be shaded in gray. If you add a new survey point to the definitive survey, the position of the projected points will be automatically recalculated.

Tip

To change the projection method, select a new projection method in the Survey Projection panel. The position of the projected points will be automatically recalculated, using the new projection method.

Adding projections

To project to additional points beyond the survey:

  1. Configure the settings for survey projection. See "Configuring survey projection".
  2. Go to the data table in the Field Operations window.
  3. Right-click on the last actual or projected survey point.
  4. Select the Add Projection option, then select a method for projecting the survey. See "Projection methods".

  5. Enter the parameters for the projected point in the

The additional projected survey points will appear in the data table, shaded in yellow. The parameters that were used to calculate the projected points will be shaded in gray, and the remaining parameters will be automatically calculated. If you add a new survey point to the definitive survey, the position of the projected points will be automatically recalculated.

Tip

To project to a target, select the TVD, North, East projection method, then enter the TVD, northing, and easting coordinates of the target. See "Projection methods".

Tip

To change the projection method, right-click on the projection and select the Change Projection option. The position of the projected points will be automatically recalculated, using the new projection method.

Projection methods

The table below defines the projection methods available for projecting to the bit and projecting beyond the bit.

Definitions of projection methods
Projection Method Definition
Trend to CL Position is calculated based on the trend in course length over multiple sections. This option is only available when projecting to the bit.
Trend to TVD Position is calculated based on the trend in TVD over multiple sections. This option is only available when projecting to the bit.
B. Rate, T. Rate and CL Position is calculated based on the build rate, turn rate, and course length. The default values for build rate, turn rate, and course length can be overwritten, but all other values are automatically calculated.
B. Rate, T. Rate and TVD Position is calculated based on the build rate, turn rate, and TVD. The default values for build rate, turn rate, and TVD can be overwritten, but all other values are automatically calculated.
DLS, T. Face and CL Position is calculated based on the dogleg severity, toolface, and course length. The default values for dogleg severity, toolface, and course length can be overwritten, but all other values are automatically calculated.
DLS, T. Face and TVD Position is calculated based on the dogleg severity, toolface, and TVD. The default values for dogleg severity, toolface, and TVD can be overwritten, but all other values are automatically calculated.
Inc, Az and CL Position is calculated based on the inclination, azimuth, and course length. The default values for inclination, azimuth, and course length can be overwritten, but all other values are automatically calculated.
Inc, Az and TVD Position is calculated based on the inclination, azimuth, and TVD. The default values for inclination, azimuth, and TVD can be overwritten, but all other values are automatically calculated.
TVD, North, East Position is calculated based on TVD, northing, and easting. The default values for TVD, northing, and easting can be overwritten, but all other values are automatically calculated.
TVD, North, East, Inc and Az Position is calculated based on TVD, northing, easting, inclination, and azimuth. The default values for TVD, northing, easting, inclination, and azimuth can be overwritten, but all other values are automatically calculated.
