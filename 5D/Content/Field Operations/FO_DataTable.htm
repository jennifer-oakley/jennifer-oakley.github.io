About the data table

The Data Table pane contains tables with information from the definitive survey, current plan, targets, and AC101 projections. There are four tabs in the Data Table pane:

  • The Survey tab shows the last point in the definitive survey, as well as any survey projections or recovery sections. The data on the Survey tab is color-coded, based on the type of survey point. See "Color coding".
  • The Current Plan tab shows all of the points in the current plan. The information on the Current Plan tab is read only, and cannot be edited.
  • The Targets tab allows you to view and edit the targets from the target set associated with the current well.
  • The AC101 tab shows TVD projections for the nearest well, based on the selected TVD projection method. The data on the AC101 tab is color-coded, based on the type of survey point. See "Color coding".

Color coding

The rows in the data table are color-coded to indicate the type of wellpath section for each row. There are five types of wellpath sections in the data table:

  • Definitive survey sections are shown in red. The definitive survey section includes the last point in the definitive survey and a blank line, which can be used to add new survey points to the definitive survey.
  • Projection sections are shown in green. The projection section includes projections to the bit and projections beyond the bit, but does not include any additional projections. See "Projecting beyond the survey".
  • Additional projection sections are shown in yellow. The additional projections section can include an unlimited number of projections beyond the survey. See "Adding projections".
  • Recovery sections are shown in orange. The recovery section includes the recovery survey points required to join the definitive survey to the current plan. See "Wellplan recovery".
  • Recovered plan sections are shown in blue. The recovered plan section includes the points from the current plan that are included in the reactive plan.
Tip

You can change the survey tool assigned to survey points in the definitive survey. To change the survey tool, add the Survey Tool column to the data table. Go to the Survey Tool field in the data table and click the Browse button (...).

Column options

The table below defines that columns that can be added to the data table on the Survey tab, the Current Plan tab, and the AC101 tab. To add a column to the data table, click the arrow next to the Options button () and select the column you want to add.

List of available columns for the data table
Column Name Definition
MD

Measured depth of the current point from the drill floor
Inc

Inclination of well at the current point, with zero degrees being vertical down
Az

Azimuth of well at the current point, measured clockwise from the site North reference
TVD

True vertical depth of the current point, measured from the TVD reference point
Elevation

Elevation above the vertical reference datum (VRD)
N. Offset

North offset of the current point from the site or slot center in the local North direction
E. Offset

East offset of the current point from the site or slot center in the local East direction
DLS

Dogleg severity of the well path between this point and the previous salient point
T. Face

Angle of the toolface from the high side of the wellbore
B. Rate

Change in inclination of the well path between this point and the previous salient point
T. Rate

Effective change in azimuth of the well path between this point and the previous salient point
VS

Projected distance of the current point from the vertical section starting position, along the vertical section azimuth
CL

Change in measured depth between this point and the previous point
Dist to Plan Distance between the survey point and the equivalent point in the current plan. This column is not available on the Current Plan tab.
Right to Plan Distance between the survey point and the right side of the equivalent point in the current plan. This column is not available on the Current Plan tab.
High to Plan Distance between the survey point and the high side of the equivalent point in the current plan. This column is not available on the Current Plan tab.
T. Face to Plan Angle of the toolface from the equivalent point in the current plan. This column is not available on the Current Plan tab.
Survey Tool Name of the survey tool used to take the survey. This column is not available on the Current Plan tab.
Source Survey Name of the source survey where the survey point is stored. This column is not available on the Current Plan tab.
P. Type

Type of plan point, such as anchor point, survey point, or ordinary point
Sec. Type Type of section, such as projection section or recovery section. This column is not available on the Current Plan tab.
Comment

Any comments about the current point
Tortuosity

Tortuosity of the wellpath at the current point
DDI

Rating for the Directional Difficulty Index (DDI)

Using the data table

The toolbar on the right side of the data table is used to customize the columns shown, create reactive plans, and create drilling data sheets.

Toolbar buttons in the Data Table pane
Button Function
Adds and removes columns from the view in the data table. See "Column options".
Creates a new reactive plan
Opens, creates, exports, or deletes the current drilling data sheet
Log Opens the Job Log window, which shows all changes made in the Field Operations window
