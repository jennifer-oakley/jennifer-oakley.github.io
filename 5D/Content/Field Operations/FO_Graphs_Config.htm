About display settings

You can add several items to the Section View, Top View, and Projected charts, such as lease lines, depth labels, casing points, well annotations, targets, salient points,, formations, wellpaths, and error ellipsoids. The toolbar on the right side of the Graph pane is used customize the items shown on the charts. This topic defines the settings that control the appearance of these items.

Lease line settings

The table below defines the display settings for lease lines in the Field Operations window. To change the display settings for lease lines, click the Lease Lines button () in the Graph pane. Lease lines are only shown on the Top View chart.

Display settings for lease lines
Property Name Definition
Show Lease Lines

Shows the lease lines on the plot
Lease Line Size Width and style of the lease lines associated with the well
Offset Line Size Width and style of the lease lines associated with offset wells
Change Offset Lease Line's Colours Uses a different line color to represent offset lease lines.
Color Color used to represent lease lines in the chart. When the Change Offset Lease Line's Colours option is selected, the color used for offset lease lines will be updated.

Depth label settings

The table below defines the display settings for depth labels in the Field Operations window. To change the display settings for depth labels, click the Depth Labels button () in the Graph pane. Depth labels are shown on the Section View, Top View, and Projected charts.

Display settings for depth labels
Property Name Definition
Show Plan Depth Labels

Adds depth labels for the _wellpathType at intervals, based on the settings in the Interval Type and Interval sections
Show Survey Depth Labels

Adds depth labels for the definitive survey at intervals, based on the settings in the Interval Type and Interval sections
Text Size

Text size to use for Depth labels
Font

Font to use for Depth labels
Contents

Information that will be included in each depth label
Interval Type

Survey point property used to define the intervals
Interval

Interval over which depth labels will be shown

  • Start – Start depth of the first interval.
  • Stop – End depth of the last interval.
  • Step – Length of each interval.
  • Increment – Controls the amount that the values in the Start, Stop, and Step fields change when you use the arrows to change the values.
  • Always show final point –Shows a depth label for the last point in the current plan or definitive survey.
Color

Font color to use for Depth labels

Wellpath settings

The table below defines the display settings for wellpaths in the Field Operations window. To change the display settings for wellpaths, click the Wellpaths button () in the Graph pane. Wellpaths are shown on the Section View, Top View, and Projected charts.

There are separate display settings for labels, the primary well, offset wells, and the highlighted offset well. To update the display settings, select the component that you want to update.

  • Labels – Display settings for the labels for each wellpath.
  • Main well – Display settings for the lines representing the current plan and definitive survey of the primary well.
  • Offset wells – Display settings for the lines representing the wellpaths of offset wells included in field operations.
  • Highlighted offset well – Display settings for the line representing the offset well that is currently selected in the Proximity Analysis pane.
Display settings for wellpaths
Property Name Definition
Show Well Path Labels Shows a label for each wellpath shown in the graph
Show Offset Wells in Section View Shows the offset wells included in field operations
Show Drilling Corridors Shows the drilling corridor for the primary well
Label Angle

Angle of the text in each label

Length of the lines connecting the labels to the Wellpaths

Adds arrows that connect the labels to the Wellpaths

Error ellipsoid settings

The table below defines the display settings for error ellipsoids in the Field Operations window. To change the display settings for error ellipsoids, click the Error Ellipsoids button () in the Graph pane. Error ellipsoids are shown on the Section View, Top View, and Projected charts.

There are separate display settings for the definitive survey, current plan, and offset wells. To update the display settings, select the component that you want to update.

  • Survey – Display settings for the definitive survey.
  • Plan – Display settings for the current plan.
  • Offset wells – Display settings for the offset wells included in field operations.
Display settings for error ellipsoids
Property Name Definition
Show Plan Error Ellipsoids Shows the error ellipsoids for the current plan
Show Survey Error Ellipsoids Shows the error ellipsoids for the definitive survey
Show Offset Wells Error Ellipsoids Shows the error ellipsoids for the offset wells included in field operations
Line Size

Width of the lines used to draw Error ellipsoids
Style Line style used to draw error ellipsoids
Start Start depth of the section over which error ellipsoids will be shown
Stop End depth of the section over which error ellipsoids will be shown
Update with current end MD Sets the value in the Stop field to match the last point in the current plan or definitive survey
Step Length of each interval in the section showing error ellipsoids
Always show final point Shows an error ellipsoid for the last point in the current plan or definitive survey.
Increment Controls the amount that the values in the Start, Stop, and Step fields change when you use the arrows to change the values.
Display

Method to use for drawing error ellipsoids

  • Bounding Ellipse – Draws a two-dimensional error ellipsoid.
  • 3D Ellipsoid – Draws a three-dimensional error ellipsoid.
  • Both – Draws a two-dimensional error ellipsoid and a three-dimensional error ellipsoid.
Draw Vector Arrows Adds vector arrows to represent the high side vector and lateral vector of each error ellipsoid
Color

Color to use for Error ellipsoids
