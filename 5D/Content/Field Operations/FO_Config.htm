About field operation settings

The Field Operations Settings window allows you to customize the settings for surveys, plots, and well data. There are three tabs in the Field Operation Settings window:

  • The Survey Settings tab contains settings for recovery sections, survey reports, anti-collision reports, magnetic parameters, and projection distances. See "Survey settings".
  • The Plot Settings tab contains settings for the charts in the graph pane. These settings only apply to the Section View, Top View, and Projected charts. See "Plot settings".
  • The Well Data tab contains settings for the start position and end position of the vertical section in the well. See "Well data settings".

To open the Field Operations Settings window, click the Settings button () in the toolbar at the top of the Field Operations window.

Survey settings

The table below defines the settings on the Survey Settings tab in the Field Operation Settings window.

Survey settings in the Field Operations Settings window
Section Property Name Definition
Recovery Section DLS DLS1 Default dogleg severity of the first curved section in the recovery section
DLS2 Default dogleg severity of the second curved section in the recovery section
DLS Iteration Method

Method for preserving dogleg severity in the recovery section.

  • Preserve DLS1/DLS2 Diff – Preserves the difference between the dogleg severity in the first curved section and the second curved section.
  • Preserve DLS1 – Preserves dogleg severity in the first curved section, and allows dogleg severity to change in the second curved section.
  • Preserve DLS2 – Preserves dogleg severity in the second curved section, and allows dogleg severity to change in the first curved section.
-- Show All Survey Points Displays all of the survey points in the definitive survey
Survey Report Survey Report includes

Controls the survey points included in a survey report.

  • Survey section only – Report will only include the survey points in the definitive survey.
  • Survey and projection sections – Report will include the survey points in the definitive survey and projected survey points.
  • Survey, projection, and recovery sections – Report will include the survey points in the definitive survey, projected survey points, and recovery survey points.
Anti-Collision Interval Length of the interval used for anti-collision analysis
AC report includes

Controls the survey points included in an anti-collision report.

  • Survey section only – Report will only include the survey points in the definitive survey.
  • Survey and projection sections – Report will include the survey points in the definitive survey and projected survey points.
  • Survey, projection, and recovery sections – Report will include the survey points in the definitive survey, projected survey points, and recovery survey points.
Show AC warning for SF Threshold Shows an anti-collision warning when the threshold for the separation factor is exceeded
Magnetic Parameters Date

Date the magnetic parameters were calculated. This should typically be the survey date when directional measurements are taken, as Earth's magnetic field changes over time.
Field Strength

Strength of the magnetic field at the well's location
Model

Model used to calculate the magnetic parameters
Declination

Clockwise bearing of the horizontal component of the Earth's magnetic field vector from true north
Dip

Angle between the local horizontal vector and the Earth's magnetic field vector
Advanced AC 101 Options Enable showing proximity data for AC101 steps Shows the proximity data for anti-collision intervals. Only available when advanced AC101 is enabled.
Show all offsets in tracking sheet Shows all offset wells that have been logged in AC101 in the tracking sheet. By default, only the current offset wells are shown.
Projection Options Project to Bit Distance to project from the last survey point to the bit
Project Beyond Bit Distance to project beyond the bit

Well data settings

The table below defines the settings on the Well Data tab in the Field Operation Settings window.

Well data settings in the Field Operations Settings window
Section Property Name Definition
Start Position Relative To

Reference datum that the starting position of the vertical section is relative to

  • Site – Start position is relative to the site center.
  • Slot – Start position is relative to the slot center.
North Offset

Distance that northing is offset from the reference datum
East Offset

Distance that easting is offset from the reference datum
End Position Reference

Reference used to determine the azimuth for the ending position of the vertical section

  • None – Azimuth is not calculated, and must be manually entered.
  • Total Depth – Azimuth is calculated from the total depth of the current plan or the definitive survey.
  • Target – Azimuth is calculated from the target depth of the current plan or the definitive survey. Only available when there is a target set associated with the well.
Well Path Type

Type of well path used to calculate azimuth. Not available when the None option is selected for the end position reference.
-- Azimuth

Azimuth along which to project the wellpath. This value may be manually entered or calculated, depending on the option selected for the end position reference.

  • If the None option is selected, this value must be manually entered.
  • If the Total Depth or Target option is selected, this value will be calculated when you click the Calculate button.
