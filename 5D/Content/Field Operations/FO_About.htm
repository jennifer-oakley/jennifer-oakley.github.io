About field operations

The Field Operations window gives you an at-a-glance view of a well. You can view the results of proximity analysis and anti-collision analysis, view charts, project beyond the survey, create reports, and recover the wellplan.

The Field Operations window is divided into five sections:

  • The Proximity Analysis table shows the proximity of the primary wellpath to each secondary wellpath. See "Anti-Collision Analysis".
  • The Graph pane contains charts that show different views of the wellpath, as well as diagnostic information about the wellpath. See "Graph pane".
  • The Survey Projection panel allows you to project the survey beyond the last point in the definitive survey. See "Survey projection".
  • The Wellplan Recovery panel allows you to recover the wellplan by joining the definitive survey to the current plan. See "Wellplan recovery".
  • The Data Table pane contains tables with information from the definitive survey, current plan, targets, and AC101 projections. See "Data Table".

To open the Field Operations window, right-click on the well name in the Data List pane and select the Field Operations option. See "Starting field operations".

Starting field operations

When you start field operations, the data from the current plan and the definitive survey will be loaded with the data from the most recent survey. The selected survey must have the same drill floor TVD as the current plan and the definitive survey. Additionally, the measured depth of the last survey point in the selected survey must be greater than or equal to the last point in the definitive survey.

To start field operations:

  1. In the Data List pane, right-click on the well and select the Field Operations option.
  2. In the Current Survey Selection window, select the most recent survey for the well. If there are no surveys, you will be prompted to create one.
  3. To enable anti-collision analysis, select the Enable AC101 when Field Ops starts option.
  4. Enter the rig name, run number, and BHA number.
  5. In the Surface Slot Separation field, enter the distance between slots at the surface.
  6. Click OK.
  7. In the Well Path Selector window, select the offset wellpaths that you want to include for proximity analysis.
  8. Click OK.

The Field Operations window will open. All survey points beyond the last point in the definitive survey will be shown in the data table. Any survey points added in the Field Operations window will also be added to the selected survey and the definitive survey.

Note

In the Well Path Selector window, the plan and survey options will disabled for well that you are running field operations on.

Using the Field Operations window

The toolbar at the top of the Field Operations window is used to configure the settings for field operations, generate reports, open the Plot window, perform a proximity analysis, and print charts. To zoom the selected chart, change the value in the Zoom field on the right side of the toolbar.
