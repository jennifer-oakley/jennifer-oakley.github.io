About report utilities

5D includes several report utilities, which are launched outside of 5D. These utilities generate reports using specific, pre-defined templates. There are several utilities available:

  • The Shell Canada Header Report utility creates a definitive survey in the format required by Shell Canada. To create a Shell Canada Header report, go to the 5D 9.8.0 folder in the Start Menu and select the Shell Header Report option. The report will be generated in a CSV and text format, and it will be saved to your Documents folder on your C drive.
  • The Manitoba Header Report utility creates a definitive survey in the format required for wells in Manitoba, Canada. To create a Manitoba Header report, go to the 5D 9.8.0 folder in the Start Menu and select the Mantitoba Header Report option. The report will be generated in a CSV format, and it will be saved to your Documents folder on your C drive.
  • The AER report utility creates a definitive survey report using the format required for energy regulators. AER header reports are created in a CSV format and an XLSX format. See "AER reports".
  • The P7_2000 report utility creates a definitive survey report using the P7/2000 data exchange format for well deviation data. P7_2000 header reports are created in a text format. See "P7_2000 reports".

Before you can create a report using these utilities, you will need to generate a definitive survey report. To generate a definitive survey report, right-click on the well that you want to create a report for. Go to Reports and select the Definitive Survey option.

AER reports

Using the AER Report Generator, you can create single reports or batch reports. When you create a single AER report, the report will be generated from the most recent definitive survey report created in 5D. See "Creating single AER reports". When you create a batch report, you will need to select the batch report directory that you want to generate reports from. See "Creating multiple AER reports".

Creating single AER reports

To create a single AER report:

  1. Right-click on the well that you want to create a report for, then go to Reports and select the Definitive Survey option.
  2. In the Report Config window, click OK to create the report. The report will open automatically.
  3. Double-click the 5D AER Header Report icon on your desktop.
  4. In the AER Report Generator window that opens, select the For Single AER Report option.
  5. Click the Choose Directory button.
  6. Navigate to the location where you want to save the report, then click OK.
  7. In the AER Report Generator window, click OK.

When the report has finished generating, a notification message will appear. The report will be saved to the selected directory, in a CSV format and an XLSX format.

Creating multiple AER reports

To create multiple AER reports:

  1. Right-click on the field that you want to generate batch reports for, then go to Reports and select the Field Batch Reports option.
  2. In the Report Config window, select the plans and surveys that you want to create reports for. The directory that the reports are saved to will open automatically.
  3. Double-click the 5D AER Header Report icon on your desktop.
  4. In the AER Report Generator window that opens, select the For Batch AER Report option.
  5. Click the Select Batch Directory button.
  6. Navigate to the location where the batch reports were saved, then click OK.
  7. In the AER Report Generator window, click OK.

When the report has finished generating, a notification message will appear. The reports will be saved to the selected directory, in a CSV format and an XLSX format.

P7_2000 reports

To create a P7_2000 report:

  1. Right-click on the well that you want to create a report for, then go to Reports and select the Definitive Survey option.
  2. In the Report Config window, click OK to create the report. The report will open automatically.
  3. Double-click the P7_2000 icon on your desktop.
  4. In the P7_2000 Report Window window that opens, verify the correct well and field names are shown.
  5. Click the Choose File button, then navigate to the location where you want to save the report,
  6. Enter a name for the report, then click Save.
  7. In the P7_2000 Report Window window, click OK.

The report will be saved to the selected directory in a text format.
