Report Formatting

The Format page contains the default settings for reports. The settings on this page apply to all reports. To edit the default report settings, click the Options and Settings button () in the main menu and go to the Reporting > Format page.

List of report format settings
Section Field Name Description
-- Latitude / longitude format Display format to use for latitude and longitude values
-- Coordinate precision Decimal precision to use for coordinate values
-- Inclination precision Decimal precision to use for inclination values
-- Latitude /longitude precision Decimal precision to use for latitude and longitude values
-- Azimuth precision Decimal precision to use for azimuth values
-- DLS precision Decimal precision to use for dogleg severity values
-- Toolface precision Decimal precision to use for toolface values
-- Depth precision Decimal precision to use for depth values
-- Casing diameter precision Decimal precision to use for casing diameters
-- Vertical depth display

Method to use for displaying elevation values

  • As elevation – Displays elevation values relative to the elevation of the drill floor.

  • As TVD relative to field vertical reference datum – Displays elevation values relative to the field vertical reference
Cover Page (checkbox) Adds a cover page to the report
Client logo Path to the image for the client logo
Prompt for comment Includes a comment on the cover page. When you generate a report, you will be prompted to enter a comment.
Show well comment Includes the well comment on the cover page. To enter a comment for the well, edit the Comment field in the well properties. See "Well properties".
Show well identifier Includes the well identifier on the cover page. To enter an identifier for the well, edit the Identifier field in the well properties. See "Well properties".
Output Options Table font size Font size to use for tables in the report
Language Language to use for text in the report
Orientation Page orientation to use for the report
-- Report in an alternative co-ordinate system Generates the report in a different coordinate system than the projected coordinate system for the field
-- Report in units Distance unit to use for the report
-- Prompt for report name Uses a custom file name for the report. When you generate a report, you will be prompted to enter a file name for the report.
