Anti-collision report settings

Related Topics
Related Topics
  • Anti-Collision Analysis
  • Anti-collision reports

The Anti-Collision Report page contains the settings for anti-collision reports. The settings on this page apply to anti-collision reports generated from the Anti-Collision window. To edit the anti-collision report settings, click the Options and Settings button () in the main menu and go to the Reporting > Anti-Collision Report page.

When you click the Customise button in each section, you can select which columns you want to appear in the report.

Definitions for anti-collision report settings
Option Definition
Header

Includes a header with the report type and page number. For additional header options, click the Customise button.
Field

Includes information about the field in the report, including the map unit, vertical reference datum, projected coordinate system, and company name. To add custom parameters about the field, click the Customise button.
Site

Includes information about the site in the report, including the site unit, north reference, convergence angle, site position, TVD reference, and elevation above MSL.
Slot

Includes information about the slot in the report, including the slot position and elevation above MSL.
Well

Includes information about the well in the report, including the type of well, file number, UWI, plan version, closure distance, closure azimuth, vertical section, and magnetic parameters.
Drill Floor

Includes information about the drill floor in the report, including the rig height, elevation above MSL, inclination, and azimuth.
Well schematic

Includes a schematic of the well sections, including the casing depths, casing diameters, and survey tool ranges. Only applies to uncertainty reports.
Primary well Includes the complete analysis results for the primary well. To customize the information shown in the Primary Well section of the report, click the Customise button.
Secondary wells Includes the complete analysis results for each secondary well. To customize the information shown in the Secondary Well sections of the report, click the Customise button.
List secondary wells Includes a list of the secondary wells used in the analysis, listed under the Anti-Collision Analysis Criteria section in the report.
AC summary Includes a summary of the points on the secondary wellpaths that pose the highest risk potential, based on the separation factors, ellipsoid separations, and center-to-center distances from the analysis results. To customize the information shown in the Anti-Collision Summary section of the report, click the Customise button.
Survey tool information

Includes information about the survey tool ranges for the primary wellpath, including tool name, start MD, end MD, and the source survey.
Risk summary Includes a summary of depth ranges of each secondary wellpath that pose a risk of collision, based on the separation factors, ellipsoid separations, and center-to-center distances from the analysis results.
Points to include

Specifies which analysis points to include in the report

  • Interpolated points only – Only includes interpolated points between survey points. Survey points will not be included.
  • Survey points with interpolated points – Includes survey points and interpolated points.
  • Survey points without interpolated points between sections – Only includes survey points. Interpolated points will not be included.
Specify depth range

Filters the points included by a specified depth range. When this option is selected, you will be prompted to enter a depth range when you generate the report.
AC report filter

Filters the report to only include analysis points which fall below the threshold values set in the AC Options > Scan Values panel. See "Anti-collision options".

  • Separation factor – Only includes analysis points with a separation factor below the Sep. Factor threshold in the Scan Values panel.
  • Centre to centre distance – Only includes analysis points with a center-to-center distance below the Cr-Cr threshold in the Scan Values panel.
  • Include all secondary well paths – Includes analysis points below the selected thresholds for all secondary wellpaths.
Orientation of ellipsoid dimensions

Direction onto which the uncertainty ellipsoid will be projected.

  • Wellbore (Highside, Lateral, Along Hole) – Uncertainty ellipsoid will be projected relative to the wellbore. The high side will be the secondary major axis, and the lateral plane will be the secondary minor axis.
  • Grid (TVD, North, East) – Uncertainty ellipsoid will be projected relative to the grid. North will be the secondary major axis, and East will be the secondary minor axis
Display dimensions Includes the dimensions of the uncertainty ellipsoids in the report
Risk thresholds Applies the selected risk thresholds to the results included in the report. Risk thresholds are defined on the Anti-Collision Parameters page in the Options and Settings window. See "Anti-collision parameters".
  • Separation factor – Applies the Separation factor thresholds to the results.
  • Ellipsoid separation – Applies the Ell Sep thresholds to the results.
  • Centre to centre distance – Applies the Cr-Cr thresholds to the results.
Centre to centre distance unit

Distance unit to use for center-to-center distances.

  • Site – Uses the site unit for the primary well, selected in the site properties.
  • Map – Uses the map unit for the field, selected in the field properties.
  • Preferred site unit – Uses the preferred site unit for the field, selected in the field properties.
Azimuth north reference

North reference to use for azimuth values.

  • Grid – Azimuth values will be relative to grid north.
  • North reference of primary well – Azimuth values will be relative to the north reference for the primary well, set in the site properties.
Enable Rolling AC Reports Enables rolling anti-collision reports. When this option is selected, separate anti-collision reports will be generated for each plan included in the analysis. See "Creating rolling reports".
