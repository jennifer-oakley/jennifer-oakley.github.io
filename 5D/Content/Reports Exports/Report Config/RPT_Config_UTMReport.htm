UTM reports

Related Topics
Related Topics
  • Well reports

The UTM Report page contains the settings for UTM reports. To edit the UTM report settings, click the Options and Settings button () in the main menu and go to the Reporting > UTM Report page.

When you click the Customise button in each section, you can select which columns you want to appear in the report.

List of UTM report settings
Section Field Name Description
-- Header

Includes a header with the report type and page number. For additional header options, click the Customise button.
-- Field

Includes information about the field in the report, including the map unit, vertical reference datum, projected coordinate system, and company name. To add custom parameters about the field, click the Customise button.
-- Site

Includes information about the site in the report, including the site unit, north reference, convergence angle, site position, TVD reference, and elevation above MSL.
-- Slot

Includes information about the slot in the report, including the slot position and elevation above MSL.
-- Well

Includes information about the well in the report, including the type of well, file number, UWI, plan version, closure distance, closure azimuth, vertical section, and magnetic parameters.
-- Plan Archive Includes a list of plans in the plan archive
-- Drill floor

Includes information about the drill floor in the report, including the rig height, elevation above MSL, inclination, and azimuth.
Salient points Salient points

Includes information about the salient points in the wellpath, such as the MD, inclination, azimuth, TVD, and northing/easting offsets. Salient points include anchor points, ordinary points, and targets. You can also include annotation points, casing points, formations, and sidetracks. To change the parameters included in the report, click the Customise button.
Annotation points

Includes points that contain annotations in the list of Salient Points. Only available when the Salient Points option is selected.
Casing points

Includes casing points in the list of Salient Points. Only available when the Salient Points option is selected.
Formations

Includes formation points in the list of of Salient Points. Only available when the Salient Points option is selected.
Sidetrack reported from drill floor

Includes Salient Points from the _wellpathType for the parent well, from the drill floor to the tie-on point. Only applies to sidetrack wells. Only available when the Salient Points option is selected.
Interpolated points Interpolated points

Includes information about the interpolated points in the wellpath, such as the MD, inclination, azimuth, TVD, and northing/easting offsets. Interpolated points are the calculated points between salient points. You can also include annotation points, casing points, formations, sidetracks, and survey sections. To change the parameters included in the report, click the Customise button.
Annotation points

Includes points that contain annotations in the list of Interpolated Points. Only available when the Interpolated Points option is selected.
Casing points

Includes casing points in the list of Interpolated Points. Only available when the Interpolated Points option is selected.
Formations

Includes formation points in the list of of Interpolated Points. Only available when the Interpolated Points option is selected.
Sidetrack reported from drill floor

Includes Interpolated Points from the _wellpathType for the parent well, from the drill floor to the tie-on point. Only applies to sidetrack wells. Only available when the Interpolated Points option is selected.
Survey sections

Includes survey sections in the list of interpolated points. Only available when the Interpolated Points option is selected.
Specify depth range

Filters the points included by a specified depth range. When this option is selected, you will be prompted to enter a depth range when you generate the report.
Interpolate with respect to

Specifies which depth value should be used to calculate interpolated points. Only available when the Interpolated Points option is selected.
Interpolation increment

Specifies the depth increment at which interpolated points should be calculated. Only available when the Interpolated Points option is selected.
