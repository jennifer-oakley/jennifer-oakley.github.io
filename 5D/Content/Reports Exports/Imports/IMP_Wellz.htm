About Wellz Imports

You can import wellpath data into 5D from Wellz files. The import can include either the current well plan or the definitive survey from each file. Because Wellz files do not use the site/slot/well hierarchy structure, there are restrictions on how well path data can be imported from Wellz files.

When you import well path data from Wellz files, 5D creates a slot and well relative to a specified site. The offset coordinate of the slot and well is relative to the site center, and all coordinates and azimuths are measured relative to the site North reference. You can select and import multiple Wellz files to the specified site.

Note

Before you can import well path data from Wellz files, you'll need to create the site where the well path data will be imported.

Importing Wellz files

To import well path data from Wellz files:

  1. Right-click on the site you want to import data into, then select the Import from Wellz Format option.
  2. In the 5D Wellz Decoder window that appears, go to the Site Details section and verify the site parameters where the Wellz data will be imported. This information is for reference only.
  3. In the Directory containing Wellz files field, click the Choose directory from which to load Wellz files button (). Navigate to the folder that contains the Wellz files you want to import, then click OK.
  4. In the Files list on the right, select the Wellz files you want to import.
  5. In the Offset between Site Center and well path origin fields, enter the East and North offset values to position the imported Wellz plan or survey data relative to the site center.
  6. Click the OK button to import the selected wells.

The selected wells will be imported into 5D, and the slots for the wells will be automatically created. All well distances are converted into the site's configured unit system during import. If the site already contains a slot with the same name, you prompted to replace the existing slot.

If any selected Wellz files are missing magnetic parameters or ground elevation:

  • If the magnetic parameters are not defined, you will be prompted to enter the default magnetic parameters. These parameters are applied to the current well and any subsequent wells that do not have magnetic parameters defined.
  • If ground elevation data is missing, you will be prompted to enter the default ground elevation. This elevation is applied to the current well and any subsequent wells that do not have ground elevation defined.

If a well has a different North reference than the site, it will not be imported.

Note

If any errors are found during import, they will be listed on the Errors tab. No wells will be imported until the error has been corrected or the well with the error is removed from the import list.

Field definitions

The table below defines the fields in the Wellz Import window.

Field definitions for the Wellz Import window
Field Definition
Site Details Parameters from the site where the Wellz data will be imported. This information is for reference only.
Directory containing Wellz files Location of the folder that contains the Wellz files. All Wellz files in the selected directory appear in the data panel.
Offset between Site Center and Well Path Origin East and North offset values to add to each imported Wellz plan or survey to position it relative to the site center.
