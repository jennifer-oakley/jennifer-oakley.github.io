Importing Compass 2000 files

About Compass 200 imports

If you have well data from Compass 2000, you can import the data into 5D after you create a field. Imported data includes sites, slots, wells, and definitive surveys. If the file contains data for plans, targets, casing points, formations, or other data, that data will not be imported.

When you initially begin the import, 5D will parse the import file to look for wells that can be imported. The results will appear in the Import Surveyed Well window, where you can review and update the well data. This window will appear for each well found in the import file. After you import a well, the Import Surveyed Well window will open, with the data for the next well in the import file.

By default, 5D will assume the definitive survey uses a standard MWD survey tool. However, you can change the selected survey tool before importing the well.

Warning

Before you import data from Compass, you will need to contact the Global Drilling Engineering group in Drilling Services.

Warning

If you attempt to import a Compass 2000 file that contains Russian characters in the file name, the import will fail.

Importing Compass 2000 files

To import well data from Compass 2000:

  1. In the Data List pane, right-click on the field and select the Import Wells option.
  2. In the Import Into 5D window that opens, go to the Import Format field and select the Compass 2000 option.
  3. In the Site Unit field, select the distance unit used in the import file.
  4. In the Site North Reference field, select the north reference used in the import file.
  5. Click the Browse button to select the import file.
  6. In the window that opens, navigate to the file you want to import and click Open.
  7. Click the OK button.
  8. In the Import Surveyed Well window, go to the Survey Tool field and select the survey tool to use for the definitive survey.
  9. Review and update the remaining data to be imported, then click the OK button to begin the import process.

The selected well and any sidetrack wells will be imported into 5D and added to the current field. If a well does not contain a definitive survey, a current plan and a definitive survey will be added to the well. A blank current plan will be added to the imported well. If the import file contains multiple wells, the Import Surveyed Well window will open again with the information for the next well listed in the import file.

Tip

To import all wells in the import file at once, select the Import all wells option.
