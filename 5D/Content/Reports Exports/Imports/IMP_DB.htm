About 5D imports

The Import 5D Items window allows you to import wells from another 5D database into your current field. This is useful when you need to add wells from other projects, restore wells from backup databases, or merge data shared by a co-worker.

You can import wells at different levels of the hierarchy:

  • Import all wells in a site by selecting the site
  • Import all wells in a slot by selecting the slot
  • Import individual wells from any combination of sites and slots

When you import wells, all associated data for those wells is included, such as surveys, plans, and well configurations. However, individual plans or surveys cannot be imported separately.

The import window shows the differences between the import database and your current database. New items that don't exist in your current database are marked with a New Item icon () , while existing items that have been modified are marked with a Modified Item icon (). When you select a modified item, the Differences panel at the bottom of the window shows a detailed comparison between the import version and your current version.

If the coordinate systems between your current field and the imported database differ, you'll be notified when the import file is read. If you choose to continue, the imported items will be converted to your current coordinate system before being displayed.

Note

When you select a site or slot, all wells contained within it are automatically selected for import. If you select a sidetrack well, its parent well is also automatically selected to maintain the well relationship in the imported database.

Importing wells

To import wells from a 5D database:

  1. Go to File > Import in the main menu.
  2. In the Browse window that opens, select the database you want to import and click OK.
  3. In the Import 5D Items window that opens, review any modified or new items.
  4. To view the differences for modified items, click on the modified item.
  5. Select the wells you want to import, then click the Import button.

The selected wells will be imported into the current field, and the import status will be updated in the Import Status column. If any of the selected items already exist in the current database, the existing item will be replaced with the imported item. Successfully imported items will be grayed out and cannot be selected for import again.

Using the Import 5D Items window

The buttons and options in the Import 5D Items window provide options for navigating and filtering the well hierarchy, as well as options for highlighting new and modified items.

Button and option definitions for the Import 5D Items window
Button/Option Function
Show Sites Collapses the hierarchy view to show only site-level items
Expand All Expands the full hierarchy to show all sites, slots, and wells
Search Filters the displayed items based on the text entered in the search field
No highlight Removes any highlight from import items
Highlight new items Displays new items in bold
Highlight modified items Displays existing items with changes in bold
Import Imports the selected wells into the current database
