About anti-collision analysis results

When you run an anti-collision analysis, there are several ways to view the results.

  • The proximity analysis table contains the full analysis results for all secondary wells, including separation factors, center-to center distances, and ellipsoid separation distances. See "Proximity analysis table".
  • Points of potential collision will be indicated by colored lines in the 3D Viewer window. See "3D viewer".
  • Ladder plots contain a depth profile of the distances between the primary well and each secondary well. See "Ladder plots".
  • Travelling cylinder plots graph the center-to-center distances between the primary well and each secondary well. See "Travelling cylinder plots".

Proximity analysis table

A table of the results will appear in the Anti-Collision window. The table below defines the columns that are available for anti-collision analysis results. To add columns to the table, go to the AC Options tab > Options screen, then click the Table Options button.

3D viewer

When you run an anti-collision analysis, points of potential collision will be indicated by colored lines in the 3D Viewer window. These lines are color-coded to indicate the severity of the risk.

  • Red lines indicate a high risk.
  • Yellow lines indicate a medium risk.
  • Green lines indicate a low risk.

Different line styles are also used to indicate the type of risk threshold that has violated.

  • Solid lines indicate the risk thresholds for separation factors have been violated.
  • Dotted lines indicate the risk thresholds for center-to-center distances or ellipsoid separation distances have been violated.

Ladder plots

A ladder plot is a depth profile of the distances between the primary well and each secondary well, based on center-to-center distances or separation factors.

To generate a ladder plot:

  1. Go to Anti-Collision > Generate Ladder Plot in the main menu.
  2. Select the type of ladder plot you want to create.
    • To create a ladder plot based on center-to-center distances, select the Cr-Cr option.
    • To create a ladder plot based on separation factors, select the Sep. Factor option.

The selected ladder plot will open in the Plot window. To customize the appearance of the plot, select the chart object you want to customize in the pane on the left side of the window. The customization options for the selected object will appear below the list of chart objects. See "Plotting".

Travelling cylinder plots

A traveling cylinder plot is a top view chart that graphs the center-to-center distances between the primary well and the secondary wells, with respect to the travelling cylinder reference. The travelling cylinder reference can be either the high side or North. To change the travelling cylinder reference, go to the AC Options tab in the Anti-Collision window and click the Analysis Options button. See "Anti-collision parameters".

To create a travelling cylinder plot:

  1. Go to Anti-Collision > Travelling Cylinder in the main menu.
  2. In the Well Path Selector window that appears, select the secondary wells that you want to include in the plot, then click OK. The secondary wells included in the anti-collision analysis will be selected by default.

The travelling cylinder plot will open in the Plot window. To customize the appearance of the plot, select the chart object you want to customize in the pane on the left side of the window. The customization options for the selected object will appear below the list of chart objects. See "Plotting".
