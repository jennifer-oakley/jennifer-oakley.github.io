About wellpath accessories

The Well Path Accessories window contains additional display and labeling options for the selected graph’s well path. These settings control which data is shown—such as measured depths, casing points, and targets—and how labels and annotations are drawn. There are several tabs in the Well Path Accessories window:

  • The Well Path Labels tab contains the display settings for the well name, annotations, salient points, casing points, and point-to-point measurements. See "Wellpath labels".
  • The Targets tab contains the display settings for targets. See "Targets".
  • The Error Ellipses tab contains the display settings for 3D ellipsoids and bounding ellipses. See "Error ellipses".
  • The Depth Labels tab contains the display settings for well intervals. See "Depth labels".
  • The Drilling Corridors tab contains the display settings for drilling corridors. See "Drilling corridors".

To open Well Path Accessories window, select a graph in the plot object browser and click the Well Path Accessories button in the properties panel. You can also right-click on the graph and select the Well Path Accessories option.

Wellpath labels

The table below defines the options available on the Well Path Labels tab.

Field definitions for the Well Path Labels tab
Field Description
Well Name

Adds a label at the end of the well path showing the well name. By default, well name labels are drawn parallel to the horizontal axis.

  • Draw Well Name – Adds a label with the well name to the graph.
  • Colour – Sets the text color for the well name label.
  • Text Size – Sets the text size for the well name label.
  • Perpendicular – Draws the label perpendicular to the horizontal axis.
Well Path Annotation

Displays annotations associated with the wellpath.

  • Draw Annotation – Adds the wellpath annotations to the graph.
  • Colour – Sets the text color for the annotation.
  • Text Size – Sets the text size for the annotation.
Salient Points

Labels key points along the well path.

  • Draw Points – Adds the salient points to the graph.
  • Colour – Sets the color of the salient points and the label text.
  • Text Size – Sets the text size for the salient point labels.
  • Comment – Adds a description to each salient point.
  • MD / Inc / Azi / Txt – Displays measured depth, inclination, azimuth, and comment text for each salient point.
Casing Points

Displays casing points along the well path.

  • Draw Points – Adds casing points to the graph.
  • Colour – Sets the color of the casing point indicators and labels.
  • Text Size – Sets the text size for casing point labels.
  • Crown – Sets the display size for the casing crown symbol.
  • Display – Sets the depth type for casing points (MD or TVD).
  • Line – Sets line thickness for casing point indicators.
Point-To-Point

Controls the appearance of point-to-point measurements.

  • Colour – Sets the color of the point-to-point annotation.
  • Text Size – Sets the size of the label text for point-to-point measurements.
  • Line Size – Sets the thickness of the connector line.
  • Font – Sets the font used for point-to-point labels.

Targets

The table below defines the options available on the Targets tab.

Field definitions for the Targets tab
Field Description
Draw

Displays the targets associated with the well path.

  • Colour – Sets the color of the target symbol.
  • Text – Sets the label text for each target.
  • Line – Sets the thickness of the target outline.
  • Font – Sets the font used for the label text.
Name Labels

Adds a label showing the name of each target.

  • Colour – Sets the text color for the name label.
  • Text – Sets the name text displayed in the label.
Comment Labels

Displays comments for each target.

  • Colour – Sets the text color for the comment label.
  • Text – Sets the comment text displayed in the label.
Penetration Points

Marks where well paths intersect targets.

  • Penetration Point Labels – Adds a label for each penetration point.
  • Colour – Sets the color for the symbol and label.
  • Text – Sets the text for the penetration label.
  • Cross Length – Sets the size of the cross symbol.
  • Line – Sets the line thickness of the symbol.
Aim Points / Target Uncertainty

Displays aim points and associated uncertainty regions.

  • Colour – Sets the color of the uncertainty ellipse or aim point.
  • Size – Sets the diameter of the uncertainty region.
Show Target Uncertainty Enables or disables the uncertainty display for all targets.

Error ellipses

The table below defines the options available on the Error Ellipses tab.

Field definitions for the Error Ellipses tab
Field Description
Draw Displays ellipsoids representing positional uncertainty at survey points along the well path.
Display Properties

Controls how the ellipsoids are displayed on the graph.

  • 3D Ellipsoid – Shows a full three-dimensional ellipsoid at each survey point.
  • Bounding Ellipse – Shows a two-dimensional projection of the uncertainty area.
  • Both – Displays both the 3D ellipsoid and the bounding ellipse.
3D Ellipsoid

Controls the appearance of the 3D ellipsoid.

  • Colour – Sets the color of the 3D ellipsoid outline.
  • Style – Sets the line style for the ellipsoid (solid, dashed, etc.).
  • Size – Sets a scale factor for the ellipsoid.
Bounding Ellipse Controls the appearance of the bounding ellipse.
  • Colour – Sets the color of the bounding ellipse.
  • Style – Sets the line style for the ellipse.
  • Size – Sets a scale factor for the bounding ellipse.
Draw Ellipsoid Vector Arrows Adds directional vector arrows to the ellipsoids to indicate orientation and uncertainty direction.
Points Selection

Controls which survey points will display error ellipses.

  • Start / End – Defines the starting and ending measured depths for the ellipsoids.
  • Step / Increment – Sets how often ellipsoids are drawn (e.g., every 50 ft).
  • Always show final ellipsoid – Ensures an ellipsoid is drawn at the final survey point, even if it doesn’t align with the increment setting.

Depth labels

The table below defines the options available on the Depth Labels tab.

Field definitions for the Depth Labels tab
Field Description
Labels Generate / Delete Creates or removes depth label plot objects on the graph. Use this option to generate new depth labels or remove existing ones.
Draw Displays static depth text along the well path instead of using editable label plot objects. This method is faster and lighter for dense well paths.
Label Properties

Controls the appearance of depth labels for the wellpath.

  • Colour – Sets the color of the depth label text.
  • Size – Sets the text size for the depth labels.
  • Font – Sets the font used for the label text.
Contents

Specifies which values are included in the depth labels.

  • TVD – Displays true vertical depth.
  • MD – Displays measured depth.
  • Inclination – Displays the inclination angle.
  • Azimuth – Displays the azimuth angle.
  • Include Text Prefixes – Adds prefixes such as “MD:” before the value for clarity.
Interval Type

Defines the property used to determine how labels are spaced along the well path.

  • TVD – Labels are placed at intervals of true vertical depth.
  • MD – Labels are placed at intervals of measured depth.
  • Inclination – Labels are placed at intervals of inclination.
  • Azimuth – Labels are placed at intervals of azimuth.
Interval

Specifies the range and spacing of depth labels.

  • Start / End – Sets the minimum and maximum values for the selected interval type.
  • Step / Increment – Defines the distance or value interval between labels.
  • Always show final point – Ensures a label appears at the final value, even if it does not align with the interval.
Filter for Depth Labels

Removes depth labels that fall within a defined area of the graph.

  • View / Edit Filter Rectangle – Enables a red box that defines the filtering area.
  • VS / TVD – Sets the dimensions of the filter rectangle.
  • Delete Labels in Rectangle – Deletes all depth labels within the defined rectangle.

Field definitions for the Drilling Corridors tab
Field Description
Show Drilling Corridors Displays the drilling corridors associated with the well path. Drilling corridors are visual boundaries that indicate the acceptable trajectory envelope for drilling operations.
