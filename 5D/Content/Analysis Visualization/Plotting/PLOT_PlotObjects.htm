About plot objects

You can add and customize a variety of plot objects—such as axes, grids, labels, boxes, and lines—within each graph. These objects help organize, annotate, and highlight specific aspects of the data shown in the graph.

The following objects can be added to a graph to display additional data, provide context, or highlight key areas:

  • Axis – A horizontal or vertical axis that defines the coordinate system for a graph. Axes are typically added automatically when a graph is created but can also be added manually. See "Axis settings".
  • Grid – A network of horizontal and vertical lines that correspond to the axis scales. Grids help visually align data points in the chart view. See "Grid settings".
  • Label – Text annotations that can be placed anywhere on the graph. Labels can show well names, measurement values, or annotations. See "Label settings".
  • Box – Boxes containing different types of data such as text, bitmaps, wellpath data, slot data, legend information, well schematic data, magnetic references, sign-off information, and graph creation information. See "Box settings".
  • Line – A straight line that can be used for reference, emphasis, or marking boundaries within a graph. See "Line settings".

These plot objects can only be added to graphs—not to chart objects. Many are created automatically when using shortcut tools, but you can also add and edit them manually using the Plot Object Browser and Properties Panel.

Editing plot objects

There are several ways to edit plot objects:

  • To edit a plot object, click on it the chart view or select it in the plot object browser. When selected, the plot object will be outlined with red anchor points at each corner and midpoint.
  • To move the plot object, drag it with the mouse. To lock the current selection, hold the Shift key while dragging. This makes it easier to reposition the plot object, without accidentally selecting a different object.
  • To resize a plot object, select the object and drag the anchor points as needed. Labels and boxes are automatically resized when you edit the text or the text size.
  • To edit the properties of a plot object, select the plot object in the chart view or the plot object browser. The properties for the selected plot object will appear in the properties panel.
  • To delete a plot object, select the plot object in the chart view or the plot object browser, then click the Delete button. The plot object will be removed from the graph.
Note

Axes and grids can only be selected in the plot object browser. When you click on an axis or grid in the chart view, the graph object will be selected instead.

Context menu

When you right-click on an axis, grid, label, box, or line, the following options will appear:

Context menu options for graphs
Option Description
Select Parent Graph Selects the parent graph that the plot object is associated with.
File Options

Options for loading a chart, saving the chart, and printing the chart.

  • Load Chart – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will be reset to their original position.
  • Load Chart With Fixed Labels – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will not change.
  • Save Chart – Exports the current chart and all associated settings to a text file.
  • Print – Prints the current chart to a printer on your computer.
  • Print to Bitmap – Exports the current chart image to a bitmap file.
  • Print to PDF – Exports the current chart image to a PDF file.
Detach from Graph Detaches the selected object from the parent graph. A new graph object will be created for the object. Only available for labels and boxes.
