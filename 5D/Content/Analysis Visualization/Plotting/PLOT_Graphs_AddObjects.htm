About adding objects

There are several types of objects you can add to a graph:

  • Well – Automatically generated graphs for the top view or section view of a wellpath. See "Adding a wellpath to graph".
  • Label – Text annotations that can be placed anywhere on the graph. Labels can show well names, measurement values, or annotations. See "Adding labels to a graph".
  • Box – Boxes containing different types of data such as text, bitmaps, wellpath data, slot data, legend information, well schematic data, magnetic references, sign-off information, and graph creation information. See "Adding boxes to a graph".
  • Line – A straight line that can be used for reference, emphasis, or marking boundaries within a graph. See "Adding lines to a graph".

Adding a wellpath to graph

When you add a wellpath to an existing graph, the new wellpath will be added to the graph as a secondary wellpath.

To add a wellpath to the chart:

  1. Open the Plot window from the toolbar or a context menu.
  2. Right-click on the graph you want to add a wellpath, then select the Add Well option.
  3. In the Select Well Path window, choose the well and well path you want to graph.

The chart will be generated with the appropriate axes, labels, and grids. You can adjust the layout by dragging elements in the chart view or editing properties in the left panel. You can also add optional items like labels or boxes by right-clicking in the chart and choosing the Add Label or Add Box option.

The new wellpath will be added to the existing graph. If the Multi-Colour Spider Plots option is selected in the chart properties, the wellpaths will be color-coded using the color settings for spider plots. See "Customizing color settings for spider plots".

Adding labels to a graph

To add a new label to a graph:

  1. Select the graph you want to add an object to in the plot object browser.
  2. Right-click on the graph in the chart view, then select the Add Label option.
  3. Select the new label in the plot object browser to display the label properties.
  4. Go to the text box field and enter the text for the label, then update the remaining properties as needed. See "Label settings".

The new label will appear on the graph. The label will be automatically resized to fit the label text.

Tip

Labels can contain multiple lines of text. To add a new line of text, press the Enter key inside the text box in the label properties.

Adding boxes to a graph

There are several types of boxes that you can add to a graph:

  • Normal – Basic text box with a customizable title and text
  • Well Path Data – Automatically generated box containing wellpath data, such as salient points, casing points, formation points, and targets. The data included can be customized in the Well-Path Data Box Options window, which appears when you add a Well Path Data box to the chart.
  • Slot Data – Automatically generated box containing information about the slot, such as elevation, offsets, northing, easting, latitude, and longitude.
  • Well Path Colour Legend – Automatically generated box containing a legend for the wells in the graph.
  • Well Schematic – Automatically generated box containing a schematic of the well.
  • Well Name Aliases – Customizable text box for any aliases associated with the well.
  • North Arrows – Customizable box illustrating the relationships between grid north, true north, and magnetic north. The options for this box can be customized in the North Reference Schematic section of the box properties.
  • Bitmap – Customizable box showing a bitmap image, such as a logo. The bitmap path can be set in the Bitmap section of the box properties.
  • Sign Off Box – Customizable text box for signatory purposes.
  • Information – Customizable text box showing the graph creation date and other information.

To add a new box to a graph:

  1. Select the graph you want to add an object to in the plot object browser.
  2. Right-click on the graph in the chart view.
  3. Go to the Add Box option, then select the type of box you want to add.
  4. If you selected the Well Path Data option or the Slot Data option, select the items you want to include in the window that appears.
  5. Select the new box in the plot object browser to display the box properties.
  6. Update the box properties as needed. See "Box settings".

The new box will appear on the graph. The label will be automatically resized to fit the label text.

Tip

To format the contents of the Well Path Data box as a table, select the HTML Formatting option in the Well-Path Data Box Options window.

Adding lines to a graph

To add a line to a graph:

  1. Select the graph you want to add an object to in the plot object browser.
  2. Select the Line option, then click the New button at the bottom of the plot object browser.
  3. Update the start and end points for the line in the properties panel, then update the remaining properties as needed. See "Line settings".
  4. Click the Apply button at the bottom of the plot object browser.

The new line will appear on the graph.
