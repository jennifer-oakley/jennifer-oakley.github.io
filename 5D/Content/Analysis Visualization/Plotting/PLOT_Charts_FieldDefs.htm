About chart properties

When the chart object is selected, you can update global properties that affect the entire chart layout. These properties appear in the properties panel, grouped into categories that match the layout of the Plot window.

The snap distance for the chart defines an invisible background grid used to align plot objects. The origin point of each plot object will automatically align to the nearest point on this grid as you move or position it. Snap distance is configured in the chart properties. You can adjust it to control how finely objects snap into place. A smaller snap value allows more precise placement, while a larger value enforces a more uniform layout. See "Miscellaneous settings". This behavior is applied to all plot objects except labels. Label positioning is not restricted by the snap grid, allowing for free placement anywhere on the graph.

General settings

The table below defines the general settings for the chart object.

Field definitions for the general settings of a chart object
Setting Button Description
Mouse / Key Control

Opens an information window showing mouse and keyboard shortcuts. See "Keyboard and mouse controls".
Re-sizes the Plot window to the default size.
Maximizes the Plot window.
File Operations

Options for loading a chart, saving the chart, and printing the chart.

  • Load Chart – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will be reset to their original position.
  • Load Chart With Fixed Labels – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will not change.
  • Save Chart – Exports the current chart and all associated settings to a text file.
  • Print – Prints the current chart to a printer on your computer.
  • Print to Bitmap – Exports the current chart image to a bitmap file.
  • Print to PDF – Exports the current chart image to a PDF file.
Short Cut Operations Opens the shortcut plotting menu. See "Shortcut Operations".
Multi-Colour Spider Plots (check box) When selected, wellpaths in spider plots will be drawn in different colors. Otherwise, all wellpaths use the default plan and survey colors.

Chart unit and size settings

The table below defines the settings in the Chart Unit and Chart Size sections of the chart properties.

Field definitions for the unit and size settings of a chart object
Setting Button Description
Chart Unit — Sets the unit of measurement for the chart: inches or centimeters.
Chart Size — Choose from standard paper sizes or define a custom height and width.
Landscape (check box) When selected, the width and height values are swapped to landscape orientation.
Zoom — Sets the zoom level of the chart.
Resets the zoom level to 1.0.
Draw Background Grid on Prints (check box) When selected, includes the visible background grid when printing the chart.

Border settings

The table below defines the settings in the Border section of the chart properties.

Field definitions for the border settings of a chart object
Setting Button Description
Show Border (check box) Enables a border around the chart object.
Margins Left Sets the spacing between the left chart edge and the border, in chart units.
Right Sets the spacing between the right chart edge and the border, in chart units.
Upper Sets the spacing between the top chart edge and the border, in chart units.
Lower Sets the spacing between the bottom chart edge and the border, in chart units.

Miscellaneous settings

The table below defines the settings in the Miscellaneous section of the chart properties.

Field definitions for miscellaneous settings of a chart object
Setting Button Description
Font Sets the default font for chart objects (e.g., labels, annotations).
Snap — Defines how precisely plot objects can be moved. Most objects snap to an invisible grid, except for labels. See "About chart properties".
Cross Hair at Well Path Origin (check box) Displays crosshairs at the origin of each wellpath.
Edit 5D Plot Default Values Opens the Defaults window where you can configure global Plot settings.

Colour settings

The table below defines the settings in the Colour section of the chart properties.

Field definitions for the color and size settings of a chart object
Setting Description
Border

Sets the color and thickness of the chart border. Click the Border button to set the border color.
Back

Sets the chart's background color. Click the Back button to set the background color.
Grid Sets the color and thickness of the grid lines. Click the Grid button to set the grid color.
Grid Numbers Sets the size of the numbers shown along the grid lines.
X-Hair Sets the color and thickness of the crosshairs at the wellpath origin. Click the X-Hair button to set the crosshair color.
