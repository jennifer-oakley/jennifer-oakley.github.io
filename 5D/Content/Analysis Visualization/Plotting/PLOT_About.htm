About plotting

The Plot window lets you create customized charts to display well paths, survey data, targets, and more. Whether you're working with a single well or planning an entire field, the Plot window gives you flexible tools for building clear, detailed visuals.

To open the Plot window, click the Plotting button () in the main toolbar right-click a well, plan, or survey in a the Data List pane and select the Plot option. There are two ways to create a graph in the Plot window: use one of the built-in shortcut tools, or by adding a wellpath to the chart. See "Creating a shortcut graph" and "Adding a wellpath to the chart".

When you open the Plot window, you’ll see three main areas:

  • Chart view (right) – where your graphs are displayed.See "Chart objects".
  • Plot object browser (left, top) – lets you select and manage the chart, graphs, and individual plot objects. See "Plot object browser".
  • Properties panel (left, bottom) – shows editable settings for the selected chart or graph. See "Chart properties" and "Graphs". The properties for other types of objects, such as labels, boxes, and lines, are also shown here. See "Plot objects".

A chart is the canvas where your plots live. Each chart can contain one or more graphs, such as a top view or a vertical section of a well path. When you create a graph, 5D automatically adds the standard elements you need—axes, grids, and labels—to help you interpret the data. You can also add other elements to your chart, like boxes, lines, and additional labels. Each of these elements is called a plot object. See "Plot object browser".

Plot object browser

The plot object browser helps you navigate and manage the various components of your chart. It's located on the left side of the Plot window and is divided into three levels: the chart object, graphs, and plot objects (such as axes, grids, labels, boxes, and lines).

Use the radio buttons to choose which level of the hierarchy you want to work with. For example, select the Graph option to view and edit graph-level properties, or select the Box option to work with boxes associated with the selected graph. When you choose an object type, the menu next to the object type updates to show all objects of that type in the current graph. Select an item from the list to highlight it in the chart view and display its properties in the properties panel.

The buttons at the bottom of the plot object browser allow you to create and delete plot objects, as well as create a snapshot, undo an action, and redo an action.

  • The New button creates a new graph, label, box, or line. Select the type of item you want to create, then click the New button. Update the object properties, then click Apply.
  • The Delete button deletes the selected object from the chart.
  • The Snapshot button creates a snapshot of the chart and the current settings. Snapshots are also automatically created when you perform any of the following actions: add or delete a graph or plot object, resize or move a graph, load a chart, reload all wells or information boxes, update the section view or top view properties, or edit the wellpath accessories.
  • The Undo button reverts to the most recent snapshot of the chart.
  • The Redo button reapplies the changes that were undone.

Creating a shortcut graph

There are several pre-formatted graphs with common layouts that you can add to a chart. These graphs include spider plots, plan vs. survey plots, slot view plots, ladder plots, and more. See "Shortcut Operations".

To add a shortcut graph:

  1. Right-click the chart object, then go to the Short Cut Operations option.
  2. Select the type of graph you want to create. Top view and section view graphs are available under the Quick Plots menu.
  3. Use the mouse cursor to draw a rectangle where you want the graph to appear.

The chart will be generated with the appropriate axes, labels, and grids. You can adjust the layout by dragging elements in the chart view or editing properties in the left panel. You can also add optional items like labels or boxes by right-clicking in the chart and choosing the Add Label or Add Box option.

Note

Some graphs are only available when a wellpath or site is selected. See "Selecting a site or wellpath".

Tip

Shortcut charts are auto-generated with default formatting, but you can fine-tune them afterward using the object browser and properties panel.

Adding a wellpath to the chart

When you add a wellpath to a chart, a top view or section view graph will be automatically created for the selected wellpath.

To add a wellpath to the chart:

  1. Open the Plot window from the toolbar or a context menu.
  2. Right-click in the chart view and go to the Add Well option, then select the view you want to use in the graph.
  3. In the Select Well Path window, choose the well and well path you want to graph. The graph will be created automatically and will include axes, a grid, and labels by default.

The chart will be generated with the appropriate axes, labels, and grids. You can adjust the layout by dragging elements in the chart view or editing properties in the left panel. You can also add optional items like labels or boxes by right-clicking in the chart and choosing the Add Label or Add Box option.
