About chart objects

The chart object is the canvas where your plots live. Each chart can contain one or more graphs, such as a top view or a vertical section of a well path. When you add a graph, it appears in the chart view, and any plot objects you attach to that graph are shown here as well.

There are two ways to create a graph in the Plot window: use one of the built-in shortcut tools, or by adding a wellpath to the chart. See "Adding a shortcut graph" and "Adding a wellpath to the chart".

After you’ve created a graph, you can adjust almost every part of the chart layout. You can move or resize objects, adjust projection and scaling, and change fonts, colors, or annotations to suit your needs.

To display the properties of the chart object, select the chart object in the plot object browser or click on a blank area in the chart view. When an object is selected, red anchor points appear at its corners and midpoints. See "Chart properties".

Context menu

When you right-click on the chart object, the following options will appear:

Context menu options for chart objects
Option Description
File Options

Options for loading a chart, saving the chart, and printing the chart.

  • Load Chart – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will be reset to their original position.
  • Load Chart With Fixed Labels – Opens a previously exported chart in the chart view. Any existing objects in the chart view will be deleted. If you reload the wells in the chart, label positions will not change.
  • Save Chart – Exports the current chart and all associated settings to a text file.
  • Print – Prints the current chart to a printer on your computer.
  • Print to Bitmap – Exports the current chart image to a bitmap file.
  • Print to PDF – Exports the current chart image to a PDF file.
Short Cut Operations Options for adding spider plots, quick plots, slot view plots, plan vs. survey plots, and more. See "Shortcut Operations".
Add Well Generates a graph for the top view or section view of a wellpath. See "Adding a wellpath to the chart".
Add Label Adds a text label to the chart.. See "Adding labels to a chart".
Add Box Adds a box containing text, bitmaps, sign-off information, and graph creation information to the chart. See "Adding boxes to a chart".
Re-Load All Wells With Latest Vs

Updates the wellpath data shown in the chart with the latest data.
Replace Chart Well With Another

Replaces the wellpath shown in all graph with another wellpath. Only available when there is only one graph in the chart view.
Re-Load All Information Box

Updates all information boxes, such as wellpath data boxes and slot view boxes, with the latest data.

Keyboard and mouse controls

You can use the following keyboard and mouse controls to manipulate the objects on the chart.

Keyboard and mouse controls
Field Name Definition
Left click Selects the plot object
Shift key + left click Moves the selected plot object
Left click on red anchor point Resizes the selected plot object
Ctrl key + left click Zooms the chart in the direction that the mouse is moved
Alt key + left click Pans the chart in the direction that the mouse is moved
Spin mouse wheel Changes the size of the selected axis, label, or text-based box
Right click Opens the context menu for the selected plot object
Ctrl key + C Copies a bitmap of the chart to the clipboard
Delete key Deletes the selected plot object
