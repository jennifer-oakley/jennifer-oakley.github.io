General settings

The table below defines the settings in the Graph Position, Master / Projection, and Point-to-Point sections of the graph properties.

Field definitions for the general graph settings
Setting Description
Graph Position Defines the location of the graph’s origin relative to the lower-left corner of the chart object.
  • X – Horizontal position in chart units
  • Y – Vertical position in chart units
  • Chart Unit – Read-only; set in the chart object
  • Increment – The step size used to change the graph position using the arrows in the X and Y fields.
Master / Projection Defines whether the graph is a master graph, and which projection to use.
  • Master – When selected, designates the graph as a master. Sub-graphs are prefixed with an underscore (_) and inherit properties from their master.
  • Select Master – When selected, moving or resizing the grid or axes for the graph will move and resize the entire graph. When not selected, the grid and axes can be resized independently.
  • Vertical Section – Shows the graph as a vertical plane projection.
  • Top View – Displays the graph from a top-down perspective.
Point-to-Point Opens the point-to-point measuring tool. When active, you can click two points in the chart view to measure the distance and direction between them.

Wellpath settings

The table below defines the settings in the Wellpath section of the graph properties.

Field definitions for the wellpath graph settings
Setting Description
Load Opens the Select Well Path dialog. This is required before plotting any well path data.
Clear Removes the currently loaded well path from the graph.
Auto-Axes and Grid When checked, automatically adds axes and a grid to the graph when a well path is loaded or the scale is updated. These objects are resized automatically unless edited manually.

Data range settings

The table below defines the settings in the Data Range section of the graph properties.

Field definitions for the data range graph settings
Setting Description
E Minimum and maximum extents of the graph in the Easting direction. Only available for top view projections.
N Minimum and maximum extents of the graph in the Northing direction. Only available for top view projections.
VS Minimum and maximum range for the vertical section. Only available for section view projections.
TVD Minimum and maximum range for TVD. Only available for section view projections.
Increment The minimum step size used to edit data range extents.
Site Unit Read-only; inherited from the site properties.

Scale / data range settings

The table below defines the settings in the Scale / Data Range section of the graph properties. Click the Set button () to apply optimization changes.

Field definitions for the scale and optimization graph settings
Setting Description
Optimise scale for full data range Calculates a round-number scale value that fills the current graph bounds with all well path data visible. When the Excluding Lease Lines option is selected, lease lines will be excluded from the optimization calculation.
Optimise edited DR (Const scale / resize) Keeps the current scale and adjusts the data range to center the axes at (0,0).
No Optimisation Disables optimization. Use this option when applying extreme scale factors or custom ranges.
X / Y Editable fields for setting the graph scale in feet or meters per chart unit (e.g., ft/in).
Note

When optimization is disabled, the Set button will have no effect.

Travelling cylinder settings

The table below defines the settings in the Travelling Cylinder section of the graph properties. These options are only available for travelling cylinder plots. See "Well plots".

Field definitions for the travelling cylinder graph settings
Setting Description
Range Sets the radius or extent of the travelling cylinder used in anti-collision analysis.
Grid Int. Grid interval setting for the cylinder visualization.
Ellipse Enables display of a 2D ellipse representing the traveling cylinder's cross-section.

Layering settings

The table below defines the settings in the Opaque Background section of the graph properties.

Field definitions for the background and layering graph settings
Setting Description
Opaque Background When selected, fills the graph background with a solid color to mask overlapping content.
Border Width Sets how far the opaque fill extends beyond the graph border.

Inheritance and miscellaneous settings

The table below defines inheritance and other miscellaneous settings for a graph.

Field definitions for inheritance and other miscellaneous graph settings
Setting Description
Edit Independently When checked on a sub-graph, allows it to override properties inherited from its master. When checked on a master graph, property changes apply only to that graph, not to its sub-graphs.
Well Path Accessories Opens the Well Path Accessorieswindow for configuring wellpath labels, target labels, error ellipses, depth labels, and drilling corridors. See "Wellpath Accessories".
Section Properties Opens the Vertical Section window, which allows you to configure depth labels, formation labels, and bearing/TVD exaggerations. Only available for section view projections. See "Section Properties".
Top View Properties Opens the Top View window, which allows you to configure depth labels, slot labels, and lease lines labels. Only available for top view projections. See "Top View Properties".
Point Sets the color and thickness of salient points shown on the graph. Click the Point button to set the point color.
Survey Point Sets the thickness of survey points shown on the graph. Survey points are shown in the same color as salient points.
Line Sets the color and thickness of the line representing the wellpath. Click the Line button to set the line color.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values
