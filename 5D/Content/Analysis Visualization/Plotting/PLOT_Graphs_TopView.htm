About top view properties

The Top View window contains settings for configuring how top-down views of the well path are displayed in a graph. These settings include options for adding labels, displaying slot view data, and customizing lease lines. This window is only available when the selected graph is set to show the top view of the wellpath.

To open this window, select a graph with Top View selected, then click the Section Properties button in the properties panel. You can also right-click on the graph and select the Top View Properties option.

Depth labels

The table below defines the options available on the Depth Labels tab.

Field definitions for the Depth Labels tab
Field Description
Labels Generate / Delete Creates or removes depth labels along the well path in the top view.
Draw Displays static depth values as fixed text on the graph, instead of creating label plot objects.
Label Properties

Controls the appearance of depth labels for the top view.

  • Colour – Sets the color of the depth label text.
  • Size – Sets the text size for the depth labels.
  • Font – Sets the font used for the label text.
Contents

Specifies which values are included in the depth labels.

  • TVD – Displays true vertical depth.
  • MD – Displays measured depth.
  • Inclination – Displays the inclination angle.
  • Azimuth – Displays the azimuth angle.
  • Include Text Prefixes – Adds prefixes such as “MD:” before the value for clarity.
Interval Type

Defines the property used to determine how labels are spaced along the well path.

  • TVD – Labels are placed at intervals of true vertical depth.
  • MD – Labels are placed at intervals of measured depth.
  • Inclination – Labels are placed at intervals of inclination.
  • Azimuth – Labels are placed at intervals of azimuth.
Interval

Specifies the range and spacing of depth labels.

  • Start / End – Sets the minimum and maximum values for the selected interval type.
  • Step / Increment – Defines the distance or value interval between labels.
  • Always show final point – Ensures a label appears at the final value, even if it does not align with the interval.

Slot view

The table below defines the options available on the Slot View tab.

Field definitions for the Slot View tab
Field Description
Slot Labels Adds labels for each slot in the slot view, showing the slot name at the corresponding location.
Data Range Limits the graph’s data range to slot positions, excluding well path length. This is useful for displaying a dense slot arrangement.
Well Paths Toggles whether to display well paths extending from each slot.
Open Hole Diameter Controls appearance of the open hole indicators.
  • Colour – Sets the color of the open hole symbol.
  • Text Size – Sets the size of any associated text.

Lease lines

The table below defines the options available on the Lease Lines tab.

Field definitions for the Lease Lines tab
Field Description
Show all lease lines Displays all lease lines available for the site in the top view.
Edit lease line Enables editing of a specific lease line’s location and properties.
Display Details Customizes the appearance of lease lines.
  • Draw – Enables or disables drawing of the lease line.
  • Colour – Sets the line color.
  • Size – Sets the line thickness.
  • Style – Sets the line style (solid, dashed, etc.).
