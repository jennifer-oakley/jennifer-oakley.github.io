About shortcut operations

The shortcut operations in the Plot window let you quickly create common chart layouts with just a few clicks. These operations automatically generate graphs with default formatting—axes, grids, and labels included—so you can get started right away and make adjustments as needed.

  • Use Quick Plot for a simple chart of a single wellpath.
  • Use Spider Plot or Field Spider Plot to compare wells across a site or field.
  • Use Slot View Plot to show well placement within a lease or pad.
  • Use Ladder Plot and AC Spider Plot to review anti-collision spacing between wellpaths.
  • Use Plan vs Survey to spot deviations between planned and actual wellpaths.
  • Use Tortuosity and DDI to evaluate wellbore curvature or directional performance.

To access the shortcut operations, right-click on the chart object, then go to the Short Cut Operations option. You can also click the Short Cut Operations button () in the properties panel. There are several options available in the Short Cut Operations menu:

Options in the Short Cut Operations menu
Option Description
Header / Footer Adds a text header or footer to the chart. See "Adding a header or footer".
Select Well Sets the active wellpath for quick plots, plan vs. survey plots, tortuosity and DDI plots, and anti-collision plots. See "Selecting a site or wellpath".
Quick Plot Generates a basic layout for the selected wellpath, showing either the top view, the section view or a combined view. Only available when a wellpath is selected. See "Adding a shortcut graph".
Plan vs Survey Overlays the planned and actual wellpaths for comparison. Only available when a wellpath is selected. See "Adding a shortcut graph".
Multi-Colour? Applies multi-color styling to wellpaths (affects new charts only). See "Customizing color settings for spider plots".
Edit Colours Opens a dialog for customizing color settings (applies to new charts only). See "Customizing color settings for spider plots".
Select Site Sets the active site for spider, slot, or field spider plots. See "Selecting a site or wellpath".
Spider Plot Creates a top view of all the wells in the selected site. Only available when a site is selected. See "Adding a shortcut graph".
Slot View Plot Displays a top-down view of wells in a slot or lease layout. Only available when a site is selected. See "Adding a shortcut graph".
Field Spider Plot Creates a top view of all the wells in the field. See "Adding a shortcut graph".
Tortuosity and DDI Displays a well’s tortuosity and directional difficulty index (DDI) along its path. Only available when a wellpath is selected. See "Adding a shortcut graph".
Ladder Plot Displays separation factors or center-to-center distances for the secondary wellpaths included in an anti-collision analysis. See "Adding a shortcut graph".
AC Spider Plot Charts secondary wellpaths from an anti-collision analysis relative to the primary wellpath, showing either the top view or the section view. See "Adding a shortcut graph".

Selecting a site or wellpath

Some graphs are only available when a wellpath or site is selected. If you opened the Plot window by right-clicking a well in a data list, the site and wellpath will already be selected. If you clicked the Plotting button () in the main toolbar, you'll need to select a well or site manually.

Selecting a site

To select a site:

  1. Right-click the chart object, then go to the Short Cut Operations option.
  2. Select the Select Site option.
  3. In the Select Site for Spider Plot window, select the name of the site you want use in the spider plot.
  4. Select the wellpath you want to use in the plot, then click OK.

A confirmation message will appear, and the site will be designated as the active site. When you create a spider plot or slot view plot, the selected site will be used.

Selecting a wellpath

To select a wellpath:

  1. Right-click the chart object, then go to the Short Cut Operations option.
  2. Select the Select Well option.
  3. In the Select Well Path window, select the site, slot, and well that you want to create graphs for.
  4. Select the wellpath you want to use in the graph, then click OK.

A confirmation message will appear, and the well will be designated as the active well. When you create quick plot, plan vs. survey graph, or tortuosity and DDI graph, the selected wellpath will be used.

Adding a shortcut graph

To add a shortcut graph:

  1. Right-click the chart object, then go to the Short Cut Operations option.
  2. Select the type of graph you want to create. Top view and section view graphs are available under the Quick Plots menu.
  3. Use the mouse cursor to draw a rectangle where you want the graph to appear.

The chart will be generated with the appropriate axes, labels, and grids. You can adjust the layout by dragging elements in the chart view or editing properties in the left panel. You can also add optional items like labels or boxes by right-clicking in the chart and choosing the Add Label or Add Box option.

Note

Some graphs are only available when a wellpath or site is selected. See "Selecting a site or wellpath".

Tip

Shortcut charts are auto-generated with default formatting, but you can fine-tune them afterward using the object browser and properties panel.

Adding a header or footer

To add a header or footer to a chart:

  1. Right-click the chart object, then go to the Short Cut Operations option.
  2. Go to the Header / Footer option, then select the type of item you want to add to the chart.
  3. Go to the properties panel to change the text or alignment of the components in the header or footer.

A default header or footer will be added to the chart. To edit a component in the header or footer, click on the item in the chart. The properties of the selected item will be displayed in the properties panel.

Customizing color settings for spider plots

You can enable or disable multi-color spider plots in the settings for the chart object.

  • When the Multi-Colour Spider Plots option is enabled, each wellpath in a spider plot will be shown in a different color. You can customize the color settings in the Well Path Colors window.
  • When the Multi-Colour Spider Plots option is disabled, all plans will be shown in blue, and all surveys will be shown in red.

You can also customize the color settings in the Well Path Colours window. To open the Well Path Colours window, right-click on the chart object, then go to Short Cut Operations and select the Edit Colours option.

  • To insert a new color above an existing color, select the existing color and click the Insert button. Select the new color in the Select Color window, then click OK.
  • To add a new color to the end of the list, click the Append button. Select the new color in the Select Color window, then click OK.
  • To change an existing color, select the existing color and click the Edit button. Select the new color in the Select Color window, then click OK.
  • To remove a color from the list, select the color you want to remove and click the Delete button.
  • To reset the color settings to the defaults, click the Default button.

When you have finished editing the color settings, click OK in the Well Path Colours window. The new color settings will be applied when you create a new spider plot.

Note

Changes to color settings do not affect existing charts. You must apply the color changes before adding the chart.
