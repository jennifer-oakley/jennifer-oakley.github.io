Axis settings

The table below defines the fields shown in the properties panel when an axis is selected.

Field definitions for axes
Section Description
Axis Position

Sets the horizontal and vertical origin of the axis.
Orientation Sets the axis orientation, either horizontal or vertical
Disposition Sets the position of the axis
Length Specifies the length of the axis in chart units.
Ticks Specifies the increment and length for major and minor tick marks along the axis
Annotation
  • Label – When selected, adds a label to the axis. The corresponding text box controls the text shown in the axis label.
  • Numbers – When selected, adds numbers to the axis at the major tick marks.
Axis Sets the color and line thickness for the axis. Click the Axis button to set the axis color.
Number Sets the color and text size for the numbers on the axis. Click the Number button to set the number color.
Label Sets the color and text size for the axis label. Click the Label button to set the number color.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values

Grid settings

The table below defines the fields shown in the properties panel when a grid is selected.

Field definitions for grids
Field Description
Grid Position

Sets the horizontal and vertical origin of the grid.
Chart Unit

Measurement unit used for the grid. This value is read only, inherited from the chart properties. See "Chart unit and size settings".
Size

Sets the width and height of the grid
Interval Sets the increment for the major and minor grid marks
Coarse Sets the color and line thickness for the major grid marks. Click the Coarse button to set the mark color.
Fine Sets the color and line thickness for the minor grid marks. Click the Fine button to set the mark color.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values

Label settings

The table below defines the fields shown in the properties panel when a label is selected.

Field definitions for labels
Section Description
Label Position

Sets the horizontal and vertical position of the labell, relative to the graph origin. The Increment field sets the increment at which the X and Y positions can be adjusted.
Line

When selected, adds a line to the label.

  • Arrow – When selected, adds an arrow to the line. If the Line option is not selected, the arrow will not be added to the label.
  • Del X – Sets the horizontal position of the end point of the line, relative the graph origin.
  • Del Y – Sets the vertical position of the end point of the line, relative the graph origin.
Chart Unit

Measurement unit used for the label. This value is read only, inherited from the chart properties. See "Chart unit and size settings".
(text box) Text that will be shown in the label
Justify Text Justification for the text in the label
Orientation Angle at which the label will be oriented. When the Impose On Line option is selected, any line added to the label will be oriented at the same angle as the label
Do Not Move Label With Graph

When selected, the label will not be moved with the graph.
Box When selected, adds a box with a background and border around the label
Back

Sets the label's background color. Click the Back button to set the background color.
Border

Sets the color and thickness of the label border. Click the Border button to set the border color.
Text Sets the color and text size for the label text. Click the Text button to set the text color.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values

Box settings

The table below defines the fields shown in the properties panel when a box is selected.

Field definitions for boxes
Section Description
Box Position

Sets the horizontal and vertical position of the boxl, relative to the graph origin. The Increment field sets the increment at which the X and Y positions can be adjusted.
Chart Unit

Measurement unit used for the box. This value is read only, inherited from the chart properties. See "Chart unit and size settings".
Type

Type of box.

  • Normal – Basic text box with a customizable title and text
  • Well Path Data – Automatically generated box containing wellpath data, such as salient points, casing points, formation points, and targets. The data included can be customized in the Well-Path Data Box Options window, which appears when you add a Well Path Data box to the chart.
  • Slot Data – Automatically generated box containing information about the slot, such as elevation, offsets, northing, easting, latitude, and longitude.
  • Well Path Colour Legend – Automatically generated box containing a legend for the wells in the graph.
  • Well Schematic – Automatically generated box containing a schematic of the well.
  • Well Name Aliases – Customizable text box for any aliases associated with the well.
  • North Arrows – Customizable box illustrating the relationships between grid north, true north, and magnetic north. The options for this box can be customized in the North Reference Schematic section of the box properties.
  • Bitmap – Customizable box showing a bitmap image, such as a logo. The bitmap path can be set in the Bitmap section of the box properties.
  • Sign Off Box – Customizable text box for signatory purposes.
  • Information – Customizable text box showing the graph creation date and other information.
Text Text that will be shown in the box. Click the Editor button to edit the text. Not available for legend, well schematic, north arrows, and bitmap boxes.
North Reference Schematic Sets the north references that will appear in the box. Only available for north arrows boxes.
Font Sets the font for the text in the box. Click the Font button () to change the font.
Bitmap File path for the image attached to the box. Only available for bitmap boxes.
Do Move Box With Graph

When selected, the box will not be moved with the graph.
Box When selected, adds a box with a background and border around the label
Back

Sets the label's background color. Click the Back button to set the background color.
Border

Sets the color and thickness of the label border. Click the Border button to set the border color.
Text Sets the color and text size for the label text. Click the Text button to set the text color.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values

Line settings

The table below defines the fields shown in the properties panel when a line is selected.

Field definitions for lines
Field Description
Line Position

Defines the start and end points of the line.

  • Chart Unit – Measurement unit used for the line. This value is read only, inherited from the chart properties. See "Chart unit and size settings".
  • Start – Sets the horizontal and vertical position of the start point.
  • End – Sets the horizontal and vertical position of the end point.
  • Increment – Sets the increment at which the X and Y positions can be adjusted.
Line Sets the color and thickness of the line. Click the Line button to set the line color.
Arrow Adds an arrow to the line's end point.
Defaults

– Sets the default color and size values to the current values

– Overrides the current color and size values with the default values
