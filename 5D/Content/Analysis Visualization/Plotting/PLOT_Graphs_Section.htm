About section properties

The Vertical Section window contains settings for configuring how section views are displayed in a graph. These settings control aspects such as depth label appearance, formation markers, bearing angle, and vertical exaggeration. This window is only available when the selected graph is set to show the vertical section.

To open this window, select a graph with Vertical Section view, then click the Section Properties button in the properties panel. You can also right-click on the graph and select the Section Properties option.

Depth labels

The table below defines the options available on the Depth Labels tab.

Field definitions for the Depth Labels tab
Field Description
Labels Generate / Delete Creates or removes depth labels along the vertical section.
Draw Displays depth values as static text instead of using label plot objects. This option may be used to preview formatting before generating label objects.
Label Properties

Controls the appearance of depth labels for the vertical section.

  • Colour – Sets the color of the depth label text.
  • Size – Sets the text size for the depth labels.
  • Font – Sets the font used for the label text.
Contents

Specifies which values are included in the depth labels.

  • TVD – Displays true vertical depth.
  • MD – Displays measured depth.
  • Inclination – Displays the inclination angle.
  • Azimuth – Displays the azimuth angle.
  • Include Text Prefixes – Adds prefixes such as “MD:” before the value for clarity.
Interval Type

Defines the property used to determine how labels are spaced along the well path.

  • TVD – Labels are placed at intervals of true vertical depth.
  • MD – Labels are placed at intervals of measured depth.
  • Inclination – Labels are placed at intervals of inclination.
  • Azimuth – Labels are placed at intervals of azimuth.
Interval

Specifies the range and spacing of depth labels.

  • Start / End – Sets the minimum and maximum values for the selected interval type.
  • Step / Increment – Defines the distance or value interval between labels.
  • Always show final point – Ensures a label appears at the final value, even if it does not align with the interval.

Formations

The table below defines the options available on the Formations tab.

Field definitions for the Formations tab
Field Description
Generate / Delete Labels Creates or removes formation point labels on the graph.
Colour Sets the text color for the formation point labels.
Size Controls the display size of the formation label text and lines.
  • Text – Sets the font size of the label.
  • Line – Sets the thickness of the line indicating the formation point.
Offset angle Sets the angle at which the label is offset from the formation point.
Offset grid unit Defines the spacing used for the offset angle, in chart units.

Bearing / TVD exaggeration

The table below defines the options available on the Bearing/TVD Exaggeration tab.

Field definitions for the Bearing/TVD Exaggeration tab
Field Description
Bearing Defines the orientation of the vertical section projection.
  • Free Edit – Allows manual entry of the bearing.
  • 5D Value – Uses the default bearing calculated by 5D.
  • Total Depth – Uses the bearing from the start to the end of the well path.
  • 1st Target – Uses the bearing from the start of the well path to the first target.
TVD Vertical Exaggeration Sets a vertical exaggeration factor to visually stretch the depth dimension. This can make small vertical changes easier to see in the section view.
