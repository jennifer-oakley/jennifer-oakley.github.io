About surfaces

The Surfaces feature allows you to view three-dimensional surfaces in the 3D Viewer window, using imported grid data. Grid data can imported from two different file formats:

  • CSV – CSV import files must contain three columns, in the following order: easting coordinates, northing coordinates, and TVD.
  • ZMAP+ – ZMAP+ is a file format that stores grid data in an ASCII line format. ZMAP+ data can be exported from many oil and gas applications.

After you import the data, it will be graphed onto a color map in the 3D Viewer window. The color mapping is based on the TVD of each coordinate. By default, the TVD scale used to create the color map is the minimum and maximum TVD found in the imported data.

Note

The Surfaces button () is only visible when surfaces have been enabled in the default application settings. See "Application Settings".

Importing surface data

To import surface data:

  1. Click the Surfaces button () in the 3D Viewer window.
  2. In the panel that opens, click Add.
  3. Navigate to the file you want to import, then click Open.

The imported data will be shown in the Surfaces panel. After the data has been imported, you can control the appearance and visibility of each imported surface. See "Customizing surfaces".

Note

If you are connected to the central database, you must lock the field before adding or deleting surface data.

Customizing surfaces

There are several ways to customize the appearance of the surfaces shown in the Surfaces panel.

  • If the imported surface data uses elevation for the depth values, select the Depth is Elevation option.
  • To change the transparency of a surface, select a surface and drag the Transparency slider bar to the desired transparency level.
  • To change the TVD scale used for the color map, go to the Colour Map section and enter new minimum and maximum TVD values. The minimum TVD is the first value shown, and the maximum is the second value shown. By default, these values are set to the minimum and maximum TVD found in the imported data.
  • To reset the TVD scale used for the color map, go to the Colour Map section and click the Reset Scale to Data button.
  • To change the colors used in the color map, go to the Colour Map section, click on the color you want to change, then select a new color.
  • To hide a surface from the 3D Viewer window, uncheck the Visibility option next to the name of the surface.
  • To hide all surfaces from the 3D Viewer window, uncheck the Show Surfaces option at the top of the Surfaces panel.
  • To show intersections with the wellpath, select the Show Intersections option at the top of the Surfaces panel. An icon will appear at the points where the surface intersects with a wellpath. The icon will be the same color as the intersecting wellpath.
