Using the 3D Viewer

The toolbar in the 3D Viewer window is used to customize how the data is visualized.

Toolbar buttons in the 3D Viewer window
Button Function

Controls the display of casing and survey points.

  • Turn On/Off – Toggles the display of all salient points.
  • Show Survey Points of Selected Well – Toggles the display of survey points.

Controls the display of wellpaths.

  • Show Thickness – Shows the thickness defined by casing points
  • Display Error Envelope – Displays the uncertainty envelopes around the selected primary and secondary wells. Only available when anti-collision is enabled.
  • Display Error Mesh – Shows the wireframe mesh that the uncertainty envelope is constructed from. Only available when anti-collision is enabled.
  • Display Error Ellipses – Shows the ellipses that the uncertainty envelope is constructed from. Only available when anti-collision is enabled.
  • Restrict TVD Range – Set a TVD range within which all objects are displayed. Everything outside this range is removed. The min/max TVD range is specified using edit boxes at the bottom of the 3D viewer. This function is particularly useful in the top view when checking the anti-collision of well paths near the site.
  • Show Current Plan – Displays the current plan
  • Show Definitive Survey – Displays the definitive survey.
  • Shows Formations – Displays any formations for the well.
  • Show Annotations – Displays any annotations.
  • Show Casing Points – Displays the casing points.
  • Show Drilling Corridor –

Controls the display of the boundaries for the field.

  • Turn On/Off – Toggles the display of the field boundaries.
  • Top Boundary – Displays the top boundary of the field.
  • Bottom Boundary – Displays the bottom boundary of the field.
  • Bounding Box – Displays a cube, which represents all four field boundaries.

Controls the display of site labels and symbols

  • Site Labels – Displays a label to indicate the location of the site.
  • Site Symbols – Displays a symbol to indicate the location of the site.
  • Turn On/Off – Toggles the display of site labels and symbols.

Displays a set of three to represent northing, easting, and TVD.

  • The blue line points toward northing.
  • The green line points toward easting.
  • The red line points toward TVD.

Controls the current camera view for the 3D Viewer window. See "Camera views".

  • Walk View – Displays the field as though you were walking through a 3D model.
  • Top View – Displays the field from above in plan view.
  • Side View – Displays the whole field from the side in two dimensions.
  • All View – Displays the Walk View, Top View, and Side View simultaneously.
  • 2D Profile Plot – Generates two-dimensional plots for the top view and the side view of the selected well.
  • Save Camera Position – Saves the current camera position for the current view.
  • Restore Camera Position – Resets the camera position to the saved camera position for the current view.
Resets the current view to the default view

Allows you to change the center point of the field and rotate the field around the center point.

  • To change the center point of the field, click on the desired center point.
  • To rotate the field around the center point, use the left and right arrow keys to rotate the view.

Controls the visibility of error ellipsoids

  • Turn On / Off Ellipsoids Mode –
  • Show Bounding Box Along Hole –
  • Show Horizontal Bounding Box –
Controls the visibility of the measuring tool and allows you to change the settings used to display the measuring tool in the 3D Viewer window
Enables object identification in the 3D Viewer window. To identify an object, hover the mouse over the object you want to identify.

Controls the wellpaths shown in the 3D Viewer window.

  • Show All – Shows all of the wellpaths in the field.
  • Show Selected – Shows the wellpaths for well selected in the Data List pane.
  • Show AC Well Paths – Shows the primary and secondary wellpaths selected for anti-collision analysis.
  • Multiple Well Selection – Allows you to select the wells you want to view in the Data List pane. After you select the desired well, click the View selected option to view the selected well.
Controls the visibility of targets, aim points, and target uncertainty in the 3D Viewer window.
Controls the visibility of lease lines and allows you to change the settings used to display lease lines in the 3D Viewer window.

Controls the visibility of surfaces in the 3D Viewer window. See "Surfaces".

  • Show Surfaces – Toggles the display of surfaces.
  • Show Intersections – Shows an icon at the points where the surface intersects with a wellpath.
