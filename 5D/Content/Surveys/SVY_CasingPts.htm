About casing points

Casing points mark the depths where you stop drilling a particular hole diameter so you can run and cement casing. Setting the right casing points is a critical part of wellbore design – they determine your hole sections and casing program throughout the well.

You can set casing points at predetermined depths based on your well design, or choose them based on geological conditions you encounter while drilling. Each casing point represents the bottom (setting depth) of a casing string, and the wellpath parameters are automatically calculated at each point. Until you set casing points, 5D assumes the entire wellbore is open hole. Once you add casing points, they help define the hole sections and casing strings for your well construction program.

Casing points are displayed in a separate view within the data table, showing the measured depth, inclination, azimuth, coordinates, and casing details for each point. You can add, edit, or delete casing points as your well design evolves. To view the existing casing point, click the Show Casing Points button () on the right side of the data table.

Tip

You can also set the casing points for the current plan. See "Casing Points".

Adding casing points

To add a casing point:

  1. In the Data List pane, go to the List Wells tab and select the well that you want to add casing points for.
  2. Go to the List Surveys pane. The data table will automatically open at the bottom of the screen.
  3. If the Casing Points pane is not visible, click the Show Casing Points button () on the right side of the data table.
  4. Right-click on the data grid and select the Add option.
  5. In the Add Casing Point window, enter the depth at the bottom of the casing section, then select the type of depth entered.
  6. In the Casing field, select the outer diameter of the casing section.
  7. In the Name field, enter a description of the casing section.
  8. Click OK to add the casing point.

The casing section will be added to the data grid. The inclination, azimuth, north offset, and east offset of the casing point will be automatically calculated. If you entered the MD of the casing point, the TVD will also be calculated. If you entered the TVD, the MD will be calculated.

Tip

If the casing size you need isn't listed in the Casing field, you can add a custom casing size in the Casing List window. See "Casing list".

Field definitions

The table below defines the columns that are shown in the Data Table pane, when you are viewing the Casing Points pane. To view the Casing Points pane, click the Show Casing Points button () on the right side of the data table.

Column definitions for the Casing Points pane
Column Name

Description
MD

Measured depth at the bottom of the casing section, also known as the setting depth. If you entered the TVD of casing point, this value is automatically calculated, based on the definitive survey.
Inclination

Inclination at the bottom of casing section. This value is automatically calculated, based on the definitive survey.
Azimuth

Azimuth at the bottom of the casing section. This value is automatically calculated, based on the definitive survey.
TVD

True vertical depth at the bottom of the casing section. If you entered the MD of casing point, this value is automatically calculated, based on the definitive survey.
North Offset

North offset at the bottom of the casing section. This value is automatically calculated, based on the definitive survey.
East Offset

East offset at the bottom of the casing section.
Casing Name Description of the casing section

The table below defines the fields in the Add Casing Point window and the Edit Casing Point window.

Fields in the Add Casing Point and the Edit Casing Point windows
Field Name Description
Depth

Setting depth at which the casing section ends and the type of depth entered.

  • TVD – Depth was entered using the TVD of the bottom of the casing section.
  • Delta TVD – Depth was entered using the difference in TVD between the original depth and the new depth. Use a positive value if the setting depth has increased, and use a negative value if the setting depth has decreased. Only available when editing a casing point.
  • MD – Depth was entered using the MD of the bottom of the casing section.
  • Delta MD – Depth was entered using the difference in MD between the original depth and the new depth. Use a positive value if the setting depth has increased, and use a negative value if the setting depth has decreased. Only available when editing a casing point.
Casing Outside diameter of the casing
Name Description of the casing section
