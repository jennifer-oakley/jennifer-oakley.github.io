About the survey tool picker

When you add or edit a survey, you can use the Survey Tool Picker window to select the survey tool. See "Selecting a survey tool". To open the Survey Tool Picker window, open the Survey Editor window and click the Browse button (...) next to the Tool Used field.

The Survey Tool Picker window makes it easier to locate the correct survey tool, by allowing you to filter the list of survey tools. See "Filtering the list of survey tools". The Survey Tool Picker window also allows you to create custom tool sets, view source information, and view detailed information about each tool.

In addition to the filters at the top of the window, there are three tabs in the Survey Tool Picker window:

  • On Tool Details tab, you can view general information about the selected tool, such as the platform, classification, tool type, reference field, drillstring correction, SAG correction, set, and revision.
  • On the Sources tab, you can view information about the error terms associated with the selected survey tool, the magnitude of each error term, and the measurement unit used to define the error term.
  • On the Edit Tool Sets tab, you can create custom tool sets for frequently used tools. When a tool set is selected in the Active Tool Set field, the tools in that tool set will appear at the top of the list on the Tool Details tab and the Sources tab. See "Adding tool sets".

Selecting a survey tool

To select a survey tool in the Survey Tool Picker window:

  1. From the Survey Editor window, click the ... button next to the Tool Used field to open the Survey Tool Picker window.
  2. Use the filters at the top of the window to refine the list of survey tools. See "Filtering the list of survey tools".
  3. Click the Use Selected button at the bottom of the window.

The Survey Tool Picker window will close, and the tool used for the survey will be updated to the selected tool.

Filtering the list of survey tools

Each tab in the Survey Tool Picker window contains a list of the survey tools in 5D. The filters at the top of the window are used to filter the tools shown in the list. The selected filters will be applied to all of the tabs in the Survey Tool Picker window. Available filters include name, tool type, reference field, drillstring correction, platform, SAG correction, set, company, and revision.

When you select a tool set in the Active Tool Set field, the tools in that tool set will appear at the top of the list. Any filters selected will also be applied to the tools in the active tool set. To create a tool set, go to the Edit Tool Sets tab. See "Adding tool sets".

Deprecated tools are automatically excluded from the list in the Survey Tool Picker window. To include deprecated tools in your results, select the Show Deprecated Tools option at the top of the window.

Adding tool sets

To add a custom tool set:

  1. Go to the Edit Tool Sets tab in the Survey Tool Picker window.
  2. Click the New button at the bottom of the window.
  3. Enter a name for the tool set in the window that appears, then click OK.
  4. Select the tool you want to add from the list in the right side of the window.
  5. Click the Add -> button to add the tool to the tool set.
  6. When you have finished adding tools, click the Save button at the bottom of the window.

The new tool set will be saved to your computer and can be used from any 5D database. When you select the new tool set in the Active Tool Set field at the top of the window, the tools that you added to the tool set will appear at the top of the list of survey tools. To edit the tool set, go to the Edit Tool Sets tab and select the tool set in the Edit Tool Set field.
