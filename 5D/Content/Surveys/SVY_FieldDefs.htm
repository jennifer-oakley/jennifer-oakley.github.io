Survey properties

The table below defines the properties in the Survey Editor - Edit window. This window appears when you create a new survey or edit the survey properties. To edit the properties of a survey, select the survey you want to edit on the List Surveys tab in the Data List pane, then go to Plan / Surveys > Surveys in the main menu and select the Edit Survey option.

Property definitions for the Survey Editor - Edit window
Section Property Name Definition
Survey Properties Name Name of the survey
Data Acquired Date the survey was taken
Company Name of the company that took the survey
Tool Used Survey tool used to take the survey readings. This information is used to calculate the results for anti-collision analysis. See "Anti-Collision Analysis".
Job Type Type of directional tool used to drill the well
Comment Any comment relevant to the survey
Survey Data / Drill Floor Well TVD Reference Elevation point used to define the TVD of the well
Relative To

Vertical reference datum that the well TVD is relative to

  • Mean Sea Level – Elevation is relative to mean sea level.
  • Site TVD Reference – Elevation is relative to the TVD reference defined in the site properties.
  • Slot TVD Reference – Elevation is relative to the TVD reference defined in the slot properties.
  • Well TVD Reference – Elevation is relative to the TVD reference defined in the well properties, either the drill floor or the kelly bushing.
Elevation Elevation of the well, relative to the selected vertical reference datum
Rig Height Elevation of the drill floor or the kelly bushing, relative to the elevation of the slot. This value is calculated automatically, based on the value entered in the Elevation field.
Inc Inclination at the drill floor or the kelly bushing
Az Azimuth at the drill floor or the kelly bushing
Survey Data / Tie point MD Measured depth of the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
Inc Inclination at the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
Az Azimuth at the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
TVD TVD at the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
North Offset Offset from northing at the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
East Offset Offset from easting at the tie-in point. If you tie the survey to the definitive survey, this value will be automatically populated.
Survey Data MD Measured depth of the survey point
Inclination Inclination at the survey point's MD
Azimuth Azimuth at the survey point's MD
TVD TVD at the survey point's MD. This value is calculated automatically.
North Offset Offset from northing at the survey point's MD. This value is calculated automatically.
East Offset Offset from easting at the survey point's MD. This value is calculated automatically.
DLS Dogleg severity at the survey point's MD. This value is calculated automatically.
Survey Tool Survey tool used to take the survey. To change the tool used for the survey point, click the button on the right side of the cell.
Comment Any comment relevant to the survey point.
Add New Survey Points (beyond MD 0.00) to Definitive Survey Adds any new points in the survey to the definitive survey, and updates existing survey points
Options Display MD relative to

Controls the drill floor reference used to display MD values

  • Definitive survey drill floor – Displays MD values relative to the drill floor of the definitive survey.
  • Current survey drill floor – Displays MD values relative to the drill floor of the current survey.
Survey Unit Distance unit used for MD, TVD, north offset, and east offset values. Existing values will be automatically converted to the new unit, unless the Preserve Existing Values option is selected.
Preserve Existing Values Prevents MD, TVD, north offset, and east offset values from being converted when a new survey unit is selected

The table below defines the properties in the Survey Options window. This window appears when you click the Survey Options button in the Survey Editor - Edit window. If you do not know the magnetic parameters, click the Calculate button to open the geo-magnetic calculator. See "Survey Magnetic Parameters".

Property definitions for the Survey Options window
Property Name Definition
Field Strength Strength of the magnetic field at the well's location
Model Model used to calculate the magnetic parameters
Dip Angle between the local horizontal vector and the Earth's magnetic field vector
Date Date the magnetic parameters were calculated
Dec Clockwise bearing of the horizontal component of the Earth's magnetic field vector from true north
Survey Type

Type of survey. This value can't be edited after the survey is created.

  • Normal MWD – Normal survey using MD, inclination, and azimuth to calculate TVD, north offset, and east offset.
  • Positional SAG D – Positional survey using TVD, north offset, and east offset to calculate MD, inclination, and azimuth. See "Positional Surveys".
