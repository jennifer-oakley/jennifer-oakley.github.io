The Well Parameter Lists page contains the default settings for well parameters. To edit the default well parameter settings, click the Options and Settings button () in the main menu and go to the Well Parameter Lists page.

List of company names currently available in 5D.

  • To remove a company name, select the company name from the list and click Remove Item.
  • To add a company name, enter the company name in the blank field and click Add Item.
Specific Well Plans 2 of 7 Creates a well path from the surface to the target using seven parameters, two of which are unknown. See "Two of Seven planning".

-- Overwrite current plan casing points with previous plan casing points Overwrites the casing points of the current plan with the casing points of the previous plan, when a previous plan is promoted to the be current plan
-- On plan drill floor change: Create new plan?

Default behavior for new plans when the elevation of the drill floor is changed.

  • Yes – A new plan will be automatically created with the updated drill floor elevation.
  • No – A new plan will not be created, and the current plan will be updated to reflect the new drill floor elevation.
  • Prompt – A message will appear when the drill floor elevation changes, asking if you want to create a new plan.
