Anti-collision parameters

The table below defines the analysis options for an anti-collision analysis. The parameters for anti-collision analysis are stored in the 5D database for the field. To open the analysis settings, click the Options and Settings button () in the main toolbar and go to the Anti-Collision Parameters page.

Field definitions for anti-collision parameters
Section Field Name Definition
Anti-Collision Parameters Separation Factor Method Method used to calculate the separation factors for each wellpath.
Ellipsoid spacing Interval used to determine the placement of uncertainty ellipsoids along the current plan. This setting does not apply to surveys.
σ for uncertainties Standard deviation used to determine the size of the uncertainty ellipsoid. A larger value increases the probability that the actual wellpath will lie within the ellipsoid. Typically, this value should be between one and three.
Ellipsoid combination method

Method for combining the separation factors for each ellipsoid of uncertainty. Only available when the Standard method is selected.

  • Independent – The positional error of each ellipsoid is used to calculate the separation factors.
  • Combined – The positional errors of all ellipsoids are combined before the separation factor is calculated.
Surface Uncertainty Indicates that surface uncertainty is accounted for when calculating the uncertainty ellipsoid
Default σ for supplied uncertainties Standard deviation used to determine the amount of surface uncertainty. Typically, this value should be between one and three.
Analysis method

Method for calculating the gap between ellipsoids on different wellpaths. Only available when the Standard method is selected.See "Separation factors".

  • Vector separation – Measures the shortest distance between corresponding points on the ellipsoids.
  • Pedal curve – Measures the shortest distance between tangents of corresponding points on the ellipsoids.
WPTS Parameters Analysis parameters for the WPTS method for calculating separation factors
Misc Parameters Centre to centre distance unit Unit for center-to-center distance measurements
Travelling cylinder reference

Reference used to define the traveling cylinder
AC Visualisation Method

Method used to display the uncertainty around each wellpath

  • Absolute – Uncertainty represents the absolute spatial error of the wellpath position.
  • Relative – Uncertainty represents the spatial error of wellpath position, relative to another wellpath.
Risk Thresholds Separation factor Indicates that separation factors will be evaluated to predict the risk of a collision, using the thresholds defined for high, medium, and low risk
Cr-Cr Indicates that center-to-center distances will be evaluated to predict the risk of a collision, using the thresholds defined for high, medium, and low risk
Ell Sep Indicates that ellipsoid separation will be evaluated to predict the risk of a collision, using the thresholds defined for high, medium, and low risk
Graphical Options Display Ellipsoids Indicates the individual ellipsoids will be displayed in the 3D Viewer
Ellipsoid Bounding Box Indicates which sides of the ellipsoid bounding box will be displayed in the 3D Viewer
Display Discs Indicates that discs will be displayed in the 3D Viewer
