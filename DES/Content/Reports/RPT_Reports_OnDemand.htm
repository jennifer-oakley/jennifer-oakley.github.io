About on-demand reports

The Reports > Reports > On-Demand page allows you to create an on-demand real-time report that contains a detailed summary of the well and the drilling performance on the rig. On-demand reports are generated based on the date selected in the Range field.

You can create reports for four different report ranges:

  • Date – Creates a report for the selected date. To select a different day, click the Calendar button () in the Report Date field.
  • Entire Well – Creates a cumulative report for the entire well.
  • Time Frame – Creates a cumulative report for the selected time frame. To change the start date, click the Calendar button () in the From field. To change the end date, click the Calendar button () in the To field.

There are two on-demand templates available:

  • Standard – The Standard template includes information about rig activities, friction factors, torque & drag effects, planned vs. actual comparisons, hydraulics calculations, and custom charts.
  • Milling Report – The Milling Report template includes information about milling performance, such as ROP, weight on mill, and milling torque.

Creating an on-demand report

To generate an on-demand real-time report:

  1. Go to the Reports > Reports > On-Demand page.
  2. In the Template field, select the report template you want to use.
  3. Select the information you want to include in the report. See "Report details".
  4. In the Range field on the right side of the screen, select a method for filtering the report data. Use the Calendar button () to select different dates, or click the Well Activities button to select a well activity.
  5. If you are using the Standard template, click the Add Chart button () to add charts to the report.
  6. Click the Save Report button () at the top of the screen.
  7. In the window that appears, navigate to the location where you want to save the report and click Save.

The report will be automatically generated for the selected report range. When the report finishes generating, it will open automatically. When the report opens in MS Excel, click the Enable Content button to enable all of the report features.

Using the On-Demand page

The toolbar at the top of the Reports > Reports > On-Demand page is used to add charts and save reports.
