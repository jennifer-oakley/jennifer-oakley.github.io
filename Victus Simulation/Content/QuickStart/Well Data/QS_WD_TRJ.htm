About wellpath data

The next step is to add the wellpath information. The Setup > Well Data > Wellpath tab contains the wellpath data for the planned well trajectory and real-time surveys. The drop-down menu at the bottom of the screen allows you to switch between the trajectory data and the survey data.

  • The Trajectory view should be used for the survey points in the planned trajectory of the well. The data in the Trajectory view is only for reference, and is not used in any calculations.
  • The Survey view should be used for the actual, real-time survey data. All calculations in Victus™ Simulation use the data in the Survey view.

The measured depth, inclination, and azimuth of each survey point are used to calculate several values, such as TVD, dogleg severity, build rate, and turn rate. These values are updated automatically as you add and edit survey points.

The first two survey points of the wellpath are automatically calculated, based on the rig elevations and the properties of the derrick. These surveys points are highlighted in pink on the Wellpath tab. To change these survey points, you must update the rig elevations on the Setup > Surface Equipment page. See "Rig Elevations".

Importing wellpath data from a file

To import survey data from a CSV, LAS, or Compass file:

  1. Go to the Wellpath tab on the Setup > Well Data page.
  2. In the drop-down menu at the bottom of the screen, select the Survey option.
  3. Click the Import Wellpath button () at the bottom of the screen.
  4. In the menu that appears, select the File option. This menu will only appear if you are connected to a WITSML channel with trajectory data.
  5. Click Yes on the warning message that appears.
  6. Navigate to the file you want to import and click Open. A list of import options will appear in Victus™ Simulation.
  7. Select the options for the measured depth data in the file you are importing.
    • MD data is the name of the MD column.
    • MD unit is the measurement unit for MD.
  8. Select the options for the inclination data in the import file.
    • Inclination data is the name of the inclination column.
    • Inclination unit is the measurement unit for inclination.
  9. Select the options for the azimuth data, then click the OK button ().
    • Azimuth data is the name of the azimuth column.
    • Azimuth unit is the measurement unit for azimuth.
  10. When the import finishes, click the Save Changes button () at the bottom of the screen.

The calculated values and charts on the Wellpath tab will be updated automatically. Any existing survey points will be deleted and replaced with the new data. If you updated the survey data, the formation layers on the Formation tab will be updated to reflect the new TVD values. If you updated the trajectory data, the formation layers will not be affected.

After you import the survey data, the next step is to enter the formation data. See "Describing the formation".
