This section guides you through the basic steps for setting up Victus™ Simulation. After you have completed the steps in this guide, you'll be ready to create dashboards to monitor incoming data, simulate MPD operations, and create automated reports. See "Dashboards" and "Control Tiles".

Step 1: Start a project

The first step for using Victus™ Simulation is to start a new project. When you start a new job, you'll need to create a well and choose the unit system for the new well.

  • Create a well. When you create a well, you'll need to enter basic details about the job, such as the well location, rig name, and operating company. See "Adding a new well".
  • Assign a unit system. The unit system controls all of the measurement units used throughout Victus™ Simulation. See "Choosing a unit system".

Step 2: Update rig equipment

The next step is to add information about the surface equipment for the well on the Setup > Surface Equipment page. The Surface Equipment page contains information about the rig, such as the type of rig, the elevation of the rig, and the equipment on the rig. There are five basic steps for entering the rig details:

  • Select the equipment on the rig. The equipment configuration includes the rig type and size, as well as the BOP, pumps, RCD, and other equipment on the rig. See "Configuring rig equipment".
  • Enter the elevations on the rig. Rig elevations are used to determine the starting depth of the well, which is the measured depth at the top of the well. See "Entering rig elevations".
  • Enter rig limits. The specifications of the derrick are required for torque and drag calculations. See "Entering rig limits".
  • Update sensor information. You'll need to specify the type of sensor connected to each source selected on the Equipment Configuration tab. See "Updating sensor information".
  • Activate additional pumps. If the rig has an auxiliary or boost pump, you'll need to enable it on the Piping and Instrumentation tab. You can also activate additional main pumps. See "Activating additional pumps".

Step 3: Calibrate sensors

After you've updated the rig equipment, you'll need to calibrate the spare analog sensors. Spare analogs are additional plug and play sensors that can be connected directly to the analog sensor ports on the UPI with a three-pin cable. The spare analog sensor ports can be connected to any type of 4-20 mA sensor, such as temperature sensors. See "Calibrating spare analogs".

Step 4: Set up incoming data

After you've updated the rig equipment, you'll need to configure the incoming data. There are five basic steps for configuring incoming data:

  • Set up WITS data. Incoming WITS data is configured on the Setup > DAQ > WITS tab. See "Setting up WITS data".
  • Set up WITSML data. Incoming WITSML data is configured on the Setup > DAQ > WITSML tab. See "Setting up WITSML data".
  • Set up modbus data. Incoming modbus data is configured on the Setup > DAQ > Modbus tab. See "Setting up modbus data".
  • Set up OPC UA data. Incoming OPC UA data is configured on the Setup > DAQ > OPC UA tab. See "Setting up OPC UA channels".
  • Define well parameters. The mappings on the Setup > Mappings > Parameters tab tell Victus™ Simulation which data source to use for each well parameter, such as bit depth and well depth. See "Defining well parameters".

Step 5: Add well data

The next step is to add data about the well, such as the well trajectory, formation characteristics, and well intervals. All of the data needed to describe the well can be entered on the Setup > Well Data page. Before receiving data or running an analysis, you'll need to add the data listed below.

  • Wellpath data. The real-time surveys for the well are used for all calculations. The easiest way to add survey data is to import the data from the WITSML server. See "Importing wellpath data".
  • Formation data. The characteristics of the wellbore formation are used to calculate pore pressure, allowable ECD, and pressure limits for the annulus. See "Describing the formation".
  • Wellbore intervals. The cased and open hole sections are defined using well intervals. The depth of the current interval is used by the On/Off Bottom indicator () to determine if the bit is on-bottom. See "Creating wellbore intervals".
  • Formation temperatures. The formation temperatures along the length of the wellbore are used to determine the temperature gradient. See "Entering temperature data".
  • Influx zones. An influx zone is a section of the formation where a kick or a loss is more likely to happen. Influx zones are used in influx simulations and underbalanced drilling analyses. See "Defining influx zones".

Step 6: Add run data

The next step is to add the data for each run, including the work strings and drilling fluids for each bit run.

  • Work string configurations. Victus™ Simulation includes the Weatherford catalog, which is the master catalog for BHA and drillstring components.. See "Building a work string".
  • Drilling fluids. The fluid's composition and rheological properties are used in many hydraulics calculations, such as ECD, bottomhole pressure, fluid volume, and influx detection. See "Adding fluids".
  • Pump fluids. The current pump fluid determines which drilling fluid is being pumped into the wellbore. See "Setting pump fluids".
