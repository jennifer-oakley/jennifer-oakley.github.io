About pressure controls

The Pressure tile in the Well View pane controls the way Victus™ Simulation manages pressure in the well. The Pressure tile shows two primary pieces of information.

  • The abbreviation in the bottom left corner of the tile shows the current pressure control mode.
  • The number in the top right corner of the tile shows the set point for the current pressure control mode.

Victus™ Simulation has seven modes for managing pressure:

  • SBP mode – Manages pressure by maintaining a constant surface backpressure. See "Using SBP mode".
  • SPP mode – Manages pressure by maintaining a constant standpipe pressure. See "Using SPP mode".
  • AP mode – Manages pressure by maintaining a constant bottomhole pressure at a specified depth. See "Using AP mode".
  • Flow mode – Manages pressure by maintaining a constant gain or loss for the flow rate. See "Using Flow mode".
  • Connection mode – Manages pressure by maintaining a constant bottomhole pressure during an MPD connection. See "Using Connection mode".
  • S&S mode – Manages pressure by controlling the surging or swabbing pressure changes resulting from the bit movement up or down the well. See "Using S&S mode".
  • None – Pressure is not managed by Victus™ Simulation, and the chokes are fully opened. See "Disabling pressure control".

At least one of the chokes must be enabled in order to use any of the pressure control modes. If all chokes are disabled, you cannot use Victus™ Simulation to manage pressure.

Using SBP mode

In SBP mode, Victus™ Simulation maintains a constant surface backpressure using a customizable set point. Victus™ Simulation compares readings from the SBP sensor to the set point for surface backpressure and adjusts the choke positions accordingly. Victus™ Simulation will automatically switch to SBP mode when the high limit reaches the threshold set on the Setup > Control Algorithms > Limits tab. See "Limits algorithm".

To enable SBP mode:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the SBP option.
  3. Click Yes on the warning message that appears.
  4. In the SBP window, enter the set point for surface backpressure. The default set point is the current surface backpressure.
  5. Click the Enter button (), then click the Close button ().

The chokes will be adjusted as needed to maintain the SBP set point. The current surface backpressure will be displayed in the banner at the top of the screen.

Warning

If any active chokes are switched to Manual mode when Victus™ Simulation switches to SBP mode, the SBP setpoint will update, according to the current pressure. If any chokes are switched back to Auto mode, and the surface backpressure does not exceed the threshold selected in the System goes to SBP mode at field on the Setup > Control Algorithms > Limits tab, the SBP setpoint will be set to the current pressure at the time the chokes were switched to Auto mode. See "Limits algorithm".

Using SPP mode

In SPP mode, Victus™ Simulation maintains a constant standpipe pressure using a customizable set point. Victus™ Simulation compares readings from the SPP sensor to the set point for standpipe pressure and adjusts the choke positions accordingly.

When using SPP mode, it is important to consider the effects of hydraulic delay. Choke movements create additional pressure pulses, which travel from the choke to the standpipe. Hydraulic delay is the total travel time for these pulses, which depends on the compressibility of the fluid. Victus™ Simulation does not account for hydraulic delay, so it is important to exercise caution when using SPP mode. The SPP set point should be adjusted gradually to minimize unwanted pressure changes.

To enable SPP mode:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the SPP option.
  3. Click Yes on the warning message that appears.
  4. In the SPP window, enter the set point for standpipe pressure. The default set point is the current standpipe pressure.
  5. Click the Enter button (), then click the Close button ().

The chokes will be adjusted as needed to maintain the SPP set point. The standpipe pressure will be displayed in the banner at the top of the screen.

Using AP mode

In AP mode, Victus™ Simulation maintains a constant bottomhole pressure using a customizable set point. Victus™ Simulation compares the calculated bottomhole pressure to the set point for annular pressure and adjusts the choke positions accordingly. Because bottomhole pressure is calculated at the depth specified for the set point, AP mode can be used to control annular pressure at any depth.

To enable AP mode:

  1. Click the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the AP option.
  3. Click Yes on the warning message that appears.
  4. In the AP window, enter the measured depth for the set point. The default depth is the current well depth.
  5. Click the Enter button (), then click the Close button ().

The set point will be automatically calculated, based on the current bottomhole pressure at the specified depth. The chokes will be adjusted as needed to maintain the AP set point. The annular pressure and depth will be displayed in the banner at the top of the screen.

Tip

In the AP window, you can override the calculated set point by entering a custom value in the Setpoint field.

Tip

The set point for annular pressure can be viewed in pressure units or density units. To toggle between units, go to the Setpoint field and click on the unit label.

Using Flow mode

In Flow mode, Victus™ Simulation maintains a constant flow gain/loss using a customizable set point. Victus™ Simulation compares the current flow gain/loss to the set point for flow, and adjusts the choke positions accordingly. Flow gain/loss is the difference between the incoming and outgoing flow rates, calculated by subtracting flow in from flow out. If the current flow gain/loss is less than the set point, the chokes will open to increase flow out. If the current flow gain/loss is larger than the set point, the chokes will close to decrease flow out.

To enable Flow mode:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the F option.
  3. Click Yes on the warning message that appears.
  4. In the F window, enter the set point for flow gain/loss. The default set point is the current flow gain/loss.
  5. Click the Enter button (), then click the Close button ().

The chokes will be adjusted as needed to maintain the flow gain/loss set point. The flow gain/loss will be displayed in the banner at the top of the screen.

Using Connection mode

In Connection mode, Victus™ Simulation maintains a constant annular pressure during an MPD connection. As the flow from the rig pumps decreases during a connection, surface backpressure must be increased to compensate for the loss in frictional pressure. Victus™ Simulation follows the connection ramp table on the Setup > Control Algorithms > Connection tab to adjust surface backpressure in order to keep annular pressure constant.

When you switch to Connection mode:

  • If SPP is below the threshold, Victus™ Simulation will use the ramp table to increase the SBP set point. However, the SBP set point will not be decreased until the SPP threshold has been reached.
  • If SPP is above the threshold, Victus™ Simulation will use the ramp table to increase and decrease the SBP set point as needed to maintain a constant bottomhole pressure.

If an SPP sensor fails during the connection, the SPP threshold will be disregarded and the SBP set point will be updated using the ramp table.

To enable Connection mode:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the C option.
  3. Click Yes on the warning message that appears.
  4. In the C window, enter the measured depth for the set point. The default depth is the current well depth.
  5. Go to the Pipe Fill SPP Threshold field and enter the expected minimum standpipe pressure when the drillstring is full of fluid.
  6. Go to the AP Setpoint field and enter the desired annular pressure set point. This option is not available on Driller mode.
  7. Click the Enter button (), then click the Close button ().

After the SPP threshold has been reached, surface backpressure will be released using the connection ramp table. The current rig activity will automatically switch to Connection - MPD. The surface backpressure, set point depth, and flow rate will be displayed in the banner at the top of the screen.

Tip

In the C window, you can customize the default SPP threshold. In the Pipe Fill SPP Threshold field, enter the expected minimum standpipe pressure when the drillstring is full of fluid.

Using S&S mode

In S&S mode, Victus™ Simulation controls the surging or swabbing pressure changes resulting from the bit movement up or down the well. Swab and surge pressures are affected by many factors, including the fluid's rheological properties, the running speed and acceleration at which the pipe enters or exits the well, the dimensions of the annulus, and the length of the drillstring in the well.

When the work string is pulled out of the well, the fluid falls to replace the volume of the pipe removed from the well. In a closed system, the fluid falls from the annulus. In an open system, the fluid falls from the pipe and the annulus. This is known as swabbing, which causes pressure to decrease while the mud fills the volume of the displaced pipe.

When the work string is run into the well, fluid is displaced from the well by the volume of the pipe added to the well. This is known as surging, which causes pressure to increase due to induced flow from the displaced mud in the annulus.

S&S mode is only available when tripping or reaming. You need to enable the Block Speed parameter in the Mappings section and set a value between 0 and 1 ft/s before enabling S&S mode. See "Mappings".

To enable S&S mode:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the S&S option.
  3. Click Yes on the warning message that appears.
  4. In the S&S window, enter the set point for reference depth and hookload threshold. The default set point is the well depth.
  5. Click the Enter button (), then click the Close button ().

When the hookload threshold is reached, surface backpressure will be released using the set point on the Setup > Control Algorithms > Surge / Swab tab. The current set point will be determined by the trip direction and current block speed. See "Surge/Swab algorithm". The reference depth, SBP set point, and current block speed will be displayed in the banner at the top of the screen.

Disabling pressure control

When None is selected for the pressure control mode, Victus™ Simulation stops managing pressure and the chokes will be fully opened. Pressure control must be disabled in order to turn off all chokes.

To disable pressure control:

  1. Click on the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the None option.
  3. Click Yes on the warning message that appears.

The chokes will be fully opened, and Victus™ Simulation will stop managing pressure.
