About choke controls

The MPD manifold includes up to four chokes, which are controlled by Victus™ Simulation. Victus™ Simulation opens and closes the chokes as needed to control pressure. Each choke has a position indicator, which shows the actual choke position during operation.

The Chokes tile in the Well View pane allows you to choose the method for controlling the chokes. There are four methods for controlling the chokes:

  • Auto – Sets the choke position automatically. The desired choke position is determined by Victus™ Simulation, and the choke is adjusted accordingly. See "Automatically controlling chokes".
  • Manual – Sets the choke position to a manually entered value. Victus™ Simulation will adjust the choke to the desired choke position. See "Manually controlling chokes".
  • Off – Disables the choke, without changing the current choke position. The choke will no longer be controlled by Victus™ Simulation. See "Disabling the chokes".
  • Swap – Closes the choke and opens the selected choke to the choke position of the first choke, while maintaining the pressure set point. See "Swapping choke positions".
Note

Only two chokes can be in Auto mode at a time, and both chokes must be the same size. If there are more than two chokes on the MPD manifold, the remaining chokes must be disabled or in Manual mode.

Note

Auto mode cannot be enabled for chokes that are located downstream from the flowmeter, unless all chokes are located downstream from the flowmeter. Chokes that are located downstream from the flowmeter must be disabled or in Manual mode. See "MPD manifold properties".

Automatically controlling chokes

To automatically control a choke:

  1. Click on the Chokes tile at the bottom of the Well View pane.
  2. Select the choke that you want Victus™ Simulation to control, then select the Auto option.

Victus™ Simulation will adjust the choke position as needed, based on the current pressure control method. See "Pressure controls". Chokes located after the flow meter cannot be set to Auto mode, unless all the chokes are located after the flow meter.See "Pressure controls".

Manually controlling chokes

To manually control a choke:

  1. Click on the Chokes tile at the bottom of the Well View pane.
  2. Select the choke that you want to manually control, then select the Manual option.
  3. In the window that appears, enter the new position for the selected choke.
  4. Click the Enter button (), then click the Close button ().

The choke position will be updated for the selected choke. Victus™ Simulation will automatically open or close the choke to the new position as needed.

Disabling the chokes

To disable all chokes:

  1. Click the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the None option.
  3. Click Yes in the warning message that appears.
  4. Click the Chokes tile at the bottom of the Well View pane.
  5. For each choke in the menu that appears, select the choke and select the Off option.

All chokes will be disabled and ready for calibration. Each choke will remain in its current position. Victus™ Simulation will no longer control the chokes.

Swapping choke positions

The Swap option will close the current choke and open the selected choke to the choke position of the first choke. During this process, the pressure set point will be maintained. You can only swap choke positions if:

  • The current pressure control mode is set to SBP mode, AP mode, or C mode. See "Pressure controls".
  • The initial choke is in Auto mode. See "Automatically controlling chokes".
  • The receiving choke is in Manual mode or Disabled mode, and the choke position is 5% or less. See "Manually controlling chokes" and "Disabling the chokes".
  • The receiving choke is the same size and model as the initial choke.

To swap choke positions:

  1. Click on the Chokes tile at the bottom of the Well View pane.
  2. Select the choke that you want to close, then select the Swap option.
  3. In the menu that appears, select the choke that you want to open.
  4. Click Yes in the confirmation message that appears.

The first choke will be slowly closed, as the receiving choke is slowly opened. When the choke swap completes, an alert will appear in the banner at the top of the screen. The initial choke will be completely closed and set to Manual mode. The receiving choke will be opened to the initial choke's original position and set to Auto mode.

Note

Chokes located after the flow meter cannot be swapped, unless all the chokes are located after the flow meter.
