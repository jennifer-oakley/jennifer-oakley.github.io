About the Weatherford SPMS

When Victus™ Simulation is connected to a Weatherford SPMS, the PRV tile contains the following menu items:

  • PT5 – Used to enable or disable pressure transducer 5.
  • PT6 – Used to enable or disable pressure transducer 6.
  • PT7 – Used to enable or disable pressure transducer 7.
  • PT8 – Used to enable or disable pressure transducer 8.
  • ALG – Used to set the PRV control mode. See "Control modes".
  • SP – Used to enter the PRV set point. See "PRV set point and offset".
  • Offset – Used to calculate the dynamic PRV set point. See "PRV set point and offset".

Enabling the Weatherford SPMS

To enable the Weatherford SPMS:

  1. Go to the Setup > Surface Equipment > Equipment Configuration tab.
  2. In the Pressure Relief Valve field, select the Weatherford SPMS option. See "Selecting rig equipment".
  3. Click the Save Changes button () at the bottom of the screen.
  4. Enable the active pressure transducers. See "Pressure transducers".
  5. Set the PRV control mode. See "Control modes".
  6. Enter the PRV set point. See "PRV set point and offset".

The new PRV set point will be applied to both PLCs. When the set point is exceeded, the PRV will be opened to release pressure. If a pressure transducer is disabled, the readings from the sensor are disregarded by Victus™ Simulation. The readings from disabled sensors will not be used to determine whether to open or close the PRV.

Pressure transducers

To enable a pressure transducer:

  1. Click the PRV tile and select the pressure transducer you want to enable.
  2. In the menu that appears, select the Enable option.
  3. When prompted, click Yes to confirm that you want to enable the pressure transducer.

The sensor status will be changed to active, and the readings from the sensor will be used to determine whether to open or close the PRV.

Control modes

The Weatherford PRV has two control modes: Open mode and Auto mode.

  • In Auto mode, the PRV valves will remain closed until the PRV set point is exceeded. When the PRV set point is reached, the PRV values will open to release pressure. The valves will close again when pressure has decreased to 80% of the PRV set point.
  • In Open mode, the PRV valves will open and remain open until the PRV control mode is changed back to Auto mode.

To set the PRV control mode:

  1. Click the PRV tile and select the ALG option.
  2. In the menu that appears, select the PRV control mode you want to use.
  3. When prompted, click Yes to confirm that PRV control mode you selected.
  4. If you selected Auto mode, define the PRV set point. See "PRV set point and offset".

The PRV valves will be opened or closed according to the selected control mode.

Warning

Open mode should only be used for testing and inspection purposes. Auto mode should always be used when operating with pressurized mud.

PRV set point and offset

For the Weatherford PRV, Victus™ Simulation sends a set point to the PRV. The PRV opens or closes as needed to prevent overpressure. You can either use the dynamic, automatically calculated set point, or you can manually enter a static set point.

Dynamic set point

The dynamic set point is automatically calculated by Victus™ Simulation. The dynamic set point will be used as long as it does not surpass the equipment protection level. The dynamic set point is calculated using the MAASP maximum allowable annular surface pressure value, plus the offset value. The default offset value is 100 psi, and it is only used with the dynamic set point.

To use a dynamic set point:

  1. Click the PRV tile and select the SP option.
  2. In the window that appears, click the Undo button ()
  3. Click the PRV tile again and select the Offset option.
  4. Enter the desired offset value and click the Enter button ().
  5. When prompted, click Yes to confirm the new offset.

When Victus™ Simulation calculates the dynamic set point, the offset value will be added to the calculated MAASP. If the dynamic set point is less than the PRV's equipment protection level, the set point will be sent to the PRV.

Static set point

You can also manually enter a static PRV set point, instead of using the dynamic set point. However, we strongly advise you to use caution and consider the potential risks before proceeding.

To manually enter a static set point:

  1. Click the PRV tile and select the SP option.
  2. In the window that appears, enter a value to define a static set point and click the Enter button ().
  3. When prompted, click Yes to confirm the new set point.

Victus™ Simulation will send the static set point to the PRV. An orange line will appear on the window to indicate a static set point is now in use. If you wish to go back to using a dynamic set point, click the Undo button (), then click the Close button ().
