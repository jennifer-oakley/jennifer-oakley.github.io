About dashboad control tiles

There are several control tiles that you can add to dashboards. These control tiles function the same way as the control tiles at the bottom of the Well View pane. See "Control Tiles".

Victus™ Simulation has the following control tiles:

  • The MPD control () allows you to change the way Victus™ Simulation manages pressure. You can use this control to change the current pressure mode, as well as update the set point. See "Pressure controls".
  • The Chokes control () allows you to choose the method for controlling the hydraulic chokes on the MPD manifold. See "Chokes control".
  • The Pumps control () displays the current pump rates for all of the rig pumps. See "Pumps control".
  • The PRV control () allows you to enable the soft set point for a pressure relief valve. See "PRV control".
  • The High Limit control () allows you to update the maximum allowed value for surface backpressure. See "Pressure Limit control".
  • The Detection control () allows you to enable automatic kick/loss detection. See "Detection control".
  • The Pump control () allows you to turn the RCD HPU pump on or off remotely.
  • The Bearing Latch control () allows you to open, close, or disable the primary bearing latch remotely.
  • The Sec. Latch control () allows you to open or disable the secondary bearing latch remotely. The secondary bearing latch is used only when there is a problem with the primary bearing latch. The Sec. Latch control can not be used to close the secondary bearing latch.
  • The Transducer control () allows you to switch the source of the active wellbore sensor between the primary and secondary pressure transducer remotely.
  • The LPS control () allows you to enable or disable the latch position sensor remotely.

Adding control tiles

To add a control tile to a dashboard:

  1. On the Home screen, select the dashboard that you want to add a control tile to.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, expand the Control Operations heading.
  4. Click on the icon for the control tile you want to create, then drag it to a blank area on the dashboard.
  5. If you want the control tile to span across multiple grid areas, click on the bottom right corner of the control tile and drag it into an adjacent blank area.

The selected control tile will appear on the dashboard. To view the menu for the control, click on the control tile in the dashboard.
