About gauges

Gauges give you a quick visual indication of the status for any operating parameter or calculated value. You can add gauges to dashboards on the Home screen, as well as to custom panels. Custom panels are standalone windows that can be re-sized and moved anywhere on your screen. See "Custom panels".

Gauges are most useful when they are connected to limits and alarms. When alarm limits are enabled, yellow and red areas will appear on the gauge to indicate when an alarm limit has been exceeded. To connect a gauge to a limit, you'll need to enable the alarm limits for that parameter on the Setup > Limits and Alarms page. See " Limits and Alarms".

Note

When you are using Playback mode, the limits on some gauges will temporarily disappear after you begin playing back the data. The limits should reappear after playback reaches the time at which the limits were originally defined.

Adding gauges to a dashboard

To add a gauge to a dashboard:

  1. On the Home screen, select the dashboard that you want to add a gauge to.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, locate the curve that you want to create a gauge for.
  4. Select the name of the curve and drag it to one of the blank areas on the dashboard.
  5. If you want the gauge to span across multiple grid areas, click on the bottom right corner of the gauge and drag it into an adjacent blank area.

The new gauge will appear on the dashboard. Click the Menu button () to change the gauge settings. You can change the type of gauge, as well as customize the units and limits applied to the gauge. See "Changing the gauge type" and "Customizing units and limits".

Note

Raw data curves, such as mnemonics for a WITS channel, will be deleted from the dashboard if the associated channel is deleted. However, WITSML curves will be retained if a new channel is created with matching mnemonics.

Tip

If there aren't any blank areas on the dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

Changing the gauge type

To change the type of gauge:

  1. On the Home screen, go to the dashboard that contains the gauge you want to update.
  2. Hover your mouse over the top left corner of the gauge. The Menu button () will appear on top of the gauge.
  3. Click the Menu button (), then select the type of gauge you want to use.

The gauge will be updated to the selected type of gauge.

Customizing units and limits

To customize the units and limits applied to a gauge:

  1. On the Home screen, go to the dashboard that contains the gauge you want to update.
  2. Hover your mouse over the top left corner of the gauge. The Menu button () will appear on top of the gauge.
  3. Click the Menu button (), then click the Options button ().
  4. To use a different measurement unit to the gauge, select the unit you want to use in the Unit field.
  5. To apply a different set of limits to the gauge, enter the desired limits in the Minimum, Min Severe, Min High, Max Severe, and Maximum fields.

The gauge will be automatically updated as you make changes. The custom settings applied to the gauge will override the settings entered on the Setup screen. However, the gauge settings will only be applied to that particular gauge. If you have another gauge for the same parameter, these settings will not be applied to it.
