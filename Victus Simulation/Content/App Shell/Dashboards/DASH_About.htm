About dashboards

Dashboards are fully customizable grids that you can add gauges and graphs to. See "Gauges" and "Graphs". You can also add indicators, control tiles, and other items to a dashboard. See "Indicators" and "Dashboard control tiles".

All of the dashboards that have been created for the current well can be viewed on the Home screen. The left side of the Home screen contains a thumbnail image of each dashboard. These thumbnails allow you to quickly navigate between the different dashboards for the well.

A check mark () in the top left corner of the dashboard thumbnail indicates which dashboard is the default dashboard. The default dashboard is the dashboard that opens when you open the Victus™ Simulation application. To change the default dashboard, hover your mouse over the top left corner of the dashboard thumbnail and click the check mark icon ().

Creating dashboards

To create a dashboard:

  1. On the Home screen, click the Edit Dashboard button () in the bottom left corner of the screen.
  2. Click the Add Dashboard button () in the Dashboard Editor panel.
  3. Drag the mouse to select the grid size of the new dashboard.
  4. Add graphs and gauges to the new dashboard that appears. See "Graphs" and "Gauges".
  5. When you have finished creating your dashboard, click the Minimize button () to close the Dashboard Editor panel and save your changes.

A blank dashboard will appear on the Home screen. A thumbnail image for the new dashboard will appear in the Dashboard Editor panel. As you add graphs and gauges to the dashboard, the thumbnail image will be updated with a preview of the dashboard.

Adding other items to a dashboard

To add a label or process flow arrow to a dashboard:

  1. On the Home screen, select the dashboard that you want to add an item to.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, expand the Others heading.
  4. Click on the icon for the item you want to create, then drag it to a blank area on the dashboard.
    • To add a text box, click on the Label icon ().
    • To add a process flow arrow, click on the Process Flow icon ().
  5. If you want the item to span across multiple grid areas, click on the bottom right corner of the item and drag it into an adjacent blank area.

If you added a label, a text box will appear on the dashboard with the text you entered. To customize the default font settings, click the Menu button () in the top left corner of the text box.

If you added a process flow arrow, an arrow will appear on the dashboard. To change the type of arrow, click the Menu button () in the top left corner of the text box.

Importing dashboards from another well

To import dashboards from another well in Victus™ Simulation:

  1. Go to the Project tab on the File > Import Objects page.
  2. In the Well field, select the well that you want to import dashboards from.
  3. After the list of dashboards appears, select the dashboards you want to import.
  4. Click the Save Changes button () at the bottom of the screen.

The selected dashboards will be imported into the current well. They will appear on the Home screen, with the other dashboards for the well. If any of the imported dashboards has the same name as an existing dashboard, an asterisk (*) will be added to the name of the imported dashboard.

Tip

To import all items into the current well, select the checkbox in the header row. All of the objects shown will be selected.

Copying dashboards

To create a copy of an existing dashboard:

  1. On the Home screen, select the dashboard you want to copy.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. Click the Clone Dashboard button () in the Dashboard Editor panel.

A new dashboard will be created, with the same graphs, curves, and indicators as the original dashboard.

Note

The Clone Dashboard button () is disabled for the default dashboards.

Deleting dashboards

To delete a dashboard:

  1. On the Home screen, select the dashboard that you want to delete.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. Select the thumbnail image of the dashboard and drag it to the Delete Dashboard button ().

The dashboard will be deleted, and the thumbnail image of the dashboard will be removed from the Dashboards panel.

Deleting graphs and gauges

To delete a graph or gauge:

  1. On the Home screen, select the dashboard that contains the graph or gauge you want to delete.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. Hover your mouse over the top right corner of the graph or gauge, then click the Delete button ().

The item will be removed from the dashboard, and the thumbnail image of the dashboard will be updated in the Dashboards panel.

Using the Dashboard Editor panel

There are several keyboard shortcuts and mouse actions that you can use when the Dashboard Editor panel is open.

  • To open a dashboard in a separate window, double-click the thumbnail of the dashboard in the Dashboard Editor panel.
  • To resize a floating dashboard, click and drag the right edge or the bottom edge of the window.
  • To rename a dashboard, double-click on the name of the dashboard in the Dashboard Editor panel and enter a new name.
  • To add a new row or column to a dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

The toolbar in the Dashboard Editor panel is used to create dashboards, copy dashboards, take screenshots of dashboards, and delete dashboards. To view the Dashboard Editor panel, click the Edit Dashboard button () in the bottom left corner of the screen.
