Main work area

The main work area is where most of the work in Victus™ Simulation is done, such as entering data, running an analysis, monitoring well conditions, and managing wells. The main menu is located at the top of the main work area, along with the current date and time. The main menu is used to navigate between the different screens in Victus™ Simulation.

  • The File screen allows you to manage the wells in Victus™ Simulation. See "File menu".
  • The Setup screen contains all of the information related to the well, such as job details, rig equipment, well data, work strings, fluids, parameters, and unit systems. See "Setup screen".
  • The Tools screen contains simple calculators that only require a minimum amount of data, as well as geomechanics and hydraulics analysis tools. See "Planning tools".
  • The Home screen displays various dashboards that contain gauges and graphs for monitoring well conditions during drilling operations, showing incoming data from sensors, hydraulics calculations, and WITS/WITSML data. See "Home screen".
  • The Alarms screen has several tools to help you identify and resolve problems that occur within Victus™ Simulation. See "Alarms".
  • The Feedback screen allows you to add comments about well operations, view maintenance logs, generate reports, and view the Victus™ Simulation application logs. See "Feedback".
Tip

The side menu bar can be minimized so that only the menu icons are visible. You can also hide the navigation bar completely.

  • To show only the menu icons, click the Hide Labels button () in the main menu.
  • To hide the menu bar completely, click the Hide Menu button ().
  • To return to showing menu icons and labels, click the Show Menu button ().
Note

The date format shown in the main menu and throughout Victus™ Simulation is based on the date format defined in your Windows settings. To change the date format shown in Victus™ Simulation, update your Windows settings and restart Victus™ Simulation.
