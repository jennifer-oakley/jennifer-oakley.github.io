About counters

The indicators under the Counters menu are used to monitor and control fluid circulation times, pill tracking, pump rates, and tank volume:

  • The Time window shows the amount of time it will take for regular drilling fluid to circulate. The Time window includes three counters for circulation time: from the surface to the bit, from the bit to the surface, and the full circulation throughout the entire wellbore.
  • The Pill Tracker window tracks the progress of a pill as it is circulated through the well. The circulation progress updates automatically as the pump rate changes. See "Using the pill tracker".
  • The Strokes window tracks the pump rate, total strokes, and stroke volume for each pump.
  • The Virtual Tank windows allow you to adjust the calculated trip tank volume, which is calculated from the flow-in and flow-out rates. See "Using virtual tanks".

To open the indicators under the Counters menu, click the Indicators button () in the Utilities toolbar. Select the Counters option, then click on the indicator that you want to open.

Using the pill tracker

To start the pill tracker:

  1. Click the Indicators button () in the Utilities toolbar.
  2. Select the Counters option.
  3. In the menu that appears, select the Pill Tracker option.
  4. In the Pill Tracker window, click the Calculate button () to start the calculations.

The Pill Tracker window will show the number of pump strokes since the pill was added, the volume of mud pumped after the pill, the percentage of the annulus that the pill has circulated through, and the percentage of the drillstring that pill has circulated through. These values will be automatically updated as the pill pumps through the well. To stop the calculations, click the Stop Calculations button ().

Using virtual tanks

There are three virtual trip tanks:

  • ICU Virtual Trip Tank reads data directly from the flow-out metering device connected to the ICU, and it can only be controlled from the operation station. On all other stations, the ICU Virtual Trip Tank window will display the values calculated on the operation station.
  • Virtual Trip Tank 1 and Virtual Trip Tank 2 reads data from the flow-out metering device connected to the ICU or the data source mapped on the Setup > Mappings page. When the Flow Out parameter is mapped to an external data source, the incoming data will be used for gain/loss calculations. See "Mappings".

To start a virtual tank:

  1. Click the Indicators button () in the Utilities toolbar.
  2. Select the Counters option.
  3. In the menu that appears, select the virtual tank that you want to use.
  4. In the Virtual Tank window, click the Calculate button () to start the calculations.

The Virtual Tank window will show the adjustment made to the total volume calculation so it shows zero output in the unloaded state, the amount of time that has elapsed since the virtual tank started filling, and the flow difference over time. To stop the calculations, click the Stop Calculations button ().

Note

Outgoing virtual tank data can only be sent for the ICU Virtual Trip Tank. Data for the other virtual tanks cannot be sent as outgoing data. See "Data acquisition".
