About the Parameters indicator

The Parameters window shows the current values for the well parameters, defined on the Setup > Mappings > Parameters tab. To open the Parameters window, click the Indicators button () in the Utilities toolbar and select the Parameters option.

Tip

Only well parameters with the Display option selected will appear in the Parameters window. To add or remove well parameters, go to the Setup > Mappings > Parameters tab. See "Mappings".

Tip

You can manually override any of the values in the Parameters window. To override the value of a parameter, click on the current value in the Parameters window. In the window that appears, enter a new value and click the Enter button ().

Bit depth overwrite

If the bit depth received from WITS or WITSML data changes more than 90 ft between WITS/WITSML transactions, the bit depth will be overwritten to the last known value. When this occurs, an orange line will appear on the Bit Depth tile in the Parameters window.

To remove the overwrite:

  1. Click the Indicators button () in the Utilities toolbar and select the Parameters option.
  2. In the Parameters window, click the Bit Depth tile.
  3. In the window that appears, enter the current bit depth and click the Enter button ().
  4. Click the Undo button (), then close the window.

The overwrite will be removed and the orange line will disappear. The bit depth received from WITS/WITSML data will appear on the Bit Depth tile.
