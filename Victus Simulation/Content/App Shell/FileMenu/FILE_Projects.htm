About projects

In the Victus™ application suite, all of the data related to a job is stored with a project. This includes job information, the rig configuration, well data, fluid properties, limits and alarms, settings, and other data. On the File > Projects page, you can create a new well, change the active well, switch between modes, and delete inactive wells. The active well is marked by an arrow icon ().

Creating a new well

All of the data related to a job is stored within a well in the Victus™ Simulation database. You can create a well using the default configuration, which uses a set of default data that includes surface equipment, well data, drillstrings, and fluids. You can also copy the configuration of another well in Victus™ Simulation. The new well will be an exact copy of the original well, but without data acquired through WITSML, modbus, or WITS.

To create a new well:

  1. Go to the File > Projects page.
  2. Enter the project name and the well name.
  3. In the Based On Well field, select the well configuration you want to copy. To use the default configuration, select the Default option.
  4. Enter the remaining job and well information.
  5. Click the Save Changes button () at the bottom of the screen to save the project.

The well will be added to the Victus™ Simulation database, and it will automatically become the active well. The new well will open on the Home screen, and the project name will appear in the banner at the top of the screen. All of the information entered on the File > Projects page will appear on the Setup > Job Info page.

Note

Adding a well to Victus™ Simulation does not add it to any of the other Victus™ applications. To add the new well to another Victus™ application, you'll need to export the well from Victus™ Simulation, then import it into another Victus™ application. See "Exports".

Changing the active well

If there are multiple wells in the Victus™ Simulation database, you can change the active well on the Projects page.

To change the active well:

  1. Go to the File > Projects page.
  2. Select the row for the well that you want to activate.
  3. Click the Select button () at the bottom of the screen.

The selected well will become the active well. When Victus™ Simulation finishes loading the data for the well, the well will open on the Home screen. Inactive wells will remain in the Victus™ Simulation database.

Switching between modes

There are two modes for Victus™ Simulation: Real-Time mode and Playback mode.

  • Real-Time mode acquires new data from WITS, modbus, or WITSML data sources and performs real-time hydraulics calculations. In Real-Time mode, you can edit any part of the well configuration.
  • Playback mode replays the well data acquired in Victus™ Simulation or Victus™ Operation. The well configuration and rig states will change as they were changed in real-time. In Playback mode, you can speed up the data and fast-forward to a specific point of time. However, the well data can't be edited when you're using Playback mode.

To switch to Real-Time or Playback mode:

  1. Go to the File > Projects page.
  2. Select the well you want to simulate or replay.
  3. If you are currently in Real-Time mode, click the Play Well button () to switch to Playback mode.
  4. If you are currently in Playback mode, click theSimulate Well button () to switch to Real-Time mode.

If you switched to Playback mode, you will no longer be able to edit the well data. To start replaying the data, click on the clock and select a playback rate from the menu that appears. See "Playback controls".

If you switched to Real-Time mode, you will be able to edit any of the data for the well. To start acquiring data, click on the clock and press the Play button (). See "Real-time controls".

In addition, the selected well will become the active well. When Victus™ Simulation finishes loading the data for the well, the well will open on the Home screen.

Deleting wells

If you no longer need one of the wells in the Victus™ Simulation database, you can remove it from the database on the Projects page.

To delete a well from the database:

  1. Go to the File > Projects page.
  2. Select the row for the well you want to delete.
  3. Click the Delete Item button () at the bottom of the screen.
  4. When prompted, click Yes on the warning message that appears.

When the well has been deleted, a confirmation message will appear. The well will be deleted from the Victus™ Simulation database, and will not be listed on any screen in Victus™ Simulation. If you deleted the active well, the first well listed on the Projects page will automatically become the active well.

Using the Projects page

The toolbar at the bottom of the Projects page is used to change the active well, delete wells, and create new wells.

Toolbar buttons on the File > Projects page
Button Function
Switches the active well to the selected well. See "Changing the active well".
Switches the application mode from Real-Time mode to Playback mode. See "Switching between modes".
Switches the application mode from Playback mode to Real-Time mode. See "Switching between modes".
Deletes the selected well. See "Deleting wells".
Opens the New Project panel, which allows you to create a new well. See "Creating a new well".
