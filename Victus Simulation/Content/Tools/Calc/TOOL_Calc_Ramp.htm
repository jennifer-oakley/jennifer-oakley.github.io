About the ramp tool

The ramp tool on the Tools > Calculators > Ramp tab is used to calculate the surface backpressure required to maintain a constant wellbore pressure when ramping the pumps up or down. The ramp tool determines a flow rate schedule, based on the current flow rate. The flow rate schedule starts at zero and increases in increments, until it reaches the current flow rate. For each flow rate, the ramp tool calculates the surface backpressure required to maintain constant annular pressure at the casing shoe, at the bit, and at bottomhole.

To use the ramp tool:

  1. Go to the Ramp tab on the Tools > Calculators page.
  2. Click the Get Ramp Data button () at the bottom of the screen.

The flow rate schedule will be calculated, using the current operating parameters. To refresh the ramp calculations, click the Get Ramp Data button () again.

Note

In order to use the ramp tool, there must be flow present in the well. If the flow rate is zero, no results will be calculated.

Field definitions

The table below defines the values calculated on the Tools > Calculators > Ramp tab.

Calculated values for the Ramp tool
Field Name Definition
Flow Incremental flow rate
SBP - Casing Shoe Surface backpressure required to maintain constant pressure at the casing shoe
SBP - Bit Surface backpressure required to maintain constant pressure at the bit
SBP - Bottom Hole Surface backpressure required to maintain constant bottomhole pressure
