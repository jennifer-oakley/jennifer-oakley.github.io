About the rollover tool

The rollover tool on the Tools > Calculators > Rollover tab is used to calculate the optimum surface backpressure and rollover schedule to maintain bottomhole pressure when displacing a lighter fluid with a heavier fluid or a heavier fluid with a lighter fluid. The rollover tool calculates the rollover schedule starting with the current fluid in the well, and ending with the selected rollover fluid. If you want to start the simulation with a different mud than the fluid currently in the well, you'll need to change the current pump fluid. See "Mud utility".

By default, the rollover tool uses the current operating parameters as inputs for the rollover calculations. However, these values can be changed as needed.

To use the rollover tool:

  1. Go to the Rollover tab on the Tools > Calculators page.
  2. Click the Reset button () to reset the settings to match the current operating parameters.
  3. In the Fluid field, select the new fluid that will be pumped into the well.
  4. Change the remaining settings as needed.
  5. Click the Calculate button () to begin the rollover calculations.

When the rollover calculations have completed, the rollover schedule will automatically load on top of the rollover settings. If you want to re-run the calculations with different settings, click the Back to Input button () and edit the settings as needed.

If the measured depth at the end of the rollover schedule is not zero, then the simulation stopped before the top of the rollover fluid reached the surface. This occurs when it is not possible to complete the full circulation of the rollover fluid while maintaining bottomhole pressure, typically because the rollover fluid is too heavy or too light.

Note

The rollover fluid must have a different density than the current pump fluid. If the new fluid has the same density as the current fluid, no results will be calculated.

Field definitions

The table below defines the settings required for rollover calculations.

Field definitions for rollover settings
Field Name Definition
Flow Rate Flow rate at which the rollover schedule will start. This flow rate should be between the drilling flow rate and minimum flow rate.
Drilling Flow Rate Flow rate at which the well is circulated during drilling. This flow rate should not include flow from booster or auxiliary pumps.
Minimum Flow Rate Minimum flow rate required to circulate the well, also known as the slow circulation rate.
Surface Back Pressure Surface backpressure at which the rollover schedule will start. By default, this is the surface backpressure that is maintained while circulating at the drilling flow rate.
System Back Pressure Surface backpressure at the drilling flow rate, with the chokes fully open. This value is measured during the in-casing test.
High Limit High limit for surface backpressure
Rotary Speed Planned top drive speed during the rollover
Fluid New fluid that will be pumped into the well
min. BHP Minimum allowable bottomhole pressure
max. BHP Maximum allowable bottomhole pressure

The table below defines the values in the rollover schedule, which are calculated by the rollover tool.

Values calculated by the rollover tool
Field Name Definition
MD Measured depth of the head of the displaced fluid
TVD TVD of the head of the displaced fluid
Flow Rate Flow rate required to maintain surface backpressure
Pump Speed Pump rate required to achieve the desired flow rate, based on the pump output specified in the pump properties on the Setup > Surface Equipment > Piping and Instrumentation tab. See " Pump configuration".
Total Time Total time since the rollover began
Strokes Number of pump strokes required for each step of the rollover
Total Strokes Total number of pump strokes required for the rollover
Volume Total volume of fluid pumped into the well. Fluid pumped through booster or auxiliary pumps are not included.
SBP Surface backpressure required to maintain bottomhole pressure within the limits defined in the rollover settings
BHP Bottomhole pressure at each stage of the rollover
BH ECD

Bottomhole ECD for each step in the rollover. The openhole section is color-coded, based on the pore pressure gradient and fracture pressure gradient of the openhole section.

  • Yellow – Bottomhole ECD is less than the pore pressure gradient of the openhole section.

  • Green – Bottomhole ECD is between the pore pressure gradient and the fracture pressure gradient of the openhole section.

  • Red – Bottomhole ECD is greater than the fracture pressure gradient of the openhole section.
SBP to trap if pumps fail Surface backpressure that must be maintained in the event of pump failure during the rollover

Using the Rollover page

The toolbar at the bottom of the screen allows you to begin Rollover calculations and import the current operating parameters. When you are viewing the calculation results, additional buttons will appear in the bottom toolbar.

Toolbar buttons on the Rollover tab
Button Function
Resets the Rollover settings to match the current operating parameters
Begins Rollover calculations
Closes the calculation results and returns to the Rollover settings
Pins the chart legends to the top of the Rollover charts
