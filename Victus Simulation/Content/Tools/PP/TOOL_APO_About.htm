About overburden analysis

The Overburden tab examines the pressure exerted by the weight of the overlying sediments in the formation, using imported log data and the formation density data defined in the Setup screen. On the Overburden tab, you can import density, sonic, seismic, and check shot data.

There are four charts on the Overburden tab. The first three charts graph data from various sources, and the last chart is used to graph the results for the Overburden tab.

  • The Well Data chart graphs the formation pressure and pore pressure data entered in the Setup screen.
  • The Offset Well chart graphs imported log data from an offset well.
  • The Current Well chart graphs real-time data from the current well.
  • The Results chart shows the calculated overburden and hydrostatic pressure gradients.

You'll need to add data for the Offset Well chart and the Current Well chart. If there is no log data for the current well available, you can import additional offset well data into the Current Well chart. However, the data in the Current Well chart is weighted more heavily in the calculated results than the data in the Offset Well chart. When both charts contain data, the data in the Offset Well chart may be completely disregarded in the results.

There are two ways to view the results on the Overburden tab: the gradient view and the pressure view. To switch between the gradient view and the pressure view, click the Pressure/Gradient button () at the bottom of the Results chart.

  • The gradient view shows the calculated overburden pressure and hydrostatic pressure gradients.
  • The pressure view shows linear trends for lithostatic pressure and hydrostatic pressure.

Determining overburden pressure

To configure overburden pressure:

  1. Go to the Overburden tab on the Tools > Geomechanics page.
  2. If the experiment is not unlocked for editing, click the Edit button () at the bottom of the screen.
  3. Import log data into the Offset Well and Current Well charts. See "Importing data".
  4. Use the Extrapolation tool () to add a density trend line to the Offset Well and Current Well charts. See "Density trend lines".

  5. When you are finished editing the experiment, click the Save Changes button ().

The calculated gradients for overburden and hydrostatic pressure will be updated in the Results chart. The imported data and density trend line will be saved to the experiment.

Importing data

On the Overburden tab, there are four types of log data you can use to calculate overburden and hydrostatic pressure: density, sonic, seismic, and check shots. However, seismic and check shot data can only be imported into the Offset Well chart.

You can import LAS and CSV files into the Overburden charts. The import data must be formatted into vertical columns, with the column names in the first row. The options listed for the Value axis are automatically determined by Victus™ Simulation, based on the column names in the first row of the import file.

To import Overburden data:

  1. Select the experiment you want to edit, then click the Edit button ().

  2. Go to the chart you want to import data into, then click the Import button ().
  3. Navigate to the file you want to import and click Open. A list of import options will appear on top of the graph.
  4. Select the options for the TVD data in the file you are importing.
    • The TVD axis is the name of the column for TVD.
    • The TVD unit is the measurement unit used for TVD.
  5. Select the options for the type of log data you want to import.
    • The Value axis is the name of the column that contains the log data.
    • The Log type is type of log data you're importing (e.g. density or sonic).
  6. Select the options for the measurement unit.
    • The Value unit type is the type of measurement unit used for the log data.
    • The Value unit is the measurement unit used for the log data.
  7. Change the default line and symbol options as desired.
  8. Click the OK button () to save your changes.

The log data will be imported into Victus™ Simulation, and the data will be graphed in the selected chart. When you are finished editing the experiment, click the Save Changes button ().

Using the Overburden tab

When you are editing an experiment, the toolbar at the bottom of the Overburden tab contains tools for refining the data in the charts. To unlock an experiment for editing, click the Edit button ().

Toolbar on the Overburden tab
Button Function
Turns off the selected tool so that the mouse doesn't have any special functions
Adds an area of interest to the selected chart. See "Boundary areas".
Adds an extrapolated density trend line to the selected chart. See "Density trend lines".
Allows you to edit the settings for the selected chart or an item on a chart
Deletes the selected item from a chart
Deletes all items from the selected chart
Locks the experiment for editing, without saving any changes
Locks the experiment for editing and saves all changes

Each chart also has its own set of toolbar buttons at the bottom of the chart.

Chart toolbar on the Overburden tab
Button Function
Opens the calculation settings for the Results chart. See "Overburden calculation settings".
Imports log data into the Offset Well or Current Well charts. See "Importing data".
Exports the data for the selected chart into a CSV file
Switches between the gradient and pressure views in the Results chart.
