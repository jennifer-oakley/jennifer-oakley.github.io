About deliver lines

On the Lithology tab, the Deliver Line tool allows you to manually define shale areas. Deliver lines are used to set the minimum ROP and gamma ray values found in shale formation layers, creating a threshold that separates shale areas from the rest of the formation. The log values in a shale area should exceed this threshold, while the log values in other formation layers should be below the threshold.

When you draw a deliver line, shale areas should be on the right side of the line. Non-shale areas should be on the left side of the line. You can add multiple deliver lines to the Offset Well and Current Well charts for ROP and gamma ray logs. However, deliver lines can not overlap within the same chart.

Adding deliver lines

To add a deliver line:

  1. Select the experiment you want to edit, then click the Edit button ().
  2. Go to the Lithology tab.
  3. Click the Deliver Line button () in the bottom toolbar.
  4. Go to the graph that you want to add a deliver line to.
  5. Locate the starting and ending TVD of the shale area in the graph.
  6. Drag your mouse from the starting TVD to the ending TVD.

The shale areas identified by the deliver line will appear in the Results chart. When you are finished editing the experiment, click the Save Changes button ().

Editing deliver lines

To edit a deliver line:

  1. Click the Edit button (), then click on the deliver line you want to edit.
  2. In the window that appears, update the values for the line's start point as needed.
    • TVD 1 is the TVD of the start point.
    • Value 1 is the minimum value for a shale area at the TVD of the starting point.
  3. Update the values for the line's end point as needed.
    • TVD 2 is the TVD of the end point.
    • Value 2 is the minimum value for a shale area at the TVD of the end point.
  4. Click the Enter button () to update the graph with your changes.

The deliver line will move to the new position. The shale areas identified by the deliver line will be updated in the Results chart. When you are finished making changes, click the Close button () to close the window.
