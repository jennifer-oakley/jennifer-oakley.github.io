Viewing analysis charts

After you run A Displacement analysis, the Calculate button () in the bottom toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

To view the charts for A Displacement experiment:

  1. Go to the Displacement tab on the Tools > Hydraulics page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Graphs button () in the bottom toolbar.

The charts will appear on top of the analysis settings. To close the analysis charts, click the Show Graphs button () again.

Viewing chart legends

By default, the chart legends are not shown. To pin the legends to the charts:

  1. Hover your mouse over the bottom right corner of the graph index, on the right side of the analysis charts.
  2. In the toolbar that appears, click the Show/Hide Legend button ().

The legend for each chart will appear in the top right corner of the graph.

List of analysis charts

The charts listed below are available after you run an analysis on the Displacement tab.

Analysis charts for a displacement analysis
Chart Name Description Chart Sample
Flow Shows the changes to the flow rate for fluids entering and exiting the well.
SPP Shows the changes in standpipe pressure caused by displacement.
ECD Shows the changes in ECD equivalent circulating density at the bottom of the wellbore and at the casing shoe.
SBP Shows the changes in surface backpressure caused by displacement.
BHP Shows the changes in bottomhole pressure caused by displacement
DOI - Annulus Pressure Shows the changes in annular pressure at the depth of interest
DOI - Pipe Pressure Shows the changes in pipe pressure at the depth of interest
DOI-Differential Pressure Shows the changes in differential pressure at the depth of interest
Fluid Positions Shows the changes in the fluid depth caused by displacement
