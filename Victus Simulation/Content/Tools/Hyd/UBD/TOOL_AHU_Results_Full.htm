Viewing detailed results

After you run An Underbalanced Drilling analysis, the Show Full Results button () will appear next to the Show Graphs button () in the bottom toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the detailed analysis results.

To view the detailed results for AnUnderbalanced Drilling experiment:

  1. Go to the UBD tab on the Tools > Hydraulics page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Full Results button () in the bottom toolbar.

The detailed results will appear on top of the analysis settings. To close the detailed results, click the Show Full Results button () again.

Split flow components

The Split Flow Components tab contains additional calculations for the split flow components in the work string.

Detailed results for split flow components
Column Definition
Tool Name of the configurable tool
Flow Path Position of the nozzles on the tool. The options available vary, depending on the type of tool.
Port State State of the ports on the component
Nozzle Config. Number and size of the nozzles on the tool. This value is automatically populated from the tool properties on the Setup > Work String > Drillstring tab.
TFA Total flow area. This value is automatically populated from the tool properties on the Setup > Work String > Drillstring tab.
Discharge Efficiency of the fluid discharge through the component. This value is automatically populated from the pump properties on the Setup > Surface Equipment page.
Flow Split Percentage of the flow that has split to the annulus
Noz. P. Drop Pressure loss through the tool nozzles
Jet Vel Jet velocity
Impact Force Total impact force of the fluid exiting the tool nozzles
Impact Pressure Impact pressure of the fluid exiting each tool nozzle
Power Hydraulic power of the flow through the port on the tool
Power Expended Power expended across a cross section of the tool
Int Temp Internal temperature
Ext Temp External temperature
Diff. Pressure at Top Differential pressure at the top of the tool
Max Press Maximum pressure required to override the configurable component
Flow Rate @ Max Press Flow rate at which the configurable component will override
SPP @ Max Press Standpipe pressure at which the configurable component will override

Column Definition

Annulus Volume Volume of the fluid in the annulus interval

Influx zone calculations

The Kick Zones table contains calculated values for each influx zone included in the analysis. If there were no influx zones included in the analysis, there will not be any data in this table.

Calculated values for an influx zone
Column Definition
Zone Name of the influx zone
Temperature Formation temperature of the influx zone
Pressure Annular pressure in the influx zone
Density Density of the underbalanced fluid as it moves through the influx zone
Flow Rate Flow rate of the underbalanced fluid as it moves through the influx zone
