Viewing analysis charts

After you run A Parametric snapshot analysis, the Calculate button () in the bottom toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

To view the charts for A Parametric snapshot experiment:

  1. Go to the Parametric tab on the Tools > Hydraulics page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Graphs button () in the bottom toolbar.

The charts will appear on top of the analysis settings. To close the analysis charts, click the Show Graphs button () again.

Viewing chart legends

By default, the chart legends are not shown. To pin the legends to the charts:

  1. Hover your mouse over the bottom right corner of the graph index, on the right side of the analysis charts.
  2. In the toolbar that appears, click the Show/Hide Legend button ().

The legend for each chart will appear in the top right corner of the graph.

Working with charts

All of the charts for parametric analysis are plotted on the same chart. The chart that is currently shown depends on which chart is selected in the Dependent Parameter field. Each chart is plotted against two of the variable operating parameters selected in the analysis settings. All of the analysis charts have three axes:

  • The Independent Parameter (X) is the graph index. The data in every analysis chart is plotted against the data for the graph index. The graph index can be any of the variable parameters selected in the analysis settings, except drillstrings or fluids.
  • The Dependent Parameter (Y) contains the chart data. The chart data is plotted against the parameter selected in the Horizontal Axis field. The chart selected in the Vertical Axis field is the chart that is currently shown.
  • The Category Axis (Z) slices the chart data. The chart will show one line for each value of the selected operating parameter. The category axis can be any of the variable parameters selected in the analysis settings, including drillstrings or fluids.

Sliders will appear for remaining variable operating parameters. You can use the sliders to adjust the values of the parameters.

List of analysis charts

The charts listed below are available after you run an analysis on the Parametric tab.

Analysis charts for an parametric analysis
Chart Name Description Chart Sample
Bed Height Shows the effect of the selected operating parameter on the height of cuttings bed.
Bit HHP Shows the effect of the selected operating parameter on the hydraulic horsepower of the drill bit.
Bit Pressure Shows the effect of the selected operating parameter on the pressure at the bit.
Cuttings Shows the effect of the selected operating parameter on the concentration of the cuttings in the fluid.
ECD+ Cuttings DOI Shows the effect of the selected operating parameter on ECD and cuttings density at the depth of interest.
ECD+ Cuttings Shoe Shows the effect of the selected operating parameter on ECD and cuttings density at the casing shoe.
ECD+Cuttings TD Shows the effect of the selected operating parameter on ECD and cuttings density at total depth.
Pump HHP Shows the effect of the selected operating parameter on the hydraulic horsepower of the pump.
SPP Shows the effect of the selected operating parameter on standpipe pressure
WS Pressure Shows the effect of the selected operating parameter on the pressure in the drillstring.
