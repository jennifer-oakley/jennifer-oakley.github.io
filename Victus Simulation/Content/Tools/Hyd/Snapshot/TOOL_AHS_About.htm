About snapshot analysis

On the Tools > Hydraulics > Snapshot tab, you can run a snapshot hydraulics analysis. A snapshot analysis gives you a quick summary of the fluid's predicted behavior in the wellbore, based on a single value for each operating parameter. It calculates several depth profiles that predict how the fluid properties and wellbore pressures will change throughout the well.

There are several factors that influence the effectiveness of a hydraulics program, including:

  • The rheological properties of a fluid, such as plastic viscosity, yield point, and low-shear yield point.
  • The velocity at which the fluid circulates through the drillstring and the annulus.
  • The equivalent circulating and static densities of the fluid.
  • The pressure in the drillstring and the annulus.
  • The temperature of the fluid in the drillstring, annulus, and formation.
  • The ability to effectively remove cuttings from the well.

Each well is unique, which means that the requirements for the fluid are unique to a specific well and the fluid being used. The properties of a fluid can positively impact one aspect of drilling a well, but negatively impact another. For this reason, it is important to find a balance that allows you to maximize hole cleaning, minimize pump pressures, avoid influxes, and prevent circulation losses to the formation.

Note

When the Choke Kill Lines option is selected for the fluid return path, the calculated value for flow in may exceed the flow rate specified in the analysis settings.

Using the Snapshot tool

The toolbar on the right side of the screen allows you to compare, add, and remove Snapshot experiments. The toolbar at the bottom of the screen allows you to begin an analysis and view the result charts.

Toolbar on the Snapshot tab
Button Function

Begins Snapshot calculations

Adds a new Snapshot analysis experiment

Deletes the current experiment, including all analysis results

Opens the Snapshot Experiments window, which allows you to compare multiple Snapshot experiments

Opens the analysis charts for the current experiment

Opens a detailed summary of the analysis results. See "Detailed snapshot analysis results".
Generates a report of the Snapshot calculation results. See "Creating reports".

When you are viewing analysis charts, additional buttons will appear in the bottom toolbar. When you hover over the graph index, next to the analysis charts, another toolbar with additional options will appear.

Chart toolbar on the Snapshot tab
Button Function

Closes the analysis charts and returns to the analysis settings

Pins the chart legends to the top of the analysis charts

Exports all chart data to a CSV file

Generates a one-page report of the analysis charts currently displayed on the Tools > Hydraulics > Snapshot tab

Opens the Data Browser window, which lists the results for the calculated depth profile

Adds an analysis chart to the charts view
