Swab and surge analysis settings

In this article

Starting the analysis

Field definitions

Starting the analysis

To run a swab and surge analysis:

  1. Go to the Surge/Swab tab on the Tools > Hydraulics page.
  2. Select the well interval, work string, and circulating fluid you want to use for the analysis.
  3. In the Run Direction field, select the trip direction.
    • For tripping in, select Downwards (Surge).
    • For tripping out, select Upwards (Swab).
    • For tripping in and out, select Both (Surge/Swab).
  4. In the Pipe End field, select the pipe configuration for the trip.
  5. Enter the remaining operating parameters for the analysis.
  6. Go to the Advanced Settings section to update the trip information and other advanced settings. See "Field definitions".
  7. To begin the analysis, click the Calculate button () in the lower right corner of the screen.

After you run A Swab and surge analysis, the Calculate button () in the bottom toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

If you want to re-run the analysis with different settings, just update the experiment settings as needed and click the Calculate button (). When you edit the experiment settings, the existing results will be erased and the Calculate button will reappear.

Field definitions

The table below defines the settings required for a swab and surge analysis on the Tools > Hydraulics > Surge/Swab tab.

Settings for a swab and surge analysis
Section Field Definition
-- Well Interval

Well interval used for the analysis
-- Work String

Drillstring or casing string used for the analysis. Work strings that contain a tool with 0 ID cannot be used in a hydraulics analysis.
-- Fluid

Circulating fluid used for the analysis
Operating Parameters Run Direction

Type of trip, based on the trip direction.

  • Downwards (Surge) – Calculates surge effects when tripping in.
  • Upwards (Swab) – Calculates swab effects when tripping out.
  • Both (Surge/Swab) – Calculates surge effects when tripping in, as well as swab effects when tripping out.
Pipe End

Pipe configuration for the trip.

  • Open-Ended – Allows fluid to enter and exit the drillstring during the trip.
  • Closed-End – Does not allow fluid to enter and exit the drillstring during the trip.
Flow Rate Flow rate at the pump. Only available for swab analysis.
SBP

Surface backpressure at the choke that's applied to the annulus
Analysis Type

Option for optimizing running speed.

  • Running Speed Optimization – Calculates the optimum running speed, using the value in the Max Allowable Speed field.
  • Fixed Running Speed – Uses a constant running speed for the entire trip, using the value in the Running Speed field.
Max. Allowable Speed Maximum running speed during the trip. Only available when the Running Speed Optimization option is selected.
Running Speed Fixed running to use for the analysis. Only available when the Fixed Running Speed option is selected.
Advanced Settings Tolerance Method

Method for determining allowable ECD equivalent circulating density.

  • Use PPG/FPG – Uses the formation's pore pressure and fracture gradients to determine allowable ECD.
  • Use Ref. ECD/Ref. Depth – Uses the ECD at a reference point to determine allowable ECD.
  • Both – Uses both of the methods above to determine allowable ECD.
Reference Depth MD at the reference point for ECD. Not available when using the Use PPG/FPG tolerance method.
ECD at Ref. Depth - Surge ECD equivalent circulating density for surge analysis at the reference point. Not available when using the Use PPG/FPG tolerance method.
ECD at Ref. Depth - Swab ECD equivalent circulating density for swab analysis at the reference point. Not available when using the Use PPG/FPG tolerance method.
Acceleration Time Time required to reach the running speed
Connection Time Time required to make or break a connection
Stand Length Length of one stand of pipe
