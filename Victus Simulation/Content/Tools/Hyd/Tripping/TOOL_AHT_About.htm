Using the Surge/Swab tab

About surge/swab analysis

On the Tools > Hydraulics > Surge/Swab tab, you can run a swab and surge analysis. Swab and surge analysis predicts the swab and surge effects on pressure when running the bit in or out of the well, for open or closed systems. You can also use the analysis to calculate an optimal running speed profile that does not violate the thresholds for pore pressure or fracture pressure.

When the work string is pulled out of the well, the fluid falls to replace the volume of the pipe removed from the well. In a closed system, the fluid falls from the annulus. In an open system, the fluid falls from the pipe and the annulus. This is known as swabbing, which causes pressure to decrease while the mud fills the volume of the displaced pipe.

When the work string is run into the well, fluid is displaced from the well by the volume of the pipe added to the well. This is known as surging, which causes pressure to increase due to induced flow from the displaced mud in the annulus.

Swab and surge pressures are affected by many factors, including the fluid's rheological properties, the running speed and acceleration at which the pipe enters or exits the well, the dimensions of the annulus, and the length of the drillstring in the well.

Using the Surge/Swab tab

The toolbar on the right side of the screen allows you to add and remove Swab and surge experiments. The toolbar at the bottom of the screen allows you to begin an analysis and view the result charts.

Toolbar on the Surge/Swab tab
Button Function

Begins Swab and surge calculations

Adds a new Swab and surge analysis experiment

Deletes the current experiment, including all analysis results
Generates a report of the Swab and surge calculation results. See "Creating reports".

Opens the analysis charts for the current experiment

When you are viewing analysis charts, additional buttons will appear in the bottom toolbar. When you hover over the graph index, next to the analysis charts, another toolbar with additional options will appear.

Chart toolbar on the Surge/Swab tab
Button Function

Closes the analysis charts and returns to the analysis settings

Pins the chart legends to the top of the analysis charts

Exports all chart data to a CSV file

Generates a one-page report of the analysis charts currently displayed on the Tools > Hydraulics > Surge/Swab tab

Opens the Data Browser window, which lists the results for the calculated depth profile
