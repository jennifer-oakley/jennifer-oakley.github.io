About flow meter configuration

The following instructions show you how to connect different flow meters, such as high pressure flow meters, to Victus™ Simulation. These configurations allow Victus™ Simulation to use flow data to obtain rheology information from the high pressure flow meter's rheometer, calculate pump stroke rates, or measure drilling fluid flow rates.

Connecting flow meters to spare sensors

To connect a flow meter or another flow measurement device to a spare sensor:

  1. Go to the Piping and Instrumentation tab on the Setup > Surface Equipment page.
  2. Click the Table View button () to view the list of equipment on the rig.
  3. In the equipment list, select the row for the pump you want to connect to a sensor. A panel with the pump properties will appear next to the equipment list.
  4. In the Pump Source field, select the spare sensor that the flow meter is connected to.
  5. In the equipment list, select the row for Instrumentation.
  6. In the panel that appears, go to the sensor that the flow meter is connected to and select the Flow option.
  7. Click the Save Changes button () to save your changes.

Victus™ Simulation will use the flow readings from the flow meter to calculate the stroke rate for that pump. The calculated stroke rate will be used to determine the total flow rate from all pumps. After you connect a flow meter to a spare sensor, you'll need to calibrate the flow meter readings on the Setup > Calibration > Default Sensors tab. See "Default sensors".

Connecting high-pressure flow meters to pumps

To connect a high-pressure flow meter to a main or auxiliary pump:

  1. Go to the Setup > Surface Equipment > Equipment Configuration tab.
  2. In the Main Flow In Source field, select the main high-pressure flow meter the pump will connect to.
  3. Enable the HPFM 1 or HPFM 2 box according to the high-pressure flow meter option selected in the previous step.
  4. Click the Save Changes button () to save your changes.

Victus™ Simulation will read the rate at which high-flow drilling fluids are flowing through the wellbore from all pumps.

Connecting the HPFM rheometer

To start reading the rheology data from the high-pressure flow meter's rheometer:

  1. Go to the Setup > Surface Equipment > Equipment Configuration tab.
  2. In the Main Rheometer Source field, select the Main HPFM 1 Rheometer or Main HPFM 2 Rheometer option.
  3. Enable the HPFM 1 or HPFM 2 box according to the high-pressure flow meter option selected in the step 2.
  4. Click the Save Changes button () to save your changes.

Victus™ Simulation will use the readings from the high-pressure flow meter's rheometer to obtain the rheology information.
