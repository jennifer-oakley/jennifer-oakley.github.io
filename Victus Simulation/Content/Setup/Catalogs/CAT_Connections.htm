Connection catalog

The table on the Setup > Catalogs > Connections tab lists the properties for all of the work string connections that are available in Victus™ Simulation.

Catalog properties of connections
Column Name Definition
Connection Name of the connection
Taper Amount that the connection tapers at the pin end of a connection
Relief groove Ø Diameter of the stress relief groove, if present
Pitch Ø at gauge point Diameter at which the distance across the threads is equal to the distance between the threads
Recommended make up stress Maximum recommended makeup stress for a connection
Thread height Distance between the crest and the root of the thread
Root truncation Clearance between the root of the thread and the truncation of the root
Nominal counterbore Nominal size of the box end of a connection
Box Bore Back Ø Cylinder diameter of a boreback contour on a box connection
Length of the pin Length of the pin end of a connection
Lead of thread Distance from a point on a thread to the corresponding point on the next thread
Mating surface friction coeff Coefficient representing the surface friction between the plug and ring members of a gauge set
1/2 included angle of thread Angle between the root and the crest of a thread
