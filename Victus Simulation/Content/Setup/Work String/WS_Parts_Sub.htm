About subs

The Victus™ catalog contains several types of rotary subs and circulation subs, including:

  • Crossover subs, which are used to connect drillstring parts with different connections or sizes.
  • Bit subs, which are located directly above the drill bit.
  • Saver subs, which are used to absorb connection wear between the drillstring and the top drive, or between the drillstring and the kelly.
  • Bottle-neck crossover subs, which are used to connect a tool with a larger connection to a tool with a smaller connection.
  • UBHO Universal Bottom Hole Orientation subs, also known as a muleshoe sub, are used in directional drilling to establish the high side of the tool.
  • Float subs, which contain a float valve to prevent drilling mud and cuttings from flowing back up the drillstring.
  • Filter subs, which are used to prevent nozzles and motors from clogging with debris.
  • Ball lock subs are ball-activated circulation subs that are locked open when running and circulating.

Adding subs

To add a sub to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the numbered tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Sub option.
  5. In the Type field, select the type of sub.
  6. In the Tool Size field, select the nominal size of the sub.
  7. Update the remaining properties of the sub as needed. See "Properties of subs".
  8. Click the OK button () to add the sub to the work string.

The sub will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of subs

The table below defines the basic properties for subs in Victus™ Simulation. To view the advanced properties of a sub, click the Show/Hide button ().

Properties of subs
Section Field Definition
-- Type

Type of sub
-- Tool Size

Nominal size of the sub
General Info Description

Name or description of the sub
Length

Total length of the sub. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the sub.

  • Calculated – The mass and adjusted weight of the sub are calculated, based on the dimensions of the sub. When the dimensions of the sub are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the sub is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the sub. When the dimensions of the sub are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the sub is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the sub. When the dimensions of the sub are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the sub. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the sub. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
NW

Nominal weight of the sub. This value is calculated automatically from the length and mass of the sub.
Material

Symbol for the material that the sub is composed of and the grade of the material.
-- TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value. Only available for ball lock subs.
-- Efficiency Efficiency of the nozzles on the sub, relative to the maximum possible performance. Only available for ball lock subs.
-- Nozzles

Number of nozzles on the sub. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles. Only available for ball lock subs.
-- Nozzle Diameter

Size of each nozzle on the sub. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field. Only available for ball lock subs.
Connection Data Connection Configuration

Configuration of the connections on the sub.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection.
Bottom Connection Thread Type Type of threads used on the bottom connection.
