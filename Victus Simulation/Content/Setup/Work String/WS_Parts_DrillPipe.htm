About drill pipes

Drill pipes are composed of three uniform pipe sections: the top connection, the body, and the bottom connection. Each uniform section has the same OD, ID, and nominal weight across the length of the section. In a drillstring, drill pipe is the primary drillstring component from the top of the BHA to the top of the well. Victus™ Simulation automatically replicates the last component in a work string to the top of the well, so a drill pipe component should always be the last component in a drillstring.

The part sections for most drill pipes are already defined in the built-in Victus™ catalog. If the catalog doesn't contain the part you need, you can create a custom drill pipe. See "Creating custom drill pipes".

Adding drill pipe

To add a drill pipe:

  1. Select a work string using the numbered tabs on the right side of the screen, then select the component below the one you want to add.
  2. Click the Insert Tool button ().
  3. In the panel that appears on the right, go to the Tool field and select the Drill Pipe option.
  4. In the Catalog field, select catalog that contains the drill pipe you want to use.
  5. In the Range field, select the range for the length of the pipe.
    • Range 1 – Initial mean length is 17 to 23 ft.
    • Range 2 – Initial mean length is 26 to 32 ft.
    • Range 3 – Initial mean length is 41 to 46 ft.
  6. Select the nominal size and weight in the Tool Size and NW fields.
  7. In the Grade and Upset Type field, select the material grade and upset on the drill pipe.
  8. In the Connection Type field, select the type of connection on the drill pipe. The list of available connections will be filtered by your previous selections.
  9. In the Service Class field, select the appropriate wear class for the pipe.
  10. Click the OK button () to add the drill pipe to the work string.

The mass of the drill pipe will be calculated, based on the selected service class. The drill pipe will be added to the work string, and the work string preview will be updated in the Well View pane.

Creating custom drill pipes

To add a custom drill pipe:

  1. Select a work string using the numbered tabs on the right side of the screen, then select the component below the one you want to add.
  2. Click the Insert Tool button ().
  3. In the panel that appears on the right, go to the Tool field and select the Drill Pipe option.
  4. In the Catalog field, select the Generic option.
  5. In the Tool Size field, select the nominal size of the part.
  6. In the Description field, enter a brief description of the part.
  7. Go to the Connections section, and update the connections as needed.
  8. Enter the number of part sections in the Sections field, then enter the dimensions of each section.
  9. Click the OK button () to add the custom part to the work string.

The drill pipe will be added to the work string, and the work string preview will be updated in the Well View pane. The custom drill pipe will be added to the Generic catalog.

Tip

If the Generic catalog already contains a custom part for the selected category, click the Edit button () at the bottom of the panel to create a new custom part.

Properties of standard drill pipes

The table below defines the basic properties for drill pipes in Victus™ Simulation. To view the advanced properties of A drill pipe, click the Show/Hide button ().

Basic properties of standard drill pipes
Section Field Definition
-- Type

Type of drill pipe
-- Range

Range representing the initial mean length of a drill pipe.

  • Range 1 – Initial mean length is 17 to 23 ft.
  • Range 2 – Initial mean length is 26 to 32 ft.
  • Range 3 – Initial mean length is 41 to 46 ft.
-- Tool Size

Nominal size of the drill pipe
-- NW Nominal weight of the drill pipe
-- Grade and Upset Type Material grade and upset type
-- Connection Type Type of connection on the part
General Info Description

Name or description of the drill pipe
Service Class

Service classification for the amount of pipe wear.

  • New – Drill pipe has never been used, with 100% wall thickness.
  • Premium – Drill pipe has 80% of its original wall thickness. Tensile strength and torsional yield strength are also reduced to 80%.
  • Premium (Reduced TSR) – Drill pipe has 80% of its original wall thickness. Tensile strength and torsional yield strength are also reduced to 50%.
  • Class 2 – Drill pipe has 70% of its original wall thickness. Tensile strength and torsional yield strength are also reduced to 70%.
Length

Total length of the drill pipe. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the drill pipe.

  • Calculated – The mass and adjusted weight of the drill pipe are calculated, based on the dimensions of the drill pipe. When the dimensions of the drill pipe are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the drill pipe is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the drill pipe. When the dimensions of the drill pipe are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the drill pipe is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the drill pipe. When the dimensions of the drill pipe are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the drill pipe. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the drill pipe. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Material

Symbol for the material that the drill pipe is composed of and the grade of the material.
Connection Data Thread Type

Thread type for both connections on the drill pipe.

When you click the Show/Hide button (), the following advanced properties for the drill pipe will appear.

Advanced properties of standard drill pipes

Outside diameter of the part section.

Inside diameter of the part section.

NW
