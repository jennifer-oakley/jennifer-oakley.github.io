About RSS units

RSS bias units are used to drill a directional well with continuous rotation from the surface. A standard configuration for an RSS BHA includes a pivot stabilizer, a bias unit, a barrel collar, a stabilized IDS, and a HEL suite, as well as a roller reamer or string stabilizer. The BHA for a motorized RSS configuration varies, but generally includes a pivot stabilizer, a bias unit, a barrel collar, and a mud motor.

The specifications for most Weatherford RSS units are included in the database for Victus™ Simulation. When you select an RSS unit, the specifications will appear in the Drillstring Component panel. Most of these properties are provided for your reference, and can't be edited.

Pivot stabilizers

Pivot stabilizers are near-bit stabilizers that are only used with RSS units. They provide a fulcrum point for the deflection of the RSS and stabilize the bit to minimize vibration and maximize bit life.

There are three factors to consider when selecting a pivot stabilizer: the RSS bias unit that will be used, the hole size, and the configuration of the pivot stabilizer.

Pivot stabilizers have two possible configurations: fishneck down and fishneck up. Fishneck down is the standard configuration, where the fishneck is next to the bit and the stabilizer blades are next to the bias unit. Fishneck up is an inverted configuration, with the stabilizer blades next to the bit and the fishneck next to the bias unit.

In Victus™ Simulation, the names of the pivot stabilizers in the built-in catalog use the following format: [Bias Unit Model] [Hole Size] [Pivot Configuration] RSS Pivot Stab. The hole size is given in inches, with the slash removed from the fraction (e.g. 6 1/2" hole size is formatted as "6-12").

For example, if you are using a 4.75" RSS unit in a 5.75" hole, there are two possible pivot stabilizers in the catalog:

  • 475 5-34 Fishneck UP RSS Pivot Stab
  • 475 5-34 Fishneck DOWN RSS Pivot Stab

Adding a Revolution© RSS unit

To add a Revolution© RSS unit:

  1. Select a work string using the numbered tabs on the right side of the screen.
  2. Select the component below the RSS unit you want to add.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the RSS option.
  5. In the Type field, select the type of Revolution© RSS tool.
  6. In the Tool Size field, select the nominal size of the bias unit.
  7. In the Pivot Stabilizer field, select the configuration for the pivot stabilizer attached to bias unit. See "Pivot stabilizers".
  8. In the Bias Unit field, select the configuration for the bias unit.
  9. In the Controller Collar field, select the controller collar used with the bias unit.
  10. Click the OK button () to add the RSS unit to the drillstring.

The Revolution© RSS unit will be added to the work string, and the work string preview will be updated in the Well View pane.

Adding a Magnus™ RSS unit

To add a Magnus™ RSS unit:

  1. Select a work string using the numbered tabs on the right side of the screen.
  2. Select the component below the RSS unit you want to add.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the RSS option.
  5. In the Type field, select the Magnus™ RSS option.
  6. In the Tool Size field, select the nominal size of the bias unit.
  7. In the Tool Config field, select the configuration of the collars on the RSS unit.
  8. In the Power Setup field, select the configuration of the turbine and batteries used to power the RSS unit.
  9. Updated the remaining properties as needed. See "Properties of Magnus™ RSS units".
  10. Click the OK button () to add the RSS unit to the drillstring.

The Magnus™ RSS unit will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of Revolution® RSS units

The table below defines the basic properties for Revolution® RSS units in Victus™ Simulation. To view the advanced properties of A Revolution® RSS unit, click the Show/Hide button ().

Basic properties for Revolution® RSS units
Section Field Definition
-- Type

Type of RSS unit
-- Tool Size

Nominal size of the RSS unit
General Info Description

Name or description of the RSS unit
Length

Total length of the RSS unit. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the RSS unit.

  • Calculated – The mass and adjusted weight of the RSS unit are calculated, based on the dimensions of the RSS unit. When the dimensions of the RSS unit are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the RSS unit is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the RSS unit. When the dimensions of the RSS unit are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the RSS unit is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the RSS unit. When the dimensions of the RSS unit are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the RSS unit. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the RSS unit. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Orifice Angle Nozzle size of the pulser. Only available when a DualPulser collar is selected.
Dimensional Data Pivot Stabilizer Size and configuration of the pivot stabilizer
Stab. Length Length of the pivot stabilizer
Blade OD Outside diameter of the stabilizer blades
-- Bias Unit Configuration of the bias unit.
-- Controller Collar Configuration of the controller collar used with the RSS unit.
-- Collar Length Length of the controller collar.
-- Connection Configuration

Configuration of the connections on the RSS unit.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Sensor Data Sensor <Number> Description of the sensor attached to the controller collar. There are two types of sensors possible: near-bit inclination (NBI) and near-bit gamma (NBG). The list of sensors is populated automatically from the Victus™ catalog, based on the selected controller collar. The checkbox next to the sensor name indicates that the sensor is active on the RSS unit.
Sensor Offset Distance between the sensor and the bottom of the RSS unit.
Top Connection Thread Type

Type of threads used on the top connection
Bottom Connection Thread Type

Type of threads used on both connections
