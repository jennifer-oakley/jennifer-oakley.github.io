About custom parts

The Generic Victus™ catalog is a local catalog that stores custom, user-created drillstring parts. When you create a tool using the Generic catalog, it will be permanently added to the catalog so that it can be used again in the future. Currently, the Generic catalog contains five types of drillstring parts: drill bits, drill collars, drill pipe, HWDP, and custom tools.

If none of the part categories in Victus™ Simulation apply to the work string part you want to add, you can also create a custom part in the Generic catalog. Custom tools can be used to create virtually any type of work string component. However, custom tools should only be used if there isn't a more appropriate tool type in the Victus™ catalog. See "Adding other custom tools".

Warning

Do not use a custom tool to create a drill bit. Victus™ Simulation will not recognize the tool as a bit. To create a custom drill bit, use the Bit option under the Generic catalog. See "Creating a custom drill bit".

Adding other custom tools

To add a custom tool to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the numbered tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Custom option.
  5. In the Catalog field, select the Generic option.
  6. In the Type field, select the type of custom tool.
  7. In the Tool Size field, select the nominal size of the custom tool.
  8. In the Description field, select the name of the custom tool you want to use.
  9. Update the remaining properties of the custom tool as needed. See "Custom tool properties".
  10. Click the OK button () to add the custom tool to the work string.

The custom tool will be added to the work string, and the work string preview will be updated in the Well View pane.

Custom tool properties

The table below defines the properties for custom tools in Victus™ Simulation.

Properties of custom parts
Section Field Definition
-- Tool Size

Nominal size of the custom tool
-- Description

Name or description of the custom tool
General Info Length

Total length of the custom tool. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the custom tool.

  • Calculated – The mass and adjusted weight of the custom tool are calculated, based on the dimensions of the custom tool. When the dimensions of the custom tool are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the custom tool is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the custom tool. When the dimensions of the custom tool are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the custom tool is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the custom tool. When the dimensions of the custom tool are updated, the value for mass will not be affected.
Mass

Total mass of the custom tool. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Adjusted Weight

Total adjusted weight of the custom tool. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
NW

Nominal weight of the custom tool. This value is calculated automatically from the length and mass of the custom tool.
Material

Symbol for the material that the custom tool is composed of and the grade of the material.
Hydraulics Pressure Drop Coefficients (checkbox)

Indicates that fluid flows through the custom tool. When selected, additional fields will appear.
Hyd. Model

Method for calculating pressure loss through the custom tool
Pressure Drop

Pressure loss limits that apply to the custom tool. Only available for linear pressure loss calculations.

Flow Rate

Flow rate limits that apply to the custom tool. Only available for linear pressure loss calculations.

Coefficient A

Fluid density exponent, based on the flow characteristics of the custom tool. Only available for Power Law pressure loss calculations.

Coefficient B

Scaling constant, based on the flow characteristics of the custom tool. Only available for Power Law pressure loss calculations.

Coefficient C

Flow rate exponent, based on the flow characteristics of the custom tool. Only available for Power Law pressure loss calculations.
Nozzles (checkbox) Indicates that nozzles are present on the custom tool. When selected, additional fields will appear
TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
Efficiency Efficiency of the nozzles on the custom tool, relative to the maximum possible performance
Nozzles

Number of nozzles on the custom tool. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
Nozzle Diameter

Size of each nozzle on the custom tool. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Sensors (checkbox) Indicates that sensors are present on the custom tool. When selected, additional fields will appear. The number of sensors shown depends on the number entered in the Sensors field.
Sensors Number of sensors on the tool
Sensor Name Description of the sensor. The checkbox next to the name of the sensor indicates that the sensor is active on the custom tool.
-- Connection Configuration

Configuration of the connections on the custom tool.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection.
MUT

Make-up torque of the top connection
Tensile Capacity

Tensile strength of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection.
MUT

Make-up torque of the bottom connection
Tensile Capacity

Tensile strength of the bottom connection
-- Sections

Number of part sections in the custom tool.
Dimensional Data OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Material

Symbol for the material that the part section is composed of and the grade of the material.
Caption

Description of the part section.
NW

Nominal weight of the part section. This value is calculated automatically from the length and mass of the part section.
Max Contact Force

Maximum allowable contact force for the section.
Max Von Mises Stress

Maximum allowable effective stress for the section.
Stabilizer Indicates that a stabilizer is included in the part section
Blade OD Outside diameter of the stabilizer blades. Only available for part sections containing a stabilizer.
Blade Width Width of the each stabilizer blade. Only available for part sections containing a stabilizer.
Blade Angle Angle between the blade and the stabilizer body. Only available for part sections containing a stabilizer.
Blade Count Number of blades on the custom tool. Only available for part sections containing a stabilizer.
Junk Slot Area Area between stabilizer blades. This value is calculated automatically. Only available for part sections containing a stabilizer.
Taper Angle

Taper angle between the stabilizer blades and the body of the stabilizer
Mid-Blade Offset

Distance between the middle of the stabilizer blade and the bottom of the stabilizer
