About casing pipe

During a casing operation, casing is run into an openhole section and cemented into place. This stabilizes the wellbore and prevents the formation from caving into the wellbore. The Victus™ catalog includes casing pipe from the API specifications for standard and special clearance casing pipe, as well as casing pipe from specific manufacturers.

Adding casing pipe

To add standard casing pipe to a casing string:

  1. Select a work string using the numbered tabs on the right side of the screen, then select the component below the one you want to add.
  2. Click the Insert Tool button ().
  3. In the panel that appears on the right, go to the Tool field and select the Casing Pipe option.
  4. In the Catalog field, select catalog that contains the casing pipe you want to use.
  5. Select the nominal size and weight in the Tool Size and NW fields.
  6. In the Grade field, select the material grade and upset on the casing pipe.
  7. In the Range field, select the range for the length of the pipe.
    • Range 1 – Initial mean length is 17 to 23 ft.
    • Range 2 – Initial mean length is 26 to 32 ft.
    • Range 3 – Initial mean length is 41 to 46 ft.
  8. In the Connection field, select the type of connection on the casing pipe. The list of available connections will be filtered by your previous selections.
  9. Click the OK button () to add the drill pipe to the work string.

The casing pipe will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of casing pipe

The table below defines the basic properties for casing pipes in Victus™ Simulation. To view the advanced properties of A casing pipe, click the Show/Hide button ().

Basic properties of casing pipes
Section Field Definition
-- Nominal Size

Nominal size of the casing pipe
-- NW Nominal weight of the casing pipe. This value is calculated automatically from the length and mass of the casing pipe.
-- Grade Material grade and upset type
-- Range

Range representing the initial mean length of a casing pipe.

  • Range 1 – Initial mean length is 17 to 23 ft.
  • Range 2 – Initial mean length is 26 to 32 ft.
  • Range 3 – Initial mean length is 41 to 46 ft.
-- Standard API/Non API

Indicates whether or not the casing pipe follows standard API specifications
-- Manufacturer Manufacturer of the casing pipe. Only available for non-API casing pipe and custom casing pipes.
-- Connection

Type of connection on the casing pipe. The list of available connections will be filtered by the selected nominal size and nominal weight.
General Info Description

Name or description of the casing pipe
Length

Total length of the casing pipe. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the casing pipe.

  • Calculated – The mass and adjusted weight of the casing pipe are calculated, based on the dimensions of the casing pipe. When the dimensions of the casing pipe are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the casing pipe is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the casing pipe. When the dimensions of the casing pipe are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the casing pipe is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the casing pipe. When the dimensions of the casing pipe are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the casing pipe. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the casing pipe. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Material Symbol for the material that the casing pipe is composed of and the grade of the material. Only available for custom casing pipes.
Drift

Drift ID, per API 5CT standard specifications or alternate specifications.
Special Drift

Any drift ID other than the drift size listed in the API 5CT specification.
Torque

Torque rating for the casing pipe
Wall Thickness

Thickness of the casing pipe
