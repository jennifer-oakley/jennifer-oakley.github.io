Work string components

In Victus™ Simulation, the Weatherford catalog contains several types of BHA and drillstring components. For more information about customizing specific tools, refer to the topics below.

  • Adjustable Gauge Stabilizers. Adjustable gauge stabilizers are used above the drilling motor to make changes to the build rate and rotary build rate characteristics of the BHA while drilling. See "Adjustable gauge stabilizers".
  • Bits. In addition to the dimensions of the bit, you'll also need to enter the sizes of the nozzles on the bit. See "Drill bits".
  • Casing Pipe. Casing pipe is steel pipe that is cemented into an openhole interval. See "Casing pipe".
  • Custom Tools. If none of the part categories in Victus™ Simulation apply to the work string part you want to add, you can also create a custom part. See "Custom parts".
  • Drill Collars. Drill collars are thick, heavy-walled tubes used between the bit and the drill pipe to provide a pendulum effect for the drillstring and add weight to the bit. See "Drill collars".
  • Drilling Motors. By default, the drilling motors included in the Victus™ catalog do not have any active stabilizers. If you are using a stabilized drilling motor, you'll need to activate the stabilizers. See "Drilling motors".
  • Drill Pipe. Drill pipes are described using uniform pipe section. Each uniform section has the same OD, ID, and nominal weight across the entire length of the section. See "Drill pipes".
  • Ghost Reamers. Ghost reamers are used for reaming, drilling, and hole cleaning in horizontal shale wells and extended-reach wells. They combine a keyseat wiper and a tapered-blade reamer to address unstable formations, swabbing, and hole-cleaning issues. See "Ghost reamers".
  • Hole Openers. When you add a hole opener, you'll also need to set the activation type and opening size. See "Hole openers".
  • HWDP. Heavyweight drill pipe is drill pipe that has thicker walls and longer tool joints, as well as a wear pad in the middle. HWDP is used between the bit and the drill pipe to reduce the chances of drill pipe fatigue or failure. See "HWDPs".
  • Jars. Jars are percussion tools that are operated mechanically or hydraulically to deliver a heavy hammer blow to objects in the wellbore. Jars are used to free objects during drilling or fishing operations. See "Jars".
  • LWD Tools. The Victus™ catalog includes Weatherford's most popular LWD tools, including AZD™/TNP™, MFR™, GuideWave™, HEL™, IDS™, ShockWave™, UltraWave™, and EMpulse™ HEL™ tools. See "LWD tools".
  • MWD Tools. The Victus™ catalog includes ABM™, HyperPulse™, and EMpulse™ MWD tools. See "MWD tools".
  • RSS Units. The Victus™ catalog includes all Weatherford RSS units. The configuration for an RSS unit includes a pivot stabilizer, a bias unit, and a controller collar. See "RSS units".
  • Sand Control Tools. Sand control tools are tools used to prevent large amounts of sand from entering the wellbore. See "Sand control tools".
  • Special Tools. Special tools are circulation tools that are used to control the flow between the pipe and the annulus. See "Special tools".
  • Stabilizers. The Victus™ catalog includes string stabilizers, near-bit stabilizers, centralizers, roller reamers, and non-magnetic stabilizers. See "Stabilizers".
  • Subs. The Victus™ catalog contains several types of rotary subs and circulation subs, including crossover subs, bit subs, saver subs, UBHO subs, float subs, filter subs, and ball lock subs. See "Subs".
  • Thru-tubing Motors. Thru-tubing motors are drilling motors that are deployed on either jointed pipe or coiled tubing. They are powered by drilling fluid, and the power section converts hydraulic horsepower from the mud into mechanical horsepower to drive the bit. See "Thru-tubing motors".
  • Well Services Tools. The tools listed under the Well Services tool type are coiled tubing tools. Coiled tubing can be used for well construction and well intervention services. See "Well services tools".
  • Wireline Cable. Wireline cables are armored electrical cables that are used to lower wireline tools into the wellbore, as well as to transmit the data gathered by wireline tools. See "Wireline cable".
  • Wireline Tools. Wireline tools are electric logging tools that use an electrical cable to lower the tool into the wellbore and transmit data. See "Wireline tools".
