About calculations

On the Setup > OneClick™ > Calculations page, you can create custom expressions to find calculated values, such as average hookload and average torque. You can use incoming data, well configuration data, or other calculations as inputs in a custom expression. You can also create calculations using predefined templates. Calculations can be used in condition checks and other calculations. There are two types of calculations in Victus™ Simulation:

  • Public calculations are global calculations that can be used throughout Victus™ Simulation. The values for public calculations are stored in the Victus™ Simulation database, along with the formula for the calculation. These values are re-calculated in real-time, at one-second intervals.
  • Private calculations are interim calculations that can be used in other calculations. Private calculations can only be used within the OneClick™ page. The values for private calculations are not stored in the Victus™ Simulation database. Unlike public calculations, the values for private calculations are not re-calculated in real-time. They are only re-calculated when they are needed by a public calculation.

The Access Modifier menu, located at the bottom of the screen, allows you to switch between the two sets of calculations. The current values for public calculations can be viewed in the Calculations indicator. To open the Calculations indicator, click the Indicators button () in the Utilities toolbar and select the Calculations option. See "Calculations indicator".

Adding calculations

To create a new calculation:

  1. Go to the Calculations tab on the Setup > OneClick™ page.
  2. Use the Access Modifier menu to select the type of calculation you want to create.
  3. Click the Insert Item button ().
  4. In the Expression pane that appears, click on the preview of the expression.
  5. Use the tabs to locate the appropriate formula for the new expression, then click on the formula. The new expression will appear in the Expression pane.
  6. Select a data source for each value in the expression.
    • Double-click the value you want to update.
    • Click the shaded area around the text box that appears.
    • Select the data source from the menu that appears.
  7. Click the Apply Expression button () to save the expression and close the Expression pane.
  8. Update the calculation name and measurement unit, then click the Save Changes button ().

The new calculation will be saved to the Victus™ Simulation database. It will also be available as data source for condition checks and other calculations. If you created a public calculation, it will also appear in the list of curves on the Home screen and the Setup > Limits and Alarms > Calculations tab.

Using templates

Predefined templates help simplify the creation of frequently used calculations by removing the need to start from scratch each time. The template available currently is the HPFM Totalizers template, which includes predefined expressions to calculate the total volume pumped through the high-pressure flow meters.

To select a template:

  1. Go to the Setup > OneClick™ page.
  2. Click the +Template button to display the available templates.
  3. Select the template you want to use to perform the calculations you need.
  4. Update the measurement unit, then click the Save Changes button ().

Once you select a template, the expressions included in it will be added to the Expression pane. To reset the calculations to the default value, select the desired expression and click the Reset button ().
