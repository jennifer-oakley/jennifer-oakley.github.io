About limits and alarms

The Setup > Limits and Alarms page contains limits for different types of alarms, such as alarms for sensor data, hydraulics calculations, calculated values, RCD parameters, well parameters, and incoming data.

When the limit for an alarm is exceeded, the banner at the top of the screen will flash yellow. An orange warning symbol () will also appear in the banner, and the alarm will be listed on the Alarms > Active page. See "Active alarms".

Values outside of the Minimum and Maximum range will be considered invalid, even if the alarm for that parameter is not enabled. When Victus™ Simulation receives an invalid value, the mapping action selected on the Setup > Mappings > Parameters tab determines how Victus™ Simulation handles the invalid value. See "Mappings".

Alarm ranges

There are three alarm ranges:

  • Red — Red indicates a danger zone. If a parameter falls within a red zone, the minimum or maximum recommended limits have been exceeded. Immediate corrective steps should be taken to prevent damage.
  • Yellow — Yellow indicates a warning zone. If a parameter falls within a yellow zone, the limits are close to being exceeded. Caution should be used to make sure the limits are not exceeded.
  • Green — Green indicates an acceptable zone. If a parameter falls within a green zone, the limits are within the normal range.

The ranges for these zones depend on how you define the limits for the alarm. The Minimum and Maximum limits are required for all alarms. However, the other limits are optional. The table below shows sample limits and the resulting alarm ranges.

Enabling alarms

To enable an alarm:

  1. On the Setup > Limits and Alarms page, go to the tab that contains the alarm you want to enable.
  2. Select the Enable checkbox next to the name of the alarm.
  3. Enter the Minimum and Maximum limits for the alarm.
  4. Enter the remaining limits as needed.
  5. Click the Save Changes button () to save your changes.

If you have created a gauge for the alarm, the gauge will be color-coded with the new alarm ranges. The minimum and maximum values on the gauge will be updated to match the Minimum and Maximum limits on the Setup > Limits and Alarms page. If the Minimum or Maximum values are exceeded, an alert will appear in the notification banner at the top of the screen.

Tip

You can also map a limit to a calculation or well parameter. Go to the limit you want to set, click on the drop-down arrow and select the desired calculation or well parameter.
