About wellpath data

There are multiple ways to add trajectory and survey data on the Setup > Well Data > Wellpath tab.

  • If the planned trajectory and real-time surveys are stored on a WITSML server, you can import the data directly from the WITSML server. See "Importing the planned trajectory from WITSML" and "Importing real-time surveys".
  • If your wellpath data is in a CSV or LAS file, you can import the data from a file. See "Importing wellpath data from a file".
  • If your wellpath data is in an Excel or CSV file, you copy and paste the data from Excel into Victus™ Simulation. See "Copying survey data from a file".
  • If you do not have any wellpath data to import, you can also enter each survey point individually. See "Adding survey points".

After you have added the planned well trajectory, you can interpolate additional survey points by applying tortuosity to the trajectory. See "Applying tortuosity".

Warning

When you apply tortuosity, any existing survey points in the Survey view will be overwritten by the interpolated survey points.

Importing wellpath data from a file

To import survey data from a CSV, LAS, or Compass file:

  1. Go to the Wellpath tab on the Setup > Well Data page.
  2. In the drop-down menu at the bottom of the screen, select data that you want to update.
    • To view the planned trajectory, select the Trajectory option.
    • To view real-time surveys, select the Survey option.
  3. Click the Import Wellpath button () at the bottom of the screen.
  4. In the menu that appears, select the File option. This menu will only appear if you are connected to a WITSML channel with trajectory data.
  5. Click Yes on the warning message that appears.
  6. Navigate to the file you want to import and click Open. A list of import options will appear in Victus™ Simulation.
  7. Select the options for the measured depth data in the file you are importing.
    • MD data is the name of the MD column.
    • MD unit is the measurement unit for MD.
  8. Select the options for the inclination data in the import file.
    • Inclination data is the name of the inclination column.
    • Inclination unit is the measurement unit for inclination.
  9. Select the options for the azimuth data, then click the OK button ().
    • Azimuth data is the name of the azimuth column.
    • Azimuth unit is the measurement unit for azimuth.
  10. When the import finishes, click the Save Changes button () at the bottom of the screen.

The calculated values and charts on the Wellpath tab will be updated automatically. Any existing survey points will be deleted and replaced with the new data. If you updated the survey data, the formation layers on the Formation tab will be updated to reflect the new TVD values. If you updated the trajectory data, the formation layers will not be affected.

Importing the planned trajectory from WITSML

If the planned trajectory is stored on a WITSML server, you can import trajectory data directly from the WITSML server. The well on the WITSML server must have a trajectory log with the planned trajectory data. Before you can import the data, you'll need to create a WITSML channel on the Setup > DAQ > WITSML tab. See "WITSML channels".

  1. Go to the Wellpath tab on the Setup > Well Data screen.
  2. Select the Trajectory option in the drop-down menu at the bottom of the screen.
  3. Click the Import Wellpath button ().
  4. In the menu that appears, select the WITSML channel and trajectory log that contains the planned trajectory.
  5. A list of import options will appear. Select the appropriate options for measured depth, inclination, and azimuth.
  6. Click the OK button (), then click the Save Changes button ().

The trajectory data from the WITSML server will be imported into Victus™ Simulation. Any existing survey points will be deleted and replaced with the new data.

Importing real-time surveys

If the real-time survey data is stored on a WITSML server, you can import data directly from the WITSML server. The well on the WITSML server must have a trajectory log with the real-time survey data. Before you can import the data, you'll need to create a WITSML channel on the Setup > DAQ > WITSML tab. See "WITSML channels".

To import real-time survey data:

  1. Go to the Wellpath tab on the Setup > Well Data screen.
  2. Select the Survey option in the drop-down menu at the bottom of the screen.
  3. Click the Import Wellpath button ().
  4. In the menu that appears, select the WITSML channel and trajectory log that contains the real-time survey data.
  5. A list of import options will appear. Select the appropriate options for measured depth, inclination, and azimuth.
  6. Select the Auto Import option, then select the frequency with which you want real-time surveys to be updated.
  7. In the Tie-On Point field, enter the measured depth of the tie-on point.
  8. Click the OK button (), then click the Save Changes button ().

Any existing survey points below the tie-on point will be deleted and replaced with the new survey points from the WITSML server. Survey points above the tie-on point will not be affected. Victus™ Simulation will periodically query the WITSML server and update the list of real-time surveys.

Copying survey data from a file

To copy survey points from a file:

  1. Open the file containing your data, then copy the data you want to paste into Victus™ Simulation.
  2. Go to the Wellpath tab on the Setup > Well Data page.
  3. Select the row where you want to insert the data, and press Ctrl + V.
  4. Click the Save Changes button () at the bottom of the screen.

The selected row and any rows below it will be replaced with the copied survey data. The calculated values and charts on the Wellpath tab will be updated automatically.

Adding survey points

To add a survey point:

  1. Go to the Wellpath tab on the Setup > Well Data page.
  2. In the drop-down menu at the bottom of the screen, select the data that you want to update.
    • To view the planned trajectory, select the Trajectory option.
    • To view real-time surveys, select the Survey option.
  3. Select a row in the table.
    • To add a survey point above that row, click the Insert Above button ().
    • To add a survey point below that row, click the Insert Below button ().
  4. In the row that appears, go to the MD column and enter the measured depth of the survey point.
  5. Enter the inclination and azimuth of the survey point.
  6. Click the Save Changes button () at the bottom of the screen.

As you add and edit survey points, the calculated values and charts on the Wellpath tab will be updated automatically. If you updated the survey data, the formation layers on the Formations tab will be updated to reflect the new TVD values. If you updated the trajectory data, the formation layers will not be affected.

Applying tortuosity

To apply tortuosity to the planned trajectory:

  1. Go to the Wellpath tab on the Setup > Well Data screen.
  2. Select the Trajectory option in the drop-down menu at the bottom of the screen.
  3. Add the survey points for the planned well trajectory.
  4. Click the Tortuosity button ().
  5. In the window that appears, click the Add button () to add a new tortuosity range.
  6. In the new row, enter the start and end depths of the tortuosity range in the From and To fields.
  7. In the Interpolation Step field, enter the desired distance between each interpolated trajectory point.
  8. In the Tortuosity Method field, select the desired method for applying tortuosity to the trajectory.
  9. Enter the amplitude, period, phase, and shift in the remaining fields.
  10. Click the Apply button () to apply your changes to the trajectory.

Tortuosity will be applied to the trajectory, and the interpolated survey points will appear in the Survey view. Any existing survey points will be overwritten by the interpolated survey points. When you are finished adding tortuosity ranges, click the Close button () to close the window.

Field definitions

Most of the values on the Wellpath tab are calculated automatically, based on the MD, inclination, and azimuth of the survey point. The table on the Wellpath tab contains the following values:

Field definitions for survey point properties
Column Definition
MD Measured depth
Inclination Inclination at the survey point's MD
Azimuth Azimuth at the survey point's MD
TVD True vertical depth
TVDSS True vertical depth subsea
North Northing coordinate
East Easting coordinate
L.Distance Lateral distance
DLS Dogleg severity
B. Rate Build rate
T. Rate Turn rate
T. Face Tool face
VS Vertical section
H. Disp Horizontal displacement
CL Course length
Tort Tortuosity
ABSTort Absolute tortuosity
DDI Rating for the Directional Difficulty Index. The values in the DDI column are color-coded, based on the directional difficulty for the survey point. This color-coding system is also applied to the charts in the Well View pane. See "Trajectory charts".
  • Green — Difficulty is 6.00 or lower, for relatively short wells and simple profiles with low tortuosity.
  • Yellow — Difficulty of 6.10 - 6.40, for shorter wells with high tortuosity and longer wells with lower tortuosity.
  • Orange — Difficulty of 6.50 - 6.80, for longer wells with relatively tortuous well paths.
  • Dark orange — Difficulty of 6.90 or higher, for long, tortuous well profiles with a high degree of difficulty

The table below defines the fields in the Tortuosity window.

Field definitions for the Tortuosity window
Column Definition
From MD at the beginning of the tortuosity range
To MD at the end of the tortuosity range
Interpolation Step Distance between each interpolated trajectory point
Tortuosity Method

Method for applying tortuosity

  • None – No tortuosity is applied to the tortuosity range.
  • Random Inclination and Azimuth Amplitudes – Applies random inclination and azimuth angles to the tortuosity range.
  • Random inclination and dependent amplitude sine azimuth – Applies random angles to the inclination and a sine wave to the azimuth.
  • Sine Wave Inclination – Applies a sine wave to the inclination. Azimuth is not affected.
  • Sine Wave Azimuth – Applies a sine wave to the azimuth. Inclination is not affected.
  • Sine Wave Inclination and Azimuth – Applies sine waves to the inclination and the azimuth.
Amplitude Amount of displacement in a wave, measured from the high point to the low point
Period Distance between the peak of two consecutive waves
Phase Horizontal shift of the wave
Shift Vertical offset of the wave
Retain Original Retains a copy of the original trajectory
