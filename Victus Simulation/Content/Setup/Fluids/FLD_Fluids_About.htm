About muds

There are three basic types of fluids available on the Fluids tab:

  • Muds are used to maintain pressure in the wellbore and clean cuttings away from the bit during drilling operations. There are four types of muds in Victus™ Simulation: water-based muds, oil-based muds, synthetic-based muds, and mineral oil-based muds.
  • Cement Slurries are used to secure casing to sections of the well that have already been drilled. Cements primarily contain limestone and clay, as well as various types of additives.
  • Spacers are used to provide a buffer between different fluids in the well, minimizing contamination between fluids. Spacers are generally water-based Newtonian fluids that contain viscosifiers and other additives.

There are two steps to adding mud and cement information:

  1. Enter the basic properties of the mud or cement, including the type of fluid and the fluid density. See "Basic fluid properties".
  2. Enter the fluid's rheological properties, defined from dial readings. See "Isothermal dial readings".

If there are fluid reports stored on a WITSML server, you can import some of the fluid properties directly from the WITSML server. See "Importing fluids".

Importing fluids

Before you can import fluid reports, you'll need to create a WITSML channel on the Setup > DAQ > WITSML tab. Fluid reports can only be imported from the active WITSML channel. See "WITSML channels".

To import a fluid:

  1. Go to the Fluids tab on the Setup > Fluid Properties page.
  2. Click the Import From WITSML button ()at the bottom of the screen.
  3. Click Yes on the warning message that appears.

A new fluid will be created, using the data from the fluid report on the WITSML server.

Note

Only the mud weight and fluid type will be imported. The remaining properties must be manually entered.

Using the Fluids tab

Each fluid in the well has its own numbered tab. To switch to another fluid, click on the numbered tab for that fluid. The toolbar at the bottom of the screen is used to manage the fluids in your well.
