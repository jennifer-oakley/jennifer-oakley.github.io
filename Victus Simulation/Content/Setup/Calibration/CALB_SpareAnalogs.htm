About spare analogs

On the Setup > Calibration > Spare Analogs tab, you can configure any additional analog sensors connected to the Victus™ MPD System. Spare analogs are additional plug and play sensors that can be connected directly to the analog sensor ports on the UPI with a three-pin cable. The spare analog sensor ports can be connected to any type of 4-20 mA sensor, such as temperature sensors.

If the sensor data is going to be received through a remote analog interface device, you need to check the Remote Analog Device box in the Setup > Surface Equipment > Equipment Configuration tab. A remote analog device is a hardware component designed to transmit and receive analog signals over long distances. See "Equipment configuration".

When you calibrate the sensors, the raw values should be entered in amps. These values should come from the sensor's calibration certificates. Sensor ratings, however, should be entered in engineering units. These values should come from the sensor's specification sheet.

On the Setup > Calibration > Spare Analogs tab, you can calibrate the readings received from these sensors. After you have calibrated the spare analog sensors, you can map the sensor data to a well parameter on the Setup > Mappings > Parameters tab. See "Mappings".

Calibrating spare analogs

To calibrate a spare analog sensor:

  1. Go to the Spare Analogs tab on the Setup > Calibration page.
  2. Select the sensor you want to calibrate.
  3. In the Title field, enter a brief description of the sensor.
  4. Enter the raw minimum and maximum values in the Raw Low and Raw High fields, in amps.
  5. Enter the minimum and maximum sensor ratings in the EU Low and EU High fields, in engineering units.
  6. In the Unit Type field, select the unit type for the sensor.
  7. In the Unit field, select the engineering unit for the sensor.
  8. Click the Save Changes button () to save your changes.

The upper and lower ranges for the spare analog sensor will be calibrated according to the sensor's specifications. To view the current analog readings, click on the Indicators View utility () and select the DAQ option. See "Data acquisition indicators".

Field definitions

Fields on the Calibration > Spare Analogs tab
Field Definition
Title

Name of the sensor
Raw Low

Lowest allowable electrical measurement for the sensor, entered in amps. This value can be found in the sensor's calibration certificate.
Raw High Highest allowable electrical measurement for the sensor, entered in amps. This value can be found in the sensor's calibration certificate.
EU Low Minimum reading from the sensor, entered in engineering units. This value can be found in the sensor's specification sheet.
EU High Maximum reading from the sensor, entered in engineering units. This value can be found in the sensor's specification sheet.
Meas. Current raw measured reading from the analog sensor. This value is read-only, and cannot be edited.
Value Current reading from the sensor, converted into engineering units. This value is read-only, and cannot be edited.
Unit Type Unit type for the sensor
Unit Engineering unit for the sensor
