About choke positions

The MPD manifold includes up to four chokes, which are controlled by the Victus™ MPD System. These chokes can be either hydraulic-powered chokes or electric chokes. The chokes are opened and closed as needed to control the pressure. Each choke has a position indicator, which shows the choke position during operation.

On the Setup > Calibration > Chokes tab, you can view the calibrated positions of each choke.

You can also manually set the positions for all chokes simultaneously. See "Setting choke positions".

Disabling the chokes

To disable all chokes:

  1. Click the Pressure tile at the bottom of the Well View pane.
  2. In the menu that appears, select the None option.
  3. Click Yes in the warning message that appears.
  4. Click the Chokes tile at the bottom of the Well View pane.
  5. For each choke in the menu that appears, select the choke and select the Off option.

All chokes will be disabled and ready for calibration. Each choke will remain in its current position. Victus™ Simulation will no longer control the chokes.

Setting choke positions

When the chokes are in manual mode, you can set the positions for all chokes simultaneously. To manually set the positions for the chokes:

  1. Set all chokes to manual mode. See "Manually controlling chokes".
  2. Go to the Chokes tab on the Setup > Calibration page.
  3. Click the Chokes button ().
  4. In the window that appears, enter the new choke position for both chokes.
  5. Click the Enter button (), then click the Close button ().

The choke position will be updated for all chokes, and Victus™ Simulation will open or close the chokes as needed.

Field definitions

The table below defines the fields for hydraulic chokes on the Setup > Calibration > Chokes tab.

Fields for hydraulic chokes on the Calibration > Chokes tab
Field Definition
Raw Low

Lowest reading registered on the position indicator when the choke is fully closed, measured in volts
Raw High

Highest reading registered on the position indicator when the choke is fully open, measured in volts
Unit Unit type used for measuring raw readings
EU Low

Percentage at which the choke is fully closed
EU high Percentage at which the choke is fully open
Meas.

Current measured reading from the position indicator, measured in volts
Value Current reading from the position indicator, converted into percentage
Unit Engineering unit for the position indicator

The table below defines the fields for electric chokes on the Setup > Calibration > Chokes tab.

Fields for electric chokes on the Calibration > Chokes tab
Field Definition
Value Current reading from the position indicator, converted into percentage
Unit Engineering unit for the position indicator
Calibration Status

Indicates the calibration status of the choke.

  • A green box indicates that the choke is fully calibrated.
  • A red box indicates that the choke is not calibrated.

Using the Chokes tab

When you are using hydraulic chokes, the toolbar at the bottom of the page is used to open and close the chokes, as well as set the choke positions.

Toolbar buttons for hydraulic chokes on the Calibration > Chokes tab
Button Function

Discards all changes made since the last time you saved

Saves all changes
Allows you to manually control all chokes simultaneously

Closes the selected choke
Sets the closed position for the selected choke

Opens the selected choke
Sets the open position for the selected choke

When you are using electric chokes, the toolbar at the bottom of the page is used to calibrate the chokes.

Toolbar buttons for electric chokes on the Calibration > Chokes tab
Button Function

Calibrates the open and closed positions of the selected choke
