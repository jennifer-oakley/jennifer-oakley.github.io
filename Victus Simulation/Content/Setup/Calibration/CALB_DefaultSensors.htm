About default sensors

On the Setup > Calibration > Default Sensors tab, you can configure the sensors connected to the Victus™ MPD System. The default sensors are used to monitor rig air pressure, hydraulic power unit (HPU) pressure, SBP, SPP, and flow. The sensors must be analog pressure sensors or flow meters, whose range is from 4 to 20 mA. They are connected to the ICU or the IU-CU.

When you calibrate the sensors, the raw values should be entered in amps. These values should come from the sensor's calibration certificates. Sensor ratings, however, should be entered in engineering units. These values should come from the sensor's specification sheet.

On the Setup > Calibration > Default Sensors tab, you can calibrate the default sensors by entering the minimum and maximum readings from the sensor. Before calibrating the sensors, you'll need to select the type of sensor connected to each sensor source on the Setup > Calibration > Piping and Instrumentation tab. See "Sensor configuration".

Note

The unit types shown on the Default Sensors tab are determined by the types of sensors selected on the Setup > Surface Equipment > Piping and Instrumentation tab. See "Sensor configuration".

Calibrating sensors

To calibrate the default sensors:

  1. Go to the Default Sensors tab on the Setup > Calibration page.
  2. Select the sensor you want to calibrate.
  3. Enter the raw minimum and maximum values in the Raw Low and Raw High fields, in amps.
  4. Enter the minimum and maximum sensor ratings in the EU Low and EU High fields, in engineering units.
  5. Click the Save Changes button () to save your changes.

The upper and lower ranges of values for the default sensors will be calibrated to match the minimum and maximum readings from each sensor. To view the current sensor readings, click on the Indicators View utility () and select the Sensors option. See "Sensors indicator".

Field definitions

The table below defines the fields on the Setup > Calibration > Default Sensors tab.

Fields on the Calibration > Default Sensors tab
Field Definition
Raw Low

Lowest reading registered on the sensor when there is no pressure, entered in amps
Raw High

Highest reading registered on the sensor at maximum pressure, entered in amps
Unit Unit type used for measuring raw readings
EU Low Minimum amount of pressure or flow that the sensor is rated for, entered in engineering units.
EU High Maximum amount of pressure or flow that the sensor is rated for, entered in engineering units.
Meas. Current measured reading from the sensor, in amps. This value is read-only, and cannot be edited.
Value Current reading from the sensor, converted into engineering units. This value is read-only, and cannot be edited.
Type Unit type for the sensor. The unit type is determined by the sensor type selected on the Setup > Surface Equipment > Piping and Instrumentation tab. See "Sensor configuration".
Unit Engineering unit for the sensor
