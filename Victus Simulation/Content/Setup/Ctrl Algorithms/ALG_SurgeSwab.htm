About the surge and swab algorithm

The surge and swab algorithm on the Setup > Control Algorithms > Surge/Swab tab is used to calculate the surface backpressure required to maintain a constant wellbore pressure due to the swab and surge effects on pressure when running the bit in or out of the well. When S&S mode is enabled, Victus™ Simulation uses these tables to determine the current set point for surface backpressure. The current set point will be determined by the trip direction and current block speed. See "Using S&S mode".

Adding surge and swab intervals

To manually edit the Surge and Swab tables:

  1. Go to the Setup > Control Algorithms > Surge/Swab tab.
  2. Click the Insert item () button and select the Surge or Swab option.
  3. Enter the block speed for the interval in the Speed column.
  4. Enter the surface backpressure for the interval in the SBP column.
  5. Click the Save Changes button () at the bottom of the screen.

The table will be updated with the new surge or swab interval. The new interval will be included in the chart.

Using the Surge/Swab tab

The toolbar at the bottom of the Surge/Swab tab is used to add or remove rows from the tables and to refresh the page.
