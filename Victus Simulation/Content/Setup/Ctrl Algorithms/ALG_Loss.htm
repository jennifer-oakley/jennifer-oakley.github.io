About the loss algorithm

The loss algorithm on the Setup > Control Algorithms > Loss tab is used to automatically detect and control lost circulation in the well. A loss occurs when the drilling fluids flow into the formation. Lost circulation is caused by natural fissures and fractures in the formation, as well as by mud pressure that exceeds the formation pressure.

When kick/loss detection is enabled, Victus™ Simulation will use the settings under the Detection heading to determine when a loss occurs. See "Loss detection". If the Auto Control option is enabled, the loss will be controlled using the settings under the Control heading. See "Loss control".

Loss detection

A loss will be detected if, within the time frame specified in the Trend Time field:

  • Flow in is larger than flow out.
  • Flow out decreases from the last measured value, at a rate higher than the Flow Difference value.

When a loss is detected, an alarm notification will appear in the banner at the top of the screen. If the Auto Control option is enabled, Victus™ Simulation will control the loss using the settings in the Control section of the loss algorithm. See "Loss control".

Loss control

When Victus™ Simulation controls a loss, the chokes will open to decrease surface backpressure until equilibrium is re-established between the incoming and outgoing flow rates. During this process, Victus™ Simulation will switch to Loss pressure control mode. In Loss mode, pressure is decreased until it reaches the minimum allowable value or the loss is controlled. The minimum value used depends on the pressure control mode when the loss was detected.

  • If the loss is detected in SBP, Flow, or Connection mode, surface backpressure is decreased until it reaches the Minimum Allowable SBP value.
  • If the loss is detected in SPP mode, standpipe pressure is decreased until it reaches the Minimum Allowable SPP value.
  • If the loss is detected in AP mode, annular pressure is decreased until it reaches the Minimum Allowable AP value.

After the flow stabilizes, you'll need to click on the banner to confirm that the loss was controlled. Victus™ Simulation will automatically switch from Loss mode to the pressure control mode you were using before the loss, using the current pressure as the setpoint.

Field definitions

The table below defines the detection and control settings on the Setup > Control Algorithms > Loss tab.

Field definitions for the loss algorithm
Section Name Field Name Description
Detection Trend Time Time interval over which the loss detection factors are monitored
Flow Difference Difference between flow in and flow out
Auto Control Enable Allows Victus™ Simulation to automatically control a loss
Control Proportional Gain Tuning parameter for the responsiveness of the movement of the chokes. A low value can result in slow responses, and a high value cause the choke movement to oscillate.
Minimum Allowable SPP Minimum level that standpipe pressure can be reduced to. Once a loss has been detected, Victus™ Simulation will not lower standpipe pressure below this level when using SPP pressure control mode.
Minimum Allowable SBP Minimum level that surface backpressure can be reduced to. Once a loss has been detected, Victus™ Simulation will not lower surface backpressure below this level when using SBP pressure control mode.
Minimum Allowable AP Minimum level that annular pressure can be reduced to, at the depth for the AP set point. Once a loss has been detected, Victus™ Simulation will not lower annular pressure below this level when using AP pressure control mode.
