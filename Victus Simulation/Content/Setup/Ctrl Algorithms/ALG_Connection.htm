About the connection algorithm

The connection algorithm on the Setup > Control Algorithms > Connection tab is used to calculate the surface backpressure required to maintain a constant wellbore pressure when ramping the pumps up or down. The connection algorithm determines a flow rate schedule, based on the current flow rate. The flow rate schedule starts at zero and increases in increments, until it reaches the current flow rate.

When you are using Connection mode to manage pressure, Victus™ Simulation maintains a constant annular pressure during an MPD connection. As the flow from the rig pumps decreases during a connection, surface backpressure must be increased to compensate for the loss in frictional pressure. Victus™ Simulation follows the connection ramp table on the Setup > Control Algorithms > Connection tab to adjust surface backpressure in order to keep annular pressure constant.

When you switch to Connection mode:

  • If SPP is below the threshold, Victus™ Simulation will use the ramp table to increase the SBP set point. However, the SBP set point will not be decreased until the SPP threshold has been reached.
  • If SPP is above the threshold, Victus™ Simulation will use the ramp table to increase and decrease the SBP set point as needed to maintain a constant bottomhole pressure.

If an SPP sensor fails during the connection, the SPP threshold will be disregarded and the SBP set point will be updated using the ramp table.

When the connection ramp is calculated, Victus™ Simulation creates ten equal intervals to ramp down the pumps. For each interval, Victus™ Simulation calculates surface backpressure required to maintain bottomhole pressure as the flow rate decreases. Surface backpressure is calculated using the following formula:

  • SBP = BHP – pa,
  • where
    • BHP is bottomhole pressure
    • pa is annular pressure, found by pa = pha + pfa

Surface backpressure

After the connection ramp is calculated, you can manually modify the ramp table by adding or removing flow rate intervals. See "Adding ramp intervals".

Calculating the connection ramp

To calculate the connection ramp:

  1. Go to the Setup > Control Algorithms > Connection tab.
  2. Click the Get Ramp Data button () at the bottom of the screen.
  3. Click the Save Changes button ().

The flow rate schedule will be calculated, using the current operating parameters. To refresh the ramp calculations, click the Get Ramp Data button () again.

Note

In order to calculate the connection ramp, there must be flow present in the well. If the flow rate is zero, no results will be calculated.

Adding ramp intervals

To manually edit the connection ramp:

  1. Go to the Setup > Control Algorithms > Connection tab.
  2. Select a row in the connection ramp table.
    • To add a ramp interval above that row, click the Insert Above button ().
    • To add a ramp interval below that row, click the Insert Below button ().
  3. Enter the flow rate for the interval in the Flow column.
  4. Enter the surface backpressure for the interval in the SBP column.
  5. Click the Save Changes button () at the bottom of the screen.

The ramp table will be updated with the new ramp interval. The new ramp interval will be included in the flow rate schedule when Victus™ Simulation ramps down the pumps during an MPD connection.

Using the Connection tab

The toolbar at the bottom of the Connection tab is used to add and remove ramp intervals for MPD connections.
