About unit systems

The Setup > Units page contains the settings for the measurement units used throughout Victus™ Simulation. It shows the available unit systems, as well as the measurement units and decimal places used for each type of unit.

The tabs on the right side of the screen show the unit systems that have been created. The base unit system is shown at the top of each unit system, in the Unit System field.

There are three base unit systems in Victus™ Simulation:

  • Imperial – Uses API-recommended units.
  • Metric – Uses SI-recommended units.
  • Custom – Customizable unit system that uses API units by default.

The settings for the assigned unit system are shown in the table below the Unit System field. This table lists all of the unit types in Victus™ Simulation, as well as the measurement units and decimal precision for each unit type.

  • The type of unit is shown in the first column. Unit types classify the measurement units for Victus™ Simulation into groups, such as units for length, density, and pressure.
  • The measurement unit for each unit type is shown in the middle column. The available units for each unit type are listed in drop-down menus. For example, the units available for the Length/Depth unit type include feet, meters, inches, and kilometers.
  • The precision for each unit type is shown in the last column. The precision setting controls the number of decimal places displayed for the unit type. The default precision for most unit types is two decimal places.

You can change the measurement units and precision settings for any of the unit systems in Victus™ Simulation. However, changes to the Imperial or Metric unit systems will not overwrite the default settings. If you switch to another unit system, any changes will be lost.

If the data for your well uses different units than the Imperial or Metric unit system, you can create a custom unit system. Changes to the custom unit system are permanently saved, unlike changes to the Imperial and Metric unit systems. If you switch to another unit system, the settings for the custom unit system will not be lost.

Using the Units page

The toolbar at the bottom of the screen is used to save and undo changes. The toolbar on the right side of the screen is used to create, copy, and activate unit systems.
