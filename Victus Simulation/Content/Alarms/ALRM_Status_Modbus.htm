Modbus alarms

Related Topics
Related Topics
  • Data acquisition indicators
  • Modbus data

The Alarms > Status > MODBUS tab allows you to monitor the status of Victus™ Simulation's data acquisition (DAQ) service for modbus channels. The Status icon next to each channel name indicates the status of communication with the channel.

  • A green box indicates that Victus™ Simulation is communicating with the channel.
  • A red box indicates that Victus™ Simulation is not communicating with the channel.

If Victus™ Simulation loses communication with a modbus channel, you may need to restart the DAQ service.

To restart the DAQ service for modbus channels:

  1. Go to the MODBUS tab on the Alarms > Status page.
  2. Click the Reboot button () at the bottom of the screen.
  3. Click Yes on the warning message that appears.

The DAQ service will be restarted, and Victus™ Simulation will reconnect to all modbus channels. When communication has been reestablished, the status icon will change to green.

To view the unprocessed modbus data, click the View MODBUS Raw Data button (). This data can be used to troubleshoot data issues and for additional analysis. The MODBUS Raw Data window shows all of the incoming records for modbus. As the data is received, records are added to the bottom of the MODBUS Raw Data listing.
