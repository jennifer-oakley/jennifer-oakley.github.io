About daily comments

Comments for the daily activity report can be entered and viewed on the Feedback > Comments page. There are three types of comments that appear on the Comments page:

  • C – Records of daily activities on the rig, such as safety meetings, rig service, pulling out of hole, and running in hole. See "Logging daily activities".
  • E – Forecast of the expected activities for the next day. See "Entering daily summaries and forecasts".
  • S – Summary of the actual activities and events for the current day. See "Entering daily summaries and forecasts".

Comments are color-coded by the rig activity at the time the comment was entered. Once a comment is entered on the Comments page, you will not be able to edit or delete it. The records that appear on the Comments page will also appear on the Timeline page. See "Timeline report".

Logging daily activities

To log a new daily activity:

  1. Go to the Feedback > Comments page.
  2. Click the Comments button in the bottom right corner of the screen, then click the Comment button ().
  3. In the Period field, enter start and end time for the activity.
  4. Enter a brief description of the activity in the text box below the Period field.
  5. Click the Save Changes button ().

The comment will be added to the daily activity log on the Comments and Timeline pages. The comment will also appear in the daily activity report. See "Daily reports".

Entering daily summaries and forecasts

Daily summaries should be entered at the end of day, along with the forecast for the next day. To enter a daily summary or forecast:

  1. Go to the Feedback > Comments page.
  2. Click the Comments button in the bottom right corner of the screen, then select the type of comment you want to enter.
    • To enter a daily summary, click the Summary button ().
    • To enter a daily forecast, click the Expected button ().
  3. In the For field, select the date that the comment is for.
    • For summary comments, select the current date.
    • For forecast comments, select the next day's date.
  4. Enter a brief description in the text box below the For field.
  5. Click the Save Changes button ().

The comment will be added to the daily activity log on the Comments and Timeline pages. The comment will also appear in the daily activity report. See "Daily reports".

Field definitions

The table below defines the fields for each type of comment on the Feedback > Comments page.

Field definitions on the Comments page
Comment Type Field Name Definition
Comment Period Start and end date for the daily activity
Description Brief description of the daily activity
Summary For Date that the summary is for
Description Brief description of the summary for selected date
Expected For Date that the forecast is for
Description Brief description of the forecast for the selected date

Using the Comments page

The toolbar on the right side of the Comments page is used to navigate between days and add comments to the daily record. To use the calendar to view the comments from another day, click on the date bar on the right side of the page.

Toolbar buttons on the Comments page
Button Function
Navigates to the comments for the previous day
Navigates to the comments for the next day
Adds a comment to the daily record
Resets the time to the time that the last comment was entered
Allows you to select a calendar date
