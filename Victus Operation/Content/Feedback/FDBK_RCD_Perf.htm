About the RCD performance report

The Feedback > RCD Performance page contains a report of the RCD's performance since the last bearing assembly was installed. The report includes the following information:

  • Bearing hours – Amount of time the bearing assembly is latched to the RCD bowl.
  • Pipe Length passed through BA – Length of the pipe that has passed through the bearing assembly.
  • Tooljoints Stripped – Number of tool joints that have passed through the bearing assembly.
  • Bearing Installation Time – Date and time the bearing was installed. To change the latest installation date, click the Calendar button ()

The report also contains a pie chart which compares the time spent rotating to the time spent not rotating.

  • Rotation hours – Amount of time that the bearing assembly has been rotating.
  • Stripping hours – Amount of time that the bearing assembly has moved up or down while the bearing assembly was latched.

The RCD Performance Report is only available when you are connected to a 7875 BTR or BTR-S RCD. You'll also need to select the type of RCD on the Setup > Surface Equipment page. See "Selecting rig equipment". When a new bearing assembly is installed, click the Reset button () to reset the report parameters. The data shown in the report will automatically update when you go to the RCD Performance page. To manually update the data shown in the report, click the Refresh button ().

Using the RCD Performance page

The toolbar at the bottom of the RCD Performance page is used to reset or refresh the values of the bearing assembly parameters.

Toolbar on the RCD Performance page
Button Function
Resets the installation time for the bearing assembly
Refreshes the data received from the RCD
