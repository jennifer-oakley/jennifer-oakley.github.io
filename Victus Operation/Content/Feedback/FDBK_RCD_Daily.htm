About daily RCD reports

The Feedback > RCD Daily Report page contains a daily report that is automatically generated by Victus™ Operation. The report includes information about the day's RCD activity such as depth, flow, pressure, torque, and temperature. The daily RCD report is generated at the end of the day.

Note

The Feedback > RCD Daily Report page is only available when a 7875 DS, 7875 BTRS, or BTR-S RCD is selected on the Setup > Surface Equipment page. See "Selecting rig equipment".

The table below lists the information included in a daily report.

Contents of an RCD daily report
Report Section Explanation
Charts Charts plotting values for surface backpressure, standpipe pressure, flow out temperature, top drive speed, bit depth, and hole depth.
Table Table showing the values for bit depth, hole depth, top drive speed, ROP, wellbore pressure, flow out, fluid temperature, pump pressure, tank level and temperature, latch volumes, and latch position.
BA Timebreakdown (hours)

Bearing assembly graphic containing the following information:

  • Bearing hours – Amount of time the bearing is latched to the RCD bowl.

  • Rotation hours – Amount of time the bearing is rotating.

  • Stripping hours – Amount of time the bearing is moved up or down when the bearing is latched.

  • Tooljoints – Number of tooljoints that pass through the bearing assembly.

  • Pipe Length – Amount of pipe length that passes through the bearing assembly.
Daily Summary for last 24 hrs A time-stamped list of daily activity comments, summary comments, and maintenance records. See "Daily activity comments" and "Maintenance records".

Using the RCD daily report page

After the Daily Activity report has generated, a preview of it will be displayed on the RCD Daily Report page. The toolbar at the top of the report preview is used to print and save the report, as well as navigate between the report pages.

Toolbar on the RCD Daily Report page
Button Function
Shows the report in the actual size of the page
Fits the report to the current width of the window
Fits the report to show one full page
Fits the report to show multiple pages
Zooms in on the report
Zooms out on the report
Exports the report to a PDF file
Exports the report to a Word file
Exports the report to an Excel file
Prints the report to a local or network printer
