About daily reports

The Feedback > Daily Report page contains a daily report that is automatically generated by Victus™ Operation. The report includes information about the day's rig activity, summaries, personnel on the rig site, downhole pressures, and RCD data.

Daily reports are available when the Backup panel or Monitoring panel roles are active. Daily reports are automatically created for the previous day at midnight. The daily reports are saved to the Victus™ Operation database, and will be included when you export the well to an MCD file. See "Exports".

Report details

The table below lists the information included in a daily report.

Contents of a daily report
Report Section Explanation
Job Types Type of job, from the Setup > Job Info page. See "Job Info".
Equipment Configuration Equipment on the rig, from the Setup > Surface Equipment page. See "Surface equipment".
Well Preview A sketch of the well intervals, drillstring, and fluid. See "Well View pane".
Personnel A list of personnel on the rig site, from the Setup > Job Info page. See "Personnel details".
Daily Summary for last 24 hrs A time-stamped list of daily activity comments, summary comments, and maintenance records. See "Daily activity comments" and "Maintenance records".
Graphs Charts showing bottomhole pressure, SBP, choke positions, and flow rate
RCD Graphs Charts showing DPR data from the RCD
Rig Activity Charts showing the time breakdown of rig operations for the day

Using the Daily Report page

After the Daily Activity report has generated, a preview of it will be displayed on the Daily Report page. The toolbar at the top of the report preview is used to print and save the report, as well as navigate between the report pages.

Toolbar on the Daily Report page
Button Function
Shows the report in the actual size of the page
Fits the report to the current width of the window
Fits the report to show one full page
Fits the report to show multiple pages
Zooms in on the report
Zooms out on the report
Exports the report to a PDF file
Exports the report to a Word file
Exports the report to an Excel file
