Diagnostic logs

The Feedback > Logs page tracks all entries to the logs for Victus™ Operation. There are two types of logs available on the Logs page:

  • The Panel tab displays any errors related to the user interface for Victus™ Operation. The log file for the Panel tab is stored in your Victus folder, under the C:\Users\[UserName]\Documents\Victus\Operation\logs directory.
  • The Agent tab displays all log entries related to the database for Victus™ Operation. The log file for the Agent tab is stored in the Victus program data folder, under the C:\ProgramData\Weatherford\Victus\Operation\logs directory.
  • The DAQ tab displays all log entries related to data acquisition for Victus™ Operation. The log file for the DAQ tab is stored in the Victus program data folder, under the C:\ProgramData\Weatherford\Victus\Operation\logs directory.

Each set of logs is grouped by date. The toolbar on the right side of the Logs page is used to navigate between days.

  • To view the logs for the previous day, click the Previous Date button ().
  • To view the logs for the next day, click the Next Date button ().
  • To use the calendar to view the logs from another day, click on the date bar on the right side of the page.
