About additional pumps

The first pump in the rig schematic is the only pump that is activated by default. If there are additional pumps on the rig, each additional pump must be activated in order to be used in Victus™ Operation's calculations. On the Setup > Surface Equipment > Piping and Instrumentation tab, you can change the pump state assigned to each pump in the pump properties. There are four pump states:

  • Aux – Auxiliary pumps are backup pumps that may be used in addition to the main pumps. Auxiliary pumps can only be used with surface BOPs. See "Activating auxiliary pumps".
  • Boost – Booster pumps are used to pump fluid through the boost lines on a subsea rig. Booster pumps can only used with subsea BOPs. See "Activating booster pumps".
  • Disable – Disabled pumps are not used in any of Victus™ Operation's calculations. All pumps except Pump1 are disabled by default.
  • Main – The main pumps are the primary pumps used on the rig. By default, Pump1 is the only main in Victus™ Operation.

Activating auxiliary pumps

To activate an auxiliary pump:

  1. Go to the Equipment Configuration tab on the Setup > Surface Equipment page.
  2. In the Blow-Out Preventer field, select Surface BOP from the list.
  3. In the Auxiliary Pump field, select the Rig Pump or Weatherford option.
  4. Click the Save Changes button ().
  5. Go to the Piping and Instrumentation tab.
  6. Select one of the secondary pumps in the rig schematic.
  7. In the panel that appears, go to the Pump State field and select the Aux option.
  8. Update the remaining pump settings, then click the Save Changes button ().

The auxiliary pump will be activated and the updated equipment specifications will be saved to the Victus™ Operation database.

After you activate the pumps, the next step is to calibrate the sensors. See "Calibrate sensors".

Activating booster pumps

To activate a boost pump:

  1. Go to the Equipment Configuration tab on the Setup > Surface Equipment page.
  2. In the Blow-Out Preventer field, select Subsea BOP from the list.
  3. Click the Save Changes button ().
  4. Go to the Piping and Instrumentation tab.
  5. Select one of the secondary pumps in the rig schematic.
  6. In the panel that appears, go to the Pump State field and select the Boost option.
  7. Update the remaining pump settings, then click the Save Changes button ().

The boost pump will be activated and the updated equipment specifications will be saved to the Victus™ Operation database.

After you activate the pumps, the next step is to calibrate the sensors. See "Calibrate sensors".
