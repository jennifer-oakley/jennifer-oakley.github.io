About work strings

The next step is to build the work string that you will be using in the well. The Setup > Work String page is used to build the work strings used while drilling and casing a well. Victus™ Operation includes the Weatherford catalog, which is the master catalog for BHA and drillstring components. It contains many of the tools manufactured by Weatherford, such as RSS units, mud motors, and LWD/MWD tools. The Weatherford catalog also contains standard BHA and drillstring parts, such as drill bits, stabilizers, drill pipe, and HWDP. Although you can edit many of these tools, the changes will only be reflected in that particular instance of the tool. The Weatherford catalog itself will not be affected. See "Work string components".

Creating a work string

When you create a work string, you can create it with or without an explicitly-defined BHA.

Work string with an explicitly-defined BHA

  • In a work string with an explicitly-defined BHA, the BHA and drillstring components will be grouped into separate tables.
  • In a work string without an explicitly-defined BHA, all BHA and drillstring components will appear in one table.

To create a new work string:

  1. Go to the Setup > Work String page.
  2. Click the Add button ().
  3. When prompted, select an option for defining the BHA.
    • To create a work string with an explicitly-defined BHA, click Yes.
    • To create a work string without an explicitly-defined BHA, click No.
  4. Click the Insert Tool button () to add components to the work string. See "Adding components".
  5. When you have finished adding components, click the Save Changes button () at the bottom of the screen.

A new work string will be added to Victus™ Operation, and a new numbered tab will appear in the right toolbar. As you add new components, the preview of the complete work string will be updated in the Well View pane. See "Work string components".

Adding components

To add a work string component:

  1. Select a work string using the numbered tabs on the right side of the screen.
  2. Select the component below the one you want to add, then click the Insert Tool button ().
  3. In the panel that appears on the right, go to the Tool field and select the type of component you want to add.
  4. Enter the dimensions and other properties of the component. See "Work string components".
  5. Click the OK button () to save your changes.

The new part will appear in the list of components, and the preview of the complete work string will be updated in the Well View pane.

After you create the work string, the next step is to add the fluids that will be used in the next run. See "Adding fluids".
