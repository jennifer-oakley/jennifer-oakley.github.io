About fluids

The next step is to define the drilling and cementing fluids. The Setup > Fluid Properties page contains the properties for the muds and cements that will be used during drilling and cementing operations.

When you create a new fluid , the basic fluid properties are entered in the top portion of the screen. The basic fluid properties include fluid type, fluid weight, temperatures, and the fluid composition. If the fluid report includes additional information, such as salt type and salinity, you can enter this information on the right side of the screen. If not, you can simply use the default values.

There are two steps to adding mud information:

  1. Enter the basic properties of the mud, including the type of fluid and the fluid density. See "Adding a mud".
  2. Enter the fluid's rheological properties, defined from the dial readings for the fluid. See "Entering dial readings".

Adding a mud

To add a drilling mud:

  1. Go to the Setup > Fluid Properties page.
  2. Click the Add button () at the bottom of the screen. A new numbered tab will appear in the toolbar to the right.
  3. In the Fluid Type field, select the type of fluid you want to add.
  4. In the Fluid Weight In field, enter the density of the fluid when it enters the wellbore.
  5. In the Fluid Inlet Temperature field, enter the inlet temperature of the mud in the suction tank.
  6. In the Ref. Temperature field, enter the temperature at which the mud weight was measured.
  7. Enter the composition of the mud in the Retort Oil and Retort Water fields.
  8. If available, enter the remaining information. See "Field definitions".
  9. Enter the data from a dial reading on the Dial Readings tab. See "Entering dial readings".
  10. Click the Save Changes button () at the bottom of the screen.

The fluid will be stored in Victus™ Operation's database. When you change the pump fluid, you'll be able to select the new fluid under the Mud utility. However, you will not be able to edit the fluid properties or dial readings.

Entering dial readings

To enter dial readings properties for a new fluid:

  1. On the Dial Readings tab, go to the first row in the table.
  2. In the θ600 - θ3 columns, enter the viscometer readings taken at each speed.
  3. In the Gel 10" - Gel 30' columns, enter the gel strength measured for each interval.
  4. In the Rheology Fit field, select the rheological model that is the closest fit for the fluid.
  5. Click the Save Changes button ().

When you save the fluid, it will be added to the Victus™ Operation database. When you change the pump fluid, you'll be able to select the new fluid under the Mud utility. However, you will not be able to edit the fluid properties or dial readings.

After you add the drilling fluids, the next step is to choose the initial pump fluid. See "Setting pump fluids".
