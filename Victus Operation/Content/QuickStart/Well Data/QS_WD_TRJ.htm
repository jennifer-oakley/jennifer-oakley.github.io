About wellpath data

The next step is to add the wellpath information. The Setup > Well Data > Wellpath tab contains the wellpath data for the planned well trajectory and real-time surveys. The drop-down menu at the bottom of the screen allows you to switch between the trajectory data and the survey data.

  • The Trajectory view should be used for the survey points in the planned trajectory of the well. The data in the Trajectory view is only for reference, and is not used in any calculations.
  • The Survey view should be used for the actual, real-time survey data. All calculations in Victus™ Operation use the data in the Survey view.

The measured depth, inclination, and azimuth of each survey point are used to calculate several values, such as TVD, dogleg severity, build rate, and turn rate. These values are updated automatically as you add and edit survey points.

The first two survey points of the wellpath are automatically calculated, based on the rig elevations and the properties of the derrick. These surveys points are highlighted in pink on the Wellpath tab. To change these survey points, you must update the rig elevations on the Setup > Surface Equipment page. See "Rig Elevations".

Importing real-time surveys

If the real-time survey data is stored on a WITSML server, you can import data directly from the WITSML server. The well on the WITSML server must have a trajectory log with the real-time survey data. Before you can import the data, you'll need to create a WITSML channel on the Setup > DAQ > WITSML tab. See "Setting up WITSML data".

To import real-time survey data:

  1. Go to the Wellpath tab on the Setup > Well Data screen.
  2. Select the Survey option in the drop-down menu at the bottom of the screen.
  3. Click the Import Wellpath button ().
  4. In the menu that appears, select the WITSML channel and trajectory log that contains the real-time survey data.
  5. A list of import options will appear. Select the appropriate options for measured depth, inclination, and azimuth.
  6. Select the Auto Import option, then select the frequency with which you want real-time surveys to be updated.
  7. In the Tie-On Point field, enter the measured depth of the tie-on point.
  8. Click the OK button (), then click the Save Changes button ().

Any existing survey points below the tie-on point will be deleted and replaced with the new survey points from the WITSML server. Survey points above the tie-on point will not be affected. Victus™ Operation will periodically query the WITSML server and update the list of real-time surveys.

After you import the real-time surveys, the next step is to enter the formation data. See "Describing the formation".
