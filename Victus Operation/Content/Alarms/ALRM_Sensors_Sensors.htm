Sensor alarms

The Alarms > Sensors > Sensors tab shows the status of communication between Victus™ Operation and the Victus™ MPD System sensors. Colored boxes are used to indicate the current status of communication with each sensor.

  • A green box indicates that the sensor is successfully communicating with Victus™ Operation.
  • A red box indicates that the sensor is not communicating with Victus™ Operation.

The table below defines the sensor alarms shown on the Alarms > Sensors > Sensors tab.
