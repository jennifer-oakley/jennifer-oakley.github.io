About RCD communication

The Alarms > Sensors > RCD tab is used to enable or disable communication between Victus™ Operation and the Victus™ MPD System. When communication is enabled, the RCD provides data that can be used to decide when to open or close the RCD remotely.

Before remote communication to the RCD can be enabled in Victus™ Operation, the HMI human machine interface that controls the RCD must be set to remote mode. If the HMI is in local mode, the RCD can not be controlled from Victus™ Operation. You will only be able to view data sent by the RCD. For more information about the HMI, refer to the operation manual for the RCD.

When Victus™ Operation opens, it checks for the connection to the RCD. If the RCD is already connected, it will automatically be enabled. If the RCD is not connected when you open Victus™ Operation, you'll need to enable it. See "Enabling communication to the RCD".

When communication to the RCD is enabled, the RCD tab will list any alarms received from the HMI. Refer to the operation manual for a list of solutions to these errors.

Note

Currently, the only RCDs supported by Victus™ Operation are the SeaShield™ 7875 DS, 7875 BTR, and BTR-S RCD. Download the SeaShield™ 7875 RCD Operation and Maintenance Manual.

Enabling communication to the RCD

To enable communication to the RCD:

  1. Press Ctrl + F7 on the HMI to enable remote mode.
  2. In Victus™ Operation, go to the Setup > Surface Equipment > Equipment Configuration tab.
  3. Select the rig type and the RCD used on the rig.
  4. Click the Save Changes button ().
  5. Go to the Alarms > Sensors > RCD page.
  6. Click the Enable RCD Communication button () at the bottom of the screen.
  7. When prompted, click Yes to enable RCD communication.

The RCD status will change to Enabled. Victus™ Operation will start reading data from the RCD, and the operators will be able to control the RCD using the RCD controls. Any alarms received from the HMI will be listed on the RCD tab. See "RCD controls".

Disabling communication to the RCD

To disable communication to the RCD:

  1. Go to the Alarms > Sensors > RCD page.
  2. In the bottom, right corner of the RCD page, click the Disable RCD Communication button ().
  3. When the Disable RCD Communication prompt appears, click Yes.

The RCD status will change to Disabled. Victus™ Operation will stop reading data from the RCD, and the operators will no longer be able to control the RCD remotely.

RCD controls

After communication to the RCD is established, you need to add the RCD control tiles to a dashboard to control the RCD. See "Dashboard control tiles".

Victus™ Operation has the following RCD control tiles:

  • The Pump control () allows you to turn the RCD HPU pump on or off remotely.
  • The Bearing Latch control () allows you to open, close, or disable the primary bearing latch remotely.
  • The Sec. Latch control () allows you to open or disable the secondary bearing latch remotely. The secondary bearing latch is used only when there is a problem with the primary bearing latch. The Sec. Latch control can not be used to close the secondary bearing latch.
  • The Transducer control () allows you to switch the source of the active wellbore sensor between the primary and secondary pressure transducer remotely.
  • The LPS control () allows you to enable or disable the latch position sensor remotely.

Viewing RCD data

There are two ways to view the data received from the RCD:

  • Add the RCD indicator to a dashboard. The RCD indicator displays all of the parameters for the RCD, such as sensor readings from the bearing latch and the secondary latch. See "Dashboards".
  • Add RCD curves to a time-based graph. The curves for the RCD parameters are listed under the RCD Parameters category in the Dashboard Editor panel. See "Graphs".
