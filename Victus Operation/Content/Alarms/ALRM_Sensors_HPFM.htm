About the high pressure flow meter alarms

The Alarms > Sensors > HPFM tab shows the status of communication between Victus™ Operation and the high pressure flow meters, as well as details and parameters. Colored boxes are used to indicate the current status of communication and alarms from each active high pressure flow meter. The HPFM tab is only visible when at least one of the HPFM options is selected on the Setup > Surface Equipment > Equipment Configuration tab. See "Equipment configuration".

Note

If the communications alarm, clog alarm, or wear alarm are triggered for any HPFM, the HPFM totalizer calculation will stop running. See "OneClick™ calculations".

Alarms and other information

The table below defines the high pressure flow meter alarms shown on the HPFM tab.

Alarms shown in the HPFM tab
Alarm Definition
Comms Indicates the communication status with the high pressure flow meter.
Clog Alarm Flow path through the meter is obstructed or partially blocked.
Wear Alarm Indicates certain components of the flow meters are experiencing wear.
Qmain Flowrate Primary measurement of fluid flowing through the meter.
Compressibility Factor Compressibility factor of the fluid being measured, which is used to correct the flow rate for compressibility effects.
Corrected Flowrate Flow rate after applying any necessary corrections.

The table below defines the fields shown in the HPFM Details section on the HPFM tab.

Fields shown in the HPFM Details section
Field Definition
HPFM PLC Software Version Current PLC software version.
User Input N (Behavior Index) Current behavior index value. This value is not used by the hydraulics model.
User Input K (Consistency Index) Fluid consistency index. This value is not used by the hydraulics model.
User Input Ty (Sheer Stress) Sheer stress within the fluid. This value is not used by the hydraulics model.

The table below defines the fields shown in the HPFM Parameters section on the HPFM tab.

Fields shown in the HPFM Parameters section
Field Definition
Volumetric Flowrate Current volumetric flow rate
Mass Flowrate Mass flow rate of the fluid
Standard Density Standard density of the fluid
Drive Gain Drive gain percentage
Drive Stability Stability of the drive control output in %
Ampl Stability Stability of the amplitude input in %
Frequency Stability Stability of the sensor oscillation frequency in percent
Left Pickup Coil Voltage level of the left pickup coil
Right Pickup Coil Voltage level of the right pickup coil
Adc Tube Mean Temp Actual RHM tube temperature.
Error Status Error code of persistent errors in the register.
Soft Error Error code of transient error causes included in the register.
