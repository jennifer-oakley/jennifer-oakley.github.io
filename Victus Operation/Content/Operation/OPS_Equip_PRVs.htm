About pressure relief valves

Pressure relief valves protect the riser and surface equipment by keeping pressure within safe operating limits. There are three types of pressure relief valves that can be controlled by Victus™ Operation:

  • T3 PRV. See "Connecting the T3 PRV".
  • Weatherford SPMS.See "Connecting the Weatherford SPMS".
  • Weatherford SPMS Gen 2. See "Connecting the Weatherford SPMS Gen 2".

Connecting the T3 PRV

For the T3 PRV, Victus™ Operation uses a soft set point to stabilize surface backpressure. The soft set point is set at 94% of the high limit. Victus™ Operation will continuously open and close the PRV as needed to keep surface backpressure below the soft set point.

The PRV also has a hard set point, which can only be configured at the local valve control. The hard set point should be above the maximum high limit. If the hard set point is exceeded, the valve will open to release the excess pressure.

To connect the T3 PRV to Victus™ Operation, there are four basic steps:

  1. Select the T3 PRV option in the rig configuration. See "Equipment configuration".
  2. Verify communication between the PRV sensors and Victus™ Operation. See "Sensor diagnostics".
  3. Set the high limit for surface backpressure. See "Pressure Limit control".
  4. Enable the soft set point for the PRV. See "T3 PRV".

Victus™ Operation will begin sending and receiving data through the ICU to the T3 PRV. To view incoming PRV data, click the Indicators View button () in the Utilities toolbar and select the PRV option. See "PRV indicator".

Connecting the Weatherford SPMS

The PRV can be set to automatic or open mode. In Auto mode, the PRV valves will remain closed until the PRV set point is exceeded. When the PRV set point is reached, the PRV values will open to release pressure. The valves will close again when pressure has decreased to 80% of the PRV set point.

In Open mode, the PRV valves will open and remain open until the PRV control mode is changed back to Auto mode. Open mode should only be used for testing and inspection purposes. Auto mode should always be used when operating with pressurized mud.

For the Weatherford SPMS, there are two PRV set point options: static or dynamic. The PRV opens or closes as needed to prevent over-pressure.

In dynamic mode, the PRV set point is automatically calculated using the MAASP maximum allowable annular surface pressure value minus the offset value. The offset value is 100 psi by default, and it cannot be used with a static set point. The calculated set point will be used as long as it does not surpass the equipment protection level. It is possible to set the PRV point to static, but we strongly advise you to use caution and consider the potential risks before proceeding.

To set a static set point, select the SP tile, enter a value and press the Enter button (). An orange line will appear on the window to indicate a static set point is now in use. To go back to dynamic mode click the Undo () button and the orange line will disappear.

To connect the Weatherford SPMS to Victus™ Operation:

  1. Select the Weatherford SPMS option in the rig configuration. See "Equipment configuration".
  2. Verify communication between the PRV sensors and Victus™ Operation. See "Sensor diagnostics".
  3. Enable the pressure transducers and configure the PRV set point. See "Weatherford SPMS".

Victus™ Operation will begin sending and receiving data directly from the Weatherford SPMS. To view incoming PRV data, click the Indicators View button () in the Utilities toolbar and select the PRV option. See "PRV indicator".

Download the Weatherford PRV Operation and Maintenance Manual

Connecting the Weatherford SPMS Gen 2

To connect to the Weatherford SPMS Gen 2 web interface:

  1. Select the Weatherford SPMS Gen 2 option in the rig configuration. See "Equipment configuration".
  2. Click the Indicators button () in the Utilities toolbar.
  3. Select the SPMS option.

The SPMS Gen 2 window will open. The PRV control tile will display N/A because the SPMS Gen 2 does not send Victus™ Operation any data to display.
