About the surge-swab tool

The surge-swab tool on the Tools > Calculators > Surge-Swab tab is used to calculate changes in annular pressure due to pipe movement when tripping in or out of the hole. The trip profile shows the optimum tripping speed throughout the entire trip, as well as acceleration rates and fluctuations in bottomhole pressure.

To use the surge-swab tool:

  1. Go to the Surge-Swab tab on the Tools > Calculators page.
  2. In the Trip Type field, select the trip direction.
    • For tripping in, select Downwards (Surge).
    • For tripping out, select Upwards (Swab).
  3. In the Stand Length field, enter the length of one stand of pipe.
  4. In the Max Pipe Velocity field, enter the maximum allowed running speed during the trip.
  5. In the Pipe End State field, select the pipe configuration for the trip.
  6. In the Time To Steady Velocity field, enter the acceleration time required to reach the running speed.
  7. Enter the limits for bottomhole pressure in the Min BHP and Max BHP fields.
  8. In the Optimization Option field, select an option for optimizing running speed.
    • The Running Speed Optimization option calculates the fastest running speed possible within safe operating limits.
    • The Fixed Running Speed uses a constant running speed throughout the entire trip.
  9. Click the Calculate button () to begin the surge-swab calculations.

When the surge-swab calculations have completed, the trip profile will automatically load on top of the surge-swab settings. If you want to re-run the calculations with different settings, click the Back to Input button () and edit the settings as needed.

Field definitions

The table below defines the settings required for the surge-swab tool.

Settings for the surge-swab tool
Field Name Definition
Trip Type

Type of trip, based on the trip direction.

  • Downwards (Surge) – Calculates surge effects when tripping in.
  • Upwards (Swab) – Calculates swab effects when tripping out.
Stand Length Length of one stand of pipe
Max Pipe Velocity Maximum running speed during the trip
Pipe End State

Pipe configuration for the trip.

  • Auto-Fill - Allows fluid to enter the annulus when pulling out the drillstring.
  • Close – Does not allow fluid to enter and exit the drillstring during the trip.
  • Open – Allows fluid to enter and exit the drillstring during the trip.
Time To Steady Velocity Acceleration time required to reach the running speed
Min BHP Minimum allowable bottomhole pressure during the trip
Max BHP Maximum allowable bottomhole pressure during the trip
Optimization Option

Option for optimizing running speed.

  • Running Speed Optimization – Calculates the optimum running speed, using the values in the Max Pipe Velocity and Min Pipe Velocity fields.
  • Fixed Running Speed – Uses a constant running speed for the entire trip, using the value in the Max Pipe Velocity field.

The table below defines the values in the trip profile, which are calculated by the swab-surge tool.

Calculated values for the surge-swab tool
Column Definition
Time Amount of time elapsed since the start of the trip
Bit Depth Depth of the bit at each time step
Acceleration Average acceleration of the trip speed
Velocity Current running speed at each time step
Optimal Tripping Velocity Optimum running speed for the trip
BHP Bottomhole pressure at each time step
Equivalent MW Equivalent mud weight of the fluid
Displaced Fluid Volume Running total of the fluid volume that has been displaced during the trip

Using the Surge-Swab page

The toolbar at the bottom of the screen allows you to begin Surge-Swab calculations and import the current operating parameters. When you are viewing the calculation results, additional buttons will appear in the bottom toolbar.

Toolbar buttons on the Surge-Swab tab
Button Function
Resets the Surge-Swab settings to the default values
Begins Surge-Swab calculations
Closes the calculation results and returns to the Surge-Swab settings
Pins the chart legends to the top of the Surge-Swab charts
