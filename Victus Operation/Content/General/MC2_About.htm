About Victus™ Operation

Welcome to Victus™ Operation v2.3.9.1. Victus™ Operation is part of the Victus™ application suite, which gives you a single platform for managing pressure during drilling operations, simulating drilling operations, and monitoring incoming data. Victus™ Operation controls MPD managed pressure drilling equipment on the rig, gathers information from sensors and control systems, automatically detects kicks and losses, and alerts you when a potential problem occurs downhole.

Victus™ Operation incorporates the Victus™ MPD System with existing pressure control equipment, such as RCDs, chokes, degassers, pressure sensors, and flow meters. This allows you to switch between MPD techniques and conventional drilling at any time, based on current drilling conditions.

Use Victus™ Operation to:

  • Remotely control the equipment connected to the RCD. See "RCD communication".
  • Monitor incoming real-time data from WITS or WITSML sources. See "Data acquisition".
  • Create custom dashboards, using Victus™ Operation's powerful dashboard customization feature. See "Dashboards".
  • View incoming sensor data from the RCD.
  • Define custom calculations and control algorithms. See "OneClick™ settings" and "Control Algorithms".
  • Define custom limits and alarms for sensors, WITS data, WITSML data, and well parameters. See " Limits and Alarms".
  • Manage MPD equipment using PRV pressure relief valve controls, manual choke controls, and set point controls for surface backpressure, standpipe pressure, annular pressure, and flow control. See "Pressure controls".
  • Automatically detect and control kicks and losses in the well. See "Kick algorithm" and "Loss algorithm".
  • Create automated daily operations reports. See "Feedback".
  • Create automated daily RCD reports. See "RCD daily reports".
  • Analyze the performance of the RCD. See "RCD performance reports".
