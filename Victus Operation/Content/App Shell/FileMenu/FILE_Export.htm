About exports

On the File > Export File page, you can export data from the current well. When you export data from a Victus™ Operation project, a compressed MCD file will be created. Victus™ project files are compressed files that can contain different types of data, such as dashboards, drillstrings, and well configurations. MCD files can be exported to your computer or to a network location.

There are several options for exporting data from Victus™ Operation:

  • The Configuration option allows you to export all of the setup data for the well. See "Exporting the well configuration".
  • The Configuration + Data option allows you to export up to 24 hours of well data, as well as the setup data. See "Exporting the configuration and data".
  • The Dashboards option allows to export only the dashboards for the well. See "Exporting dashboards".
  • The Data (CSV) option allows you to export well data to a CSV file. See "Exporting well data to a CSV file".
  • The BHA/Drill String option allows you to export only the work strings on the Setup > Work String page. See "Exporting work strings".
  • The Logs and Databases option allows you to export the Victus™ Operation logs, your Victus™ Operation database, and your well configuration to a zip file. See "Exporting logs and databases".

Exporting the well configuration

To export the well configuration data:

  1. Go to the File > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Configuration option.
  4. Click the Export button () at the bottom of the screen to begin the export.

Victus™ Operation will return to the Home screen. When the export completes, a notification message will appear in the banner at the bottom of the screen. A compressed MCD file containing the well configuration will be created in the selected folder location.

Exporting the configuration and data

To export all data in the well:

  1. Go to the File > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select theConfiguration + Data option.
  4. In the From field, select the starting date and time for the data you want to export.
  5. In the To field, select the ending date and time for the data you want to export.
  6. Click the Export button () at the bottom of the screen to begin the export.

Victus™ Operation will return to the Home screen. When the export completes, a notification message will appear in the banner at the bottom of the screen. A compressed MCD file containing the well configuration and data will be created in the selected folder location.

Exporting dashboards

To export the dashboards for the current well:

  1. Go to the File > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Dashboards option.
  4. Select the dashboards you want to include in the export.
  5. Click the Export button () at the bottom of the screen to begin the export.

Victus™ Operation will return to the Home screen. When the export completes, a notification message will appear in the banner at the bottom of the screen. A compressed MCD file containing the selected dashboards will be created in the selected folder location.

Exporting well data to a CSV file

To export all data in the well to a CSV file:

  1. Go to the File > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Data (CSV) option.
  4. In the From field, select the starting date and time for the data you want to export.
  5. In the To field, select the ending date and time for the data you want to export.
  6. In the Category table, select the data that you want to include in the export.
  7. Click the Export button () at the bottom of the screen to begin the export.

Victus™ Operation will return to the Home screen. When the export completes, a notification message will appear in the banner at the bottom of the screen. A CSV file containing the selected data will be created in the selected folder location.

Exporting logs and databases

Exporting logs and databases allows you to easily share important troubleshooting information with the support team when you encounter issues with Victus™ Operation. There are three options when exporting logs and databases:

  • Logs – Exports all agent logs, DAQ logs, and panel logs.
  • Databases – Exports the primary database and all archive databases.
  • Configuration MCD – Exports the configuration of the current well.

To export logs and databases:

  1. Go to the File > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Logs and Databases option.
  4. Select the items you want to include the export.
  5. Click the Export button () at the bottom of the screen to begin the export.

Victus™ Operation will return to the Home screen. When the export completes, a notification message will appear in the banner at the bottom of the screen. A zip file containing the selected items will be created in the selected folder location.
