About new wells

All of the data related to a job is stored within a well in the Victus™ Operation database. You can create a well using the default configuration, which uses a set of default data that includes surface equipment, well data, drillstrings, and fluids. You can also copy the configuration of another well in Victus™ Operation. The new well will be an exact copy of the original well, but without data acquired through sensors, WITSML, modbus, or WITS. See "Creating a well".

Creating a well

To create a new well:

  1. Go to the File > New page.
  2. Enter the project name and the well name.
  3. In the Based On Well field, select the well configuration you want to copy. To use the default configuration, select the Default option.
  4. Enter the remaining job and well information.
  5. Click the Save Changes button () at the bottom of the screen to save the project.

The well will be added to the Victus™ Operation database, and it will automatically become the active well. The new well will open on the Home screen, and the project name will appear in the banner at the top of the screen. All of the information entered on the File > New page will appear on the Setup > Job Info page.

Note

Adding a well to Victus™ Operation does not add it to any of the other Victus™ applications. To add the new well to another Victus™ application, you'll need to export the well from Victus™ Operation, then import it into another Victus™ application. See "Exports".

Field definitions

The table below defines all of the fields on the File > New page.

Field definitions for the File > New page
Field Definition
Project Name Any name to identify the project, such as the Weatherford file number
Based On Well Creates a new well based on the configuration of the selected well
Well Name The well name should match the well name in Spectrum, if available
Lease Name of the lease where the well is located
Field Name of the field where the well is located
Unique Well ID UWI code that defines the bottomhole location
Well License Any non-API license or permit number assigned to the well
API Number API well number assigned by the appropriate government body
Rig Name Name of the rig at the well site
Operator Company Name of the company that is drilling the well
Rig Contractor Name of the company that owns the rig
Job Type Type of job
Country Country where the well is operating
Region Region where the well is operating
Latitude Geographic coordinate for the latitude of the well's location, entered in decimal format
Longitude Geographic coordinate for the longitude of the well's location, entered in decimal format
Time Zone UTC offset for the time zone where the well is located
Spud Date Date on which a machine capable of reaching total depth begins drilling a new well
