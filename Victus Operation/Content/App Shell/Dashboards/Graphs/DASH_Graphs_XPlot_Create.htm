About creating cross plots

After you add a cross plot to a dashboard, you'll need to add tracks and curves to the graph. You'll also need to define the ranges for the Z-axis.

  • Tracks separate the graph into smaller sections and function as a container for the curve data. By default, graphs include one track. However, you add more tracks as needed. See "Adding tracks to a cross plot".
  • Curves contain the data that you are graphing. In the Dashboard Editor panel, curves are categorized according to the type of data. Each track can contain multiple curves. See "Adding curves to a track".
  • The Z-axis gives context to the data plotted for each track. The data in each track is color-coded according to the ranges defined for the Z-axis. See "Defining ranges for the Z-axis".

Curves from certain categories cannot be added to cross plots. If a curve can't be added to the graph, the cursor will change to the Disabled icon ().

Defining ranges for the Z-axis

To define the ranges used for the Z-axis:

  1. On the Home screen, select the dashboard that contains the cross plot you want to edit.
  2. Hover your mouse over the bottom of the Z-axis, and click the Edit button ().
  3. In the Graph Page Settings window, enter the minimum and maximum values for the Z-axis.
  4. Under the Range 1 heading, enter the maximum value for the first range in the Range To field. To add a new range, enter a value that is less than the maximum value for the Z-axis.
  5. Select a color for the range in the Marker Color and Marker Stroke Color fields.
  6. For each additional range that you want to include, choose a maximum range value and color.

As you change the range values and colors, the cross plot will be updated to reflect the new ranges. When you are finished defining the Z-axis ranges, close the window.

Adding tracks to a cross plot

To add a track to a cross plot:

  1. On the Home screen, select the dashboard that contains the cross plot you want to add a track to.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, locate the first curve that you want to appear in the new track.
  4. Select the name of the curve. Drag it to the graph index, and drop it over the Track label.

A new track will be added to the graph. To edit the track settings, click the Edit Graphs button () in the bottom right corner of the track. See "Updating curve and track settings".

Adding curves to a track

To add a curve to track:

  1. On the Home screen, select the dashboard that contains the cross plot you want to edit.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, locate the curve that you want to add.
  4. Select the name of the curve, and drag it to the track that you want to add the curve to.

The data for the new curve will appear in the selected graph track. To edit the curve settings, click the Edit Graphs button () in the bottom right corner of the track. See "Updating curve and track settings".

Note

Raw data curves, such as mnemonics for a WITS channel, will be deleted from the graph if the associated channel is deleted. However, WITSML curves will be retained if a new channel is created with matching mnemonics.
