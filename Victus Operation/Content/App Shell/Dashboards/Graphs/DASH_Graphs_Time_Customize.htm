About customizing time graphs

There are several ways that you can customize the appearance of a time-based graph.

  • Change the range of the data shown. The Range button () sets the overall range of real-time data to display. When you select an option from the Range menu, the increment will be determined by the size of the range. You can also apply a manual time range. See "Applying a manual range".
  • Update the overall graph settings. The general graph settings control the appearance of the legend, axes, and grid lines for all of the tracks in the graph. See "Updating graph settings".
  • Update the curve and track settings. The track settings control the appearance of the caption, grid lines, and annotations for the curves in the track. The curve settings control the appearance of the line and markers for the curve, as well as scaling and smoothing options. See "Updating curve and track settings".
  • Add annotations. Annotations are notes or comments for a specific point on a curve. See "Adding annotations".

Applying a manual range

There are multiple ways to apply a manual time range to a time-based graph:

  • Manually enter a time frame. To change the start date, click the Calendar button () in the From field. To change the end date, click the Calendar button () in the To field.
  • Choose a bit run. To select a time range based on a bit run, click the Bit Runs button and select the bit run you want to view data for. Bit runs are defined in the Bit Runs indicator. See "Bit runs".

To apply a manual range to a graph on the Home screen:

  1. Go to the Home screen, then select a dashboard.
  2. Hover your mouse over the bottom right corner of the graph you want to filter. A toolbar will appear on top of the graph index.
  3. Click the Manual Range button () to open the Time Range window.
  4. Select the time range that you want to view data for.
  5. Click the Set Range button () to apply the filter.

The data from the selected time range will load in the curves on the current graph. Gray vertical bars will appear in the middle of each track to indicate that the data is not displayed in real-time.

Tip

The end date for the time range can also be set to a future date.

Updating graph settings

The general time-based graph settings allow you to customize the appearance of the legend, axes, and grid lines in the graph. Any changes made to the general settings will affect all of the tracks in the time-based graph. If the dashboard contains multiple graphs, these changes will not affect the other graphs in the dashboard.

To update the settings for the tracks in a time-based graph:

  1. On the Home screen, select the dashboard that contains the time-based graph you want to edit.
  2. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  3. Click the Edit Graph Page button () to open the Graph Page Settings window.
  4. Update the general settings as desired.

As you make changes to the graph, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

Updating curve and track settings

The Graph Page Settings window contains several groups of settings for each track in the time-based graph:

  • The Track tab contains settings for the caption, grid lines, and annotations.
  • The Curve tab contains settings for each curve in the track, including scaling options, line and marker colors, and smoothing methods.
  • The Axis tab contains settings for the axis of each curve, such as the axis direction, measurement unit, and tick mark options.

To update the settings for one of the tracks in a time-based graph:

  1. On the Home screen, select the dashboard that contains the time-based graph with the track you want to edit.
  2. Hover your mouse over the bottom right corner of the track. A toolbar will appear on top of the graph.
  3. Click the Edit Graphs button () to open the Graph Page Settings window.
  4. Update the settings as desired.
    • To edit the track settings, go to the Track tab.
    • To edit curve settings, go to the Curve tab and select the curve you want to edit.
    • To edit axis settings, go to the Axis tab and select the curve you want to edit.

As you make changes to the track, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

Adding annotations

Annotations allow you to add notes and comments to a specific point on a curve in a graph.

To add an annotation to a curve:

  1. On the Home screen, select the dashboard that contains the time-based graph you want to add a comment to.
  2. Locate the point on the curve where you want the annotation to appear, then click and hold the mouse button.
  3. Enter your comment in the text box that appears, then press the Enter key.

The annotation will appear in a yellow box, at the selected point in the curve. To edit the appearance of the annotation, go to the Annotations section of the track settings. See "Updating curve and track settings". To delete the annotation, click on the annotation and press the Delete key.
