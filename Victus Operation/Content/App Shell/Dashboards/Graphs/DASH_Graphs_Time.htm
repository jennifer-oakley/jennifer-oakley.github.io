About time graphs

Time graphs plot curve data against time. Time graphs have two axes:

  • The primary axis is the graph index, time. The data in every track is plotted against the data for the graph index. In vertical graphs, the graph index is on the right side of the screen. In horizontal graphs, the graph index is at the bottom of the screen.
  • The secondary axis contains the curve data. The curve data is plotted against the graph index. In vertical graphs, the secondary axis is at the top of the screen. In horizontal graphs, the secondary axis is on the left side of the screen.

Adding time graphs to a dashboard

To add a time-based graph to a dashboard:

  1. On the Home screen, select the dashboard that you want to add a time-based graph to.
  2. Click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, expand the Graphs heading.
  4. Click the Vertical Time Based Graph button (), then drag it to a blank area on the dashboard.
  5. If you want the graph to span across multiple grid areas, click on the bottom right corner of the graph and drag it into an adjacent blank area.

A blank graph area will appear on the dashboard, which you can add tracks and curves to. See "Adding tracks to a graph" and "Adding curves to a track".

Tip

If there aren't any blank areas on the dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

Using time graphs

The toolbar for the graph is used to take screenshots, set a display range, and customize the settings for all of the tracks in the graph. To open the toolbar for the graph, hover your mouse over the bottom right corner of the graph index.

Toolbar buttons for graphs
Button Function

Resets the graph range to the default range
Opens the Graph Page Settings window for editing the general graph settings. See "Updating graph settings".

Exports all chart data to a CSV or LAS file. The export will include all data in the current range, at one-second intervals.
Captures a screenshot of the graph. Screen captures are stored in your Victus folder, under the C:\Users\[UserName]\Documents\Victus\Images directory. The name of each screen capture includes the date and time that the screen capture was taken.

Sets the range of real-time data to display
Sets the range of historical data to display. Real-time updates are disabled while using a historical data range. See "Applying a manual range".

Each track in the graph also has its own set of toolbar buttons. To open the track toolbar, hover your mouse over the bottom right corner of the track.

Toolbar buttons for tracks and curves
Button Function
Opens the Graph Settings window for editing the track and curve settings. See "Updating curve and track settings".
