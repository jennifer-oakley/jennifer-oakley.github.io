Control Tiles

The tiles at the bottom of the Well View pane allow you to control the chokes and PRV, as well as manage pressure, view current pump rates, and enable kick/loss detection. There are six control tiles in Victus™ Operation:

  • The Pressure tile controls the way Victus™ Operation manages pressure in the well. See "Pressure controls".
  • The Chokes tile controls the chokes on the MPD manifold. See "Chokes control".
  • The Pumps tile shows the current pump rates for all of the rig pumps. See "Pumps control".
  • The PRV tile is used to automatically control the PRV on the rig. See "PRV control".
  • The High Limit tile. Pressure limits are used to control the maximum and minimum values for surface backpressure. See "Pressure Limit control".
  • The Detection tile is used to enable automatic kick/loss detection. See "Detection control".
