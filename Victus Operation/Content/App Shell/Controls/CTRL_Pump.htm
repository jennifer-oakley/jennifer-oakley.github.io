About the Pumps control

The Pumps tile shows the current pump rates for all of the rig pumps. There are four possible sources for this information:

  • Pump data, such as data from the stroke counter or spare sensors.
  • Flow rate data from low-pressure flow meters.
  • Flow rate data from high-pressure flow meters.
  • External data, such as data from WITS or analog sensors.

The pumps listed on the Pumps tile are numbered, labeled, and color-coded according to the equipment configuration on the Setup > Surface Equipment > Piping and Instrumentation tab. See "Equipment specifications".

  • Main pumps are labeled in green, using the RIG label.
  • Auxiliary pumps are labeled in yellow, using the AUX label.
  • Boost pumps are labeled in orange, using the BST label.
  • Disabled or inactive pumps are labeled in gray, using a dash for the label.

Specifying the pump source

To specify the pump source for pump rate:

  1. Go to the Piping and Instrumentation tab on the Setup > Surface Equipment page.
  2. Select the pump in the rig schematic. A panel with the pump properties will appear.
  3. In the Pump Source field, select the desired option.
  4. Click the Save Changes button () to save your changes.

If you are using an external data source such as WITS / WITSML to obtain the pump rate, go to the Setup > Mappings > Parameters tab, then select the data source you want to use for each pump. See "Defining well parameters".

If you are using a low-pressure flow meter, additionally go to the Equipment Configuration tab and enable the Low Pressure Flow Meters box. See "Field definitions".

If you are using a high-pressure flow meter, go to the Setup > Surface Equipment page and select the main high-pressure flow meter the pump will connect to in the Main Flow In Source field. See "Connecting high-pressure flow meters to pumps".
