About custom panels

Custom panels are standalone windows that can be re-sized and moved anywhere on your screen. Custom panels contain gauges, which give you a quick visual indication of the status for any parameter, calculated value, analog sensor, or WITS/WITSML curve. The gauges in a custom panel function the same way as gauges in a dashboard. See "Gauges".

Gauges are most useful when they are connected to limits and alarms. When alarm limits are enabled, yellow and red areas will appear on the gauge to indicate when an alarm limit has been exceeded. To connect a gauge to a limit, you'll need to enable the alarm limits for that parameter on the Setup > Limits and Alarms page. See " Limits and Alarms".

Note

Unlike dashboards, custom panels can only contain gauges. Other items, such as graphs, can not be added to a custom panel.

Creating a custom panel

To create a custom panel:

  1. Click the Indicators button () in the Utilities toolbar.
  2. Go to the Custom Panels option.
  3. In the menu that appears, select the NEW option.

A blank custom panel will open in a new window. This window can be re-sized and moved to anywhere on your screen. After creating the custom panel, you'll need to add gauges to the panel. See "Adding gauges to a custom panel".

Opening custom panels

To open a custom panel that you've already created:

  1. Click the Indicators button () in the Utilities toolbar.
  2. Go to the Custom Panels option.
  3. Select the custom panel that you want to open.

The selected custom panel will open in a new window.

Adding gauges to a custom panel

To add a gauge to a custom panel:

  1. Open the custom panel that you want to add a gauge to. See "Opening custom panels".
  2. On the Home screen, click the Edit Dashboard button () in the bottom left corner of the screen.
  3. In the Dashboard Editor panel that appears on the left, locate the curve that you want to create a gauge for.
  4. Select the name of the curve and drag it the custom panel.

The new gauge will appear on the custom panel. You can change the type of gauge, as well as customize the units and limits applied to the gauge. See "Changing the gauge type" and "Customizing units and limits".

Changing the gauge type

To change the type of gauge:

  1. Open the custom panel that contains the gauge you want to update.
  2. Hover your mouse over the top left corner of the gauge. The Menu button () will appear on top of the gauge.
  3. Click the Menu button (), then select the type of gauge you want to use.

The gauge will be updated to the selected type of gauge.

Customizing units and limits

To customize the units and limits applied to a gauge:

  1. Open the custom panel that contains the gauge you want to update.
  2. Hover your mouse over the top left corner of the gauge. The Menu button () will appear on top of the gauge.
  3. Click the Menu button (), then click the Options button ().
  4. To use a different measurement unit to the gauge, select the unit you want to use in the Unit field.
  5. To apply a different set of limits to the gauge, enter the desired limits in the Minimum, Min Severe, Min High, Max Severe, and Maximum fields.

The gauge will be automatically updated as you make changes. The custom settings applied to the gauge will override the settings entered on the Setup screen. However, the gauge settings will only be applied to that particular gauge. If you have another gauge for the same parameter, these settings will not be applied to it.
