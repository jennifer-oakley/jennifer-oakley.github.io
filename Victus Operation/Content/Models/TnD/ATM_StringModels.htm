About string models

In Victus™ Operation, torque and drag analysis supports four models for predicting the effects of torque and drag:

  • The simplified soft string model is the most basic model, featuring instantaneous calculations. This model considers the work string to be a perfectly flexible rod and approximates the effect of helical buckling on drag. See "Simplified soft string model".
  • The soft string model is similar to the simplified soft string model, but also features updated buckling models and modified stress calculations. See "Soft string model".
  • The semi-stiff string model is based on the soft string model, but also takes bending stiffness into account and includes the calculations for bending moments, tool joint positions, and contact force at the tool joints and pipe body. See "Semi-stiff string model".
  • The stiff string model is a completely separate string model, with a transient model of lateral motion. This model includes calculations for bending moments, tool joint positions, contact force at the tool joints and pipe body, axial and torsional movement, and lateral movement and vibration. See "Stiff string model".

Simplified soft string model

Assumptions:

  • The work string is considered to be a perfectly flexible rod.
  • The effect of helical buckling on drag is approximate.
  • The drill pipe is considered to have a constant OD and ID, without accounting for differences between tool joints and the body.
  • Bending moments are not taken into account.
  • There is a calculation data point every 30 ft.

Features:

  • Instantaneous calculation.
  • Validated results with field data.

Soft string model

Assumptions:

  • The work string is considered to be a perfectly flexible rod.
  • The effect of helical buckling on drag is approximate.
  • The drill pipe is considered to have a constant OD and ID, without accounting for differences between tool joints and the body. The OD of the tool joints is only taken into account when calculating the helical buckling force.
  • Bending moments are not taken into account.
  • There is a calculation data point every connection.

Features:

  • Updated buckling models from the simplified soft string model.
  • Modified stress calculations from the simplified soft string model.
  • Calculations completed within one minute.
  • Validated results with field data.

Semi-stiff string model

Assumptions:

  • Based on the soft string model, but approximates the string bending stiffness.
  • Includes the calculation of bending moments, tool joint positions, and contact force at the tool joints and pipe body.
  • There is a calculation data point every 10 ft.

Features:

  • Updated buckling models from the simplified soft string model.
  • Calculations completed within five minutes.
  • Validated results with field data.

Stiff string model

Assumptions:

  • Completely separate transient model of lateral motion. See "Equations of motion".
  • Includes the calculation of bending moments, tool joint positions, contact force at the tool joints and pipe body, axial and torsional movement, and lateral movement and vibration.
  • There is a calculations data point every foot.

Features:

  • Updated buckling models from the simplified soft string model.
  • Ability to model vibration.
  • Ability to model rigid and flexible centralizers.
  • Bending stiffness taken into account.
  • True effect of contact with wellbore wall.
  • Calculations completed within 20 minutes.
  • Validated results with field data.
