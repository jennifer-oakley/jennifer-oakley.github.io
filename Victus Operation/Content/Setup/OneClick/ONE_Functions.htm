Arithmetic functions

The table below defines the calculation functions available under the Arithmetic tab in the Expression pane.

Definitions of arithmetic functions
Function Definition
A + B Returns the sum of A and B
A - B Returns the difference between A and B
A x B Returns the product of A and B
A / B Returns the quotient of A and B
A mod B Returns the remainder after dividing A by B
|A| Returns the absolute value of A
A + Inserts an addition function on the left side of the selected function or value
A - Inserts a subtraction function on the left side of the selected function or value
A x Inserts a multiplication function on the left side of the selected function or value
A / Inserts a division function on the left side of the selected function or value
A mod Inserts a modulo function on the left side of the selected function or value
Sqrt (A) Returns the square root of A
+ B Inserts an addition function on the right side of the selected function or value
- B Inserts a subtraction function on the right side of the selected function or value
x B Inserts a multiplication function on the right side of the selected function or value
/ B Inserts a division function on the right side of the selected function or value
mod B Inserts a modulo function on the right side of the selected function or value
^B Inserts an exponent to the selected function or value

Comparison functions

The table below defines the calculation functions available under the Comparison tab in the Expression pane.

Definitions of comparison functions
Function Definition
A > B Returns true if A is greater than B
A ≥ B Returns true if A is greater than or equal to B
A < B Returns true if A is less than B
A ≤ B Returns true if A is less than or equal to B
A = B Returns true if A is equal to B
A ≠ B Returns true if A is not equal to B
A > Inserts a greater than function on the left side of the selected function or value
A ≥ Inserts a greater than or equal function on the left side of the selected function or value
A < Inserts a less than function on the left side of the selected function or value
A ≤ Inserts a less than or equal function on the left side of the selected function or value
A = Inserts an equation function on the left side of the selected function or value
A ≠ Inserts a not equal function on the left side of the selected function or value
> B Inserts a greater than function on the right side of the selected function or value
≥ B Inserts a greater than or equal function on the right side of the selected function or value
< B Inserts a less than function on the right side of the selected function or value
≤ B Inserts a less than or equal function on the right side of the selected function or value
= B Inserts an equation function on the right side of the selected function or value
≠ B Inserts a not equal function on the right side of the selected function or value

Logical functions

The table below defines the calculation functions available under the Logical tab in the Expression pane.

Definitions of logical functions
Function Definition
Not (A) Returns true if A is false
A and B Returns true if A and B are true
A or B Returns true if A or B are true
A xor B Returns true if either A or B are true. Returns false if both values are true, or if both values are false.
A and Inserts an and function on the left side of the selected function or value
A or Inserts an or function on the left side of the selected function or value
A xor Inserts an xor function on the left side of the function or value
and B Inserts an and function on the right side of the selected function or value
or B Inserts an or function on the right side of the selected function or value
xor B Inserts an xor function on the right side of the function or value

Flow functions

The table below defines the calculation functions available under the Flow tab in the Expression pane.

Definitions of flow functions
Function Definition
IF ... THEN ... ELSE ... If the first term is true, evaluates the expression in the second term. Otherwise, evaluates the expression in the third term.

Math functions

The table below defines the calculation functions available under the Math tab in the Expression pane.

Definitions of math functions
Function Definition
Cos (A) Returns the cosine of A
Sin (A) Returns the sine of A
Tan (A) Returns the tangent of A
Log (A) Returns the natural logarithm of A
Max (A, B) Returns the maximum value, either A or B
Cosh (A) Returns the hyperbolic cosine of A
Sinh (A) Returns the hyperbolic sine of A
Log10 (A) Returns the base 10 logarithm of A
Min (A, B) Returns the minimum value, either A or B
Acos (A) Returns the arccosine of A
Asin (A) Returns the arcsine of A
Atan (A) Returns the arctangent of A
Exp (A) Returns the exponential function of A

Aggregate functions

The table below defines the calculation functions available under the Aggregate tab in the Expression pane.

Definitions of aggregate functions
Function Definition
Avg (A) Returns the average value of A during the specified duration, offset from the current time by the specified shift
Inc (A) Returns the number of values for A, offset from the current time by the specified shift
Sum (A) Returns the sum of all values for A during the specified duration, offset from the current time by the specified shift
Exist (A) Returns true if the statement for A is true at any time during the specified duration, offset from the current time by the specified shift
Min (A) Returns the minimum value of A during the specified duration, offset from the current time by the specified shift
ḟ (A) Returns the first derivative of A, offset from the current time by the specified shift
Count (A) Returns the number of times the statement for A is true during the specified duration, offset from the current time by the specified shift
Sequence Returns true if the values of A match the pattern specified in the last term during the specified duration, offset from the current time by the specified shift
Max (A) Returns the maximum value of A during the specified duration, offset from the current time by the specified shift
f̈ (A) Returns the second derivative of A, offset from the current time by the specified shift
Totalizer Returns a running total for the values of A
Ā•Δt Returns the area under curve of expression A between the two most recent points of A, using a trapezoidal approximation
