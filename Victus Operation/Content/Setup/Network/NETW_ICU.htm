About the ICU

The Setup > Network > ICU tab allows you to reboot, backup, and restore the configuration for the ICU. The instrumentation and control unit (ICU) is the brain of the MPD manifold. The ICU controls the manifold and manages the data received from sensors. There are two primary components for the ICU:

  • The CU is the control unit. The CU collects and distributes the data from the sensors on the manifold.
  • The IU is the instrumentation unit. The IU monitors and controls the manifold, using the data received from the CU.

Rebooting the IU and CU

To reboot the IU and CU:

  1. Go to the Setup > Network > ICU tab.
  2. Select the IU and click the Reboot button ().
  3. Click Yes on the warning message that appears.
  4. Select the CU and click the Reboot button ().
  5. Click Yes on the warning message that appears.

The IU and CU will be rebooted. A communication error may appear briefly during the reboot.

Creating a backup

To create a backup of the IU and CU:

  1. Go to the Setup > Network > ICU tab.
  2. Select the IU and click the Backup button ().
  3. Click Yes on the warning message that appears.
  4. Select the CU and click the Backup button ().
  5. Click Yes on the warning message that appears.

The backups will be created and automatically saved. IU and CU backups are stored in your Victus folder, under the C:\Users\[UserName]\Documents\Victus\IU-CU-Images directory. A communication error may appear briefly during the backup.

Restoring the IU and CU

To restore a previous configuration of the IU and CU:

  1. Go to the Setup > Network > ICU tab.
  2. Select the IU and click the Restore button ().
  3. Click Yes on the warning message that appears.
  4. In the window that opens, navigate to the IU configuration that you want to restore and click Open.
  5. Select the CU and click the Restore button ().
  6. Click Yes on the warning message that appears.
  7. In the window that opens, navigate to the CU configuration that you want to restore and click Open.

The IU and CU configuration will be restored to the selected configuration. A communication error may appear briefly during the backup.

Using the ICU tab

The toolbar on the ICU tab allows you to reboot, backup, and restore the IU and CU.

Toolbar buttons on the ICU tab
Button Name Description
Reboots the IU and CU
Restores the IU and CU to the selected configuration
Creates a backup of the current IU and CU configuration
