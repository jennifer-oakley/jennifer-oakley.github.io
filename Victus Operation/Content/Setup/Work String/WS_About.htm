About work strings

The Setup > Work String page is used to build the work strings used while drilling and casing a well. Victus™ Operation includes the Weatherford catalog, which is the master catalog for BHA and drillstring components. It contains many of the tools manufactured by Weatherford, such as RSS units, mud motors, and LWD/MWD tools. The Weatherford catalog also contains standard BHA and drillstring parts, such as drill bits, stabilizers, drill pipe, and HWDP. Although you can edit many of these tools, the changes will only be reflected in that particular instance of the tool. The Weatherford catalog itself will not be affected. See "Work string components".

Building a new work string

When you create a work string, you can create it with or without an explicitly-defined BHA.

Work string with an explicitly-defined BHA

  • In a work string with an explicitly-defined BHA, the BHA and drillstring components will be grouped into separate tables.
  • In a work string without an explicitly-defined BHA, all BHA and drillstring components will appear in one table.

To create a new work string:

  1. Go to the Setup > Work String page.
  2. Click the Add button ().
  3. When prompted, select an option for defining the BHA.
    • To create a work string with an explicitly-defined BHA, click Yes.
    • To create a work string without an explicitly-defined BHA, click No.
  4. Click the Insert Tool button () to add components to the work string. See "Adding components".
  5. When you have finished adding components, click the Save Changes button () at the bottom of the screen.

A new work string will be added to Victus™ Operation, and a new numbered tab will appear in the right toolbar. As you add new components, the preview of the complete work string will be updated in the Well View pane. See "Work string components".

Adding components

To add a component to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Select the component below the one you want to add, then click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the type of component you want to add.
  5. Enter the dimensions and other properties of the component. See "Work string components".
  6. Click the OK button () to add the component to the work string.

The new part will appear in the list of components, and the preview of the complete work string will be updated in the Well View pane.

Note

You cannot add tools with 0 ID to the active work string.

Grouping components together

To group work string components into a cluster:

  1. Go to the Setup > Work String > page.
  2. Select a work string using the tabs on the right side of the screen.
  3. For each component that you want to include in the cluster, select the checkbox in the Group column.
  4. Click the Group Tools button (). If this button is not visible, you may have non-sequential components selected.
  5. Click the Save Changes button ().

The selected components will be grouped together into a cluster, and the preview of the complete work string will be updated in the Well View pane. To separate the cluster into individual components, select the cluster and click the Ungroup Tools button ().

Activating a work string

The active work string is the work string that is currently being used in the well. In order to activate a work string, the first component must be a drill bit.

To change the active work string:

  1. Go to the Setup > Work String >page.
  2. Select the work string you want to activate.
  3. Click the Activate Item button (). If this button is not visible, the work string is already active.
  4. Click the Save Changes button ().

The selected work string will be marked as the active work string. The newly active work string will be validated against the wellbore intervals. See "Well intervals".

Note

If the selected work string contains a tool with 0 ID, you cannot activate the work string.

Using the Work String page

There are several keyboard shortcuts and mouse actions that you can use on the Setup > Work String page.

  • To export work string data to a CSV file, press the Ctrl + S keys.
  • To move a component to a different position in the work string, click on the component you want to move and drag it to the new position.
  • To change the name of a work string, go to the toolbar on the right side of the screen and double-click on the numbered tab for the work string. In the box that appears, enter a new name for the work string and press the Enter key.
  • To edit a work string tool, double-click on the tool name in the list of work string components. You can also press the Alt key and click on the tool name.
  • To switch to another tool when the editing panel is open, click on the tool name in the list of work string components. Any changes to currently selected tool will be automatically saved when you select another tool.
  • To close the editing panel, press the Esc key.

The toolbar on the right side of the screen is used to create, copy, and activate work strings. The numbered tabs above the toolbar buttons are used to switch between work strings. Each work string has its own numbered tab.
