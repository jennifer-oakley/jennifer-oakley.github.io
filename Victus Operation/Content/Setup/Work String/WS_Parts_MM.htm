About drilling motors

Drilling motors are usually located above the bit or above the RSS in a motorized RSS system. They are powered by drilling fluid, and turn the bit while the drillstring remains fixed or rotates at a different speed.

The specifications for all Weatherford mud motors listed in the Mud-Lubricated Drilling Motors Handbook are included in the default Weatherford catalog. When you select a drilling motor, the motor specifications will appear in the Work String Component panel. Most of these properties are provided for your reference, and can't be edited. However, you will need to add information about the motor configuration.

Adding a drilling motor

To add a drilling motor:

  1. Select a work string using the numbered tabs on the right side of the screen.
  2. Select the component below the drilling motor you want to add.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Drilling Motor option.
  5. In the PS Size field, select the size of the power section.
  6. In the BS Size field, select the size of the bearing section.
  7. In the Motor Type field, select the model of the drilling motor you want to use.
  8. Update the remaining properties of the motor as needed. See "Properties of drilling motors".
  9. Click the OK button () to add the mud motor to the work string.

The drilling motor will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of drilling motors

The table below defines the basic properties for drilling motors in Victus™ Operation. To view the advanced properties of A drilling motor, click the Show/Hide button ().

Basic properties of drilling motors
Section Field Definition
-- Type

Type of drilling motor
-- PS Size

Size of the power section on the drilling motor
-- BS Size Size of the bearing section on the drilling motor
-- Motor Type

Motor size and model
General Info Description

Name or description of the drilling motor
Length

Total length of the drilling motor. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the drilling motor.

  • Calculated – The mass and adjusted weight of the drilling motor are calculated, based on the dimensions of the drilling motor. When the dimensions of the drilling motor are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the drilling motor is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the drilling motor. When the dimensions of the drilling motor are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the drilling motor is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the drilling motor. When the dimensions of the drilling motor are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the drilling motor. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the drilling motor. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Material

Symbol for the material that the drilling motor is composed of and the grade of the material.
Adjustable Setting

Current bend angle setting. You can select one of the standard bend angles, or you can manually enter a value.
Operating Diff. Pressure

Pressure loss through the motor's power section while the bit is off-bottom
Power Section Length

Length from the bottom of the power section to the top of the mud motor
Power Section Type Motor size and model, with the rotor-stator lobes ration and the number of stages. This is a read-only value, populated from the Victus™ catalog.
Bearing Section Type

Model of the bearing section used with the drilling motor. This is a read-only value, populated from the Victus™ catalog.
Performance Specifications Performance Chart Dynamic chart showing differential pressure and torque at various rotation speeds and flow rates. To view the chart, click the Expand button () on the left side of the Performance Chart field.
Speed Ratio Ratio between the motor's maximum speed and maximum flow rate.This is a read-only value, populated from the Victus™ catalog.
Torque Ratio Ratio between the maximum torque and the maximum differential pressure.This is a read-only value, populated from the Victus™ catalog.
Max. Diff. Pressure Maximum allowable pressure loss through the power section.This is a read-only value, populated from the Victus™ catalog.
Torque at Max. Diff. Pressure Amount of torque at the maximum allowable differential pressure.This is a read-only value, populated from the Victus™ catalog.
Bearing Pack Bypass (checkbox) Indicates that there is flow around the bearing pack
TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
Efficiency Efficiency of the nozzles on the drilling motor, relative to the maximum possible performance
Nozzles

Number of nozzles on the drilling motor. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
Nozzle Diameter

Size of each nozzle on the drilling motor. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Top Stabilizer (checkbox) Indicates that the top stabilizer is active on the drilling motor
Blade OD

Outside diameter of the stabilizer blades
Power Section Centralizer (checkbox) Indicates that the centralizer is active on the drilling motor
Blade OD

Outside diameter of the stabilizer blades
Kick Pad (checkbox) Indicates that the kick pad is active on the drilling motor
Blade Count

Number of blades on the kick pad
Blade Length

Length of each stabilizer blade
Connection Data Connection Configuration

Configuration of the connections on the drilling motor.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type

Type of threads used on the top connection
Bottom Connection Thread Type

Type of threads used on the bottom connection
