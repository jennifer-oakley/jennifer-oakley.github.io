Control Algorithms

The algorithms on the Setup > Control Algorithms page control Victus™ Operation's calculations for kicks, losses, dynamic tests, and MPD connections. There are six tabs on the Control Algorithms page:

  • The Limits tab contains options for configuring how the high limit will be applied to maintain pressure stability. See "Limits algorithm".
  • The Kick tab contains a flow chart that determines how kicks are detected and controlled. This flow chart contains customizable detection and control settings. See "Kick algorithm".
  • The Loss tab contains a flow chart that determines how losses are detected and controlled. This flow chart contains customizable detection and control settings. See "Loss algorithm".
  • The Dynamic FIT/LOT/PPT tab controls the settings for formation integrity tests, leak-off tests, and pore pressure tests. See "Dynamic formation tests".
  • The Connection tab calculates a flow rate schedule to ramp down the pumps during MPD connections. See "Connection algorithm".
  • The Surge and Swab tab calculates the surface backpressure required to maintain a constant wellbore pressure due to the swab and surge effects on pressure when running the bit in or out of the well. See "Surge/Swab algorithm".
