About MWD calibration

The Setup > Calibration > MWD tab allows you to use temperature and pressure measurements in hydraulics calculations. By default, the hydraulics model calculates the bottomhole temperature and the temperature of fluid exiting the well. These calculated values are used to determine the temperature profile, which is then used to calculate pressure in the well.

However, you can also use temperature measurements from the rig and MWD tools. When MWD temperature calibration is enabled, the hydraulics model will use the temperature measurements to calculate temperatures more accurately. There are three temperatures required to use MWD temperature calibration:

  • Temperature Out – The outlet temperature of the fluid is measured by the flowmeter as the fluid exits the well. See "Flow Meter".
  • Temperature In – The inlet temperature of the fluid is specified in the fluid properties when you create the fluid in Victus™ Operation. See "Fluid properties".
  • Bottom Hole Temperature – Bottomhole temperature is often measured by an MWD tool, then transmitted via WITS or WITSML. See "WITS data" and "WITSML channels".

These temperatures can also be acquired from external data, such as WITS or WITSML, or by manually entering values in the Parameters indicator. See "Parameters indicator".

Temperature calibration is intended to provide minor corrections. However, you should use your best judgment when deciding whether to use temperature calibration. If the difference between calculated and measured temperatures is significant, temperature calibration will increase the accuracy of Victus™ Operation's pressure calculations. However, if problems occur with one of the measurement devices, temperature calibration should not be used.

Warning

Use caution when manually entering values for Temperature Out and Bottom Hole Temperature. When the flow rate changes, the temperature profile in the well will also change. When this occurs, manually-entered temperature values must be updated. If these values are not updated, Victus™ Operation will not be able to calculate bottomhole pressure correctly.

You can also improve the accuracy of hydraulics calculations by enabling pressure calibration, using a PWD sensor. When pressure calibration is enabled, the hydraulics model will calculate values for the measured depth, TVD, ECD, and pressure at the depth of the PWD sensor. See "Using pressure calibration".

Updating inlet temperature

Over time, the inlet temperature may become significantly different from the temperature specified in the fluid properties. For this reason, it is a good idea to periodically update this information.

To override the mud inlet temperature:

  1. Click the Indicators button () in the Utilities toolbar and select the Parameters option.
  2. In the Parameters window, double-click the Temp In tile.
  3. In the window that appears, enter the current inlet temperature and click the Enter button ().

The current inlet temperature will appear on the Temp In tile, with an orange line to indicate that the value has been manually entered.

Note

If you are receiving inlet temperature data from WITS or WITSML, it is not necessary to update the mud inlet temperature.

Using temperature calibration

Before you can enable temperature calibration, you'll need to configure Victus™ Operation to receive bottomhole temperature data from WITS or WITSML. See "WITS data" and "WITSML channels".

To enable temperature calibration:

  1. Go to the Parameters tab on the Setup > Mappings page.
  2. In the Bottom Hole Temperature row, select a data source for bottomhole temperature data.
  3. If you are using WITS or WITSML data for inlet or outlet temperature, select a data source for Temperature In and Temperature Out.
  4. Click the Save Changes button () at the bottom of the screen.
  5. Go to the MWD tab on the Setup > Calibration page.
  6. Select the Calibration option.
  7. Click the Save Changes button () at the bottom of the screen.

The hydraulics model will begin using the temperature values received from the data sources for Temperature Out and Bottom Hole Temperature. To view the current MWD temperature reading, click on the Indicators utility () and select the Parameters option. See "Parameters indicator".

Using pressure calibration

To enable pressure calibration:

  1. Go to the MWD tab on the Setup > Calibration page.
  2. In the PWD Sensor Offset field, enter the distance between the PWD sensor and the drill bit.
  3. Click the Save Changes button () at the bottom of the screen.

The hydraulics model will use the sensor offset to calculate the measured depth, TVD, pressure, and ECD at the PWD sensor. In the Dashboard Editor panel, the following curves will be listed under the Hydraulics Model Data heading: PWD Equivalent Circulating Density, PWD Measured Depth, PWD Pressure, and PWD True Vertical Depth. To open the Dashboard Editor panel, go to the Home screen and click the Edit Dashboard button (). See "Dashboards".
