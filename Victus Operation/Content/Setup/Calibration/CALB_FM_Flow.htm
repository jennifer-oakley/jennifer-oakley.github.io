About flow meters

The Calibration > Flow Meter > Flow tab contains settings which allow you to create a more detailed plot of the flow to help detect kicks and losses. The # fitting points field sets the number of fitting points to be plotted for a running average. Each fitting point sends two data samples per second. When Flow out fitting is enabled, flow out is calculated using a running average.

Enabling curve fitting

To enable curve fitting:

  1. Go to the Flow Meter tab on the Setup > Calibration page.
  2. Select the Flow tab on the right side of the screen.
  3. In the # fitting points field, select the number of fitting points to calculate.
  4. In the Flow out fitting field, select the Enable option.
  5. Click the Save Changes button ().

Victus™ Operation will start plotting fitting points, based on the number of fitting points selected.

Field definitions

This table defines the fields on the Flow Meter > Flowtab.

Fields on the Calibration> Flow Meter > Flowtab
Field Definition
# fitting points Number of fitting points used to calculate curve fitting for flow out calculations
Flow out fitting Enables or disables curve fitting for flow out calculations
