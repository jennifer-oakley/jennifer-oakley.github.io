About pump configuration

The first pump in the rig schematic is the only pump that is activated by default. If there are additional pumps on the rig, each additional pump must be activated in order to be used in Victus™ Operation's calculations. On the Setup > Surface Equipment > Piping and Instrumentation tab, you can change the pump state assigned to each pump in the pump properties. There are four pump states:

  • Aux – Auxiliary pumps are backup pumps that may be used in addition to the main pumps. Auxiliary pumps can only be used with surface BOPs. See "Activating auxiliary pumps".
  • Boost – Booster pumps are used to pump fluid through the boost lines on a subsea rig. Booster pumps can only used with subsea BOPs. See "Activating boost pumps".
  • Disable – Disabled pumps are not used in any of Victus™ Operation's calculations. All pumps except Pump1 are disabled by default.
  • Main – The main pumps are the primary pumps used on the rig. By default, Pump1 is the only main in Victus™ Operation.

Activating auxiliary pumps

To activate an auxiliary pump:

  1. Go to the Equipment Configuration tab on the Setup > Surface Equipment page.
  2. In the Blow-Out Preventer field, select Surface BOP from the list.
  3. In the Auxiliary Pump field, select the Rig Pump or Weatherford option.
  4. Click the Save Changes button ().
  5. Go to the Piping and Instrumentation tab.
  6. Select one of the secondary pumps in the rig schematic.
  7. In the panel that appears, go to the Pump State field and select the Aux option.
  8. Update the remaining pump settings, then click the Save Changes button ().

The auxiliary pump will be activated and the updated equipment specifications will be saved to the Victus™ Operation database.

Activating boost pumps

To activate a boost pump:

  1. Go to the Equipment Configuration tab on the Setup > Surface Equipment page.
  2. In the Blow-Out Preventer field, select Subsea BOP from the list.
  3. Click the Save Changes button ().
  4. Go to the Piping and Instrumentation tab.
  5. Select one of the secondary pumps in the rig schematic.
  6. In the panel that appears, go to the Pump State field and select the Boost option.
  7. Update the remaining pump settings, then click the Save Changes button ().

The boost pump will be activated and the updated equipment specifications will be saved to the Victus™ Operation database.

Specifying the pump source

To specify the pump source for pump rate:

  1. Go to the Piping and Instrumentation tab on the Setup > Surface Equipment page.
  2. Select the pump in the rig schematic. A panel with the pump properties will appear.
  3. In the Pump Source field, select the desired option.
  4. Click the Save Changes button () to save your changes.

If you are using an external data source such as WITS / WITSML to obtain the pump rate, go to the Setup > Mappings > Parameters tab, then select the data source you want to use for each pump. See "Defining well parameters".

If you are using a low-pressure flow meter, additionally go to the Equipment Configuration tab and enable the Low Pressure Flow Meters box. See "Field definitions".

If you are using a high-pressure flow meter, go to the Setup > Surface Equipment page and select the main high-pressure flow meter the pump will connect to in the Main Flow In Source field. See "Connecting high-pressure flow meters to pumps".

To calibrate the pump efficiency:

  1. Go to the Setup > Surface Equipment page.
  2. In the Piping and Instrumentation tab, select the pump for which you want to calibrate efficiency.
  3. Click the Automatic pump efficiency button () next to the Efficiency value.
  4. Click the Save Changes button () to save your changes.

The process takes 60 seconds to complete. To stop the calibration before it is completed, click the Stop button (). After calibration finishes, the new efficiency value will appear. The same calibration process needs to be performed for each pump.

Pump properties

Properties of pumps
Field Definition
Pump State

Status of the pump

  • Aux – Auxiliary pump. Can only be used with a surface BOP, when an auxiliary pump is selected on the Equipment Configuration tab.
  • Boost – Boost pump. Can only be used with a subsea BOP.
  • Disable – Pump is not active. Default pump state for all secondary pumps.
  • Main – Primary pump on the rig.
Efficiency

Efficiency of the pump. It is also possible to manually update this value. To automatically calibrate pump efficiency, click the Automatic Pump Efficiency button (). See "Calibrating pump efficiency automatically".
Liner Size Diameter of the liner
Stroke Length Length of each pump stroke
Pump Source

Data source to use for the pump stroke rate.

  • External – Uses analog, WITS, or WITSML data for the stroke rate, as configured on the Setup > Mappings > Parameters tab. See "Specifying the pump source".
  • Half Stroke Counter – Uses the stroke counter and counts half of the pulses received. You should select this option when using a proximity switch installed anywhere along the stroke. Victus™ Operation can only read pump stroke counters when the speed is >5 spm.
  • Inverse Half Stroke Counter – Uses the stroke counter to invert the pulse wavelength and counts half of the pulses received. You should select this option when using a proximity switch installed anywhere along the stroke, and an inverted pulse signal is used. Typically, this applies to installations in which the metal piece triggering the sensor is three times wider than the sensor itself. Victus™ Operation can only read pump stroke counters when the speed is >5 spm.
  • Low Pressure Flowmeter – Uses the low-pressure flow meter to measure the flow rate of fluids in systems operating at relatively low pressures. It reads data for mass flow, density, temperature, drive gain, strokes per minute, and total strokes as well. Available for Pump 1, Pump 2 and Pump 3 only. See "Specifying the pump source".
  • RPM – Measures rotations per minute to determine the stroke rate. Typically, this applies to proximity pump sensors that are mounted on a pump shaft.
  • Spare 1 – Uses the Spare 1 sensor to receive flow readings, as configured on the Setup > Calibration > Default Sensors tab. See "Connecting flow meters to spare sensors".
  • Spare 2 – Uses the Spare 2 sensor to receive flow readings, as configured on the Setup > Calibration > Default Sensors tab. See "Connecting flow meters to spare sensors".
  • Spare 3 – Uses the Spare 3 sensor to receive flow readings, as configured on the Setup > Calibration > Default Sensors tab. See "Connecting flow meters to spare sensors".
  • Spare 4 – Uses the Spare 4 sensor to receive flow readings, as configured on the Setup > Calibration > Default Sensors tab. See "Connecting flow meters to spare sensors".
  • Stroke Counter – Uses the stroke counter to count all of the pulses received. You should select this option when using a limit switch or a proximity switch installed at the end of the stroke. Victus™ Operation can only read pump stroke counters when the speed is >5 spm.
N (# of Teeth) Number of teeth on the pump shaft. This field is only visible when the RPM option is selected for the pump source.
n1 (n1:1) First component of the gear ratio. This field is only visible when the RPM option is selected for the pump source.
n2 (n2:1) Second component of the gear ratio. This field is only visible when the RPM option is selected for the pump source.
Pump Type

Type of pump

  • Triplex – A reciprocating pump with three pistons or plungers.
  • Quadruplex – A reciprocating pump with four pistons or plungers.
  • Quintuplex – A reciprocating pump with five pistons or plungers.
Pump Output Volume of fluid passing through the pump, based on efficiency, liner size, stroke length, and pump type. This value is automatically calculated.
Max Pump HHP Maximum horsepower
Max Flow Rate Maximum allowable flow rate
Max Pump Pressure Maximum pressure rating
Pop Off Pressure Pressure at which the pressure valve will open to relieve pressure
Safety Factor Margin of error for the pump pressure. This value is automatically calculated from the values in the Max Pump Pressure and Pop Off Pressure fields.
