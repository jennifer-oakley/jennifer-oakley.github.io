The magnetic field reference data will be determined from an IFR survey. Geodec is used to determine the baseline values for field strength, declination, and dip for the slot coordinates, as well as the gravitational field and convergence angle.

The method used to enter magnetic parameters depends on the amount of variation between the reference values at the kick-off point and at total depth. If the differences are within the acceptable tolerance levels, use the average of the reference values. See "Entering average parameters". If the differences are greater than the acceptable tolerance levels, the wellpath should be divided into hole sections. See "Entering section parameters".

The acceptable tolerance levels are:

  • Magnetic Field = 100 nT
  • Dip = 0.20°
  • Declination = 0.30°

Entering magnetic parameters

To enter the magnetic parameters for the well:

  1. Go to the parameters at the bottom of the setup pane.
  2. In the Azimuth field, select the reference that should be used when reporting azimuth values.
  3. In the Model field, select the model used to calculate the magnetic parameters. If an MWD tool will be used, select the MWD option.
  4. Enter the magnetic parameters. See "Entering average parameters" and "Entering section parameters".
  5. In the Grav Field field, enter the value for total gravity from Geodec.
  6. If you are using grid azimuth for the azimuth reference, enter the convergence angle in the Convergence field.
  7. If you will be adding FMS files for MSA corrections, select the Include FMS option. See "FMS Data".
  8. Calculate the B-G misalignment. See "Calculating B-G misalignment".

The magnetic parameters will be used to assess the quality of the raw survey data. When you run MSA optimization, these values will be used in the calculations. See "Multi-Station Analysis".

Entering average parameters

To enter magnetic parameters for the entire well:

  1. In the Mag Field field, enter the average of the magnetic field strengths at the kick-off point and at total depth.
  2. In the Dip field, enter the average of the dip angles at the kick-off point and at total depth.
  3. In the Declination field, enter the average of the declinations at the kick-off point and at total depth.
  4. Verify that the value in the Checksum field matches the checksum value from the IFR spreadsheet.

The magnetic parameters will be shown on the MSA > Raw Data tab. When you run MSA optimization, these values will be used in the calculations. See "Multi-Station Analysis".

Entering section parameters

To enter magnetic parameters for multiple hole sections:

  1. Select the Lookup Table option, then click the Edit Table button.
  2. In the Background Values Lookup Table window that opens, enter the magnetic parameters at the surface.
  3. Right-click on the table and select the Insert After option.
  4. In the new row that appears, enter the magnetic parameters at the bottom of the first hole section.
  5. Add additional hole sections as needed.
  6. Verify that the checksum values for each section match the checksum values from the IFR spreadsheet.
  7. When you are finished adding hole sections, click OK.

The Background Values Lookup Table window will close. When you run MSA optimization, these values will be used in the calculations. See "Multi-Station Analysis".

Calculating B-G misalignment

Surveyor's Workbench can correct for misalignment between MWD tool accelerometers and magnetometers due to the twist around the tool axis. For the most accurate calculation, be sure to obtain the latest tool calibration sheet.

To calculate the B-G misaligment:

  1. Click the Calculate button to open the Calculate Misalignment window.
  2. In the Average Dip East field, enter the average dip angle when the tool is horizontal and the heading is close to magnetic east.
  3. In the Average Dip West field, enter the average dip angle when the tool is horizontal and the heading is close to magnetic west.
  4. Click OK.

The Calculate Misalignment window will close, and the calculated value will appear in the B-G Misalignment field.

Field definitions

The table below defines the fields for magnetic parameters.

Field definitions for magnetic parameters
Field Name Definition
Azimuth

Reference used when reporting azimuth values.

  • Grid Az – North is defined as the direction northwards along the grid lines of the map projection.
  • True Az – North is defined as the direction towards the North Pole.
Lookup Table When selected, allows you to enter magnetic parameters for specific hole sections. See "Entering section parameters".
Model Model used to calculate the magnetic parameters
Mag Field Strength of the magnetic field at the well's location. If the difference between the field strengths at the kick-off point and at total depth is less than 100 nT, the average of the two values should be used. Otherwise, the wellpath should be divided into hole sections. See "Entering section parameters".
Dip Angle between the local horizontal vector and the Earth's magnetic field vector. If the difference between the dip angles at the kick-off point and at total depth is less than 0.20°, the average of the two values should be used. Otherwise, the wellpath should be divided into hole sections. See "Entering section parameters".
Declination Clockwise bearing of the horizontal component of the Earth's magnetic field vector from true north. If the difference between the declinations at the kick-off point and at total depth is less than 0.30°, the average of the two values should be used. Otherwise, the wellpath should be divided into hole sections. See "Entering section parameters".
Checksum Validation value used to verify the integrity of the calculated magnetic parameters. This value is calculated automatically and cannot be edited.
Grav Field Strength of the gravitation field at the slot coordinates.
Convergence Angle between the center of the grid and true north.
Include FMS When selected, includes FMS data when calculating MSA corrections. This option should only be selected if the field monitoring station is used to measure variations in the Earth's field.
B-G Misalignment Amount of misalignment between the MWD tool accelerometers and magnetometers due to the twist around the tool axis. When selected, this value can be manually entered. Otherwise, this value is calculated and cannot be edited. See "Calculating B-G misalignment".

The table below defines the fields in the Background Values Lookup Table window.

Field definitions for the Background Values Lookup Table window
Field Name Definition
MD Depth at the bottom of the hole section
Bt Magnetic field strength at the specified depth
Dip Dip angle at the specified depth
Declination Declination at the specified depth
Checksum Validation value used to verify the integrity of the calculated magnetic parameters. This value is calculated automatically and cannot be edited.
