Tour of the Interface

In this article

Main menu

Navigation pane

The Surveyor's Workbench interface has three main areas:

  • The main menu is used to open and save projects, edit the project settings, re-calculate corrections, and open the user guide. See "Main menu".
  • The navigation pane at the top of the screen is used to navigate between the different screens in Surveyor's Workbench. See "Navigation pane".
  • The quick access toolbar contains buttons for updating surveys, enabling AutoPilot mode, and updating the run mode. It also contains options for survey updates. See "Quick Access Toolbar".
  • The main work area is where most of the work in Surveyor's Workbench is done, such as entering job information, running corrections, configuring the BHA, and creating reports.

Main menu

The main menu is located at the top of the screen. There are four menus in the main menu:

  • The File menu is used to manage projects and to import and export BHAs. See "Projects" and "BHAs".
  • The Settings menu opens the Settings Editor window, which is used to update the project settings. See "Settings".
  • The Actions menu is used to re-run calculations and force an update to the raw survey file.
    • The Re-Calculate option recalculates MSA and sag corrections for all selected observations, using the current parameters.
    • The Manual Update From File option reloads the raw survey data file.
    • The Re-Run All Surveys option recalculates MSA and sag corrections for a range of observations, starting with the observation specified in the Initial Number of Observations field.
  • The Help menu is used to access the user guide and the RTOC field instruction.

Navigation pane

The navigation pane is used to navigate between the different screens in Surveyor's Workbench:

  • The Inputs screen is used to select setup options, enter job information, enter magnetic parameters, and load raw survey data. See "Inputs screen".
  • The MSA screen is used to review the raw survey data, add FMS files, and run corrections for multi-station analysis. See "Multi-Station Analysis".
  • The Sag screen is used to build the BHA configuration, enter the sag settings, and run corrections for sag analysis. See "Sag Analysis".
  • The Plan screen is used to import planned survey data and compare it to the raw survey data. See "Plan screen".
  • The Reports screen is used to create reports and send corrections. See "Reports".
