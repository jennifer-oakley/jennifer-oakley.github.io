About the QC and Graphs views

In the QC view, you can review overall statistics, roll tests, and residual charts for the raw and corrected data. This allows you to verify that the optimization results are valid, which ensures that the reference values match the downhole survey data and that bad surveys are excluded.

The Graphs view contains charts that show the changes in field strength and dip angle against measured depth, for both the raw and corrected data. See "Graphs view".

Summary panel

After the optimization completes, the calculated values for gravitational and magnetic parameters will be shown in the summary panel on the left. Values will only be calculated for the selected parameters.

The table below defines the quality control criteria for the parameters in the summary panel.

Quality control criteria for parameters in the summary panel
Parameter QC Criteria
RMS Residual Value should be less than 50 nT.
X Bias Value should be less than 100 nT.
Y Bias Value should be less than 100 nT.
Z Bias

Value should be less than 1000 nT.

Calculated value should be less than twice the predicted value for axial interference in the non-mag spacing sheet.

Graphs view

In the Graphs view, you can view graphs showing the changes in field strength and dip angle. The following quality control criteria can be used to assess the Δ Bt and Δ Dip charts.

  • If the RMS Residual value in the summary panel is greater than 50 nT, any outlying raw data points should be removed from the optimization.
  • Check for any spikes or reversals that may indicate external interference in the Δ Dip chart.
  • The corrected values should be closer to zero and show less variation than the raw values.
  • There should not be any obvious outliers in the raw data.
  • The charts should not show any strong trends with depth.

To remove a point from the next iteration of optimization, click on the point in the raw data that you want to remove. A black X will appear on top of the point.

To show only the data points included in the optimization, right-click on either chart, then go to Show > Selected Only.

To change the parameter that is graphed, right-click on the chart you want to change, then go to Y Axis and select the parameter you want to view.

Statistics section

The Statistics section uses mathematical statistics, such as mean values, standard deviations, and minimum/maximum ranges, to analyze the data quality for the raw and corrected data.

The table below defines the quality control criteria for the parameters in the Statistics section.

Quality control criteria for parameters in the Statistics section
Parameter QC Criteria
Δ Bt (C)

The mean value of Δ Bt (C) should be less than the mean value of Δ Bt (R).

The standard deviation for Δ Bt (C) should be less than the standard deviation for Δ Bt (R).
Δ Dip (C)

The mean value of Δ Dip (C) should be less than the mean value of Δ Dip (R).

The standard deviation for Δ Dip (C) should be less than the standard deviation for Δ Dip (R).
Inc The range of minimum and maximum values for Inc should be more than 20°.
Azi The range of minimum and maximum values for Azi should be more than 20°.
Sensitivity The maximum value for Sensitivity should be less than 3°.

Roll Tests table

Surveyor's Workbench attempts to identify downhole roll tests by looking for repeated surveys at the same measured depths. The Roll Tests table shows the MD, the raw and corrected azimuth spread, and the raw and corrected inclination spread for each roll test detected.

The table below defines the quality control criteria for the parameters in the Roll Tests table.

Quality control criteria for parameters in the Roll Tests table
Parameter QC Criteria
Azi Spread (C) The value for Azi Spread (C) should be less than or equal to Azi Spread (R).
Dip Spread (C) The value for Dip Spread (C) should be less than or equal to Dip Spread (R).

Residual charts

The Residual Charts section includes charts for the initial and final residuals. For each station, the calculated values for inclination, azimuth, and toolface are used to determine the theoretical sensor readings. The difference between the actual and theoretical readings is the residual.

Tip

When you hover your mouse over a point in the graph, the observation number and residual value will be shown in the tooltip.

The following quality control criteria can be used to assess the residual charts.

  • The spread of the residuals for all three axes should be smaller in the Final graph than in the Initial graph.
  • There shouldn't be any clear trends in residual data. For example, a clear trend in the Z residual which correlates to a change in inclination may indicate the need for a Z SF correction.
  • Any surveys that show clear outliers in the corrected residual plots should be removed from MSA optimization. Double-click on point in graph to remove it from next iteration of optimization

To remove a residual point from MSA optimization, double-click on the point in the graph. The corresponding raw survey will be excluded from optimization.
