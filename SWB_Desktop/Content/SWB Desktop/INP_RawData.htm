About raw survey data

To import raw survey data, the imported data must meet the following requirements:

  • The data must be numeric.
  • For sensor data, the data must include Gx, Gy, Gz, Bx, By, and Bz measurements. You may also include reference declination, reference dip, and reference field strength measurements, but these measurements are optional.
  • If FMS data is included, the data must also include date and time.
  • For inclination/azimuth data, the data must include MD, inclination, azimuth, and toolface. For azimuth, either true azimuth or grid azimuth data can be used.

When you import raw survey data, you'll need to map the data to the corresponding column heading, so that Surveyor's Workbench knows how to read the data. See "Importing survey data".

If any of the surveys in the imported data are in the no-go zone, a warning message will appear in the banner at the top of the screen. Any survey that exceeds the inclination or azimuth limits is considered to be in the no-go zone. To change the inclination and azimuth limits, go to the Settings menu, then go to the No Go Zone section in the Settings Editor window.

  • The default inclination limit for all survey types is 70 degrees.
  • For non-IFR surveys, the default azimuth limit is ± 30 degrees from east or west.
  • For IFR surveys, the default azimuth limit is ± 25 degrees from east or west.

The leftmost button at the top of the screen shows the status of the raw survey data file. If there haven't been any changes to the file, the Up To Date button () will be shown. When the file is updated, the Refresh button () will be shown instead. If the Beep option is selected, Surveyor's Workbench will also beep continuously until the file changes are loaded into Surveyor's Workbench. To load the new data into Surveyor's Workbench, click the Refresh button (). The quality of the data will be automatically re-assesed on the MSA > Raw Data tab, and any FMS data will be automatically synced to the new survey data. See "MSA Raw Data" and "FMS Data".

Note

When the Inc/Azimuth option is selected, only sag analysis will be available. The MSA screen will be disabled.

Importing survey data

To import raw survey data:

  1. In the Job Setup section, go to the Data Source field and select the From File option.
  2. In the File Contents field, select the type of data that the raw data file contains.
  3. In the Analysis Type section, select the type of corrections that are required for the job.
  4. Click the Load File button at the top of the data pane.
  5. In the window that opens, navigate to the file you want to import and click Open.
  6. If there is a preset defined that matches your data, select the name of the preset from the menu next to the Load File button. See "Creating a preset template".
  7. For each column detected in the import file, right-click on the column heading and select the corresponding data type for the column. If there is no corresponding data type, select the None option.

If you selected the Apply MSA option in the setup pane, the imported data will appear on the MSA > Raw Data tab. See "Multi-Station Analysis". If you selected the Sag Only option, the imported data will appear on the Sag > Analysis tab. See "Sag Analysis". If any of the surveys in the imported data are in the no-go zone, a warning message will appear in the banner at the top of the screen.

Note

When the Manual Survey Selection option is selected, you can select which observations will be included in MSA optimization. Otherwise, all observations are automatically included.

Creating a preset template

Preset templates allows you to save the current column settings, so that you can easily assign column headings in future Surveyor's Workbench projects.

To create a preset template for raw data:

  1. Import the raw survey data and assign the corresponding data type for each column.
  2. Click the Add to Presets button at the top of the data pane.
  3. In the Raw Data Preset window, enter a name for the preset and click OK.

The new preset will appear in the list of preset templates. The current column settings will be saved to the preset template, based on the order of the column headings.

Tip

To update the settings for an existing preset, add the preset again, but use the same name as the existing preset.
