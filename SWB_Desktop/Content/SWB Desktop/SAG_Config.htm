About sag setup

Sag correction requires the dimensions of each component in the BHA, as well as stabilizer, motor, and sensor positions. You'll also need to enter the mud weight, toolface offset, and motor bend angle. Most of the information required can be found in the BHA report, which lists the BHA components with the dimension and connections for each component. However, it may be necessary to refer to fishing diagrams for more complex tools.

Complex tools, such as stabilizers, RSS units, and mud motors, may need to be split into multiple part sections. You can manually enter the dimensions of each part section by adding each section as a separate component. You can also insert a blade section to automatically create top, bottom, and stabilizer sections for a stabilizer. See "Adding blade sections".

After you've built the BHA, you can view a color-coded schematic of the BHA in the lower left panel. The schematic shows the length and diameter of each component in the BHA. To maximize the schematic, click the Maximize button () at the top of the panel.

To change the units used for OD, ID, and length, go to the Sag section in the Settings window.

Note

The value in the Category column is only used to color-code the BHA schematic.

Entering sag setup data

To configure the settings for a sag analysis:

  1. Go to the Sag > Setup tab.
  2. In the panel on the left, enter the mud weight, sensor position, and bent sub settings. See "Field definitions".
  3. In the BHA Elements pane on the right, enter the dimensions of the components in the BHA. See "Adding BHA components".
  4. Click the Validate BHA button to check the BHA configuration.

As you add components, the BHA schematic will be automatically updated. To maximize the BHA schematic, click the Maximize button () at the top of the panel.

Adding BHA components

To add a component to the BHA:

  1. Go to the Sag > Setup screen.
  2. Right-click in the white area in the BHA Elements pane, then select the Insert option.
  3. In the new row that appears, go to the Category column and select the type of component.
  4. Enter a description of the component in the Name field.
  5. Enter the dimensions in the OD, ID, and Length fields.
  6. If the tool is composed of non-magnetic material, select the Non-Mag option.
  7. To add additional components to the BHA, right-click on a row in the table and select the Insert Before option or the Insert After option.

As you add components, the BHA schematic will be automatically updated. When you have finished adding components, click the Validate BHA option to check the dimensions of the BHA components. To maximize the BHA schematic, click the Maximize button () at the top of the panel.

Adding blade sections

To add a blade section:

  1. Right-click on the component that you want to add a blade section to, then select the Insert Blade option.
  2. In the Insert Blade window, enter the distance between the drill bit and the middle of the blade in the Blade Mid-Pt to Bit field.
  3. Enter the blade dimensions in the Blade OD and Blade Length fields.
  4. Click OK to add the blade section.

The component will be split into three sections: the body section below the blade, the blade section, and the body section above the blade.

Field definitions

The table below defines the setup information required for a sag analysis.

Definitions for sag setup information
Section Field Name Definition
-- Mud Weight

Density of the drilling fluid being pumped into the well
-- Toolface Offset

Toolface offset from the MWD sensor to the motor high side
-- Manually enter hole OD Adds the Hole OD column to the table in the BHA Elements pane, which allows you to enter the hole diameter for each component. When this option is not selected, hole diameter is assumed to be the same as the bit OD.
-- Ignore toolface

When selected, the analysis will assume that the bent sub angle is 0°.
Bit to Sensor Absolute

 

Distance from the sensor to the drill bit
Relative Distance from the sensor to the bottom of the tool that contains the sensor
Tool with Sensor Tool that contains the sensor. Only available when the Relative option is selected.
Bent Sub Bit to Bend

Distance from the mud motor's bent sub to the drill bit
Bent Sub Angle

Bend angle setting for the motor

The table below defines the columns in the BHA Elements pane.

Definitions for the columns in the BHA Elements pane
Column Name Definition
No. Component number showing the component's placement within the BHA
Category Type of BHA component. The value in the Category column is only used to color code the BHA schematic.
Name Description of the BHA component
OD Nominal OD of the component
ID Inside diameter of the component
Length Length of the component
Stiffness Flexibility of the component. Typically, this value should be 1.
Hole OD Diameter of the openhole section of the wellbore. Only available when the Manually enter hole OD option is selected in the sag settings.
Cumulative Length Accumulated running length of the current and preceding BHA components
Non-Mag Indicates that the component is composed of non-magnetic material
