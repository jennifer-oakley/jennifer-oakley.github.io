About FMS data

Data from field monitoring stations (FMS) can be used to conduct IFR2 surveys. The FMS is placed at a location close to the rig site, but away from sources of magnetic interference. Since FMS data is synced with the downhole data, the time stamps in the FMS data must be within the same time range as the raw survey data. Any FMS data that falls outside of the range for the raw survey data will be excluded from the optimization results.

When you add an FMS data file, you'll need specify the magnetic parameters of the data set. Typically, this is done using the average values from an initial sample of the data. The average values needed for FMS data include:

  • Magnetic field strength
  • Azimuth
  • Dip angle
  • Gravitation field strength

Adding FMS data

To add FMS data:

  1. Go to the Inputs screen, then go to the magnetic parameters section and select the Include FMS option.
  2. Go to the MSA > FMS tab.
  3. Click the Add/Edit FMS Files button.
  4. In the FMS Files window that opens, right-click in the white area and select the Insert option.
  5. In the new row that appears, click the Browse button (...) under the File column.
  6. In the window that opens, navigate to the file that contains the FMS data and click Open.
  7. In the FMS Files window, enter the average magnetic parameters and B-G misalignment for the data in the file.
  8. Click OK to add the file.
  9. For each column detected in the FMS file, right-click on the column heading and select the corresponding data type for the column. If there is no corresponding data type, select the None option.

The raw data will appear on the right side of the screen. When you run the MSA optimization, the FMS data will be included in the calculations. If desired, you can parse the FMS data to assess the quality of the data. See "Assessing FMS data".

Note

Surveyor's Workbench does not monitor FMS files. If the FMS file is updated, you'll need to reload the data file. Click the Add/Edit FMS Files button to open the FMS Files window again, then click OK to close the window.

Tip

To add additional FMS files, right-click on the existing FMS file in the FMS Files window, then select the Insert Before or Insert After option.

Assessing FMS data

When you parse the FMS data, the raw FMS data will be analyzed to assess the quality of the data. This step is optional and does not effect the optimization results. To parse FMS data, select the Parsed Data option above the raw data table. The table on the FMS tab will be color-coded to indicate the results of the quality control assessment.

  • Green surveys fall within the tolerance levels for all QC parameters. By default, all green surveys are used for MSA optimization.
  • Red surveys exceed the tolerance level for at least one QC parameter. All red surveys are excluded from MSA optimization.
  • All remaining surveys are shown in orange. Surveys may be shown in amber due to external magnetic interference or expected effects of the BHA magnetization at that attitude. By default, all orange surveys are excluded from MSA optimization. However, you should examine orange surveys to determine if they should be included. If in doubt, it is typically best to exclude the data. See "MSA Optimization".
Note

The majority of the FMS data should be color-coded in green or orange. If most or all of the FMS data is color-coded red, the reference values entered in the FMS Files window may be incorrect. See "Adding FMS data".

Note

The majority of the raw survey data should be color-coded in green or orange. If most or all of the raw survey data is color-coded red, the reference values entered on the Inputs tab may be incorrect. See "Magnetic Parameters". If most or all of the raw surveys do not have an FMS observation, the reference values entered in the FMS Files window may be incorrect. See "FMS Data".

The table below defines the tolerance levels used to assess the FMS data.

Tolerance levels for FMS data
Parameter Passing Tolerance Failing Tolerance
Total Field Strength (BTOTAL) ±350 nT ±1050 nT
Dip (DIP) ±0.3° ±0.9°
Total Gravity (GTOTAL) ±0.003 g ±0.009 g

When Surveyor's Workbench parses the data, it also calculates the average field strength, declination, and dip of the data in each FMS file, as well as the standard deviation from the average values. To view these values, click the FMS Statistics button. In the FMS Statistics window that opens, click on the name of the file that you want to view statistics for.

After the data has been parsed, you can also view graphs of the QC results in the lower left panel. These graphs show the changes in magnetic field strength, dip, and gravitation field strength. To maximize the graphs, click the Maximize button () at the top of the panel.
