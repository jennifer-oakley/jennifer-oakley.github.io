About optimization settings

In the Combined Data view on the MSA > Optimisation tab, you can review the surveys and parameters included in the analysis. The first column in the data table shows which surveys will be included in the analysis. All green surveys included by default, unless the Manual Survey Selection option is selected on the Inputs screen. The FMS column indicates if there is FMS data associated with each survey. If a raw survey has corresponding FMS data, the FMS option will be automatically selected for that survey.

For standard jobs, the required magnetic parameters are selected by default. These parameters include X Bias, Y Bias, Z Bias, X SF, and Y SF. In some situations, however, you may need to include additional magnetic parameters or gravitational parameters.

  • Z SF – Z SF requires a large change in inclination to be calculated correctly. If the Final Residual graph in the QC view shows a variation in Z residuals which trends with the variation of inclination in the well, then you may want to include calculates for Z SF in the next iteration.
  • Dip – Dip estimation requires minimum change of 20° in the wellbore azimuth, such as in a horizontal section with an azimuth turn of 20° or more.
  • Gravitational Parameters – Variations in gravitational parameters are generally caused by tool errors or calibration changes. Therefore, calculating gravitational parameters should not normally be required. Accurate calculations for gravitational parameters requires an accurate knowledge of the reference gravity field strength that the tool would be expected to read in the field.

Configuring optimization

To configure MSA optimization:

  1. Go to the MSA > Optimisation tab, then go to the Combined Data view.
  2. Review the surveys included in the analysis, then add and remove surveys as needed.
  3. Review any surveys that are missing FMS data. If the corresponding FMS data should be included, select the FMS option for that survey.
  4. If you want to include gravitational parameters, select the Gravitational Parameters option, then select the parameters that you want to be calculated.
  5. Select the magnetic parameters that you want to be calculated. For standard MSA jobs, the required magnetic parameters are selected by default.
  6. Click the Optimise button.

After the analysis completes, you can view the corrected data in the Corrected Data view and the Corrected Survey view. In the Graphs view and the QC view, you can compare the raw data and the corrected data to evaluate the optimization results. See "QC and Graphs views".
