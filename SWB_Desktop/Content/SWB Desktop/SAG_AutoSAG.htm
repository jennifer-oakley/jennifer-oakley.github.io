About auto-sag

Auto-sag projects allow you to automatically send sag corrections for new surveys received from Spectrum.

When you create an auto-sag project, the job information, magnetic parameters, and raw survey data will be parsed from the Spectrum input file. You can also specify the well start depth, tie-on point, and sag setup parameters at the same time that you create the project.

After you create the project, new surveys are processed and corrected automatically. As new surveys are added to the Spectrum job, Spectrum will send the surveys to Surveyor's Workbench. When Surveyor's Workbench receives the new surveys, they will be automatically corrected and sent back to Spectrum.

Note

When a new run begins, you will need to create a new auto-sag project for the new run.

Note

If the tie-on point is updated after you create the project, it will be included in sag corrections the next time corrections are calculated.

Creating an auto-sag project

To create an auto-sag project:

  1. Go to File > New Auto-Sag Project in the main menu.
  2. In the Create Auto-Sag Project window, go to the Depth Units field and select the type of depth unit used in the Spectrum input file.
    • If depth is measured in feet, select the Imperial Depth option.
    • If depth is measured in meters, select the Metric Depth option.
  3. In the Sag Units field, select the unit system that you want to use for sag corrections.
  4. Update the job information as needed.
  5. In the Analysis Type field, select the type of sag analysis you want to run.
  6. Enter the start depth for the well and the tie-on point information.
  7. If you have a BHA file to import, click the Browse button (...) to select the BHA file you want to import.
  8. Enter the mud weight, sensor position, and bent sub settings.
  9. Select the locations where you want to save the project and the report.
  10. Click the OK button to create the project.

The auto-sag project will be created and saved to the selected location. The job details and magnetic parameters will be automatically loaded from the Spectrum input file. The project will open to the Sag > Setup tab. If you did not select a BHA to import, you will need to create one before running sag corrections. See "Sag Setup".

Field definitions

The table below defines the fields in the Create Auto-Sag Project window.

Field definitions for the Create Auto-Sag Project window
Field Name Definition
Depth Units

Depth unit used in the Spectrum input file.

  • Imperial Depth – Input file uses feet to measure depth.
  • Metric Depth – Input file uses meters to measure depth.
Sag Units Unit system used for sag inputs and Spectrum output
Spectrum Input File File location for the Spectrum input file. This file is generated by Spectrum and saved to the SWB directory. The file location is read-only and cannot be edited.
Job Number Job number from the Spectrum input file. The job number is parsed from the input file and shown on the Inputs screen after the project is created.
Well Name Well name from the Spectrum input file. The well name is parsed from the input file and shown on the Inputs screen after the project is created.
Run Number Run number for the well. The run number is automatically calculated, but can be edited.
Project Name Project name for the Surveyor's Workbench project. The project name is determined based on the job number, well name, and run number. This field is read-only and cannot be edited.
Analysis Type

Type of sag analysis to use.

  • Sag Only – Sag correction calculates the corrected inclination and build rate to compensate for gravitational effects on the BHA. You can run a sag analysis with or without survey data, using either long or short survey data.
  • Sag + AES – Sag correction calculated using short collar readings. Sensor data is required for sag analysis with AES corrections.
Start Depth Start depth for the well
Tie Point MD, inclination, azimuth, offsets, and TVD for the tie-on point. This information is not parsed from the Spectrum input file and must be manually entered.
Spectrum Output File File location for the Spectrum output file. This file is generated by Surveyor's Workbench and saved to the SWB directory. The file location is read-only and cannot be edited.
BHA File File location of the BHA import file.
Mud Weight

Density of the drilling fluid being pumped into the well
Toolface Offset

Toolface offset from the MWD sensor to the motor high side
Ignore Toolface

When selected, the analysis will assume that the bent sub angle is 0°.
Bit to Sensor

Distance from the sensor to the drill bit
Bit to Bend

Distance from the mud motor's bent sub to the drill bit
Bent Sub Angle

Bend angle setting for the motor
Project Location File location for the Surveyor's Workbench project
Report Location File location for the survey report
