About PDF reports

On the Reports screen, you can create standard or customized PDF reports.

  • Standard reports include the job details, raw magnetic parameters, FMS magnetic parameters, the job settings, survey data, corrected data, and charts comparing raw data and corrected data.
  • Customized reports can include any combination of the available report components, which include job details and setup, different types of results from MSA analysis, several charts from MSA analysis, and sag configuration and results. See "Report components".

Creating PDF reports

To create a PDF report:

  1. Go to the Reports screen.
  2. In the PDF Report section, select the format you want to use for the report.
  3. If you selected the Customised Report option, select the report components that you want to include in the report. See "Report components".
  4. Click the Generate button.
  5. In the window that opens, navigate to the location where you want to save the report and click Save.
  6. In the Save Project window, click Save.

The report will be saved to the selected location, and a copy of the PDF report will appear on the PDF Report tab.

Report components

The table below defines the report components available for PDF reports.

Descriptions of report components for PDF reports
Component Name Description
Job Details Job details from the Inputs > Job Details pane, start and end depth of well
Job Setup Job settings from the Inputs > Job Setup pane
Final Corrected Survey Final MSA- and sag corrected survey, indexed by observation number
Survey Data Raw and corrected surveys, amount of MSA and sag correction, and tool codes, indexed by MD
MSA Results MSA optimization summary, calculated gravitation parameters, and calculated magnetic parameters, from the MSA > Optimisation tab
QC Results Statistics, Optimisations, Roll Tests, and Residuals sections, from the MSA > Optimisation > QC view
Corrected Data Raw and corrected data for gravitational field strength, magnetic field strength, and dip angle, indexed by observation number.
Raw Data - Inc Misalignment Magnetic parameters for the raw data, from the Inputs > Background Details pane
Raw Data - Inc FMS Magnetic parameters from the FMS data, from the MSA > FMS tab
