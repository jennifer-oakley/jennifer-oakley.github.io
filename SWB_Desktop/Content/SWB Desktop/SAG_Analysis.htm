About sag analysis

Sag analysis can be used to correct inclination errors from raw survey data. You can use either long or short survey data. See "Configuring sag analysis with survey data".

  • Long survey data: Sensor data that includes Gx, Gy, Gz, Bx, By, and Bz measurements. This provides the most comprehensive correction capabilities.
  • Short survey data: Basic survey measurements including MD, inclination, azimuth, and toolface. This is suitable for standard sag correction applications.

You can also use sag analysis for theoretical sag analysis and BHA design evaluation, without importing survey data. This allows you to run sag corrections for a specific toolface orientation and visualize how the BHA deflects and lies within the wellbore. If the analysis shows that deflection at the sensor changes significantly or the BHA does not rest properly on the low side of the hole, you may need to adjust the BHA configuration. See "Configuring sag analysis without survey data".

Configuring sag analysis with survey data

For sag only projects, all surveys will be selected by default. For MSA with sag projects, all surveys included in MSA optimization will be automatically selected.

To configure sag with survey data:

  1. Go to the Sag > Setup tab and enter the sag setup information. See "Sag Setup".
  2. Go to the Analysis tab.
  3. If you selected the Apply MSA option on the Inputs screen, and you have not run an MSA analysis, click the Optimise MSA button.
  4. Review the surveys included in the analysis, then add and remove surveys as needed.
  5. Click the Correct Survey button.

The analysis will calculate the corrected build rate, inclination, and amount of sag correction for each survey point. The chart in the Graph pane will show the amount of correction required for each inclination value. To maximize the graph, click the Maximize button () at the top of the panel.

Configuring sag analysis without survey data

To configure sag analysis without survey data:

  1. Go to the Inputs screen, then select the User Entered option in the Job Setup panel.
  2. Go to the Sag > Setup tab, then enter the sag setup information. See "Sag Setup".
  3. Go to the Analysis tab.
  4. Enter the build rate and toolface for the wellbore section in the Build and Toolface fields.
  5. Enter the inclination for the wellbore section.
    • To correct a single inclination value, go to the Single Value section and enter the inclination you want to correct.
    • To correct a range of inclination values, go to the Range of Values section. Enter the minimum and maximum values for the range, as well as the increment for the range.
  6. Click the Analyse button.

If you entered a single inclination value, the chart in the Graph pane will show the deflection of the BHA within the wellbore. The red lines show the top and bottom edges of the hole, the blue lines show the top and bottom edge of the BHA, and the green line shows the deflection of the center of the BHA. The black line shows the location of the sensor. To maximize the graph, click the Maximize button () at the top of the panel.

If you entered a range of inclination values, the analysis will calculate the corrected build rate, inclination, and amount of sag correction for each point included in the range. The chart in the Graph pane will show the amount of correction required for each inclination value. To maximize the graph, click the Maximize button () at the top of the panel.

The table below defines the settings for user-entered sag analysis. See "Configuring sag analysis without survey data".

Settings for user-entered sag analysis
Section Column Name Definition
-- Sensor Position Read-only value, comes from BHA config
-- Build Expected build rate for the wellbore section
-- Toolface Expected toolface for the wellbore section
Single Value Inclination Inclination value that you want to analyze
Correction Amount of sag correction required. This value is calculated automatically during the analysis.
Range of Values Start Inc Minimum inclination value to include in the analysis
End Inc Maximum inclination value to include in the analysis
Step Size Increment for the range of inclination values
