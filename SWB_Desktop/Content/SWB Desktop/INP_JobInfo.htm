About job information

The job information is entered in the setup pane on the left side of the Inputs screen. The Job Setup section is used to configure the job settings. The options in this section allow you to select the data source, choose the type of data in the import file, choose the type of corrections needed, and filter the data in the import file. The Job Details section contains the basic job information, such as the customer name, job number, well name, field name, and sensor ID.

There are two options for importing data:

  • Inc / Azimuth – Use this option to import short survey data that contains inclination and azimuth data. Short survey data can only be used to run a sag analysis. When this option is selected, the MSA screen will not be available.
  • Sensor Data – Use this option to import long survey data that contains Gx, Gy, Gz, Bx, By, and Bz data. Long survey data can be used to run any type of analysis.

There are also three analysis options:

  • Sag Only – Sag correction calculates the corrected inclination and build rate to compensate for gravitational effects on the BHA. You can run a sag analysis with or without survey data, using either long or short survey data. When this option is selected, the MSA screen will not be available.
  • Apply MSA – Multi-station analysis uses several downhole MWD surveys and compares the readings with the expected values, based on the known magnetic parameters. Azimuth corrections will be calculated using MSA corrections to correct for axial and cross-axial magnetic interference. Sensor data is required for an MSA analysis. When this option is selected, MSA analysis must be run before sag analysis.
  • Sag + AES – Sag correction calculated using short collar readings. Sensor data is required for sag analysis with AES corrections. When this option is selected, the MSA screen will not be available.
Tip

If the import file contains the job details, click the Parse Raw Data button after the import to populate the job details from the data in the file. See "Raw Survey Data".

Entering the job information

To enter the job information:

  1. Go to the Inputs screen, then go to the Job Setup section.
  2. In the Data Source field, select the data source that will be used for the raw survey data. For a standard MSA and sag correction job, select the From File option.
  3. In the File Contents field, select the type of data in the import file.
  4. In the Analysis Type field, select the type of corrections that will be needed.
  5. Apply filtering options as needed.
  6. In the Toolface Offset field, enter the angular difference between the high side of the motor and the high side of the MWD tool.
  7. In the Job Details section, enter the basic job information.

If the User Entered option is selected, the Load File button and the MSA screen will not be available. If the From File option is selected, you will be able to import the raw survey data from a file. See "Raw Survey Data". The information entered in the Job Setup and Job Details sections will appear in generated reports. See "Reports".

Warning

If you switch between the Inc/Azimuth option and the Sensor Data option, all existing survey data will be cleared.

Field definitions

The table below defines the fields in the Job Setup section on the Inputs screen.

Field definitions for the Job Setup section
Field Name Definition
Data Source

Data source to be used for the raw survey data.

  • User Entered – Enables additional options for sag analysis. When this option is selected, the Load File button and the MSA screen will not be available.
  • From File – Allows you to import inclination/azimuth data or sensor data from a text or CSV file. See "Raw Survey Data".
File Contents

Type of data in the import file. When the User Entered option is selected, these options will not be available.

  • Inc/Azimuth – File contains inclination and azimuth data. For use with sag analysis only. When this option is selected, column heading options will include inclination, true azimuth, grid azimuth, and tooflace. In addition, the MSA screen will not be available.
  • Sensor Data – File contains long survey data. When this option is selected, column heading options will include Gx, Gy, Gz, Bx, By, Bz, reference declination, reference dip, and reference field strength.
Analysis Type

Type of corrections that will be needed. When the User Entered option is selected, these options will not be available.

  • Sag Only – Project will only require sag corrections. When this option is selected, the MSA screen will not be available.
  • Apply MSA – Project requires MSA corrections. Only available when the Sensor Data option is selected.
  • Sag + AES – Project requires sag corrections, using short collar calculations to determine azimuth. Only available when the Sensor Data option is selected. When this option is selected, the MSA screen will not be available. See demo: "Demo: SAG + AES analysis".
Filtering

Options for filtering the imported data. Only survey data that falls within this range will be included on the MSA > Raw Data tab.

  • Date – Filters the data using a date range.
  • Depth – Filters the data using a depth range.
Start Depth Minimum value for the depth range. Only available when the Depth option is selected,
End Depth Maximum value for the depth range. Only available when the Depth option is selected.
Start Time Minimum value for the date range. Only available when the Date option is selected.
End Time Maximum value for the date range. Only available when the Date option is selected.
Toolface Offset Angular difference between the high side of the motor and the high side of the MWD tool
