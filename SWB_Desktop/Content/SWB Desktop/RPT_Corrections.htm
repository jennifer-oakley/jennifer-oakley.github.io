About corrections

On the Reports > Corrections tab, you can send corrections from Surveyor's Workbench to SWB Orchestrator. The Corrections tab also shows the previous, available, and new corrections.

  • The Previous Corrections table shows the corrections that have already been sent to the rig.
  • The Available Corrections table shows the new and updated corrections that can be sent to the rig.
  • The New Corrections table shows the corrections that will be sent the next time corrections are sent, and it is color-coded to show the status of the corrections being sent.

Before sending corrections, you'll need to change the run mode to Real-Time mode in both Surveyor's Workbench and SWB Orchestrator. See "Changing the run mode".

Note

The Corrections tab is only available for SWB Orchestrator projects. See "Demo: Creating Orchestrator projects".

Changing the run mode

The default mode for all new runs is Calibration mode. To change the run mode to Real-time mode, the run mode must be changed in both Surveyor's Workbench and SWB Orchestrator.

To change the run mode:

  1. In Surveyor's Workbench, click the Calibration button at the top of the screen. The button text will change to Real Time.
  2. In SWB Orchestrator, go to the Dashboard page and click the Edit Job button () in the row for the run.
  3. Locate the run you want to edit in the Add Runs Details section. The active run will be highlighted in green.
  4. Go to the Run Mode field and select the Real-time option.
  5. Click the Save button to save your changes.

The run mode will be changed to Real-time mode. The next time you load a survey into Surveyor's Workbench, the run status will change to SWB Processing.

Sending corrections

To send corrections from Surveyor's Workbench to SWB Orchestrator:

  1. Go to the Reports > Corrections tab.
  2. Go to the New Corrections table to review the corrections that will be sent.
  3. Click the Send to Rig button at the bottom of the screen.

The corrections will be uploaded to SWB Orchestrator, and the rig engineers assigned to the job will receive a notification that new corrections are available. The run status in SWB Orchestrator will change to Corrections Sent.
