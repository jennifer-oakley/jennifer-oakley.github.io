About creating projects

The first step for using Surveyor's Workbench is to create a new project. There are three types of Surveyor's Workbench projects:

  • Local Surveyor's Workbench projects and all of the project data are stored on your local machine. For local projects, you will need to import the raw survey data into Surveyor's Workbench. Corrections can be exported to a file on the Reports screen. See "Creating local Surveyor's Workbench projects".
  • Auto-sag projects allow you to automatically send sag corrections for new surveys received from Spectrum. The job information, magnetic parameters, and raw survey data will be parsed from the Spectrum input file. As new surveys are added to the Spectrum job, Spectrum will send the surveys to Surveyor's Workbench. When Surveyor's Workbench receives the new surveys, they will be automatically corrected and sent back to Spectrum.
  • Orchestrator projects are created on the SWB Orchestrator website, and corrects are performed in Surveyor's Workbench. For Orchestrator projects, the raw survey data will automatically loaded into the Surveyor's Workbench project. Corrections can be uploaded to the Orchestrator project on the Reports screen. See "Connecting to SWB Orchestrator projects".

Creating local Surveyor's Workbench projects

To create a new Surveyor's Workbench project:

  1. Go to File > New Project in the main menu.
  2. In the New Project window, enter a name of the project.
  3. Click the OK button.

A new project will open in Surveyor's Workbench.

Creating auto-sag projects

To create an auto-sag project:

  1. Go to File > New Auto-Sag Project in the main menu.
  2. In the Create Auto-Sag Project window, go to the Depth Units field and select the type of depth unit used in the Spectrum input file.
    • If depth is measured in feet, select the Imperial Depth option.
    • If depth is measured in meters, select the Metric Depth option.
  3. In the Sag Units field, select the unit system that you want to use for sag corrections.
  4. Update the job information as needed.
  5. In the Analysis Type field, select the type of sag analysis you want to run.
  6. Enter the start depth for the well and the tie-on point information.
  7. If you have a BHA file to import, click the Browse button (...) to select the BHA file you want to import.
  8. Enter the mud weight, sensor position, and bent sub settings.
  9. Select the locations where you want to save the project and the report.
  10. Click the OK button to create the project.

The auto-sag project will be created and saved to the selected location. The job details and magnetic parameters will be automatically loaded from the Spectrum input file. The project will open to the Sag > Setup tab. If you did not select a BHA to import, you will need to create one before running sag corrections. See "Sag Setup".

Connecting to SWB Orchestrator projects

After the raw survey data is received, you'll need to connect the run to Surveyor's Workbench. After you load the raw survey data into Surveyor's Workbench, the run status will be automatically updated in SWB Orchestrator.

To connect a SWB Orchestrator project to Surveyor's Workbench:

  1. In SWB Orchestrator, go to the Dashboard page and copy the project ID for the run.
  2. In Surveyor's Workbench, go to File > New Orchestrator Project.
  3. In the New Orchestrator Project window that opens, enter a name for the project in the SWB Project Name field.
  4. In the Project ID field, paste the project number that you copied from SWB Orchestrator.
  5. In the Orchestrator URL field, select the URL for the RTOC.
  6. Click the OK button.
  7. In the window that opens, sign in with your RTOC account.

The Orchestrator project will open in Surveyor's Workbench. If you have the SWB file share mapped, click the Load File button to load the most recently uploaded survey. Otherwise, you can manually load the survey file. See "Import survey data".

Note

You must be connected to the Weatherford network in to connect to SWB Orchestrator through Surveyor's Workbench.

Saving projects

To save a local Surveyor's Workbench project or an SWB Orchestrator project:

  1. Go to File > Save Project in the main menu.
  2. In the Save Project window, go to the Project Location field and select the file location where you want to save the project.
  3. In the Report Location field, select the file location where you want reports to be saved.

The project will be saved to the selected file location. When you generate reports on the Reports screen, they will be saved to the selected report location.
