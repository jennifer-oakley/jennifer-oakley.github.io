About multi-station analysis

Multi-station analysis uses several downhole MWD surveys and compares the readings with the expected values, based on the known magnetic parameters. The analysis uses the raw data to identify magnetic bias and scale errors for each axis of the MWD tool, which are used to correct the raw MWD data and re-calculate a new survey with improved accuracy. As new surveys are received, the corrected surveys are updated to account for the new data.

To run an MSA Multi-Station Analysis analysis, you will need to:

  1. Select the From File, Sensor Data, and Apply MSA options on the Inputs screen. See "Enter job information".
  2. Enter the magnetic parameters. See "Enter magnetic parameters".
  3. Upload the raw survey data. See "Import survey data".
  4. Upload FMS data, if available. See "Adding FMS data".
  5. Configure the settings for MSA optimization. See "Configuring MSA optimization".

Adding FMS data

To add FMS data:

  1. Go to the Inputs screen, then go to the magnetic parameters section and select the Include FMS option.
  2. Go to the MSA > FMS tab.
  3. Click the Add/Edit FMS Files button.
  4. In the FMS Files window that opens, right-click in the white area and select the Insert option.
  5. In the new row that appears, click the Browse button (...) under the File column.
  6. In the window that opens, navigate to the file that contains the FMS data and click Open.
  7. In the FMS Files window, enter the average magnetic parameters and B-G misalignment for the data in the file.
  8. Click OK to add the file.
  9. For each column detected in the FMS file, right-click on the column heading and select the corresponding data type for the column. If there is no corresponding data type, select the None option.

The raw data will appear on the right side of the screen. When you run the MSA optimization, the FMS data will be included in the calculations. If desired, you can parse the FMS data to assess the quality of the data. See "Assessing FMS data".

Note

Surveyor's Workbench does not monitor FMS files. If the FMS file is updated, you'll need to reload the data file. Click the Add/Edit FMS Files button to open the FMS Files window again, then click OK to close the window.

Tip

To add additional FMS files, right-click on the existing FMS file in the FMS Files window, then select the Insert Before or Insert After option.

Configuring MSA optimization

To configure MSA optimization:

  1. Go to the MSA > Optimisation tab, then go to the Combined Data view.
  2. Review the surveys included in the analysis, then add and remove surveys as needed.
  3. Review any surveys that are missing FMS data. If the corresponding FMS data should be included, select the FMS option for that survey.
  4. If you want to include gravitational parameters, select the Gravitational Parameters option, then select the parameters that you want to be calculated.
  5. Select the magnetic parameters that you want to be calculated. For standard MSA jobs, the required magnetic parameters are selected by default.
  6. Click the Optimise button.

After the analysis completes, you can view the corrected data in the Corrected Data view and the Corrected Survey view. In the Graphs view and the QC view, you can compare the raw data and the corrected data to evaluate the optimization results. See "QC and Graphs views".
