Getting Started

Run sag corrections

This section guides you through the basic steps for using Surveyor's Workbench to correct raw survey data using multi-station analysis and sag analysis.

Step 1: Create a new project

The first step for using Surveyor's Workbench is to create a new project. See "Create a new project". There are three types of Surveyor's Workbench projects:

  • Local Surveyor's Workbench projects and all of the project data are stored on your local machine. For local projects, you will need to import the raw survey data into Surveyor's Workbench. Corrections can be exported to a file on the Reports screen.
  • Auto-sag projects allow you to automatically send sag corrections for new surveys received from Spectrum. The job information, magnetic parameters, and raw survey data will be parsed from the Spectrum input file. As new surveys are added to the Spectrum job, Spectrum will send the surveys to Surveyor's Workbench. When Surveyor's Workbench receives the new surveys, they will be automatically corrected and sent back to Spectrum.
  • Orchestrator projects are created on the SWB Orchestrator website, and corrects are performed in Surveyor's Workbench. For Orchestrator projects, the raw survey data will automatically loaded into the Surveyor's Workbench project. Corrections can be uploaded to the Orchestrator project on the Reports screen.

Step 2: Enter job information

After you create a new project, the first step is to add the project information. This includes the settings for the job and the job details. The job settings include the data source, the type of data in the import file, and the type of corrections needed. The job details include the basic job information that appears in reports. See "Enter job information".

Step 3: Enter magnetic parameters

The next step is to enter the magnetic parameters. Magnetic parameters are typically entered using the average values measured at the kick-off point and at total depth. The magnetic field reference data will be determined from an IFR survey. Geodec is used to determine the baseline values for field strength, declination, and dip for the slot coordinates, as well as the gravitational field and convergence angle. See "Enter magnetic parameters".

Step 4: Import raw survey data

The next step is to import the raw survey data. For MSA corrections, the survey data must be long survey data that includes sensor measurements for Gx, Gy, Gz, Bx, By, and Bz. For sag corrections, you can use long survey data or short survey data that includes MD, inclination, azimuth, and toolface. See "Import survey data".

Step 5: Run MSA corrections

Multi-station analysis uses several downhole MWD surveys and compares the readings with the expected values, based on the known magnetic parameters. The analysis uses the raw data to identify magnetic bias and scale errors for each axis of the MWD tool, which are used to correct the raw MWD data and re-calculate a new survey with improved accuracy. As new surveys are received, the corrected surveys are updated to account for the new data. See "Run MSA corrections".

Step 6: Run sag corrections

Sag correction calculates the corrected inclination and build rate to compensate for gravitational effects on the BHA. In an inclined well, the BHA tends to deflect under the force of gravity. Typically, the BHA will rest on the stabilizers on the low side of the hole, causing the BHA to bend slightly. Because of this, the survey sensor may not be parallel to the main axis of the wellbore, causing mismatches between the sensor inclination and the well inclination. See "Run sag corrections".
