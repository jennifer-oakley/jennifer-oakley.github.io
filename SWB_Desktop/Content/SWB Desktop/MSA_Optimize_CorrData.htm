About the Corrected Data view

In the Corrected Data view, you can view the raw and corrected data for magnetic parameters, gravitational parameters, deviation and orientation angles, and short collar readings.

The data in the Corrected Data view is automatically analyzed to assess the quality of the data.

  • Green surveys fall within the tolerance levels for all QC parameters. By default, all green surveys are used for MSA optimization.
  • Red surveys exceed the tolerance level for at least one QC parameter. All red surveys are excluded from MSA optimization.
  • All remaining surveys are shown in orange. Surveys may be shown in amber due to external magnetic interference or expected effects of the BHA magnetization at that attitude. By default, all orange surveys are excluded from MSA optimization. However, you should examine orange surveys to determine if they should be included. If in doubt, it is typically best to exclude the data.

The table below defines the tolerance levels used to assess the corrected data.

Tolerance levels for corrected data
Parameter Passing Tolerance Failing Tolerance
Total Field Strength (BTOTAL) ±350 nT ±1050 nT
Dip (DIP) ±0.3° ±0.9°
Total Gravity (GTOTAL) ±0.003 g ±0.009 g

Field definitions

The tables below define the columns available in the Corrected Data view. To add a column to the Corrected Data view, right-click on any column name, then go to Choose Columns and select the column you want to add to the view. To hide a column from the view, right-click on the column you want to hide and select the Hide option.

Basic parameters

The table below defines the columns listed in the top-level of the Choose Columns menu.

Column definitions for the top-level of the Choose Columns menu
Column Name Definition
Chosen Az Correction

Method of azimuth correction used to account for magnetic interference

  • MSA – Uses multi-station analysis to correct for axial and cross-axial magnetic interference. MSA azimuth correction is the default method used for all corrected survey points.
  • AES – Uses short collar readings to adjust the Z-axis magnetometer reading by calculating the expected clean reading for each axis and define the amount of azimuth error.
  • LCAZ – Uses the least count azimuth method, which uses magnetic declination and grid convergence to calculate the corrected azimuth.
Selected

Indicates the observation is included in MSA optimization
Obs

Observation number in the raw survey data
FMS

Indicates there is corresponding FMS data associated with the observation
Date/Time

Date and time stamp for the observation in the raw survey data
