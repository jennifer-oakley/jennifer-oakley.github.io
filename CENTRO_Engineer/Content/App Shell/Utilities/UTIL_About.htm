Utilities

The Utilities toolbar contains useful utilities that allow you to change fluids, view incoming data, record videos, capture screenshots, and open the CENTRO® Engineer user guide. These utilities can be used from any screen throughout CENTRO® Engineer.

The Guides utility () includes utilities to control the fluids in the well, monitor incoming data and calculated values, and access the user guide.

  • The Mud button () is used to set pump and offset fluids. See "Mud utility".
  • The Indicators View button () allows you to monitor hydraulics calculations, torque and drag calculations, well parameters, and WITS/WITSML data. It also allows you to view custom panels and bit runs. See "Indicators utility".
  • The Tutorial Help button () is used to access the quick start guide and software manual, as well as mathematical models. See "Tutorial Help utility".

The Capture Screen utility () includes utilities to capture videos and screenshots.

  • The Capture Video button () records a video of the screen. This can be a useful tool for reporting, presentation, and training purposes. See "Capture Video utility".
  • The Capture Screen button () captures a snapshot of your screen. See "Capture Screen utility".

The About utility () contains information about your license for CENTRO® Engineer. See "License activation".
