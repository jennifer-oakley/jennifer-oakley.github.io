About the Navigation pane

The Navigation pane is used to navigate between the different screens and pages in CENTRO® Engineer. There are three main areas in the Navigation pane:

  • The main menu, located on the left side of the Navigation pane, is used to navigate between the primary screens in CENTRO® Engineer. See "Main menu".
  • The secondary navigation area, located on the right side of the Navigation pane, is used to navigate between the sections and pages for each primary screen. See "Secondary navigation".
  • The Utilities toolbar, located on the lower left side of the Navigation pane, is used to quickly switch fluids, view incoming data, take a screen capture, and access the user manual. See "Utilities".

Main menu

The main menu is located on the left side of the screen. The main menu is used to navigate between the different screens in CENTRO® Engineer.

  • The Dashboards screen () displays various dashboards that contain gauges and graphs for monitoring well conditions during drilling operations, showing incoming data from hydraulics calculations, torque and drag calculations, and WITS/WITSML data. See "Dashboards".
  • The Projects screen () allows you to manage the projects and wells in CENTRO® Engineer. See "Projects screen".
  • The Setup screen () contains all of the information related to the well, such as job details, rig equipment, well data, work strings, fluids, parameters, and unit systems. See "Setup screen".
  • The Planning screen () contains simple calculators that only require a minimum amount of data, as well as planning, mechanics, geomechanics, and hydraulics analysis tools. See "Planning screen".
  • The Alarms screen () has several tools to help you identify and resolve problems that occur within CENTRO® Engineer. See "Alarms".
  • The Reports screen () allows you to generate drilling performance reports, view the KPIs for the well, and record planned and actual risks. See "Reports".
  • The Global Settings () screen is used to customize settings that apply to all of the wells in the CENTRO® Engineer database, such as unit system settings and display settings. See "Global settings".

Secondary navigation

The secondary navigation pane contains the sections and pages for each primary screen in the main menu. To expand a section, click the Expand button () next to the section name. To hide the contents of a section, click the Collapse button ().

The secondary navigation pane can be automatically hidden when it's not needed, so that only the main menu buttons are visible.

  • To auto-hide the secondary navigation pane, click the Hide Labels button () in the bottom left corner of the screen. When you hover your mouse over the main menu, the secondary navigation pane will appear.
  • To always show the secondary navigation pane, click the Show Menu button ().
