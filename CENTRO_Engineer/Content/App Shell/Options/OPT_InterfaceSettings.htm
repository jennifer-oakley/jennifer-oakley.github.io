Interface settings

Related Topics
Related Topics

  • Graphs
  • Indicators utility

Interface settings control the legends, spacing, and abbreviations in the graphs throughout CENTRO® Engineer. The table below defines the different settings that you can customize on the Global Settings > System Preferences > Interface Settings page.

Field definitions on the Interface Settings page
Field Name Definition
Abbreviation Max Length

This value sets the maximum allowed length of an abbreviation. This setting primarily applies to the labels that appear on gauges, both on the Dashboards screen and in custom panels. This setting also applies to many of the indicators available under the Guides utility () > Indicators button () in the Utilities toolbar. If the default abbreviation is longer than the Abbreviation Max Length value, the abbreviation will be shortened to fit the maximum length.
Plots Bottom Gap This value sets the size of the gap between the last data point on a graph and the footer at the bottom of the graph. This setting applies to all graphs on the Dashboards screen.
Plots Header Max Space This value sets the maximum height of the legend shown at the top of a graph. This setting applies to all graphs on the Dashboards screen. If the height of the legend exceeds the setting for Plots Header Max Space, scrolling arrows will appear above and below the legend.
Legends always visible by default

When this option is selected, graph legends will be displayed by default. You can also show or hide the legends for individual graphs by clicking the Show/Hide Legend button () in the graph index.
Show Axial Load in BHA Builder When this option is selected, the Mud Weight field, the Required WOB field, and the NP column will be visible in the BHA Builder view on the Setup > Work String page, as well as on the BHA Builder tab in the results for a OneClick™ experiment on the Planning > OneClick™ page. This option must be selected for neutral point calculations. See "Neutral point".
Default BHA Report Template Default report template to use when you create a BHA design report. When you create a report on the Planning page, this template will be the default template for the BHA & Drill String portion of the report.
Suppress Report Selection In BHA When this option is selected, the menu to select a report template will not appear when you create a BHA design report in the BHA Builder view on the Setup > Work String page. Instead, the report will be created using the template selected in the Default BHA Report Template field. See "Creating a BHA design report".
Data Trimming When this option is selected, the data shown in time-based graphs will be trimmed to reduce the loading time for graph data. When this option is not selected, all data points will be shown. This option only applies when the graph range is set to 36 hours or less. When the graph range is greater than 36 hours, the data will be automatically trimmed.
Pie Chart Cut Off

Minimum percentage required for a rig activity to be listed separately in the pie charts on the Reports > Performance > Performance tab and in the Performance indicator () on the Dashboards screen. Any rig activities that are below this percentage will be grouped together under the Other category. See "Performance reports".
Pie Chart Cut Off Color Color used for the Other category in the pie charts on the Reports > Performance > Performance tab and in the Performance indicator () on the Dashboards screen.
