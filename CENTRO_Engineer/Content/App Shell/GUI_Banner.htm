About the main banner

The banner at the top of the screen shows warnings, errors, and system notifications. It also allows you to switch application modes, start data acquisition, and view the current rig activity. In Playback mode, it allows you to control the playback speed.

  • The Application Modes area, on the left side of the banner, shows the current application mode and allows you to switch modes. See "Application modes".
  • The Notifications area displays alerts when hardware or system problems are encountered. See "Notifications".
  • The Real-Time Controls area shows the well name and the clock, which is also used to start and stop data acquisition. When you're in Playback mode, the clock is used to control the playback speed. See "Real-time and playback controls".
  • The Rig Activity area, on the right side of the banner, shows the current rig activity. See "Rig activity menu".

Application modes

When you open CENTRO® Engineer, you will be prompted to select which application you want to use. CENTRO® Engineer has three application modes: Real Time application mode, Planning application mode, and MPD application mode.

  • Real Time application mode includes all of CENTRO® Engineer's features, such as real-time data acquisition, analysis engines, alarms, and reports. Some pages on the Setup screen are only available in Real Time mode, such as the DAQ, Mappings, MVA Settings, and User Data pages. In addition, dashboards, alarms, KPIs, and monitoring reports are only available in Real-Time mode.
  • Planning application mode only includes CENTRO® Engineer's analysis features. Real-time data acquisition, alarms, and real-time reports are not available in Planning mode. The pages available on the Setup screen are limited to the following pages in Planning mode: Surface Equipment, Wells, Fluids, Work String, Catalogs, and OneClick™ Settings.
  • MPD application mode includes some of CENTRO® Engineer's analysis features, such as geomechanics and hydraulics analysis.The pages available on the Setup screen are limited to the following pages in MPD mode: Surface Equipment, Wells, Fluids, Work String, Catalogs, and OneClick™ Settings.

After CENTRO® Engineer opens, the current application mode will be shown in the banner at the top of the screen. To switch application modes:

  1. Click on the dropdown menu on the left side of the banner.
  2. Select the application mode you want to switch to.

CENTRO® Engineer will reload in the selected application mode. The screens and pages shown will be updated as described above.

Notifications

The Notifications area displays alerts when hardware or system problems are encountered, as well as other system messages. When you export a file, fill a data gap, or reprocess data, the status of the process will be shown in the Notifications area. See "Gap fill" and "Reprocessing data". When a limit is exceeded, the banner changes color to yellow. See "Active alarms".

Real-time and playback controls

The Real-Time Control area shows the date and time that the last data point as received, as well as the name of the current well. In Real Time application mode, CENTRO® Engineer can acquire new data from WITS or WITSML data sources and perform real-time hydraulics and torque and drag calculations.

When you are acquiring data from a WITSML channel, the clock will show the current rig time. The rig time is based on the time zone selected for the primary WITSML log. If data acquisition is paused or the connection to the WITSML channel is lost, the clock will show the time that data was last received from the primary log. See "WITSML channels".

  • To start acquiring data, click on the clock and press the Start button ().
  • To stop acquiring data, click on the clock again and press the Stop button ().
Note

The date format shown in the banner is based on the date format defined in your Windows settings. To change the date format shown in CENTRO® Engineer, update your Windows settings and restart CENTRO® Engineer.

In Playback mode, CENTRO® Engineer replays the well data acquired in CENTRO® Engineer. The well configuration and rig states will change as they were changed in real-time. In Playback mode, you can speed up the data and fast-forward to a specific point of time. However, the well data can't be edited when you're using Playback mode.

When CENTRO® Engineer is in Playback mode, you can adjust the rate at which the data is replayed. To replay data from a different date, click the Select Date button (). Use the slide control to skip to another section of the data for that date.

To change the playback rate, click on the clock and select a new playback rate from the menu that appears.

  • x0 – Pauses playback.
  • x1 – Plays the data in real-time.
  • x2 – Plays the data at double speed.
  • x3 – Plays the data at triple speed.
  • x4 – Plays the data at quadruple speed.

To switch to Playback mode, go to the Projects > All Projects > Projects page. See "Switching between modes".

Rig activity menu

The Rig Activity menu shows the current rig activity. The rig activity is determined automatically by CENTRO® Engineer, using a built-in activity algorithm. However, you can override the algorithm by manually choosing a rig activity. The activities in CENTRO® Engineer include various operations for drilling, tripping, and other activities.

To change the current rig activity:

  1. Click on the Rig Activity menu.
  2. Go to the appropriate category for the new rig activity.
  3. Select the current rig activity from the list that appears.

The Rig Activity menu will be updated to show the new right activity.
