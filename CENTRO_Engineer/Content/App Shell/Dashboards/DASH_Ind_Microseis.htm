About the microseismic indicator

The Microseismic indicator uses imported data to show the magnitude and location of microseismic events, with respect to the well trajectory. Microseismic events are hydraulic fractures within a formation. The Microseismic indicator can detect, plot, and visualize these events. This allows you to see patterns of seismicity that show how the formation responds to hydraulic fracturing well stimulation. This information can be used to analyze the effectiveness of a hydraulic fracture treatment and improve it.

The Microseismic indicator can display microseismic events in four ways:

  • 3D View shows a three-dimensional view of the microseismic events along the wellbore trajectory.
  • Plan View shows the wellbore trajectory from the top, looking downward.
  • Side View shows the wellbore trajectory from the side.
  • Gun Barrel View shows the wellbore trajectory from the front.

To use the Microseismic indicator, you'll need to import microseismic data before adding the indicator to a dashboard. See "Importing microseismic data" and "Adding the microseismic indicator".

Importing microseismic data

Before microseismic data can be imported into CENTRO® Engineer, it must be saved in microseismic file format. To save data as a microseismic file:

  1. Open the data file in MS Excel and save it in .csv format. Close the file.
  2. Using your browser, locate the file and change the file extension to .microseismic.

The file will be in the special format that CENTRO® Engineer will recognize as a microseismic file.

To import microseismic data:

  1. Go to the Projects > All Projects > Import Project page.
  2. In the window that opens, change the file type to Microseismic file (*.microseismic).
  3. Navigate to the file you want to import and click Open.
  4. Adjust the import settings as needed.
  5. Click the Save Changes button ().

The selected microseismic data will be imported into CENTRO® Engineer. When you configure the Microseismic indicator, the imported file will appear in the list of microseismic events.

Adding the microseismic indicator

To add the Microseismic indicator to a dashboard:

  1. On the Dashboards screen, select the dashboard that you want to add the indicator to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, expand the Indicators heading.
  4. Click the Microseismic View icon () and drag it to the dashboard.
  5. Hover your mouse over the bottom right corner of the graph.
  6. Click the Edit Settings button () to open the Microseismic View Settings window.
  7. In the View field, select the graph views that you want to use.
  8. In the Microseismic Events field, select the name of the microseismic import file you want to use.
  9. Update the remaining graph settings as desired, then click the Close button ().

The microseismic events for the selected import file will be plotted along the wellbore trajectory.

Using the microseismic indicator

There are several keyboard shortcuts and mouse actions that you can use to manipulate the graphs in the Microseismic indicator.

You can rotate, pan, or change the zoom on the 3D View graph.

  • To rotate the graph, click and hold the mouse button while dragging the graph into the desired position.
  • To pan the graph, click and hold the right mouse button, while dragging the graph into the desired position.
  • To zoom in or out, use the scroll wheel on your mouse.
  • To reset the graph, click the Reset View button () at the top of the dashboard.

For the Plan View, Side View, and Gun Barrel View graphs, you can change the zoom or scroll to a different point in the graph index.

  • To zoom in or out, press the Ctrl key and use the scroll wheel on your mouse.
  • To scroll vertically through the graph, hover over the graph and use the scroll wheel on your mouse.
