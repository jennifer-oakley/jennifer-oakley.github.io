Curves on the Dashboards screen

The table below describes the groups of curves available on the Dashboards screen. The curves in these groups can be added to graphs or used to create gauges. See "Graphs" and "Gauges".

To view the list of available curves, click the menu button () next to a dashboard name, then select the Edit option in the menu that appears. The curves will be listed in the Dashboard Editor panel that appears on the left.

Alarm groups in CENTRO® Engineer
Curve Group Description
Calculations Curves for the calculations created on the Setup > OneClick™ Settings > Real Time page.See "OneClick™ real-time calculations".
Drillstring Configuration Curves for the properties of the active work string on the Setup > Work String page. See "Work strings".
ECD Management Time-based curves related to the weak point in the formation. These curves are calculated by the hydraulics model, based on the formation properties on the Setup > Wells > Formations page. See "Formation data".
ECD Management (Depth) Depth-based curves related to the weak point in the formation. These curves are calculated by the hydraulics model, based on the formation properties on the Setup > Wells > Formations page. See "Formation data".
Events Current status of the events on the Setup > OneClick™ Settings > Real Time page. See "OneClick™ real-time events".
Flow Data Curves for incoming flow data, such as flow in and temperature in. These curves are mapped to incoming data on the Setup > Mappings > Parameters page. See "Parameters".
Hydraulic Model Data Curves for the parameters calculated by the hydraulics model
KPI Curves for the key performance indicators on the Reports > KPIs > Time Breakdown page. See "KPIs".
OPC UA Curves for the OPC data tags defined on the Setup > DAQ > OPC UA page. See "OPC UA data".
Pump Data Curves for incoming pump data. These curves are mapped to incoming data on the Setup > Mappings > Parameters page. See "Parameters".
Sensors Data Curves for incoming data for standpipe pressure and surface back pressure. These curves are mapped to incoming data on the Setup > Mappings > Parameters page. See "Parameters".
T&D Calibrated FF Curves for calibrated friction factors, calculated by the torque and drag model.
T&D Derived Loads on Bit Curves for torque on bit and WOB, calculated by the torque and drag model.
T&D FF Sensitivity Depth Data

Depth-based curves showing the effects of different friction factors on hookload and surface torque. These curves are primarily used in the T&D FF Sensitivity Depth Data indicator () . See "Friction factors indicator".
T&D Plan Depth Data Depth-based curves showing the predicted hookload and surface torque, calculated by the torque and drag model.
T&D Real-Time Time-based curves showing real-time hookload, surface torque, and other values calculated by the torque and drag model.
User Data Depth-based curves for the custom curve sets on the Setup > User Data page. See "User data".
Well Data Curves for the current rig operation and the lithology of the formation. See "Adding a lithology log".
Well Interval Configuration Curve for the casing depth of the current well interval, defined on the Setup > Wells > Well Intervals page. See "Well intervals".
Well Parameters Curves for the well parameters defined on the Setup > Mappings > Parameters page. See "Parameters".
WITS Data Curves for the WITS records defined on the Setup > DAQ > WITS > In page. See "WITS data".
WITSML Channel Curves for the WITSML data defined on the Setup > DAQ > WITSML page. WITSML channels are grouped according to the channel name. See "WITSML channels".
