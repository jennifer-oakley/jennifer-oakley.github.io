About dashboards

Dashboards are fully customizable grids that you can add graphs, gauges, indicators, and other items to, using any combination of curves. These items can be added from the Dashboard Editor, which also lists the available curves for CENTRO® Engineer. See "Curves on the Dashboards screen".

There are several types of items you can add to a dashboard:

  • Gauges show the current value for a curve, giving you a quick visual indication of the status for any parameter or calculated value. To create a gauge, drag a curve to any empty spot on the current dashboard. See "Gauges".
  • Graphs plot curve data on a time-based or depth-based index. The different types of graphs are shown under the Graphs heading in the Dashboard Editor. See "Graphs".
  • Indicators are widgets used to display a variety of information, such as wellbore trajectory, rig activity, block height, and more. The different types of indicators are shown under the Indicators heading in the Dashboard Editor. See "Indicators".
  • Other items, such as labels and process flows can also be added to dashboards. See "Adding other items to a dashboard".

All of the dashboards that have been created for the current well can be viewed on the Dashboards screen. The left side of the Dashboards screen contains the names and thumbnail images for each dashboard. A check mark () in the top left corner of the dashboard thumbnail indicates which dashboard is the default dashboard. The default dashboard is the dashboard that opens when you open the CENTRO® Engineer application.

Note

The Dashboards screen is only available in Real-Time mode. See "Application modes".

Creating dashboards

To create a dashboard:

  1. On the Dashboards screen, click the Add button at the top of the Dashboard Editor panel.
  2. In the grid that appears, drag the mouse to select the grid size of the new dashboard.
  3. Add graphs, gauges, and other items to the new dashboard that appears. See "Gauges" and "Graphs".
  4. When you have finished creating your dashboard, click the Minimize button () to close the Dashboard Editor panel and save your changes.

A new dashboard will appear on the Dashboards screen. A thumbnail image for the new dashboard will appear in the Dashboard Editor panel, with a default name for the dashboard. As you add graphs and gauges to the dashboard, the thumbnail image will be updated with a preview of the dashboard.

Adding other items to a dashboard

To add a label or process flow arrow to a dashboard:

  1. On the Dashboards screen, select the dashboard that you want to add an item to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, expand the Others heading.
  4. Click on the icon for the item you want to create, then drag it to a blank area on the dashboard.
    • To add a text box, click on the Label icon ().
    • To add a process flow arrow, click on the Process Flow icon ().
  5. If you want the item to span across multiple grid areas, click on the bottom right corner of the item and drag it into an adjacent blank area.

If you added a label, a text box will appear on the dashboard with the text you entered. To customize the default font settings, click the Menu button () in the top left corner of the text box.

If you added a process flow arrow, an arrow will appear on the dashboard. To change the type of arrow, click the Menu button () in the top left corner of the text box.

Importing dashboards from another well

To import dashboards from another well in CENTRO® Engineer:

  1. Go to the Project tab on the Projects > All Projects > Import Objects page.
  2. In the Well field, select the well that you want to import dashboards from.
  3. After the list of dashboards appears, select the dashboards you want to import.
  4. Click the Save Changes button () at the top right corner of the screen.

The selected dashboards will be imported into the current well. They will appear on the Dashboards screen, with the other dashboards for the well. If any of the imported dashboards has the same name as an existing dashboard, an asterisk (*) will be added to the name of the imported dashboard.

Deleting graphs and gauges

To delete a graph or gauge:

  1. On the Dashboards screen, select the dashboard that contains the graph or gauge you want to delete.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. Hover your mouse over the top right corner of the graph or gauge, then click the Delete button ().

The item will be removed from the dashboard, and the thumbnail image of the dashboard will be updated in the Dashboards panel.

Using the Dashboard Editor panel

There are several keyboard shortcuts and mouse actions that you can use when the Dashboard Editor panel is open.

  • To open a dashboard in a separate window, double-click the thumbnail of the dashboard in the Dashboard Editor panel.
  • To resize a floating dashboard, click and drag the right edge or the bottom edge of the window.
  • To rename a dashboard, click on the name of the dashboard in the Dashboard Editor panel and enter a new name.
  • To add a new row or column to a dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

When you click the menu button () next to a dashboard name, several options will appear:

  • Edit – Opens the Dashboard Editor panel, which allows you to add gauges, graphs, and indicators to the dashboard.
  • Copy – Copies a screenshot of the dashboard to the clipboard
  • Clone – Creates a new dashboard with the same graphs, gauges, and indicators as the original dashboard. The Clone option is disabled for the default dashboards.
  • Delete – Deletes the selected dashboard from the well.
  • Capture – Captures a screenshot of the dashboard. Screen captures are stored in your OneSync folder, under the C:\Users\[UserName]\Documents\OneSync\Images directory. The name of each screen capture includes the date and time that the screen capture was taken.

The toolbar at the top of the Dashboard Editor panel is used to create dashboards, undo changes, redo changes, and save changes.
