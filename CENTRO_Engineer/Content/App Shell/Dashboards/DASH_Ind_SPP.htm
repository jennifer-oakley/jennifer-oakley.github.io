About the SPP indicator

The SPP Modeling Static indicator () generates a depth graph which compares real-time standpipe pressure to the standpipe pressure values calculated by the hydraulics model. SPP values are only calculated for the openhole section of the well, during drilling operations. The hydraulics model calculates static SPP values for different flow rates, using the settings defined in the SPP Modeling Static indicator settings. See "Updating the indicator settings". The curve data calculated for the SPP Modeling Static indicator can also be sent to the WITSML server as outgoing depth data. See "Publishing outgoing depth data".

There are two modes for determining which work string, well interval, well path, and pump is used in SPP calculations.

  • In Standard mode, CENTRO® Engineer will use the selected work string with the active well interval, well path, and pump fluid for the calculations. By default, Standard mode uses the active work string.
  • In Well Activity mode, CENTRO® Engineer will use the selected well activity to determine the work string, well interval, well path, and pump fluid to use for the calculations. The data shown in the graphs will also be filtered by the well activity's start and end times. Well activities are defined on the Setup > MVA Settings > Well Activity Planner tab. See "Well activity planner".

You can change the model parameters, current calculation mode, and the work string in the indicator settings. See "Updating the indicator settings".

By default, depth graphs show the entire well. As well depth increases, the graph range will expand to show the new maximum well depth. You can change the graph range by clicking the Range button () or the Zoom button (), but the graph will continue to expand unless you set a manual range. See "Applying a manual range".

Adding the SPP indicator

To add the SPP Modeling Static indicator to a dashboard:

  1. On the Dashboards screen, select the dashboard that you want to add the indicator to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, expand the Indicators heading.
  4. Click the SPP Modeling Static icon () and drag it to the dashboard.

To change the default indicator settings, hover your mouse over the top left corner of the indicator, then click the Menu button (). See "Updating the indicator settings". To change the default graph settings, click the Edit Graph Page button () in the bottom right corner of the graph index. See "Updating graph settings" and "Updating curve and track settings".

If you have configured outgoing depth data, CENTRO® Engineer will automatically begin sending curve data for SPP to the WITSML server. See "Publishing outgoing depth data".

Updating the indicator settings

The indicator settings for the SPP Modeling Static indicator includes the settings for the model parameters, calculation mode, and work string used in the SPP calculations. You can also define a time range to filter the data shown in the graph. To open the indicator settings, hover your mouse over the top left corner of the SPP Modeling Static indicator, then click the Menu button ().

Publishing outgoing depth data

You can configure the SPP Modeling Static indicator to automatically publish SPP data to a depth log on the WITSML server.

To publish data from the SPP Modeling Static indicator to WITSML:

  1. On the Dashboards screen, select the dashboard that contains the SPP Modeling Static indicator.
  2. Hover your mouse over the top left corner of the indicator, then click the Menu button ().
  3. In the panel that appears, select the Publish to WITSML option and enter a name for the WITSML log.
  4. Go to the Setup > DAQ > WITSML Out (Depth) page.
  5. Select the Standpipe Pressure parameter, listed under the HydraulicsDepthData category.
  6. Click the Save Changes button ().

The next time the model is calculated, CENTRO® Engineer will create a new depth log with the log name specified in the indicator settings. Separate SPP curves will be created in the log for each flow rate. Each time new values are calculated, the existing data in the log will be deleted and replaced with the new data.

Updating graph settings

The general SPP indicator settings allow you to customize the appearance of the legend, axes, and grid lines in the graph. If the dashboard contains multiple graphs, these changes will not affect the other graphs in the dashboard.

To update the settings for the tracks in a SPP indicator:

  1. On the Dashboards screen, select the dashboard that contains the SPP indicator you want to edit.
  2. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  3. Click the Edit Graph Page button () to open the Graph Page Settings window.
  4. Update the general settings as desired.

As you make changes to the graph, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

Updating curve and track settings

The Graph Page Settings window contains several groups of settings for each track in the SPP indicator:

  • The Track tab contains settings for the caption, grid lines, and annotations.
  • The Curve tab contains settings for each curve in the track, including scaling options, line and marker colors, and smoothing methods.
  • The Axis tab contains settings for the axis of each curve, such as the axis direction, measurement unit, and tick mark options.

To update the settings for one of the tracks in a SPP indicator:

  1. On the Dashboards screen, select the dashboard that contains the SPP indicator with the track you want to edit.
  2. Hover your mouse over the bottom right corner of the track. A toolbar will appear on top of the graph.
  3. Click the Edit Graphs button () to open the Graph Page Settings window.
  4. Update the settings as desired.
    • To edit the track settings, go to the Track tab.
    • To edit curve settings, go to the Curve tab and select the curve you want to edit.
    • To edit axis settings, go to the Axis tab and select the curve you want to edit.

As you make changes to the track, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.
