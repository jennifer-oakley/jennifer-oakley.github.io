About time graphs

Time graphs plot curve data against time. Time graphs have two axes:

  • The primary axis is the graph index, time. The data in every track is plotted against the data for the graph index. In vertical graphs, the graph index is on the right side of the screen. In horizontal graphs, the graph index is at the bottom of the screen.
  • The secondary axis contains the curve data. The curve data is plotted against the graph index. In vertical graphs, the secondary axis is at the top of the screen. In horizontal graphs, the secondary axis is on the left side of the screen.

By default, time graphs automatically scroll as new data is received. You can change the graph range by clicking the Range button () or the Zoom button (), but the graph will continue to scroll unless you set a manual range. See "Applying a manual range".

Adding time graphs to a dashboard

To add a time-based graph to a dashboard:

  1. On the Dashboards screen, select the dashboard that you want to add a time-based graph to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, expand the Graphs heading.
  4. Click on the icon for the type of graph you want to create, then drag it to a blank area on the dashboard.
    • To create a vertical time-based graph, click the Vertical Time Based Graph button ().
    • To create a horizontal time-based graph, click the Horizontal Time Based Graph button ().
  5. If you want the graph to span across multiple grid areas, click on the bottom right corner of the graph and drag it into an adjacent blank area.

A blank graph area will appear on the dashboard, which you can add tracks and curves to. See "Adding tracks to a graph" and "Adding curves to a track".

Tip

If there aren't any blank areas on the dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

Printing time graphs

To print the curves on a time-based graph:

  1. On the Dashboards screen, select the dashboard that contains the time-based graph you want to print.
  2. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  3. Click the Print Data button () to open the print settings.
  4. In the window that opens, enter the starting and ending points for the report in the From and To fields.
  5. Update the remaining print settings as desired, then click the Preview button to update the print preview.
  6. On the right side of the screen, select a print option.
    • To print a physical copy of the time-based graph, click the Print button ().
    • To export to a PDF file, click the Export to PDF button ().
    • To export to a Word file, click the Export to Word button ().
    • To export to an Excel file, click the Export to Excel button ().

If you chose one of the export options, you will be prompted to save the report. When the report has finished saving, it will open automatically.

Using time graphs

The toolbar for the graph is used to take screenshots, set a display range, and customize the settings for all of the tracks in the graph. To open the toolbar for the graph, hover your mouse over the bottom right corner of the graph index.
