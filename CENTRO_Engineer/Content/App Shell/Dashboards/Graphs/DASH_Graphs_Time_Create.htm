About creating time graphs

After you add a time-based graph to a dashboard, you'll need to add tracks and curves to the graph.

  • Tracks separate the graph into smaller sections and function as a container for the curve data. By default, graphs include one track. However, you add more tracks as needed. See "Adding tracks to a graph".
  • Curves contain the data that you are graphing. In the Dashboard Editor panel, curves are categorized according to the type of data. Each track can contain multiple curves. See "Adding curves to a track".

Curves from certain categories cannot be added to time-based graphs. If a curve can't be added to the graph, the cursor will change to the Disabled icon ().

Adding tracks to a graph

To add a track to a time-based graph:

  1. On the Dashboards screen, select the dashboard that contains the time-based graph you want to add a track to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, locate the first curve that you want to appear in the new track.
  4. Select the name of the curve. Drag it to the graph index, and drop it over the Track label.

A new track will be added to the graph. To edit the track settings, click the Edit Graphs button () in the bottom right corner of the track. See "Updating curve and track settings".

Adding curves to a track

To add a curve to track:

  1. On the Dashboards screen, select the dashboard that contains the time-based graph you want to edit.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, locate the curve that you want to add.
  4. Select the name of the curve, and drag it to the track that you want to add the curve to.

The data for the new curve will appear in the selected graph track. To edit the curve settings, click the Edit Graphs button () in the bottom right corner of the track. See "Updating curve and track settings".

Note

Raw data curves, such as mnemonics for a WITS channel, will be deleted from the graph if the associated channel is deleted. However, WITSML curves will be retained if a new channel is created with matching mnemonics.

Copying a graph to a dashboard

Copying a graph allows you to copy an existing graph and paste it into another dashboard. The new graph will be identical to the original graph, with the same curves, line colors, graph settings, and graph ranges.

Note

The new graph must be pasted into a blank area on a dashboard. If there aren't any blank areas on the desired dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

To copy a graph to a dashboard:

  1. On the Dashboards screen, select the dashboard that contains the graph you want to copy.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  4. Click the Clone Item button ().
  5. Select the dashboard that you want to copy the graph to.
  6. Right-click on a blank area in the dashboard.

A copy of the original graph will appear in the blank area. The new graph will be identical to the original graph, with the same curves, line colors, graph settings, and graph ranges.
