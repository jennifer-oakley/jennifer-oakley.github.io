About customizing cross plots

There are several ways that you can customize the appearance of a cross plot.

  • Change the range of the data shown. The Calendar button () sets a manual time range for the data displayed in the cross plot. See "Applying a manual range".
  • Update the overall graph settings. The general graph settings control the appearance of the legend, axes, and grid lines for all of the tracks in the graph. See "Updating cross plot settings".
  • Update the curve and track settings. The track settings control the appearance of the caption, grid lines, and annotations for the curves in the track. The curve settings control the appearance of the line and markers for the curve, as well as scaling and smoothing options. See "Updating curve and track settings".

Applying a manual range

There are multiple ways to apply a manual time range to a cross plot:

  • Manually enter a time frame. To change the start date, click the Calendar button () in the From field. To change the end date, click the Calendar button () in the To field. If you leave the To field blank, the graph range will automatically expand as new data is received.
  • Choose a bit run. To select a time range based on a bit run, click the Bit Runs button and select the bit run you want to view data for. Bit runs are defined in the Bit Runs indicator. See "Bit runs".

To apply a manual range to a graph on the Dashboards screen:

  1. Go to the Dashboards screen, then select a dashboard.
  2. Hover your mouse over the bottom right corner of the graph you want to filter. A toolbar will appear on top of the graph index.
  3. Click the Manual Range button () to open the Time Range window.
  4. Select the time range that you want to view data for.
  5. Click the Set Range button () to apply the filter.

The data from the selected time range will load in the curves on the current graph. Gray vertical bars will appear in the middle of each track to indicate that the data is not displayed in real-time. To reset the graph back to the default view, click the Fit button ().

Tip

The end date for the time range can also be set to a future date.

Updating cross plot settings

The general cross plot settings allow you to customize the appearance of the legend, axes, and grid lines in the graph. Any changes made to the general settings will affect all of the tracks in the cross plot. If the dashboard contains multiple graphs, these changes will not affect the other graphs in the dashboard.

To update the settings for the tracks in a cross plot:

  1. On the Dashboards screen, select the dashboard that contains the cross plot you want to edit.
  2. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  3. Click the Edit Graph Page button () to open the Graph Page Settings window.
  4. Update the general settings as desired.

As you make changes to the graph, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

Updating curve and track settings

The Graph Page Settings window contains several groups of settings for each track in the cross plot:

  • The Track tab contains settings for grid lines and the caption.
  • The Curve tab contains settings for each curve in the track, including scaling options, line and marker colors, and smoothing methods.
  • The Axis tab contains settings for the axis of each curve, such as the axis direction, measurement unit, and tick mark options.

To update the settings for one of the tracks in a cross plot:

  1. On the Dashboards screen, select the dashboard that contains the cross plot with the track you want to edit.
  2. Hover your mouse over the bottom right corner of the track. A toolbar will appear on top of the graph.
  3. Click the Edit Graphs button () to open the Graph Page Settings window.
  4. Update the settings as desired.
    • To edit the track settings, go to the Track tab.
    • To edit curve settings, go to the Curve tab and select the curve you want to edit.
    • To edit axis settings, go to the Axis tab and select the curve you want to edit.

As you make changes to the track, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.
