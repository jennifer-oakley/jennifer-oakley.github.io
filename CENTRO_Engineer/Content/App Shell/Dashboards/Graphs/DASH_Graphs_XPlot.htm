About cross plots

Cross plots are scatter plots that allow you to compare one set of curve data to another. Cross plots have three axes:

  • The X-axis is the graph index. The data in every track is plotted against the data for the X-axis. In vertical cross plots, the X-axis is on the right side of the screen. In horizontal cross plots, the X-axis is on the bottom of the screen.
  • The Y-axis contains the curve data. The curve data is plotted against the data for graph index. In vertical cross plots, the Y-axis is at the top of the screen. In horizontal cross plots, the Y-axis is on the left side of the screen.
  • The Z-axis gives context to the data plotted for each track. The data in each track is color-coded according to the ranges defined for the Z-axis. In vertical cross plots, the Z-axis is on the left side of the screen. In horizontal cross plots, the Z-axis is at the top of the screen.
Note

Only time-based curve data can be plotted on a cross plots. Depth-based curve data, including data from WITSML depth logs, cannot be plotted on a cross plot.

Adding cross plots to a dashboard

To add a cross plot to a dashboard:

  1. On the Dashboards screen, select the dashboard that you want to add a cross plot to.

  2. Click the menu button () next to the dashboard name, then select the Edit option in the menu that appears.

  3. In the Dashboard Editor panel that appears on the left, expand the Graphs heading.
  4. Click on the icon for the type of cross plot you want to create, then drag it to a blank area on the dashboard.
    • To create a vertical cross plot, click the Vertical Cross Plot button ().
    • To create a horizontal cross plot, click the Horizontal Cross Plot button ().
  5. Locate the curve you want to use as the X-axis. Drag it to the graph index, and drop it over the X-Axis label.
  6. Locate the curve you want to use as the Z-axis. Drag it to the graph index, and drop it over the Z-Axis label.
  7. In the Graph Page Settings window, define the ranges you want to use for the Z-axis. See "Defining ranges for the Z-axis".
  8. Locate the curve you want to use as the Y-axis, and drag it to the center of the graph.
  9. If you want the cross plot to span across multiple grid areas, click on the bottom right corner of the cross plot and drag it into an adjacent blank area.

The Y-axis curve will appear on the graph, plotted against the data for the X-axis. The data will be color-coded according to the ranges defined for the Z-axis. You can also add additional tracks and curves to the cross plot. See "Adding tracks to a cross plot" and "Adding curves to a track".

Tip

If there aren't any blank areas on the dashboard, hover your mouse over any side of the dashboard. When the mouse cursor changes to a double arrow, drag the edge of the dashboard to create a new row or column.

Printing cross plots

To print the curves on a cross plot:

  1. On the Dashboards screen, select the dashboard that contains the cross plot you want to print.
  2. Hover your mouse over the bottom right corner of the graph index. A toolbar will appear on top of the index.
  3. Click the Print Data button () to open the print settings.
  4. In the window that opens, enter the starting and ending points for the report in the From and To fields.
  5. Update the remaining print settings as desired, then click the Preview button to update the print preview.
  6. On the right side of the screen, select a print option.
    • To print a physical copy of the cross plot, click the Print button ().
    • To export to a PDF file, click the Export to PDF button ().
    • To export to a Word file, click the Export to Word button ().
    • To export to an Excel file, click the Export to Excel button ().

If you chose one of the export options, you will be prompted to save the report. When the report has finished saving, it will open automatically.

Using cross plots

The toolbar for the cross plot is used to take screenshots, set a display range, and customize the settings for all of the tracks in the cross plot. To open the toolbar for the cross plot, hover your mouse over the bottom right corner of the graph index.

Toolbar buttons for cross plots
Button Function

Resets the graph range to the default range
Opens the Graph Page Settings window for editing the general graph settings. See "Updating cross plot settings".

Exports all chart data to a CSV or LAS file. The export will include all data in the current range, at one-second intervals.
Captures a screenshot of the graph. Screen captures are stored in your OneSync folder, under the C:\Users\[UserName]\Documents\OneSync\Images directory. The name of each screen capture includes the date and time that the screen capture was taken.
Sets the range of historical data to display. Real-time updates are disabled while using a historical data range. See "Applying a manual range".
Prints the curves on the selected cross plot. See "Printing cross plots".

Copies a screenshot of the graph to the clipboard
Creates a copy of the graph, which can be pasted into a blank area on a dashboard. See "Copying a graph to a dashboard".

Each track in the cross plot also has its own set of toolbar buttons. To open the track toolbar, hover your mouse over the bottom right corner of the track.
