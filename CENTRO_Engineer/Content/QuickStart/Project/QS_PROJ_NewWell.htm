This video tutorial will teach you about the different ways to create a new project in CENTRO® Engineer. You can create a blank project from scratch, or you can copy the configuration of an existing well. (3:49)

Warning

You must be signed into Microsoft with your Weatherford log-in to view this video. If the video does not automatically load, click the Sign In link to sign into your account. If the sign-in window does not appear, make sure that pop-ups are not blocked.

In the CENTRO® Engineer application suite, all of the data related to a job is stored with the well. Wells are managed from the Projects > All Projects section in CENTRO® Engineer. In the All Projects section, you can manage projects, import well objects into the current well, import well configurations, and export data.

To create a new well:

  1. Go to the Projects screen, then expand the All Projects heading.
  2. Go to the Projects page.
  3. Click the Add button () at the top of the screen.
  4. Enter the project name and the well name.
  5. In the Based On Well field, select the well configuration you want to copy. To use the default configuration, select the Default option.
  6. Enter the remaining job and well information.
  7. If the well is a historical well, select the Mark For Reprocessing option. See "Reprocessing data".
  8. In the Data Source field, select the primary source of data for the well.
    • WITSML – The primary source of data is a WITSML log. When this option is selected, gap filling will be enabled for the well. See "Gap fill".
    • Other – The primary source of data is not a WITSML log. When this option is selected, gap filling will not be enabled for the well.
  9. Click the Save Changes button () to save the well.

The well will be added to the CENTRO® Engineer database, and it will automatically become the active well. The new well will open on the Dashboards screen, and the project name will appear in the banner at the top of the screen. All of the information when you created the well will appear on the Projects > Job Info > General page. See "Job Info".

After you create a new well, the next step is to assign the unit system that you want to use. See "Choosing a unit system".
