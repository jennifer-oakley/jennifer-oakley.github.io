About fluids

This video tutorial will teach you how to add a drilling mud to a well. The mud properties are used to calculate the real-time hydraulics. They are also used for analyses on the Planning screen. (2:37)

Warning

You must be signed into Microsoft with your Weatherford log-in to view this video. If the video does not automatically load, click the Sign In link to sign into your account. If the sign-in window does not appear, make sure that pop-ups are not blocked.

The next step is to define the drilling and cementing fluids. The Setup > Fluids section contains the properties for the muds, cements, underbalanced drilling fluids, and hydrocarbons that will be used during drilling and cementing operations.

When you create a new mud, the basic fluid properties are entered in the left side of the screen. The basic fluid properties listed in the Parameters section include mud type, mud weight, temperatures, and the fluid composition. If the mud report includes additional information, such as salt type and salinity, you can enter this information in the Advanced Properties section. If the mud report does not include this information, you can simply use the default values.

There are two steps to adding mud information:

  1. Enter the basic properties of the mud, including the type of fluid and the fluid density. See "Adding a mud".
  2. Enter the fluid's rheological properties. See "Entering rheological properties".

Adding a mud

To add a drilling mud:

  1. Go to the Setup > Fluids > Mud page.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of fluids.
  3. In the Mud Type field, select the base used for the mud.
  4. In the Fill Color field, select a color to represent the mud in the Well View pane.
  5. In the Mud Weight In field, enter the density of the mud when it enters the wellbore.
  6. In the Mud Inlet Temperature field, enter the inlet temperature of the mud in the suction tank.
  7. In the Ref. Temperature field, enter the temperature at which the mud weight was measured.
  8. Enter the composition of the mud in the Retort Oil and Retort Water fields.
  9. In the Rheology Input menu, select the rheological properties tab to be used to perform the calculations. Only available in Plan Mode.
  10. If available, enter the remaining information under the Advanced Properties heading. See "Field definitions".
  11. Use the tabs in the bottom portion of the screen to enter the mud's rheological properties. See "Entering rheological properties".
  12. Click the Save Changes button ().

The mud will be stored in CENTRO® Engineer's database. When you change the pump fluid, you'll be able to select the new mud under the Mud utility. See "Mud utility". When you run an analysis, you'll also be able to select the new mud in the analysis settings. However, you will not be able to edit the fluid properties or dial readings in Real Time mode.

Entering rheological properties

In CENTRO® Engineer, the mud's rheology can be calculated from dial readings or rheological properties, such as plastic viscosity and yield point. There are four ways to define the rheology of a mud:

  • Dial Readings. This method uses viscometer readings to define rheological properties. You can use this method for any mud. See "Dial readings".
  • PV YP LSYP. This method is best for Bingham plastic fluids, which have a yield point that must be exceeded before the mudwill flow. See "PV YP LSYP properties"..
  • H-B Direct. This method uses parameters for the Herschel-Bulkley rheology model, which can be used for Bingham plastic fluids or Power law fluids. See "H-B Direct properties"..
  • PVT Dial Readings. This method uses multiple sets of dial readings to define rheological properties. You can use this method for any mud. See "PVT dial readings".
  • Temperature Dependent Rheology. This method calculates the dial readings for the mud automatically, based on the mud weight and temperature. See "Temperature dependent rheology".

You can only use one of these methods to enter rheological properties. When you save the mud, only the information in the active tab will be saved. The remaining tabs will be hidden for that fluid.

Warning

In Real Time mode, switching between the Dial Readings, PV YP LSYP, H-B Direct, and Dial Readings (PVT) tabs may cause you to lose data. When you select another tab, the data on the previous tab is reset to the default values. Any data that you've entered will be lost.

In Plan mode and MPD mode, the data you enter will be saved even when switching between tabs.

After you add the drilling fluids, the initial setup process for CENTRO® Engineer will be complete. Now you are ready to run analyses on the Planning screen. See "Planning screen". Before you begin monitoring incoming data, however, you'll need to choose the initial pump fluid. See "Setting pump fluids".
