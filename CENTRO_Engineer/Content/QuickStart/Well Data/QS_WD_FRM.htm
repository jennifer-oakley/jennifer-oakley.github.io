About formation data

The next step is to describe the wellbore formation. The Setup > Wells > Formations page contains the geological and geophysical characteristics of the formation layers in the wellbore. These properties include porosity, permeability, pore pressure, and fracture pressure. Formation layers are indexed by measured depth along the length of the wellbore, from the surface to the bit. The TVD of each layer is automatically calculated, based on the wellpath data for the well.

CENTRO® Engineer allows you to store multiple sets of formation data, based on the source of the data. The Source Type menu at the top of the screen is used to switch between the different sets of formation data. There are five source types:

  • None – No information is available about the source of the data.
  • Interpreted – The data was acquired by analyzing the available samples.
  • Core – The data was acquired through core testing.
  • Cuttings – The data was acquired through cuttings tests.
  • Unknown – The source of the data is unknown.

Interpreted formation data is used for model calculations in CENTRO® Engineer. The other source types are only used for record keeping, and are not used in any calculations.

Adding formation layers

To describe the wellbore formation, you'll need to update the properties of the first formation layer, then add formation layers to describe the rest of the wellbore formation. You can also import formation data from a CSV or LAS file. See "Importing formation data from a file".

The first layer of the formation is created for you, using default properties. The measured depth of this layer is calculated automatically, based on the rig elevations entered on the Equipment Configuration page. See "Entering rig elevations".

To add a formation layer:

  1. Go to the Setup > Wells > Formations page.
  2. Go to the Source Type menu at the bottom of the screen, and select the source of the formation data. See "Field definitions".
  3. Update the default properties of the first row in the table.
  4. Click the Insert Below button () at the end of the row to add a new formation layer.
  5. In the Lithology column, select the type of formation. See "Lithologies catalog".
  6. Enter the name of the formation layer in the Name column.
  7. Enter the MD at the top of the formation layer in the MD column. TVD will be automatically calculated from the survey data. See "Importing wellpath data".
  8. Enter the remaining formation properties.
  9. Add additional formation layers as needed.
  10. When you have finished entering formation data, click the Save Changes button ().

The formation data will be stored in CENTRO® Engineer's database. A preview of the formation layers in the well will appear in the Well View pane. If the rig is located offshore, a layer of water will appear at the top of the Well View pane. This layer represents the water between the kelly bushing and the sea floor.

To publish the formation layers to WITSML, click the Publish Formations button (). The name, MD, and TVD of each formation layer will published to WITSML as a separate formation marker. Formation layers are published to the first WITSML channel listed on the Setup > DAQ > WITSML page. See "WITSML channels".

After you enter the formation data, the next step is to define the wellbore intervals. See "Creating wellbore intervals".
