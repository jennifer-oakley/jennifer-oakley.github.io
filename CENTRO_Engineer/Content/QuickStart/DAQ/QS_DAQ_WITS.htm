About WITS data

WITS wellsite information transfer specification is a two-way communication system used for sending and receiving drilling data. WITS data can only be sent across the network through a TCP/IP connection.

CENTRO® Engineer acts as a client for WITS data received from a TCP/IP server. CENTRO® Engineer cannot receive data from another TCP/IP client or a serial connection. If you need to receive WITS data from a serial connection, you'll need to convert the serial signal into a TCP/IP signal. This can done using either a physical or software-based serial-to-Ethernet server. For additional assistance, please contact theCENTRO® Engineer Support Team. See "Reporting problems".

The Setup > DAQ > WITS page is used to configure CENTRO® Engineer to send and receive WITS data. There are two basic steps for sending and receiving WITS data:

  • Add a WITS channel. To connect to a WITS server, you'll need to know the IP address and port number for the server. See "Adding WITS channels".
  • Add incoming records. CENTRO® Engineer will only receive data for the records listed on the WITS > In tab. See "Adding incoming records".

On the WITSML Out page, you can configure CENTRO® Engineer to send WITS data and calculated values from CENTRO® Engineer to the WITSML server. See "WITSML Out".

Adding WITS channels

The Setup > DAQ > WITS > Channels tab lists the incoming and outgoing WITS channels. This list tells CENTRO® Engineer the IP address and port number to use for sending and receiving WITS data. Before CENTRO® Engineer can send or receive WITS data, you'll need to add incoming and outgoing WITS channels.

Warning

If you delete a WITS channel, any associated raw data curves used in a dashboard on the Dashboards screen will also be deleted.

To add a WITS channel:

  1. Go to the Channels tab on the Setup > DAQ > WITS page.
  2. Click the Add button () at the top of the screen.
  3. In the new row that appears, enter a name for the channel in the Channel field.
  4. In the IP Address field, enter the IP address of the computer that will be sending or receiving the data.
  5. In the Port field, enter the port number on which the data will be sent or received.
  6. Click the Save Changes button () to save your changes.

The new channel will appear in the list of available WITS channels on the DAQ > WITS > In and DAQ > WITS > Out tabs. The data acquisition service will also be restarted so that CENTRO® Engineer can connect to the new channel.

Adding incoming records

The Setup > DAQ > WITS > In tab lists the incoming WITS mnemonics that will be received by CENTRO® Engineer. This list tells CENTRO® Engineer what WITS data to look for and where to look for it. Before CENTRO® Engineer can receive any WITS data, you'll need to add entries for all incoming WITS mnemonics. Any incoming WITS data that is not listed on the DAQ > WITS > In tab will be ignored by CENTRO® Engineer.

To add an entry for an incoming WITS mnemonic:

  1. Go to the In tab on the Setup > DAQ > WITS page.
  2. Click the Add button () at the top of the screen.
  3. In the new row that appears, go to the Category field and select the WITS record that the data will be sent on.
  4. In the Tag ID field, select the name of the WITS mnemonic.
  5. In the Channel field, select the name of the WITS channel on which the mnemonic will be received.
  6. In the Title field, enter a name for the entry. The name can be the same as the WITS mnemonic, or it can be something different.
  7. In the Type field, select the type of measurement unit for the mnemonic, e.g. length or penetration rate. See "List of unit types".
  8. In the Unit field, select the measurement unit for the incoming data.
  9. When you have finished adding WITS mnemonics, click the Save Changes button () to save your changes.

The new entries will be added to the CENTRO® Engineer database. The names entered in the Title field will appear in the mapping settings and the WITS Data window. To start acquiring real-time data, click on the clock and press the Start button (). See "Real-time and playback controls". To view the incoming WITS data, click on the Guides utility (), then click on Indicators View and select the DAQ option. See "Data acquisition indicators".

After you set up the WITS data, the next step is to configure any incoming WITSML data. See "Setting up WITSML data".
