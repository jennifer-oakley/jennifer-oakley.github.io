About CENTRO® Engineer

Welcome to CENTRO® Engineer v2.4.15. CENTRO® Engineer gives you a single software platform for designing, planning, and monitoring drilling operations. CENTRO® Engineer allows you to plan nearly every detail of drilling a well, from the trajectory of the well to the BHAs used to drill the well. Once drilling begins, CENTRO® Engineer processes incoming real-time data, including WITS data and WITSML data.

CENTRO® Engineer uses the projected data for the well to run several analyses, allowing you to identify potential problems before you start drilling. Once drilling begins, CENTRO® Engineer uses customizable algorithms to automatically determine the rig activities and to perform real-time torque and drag calculations. Real-time data is graphed in several dashboard charts, making it easier to monitor well conditions.

Use CENTRO® Engineer to:

  • Monitor incoming real-time data from WITS, WITSML, or OPC UA sources. See "Data acquisition".
  • Create custom dashboards, using CENTRO® Engineer's powerful dashboard customization feature. See "Dashboards".
  • Compare planned data to the actual, drilled data. See "Model vs. Actual settings".
  • Define custom calculations. See "OneClick™ settings".
  • Define custom limits and alarms for the hydraulics model, calculations, WITS data, WITSML data, and well parameters. See "Set Limits".
  • Predict events and conditions that may cause non-productive time.
  • Predict ahead, based on real-time, calibrated models.
  • Create real-time drilling optimization reports. See "Monitoring reports".
  • Perform a OneClick™ analysis, which is a customizable one-click analysis that allows you to simultaneously run torque & drag, hydraulics, and BHA analyses using a common set of parameters. See "OneClick™ analysis tool".
  • Perform a geomechanics analysis, which uses offset well data to predict overburden pressure, hydrostatic pressure, and pore pressure. See "Geomechanics".
  • Perform a mechanics analysis, which examines the different forces acting on the BHA and predicts displacement, contact forces, vibrations, and critical speeds. Mechanics analysis also includes torque and drag analysis, which calculates friction factors and predicts the torque and drag forces generated by the drillstring. See "Mechanics analyses".
  • Perform a hydraulics analysis, which examines the pressure losses in the circulating system so that you can optimize the well hydraulics. See "Hydraulics analysis".
