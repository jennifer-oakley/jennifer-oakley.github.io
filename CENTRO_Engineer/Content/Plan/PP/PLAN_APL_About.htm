About lithology analysis

The Lithology tab identifies the locations of shale and clay formations, using imported log data and the shale formation layers defined on the Setup screen. On the Lithology tab, you can import log data for ROP, gamma ray, and spontaneous potential.

There are four charts on the Lithology tab. The first three charts graph data from various sources, and the last chart is used to graph the results for the Lithology tab.

  • The Well Data chart graphs the shale formation layers entered in the Setup screen.
  • The Offset Well chart graphs imported log data from an offset well.
  • The Current Well chart graphs real-time data from the current well.
  • The Results chart graphs the calculated shale areas in the formation.

You'll need to add data for the Offset Well chart and the Current Well chart. If there is no log data for the current well available, you can import additional offset well data into the Current Well chart. However, the data in the Current Well chart is weighted more heavily in the calculated results than the data in the Offset Well chart. When both charts contain data, the data in the Offset Well chart may be completely disregarded in the results.

Identifying shale areas

To configure shale analysis:

  1. Go to the Lithology tab on the Planning > Geomechanics page.
  2. If the experiment is not unlocked for editing, click the Edit button () at the top of the screen.
  3. Import log data into the Offset Well and Current Well charts. See "Importing data".
  4. For each chart that contains ROP or gamma data, use the Deliver Line tool () to define the minimum values found in shale and clay areas. See "Deliver lines".

  5. When you are finished editing the experiment, click the Save Changes button ().

The shale areas identified by the deliver lines will appear in the Results chart. The imported data and deliver lines will be saved to the experiment. To export the results data, click the Export to User Data button () in the Results graph. See "Exporting shale data".

Importing data

On the Lithology tab, there are three types of log data you can use to identify shale areas: gamma ray, ROP, and spontaneous potential.

You can import LAS and CSV files into the lithology charts. The import data must be formatted into vertical columns, with the column names in the first row. The options listed for the Value axis are automatically determined by CENTRO® Engineer, based on the column names in the first row of the import file.

To import lithology data:

  1. Select the experiment you want to edit, then click the Edit button ().

  2. Go to the chart you want to import data into, then click the Import button ().
  3. Navigate to the file you want to import and click Open. A list of import options will appear on top of the graph.
  4. Select the options for the TVD data in the file you are importing.
    • The TVD axis is the name of the column for TVD.
    • The TVD unit is the measurement unit used for TVD.
  5. Select the options for the type of log data you want to import.
    • The Value axis is the name of the column that contains the log data.
    • The Log type is type of log data you're importing (e.g. ROP or gamma ray).
  6. Select the options for the measurement unit.
    • The Value unit type is the type of measurement unit used for the log data.
    • The Value unit is the measurement unit used for the log data.
  7. Change the default line and symbol options as desired.
  8. Click the OK button () to save your changes.

The log data will be imported into CENTRO® Engineer, and the data will be graphed in the selected chart. When you are finished editing the experiment, click the Save Changes button ().

Exporting shale data

After you have identified the shale areas on the Lithology tab, you can export the data in the Results graph to the Dashboards screen. To export shale data to the list of available curves on the Dashboards screen:

  1. Go to the Lithology tab on the Planning > Geomechanics page.
  2. Select the experiment that you want to export data from.
  3. Click the Edit button () to unlock the experiment for editing.
  4. Import log data and add deliver lines to identify the shale areas. See "Identifying shale areas".
  5. Click the Export to User Data button () in the Results graph.
  6. In the window that appears, enter a name for the results data.
  7. To send the curve data to a depth-based WITSML log, select the Publish To WITSML option.
  8. Click the Apply button () to close the Lithology window.
  9. When you are finished exporting data, click the Save Changes button () in the main window to save the experiment.

The curve data will appear in a new tab on the Setup > User Data page. See "User data". If you selected the Publish To WITSML option, a new depth log will be created on the first WITSML channel listed on the Setup > DAQ > WITSML page. See "WITSML channels".

The curve data will also be added to the list of available curves on the Dashboards screen, under the User Data category. To graph the curve data, add it to a depth-based graph. See "Adding curves to a track".

Tip

To change the name of a curve, go to the Setup > User Data page. Go to the toolbar on the right side of the screen and click on the tab name. In that box that appears, enter a new name of the curve and press the Enter key.

Using the Lithology tab

When you are editing an experiment, the toolbar at the top of the Lithology tab contains tools for refining the data in the charts. To unlock an experiment for editing, click the Edit button ().
