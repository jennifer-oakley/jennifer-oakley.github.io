About geomechanics analysis

Geomechanics analysis examines the forces acting on the wellbore formation and the formation fluids. Geomechanics results are calculated using the formation data entered in the Setup screen, as well as data from an offset well and real-time data.

Each experiment on the Planning > Geomechanics page is divided into three tabs:

  • The Lithology tab identifies the locations of shale and clay formations. See "Lithology analysis".
  • The Overburden tab examines the pressure exerted by the weight of the overlying sediments in the formation. See "Overburden analysis".
  • The Pore Pressure tab analyzes the pressure acting on the formation fluids. See "Pore pressure analysis".

Before CENTRO® Engineer can calculate any results for a geomechanics analysis, you'll need to add the following data to the well:

  • The elevation of the rig. See "Rig Elevations".
  • The characteristics of the wellbore formation. See "Formation data".

Creating experiments

To add a new geomechanics experiment:

  1. Go to the Planning > Geomechanics page.
  2. Click the Add button on the right side of the screen.
  3. Click the Edit button () to unlock the experiment for editing.
  4. Configure shale analysis on the Lithology tab. See "Identifying shale areas".
  5. Configure overburden pressure analysis on the Overburden tab. See "Determining overburden pressure".
  6. Configure pore pressure analysis on the Pore Pressure tab. See "Determining pore pressure".
  7. When you are finished editing the experiment, click the Save Changes button ().

The calculated gradients for pore pressure, overburden pressure, and hydrostatic pressure will be updated in the Results chart on the Pore Pressure tab. The imported data, deliver lines, trend lines, and boundary areas will be saved to the experiment.

Using the Geomechanics page

The toolbar at the top of the screen allows you to unlock an experiment for editing.

Toolbar on the Geomechanics page
Button Function
Unlocked the selected geomechanics experiment for editing

The tabs on the right side of the screen are used to switch between geomechanics analysis experiments. Each experiment has its own tab, with a toolbar that is used to add, copy, and remove geomechanics analysis experiments.

Experiment toolbar on the Geomechanics page
Button Function
Add

Adds a blank geomechanics analysis experiment

Creates a copy of the current geomechanics analysis experiment

Deletes the current experiment, including all analysis results
