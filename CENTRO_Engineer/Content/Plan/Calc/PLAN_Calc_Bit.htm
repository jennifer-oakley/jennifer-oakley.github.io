About the bit configuration tool

The bit configuration tool on the Planning > Calculators > Bit page can be used to determine the best nozzle configuration for the drill bit. The bit configuration tool calculates bit pressure loss and hydraulic horsepower, based on the flow rate, mud weight, and nozzle configuration.

Note

The Bit page is only available in Real-Time mode and Planning mode. See "Application modes".

To use the bit configuration tool:

  1. Go to the Planning > Calculators > Bit page.
  2. In the Flow Rate field, enter the desired flow rate at the pumps.
  3. In the Mud Weight field, enter the fluid density of the mud entering the well.
  4. Add or remove bit nozzles as needed.
    • To add a nozzle, click the Add button () at the top of the screen.
    • To remove a nozzle, click the Delete button () in the row for that nozzle.
  5. For each bit nozzle, go to the Size field and select the desired nozzle size.

The flow area, flow rate, and jet velocity will be calculated for each bit nozzle. TFA, bit pressure loss, and hydraulic horsepower will also be calculated.

Field definitions

The table below defines the fields on the Planning > Calculators > Bit page.

Field definitions for the bit configuration tool
Field Name Definition
Flow Rate Desired flow rate at the pumps
Mud Weight Fluid density of the mud entering the well
Nozzles

Information about each nozzle on the bit.

  • Size – Diameter of the bit nozzle.
  • Flow Area –Calculated flow area through the nozzle, based on the size of the nozzle.
  • Flow Rate –Calculated flow rate through the nozzle, based on the overall flow rate and the size of the nozzle.
  • Jet Velocity – Calculated jet velocity through the nozzle, based on the overall flow rate and the size of the nozzle.
TFA Total flow area. TFA is automatically calculated from the size of each nozzle and the number of nozzles.
Bit Pressure Drop Pressure loss at the bit, calculated from the flow rate, mud weight, and TFA
Power Hydraulic horsepower at the bit, calculated from the flow rate, mud weight, and TFA
