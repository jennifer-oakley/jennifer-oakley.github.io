Starting a OneClick™ analysis

To run a OneClick™ analysis:

  1. Go to the Planning > OneClick™ page.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. Enter the expected operating parameters.
  4. If the BHA contains a directional sensor, enter the magnetic information about the well.
  5. To begin the analysis, click the Calculate button () in the top right corner of the screen.

When the analysis completes, a summary of the completed checks will appear on the right side of the screen. After you run a OneClick™ analysis, the Calculate button () in the top toolbar will be replaced with the Experiment Detail button (). This button is used to open a separate window, which contains the detailed results for each analysis type.

Field definitions

The table below defines all of the settings required for a OneClick™ analysis.

Required settings for a OneClick™ analysis
Field Definition
Wellpath

Wellpath used for the analysis
Well Interval

Well interval used for the analysis
Work String

Drillstring or casing string used for the analysis
Fluid

Circulating fluid used for the analysis
WOB Expected weight on bit
TOB Expected torque on bit
TDS Expected top drive speed
ROP Expected rate of penetration
Flow Rate Expected flow rate
Mag. Field Strength Magnetic field strength. Only available for BHAs that contain a directional sensor.
Mag. Dip Angle Magnetic dip angle. Only available for BHAs that contain a directional sensor.
Total Az. Correction Total azimuth correction, also known as the convergence angle. Only available for BHAs that contain a directional sensor.
