About OneClick™ analysis

On the Planning > OneClick™ page, you can run a OneClick™ analysis. OneClick™ analysis is a customizable one-click analysis that allows you to simultaneously run torque and drag, hydraulics, and BHA analyses using a common set of parameters. A OneClick™ analysis also completes a series of condition checks for drillstring design, torque and drag, hydraulics, and BHA vibrations. You can view the full list of these condition checks on the Setup > OneClick™ Settings > Planning > Conditions tab. See "OneClick™ analysis condition checks".

By default, a OneClick™ analysis includes the following checks and analyses:

  • A wellpath check to ensure that directional difficulty is within limits.
  • A series of checks for the design of the drillstring, including bending stiffness ratio, jar placement, stiffness ratio, junk slot area, and non-magnetic component spacing.
  • Several hydraulics analyses, including snapshot, parametric, tripping, and displacement analyses. See "Hydraulics analysis".
  • Mechanics analyses, including static BHA analysis, vibration BHA analysis, and torque and drag analysis. See "Mechanics analyses".

To customize the default analyses included in a OneClick™ analysis, go to the Setup > OneClick™ Settings > Planning > Flows tab. See "OneClick™ analysis flows".

Before CENTRO® Engineer can calculate any results for a OneClick™ analysis, you'll need to add the following data to the well:

  • The rig type and equipment configuration. See "Selecting rig equipment".
  • The elevation of the rig. See "Rig Elevations".
  • The planned well trajectory and survey data. See "Wellpath data".
  • The characteristics of the wellbore formation. See "Formation data".
  • The wellbore intervals for each cased and open hole section. See "Well intervals".
  • The temperatures along the length of the wellbore. See "Temperatures".
  • The influx zones in the wellbore formation. See "Kick/loss zones".
  • The proposed drillstring configuration. See "Work strings".
  • The fluid properties for the drilling mud. See "Drilling muds".
Note

If you are using an offline license key, all modules related to BHA analysis will be disabled. This includes BHA static and vibration analysis, real-time critical speed maps, and the BHA analysis portion of a OneClick™ analysis.

Note

The OneClick™ page is only available in Real Time mode and Planning mode. See "Application modes".

Using the OneClick™ page

The toolbar on the right side of the screen allows you to add and remove OneClick™ analysis experiments. The toolbar at the top of the screen allows you to inspect the wellbore trajectory, begin an analysis, and view experiment results.

Toolbar on the OneClick™ page
Button Function

Begins OneClick™ calculations
Filters the list of condition checks

Opens the Wellpath Inspection window, which allows you to compare the original survey to the corrected survey
Generates a report of the OneClick™ calculation results. See "Creating a pre-job engineering report".
Opens the results for each completed analysis in a separate window

The tabs on the right side of the screen are used to switch between OneClick™ analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove OneClick™ analysis experiments.

Experiment toolbar on the OneClick™ page
Button Function
Add

Creates a copy of the current OneClick™ analysis experiment
Opens the Drilling Schedule window, which allows you to create a custom drilling schedule report. See "Drilling schedules".

Deletes the current experiment, including all analysis results
