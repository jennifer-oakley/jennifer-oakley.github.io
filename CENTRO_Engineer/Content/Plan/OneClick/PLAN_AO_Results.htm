About OneClick™ analysis results

The results for a OneClick™ analysis include the results for all of the analyses included in the OneClick™ analysis. Each of these analyses uses the same settings. There are two types of results available for OneClick™ analysis:

  • Detailed results for each individual analysis, as well as a list of all the BHA components for the selected drillstring. See "Detailed analysis results".
  • A summary of the results, which includes a series of condition checks that allow you to quickly spot potential problems. See "Results summary".

You can also create several types of reports, such as a pre-job engineering report, a BHA design report, and a drilling schedule.

  • A pre-job engineering report summarizes the results of the analysis, including condition checks and detailed analysis results. See "Creating a pre-job engineering report".
  • A BHA design report lists all of the drillstring components. It also includes information about the bit, sensors, and stabilizers. See "BHA tool information".
  • A drilling schedule includes the drillstring, fluids, WOB, and other operating parameters that will be used to drill the interval. See "Drilling schedules".

Results summary

The results summary shows a series of condition checks that are performed during a OneClick™ analysis. During the analysis, CENTRO® Engineer uses the calculations on the Setup > OneClick™ Settings > Planning > Conditions tab to determine whether each condition passes or fails. To filter the list of condition checks, click the Filter button () in the top toolbar.

Column definitions for the results summary
Column Name Definition
(Blank)

Pass/fail status of each condition check.

  • A green box indicates that the check passed.
  • A red box indicates that the check failed.
  • A gray box indicates that the check was not run.
Category Category that the condition check is associated with
Comment Manually entered comment about the condition check. To enter a comment for a condition check, double-click inside the Comment cell for the condition check.
Measurement Name of the condition check that was evaluated
Rule Description of the criteria required in order for the condition check to pass

Detailed analysis results

The Experiment Detail window contains the settings, result summaries, and charts for each individual analysis included in the planning analysis. Each of these analyses are similar to the separate analyses available on the Planning screen. By default, there are nine tabs in the Experiment Detail window:

  • The BHA Builder tab contains a list of the BHA components in the drillstring, as well as information about the bit, sensors, and stabilizers. See "BHA tool information".
  • The Hyd. Snapshot tab contains the results from a hydraulics snapshot analysis. See "Snapshot results".
  • The Hyd. Parametric tab contains the results from a hydraulics parametric analysis. See "Parametric results".
  • The Hyd. Bit Hydraulics Opt. tab contains the results from a bit hydraulics optimization analysis. See "Bit hydraulics results".
  • The Hyd. Surge/Swab tab contains the results from a hydraulics swab surge analysis. See "Swab and surge analysis results".
  • The Torque And Drag (T&D) tab contains the results from a torque and drag analysis. See "Torque and drag results".
  • The BHA Static tab contains the results from a static analysis of the BHA. See "Static BHA analysis results".
  • The BHA Vibration tab contains the results from a vibration analysis of the BHA. See "BHA vibration analysis results".

Viewing detailed analysis results

After you run a OneClick™ analysis, the Calculate button () in the top toolbar will be replaced with the Experiment Detail button (). This button is used to open a separate window, which contains the detailed results for each analysis type.

To view the detailed results for a OneClick™ analysis:

  1. Go to the Planning > OneClick™ page.
  2. Select the experiment that you want to view results for.
  3. Click the Experiment Detail button () in the top toolbar.

The detailed analysis results will open in the Experiment Detail window. The tabs in the Experiment Detail window contain the settings, results summary, and analysis charts for each of the individual analyses included in the OneClick™ analysis. To view the charts for a specific analysis, go to the tab for that analysis and click the Show Charts button ().

Tip

You can change the settings used for any of the analyses shown in the Experiment Detail window. Update the settings as needed, then click the Calculate button () to re-run that specific analysis.

Creating a pre-job engineering report

After you run a OneClick™ analysis, you can create a report which summarizes the results of the analysis. The report includes the condition checks that are listed in the results summary, as well as detailed analysis results for each individual analysis included in the planning analysis. Pre-job engineering reports can be created in either MS Excel or PDF format.

To create a pre-job engineering report:

  1. Go to the Planning > OneClick™ page.
  2. Select the experiment that you want to view results for.
  3. Click the Save Report button () in the top toolbar.
  4. In the OneClick™ Report Options window, select the components that you want to include in the report.
  5. To change the default folder location and file type, click the Browse button () in the Path field.
  6. Click the Save Report button () at the bottom of the window.

The report will be generated with the selected report components. When the report finishes generating, it will open automatically. If you created the report in MS Excel format, click the Enable Content button to enable all of the report features when the report opens.
