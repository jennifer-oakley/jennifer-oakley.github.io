About the motor split flow tool

The motor split flow tool on the Planning > HyperLine™ Toolbox > Motor Split Flow page is used to predict the split flow through the motor bearing back and the Magnus RSS pads, based on the actual motor bearing measurements. It also calculates the nozzle size for the bearing pack bypass, which can be entered in the drilling motor properties on the Setup > Work String page. See "Drilling motors".

To use the motor split flow tool:

  1. Go to the Planning > HyperLine™ Toolbox > Motor Split Flow page.
  2. Enter the properties of the Magnus RSS, bit, drilling motor, and bearing pack. See "Inputs".
  3. Click the Calculate button.

The split flow results will be calculated for the drilling motor and the Magnus RSS. The results will appear in the Results Summary section, below the bearing pack measurements.

Inputs

The table below defines the calculation inputs on the Planning > HyperLine™ Toolbox > Motor Split Flow page.

Calculation inputs for the motor split flow tool
Section Field Name Definition
Fluid Data Total Flow Rate Total drilling flow rate through the pumps and surface lines
Mud Weight Mud weight of the drilling fluid
Viscosity Plastic viscosity of the drilling fluid
Magnus RSS Data RSS Size Size of the Magnus RSS tool
Restrictor Size

Diameter of the screen on the restrictor. If there is no restrictor on the RSS, select the No Restrictor option.
TFA (Restrictor) Total flow area through the restrictor
Cd (Restrictor) Coefficient used to calculate pressure loss across the restrictor. This is a read-only value, populated from the CENTRO® Engineer catalog.
Pad Port OD Diameter of the nozzles on the pads. This is a read-only value, populated from the CENTRO® Engineer catalog.
TFA (Pad) Total flow area through the nozzles on the pads. This is a read-only value, populated from the CENTRO® Engineer catalog.
Cd (Pad) Coefficient for the efficiency of the flow through the nozzles on the pads
Bit Data Qty

Number of bit nozzles at the specified size
Size

Size of each nozzle on the bit
TFA (Bit)

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles.
Cd (Bit) Coefficient for the efficiency of the flow through the nozzles on the bit
Additional BHA Pressure Loss HEL PL Pressure loss through the HEL tool
Other Tools PL Total of any other pressure losses between the drilling motor and the Magnus RSS
Motor Data Motor Size Size of the drilling motor used with the Magnus RSS
Motor dP Expected differential pressure through the drilling motor
Cd Motor Coefficient for the efficiency of the flow through the drilling motor
Bearing Pack Details Bearing Pack Serial # Serial number for the bearing pack on the drilling motor
Bearing Build Date Date that the bearing pack was assembled
Bearing Pack Actual Measurements Lower Static Sleeve Diameter Diameter of the lower static radial sleeve, obtained from the latest actual motor bearing measurements
Lower Dynamic Sleeve Diameter Diameter of the lower dynamic radial sleeve, obtained from the latest actual motor bearing measurements
Bearing Mandrel Adapter Diameter Diameter of the bearing mandrel adapter, obtained from the latest actual motor bearing measurements
Upper Static Sleeve Diameter Diameter of the upper static radial sleeve, obtained from the latest actual motor bearing measurements
Lower Radial Bearing Clearance Amount of clearance between the lower static sleeve and the lower dynamic sleeve. This value is calculated automatically and cannot be edited.
Upper Radial Bearing Clearance Amount of clearance between the bearing mandrel adapter and the upper static sleeve. This value is calculated automatically and cannot be edited.

Results

The table below defines the calculation results on the Planning > HyperLine™ Toolbox > Motor Split Flow page.

Calculation results for the motor split flow tool
Section Field Name Definition
Motor Split Flow Results Flow Through Bearing Estimated flow rate through the motor bearing assembly while drilling
Motor Split Flow Estimated percentage of split flow through the drilling motor, based on actual measurements for the bearing pack
Magnus Pad Split Flow Estimated percentage of split flow through the Magnus RSS pads, based on actual measurements for the bearing pack
Total Split Flow Estimated total split flow through the drilling motor and the Magnus RSS pads
Equivalent Motor Nozzle Results Nozzle Size Equivalent nozzle size for the bearing pack bypass, which can be entered in the drilling motor properties. See "Drilling motors".
TFA (Motor) Total flow area through the nozzles on the drilling motor
Bit ΔP Differential pressure losses across the bit
Magnus Pad ΔP Differential pressure losses across the Magnus RSS pads

The toolbar at the top of the Motor Split Flow page is used to calculate the split flow results and generate reports.

Button Function

Generates a report of the split flow calculation results. Only available after the analysis completes.
