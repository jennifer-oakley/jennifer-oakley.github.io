About the dogleg severity predictor tool

The dogleg severity predictor tool on the Planning > HyperLine™ Toolbox > DLS Predictor page is used to calculate the expected dogleg severity for the drilling motor configuration. You can also use it to compare the directional tendencies of the motor with different add-ons, power section models and lengths, hole overgauge effects, and other variables.

To use the dogleg severity predictor tool:

  1. Go to the Planning > HyperLine™ Toolbox > DLS Predictor page.
  2. Select the motor size, bearing section model, and power section model.
  3. Select the components you want to include in the motor configuration, then update the remaining properties as needed. See "Inputs".
  4. Click the Calculate button.

The expected dogleg severity will be calculated for the drilling motor configuration. The results will appear on the right side of the screen. See "Results".

Tip

To calculate results for two motor configurations simultaneously, select the HyperLine Motor 2 option.

The table below defines the calculation inputs on the Planning > HyperLine™ Toolbox > DLS Predictor page.

Results

The results for the DLS predictor tool includes an adjustable setting chart, a hole overgauge chart, and a hole-in-gauge table.

  • The hole-in-gauge table that lists the available bend angles for the motor's adjustable housing, as well as the expected dogleg severity and radius of curvature for each bend angle. The row for the selected bend angle will have a green dot in each column.
  • The adjustable setting chart graphs each bend angle against the expected dogleg severity and the upper limit for dogleg severity.
  • The hole overgauge chart will appear below the adjustable setting chart. The hole overgauge chart shows the predicted amount of hole overgauge for a range of bend angles for the motor configuration. The black line represents the selected bend angle, and the green dot represents the intersection of the selected hole overgauge with the selected bend angle.

If you selected the HyperLine Motor 2 option, click the toggle switch at the top of the results to view the results for the second motor configuration.
