About the casing pass-thru calculator

The casing pass-thru calculator on the Planning > HyperLine™ Toolbox > Casing Pass-Thru page is used to calculate the minimum casing drift diameter for each bend angle available for the motor's adjustable housing. The minimum casing drift is the smallest casing diameter that the motor can pass through in a straight well interval, under zero stress conditions, without deforming the BHA.

The casing pass-thru calculator accounts for the dimensions of the drilling motor, bottom stabilizer, drill bit, and kick pad, as well as the position of the bottom stabilizer.

Note

Currently, the casing pass-thru calculator does not account for curvature in the well.

To use the casing pass-thru calculator:

  1. Go to the Planning > HyperLine™ Toolbox > Casing Pass-Thru page.
  2. Enter the dimensions of the casing, drilling motor, bottom stabilizer, drill bit, and kick pad. See "Inputs".
  3. Select the type of adjustable housing for the motor and the bend angle that you plan to use.
  4. Click the Calculate button.

The minimum casing drift results will be calculated for the drilling motor. The results will appear on the right side of the screen. See "Results".

Inputs

The table below defines the calculation inputs on the Planning > HyperLine™ Toolbox > Casing Pass-Thru page.

Calculation inputs for the casing pass-thru calculator
Field Name Definition
Casing ID Smallest casing ID that the motor will pass through
Motor Nominal Diameter

Nominal size of the drilling motor
Kickpad Diameter Diameter of the kick pad on the motor
L1 Distance Distance from the bit box to the center of the bottom stabilizer or thread protector
Adjustable Setting Type

Type of adjustable housing for the motor
Bottom Stabilizer Diameter

Outside diameter of the bottom stabilizer on the motor
Bottom Stabilizer Length Length of the bottom stabilizer on the motor
L2 Distance Distance from bottom stabilizer or thread protector to the mud motor bend
Adjustable Setting

Bend angle setting that you plan to use
Bit Diameter

Outsider diameter of the drill bit
Bit Length

Total length of the drill bit
Bit Sub Length

Length of the bit sub that the drilling motor is connected to

Results

The results for the casing pass-thru calculator contains a table that lists the available bend angles for the motor's adjustable housing and the minimum casing drift for each bend angle, as well as a chart that graphs the results.

The row for the selected bend angle will have either a red or green dot in each column. If the motor can pass through the casing ID entered in the settings, the dot will be green. If not, the dot will be red. The results graphed in the chart are color-coded in the same way.

Definitions for the results tables
Column Name Definition
Adjustable Setting Bend angle setting
Min Casing Drift Smallest casing diameter that the motor can pass through in a straight well interval, under zero stress conditions, without deforming the BHA

Using the Casing Pass-Thru page

The toolbar at the top of the Casing Pass-Thru page is used to calculate the casing drift results and generate reports.

Toolbar on the Casing Pass-Thru page
Button Function

Generates a report of the casing drift calculation results. Only available after the analysis completes.
