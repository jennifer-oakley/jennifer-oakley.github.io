About actual data

In the Actual Data window on the Planning > Hydraulics > Displacement page, you can enter the actual data from the field. After you have entered the actual data, you can compare the actual data to the modeled analysis results. See "Comparing modeled vs. actual data". You can enter three types of field data:

  • Time– Length of time of the displacement operation.
  • SPP (Actual) – Total frictional pressure loss in the hydraulic circuit.
  • SBP (Actual) – Surface back pressure at the choke applied to the annulus.

To open the Actual Data window, click the Actual Data button () in the top toolbar in the results summary.

Comparing modeled vs. actual data

To compare the actual data to the modeled analysis results:

  1. Go to the Planning > Hydraulics > Displacement page.
  2. Select the experiment that contains the actual data.
  3. Click the Show Graphs button () in the top right toolbar.

Markers for the selected actual data will appear in the main analysis charts for the flow in and out, as well as in the surface back pressure and stand pipe pressure charts.

Using the Actual Data window

The buttons in the Actual Data window allows you to add and delete actual data.

Buttons in the Actual Data window
