About swab and surge analysis results

Swab and surge analysis provides you with several depth profiles, such as fluid rheology, pipe velocity, bottomhole gauge, and bit gauge. Using these results can help you avoid pressure changes that impair the stability of the wellbore.

There are two types of results available for swab and surge analysis:

  • A summary of the results in a data table, which contains the calculated depth profiles. See "Swab and surge results summary".
  • Analysis charts, which graph changes in different pressures and fluid properties during tripping operations. See "Swab and surge charts".

When the analysis produces warning messages, a warning icon () will be displayed on the top left corner of the screen. Hover over the icon to view the messages.

Exporting swab and surge curve data

After you run a swab and surge analysis, you can export any of the depth-based curves. When you export a depth-based curve, you will be able to graph it on the Dashboards screen. You can also send the curve data to the WITSML server.

To export depth-based curves for a swab and surge analysis experiment:

  1. Go to the Planning > Hydraulics > Surge/Swab page.
  2. Run a swab and surge analysis. See "Swab and surge analysis settings".
  3. After the analysis completes, click the Export Current Experiment to User Data button ().
  4. In the panel that appears, enter a name for the experiment.
  5. To send the data to the WITSML server, select the Publish To Witsml option.
  6. In the Parameters field, select the curves you want to export.
  7. Click the Save Changes button ().

The curve data will appear in a new tab on the Setup > User Data page. See "User data". The curve data will also be added to the list of available curves on the Dashboards screen, under the User Data category. To graph the curve data, add it to a depth-based graph. See "Adding curves to a track". If you selected the Publish To Witsml option, a new depth log will be created on the first WITSML channel listed on the Setup > DAQ > WITSML page. See "WITSML channels".

Tip

To change the name of a curve, go to the Setup > User Data page. Go to the toolbar on the right side of the screen and click on the tab name. In that box that appears, enter a new name of the curve and press the Enter key.

Creating reports

After you run a swab and surge analysis, you can generate a report of the results. Swab and surge analysis reports can include several components, such as a cover page, job information, wellpath data, the work string configuration, fluid information, the well interval configuration, and the analysis results. Swab and surge reports can be created in either MS Excel or PDF format.

To create A swab and surge analysis report:

  1. Go to the Planning > Hydraulics > Surge/Swab page.
  2. Select the experiment that you want to create a report for.
  3. Click the Save Report button () at the top of the screen.
  4. In the Surge Swab Report Options window that opens, select the location where you want to save the report. To change the default folder location and file type, click the Browse button () in the Path field.
  5. Select the components that you want to include in the report.
  6. If you selected the BHA & Drill String option, select the template that you want to use for the work string portion of the report.
  7. Click the Save Report button () at the bottom of the window.

The report will be generated with the selected report components. When the report finishes generating, it will open automatically. If you created the report in MS Excel format, click the Enable Content button to enable all of the report features when the report opens.
