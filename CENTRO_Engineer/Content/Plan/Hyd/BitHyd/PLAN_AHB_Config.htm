Starting the analysis

To run a bit hydraulics analysis:

  1. Go to the Hyd. Bit Hydraulics Opt. page on the Planning > Hydraulics section.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. In the Operating Parameters section, enter the operating parameters for the analysis. See "Field definitions".
  4. In the Hole Cleaning Settings section, enter the material, density, and size of the cuttings.
  5. To begin the analysis, click the Calculate button () in the top right corner of the screen.

After you run A bit hydraulics analysis, the Calculate button () in the top toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

If you want to re-run the analysis with different settings, just update the experiment settings as needed and click the Calculate button. When you edit the experiment settings, the existing results will be erased and the Calculate button will reappear.

Note

Work strings that contain wireline tools or wireline cables cannot be used in a bit hydraulics analysis.

Field definitions

The table below defines the settings required for a bit hydraulics optimization analysis on the Planning > Hydraulics > Bit Hydraulics Opt. page.

Settings for a bit hydraulics optimization analysis
Section Field Definition
-- Wellpath

Wellpath used for the analysis
-- Well Interval

Well interval used for the analysis
-- Work String

Drillstring or casing string used for the analysis. Work strings that contain a tool with 0 ID cannot be used in a hydraulics analysis.
-- Fluid

Circulating fluid used for the analysis
Operating Parameters Desired Pump Pressure Desired operating pressure for the primary pump
Desired ROP Rate of penetration that you want to achieve while drilling
Min Annular Velocity Minimum flow rate allowed in the annulus
Min Bit TFA Minimum TFA allowed through the bit. This setting allows you to eliminate unrealistic nozzle sizes from being included in the analysis.
Max Number of Nozzles Maximum number of nozzles that the bit can support
TDS

Top drive speed for the analysis
Depth of Interest Depth that you want to analyze
Mud Inlet Temperature

Inlet temperature of the fluid in the suction tank
Hole Cleaning Settings Cuttings Size

Size of the cuttings

  • Custom – Custom cuttings diameter.
  • Large – Average diameter of cuttings is 0.25 in (6.35 mm).
  • Medium – Average diameter of cuttings is 0.15 in (3.81 mm).
  • Medium Large – Average diameter of cuttings is 0.20 in (5.08 mm).
  • Medium Small – Average diameter of cuttings is 0.10 in (2.54 mm).
  • Small – Average diameter of cuttings is 0.05 in (1.27 mm).
Cuttings Diameter

Average diameter of the cuttings. Can only be edited when the Custom option is selected for Cuttings Size.
Cuttings Material

Composition of the cuttings from the formation
Cuttings Density

Density of the cuttings from the formation
