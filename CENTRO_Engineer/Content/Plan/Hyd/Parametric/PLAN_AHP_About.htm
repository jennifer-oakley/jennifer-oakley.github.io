About parametric analysis

On the Planning > Hydraulics > Parametric page, you can run a parametric hydraulics snapshot analysis. Parametric analysis examines the impact of flow rate, top drive speed, ROP, and standpipe pressure on ECD equivalent circulating density and hole cleaning. The analysis uses multiple values for each of these key operating parameters. It calculates several profiles to predict the fluid's ability to clean the wellbore and maintain pressure.

Flow rate, top drive speed, and ROP are important operating parameters for many reasons. The flow rate must be high enough to maintain the minimum annular velocity required to remove cuttings from the well. Increasing the flow rate also increases the pressure required to circulate the fluid, so the flow rate must also remain low enough to avoid formation fracturing. ROP influences the amount of cuttings generated. Drillers want to maximize ROP, while still maintaining the ability to clean the hole. Top drive speed influences pipe rotation, which can help alleviate hole cleaning problems by lifting the settled cuttings back into the stream of fluid flow. However, increasing pipe rotation can also increase the pressure in the wellbore.

Note

Parametric analysis assumes that there is no eccentricity in the drillstring. To account for drillstring eccentricity, you'll need to run a snapshot analysis with the Include Eccentricity option selected. See "Snapshot analysis settings".

Using the Parametric page

The buttons at the top of the screen allow you to begin an analysis and view the result charts. When you are viewing analysis charts, additional buttons will appear in the top toolbar.

Toolbar on the Parametric page
Button Function

Opens a detailed summary of the analysis results. See "Parametric results summary".

Opens the analysis charts for the current experiment. Only available after the analysis completes.

The tabs on the right side of the screen are used to switch between parametric snapshot analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove parametric snapshot analysis experiments.

Experiment toolbar on the Parametric page
Button Function
Add

Creates a copy of the current parametric snapshot analysis experiment

Deletes the current experiment, including all analysis results
