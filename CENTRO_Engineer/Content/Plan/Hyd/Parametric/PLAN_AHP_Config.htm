About parametric analysis settings

When you configure the settings for a parametric analysis, you can include up to five variable operating parameters. Each variable operating parameter will be calculated for six incremented values, based on the minimum and maximum values entered in the analysis settings. The remaining operating parameters will be calculated using constant values. You can also include up to five drillstrings and fluids.

There are also two models available for predicting hole cleaning parameters:

  • The Basic hole cleaning model uses probability distribution functions to estimate downhole cleaning conditions under varying operating parameters. The primary outputs of the basic hole cleaning model are cuttings concentration, bed height, and the hole cleaning index.
  • The Advanced hole cleaning model captures fluid flow dynamics to predict the critical flow rate required for hole cleaning.
Note

Work strings that contain wireline tools or wireline cables cannot be used in a Parametric snapshot analysis.

Starting the analysis

To run a parametric analysis:

  1. Go to the Planning > Hydraulics > Parametric page.
  2. Select the wellpath, well interval, work strings, fluids, and rig operation you want to use for the analysis.
  3. In the Operating Parameters section, select the operating parameters that you want to use as variables.
  4. To include multiple well depths in the analysis, select the Work String Depth option. See "Adding well intervals to the analysis".
  5. For each variable operating parameter selected, enter the minimum and maximum values.
  6. For the remaining operating parameters, enter a constant value.
  7. Go to the Hole Cleaning Settings section and the Advanced section to update the cuttings information and other advanced settings. See "Field definitions".
  8. To begin the analysis, click the Calculate button in the lower right corner of the screen.

After you run AParametric snapshot analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

Adding well intervals to the analysis

To add a well interval to the analysis:

  1. In the Operating Parameters section, enter the minimum well interval depth in the Depth of Interest field.
  2. Select the Work String Depth option.
  3. Click the Add button () in the heading for the Work String Depth section.
  4. In the row that appears, enter the starting depth of the interval in the From field.
  5. Enter the ending depth of the interval in the To field.
  6. In the Fluid field, select the fluid you want to analyze for the interval.

When you run the analysis, the analysis will calculate results using the fluid specified for each depth range defined in the Work String Depth section.

Note

The minimum depth of the interval must be greater than or equal to the depth in the Depth of Interest field. The maximum depth must be less than or equal to the depth of the last open hole section on the Setup > Wells > Well Intervals page.

Field definitions

The table below defines the settings required for a parametric analysis on the Planning > Hydraulics > Parametric page.

Settings for a parametric analysis
Section Field Definition
-- Wellpath

Wellpath used for the analysis
-- Well Interval

Well interval used for the analysis
-- Work String Drillstring or casing string used for the analysis. Work strings that contain a tool with 0 ID cannot be used in a hydraulics analysis.. You can select up to five drillstrings to use in the analysis.
-- Fluid

Circulating fluid used for the analysis. You can select up to five fluids to use in the analysis.
Operating Parameters Operation

Rig operation used for the analysis

  • Drilling – The bit is on-bottom, and well depth is increasing. Maximum well depth is assumed to be the same as the value entered in the Work String Depth field.
  • Circulation –Fluid is pumped down the drillstring and up the annulus. The bit is assumed to be at the value entered in the Work String Depth field. Well depth is assumed to be the same as the max depth of the open hole section on the Setup > Wells > Well Intervals page.
  • Reverse Circulate – Fluid is pumped down the annulus and up the drillstring. The bit is assumed to be at the value entered in the Work String Depth field. Well depth is assumed to be the same as the max depth of the open hole section on the Setup > Wells > Well Intervals page.
Depth of Interest Additional depth that you want to include in the analysis
Work String Depth Depth at the end of the work string. You can use bit depth as a variable for the analysis, or you can enter a constant value.
Flow Rate Flow rate for the analysis. You can use flow rate as a variable for the analysis, or you can enter a constant value.
ROP Rate of penetration while drilling. Only available for drilling operations.. You can use ROP as a variable for the analysis, or you can enter a constant value.
TDS Top drive speed for the analysis. You can use top drive speed as a variable for the analysis, or you can enter a constant value.
SBP Surface backpressure at the choke that's applied to the annulus. You can use SBP as a variable for the analysis, or you can enter a constant value.
Differential Pressure on Bottom Differential pressure that powers the mud motor when the drillstring is on-bottom. This field is only available for drilling operations, when the selected drillstring contains a mud motor.. You can use differential pressure as a variable for the analysis, or you can enter a constant value.
Hole Cleaning Settings Hole Cleaning Model

Hole cleaning model that you want to use for the analysis. See "Hole cleaning".

  • The Basic hole cleaning model uses probability distribution functions to estimate downhole cleaning conditions under varying operating parameters. The primary outputs of the basic hole cleaning model are cuttings concentration, bed height, and the hole cleaning index.
  • The Advanced hole cleaning model captures fluid flow dynamics to predict the critical flow rate required for hole cleaning.
  • The Advanced With Cuttings hole cleaning model uses the interaction of cuttings in the wellbore, as well as fluid flow dynamics, to predict the critical flow rate required for hole cleaning.
Cuttings Material

Composition of the cuttings from the formation
Cuttings Density

Density of the cuttings from the formation
Cuttings Size

Size of the cuttings

  • Custom – Custom cuttings diameter.
  • Large – Average diameter of cuttings is 0.25 in (6.35 mm).
  • Medium – Average diameter of cuttings is 0.15 in (3.81 mm).
  • Medium Large – Average diameter of cuttings is 0.20 in (5.08 mm).
  • Medium Small – Average diameter of cuttings is 0.10 in (2.54 mm).
  • Small – Average diameter of cuttings is 0.05 in (1.27 mm).
Cuttings Diameter

Average diameter of the cuttings. Can only be edited when the Custom option is selected for Cuttings Size.
Advanced Settings Boost Flow Rate

Flow rate through the boost line. Only available for offshore rigs with a subsea BOP.
Mud Inlet Temperature

Inlet temperature of the fluid in the suction tank
Include Tool Joint Effect

Accounts for geometry changes at the tool joints. If the Include Tool Joint Effect option is not selected, CENTRO® Engineer will consider the work string to be slick with no tool joints.
