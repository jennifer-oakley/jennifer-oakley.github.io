About inflatables analysis settings

When you configure the settings for an inflatables analysis, you can also simulate the effects of changes in wellbore conditions. When you add changes to the analysis, the analysis will calculate the differential ratings, performance envelope, and various pressures for the initial conditions in the wellbore. Then, the analysis will run additional calculations for differential ratings and pressure that account for the changed wellbore conditions. See "Adding changes".

The calculations for bottomhole pressure assume that the well is static and under control. If the well is static and under control, then there is no flow in the well, and hydrostatic pressure is higher than the formation pressure.

When the Live Well or Lost Circulation option is selected in the analysis setting, this assumption is removed. Instead of calculating the bottomhole pressure, the analysis uses the value entered in the analysis. In a live well scenario, the well is not under control, and the hydrostatic pressure of the fluid inside the well is less than the formation pressure. This results in fluid flow inside the well or shut-in pressure. In a lost circulation scenario, the hydrostatic pressure of the fluid inside the well is significantly higher than the formation pressure. This results in fluid being lost into the formation and an inability to maintain the fluid level at the surface.

Starting an inflatables analysis

To run an inflatables analysis:

  1. Go to the Planning > Hydraulics > Inflatable page.
  2. Select the wellpath, well interval and work string you want to use for the analysis. The selected work string must contain an inflatable packer.
  3. In the Base Fluid field, select the fluid in the wellbore before the analysis begins.
  4. In the Inflation Pressure field, enter the pressure required to inflate the packer.
  5. Go to the Initial Conditions section to specify the conditions in the wellbore before the analysis begins. See "Field definitions".
  6. Click the Add Change button () to add changes to the analysis. See "Adding changes".
  7. Add additional changes as needed. The analysis will calculate each operation in the order that it is listed.
  8. To begin the analysis, click the Calculate button () in the top right corner of the screen.

When the analysis completes, the results will appear on the right side of the Inflatable page. See "Inflatables analysis results".

If you want to re-run the analysis with different settings, just update the experiment settings as needed and click the Calculate button. When you edit the experiment settings, the existing results will be erased and the Calculate button will reappear.

Adding changes

To add a change in wellbore conditions to the analysis:

  1. Click the Add Change button () to add a change in wellbore conditions to the analysis.
  2. In the Temperature at Tool field, enter the temperature of the fluid at the inflatable packer.
  3. In the Annulus (Above) section, enter the conditions in the annulus above the inflatable packer.
  4. In the Tubing (Below) section, enter the conditions in the tubing below the inflatable packer.

During the analysis, the changes will be calculated in the order that they are listed.
