About inflatables analysis results

When you run an inflatables analysis, the results will be displayed on the right side of the Planning > Hydraulics > Inflatable page. The results for an inflatables analysis includes the differential ratings, performance envelope, and various pressures for the initial conditions in the wellbore. If you included any changes in the analysis, these values will also be calculated for each change.

By default, the results for the initial conditions will be shown. To view the results for the changes included in the analysis, click on the Inflate Input Type field at the top of the screen and select the change you want to view.

Creating reports

After you run a inflatables analysis, you can generate a report of the results. Inflatables analysis reports can include several components, such as a cover page, job information, wellpath data, the work string configuration, fluid information, the well interval configuration, and the analysis results. Inflatables reports can be created in either MS Excel or PDF format.

To create An inflatables analysis report:

  1. Go to the Planning > Hydraulics > Inflatable page.
  2. Select the experiment that you want to create a report for.
  3. Click the Save Report button () at the top of the screen.
  4. In the Hydraulic Inflates Report Options window that opens, select the location where you want to save the report. To change the default folder location and file type, click the Browse button () in the Path field.
  5. Select the components that you want to include in the report.
  6. If you selected the BHA & Drill String option, select the template that you want to use for the work string portion of the report.
  7. Click the Save Report button () at the bottom of the window.

The report will be generated with the selected report components. When the report finishes generating, it will open automatically. If you created the report in MS Excel format, click the Enable Content button to enable all of the report features when the report opens.

Pressure table

The table below defines the values shown on the Pressure Table tab.

Definitions for the Pressure Table tab
Section Value Definition
Inside Element Δp Across Top Wall Pressure change across the top wall of the inflatable packer
Δp Across Bottom Wall Pressure change across the bottom wall of the inflatable packer
Change in Pressure Annulus (Above) Pressure change in the annulus, above the inflatable packer
Inflation Pressure Change in the inflation pressure
Tubing (Below) Pressure change in the tubing, below the inflatable packer
Absolute Pressure Annulus (Above) Absolute pressure in the annulus, above the inflatable packer
Inflation Pressure Absolute inflation pressure
Tubing (Below) Absolute pressure in the tubing, below the inflatable packer
-- Differential Rating of Element Maximum allowable differential pressure across the body of the inflatable packer, given the inflation pressure and the expansion of the packing element inside the host casing. This value is based on the differential ratings in the tool properties. See "Well services tools".
-- Surface Pump pressure required to set plug Pressure from the surface pump required to set the pump-out plug. Only available for the initial condition.
-- Coiled Tubing Hydrostatic Pressure Hydrostatic pressure inside the coiled tubing at the inflatable packer depth
-- Differential held from side of anchor Difference between the tubing pressure on the anchor side of the packer and the tubing pressure on the opposite side of the packer. If this value exceeds the maximum allowable rating, then the packer can lose grip with the casing and move.
-- Coiled Tubing Pressure is in Balance/Underbalanced Indicates the state of the hydrostatic pressure in the coiled tubing. If the value is negative, then the hydrostatic pressure inside the coiled tubing is less than the formation pressure. If the value is zero, then the hydrostatic pressure is balanced with the formation pressure. Initial condition only

Performance envelope

The chart on the Performance Envelope tab shows the change in pressure across the bottom wall of the packer compared to the change in pressure across the top wall of the packer. This can be used to determine if any of the load conditions defined in the initial conditions or subsequent changes fall outside the manufacturer's specifications for the packer's operating envelope, which causes the risk of possible failure.

The loads are defined by combining the differential pressures across the top and bottom walls of the inflatable into a single point on the plot. The manufacturer's operating envelope is the area inside the closed blue line. If the load falls inside the operating envelope, then they are within the manufacturer's specified maximum operating loads and are considered safe for the integrity of the tool.
