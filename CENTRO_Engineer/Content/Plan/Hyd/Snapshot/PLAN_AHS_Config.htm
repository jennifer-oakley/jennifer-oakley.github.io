About snapshot analysis settings

When you run a snapshot analysis, there are two models available for hole cleaning:

  • The Basic hole cleaning model uses probability distribution functions to estimate downhole cleaning conditions under varying operating parameters. The primary outputs of the basic hole cleaning model are cuttings concentration, bed height, and the hole cleaning index.
  • The Advanced hole cleaning model captures fluid flow dynamics to predict the critical flow rate required for hole cleaning.
Note

Work strings that contain wireline tools or wireline cables cannot be used in a snapshot analysis.

Starting the analysis

To run a snapshot analysis:

  1. Go to the Snapshot page on the Planning > Hydraulics section.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. In the Operating Parameters section, enter the operating parameters for the analysis.
  4. If the work string contains tools with nozzles, go to the Split Flow Components section to configure the flow settings for the tool.
  5. In the Hole Cleaning Settings section, enter the material, density, and size of the cuttings.
  6. In the Temperature Calibration section, enter the mud inlet temperature, the mud outlet temperature, and the bottomhole temperature.
  7. In the Advanced Settings section, select any additional options for the analysis.
  8. To begin the analysis, click the Calculate button.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Snapshot results summary".

After you run Asnapshot analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

When the analysis produces warning messages, a warning icon () will be displayed on the top left corner of the screen. Hover over the icon to view the messages.

Field definitions

The tables in this section define the settings for a snapshot analysis. There are six groups of settings:

  • The general settings in the Parameters section define the basic parameters for the analysis, including the wellpath, well interval, work string and fluid. See "General parameters".
  • The settings in the Operating Parameters section define the operation-specific settings, such as work string depth, flow rates, ROP, TDS, and SBP. See "Operating parameters".
  • The settings in Split Flow Components section define the flow path and discharge for tools with split flow, such as drill bits, hole openers, and circulation subs. See "Split flow components".
  • The settings in the Hole Cleaning Settings section define the hole cleaning model and the composition, size, and density of the cuttings. See "Hole cleaning settings".
  • The settings in the Temperature Calibration section define the inlet and outlet temperature of the mud, as well as the bottomhole temperature. See "Temperature calibration".
  • The settings in the Advanced Settings section include options for including eccentricity and the effects of the tool joints, as additional parameters for work strings that contain a thru-tubing motor or coiled tubing. See "Advanced settings".

Operating parameters

The table below defines the operating parameters for a snapshot analysis.

Operating parameters for a snapshot analysis
Field Definition
Operation

Rig operation used for the analysis

  • Drilling – The bit is on-bottom, and well depth is increasing. Maximum well depth is assumed to be the same as the value entered in the Work String Depth field.
  • Circulation –Fluid is pumped down the drillstring and up the annulus. The bit is assumed to be at the value entered in the Work String Depth field. Well depth is assumed to be the same as the max depth of the open hole section on the Setup > Wells > Well Intervals page.
  • Reverse Circulate – Fluid is pumped down the annulus and up the drillstring. The bit is assumed to be at the value entered in the Work String Depth field. Well depth is assumed to be the same as the max depth of the open hole section on the Setup > Wells > Well Intervals page.
  • Injection – Fluid is injected into the gas cap or production zone to maintain reservoir pressure. Only available for packer strings.
  • Production – Hydrocarbons are removed from the well. Only available for packer strings.
  • Pressure-Up Pipe – Pressure test to test the integrity of the completion string. The completion string has been run to the final depth, and the packers have been set. Pressure is applied from the surface to the inside of the tubing against the tubing plug. This test requires a tubing plug set at any depth inside the tubing. Only available for packer strings.
  • Pressure-Up Annulus – Pressure test to test the integrity of the packer, casing, and wellhead equipment. The completion string has been run to the final depth, and the packers have been set. Pressure is applied from the surface down to the annular space between the tubing and the casing, against the uppermost packer. Only available for packer strings.
  • Injection Annulus – Injection fluid is pumped down the annular space between the production casing and the production tubing, then into the formation through casing perforations or open hole intervals. The completion string has been run to the final depth, and the packers have been set and tested.
  • Production Annulus – Production fluids are removed from the formation through casing perforations or open hole intervals, then up the annular space between the production string and the production casing to the surface. The completion string has been run to the final depth, and the packers have been set and tested.
Fluid Return Path

Path the fluid takes out of the wellbore. Only available for offshore rigs with a subsea BOP.

  • Conventional – Fluid is pumped down the drillstring and up the annulus.
  • Choke Kill Lines – Fluid is returned through choke or kill lines.
  • Sea Floor – Fluid is returned to the sea floor.
Work String Depth

Depth at the end of the work string
Point of Interest Additional point at which you want to calculate values for the analysis. When a work string component is selected, the value in the Depth of Interest field will be automatically calculated, based on the position of the component in the work string. When the Other option is selected, the Depth of Interest value can be manually entered.
Depth of Interest

Additional depth for calculating values. When a work string component is selected in the Point of Interest field, this value is automatically calculated. When the Other option is selected in the Point of Interest field, this value can be manually entered.
Boost Flow Rate

Flow rate through the boost line. Only available for offshore rigs with a subsea BOP.
Flow Rate Flow rate for the analysis. Only available when the selected fluid does not have a gas phase.
Liquid Flow Rate Flow rate for the liquid phase of the fluid. Only available when the selected fluid is a liquid and gas mixture.
Gas Flow Rate Flow rate for the gas phase of the fluid. Only available when the selected fluid is a liquid and gas mixture.
ROP

Rate of penetration while drilling. Only available for drilling operations.
TDS

Top drive speed for the analysis
SBP

Surface backpressure at the choke that's applied to the annulus. Only available for circulation, drilling, and reverse circulation operations.
Differential Pressure on Bottom

Differential pressure that powers the mud motor when the drillstring is on-bottom. This field is only available for drilling operations, when the selected drillstring contains a mud motor.
Pipe Surface Pressure Surface pressure in the pipe. Only available for injection and pressure-up operations.
Annulus Surface Pressure Surface pressure in the annulus. Only available for injection, production, pressure-up pipe, and pressure-up annulus operations.
Plug Depth Depth of the pump out plug. Only available for pressure-up pipe, pressure-up annulus, injection annulus, and production annulus operations.
WS End State of the end of the work string. Only available for injection and production operations.

Hole cleaning settings

The table below the hole cleaning settings for a snapshot analysis.

Hole cleaning settings for a snapshot analysis
Field Definition
Hole Cleaning Model

Hole cleaning model that you want to use for the analysis. See "Hole cleaning".

  • The Basic hole cleaning model uses probability distribution functions to estimate downhole cleaning conditions under varying operating parameters. The primary outputs of the basic hole cleaning model are cuttings concentration, bed height, and the hole cleaning index.
  • The Advanced hole cleaning model captures fluid flow dynamics to predict the critical flow rate required for hole cleaning.
Cuttings Material

Composition of the cuttings from the formation
Cuttings Density

Density of the cuttings from the formation
Cuttings Size

Size of the cuttings

  • Custom – Custom cuttings diameter.
  • Large – Average diameter of cuttings is 0.25 in (6.35 mm).
  • Medium – Average diameter of cuttings is 0.15 in (3.81 mm).
  • Medium Large – Average diameter of cuttings is 0.20 in (5.08 mm).
  • Medium Small – Average diameter of cuttings is 0.10 in (2.54 mm).
  • Small – Average diameter of cuttings is 0.05 in (1.27 mm).
Cuttings Diameter

Average diameter of the cuttings. Can only be edited when the Custom option is selected for Cuttings Size.
Cuttings Effect on ECD/Pressure Includes the effect of the cuttings on pressure and ECD calculations. Only available when the advanced hole cleaning model is selected.

Temperature calibration

The table below defines the temperature calibration settings for a snapshot analysis.

Temperature calibration settings for a snapshot analysis
Field Definition
Mud Inlet Temperature

Inlet temperature of the fluid in the suction tank
Mud Outlet Temperature Temperature of the fluid when it is returned to the mud tanks
Bottom Hole (PWD) Temperature Bottomhole temperature of the fluid. This value is used to more accurately calibrate the fluid temperature profile. To use this feature, the bit depth must match the well depth for the selected well interval.

Advanced settings

The table below defines the advanced settings available for a snapshot analysis.

Advanced settings for a snapshot analysis
Field Definition
Include Tool Joint Effect

Accounts for geometry changes at the tool joints. If the Include Tool Joint Effect option is not selected, CENTRO® Engineer will consider the work string to be slick with no tool joints.
Include Eccentricity Assumes the work string is in contact with one side of the wellbore, and that the fluid will flow through an eccentric annulus. If the Include Eccentricity option is not selected, CENTRO® Engineer will assume the work string is in the center of the wellbore.
Output Torque Amount of torque produced by the power section in the thru-tubing motor. Only available for work strings that contain a thru-tubing motor.
Coiled Tubing Loss Calibration Percentage of initial pressure loss in the coiled tubing. Only available for work strings that contain coiled tubing
