Starting the simulation

To start an influx simulation:

  1. Go to the Planning > Hydraulics > Influx page.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the simulation.
  3. In the Influx Zone field, select the zone for which you are simulating an influx.
  4. In the Simulation Type field, select the calculation method you want to use.
  5. Enter the remaining operating parameters for surface backpressure, top drive speed, flow rate, pressure limit, safety factor, depth of interest, and pit gain.
  6. To begin the simulation, click the Calculate button.

When the simulation completes, a summary of the results will appear on the right side of the screen. See "Influx simulation results".

After you run Aninflux analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

When the analysis produces warning messages, a warning icon () will be displayed on the top left corner of the screen. Hover over the icon to view the messages.

Note

Work strings that contain wireline tools or wireline cables cannot be used in a influx analysis.

Field definitions

The table below defines the settings required for an influx analysis on the Planning > Hydraulics > Influx page.

Settings for an influx analysis
Section Field Definition
-- Wellpath

Wellpath used for the analysis
-- Well Interval

Well interval used for the analysis
-- Work String

Drillstring or casing string used for the analysis. Work strings that contain a tool with 0 ID cannot be used in a hydraulics analysis.
-- Fluid Drilling fluid used for the simulation
Operating Parameters Influx Zone Section of the wellbore where the influx occurs
Influx Model

Method for simulating gas influx circulating out of the well.
Simulation Type

Method for calculating pore pressure from density units to pressure units.

  • Instantaneous – Uses the density of the formation fluids at the depth of interest.
  • Pressure Driven – Uses the calculated pore pressure gradient at the depth of interest.
SBP

Surface backpressure at the choke that's applied to the annulus
TDS Expected top drive speed
Pump Flow Rate

Expected flow rate through the pumps
Pressure High Limit

Maximum anticipated surface backpressure
Safety Factor Minimum safety margin for pore pressure
Depth of Interest Assumed depth of the influx
Initial Pit Gain Volume of the initial pit gain when the influx is detected. Only available for instantaneous simulations.
Detection Time Expected amount of time required to detect the influx. Only available for pressure-driven simulations.
Control Time Minimum amount of time required to control the influx. Only available for pressure-driven simulations.
