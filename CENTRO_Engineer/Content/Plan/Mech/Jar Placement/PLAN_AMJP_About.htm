About jar placement analysis

Jar placement analysis uses stress wave analysis that allows you to optimize the location of a jar within a work string. Jars are used to free objects in the borehole by delivering a heavy hammer blow to another downhole component. The energy stored in the drillstring is released when the jar fires. The amount of force delivered by this impact is determined by the jar tripping forces, the length of hammer section, and the length of the anvil section.

The anvil section is the section directly below the jar, and the hammer section is the section directly above the jar. They are typically composed of HWDP, drill collars, or weight bars. Jar placement analysis starts with an initial hammer length, then increases the hammer length incrementally. The initial length of the anvil is determined by the length of the section immediately below the jar. During the analysis, the anvil length can either decrease or remain the same, depending on the analysis settings. See "Jar placement settings".

Jar placement also includes go-no-go analysis. Go-no-go analysis calculates upper and lower boundary ranges for WOB, which allows you to ensure that the jar will not be in the vicinity of the neutral point. See "Go-no-go analysis".

Note

Currently, jar placement analysis is not supported for work strings with multiple jars.

Using the Jar Placement (Beta) page

The toolbar at the top of the screen allows you to begin an analysis, view analysis charts, and generate a report for the results. This toolbar appears after the analysis completes.

Toolbar on the Jar Placement (Beta) page
Button Function

Generates a report of the jar placement calculation results. Only available after the analysis completes.. See "Creating reports".

Opens the analysis charts for the current experiment. Only available after the analysis completes.

The tabs on the right side of the screen are used to switch between jar placement analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove jar placement analysis experiments.
