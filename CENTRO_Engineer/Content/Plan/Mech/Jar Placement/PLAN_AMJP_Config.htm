About jar placement settings

When you run a jar placement analysis, there are two analysis methods: a fixed anvil analysis and a decreasing anvil analysis.

  • A fixed anvil length analysis is used to determine the optimum hammer length, given a constant anvil length. For this type of analysis, the work string should contain the desired amount of heavy weight drill pipe or drill collars below the jar, as well as the initial hammer length of HWDP or drill collars above the jar. During the analysis, one increment of hammer length will be added to the work string during each iteration.
  • A decreasing anvil length analysis is used to determine the optimum placement of the jar, given a constant hammer length of HWDP or drill collars. During the analysis, the jar will be moved one increment at a time, but the total number of HWDP or drill collars will remain the same.

To run a fix anvil length analysis, select the Fix Anvil Length option in the analysis settings. If this option is not selected, then the analysis will automatically use the decreasing anvil length method.

Starting the analysis

To run a jar placement analysis:

  1. Go to the Planning > Mechanics > Jar Placement (Beta) page.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. Enter the stuck point or other depth you want to analyze in the Work String Depth field.
  4. In the Preferred Jarring Direction field, select the direction that you will be jarring.
  5. Enter the amount of force that will be applied to the jar.
  6. In the Stuck Force field, enter amount of force required to move the stuck section.
  7. Enter the remaining operating parameters you want to use. See "Field definitions".
  8. Click the Calculate button to start the analysis.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Jar placement results summary".

After you run a jar placement analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch from the results summary to the charts view. See "Jar placement analysis charts".

Field definitions

The table below defines the settings required for a jar placement analysis.

Settings for jar placement analysis
Section Field Definition
-- Wellpath

Wellpath used for the analysis
-- Well Interval

Well interval used for the analysis
-- Work String

Drillstring or casing string used for the analysis
-- Fluid

Circulating fluid used for the analysis
-- Work String Depth

Measured depth used for the analysis. For drilling scenarios, the work string depth is typically the end of the deepest hole section. For fishing or thru-tubing jobs, the work string is the depth at which the work string is stuck.
Operating Parameters Preferred Jarring Direction

Direction that the jarring forces will travel along the work string.

  • Jar Up – Jarring forces will travel upwards along the work string, from the jar to the top of the work string.
  • Jar Down – Jarring forces will travel downwards along the work st ring, from the jar to the end of the work string. Not available for hydraulic fishing jars.
Up Force Amount of force to be applied to the jar for tripping, when jarring up. The default value is the maximum jar load from the jar specifications. Only available when the Jar Up option is selected for the jarring direction.
Down Force Amount of force to be applied to the jar for tripping, when jarring down. The default value is the maximum jar load from the jar specifications. Only available when the Jar Down option is selected for the jarring direction.
Stuck Force Amount of force required to move the stuck section. The stuck force must be greater than the jar tripping force.
Initial Hammer Length Amount of hammer length at the beginning of the analysis
Hammer Length Increment Amount of hammer length to add at each increment during the analysis
Weight On Bit Amount of weight on bit
Number of Increment Number of increments to use for the analysis
Fix Anvil Length

Option for the analysis assumptions about the length of the anvil.

  • When this option is selected, the analysis assumes that the anvil length is constant.
  • If this option is not selected, the analysis assumes that the anvil length decreases in increments.
Stuck Below Nearest work string component above the stuck point
Degrees Per Split Wellpath angle used to split the drillstring into segments. When the change in the wellpath inclination exceeds this value, the work string will be split into a new segment.
Output Preference

Display option for the Hammer Length table in the results summary. See "Hammer length table".

  • When the Greatest Impulse option is selected, the point where the highest impulse occurs will be automatically selected in the Hammer Length table.

  • When the Greatest Impact option is selected, the point where the highest impact occurs will be automatically selected in the Hammer Length table.
