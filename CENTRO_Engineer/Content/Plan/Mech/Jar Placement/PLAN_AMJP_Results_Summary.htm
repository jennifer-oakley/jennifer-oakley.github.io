About the results summary

The results summary for a jar placement analysis includes four tables:

  • A hammer length table, which shows the maximum forces and available impulse for each increment of hammer length. See "Hammer length table".
  • An overpull table, which shows the overpull, impact, and impulse calculations for the selected hammer length. See "Overpull table".
  • A neutral point table, which shows the tool where the neutral point occurs and the amount of tension or compression at the jar. See "Neutral point table".
  • A work string weight table, which shows the length and weight of the work string tools above and below the jar. See "Work string weight table".

Hammer length table

The table below defines the values shown in the Hammer Length table. The point where the highest impulse or the highest impact occurs will be automatically selected by default, depending on the display option selected in the Output Preference field. See "Field definitions".

Definitions for the Hammer Length table
Column Name Definition
Hammer Length Length of the section directly above the jar
Anvil Length Length of the section direction below the jar
Max Force at Jar Maximum impact force generated at the jar by the acceleration of the jar's internal hammer hitting its internal anvil
Max Force at Stuck Point Maximum impact force transferred to the stuck point to free the fish
Available Impulse Maximum possible impulse

Overpull table

By default, the overpull results are only calculated for the first hammer length listed. To calculate the overpull results for another hammer length, select the desired hammer length, then click the Calculate.

The table below defines the values shown in the Overpull table, shown below the Hammer Length table.

Definitions for the Overpull table
Column Name Definition
Overpull Amount of force applied to the jar for tripping
Impact Amount of impact force generated by the jar tripping force
Impulse Change in momentum produced by the jarring force

Neutral point table

The table below defines the values shown in the Neutral Point table, shown below the Overpull table. There are two tools shown in the table: the jar and the tool where the neutral point occurs.

Definitions for the Neutral Point table
Column Name Definition
Neutral Point Position in the work string of the neutral point, relative to the bit
Tension/Compression Amount of tension or compression at the jar. A positive value indicates the jar is in tension, and a negative value indicates the jar is in compression.

Work string weight table

There are three sets of values shown in the Work String Weight table:

  • Total Length and Weight Above Jar – Results for the tools in the work string that are above the jar.
  • Total Length and Weight Below Jar – Results for the tools in the work string that are below the jar.
  • Total Drill String Length and Weight – Results for the entire work string.

The table below defines the values shown in the Work String Weight table, shown below the Neutral Point table.

Definitions for the Work String Weight table
Column Name Definition
Length Length of the work string tools
Weight In Air Weight of the work string tools when suspended in air
Weight In Mud Weight of the work string tools when suspended in mud
