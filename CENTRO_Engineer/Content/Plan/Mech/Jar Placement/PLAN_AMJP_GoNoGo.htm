About go-no-go analysis

The neutral point is the point in the drillstring at which tension and compression are neutral. The drillstring components below the neutral point are in compression, and the components above it are in tension. As the WOB increases, the neutral point will move upwards along the drillstring. You can use the go-no-go analysis on the Jar Placement (Beta) tab to find the upper and lower boundary ranges for WOB, which allows you to ensure that the jar will not be in the vicinity of the neutral point.

There are two methods available for calculating the neutral point:

  • Tension/Compression – The neutral point is calculated based on the amount of tension or compression below the jar, where the amount of tension or compression is expressed as a percentage of WOB. You can use the following formula to determine the percentage: (weight below jar - WOB) x 100 / WOB.
  • Safety Distance – The neutral point is calculated based on the distance of the jar from the bit.

Go-no-go analysis calculates two ranges of acceptable WOB ranges. When the WOB is within the lower WOB range, the neutral point will be below the jar. When the WOB is within the upper WOB range, the neutral point will be above the jar. Using an amount of WOB that is between the lower and upper WOB ranges is not recommended, because the neutral point will be too close to the jar.

Starting the analysis

To run a go-no-go analysis:

  1. Go to the Planning > Mechanics > Jar Placement (Beta) page.
  2. Expand the Go-No-Go Analysis section at the bottom of the page.
  3. In the Method field, select the calculation method you want to use.
  4. Enter the calculation parameters and the operating conditions. See "Inputs".
  5. Click the Calculate Go-No-Go button () in the heading for the Go-No-Go Analysis section.

Upper and lower WOB ranges will be calculated for each bit depth increment included in the analysis. The results will appear on the right side of the screen. See "Results".

Inputs

The table below defines the settings required for a go-no-go analysis.

Settings for go-no-go analysis
Field Definition
Method

Method of calculating the neutral point

  • Tension/Compression – The neutral point is calculated based on the amount of tension or compression below the jar.
  • Safety Distance – The neutral point is calculated based on the distance of the jar from the bit.
Percentage Amount of tension or compression below the jar, expressed as a percentage of WOB. Only available when the Tension/Compression option is selected for the calculation method.
Distance Distance of the jar from the bit. Only available when the Safety Distance option is selected for the calculation method.
Max WOB Max WOB from the drill bit specifications
Start MD

Starting depth to include in the analysis
End MD

Ending depth to include in the analysis
Interval Count Number of increments to include in the analysis. The increment size will be automatically calculated from the start depth, end depth, and interval count.
Tool Name of the jar, with the jar position within the work string shown in parentheses. The first set of numbers is top and bottom position of the jar, starting from the top of the work string. The second set of numbers is the top and bottom position of the jar, starting from the drill bit.
Operation Operation to use for the analysis
Flow Rate

Flow rate of the circulating fluid
TDS Top drive speed while drilling. Only available when the Drilling Rotary option is selected for the operation.
ROP

Rate of penetration while drilling

Results

The results for a go-no-go analysis includes a table and a chart showing the upper and lower WOB ranges for each bit depth increment. To view a graph of these results, click the Show Graph button ().

Results for go-no-go analysis
Column Name Definition
Bit Depth MD of the bit
Lower WOB Boundary Lower range of acceptable WOB values. When the WOB is within this range, the neutral point will be below the jar.
Upper WOB Boundary Upper range of acceptable WOB values. When the WOB is within this range, the neutral point will be above the jar.
