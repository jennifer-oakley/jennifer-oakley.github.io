About the results summary

The results summary for a tubing movement analysis includes:

  • A mechanics summary of the predicted tubing and packer performance. See "Mechanics summary".
  • A hydraulics summary that contains the results of the hydraulics portion of the analysis. See "Hydraulics summary".
  • Packer envelopes that show the force and pressure limits for each packer in the work string. See "Packer envelope summary".
  • Tubing envelopes that show the force and pressure limits for each tubing component in the work string. See "Tubing envelope summary".

Mechanics summary

The mechanics summary for tubing movement analysis includes summaries of the predicted tubing and packer performance. The tubing string is divided into sections, based on the depth of the packers in the tubing string, from the surface to the end of the work string. Packers are also listed in the same order. To view the mechanics summary, click the Show Summary button () in the toolbar at the top of the screen.

The table below defines the values shown in the Tubing Performance table.

Definitions for tubing performance
Column Name Definition
Section Start and end depth of the tubing section
Length Change Amount of stretching or shortening that occurs in the tubing section, due to buckling, ballooning, or thermal properties
Max Stress Maximum effective axial force in the tubing section
Neutral Point Depth at which the axial stress in the tubing string changes from compression to tension

The table below defines the values shown in the Packer Performance table.

Definitions for packer performance
Column Name Definition
Packer # Sequential number to identify the packer
Packer Depth Setting depth of the packer
Differential Pressure Differential pressure across the packer
Packer to Tubing Force Force that the packer exerts on the tubing due to the weights and pressure inside the work string
Packer to Casing Force Force that the packer exerts on the casing due to the weights and pressure inside the work string
Pass/Fail Indicates the pass/fail status of the packer. A green icon indicates the packer is within the limits of the packer envelope, and a red icon indicates the packer is outside the limits of the packer envelope. See "Packer envelope summary".

Hydraulics summary

The hydraulics summary includes depth profiles for pressure and ECD. To view the hydraulics summary, click the Show Hydraulics button () in the toolbar at the top of the screen.

The table below defines the values shown on the table under the Pressure tab.

Definitions for values in the Pressure table
Column Name Definition
MD Measured depth
Annular Pressure Total pressure loss in the annulus
WS Pressure Total pressure loss in the work string
Packer Indicates that a packer is present

The table below defines the values shown on the table under the ECD tab.

Definitions for values in the ECD table
Column Name Definition
MD Measured depth
Annulus Density ECD of the fluid in the annulus
Pipe Density ECD of the fluid in the work string
Packer Indicates that a packer is present

Packer envelope summary

The Packer Envelope view shows the packers included in the analysis and the packer envelope for each packer. To open the Packer Envelope view, click the Packer Envelope button () in the bottom toolbar.

The table at the top of the Packer Envelope view shows the list of packers included in the analysis. The highlight color on the left side of each row indicates the pass/fail status of the packer. See "Mechanics summary".

The chart below the table shows the packer envelope for the selected packer, as well as the predicted placement of the packer within the packer envelope. The packer envelope is defined in the tool settings for the packer on the Setup > Work String page. See "Packers". To change the selected packer, click on the drop-down menu above the chart.

Tubing envelope summary

The Tubing Envelope view shows the tubing components included in the analysis and the limits for each tubing component. To open the Tubing Envelope view, click the Tubing Envelope button () in the toolbar at the top of the screen.

The table at the top of the Tubing Envelope view shows the list of tubing components included in the analysis. The highlight color on the left side of each row indicates the pass/fail status of the tubing component. The chart below the table shows the tubing envelope for the selected tubing component, as well as the predicted placement of the tubing component within the tubing envelope. The tubing envelope is composed of three elements:

  • The tool limits of the component, which includes burst, collapse, tension, and compression limits. The tool limits are defined in the tool settings for the tubing component on the Setup > Work String page. See "Work string components".
  • The limits of the connections on the component, which includes burst, collapse, tension, and compression limits.
  • The Von Mises yield criterion, which is a maximum distortion criteria that is determined by Von Mises stress and other parameters.

To change the selected tubing component, click on the drop-down menu above the chart.
