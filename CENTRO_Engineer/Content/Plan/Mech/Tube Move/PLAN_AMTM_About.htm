About tubing movement analysis

Tubing is set in the well using packers after the casing is run in and cemented in place. Packers are used for zonal isolation and tubing sealing. During operation, the tubing string is exposed to several mechanical loads caused by the string weight, differential pressures, and temperature.

On the Planning > Mechanics > Tube Move (Beta) page, you can run a tubing movement analysis. Tubing movement analysis calculates the effects of pressure, temperature, and mechanical changes on tubing strings. When the operational mode of a well changes, the resulting temperature and pressure changes inside and outside the tubing can cause length and force changes that affect the packer and downhole tools. The extent of these changes depends on the type of packer, how the packer is set, how the tubing is connected to the packer, and the amount of tubing compression on the packer. Other factors include:

  • Size, weight, material, and length of the tubing
  • Internal diameter of the casing or liner
  • Fluid density
  • Pressure and temperature changes in the fluid

Using the Tube Move (Beta) page

The toolbar at the top of the screen allows you to begin an analysis, view analysis charts, and generate a report for the results.

Toolbar on the Tube Move (Beta) page
Button Function

Begins tubing movement calculations

Generates a report of the tubing movement calculation results. Only available after the analysis completes.. See "Creating reports".

Opens the hydraulics results calculated for the analysis. See "Hydraulics summary".
Opens the packer envelope calculated for each packer in the selected work string. See "Packer envelope summary".
Opens the tubing envelope calculated for each tubing component in the selected work string. See "Tubing envelope summary".
Opens the mechanics performance results. See "Mechanics summary".

Opens the analysis charts for the current experiment. Only available after the analysis completes.

The tabs on the right side of the screen are used to switch between tubing movement analysis experiments. Each experiment has its own tab, with a toolbar that is used to add, compare, and remove tubing movement analysis experiments.

Experiment toolbar on the Tube Move (Beta) page
Button Function
