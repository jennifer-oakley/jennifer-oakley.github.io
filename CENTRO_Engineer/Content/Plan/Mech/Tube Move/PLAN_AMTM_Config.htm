About tubing movement analysis settings

Work strings used in a tubing movement analysis can only contain the following components:

  • Tools with one part section, such as drill collars and subs
  • Casing pipe
  • Downhole control valves
  • Packers
  • POST tools
  • Stage tools
  • Tubing accessories
  • Tubing compensation equipment

Starting the analysis

To run a tubing movement analysis:

  1. Go to the Planning > Mechanics > Tube Move (Beta) page.
  2. Select the wellpath, well interval, work string, and initial fluid you want to use for the analysis.
  3. Enter the bit position or other depth you want to analyze in the Work String Depth field.
  4. In the Operation field, select the operation you want to analyze.
  5. Enter the remaining operating parameters you want to use.
  6. To begin the analysis, click the Calculate button () to start the hydraulics analysis.
  7. After the hydraulics analysis completes, click the Calculate button () again to start the mechanics analysis.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Tubing movement results summary".

After you run A tubing movement analysis, the Calculate button () in the top toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view. See "Tubing movement analysis charts".

Field definitions

The table below defines the settings required for a tubing movement analysis.

Settings for tubing movement analysis
Section Field Definition
-- Wellpath

Wellpath used for the analysis
-- Well Interval

Well interval used for the analysis
-- Work String

Tubing string used for the analysis
-- Initial Fluid

Static fluid in the wellbore when the operation begins
Operating Parameters Work String Depth

Depth at the end of the work string
Operation

Operation to use for the analysis

  • Annulus Pressure Test – Pressure is applied to the annulus to test the integrity of the packers.
  • Injection – Fluid is injected into the gas cap or production zone to maintain reservoir pressure.
  • Production – Hydrocarbons are removed from the well.
  • Tubing Pressure Test – Pressure is applied to the inside of the tubing to test the integrity of the packers.
Fluid

Fluid used for the selected operation.

  • For annular or tubing pressure tests, the fluid is the pumping fluid.
  • For injection operations, the fluid is the injection fluid being injected into the well.
  • For production operations, the fluid is the producing fluid coming out of the well formation.
Flow Rate

Flow rate of the fluid for the selected operation. Only available for injection and production operations.
Pipe Surface Pressure Surface pressure in the pipe. Not available for production operations.
Annulus Surface Pressure

Surface pressure in the annulus
Packer Depth Depth of the deepest packer. This value is read-only and cannot be edited. Only available for annular pressure testing operations.
Plug Depth Depth of the pump out plug. Only available for tubing pressure testing operations.
WS End State of the end of the work string. Only available for injection and production operations.
Additional Force Amount of additional force added at the surface.
Packer Setting Type

Type of packer.
Plug Depth Depth of the pump out plug. Only available when the Hydraulic Packer option is selected.
Setting Pressure Pressure required to set the packer. Only available when the Hydraulic Packer option is selected.
Injection / Production Ports Tool Name of the configurable tool. Only available for injection and production operations.
Port State State of the ports on the configurable. The ports must be open on at least one configurable tool. Only available for injection and production operations.
Tools Configuration Mode Tool Name of the tubing compensation equipment
Configuration Mode

Configuration mode to limit upward or downward movement.

  • Fully Contracted – The tubing compensation equipment is fully contracted, and the stroke limit will be added to the total length of the tool.
  • Fully Extended – The tubing compensation equipment is fully extended, and the stroke limit will be subtracted from the total length of the tool.
  • None – No change in the tool configuration
