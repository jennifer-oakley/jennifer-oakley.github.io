Viewing analysis charts

After you run A tubing movement analysis, the Calculate button () in the top toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

To view the charts for A tubing movement experiment:

  1. Go to the Planning > Mechanics > Tube Move (Beta) page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Graphs button () in the top toolbar.

The charts will appear on top of the analysis settings. To close the analysis charts, click the Show Graphs button () again.

Displaying chart legends

By default, the chart legends are not shown. To pin the legends to the charts:

  1. Hover your mouse over the bottom right corner of the graph index, on the right side of the analysis charts.
  2. In the toolbar that appears, click the Show/Hide Legend button ().

The legend for each chart will appear in the top right corner of the graph.

Adding charts

In addition to the charts shown by default, there are more charts available. To view these charts:

  1. Click the Add Graph Page button () in the chart toolbar.
  2. In the menu that appears, click on the chart you want to view.

The selected chart will appear last in the charts view, on the right side of the current charts.

Changing graph settings

You can change the appearance of a chart in the Graph Settings window. Each chart has its own settings, so changing the appearance of one chart will not affect how another chart looks.

To customize the default graph settings:

  1. Go to the bottom right corner of the chart you want to customize, then click the Options button (). A menu with toolbar buttons will appear on top of the chart.
  2. Click the Edit button () to open the Graph Settings window.
  3. In the menu at the top of the window, select the name of the chart series that you want to customize.
  4. Change the options for the chart series as desired.

As you make changes to the chart, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

List of tubing movement analysis charts

The charts listed below are available on the Tube Move (Beta) page. If you want to view a chart that is not shown by default, you'll need to add it to the chart view. See "Adding charts".

Charts for tubing movement analysis
Chart Name Description Chart Sample
Axial Load

Shows the distribution of effective axial force along the work string. Use this plot to identify conditions that are likely to cause buckling in the drillstring. The chart shows limits that represent threshold limits for work string buckling. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The gray area represents sinusoidal buckling limits.
  • The red area represents helical buckling limits.
  • The pink area represents the compression limits for the tools in the work string.
  • The dotted pink line represents the compression limits for the connections in the work string.
  • The orange area represents the tension threshold for the allowable axial work string load.
  • The dotted orange line represents the tension limits for the connections in the work string.
  • The black line represents the axial load under the initial conditions before the current operation begins. If a line falls within the red or gray areas, buckling limits have been exceeded.
  • The blue line represents the predicted axial load under the analysis parameters. If a line falls within the red or gray areas, buckling limits have been exceeded.
Net Axial Load

Shows the distribution of true axial force along the work string. Use this plot to identify conditions that are likely to cause buckling in the drillstring. The chart shows limits that represent threshold limits for work string buckling. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The pink area represents the compression limits for the tools in the work string.
  • The dotted pink line represents the compression limits for the connections in the work string.
  • The orange area represents the tension threshold for the allowable axial work string load.
  • The dotted orange line represents the tension limits for the connections in the work string.
  • The black line represents the true axial load under the initial conditions before the current operation begins.
Elongation Change Shows the change in the amount of axial deformation in the tubing.
Change in OD Shows the changes in the outside diameter of the tubing, under the predicted axial load.
Von Mises Stress Shows the effective stress acting along the length of the work string. The turquoise area represents the yield stress limit of the work string. If a line falls within the shaded area, the work string limits have been exceeded.
Temperature

Shows the changes in the temperature of the mud and the formation fluid.

  • The black line represents the temperature of the fluid in the work string.
  • The blue line represents the temperature of the fluid in the annulus.
  • The green line represents the temperate of the fluid in the formation.
  • The gray line represents the average temperature of the fluid in the work string and the annulus.
Differential Pressure

Shows the changes in differential pressure throughout the well, which is the difference between annular pressure and drillstring pressure. The chart also shows limits that represent the pressure limits of the tools in the work string. If a line falls within the shaded area, work string pressure limits have been exceeded.

  • The black line represents the differential pressure.
  • The orange area represents the burst capacity of the tools in the work string.
  • The purple area represents the collapse capacity of the tools in the work string.
Pressure

Shows changes to the total pressure in the annulus and the work string throughout the well.

  • The black line represents the annular pressure throughout the well.
  • The blue line represents the pressure inside the work string.
Temperature Difference

Shows the differences in temperature between the work string, the annulus, and the formation.

  • The black line represents the difference between the temperature in the formation and the temperature in the work string. A positive value indicates the formation temperature is higher, and a negative value indicates the work string temperature is higher.
  • The blue line represents the difference between the temperature in the formation and the temperature in the annulus. A positive value indicates the formation temperature is higher, and a negative value indicates the annulus temperature is higher than the formation temperature.
  • The gray line represents the difference between the temperature in the formation and the average temperature of the fluid in the work string and the annulus. A positive value indicates the formation temperature is higher, and a negative value indicates the average temperature in the work string and annulus is higher.
Elongation

Shows the amount of axial deformation in the tubing.

  • The black line shows the elongation under the initial conditions before the current operation begins.
  • The blue line represents the predicted elongation under the analysis parameters.
Axial Stress Shows the axial stresses acting on the work string. The turquoise area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Bending Stress Shows the stresses resulting from bending moments along the work string.
Radial Stress Shows the average stress across the radius of the work string.
Hoop Stress Shows the average stress working on the circumference of the work string.
Pipe Apparent Viscosity Shows changes to the apparent viscosity of the fluid in the work string throughout the well.
Pipe Velocity Shows changes to the velocity of the fluid in the work string throughout the well.
Fluid Density Shows the changes in the density of the fluid inside the work string and in the annulus.
Safety Factors

Shows the changes in different safety factors throughout the wellbore.

  • The orange line represents the axial tension safety factor, which is the ratio of the minimum tensile rating in the work string to the axial load.
  • The pink line represents the axial compression safety factor, which is the minimum compression rating in the work string to the absolute value of axial load.
  • The turquoise line represents the triaxial safety factor, which is the ratio of the yield stress to the Von Mises stress.
  • The purple line represents the burst safety factor, which is the ratio of the minimum burst rating in the work string to the burst pressure.
  • The brown line represents the collapse safety factor, which is the ratio of the minimum collapse rating in the work string to the external pressure.
  • The black area represents the safety factor limit, which is the smallest value the safety factor can be without damaging the equipment. If any of the safety factors fall outside of this area, the safety factor has been exceeded.
