About torque and drag parametric analysis

Torque and drag parametric analysis performs a sensitivity analysis using a variable range with other inputs to determine the required surface value. You can calculate results for four types of sensitivity analysis:

  • Apparent Overpull – Calculates the apparent overpull during tripping and underreaming operations, based on either a range of friction factors or a range of overpull values. See "Apparent overpull".
  • Apparent Torque – Calculates the apparent torque during drilling and underreaming, based on either a range of friction factors or a range of bit torque values. See "Apparent torque".
  • Apparent WOB – Calculates the apparent WOB during drilling, underreaming, and liner set operations, based on either a range of friction factors or a range of WOB values. See "Apparent WOB".
  • Neutral Point – Calculates the neutral point during drilling operations, based on either a range of friction factors or a range of WOB values. See "Neutral point".

When you run a torque and drag parametric analysis, there are four models available for predicting the effects of torque and drag:

  • The Simplified Soft String model is the most basic model, featuring instantaneous calculations. This model considers the work string to be a perfectly flexible rod and approximates the effect of helical buckling on drag. Not available for apparent overpull analysis.
  • The Soft String model is similar to the simplified soft string model, but also features updated buckling models and modified stress calculations. Additional rig operations are also available for the soft string model, such as picking up, slacking off, underreaming, cutting, and custom operations.
  • The Semi Stiff String model is based on the soft string model, but also takes bending stiffness into account and includes the calculations for bending moments, tool joint positions, and contact force at the tool joints and pipe body. Additional rig operations are also available for the semi-stiff string model, such as picking up, slacking off, underreaming, cutting, and custom operations.
  • The Stiff String model is a completely separate string model, with a transient model of lateral motion. This model includes calculations for bending moments, tool joint positions, contact force at the tool joints and pipe body, axial and torsional movement, and lateral movement and vibration. Not available for apparent overpull analysis.

Using the T&D Parametric page

The tabs on the right side of the screen are used to switch between torque and drag parametric analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove torque and drag parametric analysis experiments.

Experiment toolbar on the T&D Parametric page
Button Function
Add

Creates a copy of the current torque and drag parametric analysis experiment

Deletes the current experiment, including all analysis results

After the analysis completes, a chart will appear on the right side of the screen. When you hover over the graph index, next to the chart, another toolbar with additional options will appear.

Chart toolbar on the T&D Parametric page
Button Function

Pins the chart legends to the top of the analysis charts

Exports all chart data to a CSV file

Generates a one-page report of the analysis charts currently displayed on the Planning > Mechanics > T&D Parametric page

Opens the Data Browser window, which lists the results for the calculated depth profile

Adds an analysis chart to the charts view
