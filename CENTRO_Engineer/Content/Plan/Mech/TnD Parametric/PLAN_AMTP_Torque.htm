About apparent torque analysis

A sensitivity study for apparent torque calculates the apparent torque for rotary drilling, custom down, custom stationary, or underreaming operations.

When you configure the analysis settings, you can choose the input you want to use as a variable range in the Sensitivity Range field. There are two options for the variable range:

  • FF – Uses a range of friction factors with a constant value for bit torque
  • TOB – Uses a range of values for bit torque with a constant friction factor.

Starting the analysis

To run an apparent torque sensitivity analysis:

  1. Go to the Planning > Mechanics > T&D Parametric page.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. In the Sensitivity Analysis field, select the Apparent Torque option.
  4. In the Sensitivity Range field, select the input that you want to use as a variable.
    • FF – Uses a range of friction factors with a constant value for bit torque.
    • TOB – Uses a range of values for bit torque with a constant value for the friction factor.
  5. In the Solver field, select the torque and drag analysis model that you want to use. See "String models".
  6. In the Start Depth and End Depth fields, enter the starting and ending depths you want to analyze.
  7. In the Range Start and Range End fields, enter the minimum and maximum values for the variable range.
  8. In the Interval Count field, enter the number of increments you want to include.
  9. If you selected the FF option in the Sensitivity Range field, enter a constant value for the bit torque in the Bit Torque field.
  10. If you selected the TOB option in the Sensitivity Range field, enter a constant value for the friction factor in the FF field.
  11. Go to the Operating Parameters section and select the operation you want to include in the analysis.
  12. Enter the operating parameters you want to use for the operation. See "Operating parameters".
  13. Enter the flow rate of the drilling fluid.
  14. To begin the analysis, click the Calculate button.

When the analysis completes, a chart of the results will appear on the right side of the screen. See "Analysis chart".

Analysis chart

After the analysis completes, the Depth vs. Apparent Torque chart will appear on the right side of the screen. The results graph will contain one line for each increment specified in the Number of Intervals field.

The chart legend shows the specific values used for the variable range. If you selected the FF option in the Sensitivity Range field, the legend will show the range of friction factors used in the analysis. If you selected the TOB option, the legend will show the range of bit torque values.

Field definitions

The table below defines the settings for an apparent torque sensitivity analysis.

Settings for an apparent torque analysis
Field Definition
Wellpath

Wellpath used for the analysis
Well Interval

Well interval used for the analysis
Work String

Drillstring or casing string used for the analysis
Fluid

Circulating fluid used for the analysis
Sensitivity Analysis

Type of sensitivity study you want to run

  • Apparent Overpull – Calculates the required surface overpull across a range of overpull values or a range of friction factors.
  • Apparent Torque – Calculates the required surface torque across a range of torque on bit values or a range of friction factors.
  • Apparent WOB – Calculates the required surface WOB across a range of WOB values or a range of friction factors.
  • Neutral Point – Calculates the neutral point of the work string across a range of WOB values or a range of friction factors.
Sensitivity Range

Input that you want to use as a variable for the analysis.

  • FF – Uses a range of friction factors with a constant value for bit torque
  • TOB – Uses a range of values for bit torque with a constant friction factor.
Solver

Torque and drag model used in the analysis. See "String models".
Start Depth

Starting depth to include in the analysis
End Depth

Ending depth to include in the analysis
Range Start

Minimum value for the variable range.

  • If you selected the FF option in the Sensitivity Range field, this value will be the minimum friction factor.
  • If you selected the TOB option in the Sensitivity Range field, this value will be the minimum bit torque value.
Range End

Maximum value for the variable range.

  • If you selected the FF option in the Sensitivity Range field, this value will be the maximum friction factor.
  • If you selected the TOB option in the Sensitivity Range field, this value will be the maximum bit torque value.
Number of Intervals

Number of increments to include in the analysis. The increment size will be automatically calculated from the starting value in the Range Start field, the ending value in the Range End field, and the number of intervals. The results graph will contain one line for each increment.
Bit Torque

Constant value to use for bit torque. Only available when the FF option is selected in the Sensitivity Range field.
Friction Factor

Constant value to use for the friction factor. Only available when the TOB option is selected in the Sensitivity Range field.
Flow Rate

Flow rate of the circulating fluid
