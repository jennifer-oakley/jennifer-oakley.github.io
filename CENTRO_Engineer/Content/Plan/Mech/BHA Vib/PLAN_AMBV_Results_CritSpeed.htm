Viewing critical speed maps

The analysis charts for vibration analysis also include critical speed maps. To view critical speed maps:

  1. Go to the BHA Vibration page.
  2. Select the experiment you want to view results for.
  3. Click the Show Graphs button () in the top toolbar.
  4. Click the Switch Graph button ().

The critical speed maps will appear on top of the analysis charts. To close the critical speed maps and return to the analysis charts, click the Switch Graph button () again.

Changing graph settings

There are several settings that you can customize for critical speed maps:

  • The Auto Scale option adjusts the scale of the lateral vibrations, using the minimum and maximum calculated values.
  • The Use Grayscale option uses shades of gray to visualize critical speeds, instead of shades of red and green.
  • The Normalize option scales both critical speed maps to the same index for lateral vibrations.

To change the settings for critical speed maps, click the Heat Map Settings button () in the top toolbar.

List of critical speed maps

The results for vibration analysis also include critical speed maps. The following charts are available when you are viewing critical speed maps:

Critical speed maps for a vibration analysis
Chart Name Description Chart Sample
Drilling Rotary CSM

Shows the lateral vibrations generated during rotary drilling, for every combination of WOB and top drive speed included in the analysis. This chart is color-coded according to the intensity of the vibration.

  • The green areas represent the smallest vibrations.
  • The yellow areas represent medium vibrations.
  • The red areas represent the largest vibrations.
  • The circle marker indicates the WOB and top drive speed at which the maximum vibrations occur.
Drilling Sliding CSM

Shows the lateral vibrations generated during slide drilling, for every combination of WOB and flow rate included in the analysis. This chart is color-coded according to the intensity of the vibration. This chart is only available for mud motor BHAs.

  • The green areas represent the smallest vibrations.
  • The yellow areas represent medium vibrations.
  • The red areas represent the largest vibrations.
  • The circle marker indicates the WOB and top drive speed at which the maximum vibrations occur.
