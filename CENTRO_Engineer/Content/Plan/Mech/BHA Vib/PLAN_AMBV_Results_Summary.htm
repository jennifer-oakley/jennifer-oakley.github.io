BHA vibration analysis results summary

The results summary for vibration analysis includes critical speed maps and a table of the calculated maximum amplitudes, as well as a list of warnings for any limits that were exceeded.

If the limits for contact force or Von Mises stress were exceeded during the analysis, a warning message will appear at the top of the results summary. Each warning includes the WOB when the limit was exceeded, as well as a list of affected BHA components. For each BHA component, the warning contains the following information:

  • The sequence number for the component's position within the BHA.
  • The name of the component.
  • CF is the contact force acting on the component. If no value is listed, the component's maximum contact force was not exceeded.
  • VM is the Von Mises stress acting on the component. If no value is listed, the component's maximum Von Mises stress was not exceeded.

The limits for contact force and Von Mises stresses are determined by the properties of the BHA component. If you believe that the limits are incorrect, you may need to update the BHA component. See "Work string components".

The maximum amplitudes for each WOB are shown in a data table at the top of the result summary. The table provides the following values for each WOB included in the analysis:

Calculated values for vibration analysis
Column Definition
WOB Weight on bit
TDS Top drive speed when the maximum amplitude occurred
Flow Rate Flow rate when the maximum amplitude occurred. If the BHA doesn't contain a mud motor, this value will be zero.
Max Amplitude Maximum amplitude calculated at the specified WOB
Operation Drilling operation in progress when the maximum amplitude occurred
Bit Bit speed when the maximum amplitude occurred

Critical speed maps are shown at the bottom of the results summary. A critical speed map for rotary drilling is created for all BHAs. This chart graphs the lateral vibrations generated by every combination of WOB and top drive speed included in the analysis. The graph is color-coded according to the intensity of the vibration. The smallest vibrations are shown in green, while the largest vibrations are shown in red.

For BHAs with a mud motor, a critical speed map for slide drilling is also created. This chart graphs the lateral vibrations generated by every WOB and flow rate included in the analysis.
