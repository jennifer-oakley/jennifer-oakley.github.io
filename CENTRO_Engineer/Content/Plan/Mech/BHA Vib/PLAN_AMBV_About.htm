About BHA vibration analysis

On the Planning > Mechanics > BHA Vibration page, you can run a vibration analysis on the BHA. Vibration analysis detects dangerous rotation speeds in the BHA, which can cause damaging downhole vibrations. Many excitations act on the BHA simultaneously during downhole operations. These excitations can include rock cutting forces, contact forces, BHA part eccentricities, and mud motor forces.

CENTRO® Engineer calculates the resulting forces for the defined BHA configuration, bit positions, WOB values, and top drive speeds. These forces include vibrations, critical speeds, contact forces, and effective stresses.

Note

If you are using an offline license key, all modules related to BHA analysis will be disabled. This includes BHA static and vibration analysis, real-time critical speed maps, and the BHA analysis portion of a OneClick™ analysis.

Note

In some cases, the contact force for the bit may not be shown on the analysis charts.

Warning

When you run a BHA analysis on a BHA that contains an RSS unit, the analysis may complete with errors after decreasing the contact diameter of the pivot stabilizer.

Warning

When you run a vibration analysis on a BHA with a motorized RSS, the lateral vibrations calculated during the analysis may be excessively high.

Using the BHA Vibration page

The toolbar at the top of the screen allows you to inspect the wellbore trajectory, begin an analysis, and view analysis charts. This toolbar appears after the analysis completes.

Top toolbar on the BHA Vibration page
Button Function
Opens the advanced settings for the critical speed maps in the results summary.

Opens the Wellpath Inspection window, which allows you to compare the original survey to the corrected survey

Generates a report of the vibration calculation results. Only available after the analysis completes.. See "Creating reports".

Opens the analysis charts for the current experiment. Only available after the analysis completes.

The toolbar in the Excitations section of the analysis settings allows you to add, edit, and delete excitations.

Toolbar for the Excitations section
Button Function
Opens the selected excitation for editing. See "Field definitions".
Adds a new excitation. See "Adding excitations".
Deletes the selected excitation

The tabs on the right side of the screen are used to switch between vibration analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove vibration analysis experiments.

Experiment toolbar on the BHA Vibration page
Button Function
Add

Creates a copy of the current vibration analysis experiment

Deletes the current experiment, including all analysis results
