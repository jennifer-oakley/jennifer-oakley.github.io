Starting a directional tendencies analysis

To run a directional tendencies analysis:

  1. Go to the Planning > Mechanics > Directional Tendency (Beta) page.
  2. Select the wellpath, well interval, drillstring, and fluid you want to use for the analysis.
  3. Enter the bit depth you want to analyze in the Bit Depth field.
  4. For BHAs with a drilling motor or an RSS unit, select the toolface setting in the Tool Face field.
  5. In the Formation field, select the type of formation lithology at the selected bit depth.
  6. In the Analysis Scope field, select the type of directional tendencies analysis you want to run.
    • WOB predicts changes in the well's trajectory due to the amount of weight on bit.
    • HOG predicts changes in the well's trajectory due to the amount of wellbore overgauge.
    • DTS predicts changes in the well's trajectory due to the settings on a steering tool.
  7. If you are running an overgauge analysis, enter the range of hole overgauge values in the Hole Overgauge row.
  8. If you are running a steering analysis, enter the mud motor and RSS settings.
    • For RSS BHAs, enter the range of deflection values in the Deflection row.
    • For mud motor BHAs, enter the minimum and maximum bend angle.
  9. Go to the Drilling Rotary section and enter the expected operating parameters.
  10. If the BHA contains a mud motor, go to the Slide Drilling section and enter the amount of weight on bit.
  11. To begin the analysis, click the Calculate button () in the top right corner of the screen.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Direction tendencies results summary".

After you run A Directional Tendencies analysis, the Calculate button () in the top toolbar will be replaced with the Show Graphs button (). This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view. See "Directional tendencies charts".

Note

Work strings that contain wireline tools or wireline cables cannot be used in a Directional Tendencies analysis.

Field definitions

The table below defines the settings required for a directional tendencies analysis.

Settings for a directional tendencies analysis
Field Definition
Wellpath

Wellpath used for the analysis
Well Interval

Well interval used for the analysis
Work String

Work string used for the analysis
Fluid

Circulating fluid used for the analysis
Bit Depth

Bit MD used for the analysis
Tool Face

Toolface setting on an RSS unit or drilling motor. Only available for BHAs with an RSS unit or a drilling motor.
WOB

Range of WOB values used in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
Formation Type of formation lithology at the selected bit depth.
Analysis Scope

Method for calculating directional tendencies.

  • WOB predicts changes in the well's trajectory due to the amount of weight on bit.
  • HOG predicts changes in the well's trajectory due to the amount of hole overgauge.
  • DTS predicts changes in the well's trajectory due to the settings on a steering tool. Only available for BHAs with an RSS unit or a drilling motor.
Hole Over Gauge Amount of washout expected during drilling. This field is only available when the HOG analysis scope is selected.
Deflection Amount of deflection on an RSS unit. Only available for BHAs with an RSS unit, when the DTS analysis scope is selected.
Min Bend Angle Minimum bend angle to calculate results for. Only available for BHAs with a drilling motor, when the DTS analysis scope is selected.
Max Bend Angle Maximum bend angle to calculate results for. Only available for BHAs with a drilling motor, when the DTS analysis scope is selected.
Drilling Rotary

Operating parameters to use during rotary drilling.

  • WOB is the amount of weight on bit. When the WOB analysis scope is selected, this field will be a range of WOB values.
  • TOB is the amount of torque on bit.
  • TDS is the expected top drive speed.
  • ROP is the expected rate of penetration
Drilling Sliding

Operating parameters to use during slide drilling. Only available for BHAs with a drilling motor.

  • WOB is the amount of weight on bit. When the WOB analysis scope is selected, this field will be a range of WOB values.
