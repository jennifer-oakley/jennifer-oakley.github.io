Mechanics analyses

Knowledge Base
Related Topics
  • Experiments from imported projects show incorrect results

Mechanics analysis examines the effects of various forces on the BHA, as well as the effects of friction on the drillstring. This analysis is useful for selecting the best BHA and drillstring, as well as for identifying potential problems with a particular BHA configuration. There are several types of mechanics analyses in the Planning > Mechanics section:

  • Torque and drag predicts the torque & drag forces generated by the drillstring while drilling, sliding, tripping in, tripping out, rotating off-bottom, reaming and backreaming. See "Torque and drag analysis".
  • Real-time torque and drag analysis calculates torque and drag effects under the current well conditions. See "Real-time torque and drag analysis".
  • Tubing movement analysis calculates the effects of pressure, temperature, and mechanical changes on tubing strings. See "Tubing movement analysis".
  • BHA static analysis predicts the contact forces and displacements of the BHA by calculating the equilibrium state of the BHA. See "Static BHA analysis".
  • BHA parametric static analysis allows you to compare static analysis results for multiple BHAs at the same time. See "Parametric static BHA analysis".
  • BHA vibration analysis calculates the effects of vibrations on the BHA, such as dangerous rotation speeds, contact forces, eccentricities, and mud motor forces. See "BHA vibration analysis".
  • BHA parametric vibration analysis allows you to compare vibration analysis results for multiple BHAs at the same time. See "Parametric BHA vibration analysis".
  • Jar placement analysis uses stress wave analysis that allows you to optimize the location of a jar within a work string. See "Jar placement analysis".
  • Directional tendencies analysis predicts the behavior of the BHA during directional drilling. See "Directional tendencies analysis".

Before CENTRO® Engineer can calculate any results for a mechanics analysis, you'll need to add the following data to the well:

  • The elevation of the rig. See "Rig Elevations".
  • The planned well trajectory and survey data. See "Wellpath data".
  • The wellbore intervals for each cased and open hole section. See "Well intervals".
  • The proposed drillstring configuration. See "Work strings".
  • The fluid properties for the drilling mud. See "Drilling muds".
Note

The Mechanics section is only available in Real Time mode and Planning mode. See "Application modes".
