About torque and drag settings

When you run a torque and drag analysis, there are four models available for predicting the effects of torque and drag:

  • The Simplified Soft String model is the most basic model, featuring instantaneous calculations. This model considers the work string to be a perfectly flexible rod and approximates the effect of helical buckling on drag.
  • The Soft String model is similar to the simplified soft string model, but also features updated buckling models and modified stress calculations. Additional rig operations are also available for the soft string model, such as picking up, slacking off, underreaming, cutting, and custom operations.
  • The Semi Stiff String model is based on the soft string model, but also takes bending stiffness into account and includes the calculations for bending moments, tool joint positions, and contact force at the tool joints and pipe body. Additional rig operations are also available for the semi-stiff string model, such as picking up, slacking off, underreaming, cutting, and custom operations.
  • The Stiff String model is a completely separate string model, with a transient model of lateral motion. This model includes calculations for bending moments, tool joint positions, contact force at the tool joints and pipe body, axial and torsional movement, and lateral movement and vibration.

When you are using the With Contact buckling model, the suggested buckling force factor will be automatically calculated and shown in the Suggested Buckling Force Factor field. This value is calculated based on the selected wellpath, fluid, and work string. If the well is vertical at the depth entered in the Work String Depth field, no value will be calculated. As you update the input parameters for the analysis, the suggested buckling force factor will be automatically recalculated.

Note

Picking up, slacking off, underreaming, cutting, and custom operations are only available when the soft string or semi-stiff string model is selected in the Solver field

Starting the analysis

To run a torque and drag analysis:

  1. Go to the Planning > Mechanics > Torque and Drag (T&D) page.
  2. Select the wellpath, well interval, work string, and fluid you want to use for the analysis.
  3. Enter the flow rate of the drilling fluid.
  4. Enter the bit position or other depth you want to analyze in the Work String Depth field.
  5. In the Solver field, select the torque and drag analysis model that you want to use. See "String models".
  6. Select the remaining analysis options. See "Field definitions".
  7. To include the effects of drillstring rotation, select the Rotational Effects option.
  8. Go to the Operating Parameters section and select the operations you want to include in the analysis.
  9. Enter the operating parameters you want to use for each operation. See "Operating parameters".
  10. To begin the analysis, click the Calculate button.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Torque & drag results summary".

After you run Atorque and drag analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view. See "Torque and drag charts".

Field definitions

The table below defines the general settings required for a torque and drag analysis. These settings are used for all operations.

Settings for torque and drag analysis
Field Definition
Wellpath

Wellpath used for the analysis
Well Interval

Well interval used for the analysis
Work String

Drillstring or casing string used for the analysis
Fluid

Circulating fluid used for the analysis
Operation Hydraulics operation that is occurring during the analysis. Only available when the selected work string contains wireline tools or cable.
Flow Rate

Flow rate of the circulating fluid
Work String Depth

Bit MD or other depth used for the analysis
Solver

Torque and drag model used in the analysis. See "String models".
FF Sensitivity Option

Friction factor sensitivity range and values input method.

  • Standard – Use the standard friction factor sensitivity values. When this option is selected, the analysis will use a range of friction factors from 0.10 to 0.60, using six increments.
  • Custom – Set a custom friction factor sensitivity range and the desired interval count. When this option is selected, the analysis will use the specified range of friction factors, using the number of increments in the Number of Intervals field.
From Minimum friction factor value. Only available when the Custom option is selected in the FF Sensitivity Option field.
To Maximum friction factor value. Only available when the Custom option is selected in the FF Sensitivity Option field.
Number of Intervals

Number of increments to include in the analysis. The increment size will be automatically calculated from the starting value in the From field, the ending value in the To field, and the number of intervals. The friction factor charts will contain one line for each increment. Only available when the Custom option is selected in the FF Sensitivity Option field.
Buckling Model

Buckling model to use in the analysis. Only available for the soft string and semi-stiff string models. See "Buckling".

  • Standard – The Standard buckling model does not account for contact forces, but requires less time to calculate.
  • With Contact – The With Contact buckling model accounts for contact forces, using a buckling force factor to calibrate the calculations. The suggested buckling force factor will be automatically calculated and shown in the Suggested Buckling Force Factor field.
Buckling Force Factor Calibration factor that equates the buckling calculations to field data. The buckling force factor may differ, based on the type of well, curvature, and other effects that are not taken into account by the model. Only available for the With Contact buckling model. See "Buckling with contact".
Suggested Buckling Force Factor Automatically calculated buckling force factor. This value is calculated based on the selected wellpath, fluid, and work string. If the well is vertical at the depth entered in the Work String Depth field, no value will be calculated. As you update the input parameters for the analysis, the suggested buckling force factor will be automatically recalculated. Only available for the With Contact buckling model. See "Buckling with contact".
Rotational Effects

Includes the effects of drillstring rotation in the analysis. Not available for the stiff string model.
String Fill-In # Of Joints

Number of joints that will be added to the drillstring when tripping in. This value is used to account for periodic pipe fill as new joints are added to the drillstring. The analysis assumes that new joints are filled with air when they are added to the drillstring, then filled with the active mud when the pipe reaches the fill-in point. Only available with the simplified soft string model.
