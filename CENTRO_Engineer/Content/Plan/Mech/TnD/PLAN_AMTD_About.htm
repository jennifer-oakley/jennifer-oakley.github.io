About torque and drag analysis

On the Planning > Mechanics > Torque and Drag (T&D) page, you can run a torque and drag analysis. Torque and drag analysis predicts the torque & drag forces generated by the drillstring during various wellbore operations. It calculates the forces, torques, and stresses that act on the drillstring at the specified bit depth. The operations available for realtime torque and drag analysis include drilling, sliding, tripping in and out, rotating off-bottom, reaming, and backreaming.

Note

Mud motor torque is not currently included in torque and drag calculations. As a workaround, adjust the amount of torque on bit to compensate for the motor torque.

Note

Changing the bit position in the Well View pane will yield different analysis results than entering the same bit position in the analysis settings.

Warning

When rotation effects are enabled, surface torque results in the analysis charts may not match the results shown in the friction factor charts.

Using the Torque and Drag page

The toolbar at the top of the screen allows you to inspect the wellbore trajectory, begin an analysis, and view analysis charts.

Toolbar on the Torque and Drag (T&D) page
Button Function
Generates a report of the torque and drag calculation results. Only available after the analysis completes.. See "Creating reports".

Opens the analysis charts for the current experiment. Only available after the analysis completes.

Opens the Wellpath Inspection window, which allows you to compare the original survey to the corrected survey
Opens the Actual Data window, which allows you to import the actual well data for hookload and surface torque. See "Actual data".

The tabs on the right side of the screen are used to switch between torque and drag analysis experiments. Each experiment has its own tab, with a toolbar that is used to add and remove torque and drag analysis experiments.

Experiment toolbar on the Torque and Drag (T&D) page
Button Function
Add

Creates a copy of the current torque and drag analysis experiment
