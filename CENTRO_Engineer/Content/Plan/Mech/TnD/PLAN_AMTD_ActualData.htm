About actual data

In the Actual Data window on the Planning > Mechanics > Torque and Drag (T&D) page, you can import the field hookload and surface torque after the well is drilled. This allows you to compare the actual data to the modeled analysis results. Currently, you can enter four types of field data:

  • Hookload while running into the well
  • Hookload while pulling out of the well
  • Hookload while rotating and circulating off-bottom
  • Surface torque while rotating and circulating off-bottom

You can enter the data manually, or you can import it from a CSV file. The import file should contain a column for the measured depth of the bit, as well as the field data that you want to import. See "Importing actual data".

Importing actual data

To import actual data into the current experiment:

  1. Go to the Planning > Mechanics > Torque and Drag (T&D) page.
  2. Select the experiment that you want to add data to, then click the Actual Data button ().
  3. In the Actual Data window, click the Import Data button ().
  4. Navigate to the file you want to import and click Open. A list of import options will appear in CENTRO® Engineer.
  5. In the Import Options field, select the method you want to use for handling existing data.
    • The Merge option adds the new data to the existing data. If the new data contains depth values that already exist, both sets of data for duplicate depths will appear in the Actual Data window.
    • The Overwrite option erases all existing data, and replaces it with the new data.
  6. In the Depth field, select the name of the column that contains the measured depth of the bit.
  7. In the remaining rows, select the column that contains the data you want to import.
  8. Verify the measurement units selected for each row.
  9. Click the OK button () to begin the import.
  10. When the import finishes, click the Save Changes button ().

The imported data will appear in the Actual Data window. After you import the actual data, you can compare it to the modeled analysis results. See "Comparing modeled vs. actual data".

Comparing modeled vs. actual data

To compare the actual data to the modeled analysis results:

  1. Go to the Planning > Mechanics > Torque and Drag (T&D) page.
  2. Select the experiment that contains the actual data.
  3. Click the Show Graphs button () in the top toolbar.
  4. Click the Filter button (), then select the Actual Data option in the menu that appears.
  5. Click the Show Friction Factor button () to view the friction factor charts.
  6. Click the Filter button (), then select the desired options for the friction factor charts.
    • The Running In Actual option shows the hookload data while tripping into the well.
    • The ROB Actual option shows the hookload and surface torque data while rotating and circulating off-bottom.
    • The Pulling Out Actual option shows the hookload data while tripping out of the well.

Markers for the selected actual data will appear in the main analysis charts for hookload and surface torque, as well as in the friction factor charts.

Using the Actual Data window

The toolbar at the bottom of the Actual Data window allows you to add, import, and delete actual data.
