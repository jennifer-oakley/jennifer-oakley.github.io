Viewing analysis charts

After you run Atorque and drag analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

To view the charts for A torque and drag experiment:

  1. Go to the Planning > Mechanics > Torque and Drag (T&D) page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Graphs button () in the top toolbar.

The charts will appear on top of the analysis settings. To close the analysis charts, click the Show Graphs button () again.

Adding charts

In addition to the charts shown by default, there are more charts available. To view these charts:

  1. Click the Add Graph Page button () in the chart toolbar.
  2. In the menu that appears, click on the chart you want to view.

The selected chart will appear last in the charts view, on the right side of the current charts.

List of torque and drag charts

The charts listed below are available after you run an analysis on the Torque and Drag (T&D) page. If you want to view a chart that is not shown by default, you'll need to add it to the chart view. See "Adding charts".

Charts for A torque and drag analysis
Chart Name Description Chart Sample
Axial Load

Shows the tension throughout the drillstring. Use this plot to identify conditions that are likely to cause buckling in the drillstring. The chart shows limits that represent threshold limits for drillstring buckling. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The gray area represents sinusoidal buckling limits.
  • The red area represents helical buckling limits.
  • The red dashed line represents rotary helical buckling limits.
  • The pink area represents the compression limits for the tools in the work string.
  • The orange area represents the tension threshold for the allowable axial drillstring load.
  • The teal line represents the operating hoisting capacity of the rig.
  • The other lines represents the axial load under the current operating conditions. If a line falls within the red or gray areas, buckling limits have been exceeded.
Torque

Shows the torque throughout the drillstring, from the surface to the selected bit position. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The orange area represents the torque threshold.
  • The dotted line represents the makeup torque of the drillstring.
  • The teal line represents the operating torque capacity for the rig.
  • The other lines represent the torque under the current operating conditions. If a line falls within the shaded area, torque limits have been exceeded.
Contact Force Shows the contact forces acting along the length of the drillstring for each selected operation.
Von Mises Stress Shows the effective stress acting along the length of the drillstring. The orange area represents the drillstring yield stress limit. If a line falls within the shaded area, drillstring limits have been exceeded.
Hook Load

Shows the trend in the maximum hookload found at the surface while the drillstring is operating. The chart shows limits that represent threshold limits for buckling and hookload. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The red area represents helical buckling limits.
  • The gray area represents sinusoidal buckling limits. This curve is only available for the soft string model and the semi-stiff string model.
  • The orange area represents the tensile overload hookload limit.
  • The teal line represents the operating hoisting capacity of the rig.
  • The other lines represents the axial load under the current operating conditions. If a line falls within the shaded areas, hookload limits have been exceeded.
Surface Torque

Shows the trend in the maximum torque found at the surface. If any of the limits are not visible on the graph, then the limit has not been exceeded.

  • The orange area represents the surface torque threshold.
  • The black dotted line represents the make up torque at surface.
  • The teal line represents the operating torque capacity for the rig.
  • The other lines represent the surface torque under the current operating conditions. If a line falls within the shaded area, torque limits have been exceeded.
Wireline Tool Shows the maximum rigid tool length and maximum dogleg severity for the wireline tools in the work string. Only available when the work string contains a wireline tool
Wireline Cable Shows the tension limits and drum crush thresholds for the wireline cable in the work string. Only available when the work string contains wireline cable.
True Axial Load Shows the effective axial load throughout the drillstring, from the surface to the selected bit position.
Axial Stress Shows the axial stresses acting on the drillstring, from the surface to the selected bit position. The orange area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Torsional Stress Shows the maximum shear stress, generated by torque, that is acting on the drillstring. The orange area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Shear Stress Shows the shear stresses acting on the drillstring, from the surface to the selected bit position. The orange area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Bending Stress Shows the stresses resulting from bending moments along the drillstring, from the surface to the selected bit position. The orange area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Hoop Stress Shows the average stress working on the circumference of the drillstring. The orange area represents the yield stress threshold. If a line falls within the shaded area, the allowable yield stress has been exceeded.
Radial Stress

Shows the average stress across the radius of the drillstring. If the yield stress threshold is not visible on the graph, then the limit has not been exceeded.

  • The orange area represents the yield stress threshold.
  • The other lines represent the radial stress under the current operating conditions. If a line falls within the shaded area, yield stress limits have been exceeded.
String Position Y Shows the vertical deviation of the drillstring from the wellbore centerline.
String Position Z Shows the tangential deviation from the wellbore centerline.
Pressure Inside Drillstring Shows the pressure inside the drillstring.
Pressure Inside Annulus Shows the pressure inside the annulus.
Twist Angle Shows the angle at which the drillstring twists along the length of the wellbore.
Eccentricity Shows the percentage of eccentricity in the drillstring throughout the wellbore.
Bending Moment Shows the amount of torque required to cause buckling along the drillstring.
Radial Clearance Shows the amount of clearance between the drillstring and the wellbore.
Stretch/Shortening Shows the amount that the drillstring stretches or shortens throughout the wellbore.
Min WOB

Shows the minimum amount of WOB required to cause buckling.

  • The red line represents the minimum amount of WOB required to cause helical buckling.
  • The gray represents the minimum amount of WOB required to cause sinusoidal buckling. This curve is only available for the soft string model and the semi-stiff string model.
String Buoyant weight per unit length Shows the weight of the drillstring as a gradient.
