About the torque & drag results summary

The results summary for torque and drag analysis includes maximum predicted values and a summary of each operation. See "Maximum predicted values" and "Operation summary". If the work string includes wireline tools or cable, the results summary also includes warning messages related to the wireline tools and cable. See "Wireline warnings".

Maximum predicted values

The maximum predicted values are shown in the Maximum Values section of the results summary. CENTRO® Engineer calculates several predicted values, including hookload, surface torque, equivalent stress, buckling, contact force, stretch, and windup. For each predicted value, the table lists the following information:

Definitions for maximum predicted values
Column Name Definition
Predicted Name of the predicted value
Max Maximum predicted value encountered during the analysis
Operation Operation that was in progress when the maximum predicted value occurred
MD Depth where the maximum predicted value occurred
Capacity Percentage of the limit that was reached by the maximum predicted value

Operation summary

A summary of the selected operation is shown in the Operating Parameters section of the results summary. Each operation summary includes a list of calculated values and a physical limits chart. The physical limits chart shows multiple physical limits, such as tension and torque, and the percentage of each limit that was reached during the operation. To select a different operation, click the Options button () next to the operation name, then select the operation you want to view.

For pulling out, backreaming, picking up, and custom up operations, the operation summary also includes a table showing the tensile limits and overpull margins for each safety factor.

Definitions for safety factor values
Column Name Definition
Safety Factor Safety factor used to calculate the allowable tensile limit and overpull margin
Components Last component in the work string selected for the analysis
Tensile Limit Maximum tensile limit for the component
Allowable Tensile Maximum allowable tensile limit for the component, at the specified safety factor
Axial Load Maximum axial load for the component
Overpull Margin Maximum allowable overpull for the component, at the specified safety factor

Wireline warnings

If the work string includes wireline tools or cable, the results summary also includes warning messages related to the wireline tools and cable.

List of warning messages for wireline tools and cable
Warning Message Cause
Tool can/cannot be conveyed to TD The hook load when running in is lower than the compression limit.
Cable tension to break weakpoint exceeds maximum cable tension The maximum tension rating for the cable is less than the amount of tension required to break the weak point
Tool weight in air cannot exceed 30% of weakpoint setting The weight of the work string in air exceeds the recommended threshold of 30% of the weak point.
Toolstring weight too low The buoyant weight of the work string is less than 0.
Maximum rigid tool of [X] OD, length [Y] will not pass depth [Z] The tool length exceeds the radius of curvature at the specified depth, based on the azimuth and inclination of the well at that point.
