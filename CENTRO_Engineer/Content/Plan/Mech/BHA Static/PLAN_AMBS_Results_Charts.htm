Viewing analysis charts

After you run Astatic analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view.

To view the charts for A static experiment:

  1. Go to the Planning > Mechanics > BHA Static page.
  2. Select the experiment that you want to view results for.
  3. Click the Show Graphs button () in the top toolbar.

The charts will appear on top of the analysis settings. To close the analysis charts, click the Show Graphs button () again.

Adding charts

In addition to the charts shown by default, there are more charts available. To view these charts:

  1. Click the Add Graph Page button () in the chart toolbar.
  2. In the menu that appears, click on the chart you want to view.

The selected chart will appear last in the charts view, on the right side of the current charts.

Using the WOB slider

You can change the amount of weight on bit reflected in the analysis charts. By default, the analysis charts show results using the minimum WOB specified in the analysis settings. Use the slider above the charts to change the current value for WOB.

Using displacement fills

Eccentricity and Von Mises fills

The analysis charts include vertical and horizontal displacement charts for the BHA included in the analysis. There are fills available for the displacement charts to help you visualize the results of a static analysis. When one of these fills is selected, a gradient will be applied to the vertical and horizontal displacement charts.

  • Eccentricity shows changes in the eccentricity of the BHA.
  • Von Mises shows changes in the effective stresses acting on the BHA.

To change the current fill, go to the toolbar above the analysis charts. Click the Options button () next to the name of the current fill, then select a fill from the list that appears.

Changing graph settings

You can change the appearance of a chart in the Graph Settings window. Each chart has its own settings, so changing the appearance of one chart will not affect how another chart looks.

To customize the default graph settings:

  1. Go to the bottom right corner of the chart you want to customize, then click the Options button (). A menu with toolbar buttons will appear on top of the chart.
  2. Click the Edit button () to open the Graph Settings window.
  3. In the menu at the top of the window, select the name of the chart series that you want to customize.
  4. Change the options for the chart series as desired.

As you make changes to the chart, your changes will be automatically saved and updated. When you are finished making changes, click the Close button () at the top of the window.

List of analysis charts

The charts listed below are available after you run an analysis on the BHA Static page. If you want to view a chart that is not shown by default, you'll need to add it to the chart view. See "Adding charts".

Analysis charts for a static analysis
Chart Name Description Chart Sample
Vertical & Horizontal Displacement

Shows vertical and horizontal displacement of the BHA from the center of the wellbore.

  • The brown areas represent the wellbore wall.
  • The purple areas represent minimum and maximum BHA displacement.
  • The blue line represents the average BHA displacement.
Contact Force

Shows the contact forces between the BHA and the wellbore. The chart shows a limit that represents the threshold for contact force. If the line falls within the shaded area, contact force limits have been exceeded.

  • The orange area represents the contact force threshold.
  • The blue line represents the wellbore contact forces along the length of The BHA.
Von Mises Stress

Shows the effective stresses acting in cross sections of The BHA, also known as equivalent tensile stress or Von Mises stress. The chart shows a limit that represents the threshold for effective stress. If the line falls within the shaded area, effective stress limits have been exceeded.

  • The orange area represents the effective stress threshold.
  • The blue line represents the effective stress along the length of The BHA.
Axial Stress Shows the normal stresses generated by joint forces along the axial axis of The BHA.
Bending Stress Shows the normal stresses that are generated by lateral bending moments along the length of The BHA.
String Position Y Shows the vertical deviation of The drillstring from the wellbore centerline.
String Position Z Shows the tangential deviation from the wellbore centerline.
Bending Moment Shows the lateral bending moments along The BHA.
Radial Clearance Shows the amount of clearance between The BHA and the wellbore.
Axial Force Shows the joint forces acting along the axial axis of The BHA. A negative value indicates that the BHA components are being compressed.
Lateral Displacement Shows the average lateral displacement of the BHA from the center of The wellbore.
Minimum Clearance Shows the minimum amount of clearance between The BHA and the wellbore.
