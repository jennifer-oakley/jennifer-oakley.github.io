Starting a parametric vibration analysis

To run a parametric vibration analysis:

  1. Go to the Planning > Mechanics > BHA Vibration Parametric page.
  2. Select the wellpath, well interval, drillstring, and fluid you want to use for the analysis.
  3. Enter the bit depth you want to analyze in the Bit Depth field.
  4. Enter mud motor and RSS settings.
    • Select the toolface setting in the Tool Face field.
    • For RSS BHAs, enter the amount of deflection in the Shaft Deflection field.
  5. Enter the range of WOB values to include in the WOB row.
  6. Enter the range of top drive speeds to include in the TDS row.
  7. For BHAs with a mud motor, enter the range of flow rates in the Flow Rate row.
  8. Enter the range of bit depths in the Bit Depth row.
  9. To begin the analysis, click the Calculate button.

When the analysis completes, a summary of the results will appear on the right side of the screen. See "Parametric BHA vibration analysis results summary".

After you run Aparametric vibration analysis, the Show Graphs button () will appear in the top toolbar. This button is used to switch between the main view, which contains the analysis settings and the results summary, and the charts view. See "Parametric BHA vibration charts".

Note

Work strings that contain wireline tools or wireline cables cannot be used in a parametric vibration analysis.

Field definitions

The table below defines the settings required for a parametric vibration analysis.

Settings for a parametric vibration analysis
Field Definition
Wellpath

Wellpath used for the analysis
Well Interval

Well interval used for the analysis
Work String

Work strings used for the analysis
Fluid

Circulating fluid used for the analysis
Bit Depth

Bit MD used for the analysis
Tool Face

Toolface setting on an RSS unit or drilling motor. Only available for BHAs with an RSS unit or a drilling motor.
Shaft Deflection

Amount of deflection on an RSS unit. Only available for BHAs with an RSS unit.
ID Option

Inside diameter that the analysis will use for calculations. Only available for work strings that contain an RSS unit or an LWD tool.

  • Standard – Assumes the RSS unit and LWD tool contain inserts.
  • ID Without Inserts – Assumes the RSS unit and LWD tool do not contain any inserts.
WOB

Range of WOB values used in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
TDS

Range of top drive speeds used in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
Flow Rate

Range of flow rates used in the analysis. Only available for BHAs with a drilling motor.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
Bit Depth

Range of bit depth used in the analysis. The minimum bit depth must be greater than the length of the longest BHA included in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
