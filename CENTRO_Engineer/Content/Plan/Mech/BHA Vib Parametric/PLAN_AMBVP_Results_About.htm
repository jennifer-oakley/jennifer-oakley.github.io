About parametric vibration results

BHA parametric vibration analysis provides you with several BHA profiles, such as lateral and axial vibrations, critical speeds, BHA displacement, contact force, and effective stress.

There are three types of results available for parametric vibration analysis:

  • A summary of the results, which includes critical speed maps and a data table containing the calculated maximum amplitudes. See "Parametric BHA vibration analysis results summary".
  • Analysis charts, which graph changes in the vibrations, forces, and stresses that act on the length of the BHA. See "Parametric BHA vibration charts".
  • Critical speed maps, which predict critical rotation speeds caused by the vibrations on the BHA. See "Parametric critical speed maps".

Creating reports

After you run a parametric vibration analysis, you can generate a report of the results. Parametric vibration analysis reports can include several components, such as a cover page, job information, wellpath data, the work string configuration, fluid information, the well interval configuration, and the analysis results. Parametric vibration reports can be created in either MS Excel or PDF format.

To create A parametric vibration analysis report:

  1. Go to the Planning > Mechanics > BHA Vibration Parametric page.
  2. Select the experiment that you want to create a report for.
  3. Click the Save Report button () at the top of the screen.
  4. In the Vibration Analysis Report Options window that opens, select the location where you want to save the report. To change the default folder location and file type, click the Browse button () in the Path field.
  5. Select the components that you want to include in the report.
  6. If you selected the BHA & Drill String option, select the template that you want to use for the work string portion of the report.
  7. Click the Save Report button () at the bottom of the window.

The report will be generated with the selected report components. When the report finishes generating, it will open automatically. If you created the report in MS Excel format, click the Enable Content button to enable all of the report features when the report opens.

Using the Charts view

When you are viewing analysis charts, additional buttons will appear in the top toolbar.

Top toolbar when viewing analysis charts
Button Function

Opens the Kinetic Energy chart in a separate window
Opens the advanced settings for the critical speed maps. Only available when viewing critical speed maps.
Opens the critical speed maps for the current experiment

Closes the analysis charts and returns to the analysis settings

The toolbar on the left side of the graphs allows you to pin chart legends, export chart data, and view the data table for the charts.
