About line styles

On the Global Settings > System Preferences > Graph Line Styles page, you can customize and create graph line styles. Graph line styles control the default appearance of the curves in the graphs on the Dashboards screen.

Before you create any custom dashboards, you can create custom line styles for different types of curves. For example, you could create one line style for calculated data and another for incoming WITS and WITSML data. Line styles are grouped into the same categories as the curves on the Dashboards screen.

Line styles are only applied to newly-added curves. When you add or edit a line style, the existing curves and graphs on the Dashboards screen will not be affected.

Adding line styles

To add a new graph line style:

  1. Go to the Global Settings > System Preferences > Graph Line Styles page.
  2. Click the Insert Item button () at the top of the screen.
  3. In the new row that appears, go to the Category field and select the category of curves that the line style will be applied to.
  4. In the Filter field, select which curves within the selected category that the line style will be applied to.
    • To apply the line style to all curves within the category, select the * option.
    • To apply the line style to a specific curve within the category, select the name of the curve.
  5. Update the line and marker styles as desired.
  6. Click the Save Changes button () to save your changes.

The new graph line style will be added to the CENTRO® Engineer database. When you add a new curve to a graph on the Dashboards screen, the new line style will be applied to any curves from the selected category. However, existing curves will not be effected.

Using the Graph Line Styles page

The toolbar at the top of the screen is used to save and undo changes, as well as manage the graph line styles in CENTRO® Engineer. To select the columns shown in the Graph Styles table, click the Show Columns button () in the first column of the table.
