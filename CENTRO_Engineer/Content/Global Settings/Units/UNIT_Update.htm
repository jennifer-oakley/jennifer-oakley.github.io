Assigning a unit system

The active unit system is indicated by a red toggle switch () in the list of unit systems. To change the active unit system:

  1. Go to the Global Settings > Units page.
  2. Select the unit system you want to activate.
  3. Click the Activate button ().
  4. Click the Save Changes button ().

The selected unit system will be marked as the active unit system, using the red toggle switch (). All of the measurement units in CENTRO® Engineer will be updated. If you are using a custom unit system, the settings for the custom unit system will be updated. See "Creating a custom unit system".

Creating a custom unit system

To create a custom unit system:

  1. Go to the Global Settings > Units page.
  2. Click the Add button to create new unit system.
  3. Change the default measurement units as needed. See "Changing a unit of measure".
  4. Change the default precision settings as needed. See "Changing the unit precision".
  5. Click the Save Changes button () to save your changes.

The settings for the custom unit system will be saved in the CENTRO® Engineer database. A new tab for the unit system will appear on the right side of the screen. However, the new unit system will not be the active unit system until you assign it to the well. See "Assigning a unit system".

Tip

To change the name of the custom unit system, go to the toolbar on the right side of the screen and click on the tab for the unit system. In the box that appears, enter a new name for the unit system and press the Enter key.

Changing a unit of measure

To change a unit of measure:

  1. Go to the Global Settings > Units page.
  2. Using the tabs on the right, select the unit system you want to edit.
  3. Locate the unit type that you want to update.
  4. Select the measurement unit that you want to use.
  5. Click the Save Changes button () to save your changes.

The measurement units for that unit type will be updated in the selected unit system.

Changing the unit precision

To change the decimal precision for a unit:

  1. Go to the Setup > Units page.
  2. Using the tabs on the right, select the unit system you want to edit.
  3. Locate the unit type that you want to update.
  4. In the field next to the measurement unit, enter the number of decimal places that you want to appear for that measurement unit.
  5. Click the Save Changes button () to save your changes.

The precision settings for that unit type will be updated in the selected unit system.

Tip

You can also control the decimal precision for unitless values, such as pH and porosity. To change the precision for unitless values, scroll to the bottom of the Global Settings > Units page and enter a new value in the Unitless Precision field.

Exporting unit systems

To export a unit system:

  1. Go to the Global Settings > Units page.
  2. Select the unit system you want to export, then click the Export Unit Settings button ().
  3. In the window that appears, navigate to the desired folder location and click Save.

The settings for the selected unit system will be saved to a *.unt file.

Importing unit systems

To import a unit system:

  1. Go to the Global Settings > Units page.
  2. Click the Import Unit Settings button ().
  3. In the window that appears, navigate to the file you want to import and click Open.
  4. Click the Save Changes button ().

The unit system will be imported into CENTRO® Engineer. A new tab for the unit system will appear on the right side of the screen. However, the new unit system will not be the active unit system until you assign it to the well. See "Assigning a unit system".
