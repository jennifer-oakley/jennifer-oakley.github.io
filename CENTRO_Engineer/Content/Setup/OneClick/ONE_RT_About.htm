OneClick™ real-time settings

The algorithm on the Setup > OneClick™ > Real Time page is used for real-time monitoring. It automatically detects the current rig operation, based on the current well parameters. This algorithm is triggered in real-time, at one-second intervals.

The Setup > OneClick™ Settings > Real Time page is divided into five tabs:

  • The Events tab defines the well states that are triggered when CENTRO® Engineer encounters them in an action node of a flow or run algorithm. See "OneClick™ real-time events".
  • The Calculations tab defines the expressions used to find calculated values, such as average hookload and average torque. Calculations can be used in condition checks, as well as other calculations. See "OneClick™ real-time calculations".
  • The Conditions tab defines the condition checks performed when CENTRO® Engineer encounters them in a trigger node of a flow or run algorithm. Condition checks are expressions that CENTRO® Engineer evaluates to determine if a statement is true or false. See "OneClick™ real-time condition checks".
  • The Flows tab defines the workflows that are used by the run algorithm. Flows can be composed of flow nodes, trigger nodes, and action nodes. See "OneClick™ real-time flows".
  • The Run tab defines the run algorithm for real-time monitoring. The run algorithm is the master workflow, and it is connected to the events, calculations, conditions, and flows on the Setup > OneClick™ Settings > Real Time page. See "OneClick™ real-time run algorithm".
