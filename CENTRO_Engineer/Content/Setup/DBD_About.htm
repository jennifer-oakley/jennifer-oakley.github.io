About user data

The Setup > User Data page contains depth-based curve data that has been exported from analysis results. You can export depth-based curves from hydraulics snapshot results or swab/surge analysis results. See "Exporting snapshot curve data" and "Exporting swab and surge curve data". You can also export geomechanics results or create a custom depth-based curve set. See "Geomechanics" and "Adding custom curve data".

The curve sets on the User Data page also appear in the list of available curves on the Dashboards screen, under the User Data category. The data on the User Data page is linked directly to these curves. When you rename or delete one of the tabs on the Setup > User Data page, the corresponding curve set on the Dashboards screen will also be renamed or deleted.

Tip

To change the name of a curve, go to the Setup > User Data page. Go to the toolbar on the right side of the screen and click on the tab name. In that box that appears, enter a new name of the curve and press the Enter key.

Note

The User Data page is only available in Real-Time mode. See "Application modes".

Adding custom curve data

To add a custom curve set:

  1. Go to the Setup > User Data page.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of curve sets.
  3. Click the Edit button () to open the curve set for editing.
  4. Go to the Unit of Depth field and select the measurement unit you want to use for depth.
  5. In the table at the top of the screen, enter the name of each curve in the Column Name column.
  6. For each curve, select the type of measurement unit and the unit that you want to use.
  7. In the table at the bottom of the screen, enter the measured depth values and curve values.
  8. When you have finished entering data, click the Save Changes button () to close the panel.
  9. Click the Save Changes button () to save your changes.

The curve data will appear in a new tab on the right side of the screen. The curve data will also be added to the list of available curves on the Dashboards screen, under the User Data category. To graph the curve data, add it to a depth-based graph. See "Adding curves to a track".

To send the curve data to a depth-based WITSML log, click the Publish To WITSML button (). A new depth log will be created on the first WITSML channel listed on the Setup > DAQ > WITSML page. See "WITSML channels".

Note

If the No Unit option is selected in the Unit Type field, the data for that curve will be sent as unitless data. Unitless data is curve data that does not have a measurement unit associated with it.

Using the User Data page

The toolbar at the top of the User Data page is used to edit curve sets and publish data to WITSML. To export curve data to a CSV file, select the curve set you want to export and press the Ctrl + S keys. To select the columns shown in the Mnemonics table or the Data table, click the Show Columns button () in the first column of the table.
