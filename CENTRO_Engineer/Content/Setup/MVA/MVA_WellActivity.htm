About the well activity planner

On the Setup > MVA Settings > Well Activity Planner page, you can plan the bit and casing runs required to drill the well. The information on the Well Activity Planner page can be used to:

  • Automatically activate well intervals and work strings. When the Auto activate well objects based on Well Activity time intervals option is selected, CENTRO速 Engineer will automatically activate the corresponding well interval, work string, and pump fluid when a new well activity begins.
  • Create well intervals. When you create a well interval on the Well Activity Planner page, an identical well interval will also be created on the Setup > Wells > Well Intervals page. See "Adding well intervals".
  • Recalculate data. When you recalculate data on the Setup > MVA Settings > Recalculation page, you can use the well activities defined on the Well Activity Planner page to recalculate sections of data that use multiple well intervals, work strings, or fluids. See "Recalculating with the Well Activity Planner".
  • Filter data in reports and graphs. You can also use well activities to filter the data in on-demand reports, performance reports, KPI reports, time-based graphs, and the T&D FF Sensitivity Depth Data indicator. See "Reports" and "Graphs".

There are two basic steps for using the well activity planner:

  • Add well intervals. Well intervals should be added sequentially, starting with the surface casing section and ending with the last open hole section. If a section will be drilled and then cased, CENTRO速 Engineer will assume that the section is cased after all of the activities for the interval have been completed. See "Adding well intervals".
  • Add activities to each interval. The activities for each interval specify which work strings will be used to drill or case the interval, the expected beginning and end of the activity, and the fluid that will be used. See "Adding activities".

Adding well intervals

To add well intervals:

  1. Go to the Setup > MVA Settings > Well Activity Planner page.
  2. Click the Add button () at the top of the screen.
  3. Enter the diameter, start depth, end depth, and friction factor for the open hole section in the fields at the top of the new well interval.
  4. If the interval is cased or will be cased, select the Casing option and enter the casing information in the fields that appear.
  5. Add activities to the interval. See "Adding activities".
  6. Continue adding intervals until the last open hole section is added, then click the Save button at the bottom of the screen.

On the Setup > Wells > Well Intervals page, a well interval will be created that includes the open hole section and all preceding well intervals from the Well Activity Planner page. If the Casing option is selected, an additional well interval will be created that includes the cased section, the open hole section, and all preceding well intervals.

If the Auto activate well objects based on Well Activity time intervals option is selected, the corresponding well interval on the Setup > Wells > Well Intervals page will be activated when the first activity begins.

Note

On the Setup > Wells > Well Intervals page, well intervals created by the well activity planner will be denoted with an asterisk (*) in front of the well interval name. Well intervals created by the well activity planner can only be deleted on the Setup > MVA Settings > Well Activity Planner page. See "Wellbores".

Adding activities

Anytime the work string or the fluid changes, you should add a new well activity to the well interval. To add an activity to a well interval:

  1. Go to the Setup > MVA Settings > Well Activity Planner page. Locate the well interval you want to add an activity to, and then expand it to show the details for the well interval.
  2. Click the Insert Item button () above the Activities table. A new line will appear in the Activities table for the well interval.
  3. In the Work String column, select the work string you plan to use for the activity.
  4. In the Start Time and Start MD columns, enter the planned start time and depth for the activity.
  5. In the End Time and End MD columns, enter the expected end time and depth for the activity.
  6. In the Fluid column, select the fluid that will be used during the activity.
  7. Update the remaining information about the activity as needed. See "Field definitions".
  8. Click the Save button at the bottom of the screen.

For each activity that you add, a new line will be added to the Activities table for the well interval. If the Auto activate well objects based on Well Activity time intervals option is selected, the corresponding work string on the Setup > Work String page will be activated when the activity begins.

When a work string is activated, the maximum OD of the activity's work string will be validated against the ID of the well interval at the activity's end depth. When you run an analysis, the max OD of each component will now be validated based on the ID of the well interval at the depth where the component is placed.

Field definitions

The table below defines the general options available for well activities on the Setup > MVA Settings > Well Activity Planner page.

Options available for well activities on the Well Activity Planner page
Option Function
Auto activate well objects based on Well Activity time intervals Indicates that automatic well object activation is enabled. When a new well activity begins, the corresponding well interval, work string, and pump fluid will be automatically activated.
Date / Time Cascade Indicates that automatic date and time shifts are enabled. When the start time of a well activity changes, the start and end times for all other well activities will be automatically updated.

The table below defines the properties of the well intervals

Field definitions for well intervals on the Well Activity Planner page
Field Name Definition
Name Name to identify the well interval
Diameter

Hole size of the open hole section
Start MD Depth at the top of the open hole section
End MD

Measured depth at the bottom of the open hole section
Friction Factor

Friction factor for the open hole section

When the Casing option is selected, additional fields for the casing properties will appear.

Field definitions for cased well intervals on the Well Activity Planner tab
Field Name Definition
Name Name to identify the well interval when it's cased
Type

Type of casing or liner

  • Conductor is a short string of large-diameter casing that's used to keep the top of the wellbore open. Also called a boot.
  • Surface is the first string of casing to be set in a well.
  • Intermediate is the string of casing set in a well after the surface casing, and is used to prevent the wellbore from collapsing. Also called protection casing.
  • Production is the last string of casing that is set in a well.
  • Liner is a short casing string, suspended inside an existing casing string.
  • Production Liner is the last string of liner that is set in a well.
Material

Symbol for the material that the casing or liner is composed of and the grade of the material
Outer Diameter

Outside diameter of the casing or liner section
Nominal Weight

Nominal weight of the casing or liner section, calculated by dividing the mass of the section by the length of the section
Inner Diameter

Inside diameter of the casing or liner section
Hanger Depth

Depth at the top of the casing or liner section. This field can only be edited for liner sections.
Setting Depth

Depth at the bottom of the casing or liner section
Friction Factor

Friction factor for the casing or liner section

Using the Well Activity Planner page

On the Setup > MVA Settings > Well Activity Planner page, the toolbar at the top of the screen is used to add new well intervals to the activity planner.

Toolbar buttons on the Well Activity Planner page
Button Function
Adds a new well interval to the well activity planner. See "Adding well intervals".

The toolbar at the top of each interval is used to display or hide the well interval details, as well as to insert new intervals and delete the current interval. When you expand an interval, additional buttons will appear.

Toolbar buttons for intervals
Button Function
Indicates that the well interval is the active well interval. Only appears when the Auto activate well objects based on Well Activity time intervals option is selected.
Adds a new well interval above the corresponding well interval
Deletes the corresponding well interval. The active well interval cannot be deleted.
Adds an activity to the corresponding well interval. See "Adding activities".

The toolbar for each activity is used to delete activities.

Toolbar buttons for activities
Button Function

Deletes the selected activity
