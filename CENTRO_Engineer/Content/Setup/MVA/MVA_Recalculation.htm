About recalculation

On the Setup > MVA Settings > Recalculation page, you can download missing data and re-run calculations for rig states, the torque and drag model, the hydraulics model, and calculations on the Setup > OneClick™ Settings page. This allows you to run calculations when there are gaps in your WITSML data or incorrect parameters were used.

There are two methods for selecting the parameters to use for re-calculating the data:

  • Use parameters from the Well Activity Planner tab. If you have planned the bit and casing runs for the well on the Setup > MVA Settings > Well Activity Planner page, you can use the same settings to recalculate data. This option is useful for recalculating sections of data that use multiple well intervals, work strings, or fluids. See "Recalculating with the Well Activity Planner".
  • Explicitly define the parameters. You can also select a single well interval, work string, and fluid to use for recalculating the data. This option is useful when all of the data you are recalculating uses the same well interval, work string, and fluid. See "Recalculating with explicit parameters".
Tip

When you select the Download & Recalculate option in the Action field, CENTRO® Engineer will automatically detect any gaps of missing data.

Recalculating with the Well Activity Planner

To recalculate data using the Well Activity Planner:

  1. On the Setup > MVA Settings > Well Activity Planner page, define the well intervals and activities for the section of data that you want to recalculate. See "Well activity planner".
  2. Go to the Setup > MVA Settings > Recalculation page.
  3. In the Action field, select which actions you want to occur during recalculation.
    • To download the data from the WITSML server and recalculate data, select the Download & Recalculate option. A table showing gaps of missing data will appear on the right side of the screen.
    • To recalculate the data without downloading WITSML data, select the Recalculate option.
  4. In the From and To fields, enter the start time and end time for the data that you want to recalculate. Use the Calendar button () to select different dates, or click the Well Activities button to select a well activity.
  5. To continue recalculating data in the event of model errors, select the Continue on error option.
  6. Select the calculations that you want to include in the recalculation. See "Field definitions".
  7. In the Parameter Selection field, select the Use Well Activity Planner option. The well activities for the selected time period will appear in a table below the Parameter Selection field.
  8. Click the Calculate button () at the top of the screen.

The recalculation will begin, using the well intervals, work strings, and fluids shown in the table of well activities. If you selected the Download & Recalculate option, WITSML data will also be downloaded from the primary log selected on the Setup > DAQ > WITSML page. The progress of the recalculation will appear above the toolbar at the bottom of the screen.

After recalculation completes, you can publish the recalculated data to the WITSML channel on the Setup > DAQ > WITSML Out page. See "Publishing historical data".

Recalculating with explicit parameters

To recalculate data using explicit parameters:

  1. Go to the Setup > MVA Settings > Recalculation page.
  2. In the Action field, select which actions you want to occur during recalculation.
    • To download the data from the WITSML server and recalculate data, select the Download & Recalculate option. A table showing gaps of missing data will appear on the right side of the screen.
    • To recalculate the data without downloading WITSML data, select the Recalculate option.
  3. In the From and To fields, enter the start time and end time for the data that you want to recalculate. Use the Calendar button () to select different dates, or click the Well Activities button to select a well activity.
  4. To continue recalculating data in the event of model errors, select the Continue on error option.
  5. Select the calculations that you want to include in the recalculation. See "Field definitions".
  6. In the Parameter Selection field, select the Explicitly Selected option.
  7. Select the well interval, work string, and fluid you want to use for the calculations.
  8. Click the Calculate button () at the top of the screen.

The recalculation will begin, using the selected parameters for the well interval, work string, and fluid. If you selected the Download & Recalculate option, WITSML data will also be downloaded from the primary log selected on the Setup > DAQ > WITSML page. The progress of the recalculation will appear above the toolbar at the bottom of the screen.

After recalculation completes, you can publish the recalculated data to the WITSML channel on the Setup > DAQ > WITSML Out page. See "Publishing historical data".

Field definitions

The table below defines the fields on the Setup > MVA Settings > Recalculation page.

Settings for recalculating data
Field Definition
Action

Actions that will occur during recalculation.

  • Download & Recalculate – Downloads data from WITSML server and recalculates the data.
  • Recalculate – Recalculates the data with downloading WITSML data.
From Starting point for recalculating data
To Ending point for recalculating data
Continue on error Option to continue recalculating data if a model error is encountered during the recalculation process
Rig States

Options for including calculations for rig activities and KPIs. Rig states calculations are required, and cannot be excluded from the recalculation.
TnD Model

Options for including calculations for torque and drag model calculations, such as calibrated friction factors, derived loads on bit, and friction factor sensitivity.

  • Recalculate – Torque and drag model will be recalculated for the selected time period.
  • No Change – Torque and drag calculations will not be affected.
Hydraulics Model

Options for including calculations for hydraulics model calculations and ECD management calculations.

  • Recalculate – Hydraulics model will be recalculated for the selected time period.
  • No Change – Hydraulics calculations will not be affected.
OneClick Calculations

Options for including calculations for OneClick™ calculations from the Setup > OneClick™ Settings page.

  • Recalculate – OneClick™ calculations will be recalculated for the selected time period.
  • No Change – OneClick™ calculations will not be affected.
Parameter Selection

Method for selecting the parameters to use for recalculating the data.

  • Use Well Activity Planner – Uses the well intervals, work strings, and fluids defined for the selected time period on the Setup > MVA Settings > Well Activity Planner page. See "Recalculating with the Well Activity Planner".
  • Explicitly Selected – Uses the well interval, work string, and fluid selected in the recalculation settings. See "Recalculating with explicit parameters".
Well Interval Well interval to use for recalculating data. Only available when the Explicitly Selected option is selected in the Parameter Selection field.
Work String Work string to use for recalculating data. Only available when the Explicitly Selected option is selected in the Parameter Selection field.
Fluid Fluid to use for recalculating data. Only available when the Explicitly Selected option is selected in the Parameter Selection field.

Using the Recalculation page

On the Setup > MVA Settings > Recalculation page, the toolbar at the top of the screen is used to start and stop recalculations.

Toolbar buttons on the MVA Settings > Recalculation page
Button Function

Discards all changes made since the last time you saved
Begins the recalculation
Stops the current recalculation. Only available when there is a recalculation in progress.
