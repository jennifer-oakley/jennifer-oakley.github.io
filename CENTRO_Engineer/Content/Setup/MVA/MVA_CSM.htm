About critical speed settings

On the Setup > MVA Settings > CSM Settings page, you can customize the settings for real-time critical speed maps. The critical speeds indicator uses real-time data to predict critical rotations caused by the vibrations of the BHA. When real-time critical speed calculations are enabled, lateral vibrations will be automatically calculated during rotary drilling operations. These calculations are based on the current flow rate, as well as the range of WOB values and top drive speeds specified in the settings.

The frequency that calculations are performed is based on either the distance drilled or the time elapsed since the last time calculations were performed. When one of the conditions in the Model To Run Every field are met, real-time critical speeds will be calculated. The timers for both conditions will be reset.

Note

If you are using an offline license key, all modules related to BHA analysis will be disabled. This includes BHA static and vibration analysis, real-time critical speed maps, and the BHA analysis portion of a OneClick™ analysis.

Note

Critical speeds are only calculated for drillstrings that contain a mud motor, during rotary drilling. Critical speeds are not calculated during any other rig operations or for drillstrings that do not contain a mud motor.

Enabling critical speed calculations

To enable real-time critical speed calculations:

  1. Go to the Setup > MVA Settings > CSM Settings page.
  2. Select the Is Enabled option.
  3. Update the remaining settings as needed. See "Field definitions".
  4. Click the Save Changes button ().

When the current rig operation changes to rotary drilling, CENTRO® Engineer will automatically run the vibration calculations for critical speed maps. To view the critical speed maps, you will need to add the CSM indicator () to a dashboard. See "Adding the critical speeds indicator".

Field definitions

The table below defines the settings required for real-time critical speed maps.

Settings for real-time critical speed maps
Field Definition
Is Enabled Indicates that real-time critical speed maps are enabled
Model Timeout Maximum length of time that calculations are permitted to run. If the calculations do not complete within this time frame, the calculations will be stopped.
Model To Run Every Frequency that calculations are performed, based on either the distance drilled or the time elapsed since the last time calculations were performed. Real-time critical speeds will be calculated when one of these conditions are met, and the timers for both conditions will be reset.
WOB

Range of WOB values used in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
TDS

Range of top drive speeds used in the analysis.

  • Min is the minimum value in the range.
  • Max is the maximum value in the range.
  • Step is the increment used to determine the next point in the range.
