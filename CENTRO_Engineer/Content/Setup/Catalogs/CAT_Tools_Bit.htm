About drill bits

Most drill bits are divided into three basic part sections: the fish neck, the gauge, and the bit face. In the drill bit shown on the right, the fish neck is represented by the green section, the gauge by the purple section, and the bit face by the blue section.

On the Setup > Catalogs > Custom Tools page, you can create custom drill bits. These drill bits can be used when you create a work string on the Setup > Work String page. See "Drill bits".

Creating a custom drill bit

To create a custom drill bit:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a bit to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the Bit option.
  4. In the Tool Size field, select the nominal size of the bit.
  5. In the Description field, enter a brief description of the drill bit.
  6. In the Blade Count field, enter the number of blades on the bit.
  7. In the Nozzles field, enter the number of bit nozzles and the size of each nozzle.
  8. In the Thread Type field, select the type of connection on the bit.
  9. Enter the number of part sections in the Sections field, then enter the dimensions of each section.
  10. Click the Save Changes button () to save your changes.

The drill bit will be added to the selected catalog. The drill bit will also appear in the list of available tools on the Setup > Work String page. See "Work strings".

Properties of custom drill bits

The table below defines the properties for custom bits in CENTRO® Engineer.

Properties of a custom drill bit
Section Field Definition
-- Tool Size

Nominal size of the bit
-- Description Name or description of the bit.
-- Length

Total length of the bit. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the bit.

  • Calculated – The mass and adjusted weight of the bit are calculated, based on the dimensions of the bit. When the dimensions of the bit are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the bit is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the bit. When the dimensions of the bit are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the bit is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the bit. When the dimensions of the bit are updated, the value for mass will not be affected.
-- Mass

Total mass of the bit. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Adjusted Weight

Total adjusted weight of the bit. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the bit
-- Material

Symbol for the material that the bit is composed of and the grade of the material.
-- Blade Count

Number of blades on the bit
-- JSA Area between bit blades, known as the junk slot area
-- TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
-- Efficiency Efficiency of the nozzles on the bit, relative to the maximum possible performance
-- Nozzles

Number of nozzles on the bit. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
-- Nozzle Diameter

Size of each nozzle on the bit. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Tensile Capacity

Tensile strength of the top connection
Part Sections Sections

Number of part sections in the bit.
OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Material

Symbol for the material that the part section is composed of and the grade of the material.
Caption

Description of the part section.
NW

Nominal weight of the part section
Max Contact Force

Maximum allowable contact force for the section.
Max Von Mises Stress

Maximum allowable effective stress for the section.
