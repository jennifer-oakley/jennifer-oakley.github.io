About custom tools

On the Setup > Catalogs > Custom Tools page, you can create custom catalogs and custom tools for work string tools that do not exist in the default CENTRO® Engineer catalog.

The default custom tool catalog is the Simplified catalog. The Simplified catalog is used to create simplified versions of RSS units and drilling motors. Tools that are added to the Simplified catalog have fewer properties than tools added to other custom catalogs. To add a tool to the Simplified catalog, you only need to enter the basic dimensions for the tool. This allows you to create custom tools more quickly and easily. Currently, you can add drilling motors and RSS tools to the Simplified catalog. See "Custom drilling motors" and "Custom RSS units".

To create other types of custom tools, or to create drilling motors and RSS units that are not simplified, you'll need to create a custom catalog. See "Creating custom catalogs". You can create the following types of custom tools in your custom catalog:

  • Bits. In addition to the dimensions of the bit, you'll also need to enter the sizes of the nozzles on the bit. See "Drill bits".
  • Generic Parts. Generic parts, such as drill pipes and HWDP, have uniform pipe sections that describe the part as a whole. Each uniform section has the same OD, ID, and nominal weight across the entire length of the section. The generic parts in the CENTRO® Engineer catalog include drill pipes, drill collars, and HWDP. See "Generic parts".
  • Casing Centralizers. Custom casing centralizers can be added to casing pipe when you create a casing string. See "Casing centralizers".
  • Casing Pipe. Casing pipe is steel pipe that is cemented into an openhole interval. See "Casing pipe".
  • Coiled Tubing. Coiled tubing is a continuous length of small-diameter pipe wound on a spool, then straightened prior to running it into the wellbore. See "Custom coiled tubing".
  • Custom Tools. If none of the part categories in CENTRO® Engineer apply to the work string part you want to add, you can also create a custom part. See "Custom parts".
  • Downhole Control Valves. Downhole control valves are downhole devices that control the flow of drilling fluid. See "Downhole control valves".
  • Drill Collars. Drill collars are thick, heavy-walled tubes used between the bit and the drill pipe to provide a pendulum effect for the drillstring and add weight to the bit. See "Generic parts".
  • Drilling Motors. By default, the drilling motors included in the CENTRO® Engineer catalog do not have any active stabilizers. If you are using a stabilized drilling motor, you'll need to activate the stabilizers. See "Custom drilling motors".
  • Drill Pipe. Drill pipes are described using uniform pipe section. Each uniform section has the same OD, ID, and nominal weight across the entire length of the section. See "Generic parts".
  • Fishing Tools. Fishing tools are tools that are designed to recover equipment and debris from the well, such as a lost bit or a drill collar. See "Fishing tools".
  • Float Collars. Float collars are special coupling devices that are inserted one or two joints above the bottom of a casing string. Float collars also prevent the back-flow of cement during cementing operations. See "Float collars".
  • Float Shoes. Float shoes are short, heavy cylindrical steel sections with rounded bottoms, which are attached to the bottom end of a casing string. The float shoe contains a check valve and functions in the same manner as a float collar, but also serves as guide shoe for the casing. See "Float shoes".
  • Guide Shoes. The CENTRO® Engineer catalog does not include any guide shoes. Before you can create a casing string, you'll need to create a guide shoe. See "Guide shoes".
  • HWDP. Heavyweight drill pipe is drill pipe that has thicker walls and longer tool joints, as well as a wear pad in the middle. HWDP is used between the bit and the drill pipe to reduce the chances of drill pipe fatigue or failure. See "Generic parts".
  • Jars. Jars are percussion tools that are operated mechanically or hydraulically to deliver a heavy hammer blow to objects in the wellbore. Jars are used to free objects during drilling or fishing operations. See "Custom jars".
  • Liner Hanger Systems. A liner hanger is a tool or system used in the oil and gas industry to support and suspend a liner in a well. Liner hangers are slip-type devices designed to connect a liner to the previous casing and provide a seal between the liner and the casing. See "Liner hangers".
  • Mills. Mills are downhole tools that have rough, sharp, and extremely hard cutting surfaces for removing metal by grinding or cutting. See "Mills".
  • Packers. Packers are downhole tools that are used to create a seal between tubing, casing, or drill pipe and the borehole wall or casing. See "Packers".
  • Re-Entry Tools. Re-entry tools are used to create sidetracks in existing wells and maneuver around wellbore obstructions. See "Re-entry tools".
  • POST Tools. POST tools are Weatherford's packoff stage cementing tools, which are specially designed for high-pressure gas and oil wells. See "POST tools".
  • RSS Units. The CENTRO® Engineer catalog includes all Weatherford RSS units. The configuration for an RSS unit includes a pivot stabilizer, a bias unit, and a controller collar. See "Custom RSS units".
  • Sand Control Tools. Sand control tools are tools used to prevent large amounts of sand from entering the wellbore. See "Sand control tools".
  • Slickline Tools. Slickline tools are wireline tools which can be attached to the slickline and lowered into the well for well workover and intervention operations. See "Custom slickline tools".
  • Special Tools. Special tools are circulation tools that are used to control the flow between the pipe and the annulus. See "Special tools".
  • Stage Tools. Stage tools are used to cement multiple sections of the well behind the same casing string or to cement a single section of the well in multiple stages. See "Stage tools".
  • Subs. The CENTRO® Engineer catalog contains several types of rotary subs and circulation subs, including crossover subs, bit subs, saver subs, UBHO subs, float subs, filter subs, and ball lock subs. You can also create custom ball lock subs and AV subs. See "Custom subs".
  • Thru-tubing Motors. Thru-tubing motors are drilling motors that are deployed on either jointed pipe or coiled tubing. They are powered by drilling fluid, and the power section converts hydraulic horsepower from the mud into mechanical horsepower to drive the bit. See "Custom thru-tubing motors".
  • Tubing Accessories. Tubing accessories are tools that are run into the well with a production tubing string. See "Tubing accessories".
  • Tubing Compensation Equipment. Tubing compensation equipment is used to compensate for the vertical and horizontal motion of the tubing string. See "Tubing compensation equipment".
  • Well Services Tools. The tools listed under the Well Services tool type are coiled tubing tools. Coiled tubing can be used for well construction and well intervention services. See "Custom well services tools".
  • Wireline Cable. Wireline cables are armored electrical cables that are used to lower wireline tools into the wellbore, as well as to transmit the data gathered by wireline tools. See "Custom wireline cable".
  • Wireline Tools. Wireline tools are electric logging tools that use an electrical cable to lower the tool into the wellbore and transmit data. See "Custom wireline tools".

Custom catalogs are shared between all of the wells in the CENTRO® Engineer database. You can share custom catalogs by exporting them to a file. See "Exporting tool catalogs".

Creating custom catalogs

To create a new custom catalog:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Click the Add button on the right side of the page.
  3. Click the Add button () at the top of the screen to add custom tools to the catalog.

A new custom catalog will be added to CENTRO® Engineer, and a new numbered tab will appear in the right toolbar. As you add custom tools to the catalog, your changes will be automatically saved. The tools in the catalog will appear in the list of available tools on the Setup > Work String page. See "Work strings".

Using the Custom Tools page

The toolbar at the top of the Custom Tools page is used to add new custom tools. To export catalog data to a CSV file, press the Ctrl + S keys. To select the columns shown in the Custom Tools table, click the Show Columns button () in the first column of the table.
