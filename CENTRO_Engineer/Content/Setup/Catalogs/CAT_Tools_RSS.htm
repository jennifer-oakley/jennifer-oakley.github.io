About custom RSS units

The specifications for most Weatherford RSS units are included in the database for CENTRO® Engineer. When you select an RSS unit, the specifications will appear in the Drillstring Component panel. Most of these properties are provided for your reference, and can't be edited.

On the Setup > Catalogs > Custom Tools page, you can also create custom RSS units. These RSS units can be used when you create a work string on the Setup > Work String page. See "RSS units".

Custom RSS units can be added to your own custom catalog or to the Simplified catalog. The Simplified catalog contains simplified versions of tools, with fewer properties than tools added to other custom catalogs. To add an RSS unit to the Simplified catalog, you only need to enter the basic dimensions for the tool.

Creating a custom RSS unit

To create a new RSS unit:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a RSS unit to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the RSS option.
  4. In the Type field, select the type of RSS unit.
  5. In the Tool Size field, select the nominal size of the RSS unit.
  6. In the Description field, enter a brief description of the RSS unit.
  7. Select the top and bottom connections for the RSS unit.
  8. Enter the dimensions of the part sections for the RSS unit.
  9. Update the remaining properties of the RSS unit as needed. See "Properties of custom RSS units".
  10. Click the Save Changes button () to save your changes.

The RSS unit will be added to the selected catalog. The RSS unit will also appear in the list of available tools on the Setup > Work String page. See "Work strings".

Properties of custom RSS units

The sections below define the properties of custom RSS units.

Properties of custom RSS units
Section Field Definition
-- Tool Size

Nominal size of the RSS unit
-- Description

Name or description of the RSS unit
-- Length

Total length of the RSS unit. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the RSS unit.

  • Calculated – The mass and adjusted weight of the RSS unit are calculated, based on the dimensions of the RSS unit. When the dimensions of the RSS unit are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the RSS unit is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the RSS unit. When the dimensions of the RSS unit are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the RSS unit is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the RSS unit. When the dimensions of the RSS unit are updated, the value for mass will not be affected.
-- Mass

Total mass of the RSS unit. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Adjusted Weight

Total adjusted weight of the RSS unit. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the RSS unit
-- Material

Symbol for the material that the RSS unit is composed of and the grade of the material.
-- Deflection Method Method used to steer the RSS unit. Not available for RSS units created in the Simplified catalog.
-- Bend Angle Current bend angle setting for the RSS unit. Only available for point-the-bit RSS units. Not available for RSS units created in the Simplified catalog.
-- Screen Restrictor Size Diameter of the screen on the restrictor. If there is no restrictor on the RSS, select the No Restrictor option. Only available for push-the-bit RSS units. Not available for RSS units created in the Simplified catalog.
-- Pad Count Number of pads on the outside of the RSS unit. Only available for push-the-bit RSS units. Not available for RSS units created in the Simplified catalog.
-- Pad Nozzle Size Diameter of the nozzles on the pads. Only available for push-the-bit RSS units. Not available for RSS units created in the Simplified catalog.
Sensors (checkbox) Indicates that a sensor is present on the RSS unit. When selected, additional fields will appear.
Sensors Number of sensors on the RSS unit
Sensor Name Description of the sensor
Measurement Type Type of measurements taken by the sensor

Dist. From Bottom

 Distance between the sensor and the bottom of the RSS unit
-- Connection Configuration

Configuration of the connections on the RSS unit.

  • Box-Box – Both connections are box connections.
  • Pin-Pin – Both connections are pin connections.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Tensile Capacity

Tensile strength of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Tensile Capacity

Tensile strength of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Stabilizer (check box) Indicates that a stabilizer is included in the part section
Hole Size Option

Method for determining the hole size for junk slot area calculations.

  • Use Bit OD – Uses the outside diameter of the drill bit for the work string to determine the hole size.
  • User Override – Uses a custom value for the hole size.
Hole Size

Diameter of the hole that the work string will be used in.

  • When the Use Bit OD option is selected, this value is read-only and cannot be edited. The value shown will be the bit OD for the work string. If the bit OD changes, the Hole Size value will be automatically updated.
  • When the User Override option is selected, this value will be unlocked for editing.
Blade OD

Outside diameter of the stabilizer blades
Blade Width

Width of the each stabilizer blade
Blade Angle

Angle between the blade and the stabilizer body
Blade Count

Number of blades on the RSS unit
Junk Slot Area

Area between stabilizer blades
