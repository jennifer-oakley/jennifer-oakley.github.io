About fishing tools

Fishing tools are tools that are designed to recover equipment and debris from the well, such as a lost bit or a drill collar. You can create several types of custom fishing tools:

  • Basket – A junk basket is a fishing tool that is designed to remove junk from the bottom of the borehole. Junk can be any form of debris or other objects, such as balls, darts, plugs, bit nozzles. Available basket types include boot baskets, globe type junk baskets, kangaroo baskets, reverse circulation junk baskets (RCJB), and Venturi baskets. Venturi baskets are also known as jet baskets.
  • Casing Repair – Casing repair tools are used to repair damaged casing. There are two types of casing repair tool available in the catalog: casing swages and external casing patches. Casing swages restore dented, buckled, or collapsed casing to its original shape and diameter, using a tapered anvil to drive the damaged casing back to its original shape. External casing patches are used to re-line damaged casing or tubing, without reducing the ID of the casing.
  • Clean Out – Clean out tools are used to remove wellbore material, such as sand, scale, and other debris from the wellbore. Available clean out tools include casing scrapers, circulation subs, string magnets, and casing brushes. Casing scrapers use a blade assembly to remove scale and debris from the inside of a casing string. Circulation subs are used to allow a higher circulation rate by opening a path to the annulus in the upper portion of the work string. String magnets are used to retrieve small pieces of magnetic debris.
  • Cutter – Cutters are used to cut pipe that is stuck in the wellbore. Cutter tools contain metal-cutting knives that cut the pipe, so that the severed portion can be brought back to the surface. Available tools include external cutters and internal cutters. External cutters are lowered over the outside of the pipe, and internal cutters are lowered into the inside of the pipe.
  • External Catch – External catch tools are used to recover tubular equipment from the wellbore by connecting to the outside diameter of the fish, either by using a grapple or by connecting directly to the outside surface. Examples of external catch tools include overshots, basket grapples, box taps, and die collars.
  • Internal Catch – Internal catch tools are used to recover tubular equipment from the wellbore by connecting to the inside diameter of the fish, either by using a grapple or by connecting directly to the inside surface. Examples of internal catch tools include taper taps, grapple releasing spears, and packer retrieval tools. Available types of internal cutters include stop subs, rope spears, and releasing spears.
  • Washover – Washover tools are used to free pipe that is stuck in the wellbore, by rotating a washover assembly around the stuck pipe. Washover pipe is connected to a rotary sub, which cuts away obstructions blocking the pipe's movement and circulates them out of the wellbore. Available washover tools include drive subs, washover shoes, and wash pipe.

Creating fishing tools

To create a new fishing tool:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a fishing tool to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the Fishing option.
  4. In the Type field and Tool Configuration field, select the type of fishing tool.
  5. In the Tool Size field, select the nominal size of the fishing tool.
  6. In the Description field, enter a brief description of the fishing tool.
  7. Select the top and bottom connections for the fishing tool.
  8. Enter the dimensions of each part section.
  9. Update the remaining properties of the fishing tool as needed. See "Properties of fishing tools".
  10. Click the Save Changes button () to save your changes.

The fishing tool will be added to the selected catalog. The fishing tool will also appear in the list of available tools on the Setup > Work String page. See "Work strings".

Properties of fishing tools

The sections below define the properties of custom fishing tools.

Properties of custom fishing tools
Section Field Definition
-- Type Type of fishing tool. See "About fishing tools".
-- Tool Configuration Configuration of the fishing tool. Not available for external catch, internal catch, or rope spear tools.
-- Tool Size

Nominal size of the fishing tool
-- Description

Name or description of the fishing tool
-- Length

Total length of the fishing tool. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the fishing tool.

  • Calculated – The mass and adjusted weight of the fishing tool are calculated, based on the dimensions of the fishing tool. When the dimensions of the fishing tool are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the fishing tool is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the fishing tool. When the dimensions of the fishing tool are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the fishing tool is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the fishing tool. When the dimensions of the fishing tool are updated, the value for mass will not be affected.
-- Adjusted Weight

Total adjusted weight of the fishing tool. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Mass

Total mass of the fishing tool. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the fishing tool
-- Material

Symbol for the material that the fishing tool is composed of and the grade of the material.
Nozzles/Hydraulic (checkbox) Indicates that nozzles are present on the fishing tool. Only available for Venturi baskets, circulation subs, internal cutters, and rope spears. When selected, additional fields will appear.
TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
Efficiency

Efficiency of the nozzles on the fishing tool, relative to the maximum possible performance
Nozzles

Number of nozzles on the fishing tool. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
Nozzle Diameter

Size of each nozzle on the fishing tool. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Junk Slot Type Type of junk slot. Only available for washover shoes.
Slot Count Number of junk slots on the shoe. Only available for washover shoes.
Slot Width Width of the junk slot. Only available for washover shoes.
Slot ID Inner diameter of the junk slot. Only available for washover shoes.
-- Connection Configuration

Configuration of the connections on the fishing tool.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Junk Slot/Blade (check box) Indicates that a stabilizer is included in the fishing tool. Only available for globe baskets, RCJB baskets, and internal cutters. When selected, additional fields will appear.
Blade OD

Outside diameter of the stabilizer blades
Blade Width

Width of the each stabilizer blade
Blade Angle

Angle between the blade and the stabilizer body
Blade Count

Number of blades on the stabilizer
Junk Slot Area

Area between stabilizer blades
