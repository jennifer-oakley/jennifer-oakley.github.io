About guide shoes

Guide shoes are tools that guide the casing into the wellbore. On the Setup > Catalogs > Custom Tools page, you can add guide shoes to a custom catalog. These guide shoes can be used when you create a casing string on the Setup > Work String page. See "Work strings" and "Casing string design".

Creating a guide shoe

To create a new guide shoe:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a guide shoe to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the Guide Shoe option.
  4. In the Tool Size field, select the nominal size of the guide shoe.
  5. In the Description field, enter a brief description of the guide shoe.
  6. In the TFA field, enter the total flow area through the guide shoe.
  7. In the Thread Type field, select the type of connection on the guide shoe.
  8. Enter the number of part sections in the Sections field, then enter the dimensions of each section.
  9. Click the Save Changes button () to save your changes.

The guide shoe will be added to the selected catalog. The guide shoe will also appear in the list of available tools on the Setup > Work String page.See "Work strings" and "Casing string design".

Properties of guide shoes

The table below defines the properties of custom guide shoes.

Properties of custom guide shoes
Section Field Definition
-- Tool Size

Nominal size of the guide shoe
-- Description

Name or description of the guide shoe
-- Length

Total length of the guide shoe. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the guide shoe.

  • Calculated – The mass and adjusted weight of the guide shoe are calculated, based on the dimensions of the guide shoe. When the dimensions of the guide shoe are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the guide shoe is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the guide shoe. When the dimensions of the guide shoe are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the guide shoe is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the guide shoe. When the dimensions of the guide shoe are updated, the value for mass will not be affected.
-- Adjusted Weight

Total adjusted weight of the guide shoe. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Mass

Total mass of the guide shoe. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the guide shoe
-- Material

Symbol for the material that the guide shoe is composed of and the grade of the material.
-- TFA

Total flow area
-- Efficiency Efficiency of the nozzles on the guide shoe, relative to the maximum possible performance
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Tensile Capacity

Tensile strength of the top connection
Part Sections Sections

Number of part sections in the guide shoe.
OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Material

Symbol for the material that the part section is composed of and the grade of the material.
Caption

Description of the part section.
NW

Nominal weight of the part section
Max Contact Force

Maximum allowable contact force for the section.
Max Von Mises Stress

Maximum allowable effective stress for the section.
