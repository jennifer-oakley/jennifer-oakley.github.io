About custom connections

On the Setup > Catalogs > Custom Connections page, you can create custom connections that do not exist in the default CENTRO® Engineer catalog. When you create a custom connection, you'll also need to include the tool size and nominal weight for the tool that the connection is used with. If the connection can be used with multiple tool sizes or nominal weights, you'll need to create an additional custom connection for each combination of tool size and nominal weight.

Custom connections are included in the list of available connection types for custom tools on the Setup > Catalogs > Custom Tools page. On the Custom Tools page, the names of custom API connections use the following format: [Tool Size] [Nominal Weight]# [Connection Name], e.g. 2 3/8" 6.65# GPDS26. For non-API connections, the name will also include the manufacturer name.

The custom connection catalog is shared between all of the wells in the CENTRO® Engineer database. You can also share the custom connection catalog by exporting it to a file. See "Exporting custom connections".

Note

Custom connections are not available for bits, drilling motors, or RSS units.

Adding custom connections

To add a custom connection:

  1. Go to the Custom Connections page in the Setup > Catalogs section.
  2. Click the Insert Item button ().
  3. In the panel that appears, go to the Parent Pipe Info section and enter the tool size, nominal weight, and material for the tool that the connection is used with.
  4. Go to the Connection Information section and enter the name, manufacturer, and dimensions of the connection.
  5. Go the Limits section and enter the connection's burst, collapse, tension, compression, and makeup torque ratings.
  6. Click the Save Changes button () to add the connection to the catalog.

The new connection will be added to the catalog of custom connections, and it will be listed as one of the available connection types on the Setup > Catalogs > Custom Tools page. In the Thread Type field on the Custom Tools page, the connection name will use the following format: [Tool Size] [Nominal Weight]# [Connection Name], e.g. 2 3/8" 6.65# GPDS26.

Field definitions

The table below defines the properties for custom connections.

Nominal weight of the pipe the connection is used with. This value can't be edited after the connection is created.

Symbol for the material that the tool the connection is used with is composed of and the grade of the material.

NW

Limits As % Indicates that limits are displayed as a percentage, rather than pressure. When selected, the existing values will be converted to a percentage value.

Using the Custom Connections page

To export custom connections data to a CSV file, press the Ctrl + S keys. The toolbar at the top of the screen is used for adding custom connections. To select the columns shown in the Custom Connections table, click the Show Columns button () in the first column of the table.

Toolbar buttons on the Setup > Catalogs > Custom Connections page
Button Function
Adds a new custom connection. See "Adding custom connections".
Refreshes the catalog of custom connections from the database

The toolbar for each custom connection is used to edit, copy, and delete custom connections.

Toolbar buttons for custom connections
Button Function

Opens the properties of the selected custom connection

Creates a copy of the selected custom connection

Deletes the selected custom connection
