About mills

Mills are downhole tools that have rough, sharp, and extremely hard cutting surfaces for removing metal by grinding or cutting. You can create several types of custom mills:

  • Bottom Mill – Bottom mills are mills that are attached to the end of the work string. There are two types of bottom mills: junk mills and taper mills. Junk mills are used to perform heavy-duty milling jobs. Taper mills are used to ream out collapsed casing and liner section, as well as to deburr whipstock windows and casing.
  • Packer Mill – Packer mills are used to mill out production packers and bridge plugs. The blades mill over the slip section to disengage the packer. Once the slip have been removed, the packer can be retrieved using a retrieval tool.
  • Pilot Mill – Pilot mills are designed for milling casing, liners, rotary shoes or large-ID tubing. Pilot mills have an extended pilot section that inserted into the drill pipe or tubing that is being milled. The pilot end of the mill guides the mill to the top of the pipe and centers it, preventing the mill from bypassing the pipe.
  • Section Mill – Section mills are used to cut and mill complete sections of casing.
  • String Mill – String mills can be run throughout the work string. There are two types of string mills: standard string mills and watermelon mills. Standard string mills are used to clean out damaged casing and liner, as well as to remove keyseats. Watermelon mills are used to extend the window of a whipstock.

Creating mills

To create a new mill:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a mill to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the Mill option.
  4. In the Type field and Tool Configuration field, select the type of mill.
  5. In the Tool Size field, select the nominal size of the mill.
  6. In the Description field, enter a brief description of the mill.
  7. Select the top and bottom connections for the mill.
  8. Enter the dimensions of each part section.
  9. Update the remaining properties of the mill as needed. See "Properties of mills".
  10. Click the Save Changes button () to save your changes.

The mill will be added to the selected catalog. The mill will also appear in the list of available tools on the Setup > Work String page. See "Work strings".

Properties of mills

The sections below define the properties of custom mills.

Properties of custom mills
Section Field Definition
-- Type Type of mill. See "About mills".
-- Tool Configuration Configuration of the mill. Only available for bottom mills and string mills.
-- Tool Size

Nominal size of the mill
-- Description

Name or description of the mill
-- Length

Total length of the mill. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the mill.

  • Calculated – The mass and adjusted weight of the mill are calculated, based on the dimensions of the mill. When the dimensions of the mill are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the mill is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the mill. When the dimensions of the mill are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the mill is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the mill. When the dimensions of the mill are updated, the value for mass will not be affected.
-- Mass

Total mass of the mill. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Adjusted Weight

Total adjusted weight of the mill. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the mill
-- Material

Symbol for the material that the mill is composed of and the grade of the material.
Nozzles (checkbox) Indicates that nozzles are present on the mill. Only available for packer mills, section mills, and pilot mills. When selected, additional fields will appear.
-- TFA Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.. Not available for string mills.
-- Efficiency Efficiency of the nozzles on the mill, relative to the maximum possible performance. Not available for string mills.
-- Nozzles Number of nozzles on the mill. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.. Not available for string mills.
-- Nozzle Diameter Size of each nozzle on the mill. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.. Not available for string mills.
Stabilizer (Fishneck) (checkbox) Indicates that a stabilizer is included on the fishneck of the mill. Not available for bottom mills and packer mills. When selected, additional fields will appear.
Blade OD

Outside diameter of the stabilizer blades
Blade Count

Number of blades on the stabilizer
Blade Width

Width of the each stabilizer blade
Blade Length

Length of each stabilizer blade
Blade Angle

Angle between the blade and the stabilizer body
-- Connection Configuration

Configuration of the connections on the mill.. The list of available configuration options is automatically filtered, based on the selected type of mill.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
Stabilizer/Blade (checkbox) Indicates that a stabilizer is included on the mill.
Blade OD

Outside diameter of the stabilizer blades
Blade Width

Width of the each stabilizer blade
Blade Angle

Angle between the blade and the stabilizer body
Blade Count

Number of blades on the stabilizer
Junk Slot Area

Area between stabilizer blades
