About custom drilling motors

Drilling motors are usually located above the bit or above the RSS in a motorized RSS system. They are powered by drilling fluid, and turn the bit while the drillstring remains fixed or rotates at a different speed.

The specifications for all Weatherford mud motors listed in the Mud-Lubricated Drilling Motors Handbook are included in the default Weatherford catalog. When you select a drilling motor, the motor specifications will appear in the Work String Component panel. Most of these properties are provided for your reference, and can't be edited. However, you will need to add information about the motor configuration.

On the Setup > Catalogs > Custom Tools page, you can also create custom drilling motors. These drilling motors can be used when you create a work string on the Setup > Work String page. See "Drilling motors".

Custom drilling motors can be added to your own custom catalog or to the Simplified catalog. The Simplified catalog contains simplified versions of tools, with fewer properties than tools added to other custom catalogs. To add an drilling motor to the Simplified catalog, you only need to enter the basic dimensions for the tool.

Creating a custom drilling motor

To create a new drilling motor:

  1. Go to the Setup > Catalogs > Custom Tools page.
  2. Select the catalog that you want to add a drilling motor to, and click the Insert Item button ().
  3. In the panel that appears on the right, go to the Tool field and select the Drilling Motor option.
  4. In the Type field, select the type of drilling motor.
  5. In the Tool Size field, select the nominal size of the drilling motor.
  6. In the Description field, enter a brief description of the drilling motor.
  7. Enter the dimensions of the power section, rotor, and top sub.
  8. In the Stabilizers section, select the stabilizers, centralizers, and kick pads that are active on the drilling motor.
  9. Select the top and bottom connections for the drilling motor.
  10. Update the remaining properties of the drilling motor as needed. See "Properties of custom drilling motors".
  11. Click the Save Changes button () to save your changes.

The drilling motor will be added to the selected catalog. The drilling motor will also appear in the list of available tools on the Setup > Work String page. See "Work strings".

Properties of custom drilling motors

The table below defines the properties for custom drilling motors in CENTRO® Engineer.

Properties of custom drilling motors
Section Field Definition
--

Tool Size

Nominal size of the drilling motor
-- Description

Name or description of the drilling motor
-- Length

Total length of the drilling motor. This value is calculated automatically.
-- Tool Weight Option

Method for entering the total mass of the drilling motor.

  • Calculated – The mass and adjusted weight of the drilling motor are calculated, based on the dimensions of the drilling motor. When the dimensions of the drilling motor are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the drilling motor is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the drilling motor. When the dimensions of the drilling motor are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the drilling motor is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the drilling motor. When the dimensions of the drilling motor are updated, the value for mass will not be affected.
-- Mass

Total mass of the drilling motor. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
-- Adjusted Weight

Total adjusted weight of the drilling motor. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
-- NW

Nominal weight of the drilling motor
-- Material

Symbol for the material that the drilling motor is composed of and the grade of the material. This is a read-only value, populated from the CENTRO® Engineer catalog.
-- Adjustable Setting Current bend angle setting. You can select one of the standard bend angles, or you can manually enter a value. Not available for drilling motors created in the Simplified catalog.
-- Operating Differential Pressure Pressure loss through the motor's power section while the bit is off-bottom. Not available for drilling motors created in the Simplified catalog.
-- Speed Ratio Ratio between the motor's maximum speed and maximum flow rate. Not available for drilling motors created in the Simplified catalog.
-- Torque Ratio Ratio between the maximum torque and the maximum differential pressure. Not available for drilling motors created in the Simplified catalog.
Hydraulics Pressure Drop Coefficients Hyd. Model

Method for calculating pressure loss through the drilling motor
Pressure Drop

Pressure loss limits that apply to the drilling motor. Only available for linear pressure loss calculations.

Flow Rate

Flow rate limits that apply to the drilling motor. Only available for linear pressure loss calculations.

Coefficient A

Fluid density exponent, based on the flow characteristics of the drilling motor. Only available for Power Law pressure loss calculations.

Coefficient B

Scaling constant, based on the flow characteristics of the drilling motor. Only available for Power Law pressure loss calculations.

Coefficient C

Flow rate exponent, based on the flow characteristics of the drilling motor. Only available for Power Law pressure loss calculations.
Bearing Pack Nominal OD Nominal outer diameter of the bearing pack section. Not available for drilling motors created in the Simplified catalog.
Length Total length of the bearing pack section, from the bit box to the bottom of the power section. Not available for drilling motors created in the Simplified catalog.
Bit-to-Bend Length Length from the bit box to the top of the adjustable housing. Not available for drilling motors created in the Simplified catalog.
Max Contact Force Maximum allowable contact force for the section. Not available for drilling motors created in the Simplified catalog.
Max Von Mises Stress Maximum allowable effective stress for the section. Not available for drilling motors created in the Simplified catalog.
Bearing Pack Bypass (checkbox) Indicates that there is flow around the bearing pack. Not available for drilling motors created in the Simplified catalog.
TFA Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value. Not available for drilling motors created in the Simplified catalog.
Efficiency Efficiency of the nozzles on the drilling motor, relative to the maximum possible performance. Not available for drilling motors created in the Simplified catalog.
Nozzles Number of nozzles on the drilling motor. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles. Not available for drilling motors created in the Simplified catalog.
Nozzle Diameter Size of each nozzle on the drilling motor. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field. Not available for drilling motors created in the Simplified catalog.
Power Section Outer Diameter Outer diameter of the power section. Not available for drilling motors created in the Simplified catalog.
Length Length from the bottom of the power section to the top of the mud motor. Not available for drilling motors created in the Simplified catalog.
Lobe Configuration Configuration of the lobes on the rotor and the stator. The first number is the number of lobes on the rotor, and the second number is the number of lobes on the stator. Not available for drilling motors created in the Simplified catalog.
Stages Number of stages for the power section Not available for drilling motors created in the Simplified catalog.
Min Flow Rate Minimum recommended flow rate. Not available for drilling motors created in the Simplified catalog.
Max Flow Rate Maximum recommended flow rate. Not available for drilling motors created in the Simplified catalog.
Max Differential Pressure Maximum allowable pressure loss through the power section. Not available for drilling motors created in the Simplified catalog.
Max Torque Amount of torque at the maximum allowable differential pressure. Not available for drilling motors created in the Simplified catalog.
Performance Specification Performance Chart Dynamic chart showing differential pressure and torque at various rotation speeds and flow rates. To view the chart, click the Expand button () on the left side of the Performance Chart field. Not available for drilling motors created in the Simplified catalog.
Rotor Mass of Rotor Mass of the rotor in the power section. This value is automatically calculated from the nominal OD of the power section. However, you can also enter your own value. Not available for drilling motors created in the Simplified catalog.
Eccentricity Eccentricity of the rotor in the power section. This value is automatically calculated from the nominal OD of the power section. However, you can also enter your own value. Not available for drilling motors created in the Simplified catalog.
Max Contact Force Maximum allowable contact force for the section. Not available for drilling motors created in the Simplified catalog.
Max Von Mises Stress Maximum allowable effective stress for the section. Not available for drilling motors created in the Simplified catalog.
Top Sub Nominal OD Outside diameter of the top sub. Not available for drilling motors created in the Simplified catalog.
Length Length of the top sub. Not available for drilling motors created in the Simplified catalog.
Max Contact Force Maximum allowable contact force for the section. Not available for drilling motors created in the Simplified catalog.
Max Von Mises Stress Maximum allowable effective stress for the section. Not available for drilling motors created in the Simplified catalog.
Stabilizers (checkbox) Indicates that the stabilizer, centralizer, or kick pad is included on the drilling motor. Not available for drilling motors created in the Simplified catalog.
Blade OD Outside diameter of the stabilizer blades. Not available for drilling motors created in the Simplified catalog.
Body OD Outside diameter of the stabilizer body. Not available for drilling motors created in the Simplified catalog.
Blade Count Number of blades on the stabilizer. Not available for drilling motors created in the Simplified catalog.
Blade Width Width of the each stabilizer blade. Not available for drilling motors created in the Simplified catalog.
Blade Length Length of each stabilizer blade. Not available for drilling motors created in the Simplified catalog.
Blade Angle Angle between the blade and the stabilizer body. Not available for drilling motors created in the Simplified catalog.
-- Connection Configuration

Configuration of the connections on the drilling motor.

  • Box-Box – Both connections are box connections.
  • Pin-Pin – Both connections are pin connections.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Tensile Capacity

Tensile strength of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Tensile Capacity

Tensile strength of the bottom connection
