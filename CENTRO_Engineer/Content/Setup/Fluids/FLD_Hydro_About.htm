About hydrocarbons

Hydrocarbons are organic compounds of hydrogen and carbon, whose densities, boiling points, and freezing points increase as their molecular weights increase. Although composed of only two elements, hydrocarbons exist in a variety of compounds because of the strong affinity of the carbon atom for other atoms and for itself. The smallest molecules of hydrocarbons are gaseous, the largest are solids.

Adding hydrocarbons

To add a hydrocarbon:

  1. Go to the Setup > Fluids > Hydrocarbons page.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of fluids.
  3. In the Type field, select the type of hydrocarbon fluid.
  4. In the Fill Color field, select a color to represent the hydrocarbon fluid in the Well View pane.
  5. In the Density field, enter the density of the hydrocarbon when it enters the wellbore.
  6. In the Rheology Input menu, select the rheological properties tab to be used to perform the calculations. Only available in Plan Mode.
  7. Use the tabs on the right side of the screen to enter the hydrocarbon's rheological properties. See "Rheological properties".
  8. Click the Save Changes button ().

The hydrocarbon will be stored in CENTRO® Engineer's database. After you save the hydrocarbon, you will not be able to edit the fluid properties.

Tip

To change the name of the hydrocarbon, go to the toolbar on the right side of the screen and click on the tab for the hydrocarbon. In the box that appears, enter a new name for the hydrocarbon and press the Enter key.

Importing fluids

Before you can import fluid reports, you'll need to create a WITSML channel on the Setup > DAQ > WITSML page. Fluid reports can only be imported from the active WITSML channel. See "WITSML channels".

To import a fluid:

  1. Go to the Setup > Fluids > Hydrocarbons page.
  2. Click the Import From WITSML button ()at the top of the screen.
  3. Click Yes on the warning message that appears.

A new fluid will be created, using the data from the fluid report on the WITSML server.

Note

Only the fluid weight and fluid type will be imported. The remaining properties must be manually entered.

Field definitions

The table below defines the basic properties for hydrocarbon fluids on the Setup > Fluids > Hydrocarbons page.

Field definitions for hydrocarbon properties
Field Definition
Type

Type of hydrocarbon fluid
Fill Color

Color used to represent the hydrocarbon in the Well View pane
Density

Density of the hydrocarbon when it enters the wellbore
Friction Multiplier

Multiplier used to account for friction reducers added to the hydrocarbon. A friction multiplier of 1 indicates that there are no friction reducers added to the hydrocarbon. Smaller multipliers indicate an increased reduction in friction. A typical value would be between 0.9 and 0.8. The friction multiplier is unlikely to be less than 0.6.
Rheology Input Rheological properties method to use for calculations. Only available in Plan Mode
