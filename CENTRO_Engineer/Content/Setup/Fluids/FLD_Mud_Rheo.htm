Rheological properties

In CENTRO® Engineer, the mud's rheology can be calculated from dial readings or rheological properties, such as plastic viscosity and yield point. There are four ways to define the rheology of a mud:

  • Dial Readings. This method uses viscometer readings to define rheological properties. You can use this method for any mud. See "Dial readings".
  • PV YP LSYP. This method is best for Bingham plastic fluids, which have a yield point that must be exceeded before the mudwill flow. See "PV YP LSYP properties"..
  • H-B Direct. This method uses parameters for the Herschel-Bulkley rheology model, which can be used for Bingham plastic fluids or Power law fluids. See "H-B Direct properties"..
  • PVT Dial Readings. This method uses multiple sets of dial readings to define rheological properties. You can use this method for any mud. See "PVT dial readings".
  • Temperature Dependent Rheology. This method calculates the dial readings for the mud automatically, based on the mud weight and temperature. See "Temperature dependent rheology".

You can only use one of these methods to enter rheological properties. When you save the mud, only the information in the active tab will be saved. The remaining tabs will be hidden for that fluid.

Warning

In Real Time mode, switching between the Dial Readings, PV YP LSYP, H-B Direct, and Dial Readings (PVT) tabs may cause you to lose data. When you select another tab, the data on the previous tab is reset to the default values. Any data that you've entered will be lost.

In Plan mode and MPD mode, the data you enter will be saved even when switching between tabs.
