About mud properties

When you create a new mud, the basic fluid properties are entered in the left side of the screen. The basic fluid properties listed in the Parameters section include mud type, mud weight, temperatures, and the fluid composition. If the mud report includes additional information, such as salt type and salinity, you can enter this information in the Advanced Properties section. If the mud report does not include this information, you can simply use the default values.

Adding muds

To add a drilling mud:

  1. Go to the Setup > Fluids > Mud page.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of fluids.
  3. In the Mud Type field, select the base used for the mud.
  4. In the Fill Color field, select a color to represent the mud in the Well View pane.
  5. In the Mud Weight In field, enter the density of the mud when it enters the wellbore.
  6. In the Mud Inlet Temperature field, enter the inlet temperature of the mud in the suction tank.
  7. In the Ref. Temperature field, enter the temperature at which the mud weight was measured.
  8. Enter the composition of the mud in the Retort Oil and Retort Water fields.
  9. In the Rheology Input menu, select the rheological properties tab to be used to perform the calculations. Only available in Plan Mode.
  10. If available, enter the remaining information under the Advanced Properties heading. See "Field definitions".
  11. Use the tabs in the bottom portion of the screen to enter the mud's rheological properties. See "Rheological properties".
  12. Click the Save Changes button ().

The mud will be stored in CENTRO® Engineer's database. When you change the pump fluid, you'll be able to select the new mud under the Mud utility. See "Mud utility". When you run an analysis, you'll also be able to select the new mud in the analysis settings. However, you will not be able to edit the fluid properties or dial readings in Real Time mode.

Tip

After you save the fluid, you can use the Mud button () in the utility bar to set the fluid as the pump fluid or boost line fluid. See "Mud utility".

Tip

To change the name of the mud, go to the toolbar on the right side of the screen and click on the tab for the mud. In the box that appears, enter a new name for the mud and press the Enter key.

Field definitions

The table below defines the basic properties for muds on the Setup > Fluids > Mud page.

Field definitions for mud properties
Section Field Definition
Parameters Mud Type

Base fluid used for the mud

  • Mineral Oil Base is a fluid in which the continuous phase is mineral oil.
  • Oil Base is a fluid in which the continuous phase is diesel oil.
  • Synthetic Base is a fluid in which the continuous phase is a synthetic fluid, such as a polyether or a linear alkyl benzene.
  • Water Base is a fluid in which the continuous phase is water or saltwater, such as fresh water, potassium, or silicate.
Fill Color

Color used to represent the mud in the Well View pane
Mud Weight In

Density of the mud when it enters the wellbore
Mud Inlet Temperature Inlet temperature of the fluid in the suction tank
Ref. Temperature

Temperature at which the mud weight was measured
Friction Multiplier

Multiplier used to account for friction reducers added to the mud. A friction multiplier of 1 indicates that there are no friction reducers added to the mud. Smaller multipliers indicate an increased reduction in friction. A typical value would be between 0.9 and 0.8. The friction multiplier is unlikely to be less than 0.6.
Retort Oil Percentage of oil in the fluid when it enters the wellbore. Not available for water-based fluids.
Retort Water Percentage of water in the fluid when it enters the wellbore.
Solids Percentage of solids in the fluid when it enters the wellbore. Read-only value, automatically calculated from the Retort Oil and Retort Water values.
Rheology Input Rheological properties tab to use for calculations. Only available in Plan Mode
Advanced Properties Salt Type Type of salt used in the fluid
Salinity Concentration of the salt in the fluid when it enters the wellbore. Salinity should be entered as a percentage of the weight of the fluid.
Conductivity Conductivity of the fluid. The default value is calculated based on the fluid composition.
Specific Heat Specific heat of the fluid. The default value is calculated based on the fluid composition.
API Fluid Loss Amount of water loss that occurs in 30 minutes at 100 psi. Only available for water-based fluids.
HTHP Fluid Loss Amount of water loss that occurs in 30 minutes at 500 psi. Not available for water-based fluids.
API Cake Thickness of the filter cake. Only available for water-based fluids.
pH Acidity of the fluid. Only available for water-based fluids.
Proppants Type of particles mixed with the fluid
Gas Type Type of gas mixed with the fluid
Gas Density Density of the gas mixed with the fluid, automatically determined by the selected gas type. Not available when the None option is selected in the Gas Type field.
