About PVT dial readings

The Dial Readings (PVT) tab uses dial readings taken at multiple temperatures to calculate the rheological properties of a fluid. Dial readings can be used to define the rheology of any mud, without calculating additional values such as PV or YP. Viscometer readings are used to calculate the mud's plastic viscosity, yield point, yield stress, consistency factor, and flow index. Gel strength readings are used to determine the viscosity of the fluid when the fluid isn't flowing.

Tip

You can enter up to three sets of dial readings taken at different temperatures.

Entering PVT dial readings

To enter PVT dial readings properties for a new mud:

  1. Go to the Setup > Fluids > Mud page to add a new mud. See "Adding muds".
  2. On the Dial Readings (PVT) tab, go to the first row in the table.
  3. In the T column, enter the temperature at which the viscometer readings were taken.
  4. In the θ600 - θ3 columns, enter the viscometer readings taken at each speed.
  5. In the Gel 10" - Gel 30' columns, enter the gel strength measured for each interval.
  6. When you have finished adding dial readings, click the Save Changes button ().

When you save the mud, it will be added to the CENTRO® Engineer database. When you change the pump fluid, you'll be able to select the new mud under the Mud utility. When you run an analysis, you'll also be able to select the new mud in the analysis settings. However, you will not be able to edit the mud properties or dial readings.

Field definitions

On the Dial Readings (PVT) tab, the mud's rheology is defined using the viscosity and gel strength readings listed below.
