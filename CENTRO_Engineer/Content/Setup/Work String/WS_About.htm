About work strings

The Setup > Work String page is used to build the work strings used while drilling and casing a well. CENTRO® Engineer includes the Weatherford catalog, which is the master catalog for BHA, drillstring, and casing components. It contains many of the tools manufactured by Weatherford, such as RSS units, mud motors, and LWD/MWD tools. The Weatherford catalog also contains standard BHA and drillstring parts, such as drill bits, stabilizers, drill pipe, and HWDP. Although you can edit many of these tools, the changes will only be reflected in that particular instance of the tool. The Weatherford catalog itself will not be affected. See "Work string components".

You can also create additional catalogs for custom, user-created drill bits, drill collars, drill pipe, HWDP, and other custom tools. When you create a custom tool, it will be added to your custom catalog so that it can be used again in the future. See "Custom tools".

The Work String page also includes a BHA builder and a casing string builder. The BHA Builder view has several tools to help you assess the quality of a BHA, such as additional weight calculators, a non-magnetic spacing calculator, and tool summaries for the bit, stabilizers, and sensors. See "BHA design". The Casing String Builder window generates a casing string by calculating the placement of the centralizers and the amount of casing pipe required. See "Casing string design".

Building a new work string

When you create a work string, you can create it with or without an explicitly-defined BHA.

Work string with an explicitly-defined BHA

  • In a work string with an explicitly-defined BHA, the BHA and drillstring components will be grouped into separate tables. When you open the BHA Builder view, only the BHA components will be listed. Drillstring components will not appear in the BHA Builder view.
  • In a work string without an explicitly-defined BHA, all BHA and drillstring components will appear in one table. When you open the BHA Builder view, all work string components will be listed.

To create a new work string:

  1. Go to the Setup > Work String page.
  2. Click the Add button on the right side of the screen.
  3. When prompted, select an option for defining the BHA.
    • To create a work string with an explicitly-defined BHA, click Yes.
    • To create a work string without an explicitly-defined BHA, click No.
  4. Click the Insert Tool button () to add components to the work string. See "Adding components".
  5. When you have finished adding components, click the Save Changes button () at the top of the screen.

A new work string will be added to CENTRO® Engineer, and a new numbered tab will appear in the right toolbar. As you add new components, the preview of the complete work string will be updated in the Well View pane. The work string components will also appear in the BHA Builder view. See "BHA design".

Tip

To export the completed work string to WITSML, click the Export to WITSML button (). The work string will be exported as a tubular object to the first WITSML channel listed on the Setup > DAQ > WITSML page. See "WITSML channels".

Adding components

To add a component to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Select the component below the one you want to add, then click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the type of component you want to add.
  5. In the Catalog field, select the catalog you want to use.
  6. Enter the dimensions and other properties of the component. See "Work string components".
  7. Click the OK button () to add the component to the work string.

The new part will appear in the list of components, and the preview of the complete work string will be updated in the Well View pane.

Note

You cannot add tools with 0 ID to the active work string.

Adding custom tools to a catalog

If your work string contains a custom tool that is not in a custom tool catalog, you can add the custom tool to a catalog.

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Select the custom tool that you want to add to a catalog.
  4. Click the Convert To Template button (). If this button is not visible, the custom tool may have already been added to a catalog.
  5. Select the catalog that you want to add the tool to.

The custom tool will be added to the selected catalog, and the Convert To Template button () will no longer be visible.

Note

If there are no custom tool catalogs in CENTRO® Engineer, a new catalog will be automatically created when you select a custom tool on the Setup > Work String page. To view the new catalog, go to the Setup > Catalogs > Custom Tools page. See "Custom tools".

Grouping components together

To group work string components into a cluster:

  1. Go to the Setup > Work String > page.
  2. Select a work string using the tabs on the right side of the screen.
  3. For each component that you want to include in the cluster, select the checkbox in the Group column.
  4. Click the Group Tools button (). If this button is not visible, you may have non-sequential components selected.
  5. Click the Save Changes button ().

The selected components will be grouped together into a cluster, and the preview of the complete work string will be updated in the Well View pane. To separate the cluster into individual components, select the cluster and click the Ungroup Tools button ().

Activating a work string

The active work string is indicated by a red toggle switch () in the list of work strings. To change the active work string:

  1. Go to the Setup > Work String page.
  2. Select the work string you want to activate.
  3. Click the Activate button ().
  4. Click the Save Changes button ().

The selected work string will be marked as the active work string, using the red toggle switch (). The newly active work string will be validated against the active wellbore intervals. See "Well intervals".

Note

If the selected work string contains a tool with 0 ID, you cannot activate the work string.

Using the Work String page

There are several keyboard shortcuts and mouse actions that you can use on the Setup > Work String page.

  • To export work string data to a CSV file, press the Ctrl + S keys.
  • To move a component to a different position in the work string, click on the component you want to move and drag it to the new position.
  • To change the name of a work string, go to the toolbar on the right side of the screen and click on the tab for the work string. In the box that appears, enter a new name for the work string and press the Enter key.
  • To change the name of a work string tool, double-click on the tool name in the list of work string components.
  • To edit a work string tool, double-click anywhere in the row for the tool, except on the name. You can also press the Alt key and click on the tool name.
  • To switch to another tool when the editing panel is open, click on the tool name in the list of work string components. Any changes to currently selected tool will be automatically saved when you select another tool.
  • To close the editing panel, press the Esc key.

The toolbar at the top of the screen is used to add components in the selected work string. To select the columns shown in the Workstring Components table, click the Show Columns button () in the first column of the table.
