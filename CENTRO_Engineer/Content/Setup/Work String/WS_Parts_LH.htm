About liner hangers

A liner hanger is a tool or system used in the oil and gas industry to support and suspend a liner in a well. Liner hangers are slip-type devices designed to connect a liner to the previous casing and provide a seal between the liner and the casing.

The CENTRO® Engineer catalog does not include any liner hangers, so you will need to create them on the Setup > Catalog > Custom Tools page. See "Liner hangers".

Adding liner hangers

To add a liner hanger to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Liner Hanger System option.
  5. In the Catalog field, select the catalog that contains the liner hanger you want to use.
  6. In the Tool Size field, select the nominal size of the liner hanger.
  7. In the Description field, select the name of the liner hanger you want to use.
  8. Update the remaining properties of the liner hanger as needed. See "Properties of liner hangers".
  9. Click the OK button () to add the liner hanger to the work string.

The liner hanger will be added to the selected catalog. It will also appear in the list of available tools on the Setup > Work String page. See "Work strings".
