About tubing accessories

Tubing accessories are tools that are run into the well with a production tubing string. The CENTRO® Engineer catalog does not include any tubing accessorys, so you will need to create them on the Setup > Catalog > Custom Tools page. See "Tubing accessories".

You can create several types of custom tubing accessories:

  • Gas Lift Mandrel – A gas lift mandrel is installed in the tubing string of gas-lift well that has a gas-lift valve. The two most common types of mandrels are the conventional mandrel and the side pocket mandrel. In a conventional mandrel, the gas-lift valve in installed inside the mandrel as the tubing is placed in the well. In a side pocket mandrel, the gas-lift valve is installed by wire line while the mandrel is in the well.
  • Gauge Mandrel – Gauge mandrels are used to protect permanent monitoring system gauges. Gauge mandrels are typically constructed from a solid block, so that there are no weld seams that create weak points in the mandrel.
  • Hydro-Trip Sub – A hydro-trip pressure sub is a temporary tubing plug used to set hydraulically actuated packers. The sub is installed in a tubing string below the packer, then pressure is applied to activate the packer and the sub.
  • Landing Nipple – A landing nipple is a short section of heavy wall tubular with a machined internal surface. Landing nipples are used in production strings to latch and seal various types of plugs and valves. The most common types of landing nipples are no-go nipples, selective-landing nipples, and safety-valve nipples.
  • Locator Seal Assembly – Locator seal assemblies are used to limit the downward movement of the seals in a packer bore. The assembly is run into the well on the production tubing string until it locates the top of the packer, which establishes a seal between the packer and the tubing.
  • Mill Out Extension – A mill out extension is a packer accessory used with seal bore packers. They are run into a tubing string directly below a seal bore packer or a seal bore extension. The ID of a mill out extension is larger than the ID of the packer, which protects the seals on the packer.
  • Pump Out Plug – A pump out plug is a temporary tubing plug used to set a packer. The plug contains a ball and seat that allow the tubing to fill as it is run into the well. After the packer is set, tubing pressure is increased to shear the plug, and the ball and seat are expelled downward to the bottom of the well.
  • Seal Bore Extension – Sealbore extensions are used to extend the sealing bore of a sealbore packer. A sealbore extension provides sufficient sealbore length for long seal assemblies to accommodate tubing movement, while remaining in the sealbore. Sealbore extensions have the same ID as the sealbore packer.
  • USB w/ LSU – An upper sealbore packer with locator seal unit is a sealbore packer that includes a locator seal unit. Upper sealbore packers have a large upper bore that allows completions with large-ID tubing to maximize flow rates and improve access to the reservoir by wireline tools.

Adding a tubing accessory

To add a tubing accessory to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Tubing Accessory option.
  5. In the Catalog field, select the catalog that contains the tubing accessory you want to use.
  6. In the Type field, select the type of tubing accessory.
  7. In the Tool Size field, select the nominal size of the tubing accessory.
  8. In the Description field, select the name of the tubing accessory you want to use.
  9. Update the remaining properties of the tubing accessory as needed. See "Properties of tubing accessories".
  10. Click the OK button () to add the tubing accessory to the work string.

The tubing accessory will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of tubing accessories

The sections below define the properties of custom tubing accessories.

Properties of custom tubing accessories
Section Field Definition
-- Type Type of tubing accessory. See "About tubing accessories".
-- Tool Size

Nominal size of the tubing accessory
-- Description

Name or description of the tubing accessory
General Info Length

Total length of the tubing accessory. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the tubing accessory.

  • Calculated – The mass and adjusted weight of the tubing accessory are calculated, based on the dimensions of the tubing accessory. When the dimensions of the tubing accessory are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the tubing accessory is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the tubing accessory. When the dimensions of the tubing accessory are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the tubing accessory is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the tubing accessory. When the dimensions of the tubing accessory are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the tubing accessory. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the tubing accessory. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Material

Symbol for the material that the tubing accessory is composed of and the grade of the material.
Drift

Drift diameter of the tubing accessory
NW

Nominal weight of the tubing accessory. This value is calculated automatically from the length and mass of the tubing accessory.
Limits Burst

Burst pressure rating for the tubing accessory
Collapse

Collapse pressure rating for the tubing accessory
Tension

Tension rating of the tubing accessory
Compression

Compression rating of the tubing accessory
Connection Data Connection Configuration

Configuration of the connections on the tubing accessory. This is a read-only value, populated from the CENTRO® Engineer catalog.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
