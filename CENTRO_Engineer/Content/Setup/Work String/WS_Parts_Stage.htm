About stage tools

Stage tools are used to cement multiple sections of the well behind the same casing string or to cement a single section of the well in multiple stages. The CENTRO® Engineer catalog does not include any stage tools, so you will need to create them on the Setup > Catalog > Custom Tools page. See "Stage tools".

You can create two types of custom stage tools:

  • Mechanical – Mechanical stage collars are mechanically opened and closed. The opening sleeve shifts open when an opening dart is landed in the stage collar. The stage collar is closed by landing a displacement plug in the stage collar and applying pressure.
  • Inflatable – Hydraulic stage collars can be set to open hydraulically at different opening pressures. The opening sleeve shifts down when pressure is applied, which allows the second stage cement to be displaced. The stage collar is closed by landing a displacement plug and applying pressure above the plug.

Adding a stage tool

To add a stage tool to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Stage Tool option.
  5. In the Catalog field, select the catalog that contains the stage tool you want to use.
  6. In the Type field, select the type of stage tool.
  7. In the Tool Size field, select the nominal size of the stage tool.
  8. In the Description field, select the name of the stage tool you want to use.
  9. Update the remaining properties of the stage tool as needed. See "Properties of stage tools".
  10. Click the OK button () to add the stage tool to the work string.

The stage tool will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of stage tools

The sections below define the properties of custom stage tools.

Properties of custom stage tools
Section Field Definition
-- Type Type of stage tool. See "About stage tools".
-- Tool Size

Nominal size of the stage tool
-- Description

Name or description of the stage tool
General Info Length

Total length of the stage tool. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the stage tool.

  • Calculated – The mass and adjusted weight of the stage tool are calculated, based on the dimensions of the stage tool. When the dimensions of the stage tool are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the stage tool is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the stage tool. When the dimensions of the stage tool are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the stage tool is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the stage tool. When the dimensions of the stage tool are updated, the value for mass will not be affected.
Mass

Total mass of the stage tool. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Adjusted Weight

Total adjusted weight of the stage tool. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
NW

Nominal weight of the stage tool. This value is calculated automatically from the length and mass of the stage tool.
Material

Symbol for the material that the stage tool is composed of and the grade of the material.
-- TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
-- Efficiency

Efficiency of the nozzles on the stage tool, relative to the maximum possible performance
-- Nozzles

Number of nozzles on the stage tool. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
-- Nozzle Diameter

Size of each nozzle on the stage tool. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Connection Data Connection Configuration

Configuration of the connections on the stage tool. This is a read-only value, populated from the CENTRO® Engineer catalog.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Seat ID Inside diameter of the opening seat. Only available for the stage tool section.
Length

Length of the part section.
