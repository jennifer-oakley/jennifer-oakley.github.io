About wireline tools

Wireline tools are electric logging tools that use an electrical cable to lower the tool into the wellbore and transmit data. On the Setup > Catalogs > Custom Tools page, you can create custom wireline tools. These wireline tools can be used when you create a work string on the Setup > Work String page. See "Custom wireline tools".

You can add three types of custom wireline tools:

  • Standard – Standard wireline tools include a wide range of openhole tools for formation testing and sampling. Openhole tools include petrophysical tools, acoustic logging tools, geological tools, and geophysical tools.
  • Compact – Compact tools are slim, self-centering versions of standard wireline tools that can be used in slim boreholes and through pipe.
  • Cased Hole – Cased hole wireline tools provide data that identify downhole issues for rig-free well remediation. Cased hole tools include production logging tool, formation evaluation tools, well integrity evaluation tools, perforation tools, mechanical and plugback tools, and pipe recovery tools.

Adding wireline tools

To add a wireline tool to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Wireline Tools option.
  5. In the Catalog field, select the catalog that contains the wireline tool you want to use.
  6. In the Type field, select the type of wireline tool.
  7. In the Config field, select the mnemonic for the data measured by the wireline tool.
  8. In the Tool Size field, select the nominal size of the wireline tool.
  9. In the Description field, select the name of the wireline tool you want to use.
  10. Update the remaining properties of the wireline tool as needed. See "Properties of wireline tools".
  11. Click the OK button () to add the wireline tool to the work string.

The wireline tool will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of wireline tools

The table below defines the properties for wireline tools in CENTRO® Engineer.

Properties of wireline tools
Section Field Definition
-- Type

Type of wireline tool
-- Config Mnemonic for the data measured by the wireline tool Only available for standard wireline tools.
-- Tool Size

Nominal size of the wireline tool
-- Description

Name or description of the wireline tool
General Info Description

Name or description of the wireline tool Only available for standard wireline tools.
Mnemonic Mnemonic for the data measured by the wireline tool Only available for customwireline tools.
Length

Total length of the wireline tool. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the wireline tool.

  • Calculated – The mass and weight in air of the wireline tool are calculated, based on the dimensions of the wireline tool. When the dimensions of the wireline tool are updated, the values for mass and weight in air will be automatically updated.
  • User-Defined Adjusted Weight – The weight in air of the wireline tool is a manually entered value. Mass is automatically calculated, based on the weight in air of the wireline tool. When the dimensions of the wireline tool are updated, the value for weight in air will not be affected.
  • User-Defined Mass – The mass of the wireline tool is a manually entered value. Weight in air is automatically calculated, based on the mass of the wireline tool. When the dimensions of the wireline tool are updated, the value for mass will not be affected.
Mass

Total mass of the wireline tool. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
Weight in Air

Total weight of the wireline tool when suspended in air. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
NW

Nominal weight of the wireline tool. This value is calculated automatically from the length and mass of the wireline tool.
Material

Symbol for the material that the wireline tool is composed of and the grade of the material.
Limits Tension

Tension rating of the wireline tool
Compression

Compression rating of the wireline tool
Knuckle Joint (checkbox) Indicates if the wireline tool includes a knuckle joint
Deflection Angle

Angle at which the knuckle joint is attached to the wireline tool
Roller (checkbox) Indicates if the wireline tool includes a roller
Centralizer (checkbox) Indicates if the wireline tool includes a centralizer
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
