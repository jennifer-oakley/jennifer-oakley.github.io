Building a BHA

To create a new BHA:

  1. Go to the Setup > Work String page.
  2. Click the BHA Builder button () in the bottom toolbar.
  3. Click the Add button on the right side of the screen.
  4. Click the Insert Tool button () to add components to the BHA. See "Adding BHA components".
  5. When you have finished adding components, click the Save Changes button () at the top of the screen.

A new BHA will be added to CENTRO® Engineer, and a new numbered tab will appear in the right toolbar. As you add new BHA components, the preview of the complete BHA will be updated in the Well View pane. BHAs created in the BHA Builder view will also appear in the Work String view. See "Work strings".

Adding BHA components

To add a BHA component:

  1. Go to the Setup > Work String page.
  2. Click the BHA Builder button () in the bottom toolbar.
  3. Select a BHA using the tabs on the right side of the screen.
  4. Select the component below the one you want to add, then click the Insert Tool button ().
  5. In the panel that appears on the right, go to the Tool field and select the type of component you want to add.
  6. In the Catalog field, select the catalog you want to use.
  7. Enter the dimensions and other properties of the component. See "Work string components".
  8. Click the OK button () to add the component to the BHA.

The new component will be added to the list of BHA components, and the preview of the complete BHA will be updated in the Well View pane.

Adding custom tools to a catalog

If your BHA contains a custom tool that is not in a custom tool catalog, you can add the custom tool to a catalog.

  1. Go to the Setup > Work String page.
  2. Select a BHA using the tabs on the right side of the screen.
  3. Select the custom tool that you want to add to a catalog.
  4. Click the Convert To Template button (). If this button is not visible, the custom tool may have already been added to a catalog.
  5. Select the catalog that you want to add the tool to.

The custom tool will be added to the selected catalog, and the Convert To Template button () will no longer be visible.

Note

If there are no custom tool catalogs in CENTRO® Engineer, a new catalog will be automatically created when you select a custom tool on the Setup > Work String page. To view the new catalog, go to the Setup > Catalogs > Custom Tools page. See "Custom tools".
