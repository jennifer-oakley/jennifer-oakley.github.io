About tool information

In the BHA Builder view, you can get a quick estimate of the quality of each BHA. The BHA Builder view contains several features to help you build a better BHA:

  • Neutral point calculations. The neutral point of the BHA is automatically calculated, based on the mud weight, hole angle, and WOB. See "Neutral point".
  • Non-magnetic calculator. If the BHA contains a directional sensor, such as an IDS, you can check the spacing of the non-magnetic components around the sensor. See "Non-magnetic component spacing".
  • Tool summaries. The BHA Builder view also includes at-a-glance summaries for the bit, stabilizers, and sensors in the BHA. See "Bit, stabilizer, and sensor information".

For work strings with an explicitly-defined BHA, the BHA Builder view only displays the BHA components for the work string. For work strings without an explicitly-defined BHA, the BHA Builder view automatically identifies the BHA components in the work string.

Neutral point

The neutral point of the BHA is the point at which axial stress changes from compression to tension. The location of the neutral point depends on the amount of WOB and the buoyancy of the drilling fluid.

Note

The Mud Weight field, the Required WOB field, and the NP column are only visible when the Show Axial Load in BHA Builder option is selected on the Global Settings > System Preferences > Interface Settings page. See "Interface settings".

Note

The neutral point calculation in the BHA Builder view is only an estimate. For a more accurate neutral point calculation, you can run a torque and drag analysis on the Planning > Mechanics > Torque and Drag page. See "Torque and drag analysis".

To find the BHA's neutral point and corrected weight:

  1. Go to the Setup > Work String page.
  2. Click the BHA Builder button () in the toolbar at the top of the page.
  3. In the Hole Angle field, enter the planned inclination.
  4. In the Mud Weight field, enter the weight of the mud that you plan to use with the BHA.
  5. In the Required WOB field, enter the maximum WOB that will be on the BHA.

The neutral point will be updated in the NP column automatically. Red arrows indicate tools where the BHA is in compression. Green arrows indicate tools where the BHA is in tension. The neutral point occurs near the bottom of the first component with a green arrow.

Non-magnetic component spacing

If the BHA contains a directional sensor, you can check the spacing of the non-magnetic components around the sensor. To check the spacing of the non-magnetic components:

  1. Go to the Setup > Work String page.
  2. Click the BHA Builder button () in the toolbar at the top of the page.
  3. In the Hole Angle field, enter the planned inclination.
  4. In the Target Azimuth field, enter the target azimuth of the well trajectory.
  5. In the Mag. Field Strength field, enter the magnetic field strength.
  6. In the Mag. Dip Angle field, enter the magnetic dip angle.
  7. In the Total Az. Correction field, enter the amount of azimuth correction.

The azimuth error will be automatically calculated and updated in the Non-Mag Spacing area, in the bottom left corner of the screen. If the amount of error is within a safe range, a green dot will appear next to the Az Error field. If the amount of error is unsafe, a red dot will appear.

Bit, stabilizer, and sensor information

The bottom portion of the BHA Builder view contains summaries about LWD/MWD sensors, stabilizers, and the drill bit. The Bit Information area, in the bottom left corner of the screen, contains details about the bit nozzles and TFA.

The Sensors tab is only visible if the work string contains tools with sensors, and the Stabilizers tab is only visible if the work string contains stabilizers or tools with stabilizers. The stabilizers are numbered in the order that they appear in the BHA, from bottom to top. The stabilizers listed in this table include mud motor stabilizers and the pivot stabilizer on an RSS unit, as well as standard standalone stabilizers.

Note

The calculation for azimuth error in the BHA builder does not account for magnetic interference from MWD tools.

The table below defines the bit, stabilizer, and sensor information on the BHA Builder view.
