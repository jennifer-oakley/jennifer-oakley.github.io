About BHA design

On theSetup > Work String page, the BHA Builder view has several tools to give you a quick estimate of the quality of each BHA. In the BHA Builder view, you can:

  • Create a BHA design report. The BHA design report includes the work string configuration, the rig and well interval details, mud information, and summaries for special tools, such as the bit, stabilizers, sensors, and drilling motors. See "Creating a BHA design report".
  • View calculation results. The BHA Builder view also includes several tools to help you assess the quality of the BHA. These tools include additional weight calculations, a non-magnetic spacing calculator, and summaries for the bit, stabilizers, and sensors. See "Tool information".
  • View engineering information. In the BHA Builder view, the overall properties of the BHA includes calculated engineering values for stiffness ratio and bending stiffness ratio, as well as flags for non-magnetic components. See "Field definitions".

For work strings with an explicitly-defined BHA, the BHA Builder view only displays the BHA components for the work string. For work strings without an explicitly-defined BHA, the BHA Builder view automatically identifies the BHA components in the work string.

You can also build additional BHAs, as well as add and edit the BHA components. Any changes made in the BHA Builder view will also appear on the Work String view. See "Editing BHAs".

Creating a BHA design report

There are four templates you can use to create a BHA design report:

  • The Standard report template includes the work string configuration, as well as information about the drill bit, mud, sensors, stabilizers, drilling motor and RSS unit.
  • The WFT - Template A report template is a simplified, landscape-oriented template for Shell or other clients with specific reporting requirements.
  • The WFT - Template B report template is a landscape-orientation version of the Standard template.
  • The DE BHA report template includes the work string configuration, as well as information about the drill bit, mud, sensors, stabilizers, drilling motor, Magnus RSS unit, and underreamers.

To create a BHA design report:

  1. Go to the Setup > Work String page.
  2. Select the BHA that you want to create a report for.
  3. Click the BHA Builder button ().
  4. Click the Save Report button ().
  5. In the menu that appears, select the template that you want to use for the report.
  6. In the window that appears, select the folder location and file type.
  7. Click Save.

The report will be automatically generated and saved to the selected location. When the report finishes generating, it will open automatically.
