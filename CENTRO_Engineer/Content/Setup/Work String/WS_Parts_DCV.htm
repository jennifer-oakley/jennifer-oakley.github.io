About downhole control valves

Downhole control valves are downhole devices that control the flow of drilling fluid. The CENTRO® Engineer catalog does not include any downhole control valves, so you will need to create them on the Setup > Catalog > Custom Tools page. See "Downhole control valves".

You can create several types of custom downhole control valves:

  • FIV (Ball Valve) – A ball valve contains a ball which rotates up to 90° to open and close the valve.
  • FLV (Flapper Valve) – A flapper valve prevents fluid from flowing up through the drillstring. It contains a spring-loaded plate which opens when fluid is pumped downhole, but closes if the fluid attempts to flow backwards from the drillstring to the surface.
  • ICV – Inflow control valves prevent unwanted fluids from entering the wellbore. ICVs are installed at completion joints and can be controlled from the surface.
  • Safety Valve – A safety valve is an automatic valve that opens when excessive pressure is reached. Safety valves are also known as relief valves or pip valves.
  • Sliding Sleeve – Sliding sleeves are used to adjust flow and manage pressure along the wellbore by choosing which zones remain open or closed during production. They can also be used to circulate fluids from the annulus to the tubing above the production packer, or to inject water into individual zones during waterflood operations.

Adding a downhole control valve

To add a downhole control valve to a work string:

  1. Go to the Setup > Work String page.
  2. Select a work string using the tabs on the right side of the screen.
  3. Click the Insert Tool button ().
  4. In the panel that appears on the right, go to the Tool field and select the Downhole Control Valve option.
  5. In the Catalog field, select the catalog that contains the downhole control valve you want to use.
  6. In the Type field, select the type of downhole control valve.
  7. In the Tool Size field, select the nominal size of the downhole control valve.
  8. In the Description field, select the name of the downhole control valve you want to use.
  9. Update the remaining properties of the downhole control valve as needed. See "Properties of downhole control valves".
  10. Click the OK button () to add the downhole control valve to the work string.

The downhole control valve will be added to the work string, and the work string preview will be updated in the Well View pane.

Properties of downhole control valves

The sections below define the properties of custom downhole control valves.

Properties of custom downhole control valves
Section Field Definition
-- Type Type of downhole control valve. See "About downhole control valves".
-- Tool Size

Nominal size of the downhole control valve
-- Description

Name or description of the downhole control valve
General Info Length

Total length of the downhole control valve. This value is calculated automatically.
Tool Weight Option

Method for entering the total mass of the downhole control valve.

  • Calculated – The mass and adjusted weight of the downhole control valve are calculated, based on the dimensions of the downhole control valve. When the dimensions of the downhole control valve are updated, the values for mass and adjusted weight will be automatically updated.
  • User-Defined Adjusted Weight – The adjusted weight of the downhole control valve is a manually entered value. Mass is automatically calculated, based on the adjusted weight of the downhole control valve. When the dimensions of the downhole control valve are updated, the value for adjusted weight will not be affected.
  • User-Defined Mass – The mass of the downhole control valve is a manually entered value. Adjusted weight is automatically calculated, based on the mass of the downhole control valve. When the dimensions of the downhole control valve are updated, the value for mass will not be affected.
Adjusted Weight

Total adjusted weight of the downhole control valve. When the Calculated option or the User-Defined Adjusted Mass option is selected in the Tool Weight Option field, this value is calculated automatically.
Mass

Total mass of the downhole control valve. When the Calculated option or the User-Defined Adjusted Weight option is selected in the Tool Weight Option field, this value is calculated automatically.
NW

Nominal weight of the downhole control valve. This value is calculated automatically from the length and mass of the downhole control valve.
Drift

Drift diameter of the downhole control valve
Material

Symbol for the material that the downhole control valve is composed of and the grade of the material.
Limits Burst

Burst pressure rating for the downhole control valve
Collapse

Collapse pressure rating for the downhole control valve
Tension

Tension rating of the downhole control valve
Compression

Compression rating of the downhole control valve
Nozzles (checkbox) Indicates that nozzles are present on the downhole control valve. Not available for safety valves.
TFA

Total flow area. TFA can be manually entered, or calculated from the size of each nozzle and the number of nozzles. If there is a value in the Nozzles field, the value in the TFA field will be overwritten with a calculated value.
Efficiency

Efficiency of the nozzles on the downhole control valve, relative to the maximum possible performance
Nozzles

Number of nozzles on the downhole control valve. If a value is entered in this field, TFA will be calculated from the size of the nozzles and the number of nozzles.
Nozzle Diameter

Size of each nozzle on the downhole control valve. The number of nozzles listed is determined by the value in the Nozzles field. The Nozzle Diameter field only appears when there is a value entered in the Nozzles field.
Connection Data Connection Configuration

Configuration of the connections on the downhole control valve. This is a read-only value, populated from the CENTRO® Engineer catalog.

  • Box-Box – Both connections are box connections.
  • Box – The top connection is a box connection, and there is no bottom connection.
  • Box-Pin – The top connection is a box connection, and the bottom connection is a pin connection.
  • Pin-Pin – Both connections are pin connections.
  • Pin – The top connection is a pin connection, and there is no bottom connection.
  • Pin-Box – The top connection is a pin connection, and the bottom connection is a box connection.
Top Connection Thread Type Type of threads used on the top connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the top connection
Bottom Connection Thread Type Type of threads used on the bottom connection. The list of available connections includes custom connections created on the Setup > Catalogs > Custom Connections page. This list is automatically filtered based on the selected tool size.
MUT

Make-up torque of the bottom connection
Part Sections OD

Outside diameter of the part section.
ID

Inside diameter of the part section.
Length

Length of the part section.
