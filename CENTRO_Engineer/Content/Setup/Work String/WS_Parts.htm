Work string components

Related Topics
Related Topics
  • Custom tools

In CENTRO® Engineer, the Weatherford catalog contains several types of work string components. For more information about customizing specific tools, refer to the topics below.

  • Adjustable Gauge Stabilizers. Adjustable gauge stabilizers are used above the drilling motor to make changes to the build rate and rotary build rate characteristics of the BHA while drilling. See "Adjustable gauge stabilizers".
  • Bits. In addition to the dimensions of the bit, you'll also need to enter the sizes of the nozzles on the bit. See "Drill bits".
  • Casing Pipe. Casing pipe is steel pipe that is cemented into an openhole interval. See "Casing pipe".
  • Coiled Tubing. Coiled tubing is a continuous length of small-diameter pipe wound on a spool, then straightened prior to running it into the wellbore. See "Coiled tubing".
  • Custom Tools. If none of the part categories in CENTRO® Engineer apply to the work string part you want to add, you can also create a custom part. See "Custom parts".
  • Downhole Control Valves. Downhole control valves are downhole devices that control the flow of drilling fluid. See "Downhole control valves".
  • Drill Collars. Drill collars are thick, heavy-walled tubes used between the bit and the drill pipe to provide a pendulum effect for the drillstring and add weight to the bit. See "Drill collars".
  • Drilling Motors. By default, the drilling motors included in the CENTRO® Engineer catalog do not have any active stabilizers. If you are using a stabilized drilling motor, you'll need to activate the stabilizers. See "Drilling motors".
  • Drill Pipe. Drill pipes are described using uniform pipe section. Each uniform section has the same OD, ID, and nominal weight across the entire length of the section. See "Drill pipes".
  • Fishing Tools. Fishing tools are tools that are designed to recover equipment and debris from the well, such as a lost bit or a drill collar. See "Fishing tools".
  • Float Collars. Float collars are special coupling devices that are inserted one or two joints above the bottom of a casing string. Float collars also prevent the back-flow of cement during cementing operations. See "Float collars".
  • Float Shoes. Float shoes are short, heavy cylindrical steel sections with rounded bottoms, which are attached to the bottom end of a casing string. The float shoe contains a check valve and functions in the same manner as a float collar, but also serves as guide shoe for the casing. See "Float shoes".
  • Guide Shoes. The CENTRO® Engineer catalog does not include any guide shoes. Before you can create a casing string, you'll need to create a guide shoe. See "Guide shoes".
  • Ghost Reamers. Ghost reamers are used for reaming, drilling, and hole cleaning in horizontal shale wells and extended-reach wells. They combine a keyseat wiper and a tapered-blade reamer to address unstable formations, swabbing, and hole-cleaning issues. See "Ghost reamers".
  • Hole Openers. When you add a hole opener, you'll also need to set the activation type and opening size. See "Hole openers".
  • HWDP. Heavyweight drill pipe is drill pipe that has thicker walls and longer tool joints, as well as a wear pad in the middle. HWDP is used between the bit and the drill pipe to reduce the chances of drill pipe fatigue or failure. See "HWDPs".
  • Jars. Jars are percussion tools that are operated mechanically or hydraulically to deliver a heavy hammer blow to objects in the wellbore. Jars are used to free objects during drilling or fishing operations. See "Jars".
  • Liner Hanger Systems. A liner hanger is a tool or system used in the oil and gas industry to support and suspend a liner in a well. Liner hangers are slip-type devices designed to connect a liner to the previous casing and provide a seal between the liner and the casing. See "Liner hangers".
  • LWD Tools. The CENTRO® Engineer catalog includes Weatherford's most popular LWD tools, including AZD™/TNP™, MFR™, GuideWave™, HEL™, IDS™, ShockWave™, UltraWave™, and EMpulse™ HEL™ tools. See "LWD tools".
  • Mills. Mills are downhole tools that have rough, sharp, and extremely hard cutting surfaces for removing metal by grinding or cutting. See "Mills".
  • MWD Tools. The CENTRO® Engineer catalog includes ABM™, HyperPulse™, and EMpulse™ MWD tools. See "MWD tools".
  • Packers. Packers are downhole tools that are used to create a seal between tubing, casing, or drill pipe and the borehole wall or casing. See "Packers".
  • Re-Entry Tools. Re-entry tools are used to create sidetracks in existing wells and maneuver around wellbore obstructions. See "Re-entry tools".
  • POST Tools. POST tools are Weatherford's packoff stage cementing tools, which are specially designed for high-pressure gas and oil wells. See "POST tools".
  • RSS Units. The CENTRO® Engineer catalog includes all Weatherford RSS units. The configuration for an RSS unit includes a pivot stabilizer, a bias unit, and a controller collar. See "RSS units".
  • Sand Control Tools. Sand control tools are tools used to prevent large amounts of sand from entering the wellbore. See "Sand control tools".
  • Slickline Tools. Slickline tools are wireline tools which can be attached to the slickline and lowered into the well for well workover and intervention operations. See "Slickline tools".
  • Special Tools. Special tools are circulation tools that are used to control the flow between the pipe and the annulus. See "Special tools".
  • Stabilizers. The CENTRO® Engineer catalog includes string stabilizers, near-bit stabilizers, centralizers, roller reamers, and non-magnetic stabilizers. See "Stabilizers".
  • Stage Tools. Stage tools are used to cement multiple sections of the well behind the same casing string or to cement a single section of the well in multiple stages. See "Stage tools".
  • Subs. The CENTRO® Engineer catalog contains several types of rotary subs and circulation subs, including crossover subs, bit subs, saver subs, UBHO subs, float subs, filter subs, and ball lock subs. See "Subs".
  • Thru-tubing Motors. Thru-tubing motors are drilling motors that are deployed on either jointed pipe or coiled tubing. They are powered by drilling fluid, and the power section converts hydraulic horsepower from the mud into mechanical horsepower to drive the bit. See "Thru-tubing motors".
  • Tubing Accessories. Tubing accessories are tools that are run into the well with a production tubing string. See "Tubing accessories".
  • Tubing Compensation Equipment. Tubing compensation equipment is used to compensate for the vertical and horizontal motion of the tubing string. See "Tubing compensation equipment".
  • Well Services Tools. The tools listed under the Well Services tool type are coiled tubing tools. Coiled tubing can be used for well construction and well intervention services. See "Well services tools".
  • Wireline Cable. Wireline cables are armored electrical cables that are used to lower wireline tools into the wellbore, as well as to transmit the data gathered by wireline tools. See "Wireline cable".
  • Wireline Tools. Wireline tools are electric logging tools that use an electrical cable to lower the tool into the wellbore and transmit data. See "Wireline tools".
