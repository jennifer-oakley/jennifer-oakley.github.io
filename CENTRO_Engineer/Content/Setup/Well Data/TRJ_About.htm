About wellpath data

The Setup > Wells > Wellpath page contains the wellpath data for the planned well trajectory and real-time surveys. The drop-down menu at the top of the screen allows you to switch between the trajectory data and the survey data.

  • The Trajectory view should be used for the survey points in the planned trajectory of the well. The data in the Trajectory view is only for reference, and is not used in any calculations.
  • The Survey view should be used for the actual, real-time survey data. All calculations and analyses in CENTRO® Engineer use the data in the Survey view.

The measured depth, inclination, and azimuth of each survey point are used to calculate several values, such as TVD, dogleg severity, build rate, and turn rate. These values are updated automatically as you add and edit survey points. See "Wellpath data".

The first two survey points of the wellpath are automatically calculated, based on the rig elevations and the properties of the derrick. These surveys points are highlighted in pink on the Wellpath page. To change these survey points, you must update the rig elevations on the Setup > Surface Equipment > Equipment Configuration page. See "Rig Elevations".

In CENTRO® Engineer, you can also create multiple wellpaths. While a well is in the planning stages, the planned wellpath may change several times. When this happens, you can create a new wellpath with the updated plan. This allows you to keep an archive of the wellpath changes that occur during the well planning process. See "Creating a wellpath".

The Wellpath page also uses several charts to graph the wellpath data. The Well View pane shows charts for the projected wellpath and the three-dimensional wellbore trajectory. The Wellpath Inspection window contains charts that allow you to compare the original trajectory and survey data to the corrected data. See "Trajectory charts".

Creating a wellpath

To create a new wellpath:

  1. Go to the Setup > Wells > Wellpath page.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of wellpaths.
  3. Add survey points to the new wellpath. See "Wellpath data".
  4. Click the Save Changes button () to save your changes.

The new wellpath will be added to the CENTRO® Engineer database. When you run an analysis, it will be listed as one of the wellpath options.

Tip

To change the name of a wellpath, go to the toolbar on the right side of the screen and click on the tab for the wellpath. In the box that appears, enter a new name for the wellpath and press the Enter key.

Activating a wellpath

The active wellpath is indicated by a red toggle switch () in the list of wellpaths. To change the active wellpath:

  1. Go to the Setup > Wells > Wellpath page.
  2. Select the wellpath you want to activate.
  3. Click the Activate button ().
  4. Click the Save Changes button ().

The selected wellpath will be marked as the active wellpath, using the red toggle switch (). The formation layers on the Formation tab will be updated to reflect the TVD values in the currently active wellpath. When you run an analysis, the active wellpath will be selected by default.

Note

When you export the well from CENTRO® Engineer and import it into another CENTRO® Engineer application, only the active wellpath will be imported.

Using the Wellpath page

To export wellpath data to a CSV file, press the Ctrl + S keys. The toolbar at the top of the screen on the Wellpath page is used to open the Wellpath Inspection window, open the Tortuosity window, import data, and create reports. To select the columns shown in the Wellpath table, click the Show Columns button () in the first column of the table.
