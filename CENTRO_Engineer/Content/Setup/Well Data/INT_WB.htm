About wellbores

In CENTRO® Engineer, you can create multiple wellbores. While a well is in the planning stages, the planned wellbore geometry may change several times. When this happens, you can create a new wellbore with the updated geometry. This allows you to run analyses on different wellbores, as well as keep an archive of the well geometry changes that occur during the well planning process. See "Creating wellbore instances".

Note

Well intervals created by the well activity planner are denoted with an asterisk (*) in front of the well interval name. Well intervals created by the well activity planner can only be deleted on the Setup > MVA Settings > Well Activity Planner page. See "Well activity planner".

Creating wellbore instances

To create a new wellbore instance:

  1. Go to the Setup > Wells > Well Intervalspage.
  2. Click the Add button on the right side of the screen. A new tab will appear in the list of well intervals.
  3. Add casing and liner sections. See "Casing and liner sections".
  4. Add open hole sections. See "Open hole sections".
  5. Click the Save Changes button () to save your changes.

All of the hole sections in the previous wellbore instance will copied to the new wellbore instance. When you save your changes, the dimensions of the new wellbore instance will be updated in the Well View pane. You will no longer be able to edit any of the hole sections in the existing wellbore instances.

Tip

To change the name of a wellbore instance, go to the toolbar on the right side of the screen and click on the tab for the wellbore instance. In the box that appears, enter a new name for the wellbore instance and press the Enter key.

Activating a wellbore instance

The active wellbore is the current wellbore, used for CENTRO® Engineer's real-time calculations. Only one wellbore instance and one drillstring can be active at a time. CENTRO® Engineer validates the dimensions of the active wellbore instance against the dimensions of the active drillstring. The ID of all casing, liner, and open hole sections must be larger than the OD of all drillstring parts. Inactive wellbore instances and drillstrings are not validated against one another.

The active wellbore instance is indicated by a red toggle switch () in the list of wellbore instances. To change the active wellbore instance:

  1. Go to the Setup > Wells > Well Intervals page.
  2. Select the wellbore instance you want to activate.
  3. Click the Activate button ().
  4. Click the Save Changes button ().

The selected wellbore instance will be marked as the active wellbore instance, using the red toggle switch (). The newly active wellbore instance will be validated against the active drillstring. See "Activating a work string".
