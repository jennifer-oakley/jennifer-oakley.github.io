About imports

CENTRO® Engineer project files are compressed files that can contain different types of data, such as dashboards, drillstrings, tool catalogs, custom connections, and well configurations. On the All Projects > Import Project page, you can import a well into CENTRO® Engineer. Importing CENTRO® Engineer projects allows you to easily collaborate on a well with a co-worker, open a well from another CENTRO® Engineer application, or load one of your own wells from a backup file.

You can also import the following types of data on the All Projects > Import Project page.

  • CSV and LAS imports allow you to import depth-based data for an offset well. See "Importing CSV or LAS files".
  • Formation tops imports allow you to import the formation data required for the Geosteering indicator. See "Importing formation tops".
  • Microseismic data imports allow you to import the microseismic event data required for the Microseismic indicator. See "Importing microseismic data".
Tip

To import other types of data from an MCD file, go to the All Projects > Import Objects > MCD tab. See "MCD imports".

Importing a well

To import a well configuration:

  1. Go to the All Projects > Import Project page.
  2. In the window that opens, navigate to the file you want to import and click Open.
  3. After the import file loads, review the well information on the Import Project page.
  4. In the OneClick™ Import Setting field, choose an import option for OneClick™ settings.
    • To use the default OneClick™ settings, select the Do not import option. OneClick™ settings will not be included in the import, and any default OneClick™ settings will remain intact.
    • To import custom OneClick™ settings, select the Import from X.X.XX configuration option. OneClick™ settings will be included in the import, and any default OneClick™ settings will be overwritten.
  5. If the import file includes analysis experiments, go to the bottom of the screen and select the experiments you want to import.
  6. Click the Save Changes button () at the bottom of the screen.

When CENTRO® Engineer finishes loading the data from the import file, the well will open on the Dashboards screen. The project name will be updated in the Notifications banner at the top of the screen.

Importing CSV or LAS files

To import CSV or LAS files:

  1. Go to the All Projects > Import Project page.
  2. In the window that opens, change the file type to CSV documents (*.csv) or LAS documents (*.las).
  3. Navigate to the file you want to import and click Open.
  4. In the Index column, select the curve used to index the data.
  5. Select the remaining curves you want to import.
  6. Update the units for each curve as needed.
  7. Click the Save Changes button () at the bottom of the screen.

The imported curve data will appear in the list of curves on the Dashboards screen, under the Well Data Offset Log heading. To access the list of curves, go to the Dashboards screen and click the Edit Dashboard button ().

Note

Only depth-based data can be imported into CENTRO® Engineer. CSV and LAS imports for time-based data are not currently supported.

Importing formation tops

The data for formation tops must be in a tab-delimited text file that includes the formation names, lateral distance, and TVD. The data should follow the basic format shown below. You may use as many points for TVD and LD lateral distance as needed, but you must have at least two sets of points.

  • Column 1 - Formation name
  • Column 2 - LD
  • Column 3 - TVD
  • Column 4 - LD
  • Column 5 - TVD

To import formation tops:

  1. Go to the Projects > All Projects > Import Project page.
  2. In the window that opens, change the file type to Formation Tops (*.txt).
  3. Navigate to the file you want to import and click Open.
  4. Update the settings for Depth Type, Depth Unit and Lateral Distance Unit as needed.
  5. For each formation top, select the color you want to use to plot the formation top.
  6. Click the Save Changes button ().

The import file will appear in the list of available formation tops when you configure the settings for the Geosteering indicator. See "Geosteering indicator".

Importing microseismic data

Before microseismic data can be imported into CENTRO® Engineer, it must be saved in microseismic file format. To save data as a microseismic file:

  1. Open the data file in MS Excel and save it in .csv format. Close the file.
  2. Using your browser, locate the file and change the file extension to .microseismic.

The file will be in the special format that CENTRO® Engineer will recognize as a microseismic file.

To import microseismic data:

  1. Go to the Projects > All Projects > Import Project page.
  2. In the window that opens, change the file type to Microseismic file (*.microseismic).
  3. Navigate to the file you want to import and click Open.
  4. Adjust the import settings as needed.
  5. Click the Save Changes button ().

The selected microseismic data will be imported into CENTRO® Engineer. When you configure the Microseismic indicator, the imported file will appear in the list of microseismic events.
