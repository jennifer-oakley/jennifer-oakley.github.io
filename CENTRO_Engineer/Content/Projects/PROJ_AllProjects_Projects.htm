About projects

In the CENTRO® Engineer application suite, all of the data related to a job is stored with a project. This includes job information, the rig configuration, well data, fluid properties, limits and alarms, settings, and other data. On the Projects > All Projects > Projects page, you can create a new well, change the active well, switch between modes, and delete inactive wells. The active well is marked by a red Active label.

Creating a new well

You can create a well using the default configuration, which uses a set of default data that includes surface equipment, well data, drillstrings, and fluids. You can also copy the configuration of another well in CENTRO® Engineer. The new well will be an exact copy of the original well, but without completed analysis results or data acquired through WITSML or WITS.

Note

If the well is a historical well, and you intend to reprocess the data for it, the Mark For Reprocessing option must be selected when you create the well. See "Reprocessing data".

To create a new well:

  1. Go to the Projects screen, then expand the All Projects heading.
  2. Go to the Projects page.
  3. Click the Add button () at the top of the screen.
  4. Enter the project name and the well name.
  5. In the Based On Well field, select the well configuration you want to copy. To use the default configuration, select the Default option.
  6. Enter the remaining job and well information.
  7. If the well is a historical well, select the Mark For Reprocessing option. See "Reprocessing data".
  8. In the Data Source field, select the primary source of data for the well.
    • WITSML – The primary source of data is a WITSML log. When this option is selected, gap filling will be enabled for the well. See "Gap fill".
    • Other – The primary source of data is not a WITSML log. When this option is selected, gap filling will not be enabled for the well.
  9. Click the Save Changes button () to save the well.

The well will be added to the CENTRO® Engineer database, and it will automatically become the active well. The new well will open on the Dashboards screen, and the project name will appear in the banner at the top of the screen. All of the information when you created the well will appear on the Projects > Job Info > General page. See "Job Info".

Changing the active well

If there are multiple wells in the CENTRO® Engineer database, you can change the active well on the Projects page.

To change the active well:

  1. Go to the Projects screen, then expand the All Projects heading.
  2. Go to the Projects page.
  3. Click the Active button in the row for the well that you want to activate.

The selected well will become the active well. When CENTRO® Engineer finishes loading the data for the well, the well will open on the Dashboards screen. Inactive wells will remain in the CENTRO® Engineer database.

Switching between modes

There are two modes for CENTRO® Engineer: Real Time mode and Player mode.

  • Real Time mode acquires new data from WITS or WITSML data sources and performs real-time hydraulics and torque and drag calculations. In Real Time mode, you can edit any part of the well configuration.
  • Player mode replays the well data acquired in CENTRO® Engineer. The well configuration and rig states will change as they were changed in real-time. In Playback mode, you can speed up the data and fast-forward to a specific point of time. However, the well data can't be edited when you're using Playback mode.

To switch to Real Time or Player mode:

  1. Go to the Projects screen, then expand the All Projects heading.
  2. Go to the Projects page.
  3. If you are currently in Real Time mode, click the Play Well button () in the row for the well you want to play.
  4. If you are currently in Player mode, click theMonitoring button () to switch to Real Time mode.

If you switched to Playback mode, you will no longer be able to edit the well data. To start replaying the data, click on the clock and select a playback rate from the menu that appears. See "Real-time and playback controls".

If you switched to Real Time mode, you will be able to edit any of the data for the well. To start acquiring data, click on the clock and press the Play button (). See "Real-time and playback controls".

In addition, the selected well will become the active well. When CENTRO® Engineer finishes loading the data for the well, the well will open on the Dashboards screen.

Deleting wells

If you no longer need one of the wells in the CENTRO® Engineer database, you can remove it from the database on the Projects page.

Note

The active well cannot be deleted. To delete the active well, you will need to switch to another well first. See "Changing the active well".

To delete a well from the database:

  1. Go to the Projects screen, then expand the All Projects heading.
  2. Go to the Projects page.
  3. Click the Delete Well button () in the row for the well you want to delete.
  4. When prompted, click Yes on the warning message that appears.

When the well has been deleted, a confirmation message will appear. The well will be deleted from the CENTRO® Engineer database, and will not be listed on any screen in CENTRO® Engineer.

Using the Projects page

The toolbar at the top of the Projects page is used to create new wells and switch between Table view and Map view.

Toolbar buttons on the All Projects > Projects page
Button Function
Switches the application mode from Real Time mode to Playback mode. See "Switching between modes".
Switches from Table view to Map view, which shows the locations of all of the wells in CENTRO® Engineer
Switches from Map view to Table view, which shows a list of all of the wells in CENTRO® Engineer
Opens the New Project panel, which allows you to create a new well. See "Creating a new well".

The toolbar next to each project is used to delete wells, switch modes, and change the active well.

Toolbar buttons for projects
Button Function
Active Switches the active well to the selected well. See "Changing the active well".
Switches the application mode from Real Time mode to Playback mode. See "Switching between modes".
Deletes the selected well. See "Deleting wells".
