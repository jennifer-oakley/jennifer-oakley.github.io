About exports

On the Projects > All Projects > Export File page, you can export data from the current well. When you export data from a CENTRO® Engineer project, a compressed MCD file will be created. CENTRO® Engineer project files are compressed files that can contain different types of data, such as dashboards, drillstrings, tool catalogs, custom connections, and well configurations. MCD files can be exported to your computer or to a network location.

There are several options for exporting data from CENTRO® Engineer:

  • The Configuration option allows you to export all of the setup data for the well. See "Exporting the well configuration".
  • The Configuration + Experiments option allows you to export analysis experiments, as well as the setup data. See "Exporting the configuration and analysis experiments".
  • The Dashboards option allows to export only the dashboards for the well. See "Exporting dashboards".
  • The BHA/Drill String option allows you to export only the work strings on the Setup > Work String section. See "Exporting work strings".
  • The Custom Tool Catalogs option allows you to export your catalogs of custom tools on the Setup > Catalogs > Custom Tools page. See "Exporting tool catalogs".
  • The Connection Catalogs option allows you to export custom connections from the catalog on the Setup > Catalogs > Custom Connections page. See "Exporting custom connections".
  • The Well/Project option allows you to export the well configuration, work strings, dashboards, custom tool catalogs, custom connection catalogs, analysis experiments, and real-time well data. See "Exporting well and project data".
  • The OneClick Calculations option allows you to export public and private calculations from the Setup > OneClick™ Settings > Planning > Calculations tab and the Setup > OneClick™ Settings > Real Time > Calculations tab. See "Exporting OneClick™ calculations".

Exporting the well configuration

To export the well configuration data:

  1. Go to the Projects > All Projects > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Configuration option.
  4. Click the Export button () at the top of the screen to begin the export.

CENTRO® Engineer will return to the Dashboards screen. When the export completes, a notification message will appear in a pop-up window. A compressed MCD file containing the well configuration will be created in the selected folder location.

Exporting the configuration and analysis experiments

To export the well configuration and analysis experiments:

  1. Go to the Projects > All Projects > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select theConfiguration + Experiments option.
  4. Select the experiments you want to include in the export.
  5. Click the Export button () at the top of the screen to begin the export.

CENTRO® Engineer will return to the Dashboards screen. When the export completes, a notification message will appear in a pop-up window. A compressed MCD file containing the well configuration and data will be created in the selected folder location.

Exporting well and project data

An export for well and project data includes the well configuration, work strings, dashboards, custom tool catalogs, custom connection catalogs, analysis experiments, and real-time well data.

To export the well and project data:

  1. Go to the Projects > All Projects > Export File page.
  2. To change the default export location, click the Browse button () in the Path field and select a new folder location.
  3. In the Save field, select the Well/Project option.
  4. Click the Export button () at the top of the screen to begin the export.

CENTRO® Engineer will return to the Dashboards screen. When the export completes, a notification message will appear in a pop-up window. A compressed MCD file containing the well configuration, the project data, and all real-time well data will be created in the selected folder location.

  4. Select the planning calculations and monitoring you want to include in the export.

CENTRO® Engineer will return to the Dashboards screen. When the export completes, a notification message will appear in a pop-up window. A compressed MCD file containing the selected calculations will be created in the selected folder location.
