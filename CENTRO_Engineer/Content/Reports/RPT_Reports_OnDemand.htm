About on-demand reports

The Reports > Reports > On-Demand page allows you to create an on-demand real-time report that contains a detailed summary of the well and the drilling performance on the rig. On-demand reports are generated based on the date selected in the Range field.

You can create reports for four different report ranges:

  • Date – Creates a report for the selected date. To select a different day, click the Calendar button () in the Report Date field.
  • Entire Well – Creates a cumulative report for the entire well.
  • Time Frame – Creates a cumulative report for the selected time frame. To change the start date, click the Calendar button () in the From field. To change the end date, click the Calendar button () in the To field.
  • Well Activities – Creates a cumulative report for the time frame of the selected well activity. To select a time frame based on a well activity, click the Well Activities button at the top of the screen, then select the well activity that you want to create a report for. See "Well activity planner".

There are two on-demand templates available:

  • Standard – The Standard template includes information about rig activities, friction factors, torque & drag effects, planned vs. actual comparisons, hydraulics calculations, and custom charts.
  • Milling Report – The Milling Report template includes information about milling performance, such as ROP, weight on mill, and milling torque.

You can also use a custom report template. Custom report templates must use the same pre-defined hidden sheets that are used in the Standard template. See "Using a custom template".

Creating an on-demand report

To generate an on-demand real-time report:

  1. Go to the Reports > Reports > On-Demand page.
  2. In the Template field, select the report template you want to use. See "Using a custom template".
  3. Select the information you want to include in the report. See "Report details".
  4. In the Range field on the right side of the screen, select a method for filtering the report data. Use the Calendar button () to select different dates, or click the Well Activities button to select a well activity.
  5. If you are using the Standard template, add any charts that you want to include in the report. See "Adding charts to a report".
  6. Click the Save Report button () at the top of the screen.
  7. In the window that appears, navigate to the location where you want to save the report and click Save.

The report will be automatically generated for the selected report range. When the report finishes generating, it will open automatically. When the report opens in MS Excel, click the Enable Content button to enable all of the report features.

Adding charts to a report

When you create a report using the Standard template, you can add up to four custom charts to the report. To add a chart to an on-demand monitoring report:

  1. Go to the Reports > Reports > On-Demand page.
  2. Click the Add Chart button () at the top of the screen. A list of chart settings will appear below the Range field.
  3. In the Chart Name field, enter a title for the chart.
  4. In the Name column, select the curves you want to appear in the chart.
  5. In the Color column, select the color you want to use for graphing each curve.
  6. To add a secondary Y-axis to the chart, select the Use Secondary Axis option.

The unit type for the first parameter listed in the table will be used as the Y-axis for the chart. When you create the report, the chart will appear on the Charts tab in the report.

Note

Charts cannot be added to the reports creating using the Milling Report template.

Tip

To delete a curve from a chart, select the curve and press the Delete key.

Using a custom template

Report templates must be saved in a *.xlsm file format, and they must be saved to the following directory: C:\ProgramData\Weatherford\OneSync\Monitoring\ReportTemplates. Custom report templates must use the same pre-defined hidden sheets that are used in the standard report template. When you create a report using a custom template, CENTRO® Engineer will attempt to populate the report with the data from the hidden sheets.

To use a custom template:

  1. Save your custom report template to the following directory: C:\ProgramData\Weatherford\OneSync\Monitoring\ReportTemplates. If the ReportTemplates folder does not exist, you'll need to create it.
  2. Close and reopen CENTRO® Engineer.
  3. Go to the Reports > Reports > On-Demand page.
  4. In the Templates field, select your custom report template. The report template will have the same name as the file you saved to the ReportTemplates directory.

When you save the report, the report will be created using the selected template.

Using the On-Demand page

The toolbar at the top of the Reports > Reports > On-Demand page is used to add charts and save reports.

Adds a new chart to the report. Only available for the Standard template.
