About rig activity reports

The Reports > KPIs > Rig Activity page contains a daily summary of the amount of time spent on different drilling operations. This information is collected from the operations triggered on the Setup > OneClick™ Settings > Real Time page. See "OneClick™ real-time settings".

There are two charts on the Rig Activity page:

  • The first chart shows all of the operations for the selected day, broken into color-coded blocks to show which operations occurred for each hour of the day.
  • The second chart filters this information by the selected operation, showing the portion of each hour was spent on the selected operation.

To view the performance summary for a different day, click the Calendar button () at the top of the screen. In the window that appears, use the arrows to navigate to the desired day.

Selecting an operation

To change the selected operation:

  1. Go to the Reports > KPIs > Rig Activity page.
  2. Click on the dropdown menu above the chart at the top of the page.
  3. Select the desired operation from the list that appears.

The chart at the bottom of the page will be updated to show the time spent on the selected operation.

Using the Rig Activity page

The toolbar at the top of the Reports > KPIs > Rig Activity page is used to navigate between days and toggle the chart legend.

Toolbar buttons on the Rig Activity page
Button Function
Pins the chart legends to the top of the performance charts
Opens the Browsable Date window, which allows you to navigate to the summary for another day
