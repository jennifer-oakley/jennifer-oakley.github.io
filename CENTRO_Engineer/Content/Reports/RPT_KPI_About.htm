KPIs

The Reports > KPIs section contains reports that allow you to view the performance of the well. There are four two pages in the KPIs section:

  • The Rig Activity page contains a report of the daily breakdown of the time spent on the different drilling operations. This information is grouped into several broad categories, such as drilling, tripping, circulating, and reaming. See "Rig activity reports".
  • The Performance page shows the detailed, time-based breakdown of rig activities for individual activities and activity groups. The information is displayed as both a legend table and as pie charts. See "Performance reports".
  • The Time Breakdown page contains a report of the daily key performance indicators for the well. This report contains more specific information about the activities that occur while drilling and tripping, as well as during connections. See "KPIs".
  • The Custom KPI Groups page lists the custom KPI groups and the rig activities assigned to them. KPI groups are used to group CENTRO® Engineer rig activities together in any way desired, regardless of the associated drilling phase. See "Custom KPI groups".

The information on both the Rig Activity and Performance pages is collected from the operations triggered by the flows on the Setup > OneClick™ Settings > Real Time page. The information on the Time Breakdown page is collected based on the same algorithm. If you are seeing inconsistencies between these reports, check your algorithm settings on the Setup > OneClick™ Settings > Real Time page. See "OneClick™ real-time settings".

Note

The KPIs section is only available in Real-Time mode. See "Application modes".
