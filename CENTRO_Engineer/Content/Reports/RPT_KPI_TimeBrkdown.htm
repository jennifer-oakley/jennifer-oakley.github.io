About KPIs

The Time Breakdown page contains a summary of the key performance indicators for drilling, tripping through open hole, and tripping through cased hole, as well as connections made during each of these activities. This information is collected from the events triggered on the Setup > OneClick™ Settings > Real Time page. See "OneClick™ real-time settings".

To view the KPIs, go to the Reports > KPIs > Time Breakdown page and select the report range you want to view.

  • Today – Shows the KPIs for the current day, starting at midnight.
  • Date – Displays the KPIs for the selected date. To select a different day, click the Calendar button ().
  • Time Frame – Displays the cumulative KPIs for the selected time frame. To change the start date, click the Calendar button () in the first date field. To change the end date, click the Calendar button () in the second date field.
  • Entire Well – Displays the cumulative KPIs since the well was created.
  • Well Activities – Displays the cumulative KPIs for the time frame of the selected well activity. To select a time frame based on a well activity, click the Well Activities button and select the well activity that you want to view KPIs for. See "Well activity planner".
