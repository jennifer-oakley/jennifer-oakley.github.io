Set Limits

Related Topics
Related Topics
  • Active alarms
  • Gauges

The Alarms > Set Limits section contains the limits for different types of alarms, such as alarms for incoming data, calculations, and well parameters.

  • The Hydraulics page contains alarm limits for some of values calculated by the hydraulics model. See "Hydraulics alarms".
  • The Parameters page contains alarm limits for the well parameters on the Setup > Mappings > Parameters page. See "Well parameter alarms".
  • The Calculations page contains alarm limits for the calculations on the OneClick™ Settings > Real Time page. See "Calculation alarms".
  • The WITS page contains alarm limits for the WITS records defined on the Setup > DAQ > WITS > In tab. See "WITS alarms".
  • The WITSML page contains alarm limits for the WITSML data defined on the Setup > DAQ > WITSML page. See "WITSML alarms".

When an alarm is enabled, the alarm limits will be applied to any associated gauges on the Dashboard screen. When the minimum or maximum limit for an enabled alarm is exceeded, the banner at the top of the screen will flash yellow. The alarm will be also listed on the Alarms > Active page. See "Active alarms".

Alarm limits are color-coded, according to the severity of the alarm. There are three alarm ranges:

  • Red — Red indicates a danger zone. If a parameter falls within a red zone, the minimum or maximum recommended limits have been exceeded. Immediate corrective steps should be taken to prevent damage.
  • Yellow — Yellow indicates a warning zone. If a parameter falls within a yellow zone, the limits are close to being exceeded. Caution should be used to make sure the limits are not exceeded.
  • Green — Green indicates an acceptable zone. If a parameter falls within a green zone, the limits are within the normal range.

The ranges for these zones depend on how you define the limits for the alarm. The Minimum and Maximum limits are required for all alarms. However, the other limits are optional. The table below shows sample limits and the resulting alarm ranges.
