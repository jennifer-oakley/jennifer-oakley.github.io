About WITSML alarms

The Alarms > Set Limits > WITSML page contains the alarm limits for the WITSML data defined on the Setup > DAQ > WITSML tab. See "WITSML channels". On the Dashboard screen, the associated curves for these alarms are listed under the name of the WITSML channel.

When an alarm is enabled, the alarm limits for the associated curve will be applied to any gauges on the Dashboard screen. See "Gauges". The alarm limits will also appear in the DAQ indicator for the WITSML channel. See "Data acquisition indicators".

Enabling WITSML alarms

To enable a WITSML alarm:

  1. Go to the Alarms > Set Limits > WITSML page.
  2. Select the Enable checkbox next to the name of the alarm you want to enable.
  3. Enter the Minimum and Maximum limits for the alarm.
  4. Enter the remaining limits as needed.
  5. Click the Save Changes button () to save your changes.

If you have created a gauge for that curve, the gauge will be color-coded with the new alarm ranges. The minimum and maximum values on the gauge will be updated to match the Minimum and Maximum limits on the WITSML page. If the Minimum or Maximum values are exceeded, an alert will appear in the notification banner at the top of the screen.

Tip

You can also map a limit to a calculation or well parameter. Go to the limit you want to set, click on the drop-down arrow and select the desired calculation or well parameter.

Using the WITSML page

To export alarm limits data to a CSV file, press the Ctrl + S keys. The toolbar on the WITSML page is used to save and undo changes.
