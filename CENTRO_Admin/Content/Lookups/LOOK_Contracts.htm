About contracts

Contracts store administrative information about the job, such as client, facility, and contractor information. You can also include billing information, such as the product line, billing entity, and client company. All managed wells should be associated with a contract. There can be multiple wells associated with a contract, but only one contract associated with each well.

On the Lookups > Contracts page, you can add and edit contracts. When you create a new well, the contracts on this page will appear in the list of available contracts.

Adding contracts

To add a new contract to CENTRO® Admin:

  1. Go to the Lookups > Contracts page.
  2. Click the Add Contract button () at the top of the page.
  3. In the window that appears, enter the contract name or job number in the Name field.
  4. In the Client field, select the name of the client company.
  5. Enter the starting and ending dates of the contract. If there is no end date, leave the End Date field blank.
  6. Select the billing entity and product line associated with the project.
  7. Enter the information for the project manager, client facility, client representative, and project supervisor.
  8. Enter the remaining contract details, then click the Save button.

The CENTRO® Admin database will be updated with the contract details. When you create a new well, the contract name will be included in the list of available contracts.

Using the Contracts page

The search box above the table on the Contracts page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the Contracts page are used to add, edit, and remove contracts.

Buttons on the Contracts page
Button Function
Adds a new contract. See "Adding contracts".
Allows you to edit the details for the current contract
Deletes the details for the selected contract
