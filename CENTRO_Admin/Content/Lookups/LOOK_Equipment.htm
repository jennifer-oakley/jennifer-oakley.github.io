About equipment records

Equipment records store administrative information about servers, gateways, switches, and other equipment. You can include information such as the user name and password, serial number, MAC address, manufacturer, and model. On the Lookups > Equipment page, you can add and edit equipment records. When you create a new well, the equipment listed on this page will appear in the list of available equipment.

Adding equipment records

To add a new equipment record to CENTRO® Admin:

  1. Go to the Lookups > Equipment page.
  2. Click the Add Equipment button () at the top of the page.
  3. In the window that appears, enter the name of the equipment in the Name field.
  4. In the Equipment Type field, select the type of equipment.
  5. In the Location field, select the location of the equipment.
  6. Enter the remaining equipment details, then click the Save button.

The CENTRO® Admin database will be updated with the equipment details. When you create a new well, the equipment name will be included in the list of available equipment.

Field definitions

The table below defines all of the fields in the Equipment Details window, which appears when you add or edit a client on the Lookups > Equipment page on the data center.

Field definitions for the Lookups > Equipment page
Field Definition
Name Name of the equipment, such as the server name
Equipment Type Type of equipment
Status

Current status of the equipment. If the In Service option is not selected, the equipment will not appear in the list of equipment names when you add wells.

  • In Service – The equipment is currently in service at the selected location
  • Obsolete – The equipment is obsolete and has been decommissioned.
  • Reassigned – The equipment has been reassigned to another project or purpose.
  • Repair – The equipment is currently being repaired.
  • Standby – The equipment is not currently being used, but is available for use.
  • Transit From – The equipment is in transit from the selected location.
  • Transit To – The equipment is in transit to the selected location.
  • Unknown – The status of the equipment is currently unknown.
Location Name of the location where the equipment is located. The client company associated with the selected location will be shown in parentheses. The list of available locations is defined on the Lookups > Locations page. See "Locations".
Username Login ID for the equipment
Password Password for the specified user name
OEM Serial Original serial number from the manufacturer
MAC Address Physical network address of the equipment
Model Model name of the equipment
Manufacturer Name of the equipment manufacturer
Purchase Date Date that the equipment was purchased
Warranty Expiration Date that the equipment warranty expires
Comments Any additional information about the equipment

Using the Equipment page

The search box above the table on the Equipment page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the Equipment page are used to add, edit, and remove equipment records.

Buttons on the Equipment page
Button Function
Adds a new equipment record. See "Adding equipment records".
Allows you to edit the details for the current equipment record
Deletes the details for the selected equipment record
