About metadata

Metadata controls which menu options are available for many of the dropdown lists throughout CENTRO® Admin. On the Lookups > Metadata page, you can add and edit the options for these lists. Each type of metadata corresponds to a dropdown list elsewhere in CENTRO® Admin. See "List of metadata types".

To view the current values for a dropdown list, use the Metadata Type field at the top of the page to select the list you want to view.

  • The Sequence field controls the order in which each item appears in the dropdown list.
  • The Name field controls the value that appears in the dropdown list.

Each metadata type contains several default values. These values can't be deleted. However, you can change the sequence number associated with a default value.

Adding metadata values

To add a metadata value:

  1. Go to the Lookups > Metadata page.
  2. In the Metadata Type field at the top of the page, select the type of metadata you want to add. See "List of metadata types".
  3. Click the Add button () at the top of the page.
  4. In the window that appears, enter a name for the new value.
  5. Enter a sequence number for the new value.
  6. Click the Save button.

The name of the new metadata value will appear in the corresponding dropdown list. The sequence number of the new value will determine the order in which the new value will appear in the corresponding dropdown list.

Field definitions

The table below defines all of the fields on the Lookups > Metadata page on the data center.

Field definitions for the Lookups > Metadata page
Field Definition
Sequence Number Number used to determine the order in which the metadata value will appear in the corresponding dropdown list
Name Name of the metadata value
Active Indicates that the metadata value is active. If this option is not enabled, the metadata value will not appear in the corresponding dropdown list.

Using the Metadata page

The search box above the table on the Metadata page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the Metadata page are used to add, edit, and remove metadata values.

Buttons on the Metadata page
Button Function
Adds a metadata value to the current list. See "Adding metadata values".
Indicates that the metadata value is active
Indicates that the metadata value is inactive
Allows you to edit the current metadata value
Deletes the selected metadata value. Default metadata values cannot be renamed or deleted.
