Using the Document Store page

About the document store

On the Document Store page, you can upload and download documents related to the wells in CENTRO® Admin. When you select a well on the Document Store, a list of the folders and files for well will open. See "Uploading documents" and "Managing documents".

The Document Store page is only available for the server that you are logged into. If you are logged into the data center, you cannot access the document store for any of the data hubs connected to it. To access the document store for a data hub, you will need to log into the data hub directly.

Note

If a client company is assigned to your account, you will only be able to view documents that are associated with the same client company. See "User accounts".

Tip

The document store can also be accessed directly from the CENTRO® landing page. To open the document store, click the CENTRO® Document button after you log-in to CENTRO®.

Adding folders

To add a folder to the document store for a well:

  1. Go to the Document Store page.
  2. Select the well that you want to add a folder to.
  3. Click the Add Folder button ().
  4. In the Create Folder window, enter a name for the folder and click Save.

The new folder will appear in the list of folders for the well. To add files to the folder, click on the folder name, then click the Upload button (). See "Uploading documents". To change the default permissions for the folder, click the Actions button () and select the ACLs option.

Uploading documents

To upload documents for a well:

  1. Go to the Document Store page.
  2. Select the well that you want to upload a document for.
  3. Navigate to the folder where you want to store the document.
  4. Click the Upload button () to open the Upload Files window.
  5. Drag the files you want to upload into the Upload Files window.
  6. When you have finished adding files, click the Upload button ().

The files will be uploaded to the document store for the well. All users who have access to the folder will be able to access the new files. When all of the documents have finished uploading, click Ok to close the window.

Managing documents

You can use the toolbar at the top of the screen to manage multiple files simultaneously. To select a file, click on the blank space in the column on the left, next to the files or folders you want to manage.

  • To move files, select the files you want to move, then click the Move button (). In the panel that opens, select the folder you want to move the files to, then click Move.
  • To download files, select the files you want to download, then click the Download button (). The selected document will be automatically downloaded to your default download location.
  • To delete files, select the files you want to delete, then click the Delete button (), then click Yes on the confirmation message that appears.

Subscribing to notifications

You can subscribe to receive an email notification when a new document is uploaded for the well. You can also subscribe others to receive email notifications. See "Managing subscriptions".

To subscribe yourself to receive notifications:

  1. Go to the Document Store page.
  2. Click the Actions button () next to the name of well you want to subscribe to.
  3. Select the Subscribe option, then click Yes on the confirmation message that appears.

When new documents are uploaded for the well, you will receive an email notification.

Using the Document Store page

The search box above the table on the Document Store page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The toolbar at the top of the Document Store page allows you to add folders and upload documents. For additional options, click the Actions button () next to the name of a well, folder, or document.

Buttons on the Document Store page
Button Function
Adds a new folder to the document store. See "Adding folders".
Opens the Upload Files window, which allows you to upload documents to the current folder. See "Uploading documents".
Moves the selected files and folders to another folder. Only visible when a file or folder is selected. See "Managing documents".
Downloads the selected files and folders to your default download location. Only visible when a file or folder is selected. See "Managing documents".
Deletes the selected files and folders from the document store. Only visible when a file or folder is selected. See "Managing documents".
Opens a menu with additional options for managing the document store. The options that appear depend on the item type and your access level.
