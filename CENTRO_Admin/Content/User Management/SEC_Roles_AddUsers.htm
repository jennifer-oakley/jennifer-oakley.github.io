About adding users to roles

When you assign user accounts or groups to a role, the assigned user accounts and members of the assigned groups will be granted all of the privileges that are assigned to the role. There are three ways to add users to a security role:

  • Add a local user account. Local user accounts are created on the User Management > Users page. In the Role Details window, existing user accounts will be listed under the Users tab. See "Adding local user accounts to roles".
  • Add an Active Directory (AD) group. AD groups are created in Active Directory by a domain administrator. See "Adding Active Directory groups to roles".
  • Add an Azure group. Azure groups are created in Azure by a server administrator. You must have the group ID and group name in order to add the group to a role. See "Adding Azure groups to roles".

Adding local user accounts to roles

To add a local user account to a role:

  1. Go to the User Management > Roles page.
  2. Click the Edit button () for the role that you want to add users to.
  3. In the window that appears, go to the Users tab.
  4. Select the user accounts you want to add in the Available Users list, then click the Add button ().
  5. Click the Save button at the bottom of the window.

The Selected Users list will be updated with the assigned users. The user accounts will be granted all of the privileges assigned to the role. If a client company is associated with the role, the users assigned to the role will only be able to view the roles associated with that client company. Any users who are currently logged in will be automatically logged out and asked to log back in.

Tip

To select multiple users at the same time, press the Ctrl key and select the users you want to add.

Adding Active Directory groups to roles

When you add an Active Directory group to a role, members of the assigned group will be able to log in to CENTRO® using the Corporate Login button on the log-in screen, without entering their log-in credentials. After a user initially logs in with their AD account, their AD account will be listed on the User Management > Users page. See "User accounts".

If they have access to CENTRO® Monitor, they will also be automatically logged in when they use the direct URL for CENTRO® Monitor. See "Security Access Control Lists".

To add an Active Directory group to a role:

  1. Go to the User Management > Roles page.
  2. Click the Edit button () for the role that you want to add users to.
  3. In the window that appears, go to the AD Groups tab.
  4. In the Search AD Group field, enter the name of the group that you want to add and click the Search button ().
  5. Select the groups you want to add in the Available AD Groups list, then click the Add button ().
  6. Click the Save button at the bottom of the window.

The Selected AD Groups list will be updated with the assigned groups. Members of the assigned groups will be granted all of the privileges assigned to the selected role.

Note

The AD Groups tab is only visible when Active Directory integration is enabled. Active Directory integration is not enabled by default. To enable AD integration, refer to the Server-Side Configuration chapter of the CENTRO® Admin Install Guide.

Tip

To select multiple groups at the same time, press the Ctrl key and select the groups you want to add.

Adding Azure groups to roles

When you add an Azure group to a role, members of the assigned group will be able to log in to CENTRO® using the Corporate Login button on the log-in screen, without entering their log-in credentials. After a user initially logs in, their Azure AD account will be listed on the User Management > Users page. See "User accounts".

To add an Azure group to a role:

  1. Go to the User Management > Roles page.
  2. Click the Edit button () for the role that you want to add users to.
  3. In the window that appears, go to the Azure Groups tab.
  4. In the Group ID field, enter the unique ID for the Azure group you want to add.
  5. In the Group Name field, enter the name of the Azure group and click the Add Group button ().
  6. Select the group in the Available Azure Groups list, then click the Add button ().
  7. Click the Save button at the bottom of the window.

The Selected Azure Groups list will be updated with the assigned group. Members of the assigned group will be granted all of the privileges assigned to the selected role.

Note

The Azure Groups tab is only visible when Azure integration is enabled. Azure integration is not enabled by default. To enable Azure integration, refer to the Server-Side Configuration chapter of the CENTRO® Admin Install Guide.

Tip

To select multiple groups at the same time, press the Ctrl key and select the groups you want to add.
