About replication servers

To set up a replication route, you will need to enter the server information for the source WITSML server and the destination WITSML server on the Select Data Servers screen.

The source server is the server that contains the WITSML data that you want to copy. The destination server is the server that the data will be copied to.

Entering server information

To enter the server information:

  1. Go to the WITSML Replication page.
  2. Click the Create WITSML Replicator button () to create a new replication route, or click the More Info button () to edit an existing route.
  3. Go to the Select Data Servers screen, then enter a name for the replication route in the Replicator Name field.
  4. In the WITSML Data Source section, enter the information for the server that you are copying data from.
    • If you are copying data from a well on the current server, select the Local option.
    • If you are copying data from a well on another server, select the External option.
  5. In the WITSML Data Destination section, enter the information for the server that you are copying data to.
    • If you are copying data to a well on the current server, select the Local option.
    • If you are copying data to a well on another server, select the External option.
  6. Click the Next button.

No data will be copied to the destination server until you start the replication. When you start the replication, the selected data will be copied from the source well to the destination well. If you are copying data to a new well, the destination well will be automatically created on the destination server. If you are copying data to an existing well, the data from the source well will be added to the destination well.

Field definitions

The table below defines the fields for the settings on the WITSML Replicators > Select Data Servers screen.

Field definitions for the settings on the Select Data Servers screen
Section Name Field Name Definition
-- Replicator Name Name for the WITSML replication route
WITSML Data Source Local Indicates that the source well is located on the current WITSML server
External Indicates that the source well is located on a different WITSML server
URL URL for source WITSML server. Only available when the External option is selected.
Username User name used to log in to the source WITSML server. Only available when the External option is selected.
Password Password used to log in to the source WITSML server. Only available when the External option is selected.
WITSML Data Destination Local Indicates that the destination well will be located on the current WITSML server
External Indicates that the destination well will be located on a different WITSML server.
URL URL for the destination WITSML server. Only available when the External option is selected.
Username User name used to log in to the destination WITSML server. Only available when the External option is selected.
Password Password used to log in to the destination WITSML server. Only available when the External option is selected.
