About well management

The Wells > Well Management page lists all of the wells on the data center. Managed wells created in CENTRO® Admin are synchronized from the data hub to the data center.

When you create a managed well on the data center and assign a data hub to the well, the well will be automatically created on the assigned data hub. The WITSML data received for the well will be automatically synchronized between the data center and the data hub. If the assigned data hub is not currently registered and authorized, the well data will be synchronized once the data hub is authorized by the system administrator. Changes to managed wells are only replicated from the data hub to the data center. If you edit a well through the Well Management page on the data center, the changes will not be replicated to the data hub.

Note

If a client company is assigned to your account, you will only be able to view wells that are associated with the same client company. See "User accounts".

Tip

Completed and archive-ready wells are not shown by default on the Well Management page. To view completed and archive-ready wells, go to the Well Status field at the top of the table and select the checkbox at the top of the list.

Creating a managed well

To create a new managed well:

  1. Go to the Wells > Well Management page.
  2. Click the Add Well button () at the top of the page.
  3. In the window that appears, select the name of the project in the Contract field.
  4. Enter the unique ID for the well and the well name.
  5. Enter the remaining job and well information.
  6. In the Equipment field, select the data hub that will be used for the job.
  7. Click the Save button at the bottom of the window.

The well will be created on the WITSML server for the selected data hub, using the unique well ID and wellbore ID that you specified. The well will be available on both the data center and the selected data hub. The WITSML data received for the well will be automatically synchronized between the data hub and the data center. To integrate the well with CENTRO® Monitor, click the Create Project/Groups button () next to the well name. See "CENTRO® Monitor well integration".

CENTRO® Monitor well integration

After you create a well in CENTRO® Admin, you can integrate the well with CENTRO® Monitor. When you integrate a well with CENTRO® Monitor, the well will be mapped to a new project in CENTRO® Monitor.

CENTRO® will also create the roles required to access the well. When you begin the integration process, three roles will be created in CENTRO® Admin:

  • [RigName]_[WellName]_ReadOnly – Grants read-only access to CENTRO® Monitor, as well as read and list access to the WITSML data store for the well.
  • [RigName]_[WellName]_NormalUser – Grants normal user access to CENTRO® Monitor, as well as read and list to the WITSML data store for the well.
  • [RigName]_[WellName]_Admin – Grants admin access to CENTRO® Monitor, as well as create, update, read, and list access to the WITSML data store for the well.

The ACLs and WITSML ACLs required for each access level will be automatically assigned to the appropriate role. See "Security Access Control Lists" and "WITSML Access Control Lists".

Two groups will also be created for the project in CENTRO® Monitor:

  • [RigName]_[WellName]_InternalUsers – User accounts with a user name that ends with @weatherford.com will be assigned to the internal users group.
  • [RigName]_[WellName]_ExternalUsers – User accounts with a user name that does not end with @weatherford.com will be assigned to the external users group.

When a user is assigned to a well-specific role in CENTRO® Admin, a user account will also be created in CENTRO® Monitor and assigned to the appropriate group.

Note

This feature is only available data centers that have IVAAP integration enabled. IVAAP integration is not enabled by default. To enable IVAAP integration, refer to the Server-Side Configuration chapter of the CENTRO® Admin Install Guide.

To integrate a well with CENTRO® Monitor:

  1. Click the Create Project/Groups button () next to the well name.
  2. In the Monitor Integration window, click the Create Group and Projects button.
  3. When the mapping process is complete, click the Close button to close the Monitor Integration window.

The status of the integration process will be automatically updated every 30 seconds. The Create Project/Groups button () next to the well name will be replaced with the View Status button (). To add users to the roles for the well, go to the User Management > Roles page. See "Adding users to roles".

Field definitions

The table below defines all of the fields on the Wells > Well Management page.

Field definitions for the Wells > Well Management page
Field Definition
Name [UID] The well name should match the well name in Spectrum, if available. The unique ID in WITSML is shown in brackets, after the well name.
Status

Current status of the well

  • New – The well is a new well, and drilling has not yet begun.
  • Current – The well is currently being drilled.
  • Completed – Drilling has completed for the well.
  • Archive Ready – The data for the well is ready to be archived. This option is not available until the well has been marked as completed.
Contract Name of the contract for the well. The list of available contracts is defined on the Lookups > Contracts page. See "Contracts".
Field Name of the field where the well is located
Country Country where the well is operating
Rig Name of the rig at the well site
Depth Units Default measurement unit used for depth
Start Date Date when drilling begins for the well
Spud Date Date on which a machine capable of reaching total depth begins drilling a new well
End Date Date when drilling ended for the well
Client Name of the client company for the well. The list of available clients is defined on the Lookups > Clients page. See "Clients".

Using the Well Management page

The search box above the table on the Well Management page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the Well Management page are used to add, edit, and remove wells.

Buttons on the Well Management page
Button Function
Adds a new well to the current server. See "Creating a managed well".
Allows you to edit the selected well
Deletes the selected well from the server
Indicates that the CENTRO® Monitor integration process has been initiated. Opens the Monitor Integration window, which allows you to view the status of the integration process. Only available for data centers with IVAAP integration enabled.
Indicates that the CENTRO® Monitor integration process has not been initiated. Opens the Monitor Integration window, which allows you to start the integration process. Only available for data centers with IVAAP integration enabled.
