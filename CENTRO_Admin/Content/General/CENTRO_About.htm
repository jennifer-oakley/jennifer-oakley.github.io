About CENTRO® Admin

Welcome to CENTRO® Admin v6.0. CENTRO® Admin allows you to manage all aspects of the Weatherford Real-Time System, from controlling user access to managing WITSML data.

CENTRO® Admin administers the data center and data hubs in the Weatherford Real-Time System. The data center is a centralized system, which collects data from all of the data hubs connected to your Weatherford Real-Time System. Data hubs collect data from various sources, such as WITS and WITSML. The data center is normally located in an office, while the data hubs are located at the rig sites.

Use CENTRO® Admin to:

  • Control user access to CENTRO® Admin and the wells on the data center. See "User management".
  • Record information about clients, vendors, contracts, and other data. See "Lookups menu".
  • Create managed wells for the WITSML server. See "Well management".
  • Create alarms to monitor the data received for the wells on the data center. See "Alarms".
  • Convert LAS and WITS data to WITSML data. See "Well Data Import".
  • Create calculations to remove invalid data from WITSML logs and to determine the current rig activity. See "Calculations".
  • Create WITSML queries to troubleshoot issues with the WITSML data store. See "WITSML Explorer".
  • Upload documents for the wells on the data center to a central location. See "Document store".

When you initially install CENTRO® Admin, a temporary license will be automatically generated. However, this license will expire seven days after installing CENTRO® Admin. To continue using CENTRO® Admin, you will need to obtain a permanent license from the CENTRO® Support Team. See "Licensing".
