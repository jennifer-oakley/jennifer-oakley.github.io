Adding a managed well

Related Topics
Related Topics
  • Main menu
  • Well management

The Wells > Well Management page lists all of the wells on the data center. Managed wells created in CENTRO® Admin are synchronized from the data hub to the data center.

When you create a managed well on the data center and assign a data hub to the well, the well will be automatically created on the assigned data hub. The WITSML data received for the well will be automatically synchronized between the data center and the data hub. If the assigned data hub is not currently registered and authorized, the well data will be synchronized once the data hub is authorized by the system administrator. Changes to managed wells are only replicated from the data hub to the data center. If you edit a well through the Well Management page on the data center, the changes will not be replicated to the data hub.

Note

If a client company is assigned to your account, you will only be able to view wells that are associated with the same client company. See "User accounts".

Tip

Completed and archive-ready wells are not shown by default on the Well Management page. To view completed and archive-ready wells, go to the Well Status field at the top of the table and select the checkbox at the top of the list.

To create a new managed well:

  1. Go to the Wells > Well Management page.
  2. Click the Add Well button () at the top of the page.
  3. In the window that appears, select the name of the project in the Contract field.
  4. Enter the unique ID for the well and the well name.
  5. Enter the remaining job and well information.
  6. In the Equipment field, select the data hub that will be used for the job.
  7. Click the Save button at the bottom of the window.

The well will be created on the WITSML server for the selected data hub, using the unique well ID and wellbore ID that you specified. The well will be available on both the data center and the selected data hub. The WITSML data received for the well will be automatically synchronized between the data hub and the data center. To integrate the well with CENTRO® Monitor, click the Create Project/Groups button () next to the well name. See "CENTRO® Monitor well integration".
