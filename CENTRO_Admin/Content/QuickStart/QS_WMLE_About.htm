About WITSML Explorer

On the WITSML Explorer page, you can edit and create WITSML objects such as wells, wellbores, logs, mud logs, well geometries, and fluid reports. The WITSML Explorer page is divided into two basic sections:

  • The Tree View pane, on the left side of the page, displays a hierarchical view of the wells on the WITSML server, as well as the WITSML objects associated with each well.
  • The Query View pane, on the right side of the page, displays the current query and query results. It also contains buttons that allow you to add and update WITSML objects.

To navigate through the Tree View pane, click on the name of an object on the left side of the screen. All of the child objects for that object will be displayed below the selected object.

Creating a query

To build a query for an existing WITSML object:

  1. Go to the WITSML Explorer page.
  2. In the WITSML Version field at the bottom of the page, select the version of WITSML that your server is using.
  3. On the left side of the page, navigate to the WITSML object that you want to build a query for.
  4. Right-click on the WITSML object and select the Build Query option.

The query for the selected WITSML object will appear on the right side of the screen, under the Request tab. You can use this query to add or update WITSML objects. See "Adding WITSML objects" and "Updating WITSML objects".

Adding WITSML objects

To add a WITSML object to an existing well:

  1. Go to the WITSML Explorer page.
  2. In the WITSML Version field at the bottom of the page, select the version of WITSML that your server is using.
  3. On the left side of the page, navigate to the well and wellbore that you want to add an object to.
  4. Select the type of object that you want to add, such as a fluid report or a mud log.
  5. Right-click on the WITSML object and select the Build Query option. A base query for that object type will appear under the Request tab.
  6. Update the query with the name of the new object.
  7. Enter the remaining WITSML properties for the object.
  8. Click the Add button at the bottom of the page.

A response from the WITSML server will appear at the bottom of the Result tab. If the new object is successfully added, a unique ID will be automatically generated for the new object. If the object can't be added, an error message will be included with the response from the WITSML server.

Updating WITSML objects

To update an existing WITSML object:

  1. Go to the WITSML Explorer page.
  2. In the WITSML Version field at the bottom of the page, select the version of WITSML that your server is using.
  3. On the left side of the screen, navigate to the WITSML object that you want to update.
  4. Right-click on the WITSML object and select the Build Query option.. A query for the selected object will appear under the request tab.
  5. Update the WITSML properties for the object as desired.
  6. Click the Update button at the bottom of the page.

A response from the WITSML server will appear at the bottom of the Result tab. If the update is successful, a confirmation message will appear. If the object can't be updated, an error message will be included with the response from the WITSML server.
