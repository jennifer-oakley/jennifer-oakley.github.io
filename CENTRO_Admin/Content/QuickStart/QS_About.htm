Getting Started

This section provides you with an introduction to several of the main features of CENTRO® Admin. For more information about any of these topics, refer to the list of related topics on the left side of the page. For a full list of all the features available in CENTRO® Admin, refer to the Welcome > Interface section of the user guide. See "Tour of the CENTRO® Admin interface".

This quick start guide covers the following topics:

  • Connecting the data hubs. The Data Hub Listing page shows a tile for each data hub connected to the data center. Each tile includes the name and authorization status of the data hub, the date that data was last synced with it, and the date that it was last seen online. See "Connecting the data hubs".
  • Approving sign-up requests. The Signup Requests page lists all of the pending sign-up requests for the data center. From this page, pending requests can be approved or rejected. See "Approving sign-up requests".
  • Adding project information. The Lookups menu allows you to add the project-specific information on the data center. This includes vendors, clients, and contracts. This information is used when you create a new well. You can also add other types of project information, such as locations, equipment, and metadata. See "Adding project information".
  • Creating wells. The Wells > Well Management page lists all of the wells on the data center. Managed wells created in CENTRO® Admin are synchronized from the data hub to the data center.
    See "Adding a managed well".
  • Managing alarms. The Alarms > Alarm Management page shows you all of the alarms that have been created for each well on the data center. To view the alarms for a well, go to the Alarms Management page and click on the name of the well. See "Managing alarms".
  • Creating calculations. The Calculations menu is used to map mnemonics to curves and configure calculations. Despike calculations automatically remove data spikes from a log, and rig state calculations determine the current activity for the well. See "Creating calculations".
  • Using WITSML Explorer. The WITSML Explorer page is used to edit and create WITSML objects such as wells, wellbores, logs, mud logs, well geometries, and fluid reports. See "Using WITSML Explorer".
