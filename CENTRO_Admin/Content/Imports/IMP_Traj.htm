About survey data imports

On the Well Data Import page, you can import Survey data from an LAS or CSV file into a WITSML Trajectory. You must import Survey data into an existing WITSML Trajectory. There are three basic steps required to import a Survey from an LAS file:

  • Select the source. When you import the file, the data will be converted into WITSML trajectory stations using the mapping settings. See "Selecting the source file".
  • Select the destination Survey. The destination settings tell CENTRO® Admin which well, wellbore, and WITSML Survey to import the data into.
  • Select mapping options. The mapping settings tell CENTRO® Admin how to convert the data in the import file into WITSML trajectory stations. See "Selecting mapping options".

When you upload the import file, CENTRO® Admin will automatically detect the mnemonics used in the file.

You can also create a template to save the settings for mapping a Survey to WITSML data. After you create a template, you can use it the next you import a Survey. The mapping settings in the template will be applied to the new import file. See "Creating mapping templates" and "Applying a template".

Note

If a client company is assigned to your account, you will only be able to import data into wells that are associated with the same client company. See "User accounts".

Selecting the source file

To import a Survey:

  1. Go to the Well Data Import page.
  2. Click the Choose button, then select the file you want to import.
  3. Click the Process button to upload the file.
  4. In the Import To field, select the Trajectory option.
  5. Click the Next button at the bottom of the screen.
  6. Select the destination for the imported data, then click Next. See "Selecting the destination".
  7. Select the mapping options for the import file, then click Next. See "Selecting mapping options".
  8. Verify the range for the data and click the Import button.

The Survey data will be imported from the selected file and converted into trajectory stations for the selected WITSML Trajectory. The mnemonics in the file will be converted into WITSML curves, based on the mapping options you selected.

Selecting the destination

To select the destination for the imported data:

  1. In the Well Name and Wellbore fields, select the name of the WITSML well and wellbore for the Trajectory.
  2. Click the Next button at the bottom of the screen.
  3. Select the mapping options for the import file, then click Next. See "Selecting mapping options".

After you finish configuring the import, the selected file will be imported into the destination Trajectory.

Note

Existing WITSML Surveys can only be used if the WITSML object does not contain any data.

Selecting mapping options

To select the mapping options for a trajectory import:

  1. Review and update the list of mnemonics detected in the import file.
  2. For each mnemonic listed, select the equivalent WITSML mnemonic and unit.
  3. In the Station Type field, select type of WITSML trajectory station.
  4. In the Tie-In Point field, enter the measured depth of the tie-in point.

When you import the file, the settings for the survey mapping will be used to convert the data into WITSML trajectory stations. To save the mapping settings for future trajectory imports, you can create a mapping template. See "Creating mapping templates".

Creating mapping templates

To create a template for a Survey import:

  1. During the import process, select the mapping settings for the imported file. See "Selecting mapping options".
  2. Click the Create Template button () at the top of the Trajectory Mappings table.
  3. In the Mapping Template window, enter a name for the template and click Save.

All of the mapping settings will be saved to the Survey template. The next time you import a Survey file, you can apply the template to use the same settings again. See "Applying a template".

Applying a template

To apply a Survey template:

  1. On the Mapping screen, click the Preview Template button () at the top of the Trajectory Mappings table.
  2. Select the template you want to use, then click the Load Template button ().
  3. Click the Apply button.
  4. Review the mapping settings, then click Next to continue the import process.

The settings in the Survey template will be applied to mapping for the import file.

Field definitions

The table below defines the columns in the mnemonics table.

Field definitions for the mnemonics table
Column Name Definition
Import Indicates that the mnemonic will be imported into the WITSML trajectory. If the Import option is not selected, the mnemonic will be skipped during the import process.
LAS Mnemonic Name of the mnemonic in the LAS import file. Only available for LAS imports.
CSV Mnemonic Name of the mnemonic in the CSV import fle. Only available for CSV imports.
LAS Unit Unit used for the mnemonic in the LAS import file. Only available for LAS imports.
CSV Unit Unit used for the mnemonic in the CSV import file. Only available for CSV imports.
WITSML Mnemonic

Name of the equivalent WITSML mnemonic. CENTRO® Admin supports the following mnemonics for trajectory stations:

  • md – Measured depth of the measurement.
  • tvd – Vertical depth of the measurement.
  • incl – Hole inclination, measured from vertical.
  • azi – Hole azimuth, corrected to the well's azimuth reference.
  • dispNs – North-south offset, positive to the north.
  • dispEw – East-west offset, positive to the east.
  • vertSect – Distance along the vertical section.
  • dls – Dogleg severity.
WITSML Unit Unit associated with the WITSML mnemonic

Indexing options for surveys
