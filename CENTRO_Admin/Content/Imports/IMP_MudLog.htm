About mud log imports

On the Well Data Import page, you can import Mud Log data from an LAS file into a WITSML Mud Log. You must import Mud Log data into an existing WITSML Mud Log. There are three basic steps required to import a Mud Log from an LAS file:

  • Select the source. When you import the file, the data will be converted into WITSML geology intervals using the mapping settings. See "Selecting the source file".
  • Select the destination Mud Log. The destination settings tell CENTRO® Admin which well, wellbore, and WITSML Mud Log to import the data into.
  • Select mapping options. The mapping settings tell CENTRO® Admin how to convert the data in the import file into WITSML geology intervals. See "Selecting mapping options".

When you upload the import file, CENTRO® Admin will automatically detect the mnemonics used in the file.

You can also create a template to save the settings for mapping a Mud Log to WITSML data. After you create a template, you can use it the next you import a Mud Log. The mapping settings in the template will be applied to the new import file. See "Creating mapping templates" and "Applying a template".

Note

If a client company is assigned to your account, you will only be able to import data into wells that are associated with the same client company. See "User accounts".

Note

Mud logs can only be imported from an LAS file. CSV imports for mud logs are not currently supported.

Selecting the source file

To import a Mud Log:

  1. Go to the Well Data Import page.
  2. Click the Choose button, then select the file you want to import.
  3. Click the Process button to upload the file.
  4. In the Import To field, select the Mud Log option.
  5. Click the Next button at the bottom of the screen.
  6. Select the destination for the imported data, then click Next. See "Selecting the destination".
  7. Select the mapping options for the import file, then click Next. See "Selecting mapping options".
  8. Verify the range for the data and click the Import button.

The Mud Log data will be imported from the selected file and converted into geology intervals for the selected WITSML Mud Log. The mnemonics in the file will be converted into WITSML lithologies, based on the mapping options you selected.

Selecting the destination

To select the destination for the imported data:

  1. In the Well Name and Wellbore fields, select the name of the WITSML well and wellbore for the Mud Log.
  2. Click the Next button at the bottom of the screen.
  3. Select the mapping options for the import file, then click Next. See "Selecting mapping options".

After you finish configuring the import, the selected file will be imported into the destination Mud Log.

Note

Existing WITSML Mud Logs can only be used if the WITSML object does not contain any data.

Selecting mapping options

To select the mapping options for a mud log import:

  1. Review and update the list of mnemonics detected in the import file.
  2. For each mnemonic listed, select the equivalent WITSML lithology type.
  3. In the Index section below the mappings table, go to the Lithology Data Type field and select the source of the lithology data in the LAS file.
  4. In the LAS Index Unit field, enter the name of the unit used for the index curve.
  5. In the WITSML Index Unit field, select the equivalent WITSML unit.

When you import the LAS file, the settings for the mud log mapping will be used to convert the data into WITSML geology intervals. To save the mapping settings for future mud log imports, you can create a mud log template. See "Creating mapping templates".

Creating mapping templates

To create a template for a Mud Log import:

  1. During the import process, select the mapping settings for the imported file. See "Selecting mapping options".
  2. Click the Create Template button () at the top of the Mud Log Mappings table.
  3. In the Mapping Template window, enter a name for the template and click Save.

All of the mapping settings will be saved to the Mud Log template. The next time you import a Mud Log file, you can apply the template to use the same settings again. See "Applying a template".

Applying a template

To apply a Mud Log template:

  1. On the Mapping screen, click the Preview Template button () at the top of the Mud Log Mappings table.
  2. Select the template you want to use, then click the Load Template button ().
  3. Click the Apply button.
  4. Review the mapping settings, then click Next to continue the import process.

The settings in the Mud Log template will be applied to mapping for the import file.

Field definitions

The table below defines the fields in the mnemonics table.

Field definitions for the mnemonics table
Column Name Definition
LAS Lithology Numerical code used to represent a lithology type in the LAS file
WITSML Lithology Equivalent WITSML lithology type

The table below defines the index options for mud log imports.

Indexing options for mud logs
Column Name Definition
Lithology Data Type Source of the lithology data in the LAS file
LAS Index Unit Unit used for the index curve in the LAS file
WITSML Index Unit Equivalent WITSML unit for the index curve
