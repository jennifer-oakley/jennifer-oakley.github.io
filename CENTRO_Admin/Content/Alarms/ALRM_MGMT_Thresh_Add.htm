Creating a threshold alarm

To add a threshold alarm:

  1. Go to the Alarms > Alarms Management page.
  2. Click on the name of the well that you want to add an alarm to.
  3. Click the Add Alarm button () and select the Threshold option.
  4. On the left side of the screen, enter a name for the alarm in the Alarm Name field.
  5. To add conditional statements to the alarm, configure the If component. See "Configuring the If component".
  6. Add logic components to each conditional statement. See "Adding logic components".
  7. Add math components and WITSML mnemonics to the logic components. See "Adding math components" and "Adding alarm mnemonics".
  8. Configure the alarm actions for each conditional statement. See "Configuring alarm actions".
  9. Click the Create button in the upper right corner to save the alarm settings.
  10. Configure the subscription settings for the alarm. See "Subscribing to an alarm".

CENTRO® will evaluate the conditions defined for each conditional statement in the If component. If a condition is true, the alarm will be triggered using the settings in the action component. When the alarm is triggered, notifications will be sent based on the settings on the Subscription tab.

Configuring the If component

To configure the If component:

  1. Click on the Settings button () in the top left corner of the If component.
  2. In the window that appears, click on the component you want to add in the left side of the window.
    • The Else If component adds a conditional statement to the If component. If the condition defined for the initial If component is false, the alarm rule will check the condition defined for the Else If component.
    • The Else component adds an alarm action to the If component. If the condition defined for the initial If component is false, the alarm rule will execute the action defined for the Else component.
  3. Connect the component to the If component in the right side of the window.
  4. Click the Settings button () again to close the window.

Additional conditional statements and alarm actions will appear in the If component. For each conditional statement, you'll need to add logic components to define the logic for the condition. See "Adding logic components". For each alarm action, you'll need to add action components to define what happens when the alarm triggered. See "Configuring alarm actions".

Adding logic components

To add logic components:

  1. On the left side of the grid, click on the Logic heading and drag a logic component to the grid area. See "Logic components".
  2. Connect the logic component to a conditional statement.
  3. If the logic component contains an operator, click on the dropdown arrow to select the operator you want to use.

For most types of logic components, you'll need to add math and mnemonic components to define the values for the logic statement. See "Adding math components" and "Adding alarm mnemonics". You can also nest additional logic components inside another logic component. See "Adding threshold alarms".

Adding math components

To add math components:

  1. On the left side of the grid, click on the Math heading and drag any required math components to the grid area.
  2. Connect the math components to a logic component.
  3. If the math component contains an operator, click on the dropdown arrow to select the operator you want to use.

When CENTRO® evaluates the logic statement, it uses the math components and mnemonic components to determine if the conditional statement is true. If the conditional statement is true, the alarm will be triggered.

Adding alarm mnemonics

To add alarm mnemonics:

  1. On the left side of the screen, select the type of WITSML object that you are adding an alarm for.
  2. Select the WITSML objects that contain the curves for the alarm.
  3. Select the curves that you will need for the alarm. If you selected multiple WITSML logs, click the arrow next to the log name to select curves from that log.
  4. On the left side of the grid, click on the Alarm Mnemonics heading and drag the required mnemonic components to the grid area.
  5. Connect the mnemonic components to a logic component.

When CENTRO® evaluates the logic statement, it uses the math components and mnemonic components to determine if the conditional statement is true. If the conditional statement is true, the alarm will be triggered.

Configuring alarm actions

To configure the alarm actions:

  1. On the left side of the grid, click on the Alarm Actions heading and drag the action component to the grid area.
  2. Connect the action component to the rule.
  3. In the alarm component, click on the dropdown arrow to select the severity of the alarm.
  4. In the text box for the alarm condition, enter the message that should be displayed when the alarm is triggered.

CENTRO® will evaluate the conditions defined for each conditional statement in the If component. If a condition is true, the alarm will be triggered using the settings in the action component.
