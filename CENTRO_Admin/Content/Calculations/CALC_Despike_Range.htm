About index ranges

Index ranges define the time or depth range that the despike calculation will be applied to. By default, the first index range that you create will be applied to the entire log. However, you can create additional index ranges as needed. See "Adding index ranges".

Each index range also defines the curves that will be included in the despike calculation, as well as the limits and despiking strategy for each curve. See "Adding curves to an index range". Despiking calculations can be used to despike two types of values: false zeroes and invalid values.

A false zero is a single zero value that occurs between consecutive non-zero values. If multiple consecutive zero values are received, they are not considered to be false zeroes. There are two options for removing false zeroes in a curve:

  • NullValue – The NullValue option replaces a false zero with a null value.
  • LastValue – The LastValue option replaces a false zero with the last valid value.

Invalid values are values that fall outside the minimum or maximum value defined in the threshold limits. There are three options for using threshold limits to despike a curve:

  • NullValue – The NullValue option replaces an invalid value with a null value.
  • LastValue – The LastValue option replaces an invalid value with the last valid value.
  • LimitValue – The LimitValue option replaces an invalid value with the defined minimum value or maximum value. If the invalid value is below the minimum limit, the value defined in the Minimum Value field will be used. If the invalid value is above the maximum limit, the value defined in the Maximum Value field will be used.

Adding index ranges

To add an index range:

  1. Click the Add Index Range button (), located below the Mapping for Physical Measurements and Log Mnemonics section.
  2. In the Start field, enter the starting time or starting depth for the index range. The default value for the first index range is the start of the log.
  3. In the End field, enter the ending time or ending depth for the index range. If this field is blank, the index range will be applied until the end of the log.
  4. Add curves to the index range. See "Adding curves to an index range".

When you start the calculation, incoming values for the curves in the index range will be evaluated for invalid values. The despiking option selected for each curve will be applied to incoming values received within the specified time or depth range.

Adding curves to an index range

To add a curve to an index range:

  1. Go to the index range you want to add curves to and click the Add Curves button ().
  2. In the Physical Measurements field, select the curve that you want to despike.
  3. If you want to despike false zeroes for the selected curve, select the Despike False Zeroes option and the desired option for handling false zeroes.
  4. In the Minimum Limit field and the Maximum Limit fields, enter the minimum and maximum allowable values.
  5. In the Despiking Strategy field, select the desired option for handling invalid values.
  6. If you selected the LimitValue option, enter the minimum and maximum replacement values in the Minimum Value field and the Maximum Value field.

When you start the calculation, incoming values for the curve will be evaluated and replaced as needed, based on the selected despiking option.

Field definitions

The table below defines the fields in the Index Range section.

Field definitions for the Index Range section
Section Field Name Definition
-- Start Date/Depth Start time or depth for the index range
-- End Date/Depth End time or depth for the index range. If this value is blank, the settings for the index range will be applied until the end of the log
-- Physical Measurements Name of the curve that will be despiked
False Zeroes Despike False Zeroes? Option to include despiking false zeroes in the calculation. When selected, zero values that occur between consecutive non-zero values will be replaced with a null value or the last valid value, depending on the selected despiking strategy.
Despiking Strategy

Despiking strategy to use for replacing false zeroes.

  • NullValue – The NullValue option replaces a false zero with a null value.
  • LastValue – The LastValue option replaces a false zero with the last valid value.
Threshold Limits Minimum Limit Minimum allowable value for the curve. Incoming values that are below this limit will be replaced with a new value, based on the option selected in the Despiking Strategy field.
Maximum Limit Maximum allowable value for the curve. Incoming values that are above this limit will be replaced with a new value, based on the option selected in the Despiking Strategy field.
Despiking Strategy

Despiking strategy to use for replacing invalid values that exceed the allowable limits.

  • NullValue – The NullValue option replaces an invalid value with a null value.
  • LastValue – The LastValue option replaces an invalid value with the last valid value.
  • LimitValue – The LimitValue option replaces an invalid value with the defined minimum value or maximum value. If the invalid value is below the minimum limit, the value defined in the Minimum Value field will be used. If the invalid value is above the maximum limit, the value defined in the Maximum Value field will be used.
Minimum Value Replacement value to use when an incoming value falls below the Minimum Limit value. Only available when the LimitValue option is selected for the despiking strategy.
Maximum Value Replacement value to use when an incoming value falls above the Maximum Limit value. Only available when the LimitValue option is selected for the despiking strategy.

Using the Index Range section

The buttons in the Index Range section allow you to add, copy, and delete index ranges, as well as add and remove curves from the index range.

Buttons in the Index Range section
Button Function
Adds a new index range to the calculation. See "Adding index ranges".
Creates a copy of the selected index range
Deletes the selected index range. If only one index range exists, it cannot be deleted.
Adds a new curve to the selected index range. See "Adding curves to an index range".
Deletes the corresponding curve from the selected index range. If only one curve exists in the index range, it cannot be deleted.
