About despike calculations

On the Calculations > Despike page, you can configure calculations to automatically remove data spikes from a time-based or depth-based log. Despiking automatically removes false zeroes and invalid data points that exceed the defined limits for a curve.

When a despike calculation is running, CENTRO® will check the incoming values of the mnemonics specified in the despike calculation. If a false zero or an invalid value is received, it will be replaced with a new value, based on the despiking strategy selected for that mnemonic. See "Despike ranges".

Note

If you stop and restart calculations, calculations will resume at the last processed index point.

Note

The Calculations > Despike page only lists calculations for the wells that you have been granted access to.

Creating despike calculations

To create a despike calculation:

  1. Go to the Calculations > Despike page.
  2. Click the Create Calculation button () at the top of the page.
  3. On the New Despike Calculation page, go to the Well field and select the name of the well.
  4. In the Wellbore field, select the wellbore name.
  5. In the Log field, select the name of the log you want to despike.
  6. In the Mapping for Physical Measurements and Log Mnemonics section, map the mnemonics you want to despike to the appropriate WITSML measurement units. See "Adding mappings to a despike calculation".
  7. Create an index range for the time period or depth range that you want the despiking calculation to be applied to. See "Adding index ranges".
  8. For each index range, add the curves you want to evaluate for invalid values. See "Adding curves to an index range".
  9. Click the Save button () to save your changes, or click the Start button () to begin calculations.

When you start calculations, a new WITSML log will be created for the despike curves. The new log will have the same name as the original log, with _DS added to the end of the name, e.g. [OriginalLogName]_DS. The new calculation will be listed on the Calculations > Despike page. The Calculations > Despike page will show the current status of the calculation and the time that the calculated was created.

Exporting despike calculations

After you have saved a despike calculation, you can export it to a file and import it into another well. To export the current rig state mapping to a file:

  1. Go to the Calculations > Despike page.
  2. Click the Edit button () for the well that contains the calculations you want to export.
  3. Click the Export to Despike Calculation File button ().

The calculation file will be automatically downloaded to the Downloads directory on your computer.

Note

Calculation files can't be exported while a calculation is running.

Importing despike calculations

To import despike calculations from a file:

  1. Go to the Calculations > Despike page.
  2. Click the Create Calculation button () at the top of the page.
  3. On the New Despike Calculation page, select the well name and wellbore name.
  4. In the Log field, select the name of the log you want to despike.
  5. Click the Import From Despike Calculation File button ().
  6. In the Import Despike Calculation File window, click the Choose button.
  7. Navigate to file you want to import, then click Open.
  8. In the Import Despike Calculation File window, click the Overwrite button.
  9. Click the Save button () to save your changes, then click the Start button () to begin calculations.

The settings for the despike calculation will be imported from the selected file. If any of the log mnemonics in the imported file do not match the log mnemonics for the current log, you will need to manually configure those mnemonics. See "Despike mappings".

Using the Despike page

The search box above the table on the Despike page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the Despike page are used to add, start, stop, and delete despike calculations.

Buttons on the Despike page
Button Function
Adds a new despike calculation. See "Creating despike calculations".
Refreshes the status of the calculations on the Despike module
Starts the corresponding despike calculation
Stops the corresponding despike calculation
Allows you to view and edit the corresponding despike calculation
Creates a copy of the corresponding despike calculation
Deletes the corresponding despike calculation. If the calculation is currently running, you must stop the calculation, before you can delete it.

When you add or edit a despike calculation, the buttons on the Edit Despike Calculation page allow you to start and stop the calculation, as well an export and import the calculation settings. When you expand the Mapping for Physical Measurements and Log Mnemonics section or the Index Range section, additional buttons will appear. See "Using the Mappings section" and "Using the Index Range section".

Buttons on the Edit Despike Calculation page
Button Function
Saves all changes
Save the current calculation settings and starts calculations
Exports the current mappings to a file. See "Exporting despike calculations".
Imports a despike calculation file into the current mappings. See "Importing despike calculations".
Stops the despike calculation
Deletes the existing output log and restarts the calculation into a new log, from the start time.
