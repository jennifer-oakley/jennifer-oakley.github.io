About mappings

The Calculations > Mappings page shows the mappings between the curves for different types of physical measurements and the corresponding WITSML unit class. You can also define the WITSML mnemonics used in your data for these curves. When you create a despike calculations, these mappings are used to map the mnemonics in the despike calculation to the curves for physical measurements. See "Despike calculations".

To filter the list of mappings, enter a keyword in the Search field at the top of the Physical Measurements column or the WITSML Class column.

Defining mnemonics

To define a mnemonic used for a curve:

  1. Go to the Calculations > Mappings page.
  2. Locate the name of the curve in the Physical Measurements column.
  3. In the Curve/Mnemonic Names column, enter the mnemonics used for the curve. If you are entering multiple mnemonics for the curve, use commas to separate the mnemonic names.
  4. Click the Save Mappings button () at the top of the page.

The mnemonic will be mapped to the corresponding curve. When you add the mnemonic to a despike calculation, the mapping will be automatically populated in the Mapping for Physical Measurements and Log Mnemonics section. See "Despike mappings".

Importing mappings

If you have previously exported the mappings from another server, you can import them. To import a mappings file:

  1. Go to the Calculations > Mappings page.
  2. Click the Import Mappings button () at the top of the page.
  3. In the Import Mappings File window that appear, click the Choose button and navigate to the file you want to import.
  4. Select the file and click Open.
  5. To overwrite all existing data, click the Overwrite button. To merge the mappings in the import file with your existing settings, click the Merge button.

If you selected the Overwrite option, all existing mappings will be deleted and replaced with the mappings in the import file. If you selected the Merge option, the mappings in the import file will be added to the existing mappings.
