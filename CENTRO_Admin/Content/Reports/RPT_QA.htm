About QA/QC reports

On the Reports > QA/QC page, you can generate a data quality report to identify missing values in your WITSML data. When you create a QA/QC report, the report analysis will look for gaps in the data that exceed the value in the Gap Size field. These gaps will be used to determine the overall quality rating for the WITSML data.

Each QA/QC report can include multiple wells, as well as multiple logs for each well. The completed report will be divided into sections for each well included in the report. See "Report details". To save a completed report as a PDF file, click the Save As PDF button () when you are viewing the report.

Note

The Reports > QA/QC page is only available on the data center.

Creating QA/QC reports

To create a QA/QC report:

  1. Go to the Reports > QA/QC page in the Datacenter menu.
  2. Click the Create Report button () at the top of the page.
  3. Enter a name for the report in the Report Name field.
  4. Select a start and end time for the report to run in the Report Start Date field and the Report End Date field.
  5. Select the wells you want to include in the report in the Available Wells list, then click the Add button ().
  6. Click the Get Log Details button to load the log information for the selected wells.
  7. In the Log Details pane, select the logs that you want to include in the report.
  8. For each selected log, enter the minimum threshold to use for identifying gaps in the Gap Size field.
  9. Click the Generate Report button to create the report.

The new report will appear in the list of reports on the QA/QC page. When the report finishes processing, the status will change to Completed. To open the report, click the View Report button (). See "Report details". To save a completed report as a PDF file, click the Save As PDF button () when you are viewing the report.

Tip

To select multiple wells at the same time, press the Ctrl key and select the wells you want to add.

Report details

QA/QC reports are divided into different sections for each well included in the report. The table below defines the report details included for each well. The section header for each well and wellbore also includes the overall quality ratings for all time and depth logs included in the report.

Contents of a QA/QC report
Column Definition
Log Name Name of the log included in the report
Data Service Start Index Start time of the report analysis for the log. By default, this is the first time stamp or depth in the log.
Data Service End Index End time of the report analysis for the log. By default, this is the last time stamp or depth in the log.
Gap Size Criteria Threshold used to identify gaps in the log
# of Gaps Number of gaps detected in the log
Gap Span Combined length of all gaps detected in the log
Total Span Total length of the data analyzed for the log
Quality Rating Quality rating for the log, determined by comparing the gap span to the total span

If a gap was detected for one of the well's logs, the Expand button () will appear next to the name of the log. To view details about the gaps detected, click the Expand button ().

Details for gaps detected in a log
Column Definition
Gap Start Index Start time of the gap
Gap End Index End time of the gap
Gap Span Length of the gap

Using the QA/QC page

The search box above the table on the QA/QC page allows you to filter the rows displayed. To view additional rows of data, click on the page numbers or arrows at the bottom of the table. To change the number of rows shown on each page, use the dropdown menu next to the page numbers.

The buttons on the QA/QC page are used to add, view, copy, and delete reports.

Buttons on the Reports > QA/QC page
Button Function
Adds a new QA/QC report
Reloads the list of reports
Opens the selected QA/QC report
Creates a copy of the selected QA/QC report
Deletes the selected report from the server

When you add a QA/QC report, the buttons on the Generate Report page allow you to add and remove wells from the report.

Buttons on the Generate Report page
Button Function
Adds the selected well to the report
Adds all available wells to the report
Removes the selected wells from the report
Removes all wells from the report
Get Log Details Loads the log information for the selected wells
Generate Report Creates the report and begins the report analysis
