About user and well reports

The Reports > User/Well page allows you to create reports showing access to wells, well status, and user status. Once you have created a report, you can save it in CSV, Excel, or PDF format. Currently, there are four reports available:

  • The User Well Access report shows the list of users that have access to a well, either directly through their user account or through a role. See "Creating a well access report".
  • The Well Status report shows the number of days since a well has been active. See "Creating a well status report".
  • The User Status report shows the number of days since a user was last active. See "Creating a user status report".
  • The Well Audit report shows a list of days on which a well was active. See "Creating a well audit report".
  • The Well Activity report shows a list of wells that were active within the selected time frame, as well as the days on which they were active. See "Creating a well activity report".

The Reports > User/Well page is only available for the server that you are logged into. If you are logged into the data center, you cannot create reports for any of the data hubs connected to it. To create reports for a data hub, you will need to log into the data hub directly.

Creating a well access report

To create a well access report:

  1. Go to the Reports > User/Well page.
  2. At the top of the page, go to the Reports field and select the User Well Access option.
  3. In the Select Well(s) field, select the wells that you want to generate a report for.
  4. Click the Create Report button ().

A list of users that have access to the selected wells will appear in the report table. This list includes all users that have been granted access to the selected wells on the User Management > WITSML ACLs page. Users that are assigned to a role that has been granted access will also be included. To export a copy of the report, click the CSV button, the Excel button, or the PDF button.

Creating a well status report

To create a well status report:

  1. Go to the Reports > User/Well page.
  2. At the top of the page, go to the Reports field and select the Well Status option.
  3. In the Well Age Filter field, enter the maximum well age that you want to include in the report. Well age is the number of days since data was received for the well.
  4. Click the Create Report button ().

A list of wells will appear in the report table. This list includes all wells that have received data within the time frame specified in the Well Age Filter field. To export a copy of the report, click the CSV button, the Excel button, or the PDF button.

Creating a user status report

To create a user status report:

  1. Go to the Reports > User/Well page.
  2. At the top of the page, go to the Reports field and select the User Status option.
  3. In the field Last x days field, enter the maximum access age that you want to include in the report. Access age is the number of days since the user has logged in.
  4. Click the Create Report button ().

A list of users that have logged in within the specified time frame will appear in the report table. To export a copy of the report, click the CSV button, the Excel button, or the PDF button.

Creating a well audit report

To create a well audit report:

  1. Go to the Reports > User/Well page.
  2. At the top of the page, go to the Reports field and select the Well Audit option.
  3. In the Select Well field, select the well that you want to view activity dates for.

A list of the activity dates for the well will automatically appear in the report table. Activity dates are dates that data was received for the well. To export a copy of the report, click the CSV button, the Excel button, or the PDF button.

Creating a well activity report

To create a well activity report:

  1. Go to the Reports > User/Well page.
  2. At the top of the page, go to the Reports field and select the Well Activity option.
  3. In the From field and the To field, select the range of months that you want to view well activity for.
  4. Click the Create Report button ().

A list of wells that were active within the specified time frame will appear in the report table. The report will also include the client, rig, country, and activity dates for each well in the report. Activity dates are dates that data was received for the well. To export a copy of the report, click the CSV button, the Excel button, or the PDF button.
